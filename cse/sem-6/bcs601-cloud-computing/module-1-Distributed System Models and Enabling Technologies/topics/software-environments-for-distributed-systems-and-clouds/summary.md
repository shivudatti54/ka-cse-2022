This module covered software environments for distributed systems and clouds, including: (1) Key characteristics like elasticity, resilience, and multi-tenancy with mathematical formulations; (2) Layered architecture from physical infrastructure to application layer; (3) Comparative analysis of IaaS, PaaS, and FaaS with technical depth on control levels, scaling, and costs; (4) Container ecosystem with Docker architecture and Kubernetes orchestration details; (5) Performance, security, and energy efficiency trade-offs with quantified overheads; (6) Cloud provider ecosystems and distributed computing frameworks. Three numerical problems test application of concepts to real scenarios.
