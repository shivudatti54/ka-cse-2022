1. **Three evolutionary models**: Cluster (tight coupling, single site), Grid (loose coupling, heterogeneous, cross-organizational), and Cloud (virtualized, elastic, service-oriented)
2. **Theoretical foundations**: Amdahl's Law for parallel speedup, CAP Theorem for distributed trade-offs
3. **Enabling technologies**: Virtualization (Type 1/2 hypervisors), Containerization (Docker, Kubernetes), SOA (REST, SOAP), Utility/Autonomic Computing, High-speed networks
4. **Service models**: IaaS, PaaS, SaaS represent increasing abstraction levels
5. **Comparative analysis**: Each model differs in geographic scope, heterogeneity, management, and business models
