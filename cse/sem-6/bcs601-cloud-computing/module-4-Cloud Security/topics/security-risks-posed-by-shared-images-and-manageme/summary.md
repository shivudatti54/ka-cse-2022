**Core Risks from Shared Images**: Malware/backdoor insertion, outdated vulnerable software (CVEs), hardcoded secrets (credentials, API keys), lack of provenance verification and image signing. **Core Risks from Management OS/Hypervisor**: VM escape attacks enabling cross-tenant compromise, hypervisor vulnerabilities (specific CVEs cited), tenant isolation failures, side-channel attacks. **Key Mitigation**: Prefer provider-official images, scan/harden before deployment, implement secret injection rather than hardcoding, understand Shared Responsibility Model boundaries, deploy network segmentation and monitoring. **Standards Reference**: NIST SP 800-144 provides guidelines for public cloud security; CIS Benchmarks offer configuration hardening standards.
