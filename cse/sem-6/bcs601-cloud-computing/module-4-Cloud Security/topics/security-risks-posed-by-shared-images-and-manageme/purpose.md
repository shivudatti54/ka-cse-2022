This module provides B.Tech/MSc/MCA students with a comprehensive understanding of security risks in cloud IaaS environments, specifically addressing threats from shared machine images and the management OS/hypervisor layer. It establishes theoretical foundations through formal threat modeling using STRIDE, presents rigorous technical analysis with proof-of-concept concepts and theorems, and connects theoretical knowledge to practical mitigation strategies aligned with NIST SP 800-144. The content prepares students to design secure cloud architectures and respond to sophisticated attacks including VM escape scenarios.
