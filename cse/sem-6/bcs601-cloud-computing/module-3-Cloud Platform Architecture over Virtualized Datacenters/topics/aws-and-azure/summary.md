This module covers cloud programming on AWS and Azure, beginning with fundamental service models (IaaS, PaaS, SaaS) and the shared responsibility model. Core AWS services are examined including EC2 for virtual servers, Lambda for serverless computing, S3 for object storage, and DynamoDB/RDS for databases—each with practical Python code examples. Azure coverage includes Virtual Machines, Azure Functions, Blob Storage, and database services with corresponding SDK demonstrations. Infrastructure as Code is explored through AWS CloudFormation YAML templates and Azure ARM JSON templates. A detailed service comparison table highlights equivalent offerings across platforms. Authentication mechanisms (IAM roles, Azure Managed Identities) are explained. The assessment includes four multiple-choice questions requiring architectural analysis, two short answer questions testing conceptual understanding, and one numerical problem calculating Lambda cost dynamics.
