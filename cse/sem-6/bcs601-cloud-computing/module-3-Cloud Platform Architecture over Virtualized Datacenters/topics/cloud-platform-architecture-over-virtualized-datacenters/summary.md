This module examined the architectural foundations of cloud computing, focusing on virtualization technology as the cornerstone enabling elastic resource delivery. Key topics included hypervisor architecture (Type 1 vs Type 2), virtual machine components, live migration, multi-tenancy security, and network virtualization (VLANs and VXLAN). The module established mathematical models for resource consolidation and virtualization overhead analysis. Students learned to differentiate between IaaS, PaaS, and SaaS service models through the lens of resource management responsibilities. The analytical frameworks provided enable quantitative evaluation of cloud architectures, preparing learners for advanced study in distributed systems and professional cloud certifications.
