This module covers cloud computing fundamentals including the NIST definition, five essential characteristics (on-demand self-service, broad network access, resource pooling, rapid elasticity, measured service), and the SPI service model framework (IaaS/PaaS/SaaS). Cloud deployment models (public, private, hybrid, community) are analyzed with use cases. The content includes detailed comparison tables, multi-tenant architecture principles, SLA considerations, and cloud cost economics. Students gain theoretical knowledge for analyzing service model selection based on organizational requirements, control needs, and economic factors.
