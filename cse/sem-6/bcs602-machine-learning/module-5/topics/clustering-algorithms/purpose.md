# Learning Objectives

After studying this topic, you should be able to:

1. Define clustering and explain its role as an unsupervised learning technique
2. Classify clustering algorithms into major categories: partitional, hierarchical, density-based, and grid-based
3. Compare the characteristics, advantages, and limitations of each clustering category
4. Understand the key parameters and considerations for choosing a clustering algorithm
5. Describe evaluation metrics for clustering: Silhouette Score, Davies-Bouldin Index, and Dunn Index
6. Identify appropriate clustering algorithms for different data characteristics and application scenarios
