# Learning Objectives

After studying this topic, you should be able to:

1. **Identify** the three main paradigms of machine learning and their positions in the ML taxonomy
2. **Distinguish** between supervised, unsupervised, and reinforcement learning based on feedback type, data characteristics, and learning objectives
3. **Explain** why reinforcement learning is uniquely suited for sequential decision-making problems
4. **Analyze** the exploration-exploitation trade-off and its significance in RL
5. **Determine** when reinforcement learning is the appropriate choice compared to other ML paradigms
6. **Describe** the core components of the reinforcement learning framework (agent, environment, state, action, reward, policy, value function)
