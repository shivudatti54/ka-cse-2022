# Learning Objectives

After studying this topic, you should be able to:

1. **Identify and explain** the eight core components of a reinforcement learning system: Agent, Environment, State, Action, Reward, Policy, Value Function, and Model

2. **Understand the interaction** between the agent and environment in the reinforcement learning loop

3. **Differentiate** between key concepts such as reward versus value, and deterministic versus stochastic policies

4. **Explain the role** of the value function in estimating long-term returns and how it differs from immediate rewards

5. **Distinguish** between model-based and model-free reinforcement learning approaches

6. **Apply** the component framework to analyze real-world reinforcement learning problems
