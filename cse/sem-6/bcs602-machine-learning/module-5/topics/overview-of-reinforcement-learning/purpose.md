# Learning Objectives

After studying this topic, you should be able to:

1. **Define Reinforcement Learning** and explain how it differs from supervised and unsupervised learning paradigms

2. **Identify the core components** of an RL system including the agent, environment, states, actions, rewards, and policies

3. **Explain the trial-and-error learning process** and how agents learn from feedback through environmental interactions

4. **Understand the exploration-exploitation dilemma** and describe common strategies for balancing them

5. **Calculate returns** using reward summation with and without discount factors

6. **Apply the RL framework** to real-world problems and identify potential applications
