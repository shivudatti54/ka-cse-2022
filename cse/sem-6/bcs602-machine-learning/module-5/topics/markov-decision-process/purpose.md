# Learning Objectives

After studying this topic, you should be able to:

1. Define and explain the Markov Property and its significance in sequential decision-making problems.

2. Identify and describe the five components of a Markov Decision Process: States, Actions, Transition Probabilities, Rewards, and Discount Factor.

3. Differentiate between deterministic and stochastic policies in MDPs.

4. Explain the concepts of state value function V(s) and action value function Q(s, a) with their mathematical formulations.

5. Apply the Bellman equations to compute value functions for given MDPs.

6. Understand the concept of optimal policy and how it relates to optimal value functions.

7. Analyze simple MDP problems and compute optimal decisions for given scenarios.

8. Connect MDP theory to reinforcement learning algorithms and real-world applications.
