# Model-based Learning

## Summary

Model-based learning is a type of reinforcement learning that uses a model of the environment to make predictions about future states and rewards. It involves two main components: model learning and planning. Model-based learning can be more sample-efficient than model-free methods, but it can also be biased towards the learned model.

### Important Formulas, Definitions, and Theorems

- Model-based learning: A type of reinforcement learning that uses a model of the environment to make predictions about future states and rewards.
- Model learning: The process of learning a model of the environment.
- Planning: The process of using the learned model to make decisions.
- Value iteration: A technique used to compute the value function.
- Policy iteration: A technique used to compute the policy.

### Key Points

- Model-based learning uses a model of the environment to make predictions about future states and rewards.
- Model-based learning involves two main components: model learning and planning.
- Model-based learning can be more sample-efficient than model-free methods.
- Model-based learning can be biased towards the learned model.
- Model-based learning has been applied to a wide range of domains, including robotics and finance.
