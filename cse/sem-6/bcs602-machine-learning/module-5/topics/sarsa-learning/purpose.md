# Learning Objectives

After studying this topic, you should be able to:

1.  Define SARSA learning and its components.
2.  Explain how the SARSA learning algorithm works.
3.  Write the mathematical representation of the SARSA learning algorithm.
4.  Apply SARSA learning to a simple problem, such as the grid world example.
5.  Compare SARSA learning with other reinforcement learning algorithms, such as Q-learning.
6.  Identify the advantages and disadvantages of SARSA learning.
7.  Implement SARSA learning in a programming language, such as Python.
