# Learning Objectives

After studying this topic, you should be able to:

1. Define the Nearest Centroid Classifier and explain its fundamental working principle of classifying data points based on their proximity to class centroids.

2. Compute class centroids from a given training dataset by calculating the mean vector for each class.

3. Apply Euclidean distance or alternative distance metrics to determine the nearest centroid for test data points.

4. Explain how decision boundaries are formed in the Nearest Centroid Classifier and identify them as linear hyperplanes.

5. Compare and contrast the Nearest Centroid Classifier with other similarity-based learning algorithms such as k-Nearest Neighbors.

6. Analyze the assumptions underlying the Nearest Centroid Classifier, including the requirement for linearly separable classes with similar covariances.

7. Implement the shrunk Nearest Centroid Classifier and understand how regularization through shrinkage improves generalization.

8. Evaluate the computational advantages of the Nearest Centroid Classifier in terms of storage requirements and prediction time complexity.
