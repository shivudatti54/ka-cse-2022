# Learning Objectives

After studying this topic, you should be able to:

1. Define logistic regression and explain why it is used for classification despite its name
2. Describe the sigmoid (logistic) function and how it maps values to probabilities
3. Derive and explain the logistic regression model equation
4. Understand Maximum Likelihood Estimation (MLE) and the log-loss cost function
5. Interpret logistic regression coefficients in terms of odds ratios
6. Extend logistic regression to multi-class classification using one-vs-rest or softmax
