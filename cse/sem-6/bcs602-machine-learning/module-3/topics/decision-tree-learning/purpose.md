# Why Study Decision Trees?

## Big Picture Foundation

Decision trees are **highly interpretable** models that form the foundation for powerful ensemble methods like Random Forest and Gradient Boosting. Understanding them is essential for both explainable AI and advanced ML.

## Exam Relevance

| Aspect         | Details                                                        |
| -------------- | -------------------------------------------------------------- |
| Frequency      | Very common in ML exams                                        |
| Marks          | 8-12 marks (algorithm, calculations, pruning)                  |
| Question Types | Explain algorithm, calculate information gain, discuss pruning |

## Practical Value

- **Interpretability**: Can explain decisions to stakeholders
- **Feature Importance**: Built-in ranking of feature relevance
- **Versatility**: Handles classification and regression
- **Foundation for Ensembles**: Random Forest, XGBoost use trees

## Learning Outcomes

After this topic, you should be able to:

1. Explain the decision tree construction algorithm
2. Calculate Gini impurity and information gain
3. Compare Gini vs entropy as splitting criteria
4. Describe pruning techniques to prevent overfitting
5. Understand limitations and when to use ensembles
