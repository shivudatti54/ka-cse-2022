# Learning Objectives

After studying this topic, you should be able to:

1. Define what a decision tree learning model is and describe its basic structure
2. Explain how decision trees represent classification and regression problems
3. Describe the key terminology: root node, internal nodes, branches, and leaf nodes
4. Understand the concept of feature-based splitting and recursive partitioning
5. Explain why decision trees are considered interpretable and transparent models
6. Identify appropriate problem types for decision tree learning
