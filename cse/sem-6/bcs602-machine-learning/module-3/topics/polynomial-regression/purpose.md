# Learning Objectives

After studying this topic, you should be able to:

1. Define polynomial regression and explain when it is used over linear regression
2. Write the mathematical formulation of polynomial regression for different degrees
3. Understand how polynomial regression is still a linear model in terms of parameters
4. Explain the relationship between polynomial degree and model complexity
5. Describe the risk of overfitting with high-degree polynomials and mitigation strategies
6. Compare polynomial regression with other non-linear regression approaches
