# Learning Objectives

After studying this topic, you should be able to:

1. Define regression analysis and explain its role in supervised learning
2. Differentiate between regression and classification problems
3. Describe different types of regression: linear, polynomial, logistic, and non-linear
4. Explain the concept of dependent and independent variables in regression
5. Understand the general goals of regression: prediction, inference, and relationship modeling
6. Identify real-world scenarios where regression analysis is appropriate
