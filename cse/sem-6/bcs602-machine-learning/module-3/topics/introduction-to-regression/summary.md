# Introduction to Regression

## Overview

Regression is a supervised learning technique used to predict continuous numerical values. Unlike classification which predicts discrete categories, regression models the relationship between input features and continuous output targets, making it fundamental for forecasting and prediction tasks.

## Key Points

- **Purpose**: Predict continuous outcomes (house prices, temperature, sales) rather than discrete categories
- **Linear vs Non-Linear**: Linear regression assumes straight-line relationship; non-linear uses curves/polynomials
- **Regression vs Classification**: Regression outputs continuous values; classification outputs discrete categories
- **Applications**: Price prediction, demand forecasting, risk assessment, trend analysis, time-series forecasting
- **Evaluation Metrics**: MSE (Mean Squared Error), RMSE (Root MSE), MAE (Mean Absolute Error), R² (proportion of variance explained)
- **Key Assumption**: Relationship exists between independent variables (features) and dependent variable (target)

## Important Concepts

- Dependent variable (y): target to predict; Independent variables (X): predictors/features used for prediction
- Best-fit line/curve minimizes prediction error (distance between predicted and actual values)
- Overfitting occurs when model memorizes training data; underfitting when model too simple
- Regularization (L1/L2) prevents overfitting by penalizing complex models

## Notes

- Know the difference between regression and classification with examples
- Memorize evaluation metrics: MSE, RMSE, MAE, R² and when to use each
- Be able to explain linear vs non-linear regression with diagrams
- Understand common applications: price prediction, sales forecasting, temperature prediction
