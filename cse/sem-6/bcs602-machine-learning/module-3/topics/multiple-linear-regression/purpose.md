# Learning Objectives

After studying this topic, you should be able to:

1. Define multiple linear regression and differentiate it from simple linear regression
2. Write the mathematical formulation of multiple linear regression
3. Explain the ordinary least squares (OLS) method for estimating coefficients in multiple regression
4. Describe the role of each coefficient and the intercept in the regression equation
5. Understand multicollinearity, its detection, and its effects on the model
6. Apply and interpret evaluation metrics: R-squared, Adjusted R-squared, and F-statistic
