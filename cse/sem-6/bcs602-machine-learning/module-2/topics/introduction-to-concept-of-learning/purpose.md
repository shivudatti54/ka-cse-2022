# Learning Objectives

After studying this topic, you should be able to:

1. Define learning in the context of machine learning using Tom Mitchell's formal definition
2. Explain the three components of learning: Task (T), Performance measure (P), and Experience (E)
3. Differentiate between rote learning and generalization-based learning
4. Describe the concept of inductive learning and its assumptions
5. Explain the role of hypothesis in learning and how models improve through experience
6. Understand different learning paradigms: supervised, unsupervised, and reinforcement learning at a conceptual level
