# Why Study Multivariate Statistics?

## Big Picture Foundation

Machine learning algorithms operate on multivariate data. Understanding multivariate statistics gives you the mathematical language to describe how features relate to each other (covariance matrix), how to measure distances properly (Mahalanobis), and how to model data distributions (multivariate normal). Without this foundation, concepts like PCA, Gaussian Naive Bayes, and linear discriminant analysis remain opaque.

## Exam Relevance

| Aspect         | Details                                                                       |
| -------------- | ----------------------------------------------------------------------------- |
| Frequency      | High — covariance matrix, Mahalanobis distance appear in most exams           |
| Marks          | 5-10 marks (matrix computation, distance formulas, distribution concepts)     |
| Question Types | Compute covariance matrix, explain Mahalanobis distance, compare ANOVA/MANOVA |

## Practical Value

- **Feature analysis**: Covariance and correlation matrices reveal feature relationships before modeling.
- **Outlier detection**: Mahalanobis distance identifies multivariate outliers that univariate methods miss.
- **Model assumptions**: Many ML algorithms assume multivariate normality; knowing how to check this matters.
- **Dimensionality understanding**: Principal component intuition explains why some features can be dropped.

## Learning Outcomes

After this topic, you should be able to:

1. Compute mean vectors and covariance matrices from data
2. Explain the difference between covariance matrix and correlation matrix
3. Calculate and interpret Mahalanobis distance
4. Describe the multivariate normal distribution and its parameters
5. Explain the intuition behind principal components
6. Compare Hotelling's T-squared test with the univariate t-test
7. Differentiate between ANOVA and MANOVA
