# Why Study Modelling in Machine Learning?

## Big Picture Foundation

Modelling is the **central activity** in machine learning. Understanding what a model is, how it is represented, and how it is optimized forms the basis for every ML algorithm you will study. Without this foundation, topics like regression, classification, and neural networks lack context.

## Exam Relevance

| Aspect         | Details                                                                 |
| -------------- | ----------------------------------------------------------------------- |
| Frequency      | Very common in exams — parametric vs non-parametric, bias-variance      |
| Marks          | 5-10 marks (definitions, comparisons, loss functions, gradient descent) |
| Question Types | Short answers, comparison tables, mathematical derivations, diagrams    |

## Practical Value

- **Model selection**: knowing parametric vs non-parametric helps choose the right algorithm for the data size and problem type.
- **Debugging models**: understanding bias-variance tradeoff helps diagnose underfitting and overfitting.
- **Optimization literacy**: gradient descent is used in virtually every modern ML system from linear regression to deep learning.

## Learning Outcomes

After this topic, you should be able to:

1. Define what a model is and explain hypothesis functions with parameters
2. Compare parametric and non-parametric models with examples
3. Explain the bias-variance tradeoff and its impact on model complexity
4. Write loss functions for regression (MSE) and classification (cross-entropy)
5. Describe the gradient descent optimization process
