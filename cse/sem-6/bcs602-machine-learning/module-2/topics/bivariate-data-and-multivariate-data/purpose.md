# Why Study Bivariate Data and Multivariate Data?

## Big Picture Foundation

Univariate analysis examines one variable at a time, but real-world ML problems involve many interacting features. Understanding bivariate and multivariate data analysis bridges the gap between single-variable exploration and building multi-feature ML models. You cannot perform meaningful feature selection, detect multicollinearity, or interpret model coefficients without this foundation.

## Exam Relevance

| Aspect         | Details                                                                 |
| -------------- | ----------------------------------------------------------------------- |
| Frequency      | High — correlation, scatter plots, and causation appear regularly       |
| Marks          | 5-10 marks (formulas, worked examples, visualization descriptions)      |
| Question Types | Compute correlation, explain correlation vs. causation, compare methods |

## Practical Value

- **Feature selection**: Identify which variables are strongly correlated with the target before model training.
- **Multicollinearity detection**: Spot highly correlated input features that can destabilize regression models.
- **Data storytelling**: Communicate insights from multi-dimensional data using appropriate visualizations.
- **Better preprocessing**: Understand variable relationships to make informed imputation and transformation decisions.

## Learning Outcomes

After this topic, you should be able to:

1. Compute and interpret covariance, Pearson correlation, and Spearman rank correlation
2. Distinguish between correlation and causation with real-world examples
3. Select appropriate visualization techniques for bivariate and multivariate data
4. Explain how bivariate analysis extends to multivariate scenarios using matrices
5. Describe multivariate visualization methods like parallel coordinates and heatmaps
