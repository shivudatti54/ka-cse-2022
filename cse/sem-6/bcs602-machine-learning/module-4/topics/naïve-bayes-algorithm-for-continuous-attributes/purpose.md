# Learning Objectives

After studying this topic, you should be able to:

1. **Understand** the fundamental differences between handling continuous and discrete attributes in Naïve Bayes classification, specifically why probability density functions are required for continuous features.

2. **Explain** the concept of Probability Density Function (PDF) and its role in calculating likelihood for continuous attributes, distinguishing between PDF and Probability Mass Function (PMF).

3. **Apply** the Gaussian (Normal) distribution formula to estimate class-conditional probabilities for continuous features, computing mean and variance parameters from training data.

4. **Analyze** how the independence assumption in Naïve Bayes affects classification performance when features are correlated, and explain why the classifier often performs well despite this assumption.

5. **Implement** the Naïve Bayes classifier for continuous attributes using the Gaussian Naïve Bayes model in Python with scikit-learn.

6. **Compare** the advantages and limitations of Gaussian Naïve Bayes versus other classification algorithms for continuous data, including computational complexity and distributional assumptions.

7. **Evaluate** the performance of a Naïve Bayes classifier using appropriate metrics such as accuracy, precision, recall, F1-score, and confusion matrix.

8. **Design** end-to-end classification solutions for real-world problems involving continuous attributes (e.g., credit approval, medical diagnosis) including data preprocessing, model training, and evaluation.
