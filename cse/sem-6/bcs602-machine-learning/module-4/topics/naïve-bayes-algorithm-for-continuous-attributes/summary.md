# Summary

This topic covered the Gaussian Naïve Bayes classifier for continuous attributes. The key theoretical concept is that continuous features require probability density functions (PDFs) rather than discrete probabilities. The Gaussian (Normal) distribution is used to model class-conditional densities, with parameters estimated via maximum likelihood from training data. The classifier applies Bayes theorem with the independence assumption, computing posterior probabilities proportional to the product of likelihood and prior. For classification, the class with highest posterior probability is selected, typically implemented using log-probabilities to prevent numerical underflow. Advantages include computational efficiency and handling high-dimensional data, while limitations include the restrictive independence and Gaussian assumptions. Implementation in Python uses scikit-learn's GaussianNB class, with performance evaluated through accuracy, precision, recall, F1-score, and confusion matrices.
