# Classification Using Bayes Model

### Definition

- Classification is a type of supervised learning where the goal is to predict a categorical label or class for a given input.
- Bayes model is a probabilistic framework that uses Bayes' theorem to classify data.

### Key Concepts

- **Bayes' Theorem**: P(A|B) = P(B|A) \* P(A) / P(B)
  - P(A|B): Posterior probability of class A given class B
  - P(B|A): Likelihood of class B given class A
  - P(A): Prior probability of class A
  - P(B): Prior probability of class B
- **Naive Bayes**: An implementation of Bayes model that assumes independence between features
- **Bernoulli Naive Bayes**: Used for binary classification problems
- **Multinomial Naive Bayes**: Used for multi-class classification problems

### Important Formulas

- **Probability Mass Function (PMF)**: Defines the probability of a single data point belonging to a particular class
- **Likelihood Function**: The probability of observing the data given a particular class
- **Prior Distribution**: The probability of a class before observing any data

### Theorems

- **Bayes' Theorem**: A cornerstone of probabilistic reasoning in classification
- **Maximum Likelihood Estimation (MLE)**: The method of estimating parameters from data by maximizing the likelihood function

### Key Points for Revision

- Understand Bayes' theorem and its application in classification
- Familiarize yourself with naive Bayes and its variants
- Know the importance of prior distributions and likelihood functions
- Review MLE for estimating parameters
