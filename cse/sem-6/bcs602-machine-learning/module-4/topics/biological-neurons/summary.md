# Biological Neurons

## Overview

Biological neurons are the fundamental units of the brain's information processing system. Understanding their structure and function provides the inspiration for artificial neural networks in machine learning. They communicate through electrochemical signals, forming complex networks capable of learning and adaptation.

## Key Points

- **Structure**: Dendrites (receive signals), Cell body/Soma (processes), Axon (transmits), Synapses (connections to other neurons)
- **Signal Processing**: Dendrites collect inputs from multiple neurons; soma integrates signals; if threshold exceeded, neuron fires action potential down axon
- **Synaptic Transmission**: Electrical signal triggers chemical neurotransmitter release at synapse; received by next neuron's dendrites; synaptic strength determines influence
- **Learning**: Synaptic plasticity - connection strengths change with use; Hebbian learning: "neurons that fire together wire together"
- **All-or-Nothing**: Action potential is binary (fires or doesn't); signal strength encoded in firing frequency not amplitude
- **Network**: Brain has ~86 billion neurons, each connected to thousands of others; parallel distributed processing

## Important Concepts

- Threshold mechanism: neuron fires only when input sum exceeds threshold (inspiration for activation functions)
- Synaptic weights: stronger synapses have more influence (analogous to artificial neural network weights)
- Plasticity: ability to strengthen/weaken connections based on experience (basis for learning in ANNs)
- Parallel processing: many neurons compute simultaneously (inspiration for parallel architectures)

## Notes

- Understand structure: dendrites (input), soma (processing), axon (output), synapses (connections)
- Threshold and all-or-nothing response inspire activation functions in ANNs
- Synaptic plasticity (Hebbian learning) inspires weight update rules
- Know biological inspiration for ANN components: weights (synaptic strength), activation (firing threshold), learning (plasticity)
