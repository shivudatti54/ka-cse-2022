**Descriptive Statistics in ML:** Fundamental data summarization techniques including: (1) Central Tendency - mean (arithmetic average, sensitive to outliers), median (middle value, robust), mode (most frequent, for categorical data); (2) Dispersion - variance (average squared deviation, sample uses n-1 for unbiasedness), standard deviation (variance square root), IQR (middle 50% spread); (3) Shape - skewness (asymmetry: positive=right tail, negative=left tail), kurtosis (tailedness: heavy vs light tails); (4) Relationships - covariance (joint variability), correlation (standardized [-1,1] measure); (5) Visualization - histograms (frequency distribution), box plots (five-number summary with outlier detection via 1.5×IQR rule), scatter plots (bivariate relationships). ML applications include variance thresholding for feature selection, correlation analysis for multicollinearity detection, outlier treatment, distribution transformation decisions, and residual analysis for model validation.
