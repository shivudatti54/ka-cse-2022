# Types of Machine Learning

## Overview

Machine learning types classify learning problems by the feedback signal received during training. The three core types are supervised, unsupervised, and reinforcement learning, each suited to different data characteristics and objectives.

## Key Points

- **Supervised Learning**: Uses labeled data (input-output pairs) for classification and regression tasks
- **Unsupervised Learning**: Uses unlabeled data to discover hidden patterns, structures, or groupings
- **Reinforcement Learning**: Agent learns through environment interaction and reward signals
- **Feedback Signal**: Labels (supervised), structure (unsupervised), or rewards (reinforcement)
- **Task Alignment**: Each type aligns with distinct tasks, metrics, and algorithms
- **Complexity Varies**: Training and inference complexity depends on algorithm choice and data size

## Important Concepts

- Supervised learning examples: spam detection, house price prediction, disease diagnosis, image classification, speech recognition
- Unsupervised learning examples: customer segmentation, anomaly detection, topic discovery, market basket analysis, data compression
- Reinforcement learning examples: game playing (Chess, Go), robotics control, resource scheduling, ad bidding
- Correctly identifying the learning type guides model selection and evaluation strategy

## Notes

- Know examples for each learning type - commonly asked in exams
- Understand the differences in feedback signals: labels vs structure vs rewards
- Be able to map real-world problems to appropriate learning types
