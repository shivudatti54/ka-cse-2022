# Learning Objectives

After studying this topic, you should be able to:

1. Define and calculate measures of central tendency (mean, median, mode) for univariate data
2. Compute measures of dispersion (range, variance, standard deviation, IQR) and interpret their meaning
3. Identify and describe different distribution shapes (skewness, kurtosis)
4. Select appropriate visualizations (histograms, box plots, bar charts) for different data types
5. Detect and handle missing values and outliers in univariate data
6. Explain the importance of univariate analysis in the machine learning pipeline
