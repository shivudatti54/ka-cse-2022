# Why Study Machine Learning Applications?

## Big Picture Foundation

Knowing real-world ML applications helps you connect theory to practice. Understanding which ML technique solves which problem is key to both exams and industry work.

## Exam Relevance

| Aspect         | Details                                                                |
| -------------- | ---------------------------------------------------------------------- |
| Frequency      | Common — asked as "list applications" or "match technique to use case" |
| Marks          | 5-10 marks (list applications with examples, match techniques)         |
| Question Types | Short answers, industry-wise listings, technique mapping               |

## Practical Value

- **Problem framing**: Know which ML approach fits a given real-world problem.
- **Industry awareness**: Understand how ML is used across healthcare, finance, retail, etc.
- **Technique selection**: Map problem types to appropriate algorithms.

## Learning Outcomes

After this topic, you should be able to:

1. List ML applications across at least 5 industries with specific examples
2. Map ML techniques (supervised, unsupervised, RL, deep learning) to applications
3. Explain why a particular technique is suited for a given application
4. Identify the data type (images, text, tabular, time-series) for each application
