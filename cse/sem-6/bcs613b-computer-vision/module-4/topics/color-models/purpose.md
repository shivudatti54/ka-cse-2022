# Learning Objectives

After studying this topic, you should be able to:

1. Explain the fundamental differences between additive and subtractive color models
2. Identify and describe the components of major color models (RGB, CMY/K, HSI, YCbCr)
3. Convert colors between RGB and other color models using appropriate formulas
4. Select the most appropriate color model for specific image processing tasks
5. Explain why certain color models are preferred for particular applications (display, printing, compression, etc.)
6. Apply color model transformations in practical image processing scenarios
