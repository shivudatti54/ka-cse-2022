# What is Computer Vision - A Brief History

## Overview

Computer Vision enables machines to interpret and understand visual information from the world, replicating human visual system capabilities. The field has evolved from simple rule-based approaches in the 1960s through statistical methods in the 1990s to the deep learning revolution beginning in 2012.

## Key Points

- **Definition**: Science and technology enabling machines to extract information from images necessary to solve tasks
- **Core Tasks**: Classification, detection, segmentation, tracking, and recognition at different complexity levels
- **Historical Milestones**: Block World (1970s), Marr's computational theory (1982), SIFT (1999), AlexNet (2012), ResNet (2015)
- **Deep Learning Era**: Started with AlexNet winning ImageNet 2012, revolutionized CV with CNNs replacing handcrafted features
- **Real-World Applications**: Autonomous vehicles, medical imaging, surveillance, AR/VR, industrial automation, retail
- **Key Challenges**: Illumination variation, occlusion, scale variation, viewpoint changes, intra-class variation, background clutter

## Important Concepts

- Computer Vision extracts semantic meaning while Image Processing performs image-to-image transformations
- CV pipeline: Image Acquisition → Preprocessing → Feature Extraction → Recognition → Decision
- Modern CV relies heavily on deep learning with CNNs, R-CNN, YOLO, and transformers
- Foundation models like CLIP and SAM represent current cutting-edge trends

## Notes

- Distinguish CV from Image Processing - CV understands images, IP transforms them
- Know major milestones: Marr's theory, Viola-Jones detector, SIFT features, deep learning revolution
- Be familiar with different application domains and their specific CV requirements
- Understand that the "inverse problem" (2D to 3D reconstruction) is inherently ill-posed
