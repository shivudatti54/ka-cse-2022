# Learning Objectives

After studying this topic, you should be able to:

1. Understand the purpose and differences between trim() and strip() methods for removing leading and trailing whitespace from strings, including Unicode whitespace handling.

2. Explain the usage of substring() method for extracting portions of strings using start and end indices, understanding index boundaries and the distinction between inclusive and exclusive indexing.

3. Apply the split() method to parse strings into arrays using specified delimiters and handle edge cases such as empty strings, limit parameters, and special regex characters.

4. Apply replace() and replaceAll() methods to substitute characters and substrings within strings, distinguishing between literal replacement and regex-based replacement.

5. Use the matches() method to validate strings against regular expression patterns for data validation in applications such as form validation and input checking.

6. Implement the format() method to create formatted output strings using format specifiers for various data types including strings, integers, and floating-point numbers.

7. Analyze which string manipulation method is appropriate for a given problem scenario based on the nature of the operation and performance considerations.

8. Design solutions that combine multiple string methods to solve complex text processing problems in real-world applications.
