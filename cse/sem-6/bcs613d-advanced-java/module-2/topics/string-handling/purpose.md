# Why Study String Handling?

## Core Java Skill

String handling is one of the **most fundamental skills** in Java programming. Strings are used everywhere - from user input processing to file handling, database queries, and network communication. Almost every Java application manipulates strings extensively.

## Exam Relevance

| Aspect             | Details                                                      |
| ------------------ | ------------------------------------------------------------ |
| **Frequency**      | Asked in nearly every exam                                   |
| **Marks**          | Typically 5-10 marks                                         |
| **Question Types** | String methods, immutability, String vs StringBuilder/Buffer |
| **Must Know**      | String pool, immutability, common methods, == vs equals()    |

## Real-World Applications

1. **Input Validation** - Checking and sanitizing user input
2. **Text Processing** - Parsing CSV files, log files, configuration
3. **Web Development** - URL manipulation, HTML generation, form handling
4. **Database Queries** - Building SQL strings safely
5. **File I/O** - Reading and writing text data

## Prerequisites

Before studying string handling, you should know:

- Java basics (variables, data types, methods)
- Object-oriented concepts (objects, classes)
- Basic understanding of memory (heap, stack)

## What You Will Learn

- String creation methods (literals vs new keyword)
- String immutability and the String pool
- Essential String methods (length, charAt, substring, etc.)
- String comparison (== vs equals())
- StringBuilder and StringBuffer for mutable strings
