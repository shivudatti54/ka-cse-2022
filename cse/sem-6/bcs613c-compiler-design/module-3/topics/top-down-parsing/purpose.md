# Learning Objectives

After studying this topic, you should be able to:

1. Explain the concept of top-down parsing and differentiate between recursive descent and LL(1) parsing techniques.

2. Identify and eliminate left recursion from context-free grammars to make them suitable for top-down parsing.

3. Apply left factoring to grammars with common prefixes to construct predictive parsers.

4. Compute FIRST and FOLLOW sets for all non-terminals in a given grammar accurately.

5. Construct LL(1) parsing tables for grammars that satisfy LL(1) conditions.

6. Demonstrate the predictive parsing process by tracing through given input strings using the parsing table.

7. Determine whether a grammar is LL(1) by checking for ambiguity, left recursion, and conflicts in the parsing table.
