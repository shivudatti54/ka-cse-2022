This chapter traced the evolution of programming languages from first-generation machine code through fifth-generation declarative languages. Machine language provided direct hardware execution with zero abstraction but maximum difficulty. Assembly language introduced symbolic representation while maintaining hardware correspondence. Third-generation languages (FORTRAN, COBOL, ALGOL) achieved hardware independence through compilation, utilizing context-free grammars for syntax specification. Structured programming languages (C, Pascal) formalized control structures and introduced modern type systems. Fourth and fifth-generation languages (SQL, Prolog) shifted toward declarative paradigms specifying outcomes rather than procedures. The chapter also covered object-oriented programming (encapsulation, inheritance, polymorphism) and functional programming (lambda calculus foundations, immutability, referential transparency), concluding with compilation versus interpretation trade-offs including modern JIT compilation approaches.
