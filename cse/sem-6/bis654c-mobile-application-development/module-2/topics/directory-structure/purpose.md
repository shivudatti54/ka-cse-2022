# Learning Objectives

After studying this topic, you should be able to:

1. Understand the complete directory structure of an Android project generated by Android Studio
2. Explain the purpose and contents of each key directory including `app/`, `res/`, `java/`, and `manifests/`
3. Identify the role of Gradle build files at both project and module levels
4. Locate and describe the components of the `AndroidManifest.xml` file within the manifests directory
5. Apply knowledge of the resource directory structure to organize layouts, drawables, and values files appropriately
6. Analyze the relationship between source code packages and the Java/Kotlin directory hierarchy
7. Design a well-organized Android project structure following standard conventions
8. Implement modifications to project files by navigating the correct directories in Android Studio
