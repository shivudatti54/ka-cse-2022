# Learning Objectives

After studying this topic, you should be able to:

1. Understand the fundamental concepts of RelativeLayout and its role in positioning child views relative to each other and the parent container.

2. Explain the various positioning attributes available in RelativeLayout such as android:layout_below, android:layout_toRightOf, android:layout_alignParentTop, and android:centerInParent.

3. Apply RelativeLayout to design responsive user interfaces that adapt to different screen sizes without hardcoded pixel values.

4. Understand the TableLayout structure and explain how it organizes child views in rows and columns using TableRow containers.

5. Analyze the use of TableLayout properties like android:stretchColumns, android:shrinkColumns, and android:collapseColumns for table management.

6. Design practical Android layouts using TableLayout for displaying structured data such as forms, schedules, and comparison tables.

7. Implement complete Android UI solutions by combining RelativeLayout and TableLayout to create professional-looking application screens.
