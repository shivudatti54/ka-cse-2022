# Learning Objectives

After studying this topic, you should be able to:

1. Understand the Android Studio development environment and identify its key components for mobile application development.
2. Explain the purpose and relationship between MainActivity and the corresponding layout XML file.
3. Apply the code-build-run-debug cycle to create, compile, and execute a basic Android application.
4. Create a new Android project with proper directory structure using Android Studio.
5. Run an Android application on an Android Virtual Device (AVD) emulator to verify functionality.
6. Analyze the AndroidManifest.xml file to understand app components and permissions.
7. Implement the onCreate() method and recognize its role in the Android activity lifecycle.
8. Design and modify a simple user interface using basic Android Views and layouts.
