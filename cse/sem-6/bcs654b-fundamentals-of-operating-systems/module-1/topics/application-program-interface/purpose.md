# Learning Objectives

After studying this topic, you should be able to:

1. Understand the concept of Application Programming Interface (API) and its role in operating systems.
2. Explain the relationship between APIs, system calls, and operating system services.
3. Differentiate between API and system calls, and describe how they interact.
4. Describe common types of APIs used in modern operating systems.
5. Explain how applications use APIs to request operating system services without directly accessing hardware.
6. Analyze the benefits of using APIs, including abstraction, portability, and modularity.
