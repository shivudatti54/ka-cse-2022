# Learning Objectives

After studying this topic, you should be able to:

1. **Define** a process and distinguish between a process and a program with examples

2. **Explain** the various process states and the state transitions in the process lifecycle

3. **Describe** the structure and components of a Process Control Block (PCB)

4. **Analyze** different process scheduling algorithms (FCFS, SJF, Round Robin, Priority) and evaluate their performance

5. **Illustrate** operations on processes including creation, termination, and inter-process communication

6. **Compare** single-threaded and multithreaded processes with respect to concurrency and resource utilization

7. **Differentiate** between the various multithreading models (many-to-one, one-to-one, many-to-many)

8. **Apply** thread libraries to create and manage threads in a programming context
