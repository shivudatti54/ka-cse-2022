A tree is a connected, acyclic graph with n vertices and n−1 edges, satisfying several equivalent characterizations. Key properties include the bridge property, leaf count theorem, and degree sum formula. A spanning tree is a subgraph containing all vertices of a connected graph while maintaining tree structure. Every connected graph has at least one spanning tree. Construction algorithms include DFS and BFS, while weighted graphs require minimum spanning tree algorithms (Kruskal's and Prim's). Cayley's Formula states that complete graph K_n has n^(n−2) distinct spanning trees.
