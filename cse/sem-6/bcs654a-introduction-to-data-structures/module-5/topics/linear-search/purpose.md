### Learning Purpose: Linear Search

**1. Why is this topic important?**
Linear search is the simplest searching algorithm that sequentially checks each element until a match is found. Despite its O(n) time complexity, it is important because it works on unsorted data and serves as a baseline for comparing more efficient search algorithms.

**2. Real-world applications:**
Linear search is used when searching small or unsorted datasets, finding elements in linked lists, searching through unindexed database records, and as a fallback when data is not sorted or indexed.

**3. Connection to other topics:**
This topic provides the baseline comparison for binary search (which requires sorted data but is much faster). It connects to array traversal from Module 1 and linked list traversal from Module 3, and contrasts with tree-based searching in Module 4.
