# Learning Objectives

After studying this topic, you should be able to:

1. Define AND-OR graphs and explain their role in problem decomposition
2. Distinguish between AND nodes and OR nodes in a search tree
3. Explain the concept of problem reduction and how complex problems are decomposed into simpler subproblems
4. Trace the execution of the AO\* (AND-OR graph search) algorithm
5. Compute cost estimates for AND nodes and OR nodes during AO\* execution
6. Identify when AND search is more appropriate than standard OR-based search strategies
7. Construct AND-OR search trees for given problem domains
8. Analyze the completeness and optimality properties of the AO\* algorithm
