# Block

Summary

- A block is a collection of transactions that are verified and validated by the network.
- The block header contains metadata about the block, such as the block number, parent hash, and timestamp.
- The transaction list contains a list of transactions that are included in the block.
- Blocks are created by miners, who are responsible for validating and verifying transactions on the network.

### Important Formulas, Definitions, and Theorems

- Block: a collection of transactions that are verified and validated by the network.
- Block header: the metadata about the block, including the block number, parent hash, and timestamp.
- Transaction list: the list of transactions that are included in the block.
- Block body: the actual data for each transaction in the block.

### Key Points

- Blocks are the fundamental component of the Ethereum blockchain.
- Each block is linked to the previous block through its parent hash.
- Miners are responsible for creating and validating blocks.
- The block chain is the sequence of blocks that make up the Ethereum blockchain.
- Blocks are broadcast to the network and validated by other nodes.
- The block header contains metadata about the block.
- The transaction list contains a list of transactions that are included in the block.
