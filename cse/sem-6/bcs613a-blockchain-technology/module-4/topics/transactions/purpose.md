# Learning Objectives

After studying this topic, you should be able to:

1. Describe the structure of an Ethereum transaction and explain each field (nonce, gas price, gas limit, to, value, data, v/r/s).
2. Differentiate between the three types of Ethereum transactions: simple ETH transfer, contract deployment, and contract interaction.
3. Explain the gas mechanism including gas costs, gas limit vs gas used, and fee calculation under both legacy and EIP-1559 models.
4. Describe the complete lifecycle of an Ethereum transaction from creation to finality.
5. Compare Ethereum's account-based model with Bitcoin's UTXO model for handling transactions.
6. Explain message calls (internal transactions) and how contracts interact with each other.
