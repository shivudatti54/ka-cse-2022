# Introduction to Ethereum

## Overview

Ethereum is a decentralized blockchain platform enabling smart contracts and decentralized applications (dApps), launched in 2015 by Vitalik Buterin. Unlike Bitcoin's primary focus on peer-to-peer currency, Ethereum serves as a programmable world computer executing arbitrary code through the Ethereum Virtual Machine.

## Key Points

- **Smart Contract Platform**: Executes Turing-complete programs called smart contracts on decentralized network
- **Ethereum Virtual Machine (EVM)**: Runtime environment executing bytecode deterministically across all nodes
- **Ether (ETH)**: Native cryptocurrency used for transaction fees (gas) and value transfer
- **Accounts**: Two types - Externally Owned Accounts (EOAs) controlled by private keys, Contract Accounts controlled by code
- **Gas Mechanism**: Computational pricing prevents infinite loops and incentivizes efficient code
- **Decentralized Applications**: Frontend interfaces combined with smart contract backend logic
- **Global State Machine**: Ethereum maintains world state transitioning through transactions and block creation

## Important Concepts

- Ethereum enables programmable money and decentralized applications beyond simple value transfer
- Smart contracts are immutable once deployed, requiring careful development and auditing
- Gas costs make computation expensive, encouraging optimization
- EVM ensures deterministic execution across all network nodes
- Ethereum transitioned from Proof-of-Work to Proof-of-Stake (The Merge, 2022)
