# Learning Objectives

After studying this topic, you should be able to:

1. Explain the architecture of Hyperledger Fabric, including peer nodes, orderer nodes, client nodes, and certificate authority.
2. Describe the features of Hyperledger Fabric, such as permissioned network, modular architecture, scalability, and security.
3. Identify the use cases of Hyperledger Fabric, such as supply chain management, financial services, and healthcare.
4. Understand how Hyperledger Fabric can be used to build secure and scalable blockchain applications.
5. Describe the example of a supply chain management system built on Hyperledger Fabric.
6. Explain the benefits of using Hyperledger Fabric, such as improved security, scalability, and transparency.
7. Analyze the advantages and disadvantages of using Hyperledger Fabric in different scenarios.
