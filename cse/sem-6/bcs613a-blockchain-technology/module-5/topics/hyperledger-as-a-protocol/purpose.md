# Learning Objectives

After studying this topic, you should be able to:

1. Explain the theoretical foundations of Hyperledger, including permissioned consensus and modular architecture.
2. Describe the major Hyperledger frameworks (Fabric, Sawtooth, Corda) and their distinguishing characteristics.
3. Analyze the four-layer protocol stack and the function of each layer.
4. Evaluate the consensus mechanisms supported by Hyperledger platforms.
5. Compare and contrast Hyperledger with public blockchain platforms such as Ethereum.
6. Identify appropriate use cases for different Hyperledger frameworks.
7. Design a basic enterprise blockchain network using Hyperledger Fabric.
