# Learning Objectives

After studying this topic, you should be able to:

1. Identify and describe the different types of blockchain transactions including value transfer, coinbase, multisig, and smart contract transactions.
2. Differentiate between Bitcoin transaction types: P2PKH, P2SH, P2WPKH (SegWit), and multisignature.
3. Explain the purpose and structure of coinbase transactions as the mechanism for creating new cryptocurrency.
4. Describe how timelocked transactions work using locktime and CLTV/CSV opcodes.
5. Compare transaction types across different blockchain platforms (Bitcoin vs Ethereum).
6. Analyze the security and use-case implications of different transaction types.
