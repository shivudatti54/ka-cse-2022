# Bitcoin Definition

## Overview

Bitcoin is the world's first decentralized digital currency and peer-to-peer electronic payment system, introduced by Satoshi Nakamoto in 2008. It solved the double-spending problem without central authority through blockchain technology and Proof-of-Work consensus, creating a trustless system for value transfer.

## Key Points

- **Distributed Protocol**: Peer-to-peer system defining rules for creating, transmitting, and verifying transactions without central control
- **Public Immutable Ledger**: Blockchain chain of blocks cryptographically linked, making alteration computationally infeasible
- **Proof-of-Work Consensus**: Miners compete to solve cryptographic puzzles for right to create blocks and earn rewards
- **Limited Supply**: Capped at 21 million BTC with disinflationary halving every 210,000 blocks (~4 years)
- **Cryptographic Ownership**: Public-private key pairs replace traditional accounts; private key authorization required for spending
- **Block Reward System**: Incentivizes miners with newly minted bitcoins plus transaction fees
- **Digital Scarcity**: Predictable issuance mimics scarce resource extraction, earning "digital gold" nickname

## Important Concepts

- Nakamoto Consensus combines PoW mining with longest chain rule
- Block structure: header (previous hash, timestamp, nonce, Merkle root) plus transaction list
- Transaction flow: create → sign → broadcast → validate → mine → confirm
- Pseudonymity through address representation rather than real identities
- Transparency with all transactions publicly auditable on blockchain

## Notes

- Understand how PoW solves Byzantine Generals Problem practically
- Know block reward halving schedule and its economic implications
- Be able to trace complete transaction lifecycle from initiation to confirmation
- Remember that Bitcoin's breakthrough is trustless consensus, not just digital money
