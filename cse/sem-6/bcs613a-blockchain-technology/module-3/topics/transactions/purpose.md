# Learning Objectives

After studying this topic, you should be able to:

1. Define what a Bitcoin transaction is and describe its fundamental properties (irreversibility, transparency, pseudonymity).
2. Explain the UTXO (Unspent Transaction Output) model and how Bitcoin tracks ownership of funds.
3. Describe the components of a Bitcoin transaction including inputs, outputs, and scripts.
4. Explain how transaction signing works using private keys and how verification uses public keys.
5. Differentiate between different types of Bitcoin transactions (P2PKH, P2SH, multisig).
6. Analyze the transaction lifecycle from creation to confirmation in a block.
