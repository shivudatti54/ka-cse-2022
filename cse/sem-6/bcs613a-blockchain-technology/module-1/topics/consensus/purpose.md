# Why Study Consensus?

## Foundation of Blockchain Technology

Consensus is the **core innovation** that makes blockchain possible. Without consensus, there would be no way for decentralized nodes to agree on the state of the ledger. Understanding consensus is essential before diving into specific mechanisms like PoW or PoS.

## Real-World Applications

1. **Cryptocurrency Networks** - Bitcoin, Ethereum use consensus to validate transactions
2. **Supply Chain Tracking** - Multiple parties agree on shipment status
3. **Healthcare Records** - Hospitals reach consensus on patient data
4. **Voting Systems** - Electronic voting with verifiable agreement
5. **IoT Networks** - Devices agreeing on sensor readings

## Connection to Other Topics

After mastering consensus basics, you'll be ready for:

- **Byzantine Fault Tolerance** - Handling malicious nodes
- **Proof of Work** - Mining-based consensus
- **Proof of Stake** - Stake-based consensus
- **PBFT** - Classical BFT for permissioned systems
- **Smart Contracts** - Code that relies on consensus

## Learning Outcomes

After studying this topic, you should be able to:

1. Define consensus in distributed systems context
2. Explain why consensus is necessary for blockchains
3. List the four key properties of consensus
4. Differentiate between crash faults and Byzantine faults
5. Compare centralized vs decentralized consensus
