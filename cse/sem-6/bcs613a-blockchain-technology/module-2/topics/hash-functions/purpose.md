# Learning Objectives

After studying this topic, you should be able to:

1. Define and explain the properties of hash functions.
2. Distinguish between cryptographic and non-cryptographic hash functions.
3. Describe the applications of hash functions in blockchain technology.
4. Identify and explain the use of specific hash functions in blockchain networks.
5. Understand the importance of collision-resistance and non-invertibility in hash functions.
6. Explain how hash functions enable data integrity, block linking, and transaction verification in blockchain.
7. Analyze the role of hash functions in ensuring the security and integrity of blockchain networks.
