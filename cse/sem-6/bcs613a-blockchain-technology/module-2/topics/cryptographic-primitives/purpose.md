# Learning Objectives

After studying this topic, you should be able to:

1.  Differentiate between symmetric and asymmetric cryptography, listing the advantages and disadvantages of each.
2.  Explain the five core properties of a cryptographic hash function and their importance in blockchain operations.
3.  Describe the step-by-step process of how a digital signature is created and verified using asymmetric cryptography.
4.  Identify the reasons for using the Elliptic Curve Digital Signature Algorithm (ECDSA) in major blockchains like Bitcoin and Ethereum.
5.  Illustrate the structure of a Merkle Tree and explain its role in ensuring data integrity and enabling efficient verification (SPV).
6.  Synthesize how all these cryptographic primitives work together to secure a blockchain transaction from creation to block confirmation.
