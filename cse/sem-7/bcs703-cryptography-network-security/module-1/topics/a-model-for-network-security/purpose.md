# Learning Objectives

After studying this topic, you should be able to:

1. Identify and explain the four fundamental components of a network security model (sender, receiver, transmission medium, adversary)

2. Describe the five primary security services defined by the OSI security architecture

3. Differentiate between passive and active security attacks with examples of each

4. Understand the mathematical representation of encryption and decryption processes

5. Explain how confidentiality is achieved through encryption in the communication model

6. Analyze real-world scenarios to identify applicable security services and potential attacks

7. Evaluate the role of security mechanisms in protecting network communications

8. Apply the network security model to understand how classical encryption techniques fit into broader security frameworks
