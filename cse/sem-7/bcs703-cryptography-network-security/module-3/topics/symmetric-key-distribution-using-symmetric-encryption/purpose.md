# Learning Objectives

After studying this topic, you should be able to:

1. Explain the fundamental key distribution problem in symmetric cryptography and why it necessitates special solutions.

2. Describe the concept of key hierarchy and distinguish between master keys and session keys, including their respective roles and lifetimes.

3. Explain the architecture and operation of a Key Distribution Center (KDC) in facilitating secure key exchange.

4. Analyze the Needham-Schroeder protocol step-by-step, understanding the purpose of each message component including nonces and encrypted payloads.

5. Identify the security vulnerability in the original Needham-Schroeder protocol (the Denning-Sacco attack) and explain how it can be exploited.

6. Describe the Kerberos authentication system, its components, and how it addresses limitations in earlier protocols.

7. Evaluate the trust assumptions underlying KDC-based key distribution systems and discuss the implications of KDC compromise.

8. Apply the concept of key reduction to calculate the number of keys required in a KDC-based system versus a point-to-point system for a given number of users.
