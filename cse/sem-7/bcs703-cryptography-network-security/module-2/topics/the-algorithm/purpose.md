# Learning Objectives

After studying this topic, you should be able to:

1. **Define** the Linear Congruential Generator algorithm and its mathematical formulation Xₙ₊₁ = (a × Xₙ + c) mod m

2. **Explain** the role and significance of each parameter (a, c, m, seed) in the LCG algorithm

3. **Apply** the LCG recurrence relation to generate sequences of pseudorandom numbers given specific parameters

4. **Analyze** the conditions required for an LCG to achieve full period according to the Hull-Dobell theorem

5. **Evaluate** the suitability of LCG for cryptographic applications and understand why it fails security requirements

6. **Compare** the LCG approach with other pseudorandom generators like Blum Blum Shub

7. **Identify** historical examples of LCG parameter choices and their consequences (e.g., RANDU)
