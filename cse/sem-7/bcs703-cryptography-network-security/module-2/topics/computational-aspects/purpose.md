# Learning Objectives

After studying this topic, you should be able to:

1. Analyze the time and space complexity of various PRNG algorithms including LCGs, LFSRs, and BBS generators.

2. Explain the relationship between state size and maximum achievable period in deterministic PRNGs.

3. Define computational entropy and explain its role in characterizing PRNG unpredictability.

4. Apply the next-bit test and Yao's Theorem to evaluate PRNG security properties.

5. Compare hardware and software PRNG implementations with respect to efficiency and security considerations.

6. Describe the security model for cryptographic PRNGs against polynomial-time adversaries.

7. Explain entropy extraction techniques and the leftover hash lemma for seed generation.

8. Identify optimization techniques for practical PRNG implementation including counter mode and hash-based approaches.
