This module provides comprehensive knowledge of Raspberry Pi as an IoT platform. Key topics include: (1) System-on-Chip architecture with comparative analysis of Pi models for IoT applications; (2) GPIO programming using Python with practical LED, button, and servo control examples; (3) I²C, SPI, and UART communication protocols with timing analysis and code implementations; (4) Linux operating system configuration, network security hardening, and Docker containerization; (5) MQTT protocol implementation for IoT messaging and edge computing deployment. Assessment includes application-level multiple choice questions covering power calculations, protocol selection, security considerations, and system design scenarios. The module equips students to develop production-ready IoT solutions using Raspberry Pi.
