Apache Hadoop is an open-source distributed computing framework comprising HDFS (storage), YARN (resource management), and MapReduce (processing). HDFS uses a NameNode-DataNode architecture with block replication (default 3x) and rack-aware placement for fault tolerance. YARN separates resource management from processing through ResourceManager, NodeManager, and ApplicationMaster components. MapReduce divides processing into map, shuffle-sort, and reduce phases for parallel execution. For IoT applications, Hadoop handles batch analytics on sensor data but requires integration with real-time frameworks for latency-sensitive operations. The framework scales horizontally on commodity hardware, offering cost-effective big data processing at petabyte scale.
