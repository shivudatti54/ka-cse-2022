# Learning Objectives

After studying this topic, you should be able to:
1. Install Python 3.10+ on Windows, Linux, and macOS operating systems using multiple methods
2. Configure Python PATH environment variables correctly on Windows for command-line accessibility
3. Set up Python on Raspberry Pi for IoT hardware projects, including interface enablement
4. Create and manage virtual environments using venv and Conda for IoT project isolation
5. Install essential IoT communication libraries including paho-mqtt, requests, and websockets
6. Install hardware interaction libraries (RPi.GPIO, gpiozero, smbus2, spidev) for sensor and actuator control
7. Configure integrated development environments (VS Code, Thonny, PlatformIO) for IoT development
8. Verify Python installation and IoT library installation through command-line testing
9. Troubleshoot common installation issues including PATH configuration, permission errors, and SSL certificate problems
10. Enable and verify I2C and SPI interfaces on Raspberry Pi for embedded hardware communication
