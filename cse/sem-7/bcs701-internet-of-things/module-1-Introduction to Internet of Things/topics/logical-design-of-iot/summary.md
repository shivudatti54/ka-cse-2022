The logical design of IoT systems represents an implementation-agnostic abstraction that defines functional components, information flow, and communication patterns. This module covers the four-layer architecture model (perception, network, processing, application), core functional blocks (device/sensing, communication, storage/processing, application/interface), and major communication protocols (MQTT, CoAP, AMQP). Key topics include protocol selection trade-offs, data serialization formats, edge/fog/cloud processing tiers, and security architecture considerations. The theoretical foundation enables students to perform system analysis, make informed technology selections, and design scalable IoT solutions.
