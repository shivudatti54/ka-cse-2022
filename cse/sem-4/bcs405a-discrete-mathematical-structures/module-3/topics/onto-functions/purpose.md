# Learning Objectives

After studying this topic, you should students able to:

1. Define and explain the concept of onto (surjective) functions with precise mathematical notation.

2. Identify whether a given function is onto by examining the relationship between its domain, codomain, and range.

3. Distinguish between onto, one-to-one, and bijective functions, and determine which category a specific function belongs to.

4. Construct onto functions between finite sets with specified properties and count the total number of such functions.

5. Apply the composition theorem for onto functions and analyze the onto property in composed functions.

6. Prove or disprove that a function is onto using formal mathematical reasoning and counterexamples.

7. Apply the pigeonhole principle to determine the possibility of onto functions based on set sizes.

8. Solve problems involving onto functions in the context of discrete mathematics applications.
