# Learning Objectives

After studying this topic, you should students able to:

1. Define a function, specifying its domain, codomain, and range, and distinguish between these concepts.

2. Identify whether a given relation is a function by verifying that each element in the domain has exactly one image.

3. Determine whether a function is one-to-one (injective) using algebraic methods, the horizontal line test, or by providing counterexamples.

4. Prove rigorously that a function is one-to-one by assuming f(a₁) = f(a₂) and deriving a₁ = a₂.

5. Explain the relationship between one-to-one functions and inverse functions, recognizing that inverses exist only for bijective functions.

6. Apply the theorem that composition of one-to-one functions results in a one-to-one function.

7. Compare and contrast one-to-one, onto, and bijective functions with appropriate examples.

8. Solve application problems involving functions in discrete mathematics contexts.
