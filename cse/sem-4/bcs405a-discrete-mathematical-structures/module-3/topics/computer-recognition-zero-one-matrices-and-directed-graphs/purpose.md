# Learning Objectives

After studying this topic, you should students able to:

1. Define and distinguish between directed graphs (digraphs) and undirected graphs, identifying key terminology such as outdegree, indegree, path, cycle, and strong connectivity.

2. Construct the adjacency matrix of a given digraph and interpret matrix entries as edge indicators between vertices.

3. Explain the concept of reachability matrix and describe its relationship to paths of arbitrary length in a digraph.

4. Apply Warshall's algorithm to compute the transitive closure of a digraph, demonstrating each iteration step-by-step.

5. Analyze graph properties including connectivity and strong connectivity using matrix representations and Boolean matrix operations.

6. Perform Boolean matrix operations (AND, OR) and explain how they differ from regular arithmetic operations in the context of graph theory.

7. Utilize zero-one matrix representations to count paths of specific lengths between vertices in a digraph.

8. Apply computer recognition techniques to identify valid graph properties from adjacency matrices for practical applications.
