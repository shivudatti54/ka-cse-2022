# Learning Objectives

After studying this topic, you should students able to:

1. Define and explain the concept of a board in the context of rook polynomials, distinguishing between complete and partial boards.

2. Calculate rook numbers r_k(B) for a given board configuration by counting the number of ways to place k non-attacking rooks.

3. Construct the rook polynomial R(B, x) for any given board using the definition R(B, x) = Σ r_k(B) x^k.

4. Apply the board splitting theorem to compute rook polynomials for complex boards by decomposing them into simpler disjoint sub-boards.

5. Solve restricted permutation problems using the connection between rook polynomials and the principle of inclusion-exclusion.

6. Apply the formula for counting valid permutations when forbidden positions are given: Σ\_{k=0}^{n} (-1)^k r_k(B) (n-k)!.

7. Distinguish between complete boards, partial boards, and various board shapes, and determine appropriate methods for each type.

8. Interpret the coefficients of rook polynomials and relate them to specific combinatorial counting problems in real-world scenarios.
