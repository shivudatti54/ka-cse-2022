# Learning Objectives

After studying this topic, you should students able to:

1. Define and distinguish between homogeneous and non-homogeneous first-order linear recurrence relations.

2. Identify the components (c, d, initial condition) of a first-order linear recurrence relation from a given equation.

3. Solve homogeneous first-order linear recurrence relations using the characteristic root method or iteration.

4. Solve non-homogeneous first-order linear recurrence relations using the general solution formula.

5. Handle the special case when the coefficient c equals 1 (arithmetic progression case).

6. Apply recurrence relations to solve real-world problems involving growth, decay, and financial calculations.

7. Verify the correctness of a solved recurrence relation by back-substitution.

8. Apply the iteration method to derive closed-form solutions from first principles.
