# Learning Objectives

After studying this topic, you should students able to:

1. State and apply the Principle of Inclusion-Exclusion for counting unions of n sets
2. Compute the number of elements in unions and intersections of multiple sets using the generalized formula
3. Apply the inclusion-exclusion principle to solve problems involving divisibility and remainders
4. Calculate derangements (subfactorials) using the inclusion-exclusion formula and recurrence relation
5. Determine the number of onto (surjective) functions between finite sets using the sieve method
6. Understand the connection between rook polynomials and inclusion-exclusion for restricted permutation problems
7. Choose between direct counting and inclusion-exclusion based on problem structure
8. Verify results for inclusion-exclusion problems using complementary counting and sanity checks
