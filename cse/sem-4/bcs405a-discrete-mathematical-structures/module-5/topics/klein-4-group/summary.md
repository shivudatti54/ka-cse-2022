# Klein 4-Group (V4) - Summary

## Key Definitions and Concepts

- **Klein 4-Group (V4 or K4)**: A finite abelian group of order 4 with four elements where every non-identity element has order 2.

- **Order of an element**: The smallest positive integer n such that aⁿ = e (identity). In V4, all non-identity elements have order 2.

- **Cyclic group**: A group generated by a single element. V4 is NOT cyclic because no single element generates all four elements.

- **Isomorphism**: Two groups are isomorphic if there exists a bijective mapping that preserves the group operation. V4 ≅ Z2 × Z2.

- **Internal direct product**: V4 can be expressed as the product of any two distinct subgroups of order 2.

## Important Formulas and Theorems

- **Cayley Table for V4**: The product of any two distinct non-identity elements equals the third non-identity element.

- **Order properties**: If G is a group of order 4 with all non-identity elements having order 2, then G ≅ V4.

- **Subgroup structure**: V4 has exactly 5 subgroups: one trivial, three of order 2, and the group itself.

- **Isomorphism theorem**: V4 is isomorphic to Z2 × Z2 (the direct product of two cyclic groups of order 2).

## Key Points

1. V4 contains exactly four elements: identity (e) and three elements of order 2.

2. V4 is abelian (commutative) because the Cayley table is symmetric along the main diagonal.

3. V4 is the smallest non-cyclic group, distinguishing it from cyclic groups of order 4.

4. Every subgroup of V4 is normal since V4 is abelian.

5. The symmetry group of a rectangle (not a square) is isomorphic to V4.

6. V4 can be generated by any two distinct non-identity elements: V4 = ⟨a, b⟩ where a ≠ b and a, b ∈ {non-identity elements}.

7. V4 has three subgroups of order 2, each cyclic, generated by one non-identity element.

8. In V4, the inverse of every element is the element itself (a = a⁻¹ for all a ∈ V4).

## Common Mistakes to Avoid

1. **Confusing with Z4**: Students often confuse V4 with the cyclic group of order 4 (Z4). Remember: Z4 has an element of order 4, while V4 has no such element.

2. **Thinking V4 is cyclic**: V4 is NOT cyclic. This is its most distinguishing feature from other groups of order 4.

3. **Forgetting commutativity**: Always remember that V4 is abelian. Some students incorrectly treat it like non-abelian groups.

4. **Incorrect subgroup counting**: V4 has exactly 5 subgroups, not more. Each non-identity element generates exactly one subgroup of order 2.

## Revision Tips

1. Memorize the Cayley table pattern: distinct non-identity elements multiply to give the third non-identity element.

2. Practice identifying whether a given group of order 4 is isomorphic to V4 or Z4 by checking element orders.

3. Remember the key property: "All elements (except identity) have order 2" = likely V4.

4. Review the rectangle symmetry example to understand practical applications of V4.

5. Practice constructing proofs showing V4 ≅ Z2 × Z2 as this is a common exam topic.
