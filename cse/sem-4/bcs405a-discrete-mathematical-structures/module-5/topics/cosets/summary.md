# Cosets - Summary

## Key Definitions and Concepts

- **Left Coset**: For group G, subgroup H, and element a ∈ G, the left coset aH = {ah : h ∈ H}
- **Right Coset**: Ha = {ha : h ∈ H}
- **Coset Representative**: The element 'a' used to generate coset aH
- **Index**: The number of distinct cosets of H in G, denoted [G:H]
- **Normal Subgroup**: A subgroup N where gN = Ng for all g ∈ G

## Important Formulas and Theorems

- **Coset Size**: |aH| = |H| (for finite H)
- **Lagrange's Theorem**: |H| divides |G|, and [G:H] = |G|/|H|
- **Coset Equality**: aH = bH if and only if a⁻¹b ∈ H
- **Coset Partition**: Distinct cosets are disjoint and G = ∪aH

## Key Points

- Cosets partition a group into disjoint, equally-sized subsets
- The number of cosets equals the index, which is |G|/|H|
- In abelian groups, left and right cosets coincide
- Every element belongs to exactly one coset
- The identity element's coset is the subgroup itself: eH = H
- Coset representatives can be chosen in infinitely many ways
- The quotient group G/H exists only when H is normal

## Common Mistakes to Avoid

- Confusing subgroup with coset: H is a subgroup, aH is a coset
- Forgetting that coset operation depends on group operation (multiplicative vs additive notation)
- Assuming left and right cosets are equal in non-abelian groups without verification
- Using wrong coset representatives when listing cosets

## Revision Tips

- Practice finding cosets in Z₁₂ and Z₁₈ as these appear frequently in exams
- Remember: coset = subgroup + representative element
- Focus on understanding why cosets partition the group (this is key for proofs)
- Memorize the relationship: Order of group = Order of subgroup × Index
- Review examples from your university textbook covering different group types
