## Introduction

Cosets are fundamental objects in abstract algebra that arise when we study the structure of groups relative to their subgroups. In the context of the university's Discrete Mathematical Structures syllabus, understanding cosets is essential because they provide a powerful mechanism for decomposing groups into smaller, more manageable pieces. The concept of cosets was developed to understand how elements of a group behave when multiplied by elements from a specific subgroup.

In computer science applications, cosets play a crucial role in fields like cryptography, coding theory, and algorithm design. For instance, in the implementation of hash functions and error-correcting codes, the theory of cosets helps organize data elements into equivalence classes. Understanding cosets also prepares students for more advanced topics like quotient groups and homomorphism theorems, which are essential for formal language theory and automata.

This topic builds directly on the previous study of groups, subgroups, and group properties. By the end of this module, students should be able to identify cosets in various groups, prove key properties, and apply Lagrange's Theorem to solve practical problems.

## Key Concepts

### Definition of Cosets

Let G be a group and H be a subgroup of G. For any element a ∈ G, the **left coset** of H in G containing a is defined as:

**aH = {ah : h ∈ H}**

Similarly, the **right coset** of H in G containing a is defined as:

**Ha = {ha : h ∈ H}**

The element 'a' is called the **coset representative** or **coset leader**. It is important to note that while we typically write aH for left cosets, the same notation can represent right cosets depending on whether the group operation is written multiplicatively or additively.

For additive groups (like Zn under addition), we write cosets as a + H rather than aH.

### Examples of Cosets in Different Groups

**Example 1: Integers under Addition**

Consider the group (Z, +) and its subgroup 3Z = {..., -6, -3, 0, 3, 6, ...} (multiples of 3).

The cosets are:

- 0 + 3Z = {..., -6, -3, 0, 3, 6, ...} (multiples of 3)
- 1 + 3Z = {..., -5, -2, 1, 4, 7, ...} (integers leaving remainder 1 when divided by 3)
- 2 + 3Z = {..., -4, -1, 2, 5, 8, ...} (integers leaving remainder 2 when divided by 3)

These three cosets partition the entire set of integers.

**Example 2: In Zn**

In the group (Zn, +), consider the subgroup generated by d where d divides n. For n = 12 and d = 4, the subgroup is ⟨4⟩ = {0, 4, 8}.

The cosets are:

- 0 + ⟨4⟩ = {0, 4, 8}
- 1 + ⟨4⟩ = {1, 5, 9}
- 2 + ⟨4⟩ = {2, 6, 10}
- 3 + ⟨4⟩ = {3, 7, 11}

We can see that there are exactly d = 4 distinct cosets, each with n/d = 3 elements.

### Properties of Cosets

**Property 1: Coset Equality**

For a ∈ G and subgroup H:

- a ∈ aH (since a = ae where e is the identity)
- aH = bH if and only if a⁻¹b ∈ H (or equivalently, b ∈ aH)

**Property 2: Coset Size**

If H is a finite subgroup of G, then |aH| = |H| for all a ∈ G. This follows because the mapping h → ah is a bijection from H to aH.

**Property 3: Partition of Group**

The left cosets of H in G form a partition of G. Specifically:

- G = ∪{aH : a ∈ G}
- If aH ≠ bH, then aH ∩ bH = ∅

This is a fundamental result showing that cosets decompose the group into disjoint, equally-sized pieces.

### Lagrange's Theorem (Related to Cosets)

Lagrange's Theorem states: If G is a finite group and H is a subgroup of G, then |H| divides |G|, and the number of distinct left cosets of H in G is |G|/|H|.

The proof relies directly on the partition property: since G is partitioned into cosets, each of size |H|, the total number of cosets (called the **index** of H in G, denoted [G:H]) must be |G|/|H|.

The **index** [G:H] = number of cosets of H in G = |G|/|H|.

### Normal Subgroups

A subgroup N of G is called **normal** if gN = Ng for all g ∈ G (left and right cosets coincide). When N is normal, we can form the **quotient group** G/N whose elements are the cosets of N in G.

For abelian groups, every subgroup is normal, so left and right cosets are always identical.

### Coset Decomposition

The process of writing G as a disjoint union of cosets is called **coset decomposition** or **coset partition**. For a subgroup H of G:

G = a₁H ∪ a₂H ∪ ... ∪ aₖH

where a₁, a₂, ..., aₖ are distinct coset representatives and k = [G:H].

## Examples

### Worked Example 1: Cosets in S₃

Consider the symmetric group S₃ (permutations of 3 elements). |S₃| = 6. Let H = {e, (12)} (the subgroup containing identity and the transposition (12)).

The left cosets are:

- eH = {e, (12)}
- (13)H = {(13), (132)} — compute: (13)(12) = (132), and (13)e = (13)
- (23)H = {(23), (123)} — compute: (23)(12) = (123), and (23)e = (23)

So we have exactly 3 cosets, each of size 2, and 3 = 6/2 = |S₃|/|H|, verifying Lagrange's Theorem.

### Worked Example 2: Using Cosets to Solve a Problem

**Problem:** In Z₁₂, find all cosets of the subgroup H = {0, 3, 6, 9}. Determine the index [Z₁₂ : H].

**Solution:**
Z₁₂ under addition modulo 12, H = {0, 3, 6, 9} = 3Z₁₂

Since |Z₁₂| = 12 and |H| = 4, by Lagrange's Theorem, we must have 12/4 = 3 cosets.

Finding cosets:

- 0 + H = {0, 3, 6, 9}
- 1 + H = {1, 4, 7, 10}
- 2 + H = {2, 5, 8, 11}

No other distinct cosets exist since:
3 + H = 3 + {0, 3, 6, 9} = {3, 6, 9, 0} = 0 + H
4 + H = 4 + {0, 3, 6, 9} = {4, 7, 10, 1} = 1 + H
and so on.

Index [Z₁₂ : H] = 3

### Worked Example 3: Verifying Coset Properties

**Problem:** Let G = Z₁₀ and H = {0, 5}. Show that H is a subgroup and find all cosets of H in G.

**Solution:**
H = {0, 5} under addition mod 10.

Check closure: 0+0=0, 0+5=5, 5+0=5, 5+5=10≡0 ∈ H. Closed.
Identity: 0 ∈ H
Inverse: 0⁻¹ = 0, 5⁻¹ = 5 (since 5+5=10≡0)
Therefore H is a subgroup. |H| = 2, |G| = 10.

By Lagrange's Theorem: number of cosets = 10/2 = 5

Cosets:

- 0 + H = {0, 5}
- 1 + H = {1, 6}
- 2 + H = {2, 7}
- 3 + H = {3, 8}
- 4 + H = {4, 9}

These 5 disjoint cosets cover all 10 elements of Z₁₀.

## Exam Tips

1. **Remember the definition clearly**: For any a ∈ G and subgroup H, aH = {ah : h ∈ H}. Practice writing cosets for both multiplicative and additive notation.

2. **Coset size equals subgroup size**: This is crucial for applying Lagrange's Theorem. Always verify |aH| = |H| when H is finite.

3. **Cosets partition the group**: Remember that distinct cosets are always disjoint, and their union equals the entire group. This is frequently used in proofs.

4. **Index equals number of cosets**: The index [G:H] = |G|/|H| gives the number of distinct cosets. This is directly tested in university exams.

5. **Practice with Zn**: Most exam problems involve cosets in Zn under addition. Be comfortable finding cosets of subgroups generated by divisors.

6. **Coset equality test**: aH = bH if and only if a⁻¹b ∈ H. This criterion is essential for proving coset equality.

7. **For abelian groups, left = right**: If G is abelian, then aH = Ha for all a. Don't assume this for non-abelian groups.

8. **Identify coset representatives**: When listing cosets, choose the smallest non-negative representatives (0, 1, 2, ...) to avoid repetition.
