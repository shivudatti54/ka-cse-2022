## Introduction

Group theory is one of the most fundamental branches of abstract algebra, serving as the foundation for understanding algebraic structures in mathematics and their applications in computer science, physics, and engineering. A group is a set equipped with a binary operation that satisfies four key properties: closure, associativity, identity, and invertibility. This module explores various particular groups that frequently appear in both theoretical and applied contexts.

Understanding particular groups is essential for CSE students as these structures appear in cryptography, coding theory, algorithm analysis, and data structures. The study of specific groups like cyclic groups, symmetric groups, and dihedral groups provides concrete illustrations of abstract algebraic concepts and equips students with the tools needed to analyze computational problems.

## Key Concepts

### Definition of a Group

A **group** (G, _) is a set G together with a binary operation _ : G × G → G satisfying four axioms:

1. **Closure**: For all a, b ∈ G, a \* b ∈ G
2. **Associativity**: For all a, b, c ∈ G, (a _ b) _ c = a _ (b _ c)
3. **Identity**: There exists an element e ∈ G such that e _ a = a _ e = a for all a ∈ G
4. **Invertibility**: For each a ∈ G, there exists an element a⁻¹ ∈ G such that a _ a⁻¹ = a⁻¹ _ a = e

If the operation is commutative (a _ b = b _ for all a, b ∈ G), the group is called an **Abelian group**. Otherwise, it is a **non-Abelian group** (or non-abelian group).

### Classification by Size

- **Finite Group**: A group with a finite number of elements. The **order** of a finite group G, denoted |G|, is the number of elements in G.
- **Infinite Group**: A group with infinitely many elements.

### Cyclic Groups

A **cyclic group** is a group that can be generated by a single element. If there exists an element g ∈ G such that every element of G can be expressed as gⁿ for some integer n, then G is cyclic and g is called a **generator**.

**Properties of Cyclic Groups**:

- Every cyclic group is abelian
- The order of a finite cyclic group equals the order of its generator
- A cyclic group of order n is isomorphic to (Zₙ, +ₙ)

**Examples**:

1. **Integers under addition** (Z, +): This is an infinite cyclic group generated by 1 (or -1). Every integer can be written as n·1 for some integer n.
2. **Integers modulo n** (Zₙ, +ₙ): The set {0, 1, 2, ..., n-1} with addition modulo n forms a cyclic group of order n. The element 1 is always a generator.

### Symmetric Groups (Sₙ)

The **symmetric group** Sₙ on n symbols is the group of all permutations of the set {1, 2, ..., n} under the composition of permutations. The order of Sₙ is n! (n factorial).

**Properties**:

- Sₙ is non-abelian for n ≥ 3
- S₁ and S₂ are abelian; S₃ is the smallest non-abelian group
- Every finite group can be embedded in some symmetric group (Cayley's Theorem)

**Example**: S₃ has 6 elements representing all permutations of {1, 2, 3}:

- e (identity): (1)(2)(3)
- (1 2), (1 3), (2 3): Transpositions
- (1 2 3), (1 3 2): 3-cycles

### Alternating Groups (Aₙ)

The **alternating group** Aₙ is the subgroup of Sₙ consisting of all even permutations. An even permutation can be expressed as an even number of transpositions.

**Properties**:

- |Aₙ| = n!/2 for n ≥ 2
- Aₙ is non-abelian for n ≥ 4
- Aₙ is simple (has no non-trivial normal subgroups) for n ≥ 5

**Example**: A₃ has 3 elements: the identity e and the two 3-cycles (1 2 3) and (1 3 2). A₃ ≅ Z₃.

### Dihedral Groups (Dₙ)

The **dihedral group** Dₙ (sometimes denoted as D₂ₙ) represents the symmetries of a regular n-gon, including n rotations and n reflections. |Dₙ| = 2n.

**Elements**:

- Rotations: r₀, r₁, r₂, ..., rₙ₋₁ where rₖ rotates by 2πk/n radians
- Reflections: s₀, s₁, s₂, ..., sₙ₋₁ (reflections about various axes)

**Properties**:

- Dₙ is non-abelian for n ≥ 3
- Dₙ can be generated by two elements r and s satisfying rⁿ = e, s² = e, and srs = r⁻¹

**Example**: D₃ (symmetries of an equilateral triangle) has 6 elements and is isomorphic to S₃.

### Klein Four-Group (V₄)

The **Klein four-group** V₄ = {e, a, b, c} is defined with operation where each element is its own inverse: a² = b² = c² = e, and all products of distinct non-identity elements commute: ab = ba = c, bc = cb = a, ac = ca = b.

**Properties**:

- V₄ is abelian
- V₄ is the smallest non-cyclic group of order 4
- V₄ is isomorphic to Z₂ × Z₂ (direct product of two cyclic groups of order 2)

### Quaternion Group (Q₈)

The **Quaternion group** Q₈ = {1, -1, i, -i, j, -j, k, -k} with multiplication defined by:

- i² = j² = k² = ijk = -1
- ij = k, jk = i, ki = j
- ji = -k, kj = -i, ik = -j

**Properties**:

- |Q₈| = 8
- Q₈ is non-abelian
- All subgroups are cyclic: ⟨1⟩, ⟨-1⟩, ⟨i⟩, ⟨j⟩, ⟨k⟩, Q₈
- Q₈ has exactly one element of order 2 (-1) and six elements of order 4

### Matrix Groups

**General Linear Group** GL(n, R): The group of all invertible n×n matrices with real entries under matrix multiplication. |GL(n, R)| is infinite for n ≥ 1.

**Special Linear Group** SL(n, R): The subgroup of GL(n, R) consisting of matrices with determinant 1.

**Example**: GL(2, R) includes all invertible 2×2 matrices. SL(2, R) includes matrices like [[1, 0], [0, 1]] and [[0, -1], [1, 0]].

### Group of Units Modulo n (U(n))

The **multiplicative group of units** modulo n is U(n) = {a ∈ {1, 2, ..., n-1} : gcd(a, n) = 1} under multiplication modulo n.

**Properties**:

- |U(n)| = φ(n), where φ is Euler's totient function
- U(n) is abelian
- U(n) is cyclic if and only if n = 1, 2, 4, pᵏ, or 2pᵏ for odd prime p

**Example**: U(10) = {1, 3, 7, 9} with multiplication mod 10. |U(10)| = φ(10) = 4.

## Examples

### Example 1: Verifying Cyclic Group Structure

**Problem**: Show that Z₆ = {0, 1, 2, 3, 4, 5} with addition modulo 6 is a cyclic group and find all generators.

**Solution**:

**Step 1**: Verify group axioms

- Closure: Adding any two elements mod 6 gives another element in Z₆
- Associativity: Addition is associative modulo n
- Identity: 0 acts as identity (0 + a ≡ a mod 6)
- Inverse: For any a ∈ Z₆, 6-a is its inverse (a + (6-a) ≡ 0 mod 6)

**Step 2**: Find generators
Test each element as a potential generator:

- ⟨0⟩ = {0} — not a generator
- ⟨1⟩ = {0, 1, 2, 3, 4, 5} = Z₆ — generator
- ⟨2⟩ = {0, 2, 4, 2, 4, 2} = {0, 2, 4} — not a generator
- ⟨3⟩ = {0, 3, 0, 3, ...} = {0, 3} — not a generator
- ⟨4⟩ = {0, 4, 2, 0, 4, 2} = {0, 2, 4} — not a generator
- ⟨5⟩ = {0, 5, 4, 3, 2, 1} = Z₆ — generator

**Conclusion**: Z₆ is cyclic with generators 1 and 5. Note that gcd(1,6) = 1 and gcd(5,6) = 1. In general, generator elements in Zₙ are those relatively prime to n.

### Example 2: Analyzing D₄ (Symmetries of a Square)

**Problem**: List all elements of D₄ (symmetries of a square) and verify the group order.

**Solution**:

D₄ has 8 elements representing symmetries of a square:

**Rotations** (r₀ through r₃):

- r₀: Identity (0° rotation)
- r₁: Rotate 90° clockwise
- r₂: Rotate 180°
- r₃: Rotate 270° clockwise

**Reflections** (s₀ through s₃):

- s₀: Reflection across vertical axis
- s₁: Reflection across horizontal axis
- s₂: Reflection across main diagonal
- s₃: Reflection across anti-diagonal

**Verification**:

- Total elements = 4 rotations + 4 reflections = 8
- Order |D₄| = 2(4) = 8 ✓

The relations: r⁴ = e, s² = e, srs = r⁻¹

### Example 3: Computing in U(12)

**Problem**: Find the structure of U(12) = {a : 1 ≤ a ≤ 11, gcd(a,12) = 1} under multiplication modulo 12.

**Solution**:

**Step 1**: Identify elements
U(12) = {1, 5, 7, 11} (removing 2, 3, 4, 6, 8, 9, 10 which share factors with 12)

**Step 2**: Compute Cayley table

| ×₁₂ | 1   | 5   | 7   | 11  |
| --- | --- | --- | --- | --- |
| 1   | 1   | 5   | 7   | 11  |
| 5   | 5   | 1   | 11  | 7   |
| 7   | 7   | 11  | 1   | 5   |
| 11  | 11  | 7   | 5   | 1   |

**Step 3**: Analyze structure

- Each element is its own inverse: 1² ≡ 1, 5² ≡ 25 ≡ 1, 7² ≡ 49 ≡ 1, 11² ≡ 121 ≡ 1 (mod 12)
- |U(12)| = φ(12) = 4

**Conclusion**: U(12) ≅ V₄ (Klein four-group), not cyclic. This is because 12 = 2² × 3, and for n = 4, 2pᵏ (where p is odd prime), U(n) is not cyclic.

## Exam Tips

1. **Memorize the order formulas**: |Sₙ| = n!, |Aₙ| = n!/2, |Dₙ| = 2n, |Zₙ| = n, |U(n)| = φ(n)

2. **Identify abelian vs non-abelian**: Z, Zₙ, U(n) are abelian; Sₙ (n≥3), Dₙ (n≥3), Q₈ are non-abelian

3. **Generator identification**: In Zₙ, elements coprime to n are generators. In cyclic groups, if g generates, so does g⁻¹

4. **Know isomorphic relationships**: A₃ ≅ Z₃, D₃ ≅ S₃, V₄ ≅ Z₂ × Z₂, Q₈ is unique among groups of order 8

5. **Apply Cayley's Theorem**: Every finite group of order n is isomorphic to a subgroup of Sₙ

6. **Normal subgroups**: Aₙ is normal in Sₙ; know why (index = 2)

7. **Lagrange's Theorem application**: If |G| = n and H is a subgroup, then |H| divides n. Use this to determine possible subgroup orders

8. **Recognize special groups**: Q₈ has the unique property among order-8 groups that all subgroups are cyclic but the group itself is non-abelian
