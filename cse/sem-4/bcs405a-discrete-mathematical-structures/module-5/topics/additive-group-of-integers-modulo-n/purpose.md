# Learning Objectives

After studying this topic, you should students able to:

1. Define the set of integers modulo n (Zₙ) and understand the concept of congruence classes
2. Explain the binary operation of addition modulo n and verify that (Zₙ, +) forms an abelian group
3. Identify the identity element and inverses in (Zₙ, +)
4. Determine whether a given element is a generator of Zₙ by checking gcd(a, n) = 1
5. Calculate the order of any element in Zₙ using the formula ord(a) = n / gcd(a, n)
6. List all subgroups of Zₙ corresponding to the divisors of n
7. Compute Euler's totient function φ(n) and relate it to the number of generators
8. Apply the concepts of cyclic groups and isomorphisms to solve related problems
