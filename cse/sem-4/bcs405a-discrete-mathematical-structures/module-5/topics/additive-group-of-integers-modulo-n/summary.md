# Additive Group of Integers Modulo n - Summary

## Key Definitions and Concepts

- **Zₙ = {0, 1, 2, ..., n-1}**: The set of integers modulo n, representing congruence classes when divided by n
- **Addition modulo n**: a ⊕ b = (a + b) mod n
- **Cyclic Group**: A group generated by a single element; (Zₙ, +) is always cyclic
- **Generator**: Element g in Zₙ such that every element can be expressed as k·g for some integer k; generators are elements coprime to n
- **Order of Element**: Smallest positive k such that k·a ≡ 0 (mod n)
- **Euler's Totient Function φ(n)**: Counts integers less than n that are coprime to n; equals the number of generators

## Important Formulas and Theorems

- **Order Formula**: ord(a) = n / gcd(a, n) in Zₙ
- **Number of Generators**: φ(n) where φ is Euler's totient function
- **Subgroup Orders**: For each divisor d of n, there exists exactly one subgroup of order d
- **Euler's Theorem**: a^φ(n) ≡ 1 (mod n) when gcd(a, n) = 1
- **Isomorphism**: Every cyclic group of order n is isomorphic to (Zₙ, +)

## Key Points

- (Zₙ, +) satisfies closure, associativity, has identity 0, every element has inverse (n-a), and is commutative → it is an abelian group
- Element 1 always generates Zₙ, and there are φ(n) generators total
- In Zₙ, gcd(a, n) = 1 if and only if a generates Zₙ
- All subgroups of Zₙ are cyclic and correspond to divisors of n
- Zₙ\* (the multiplicative group of units) has order φ(n)
- When n is prime, Zₙ is a field with p elements
- The additive group (Zₙ, +) and multiplicative group (Zₙ\*, ×) are different structures

## Common Mistakes to Avoid

1. **Forgetting to reduce modulo n**: Always apply "mod n" after addition; writing 7 ⊕ 8 = 15 in Z₉ instead of 6
2. **Confusing additive and multiplicative identities**: Identity in (Zₙ, +) is 0, not 1
3. **Wrong formula for order**: Using n × a instead of n / gcd(a, n)
4. **Including 0 as a generator**: gcd(0, n) = n ≠ 1 (for n > 1), so 0 never generates Zₙ

## Revision Tips

1. Practice computing orders of elements using both the formula and direct verification
2. Memorize the generators of small groups like Z₈, Z₁₂, Z₆ to understand patterns
3. Remember that φ(p) = p-1 for prime p, and verify with examples
4. Draw the subgroup lattice for Z₁₂ to visualize the divisor-subgroup correspondence
5. Understand that the additive and multiplicative structures are fundamentally different—never confuse them
