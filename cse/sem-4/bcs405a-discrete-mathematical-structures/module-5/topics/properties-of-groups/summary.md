# Properties of Groups - Summary

## Key Definitions and Concepts

- **Group (G, \*)**: A set with binary operation satisfying closure, associativity, identity, and inverses
- **Abelian Group**: A group where a _ b = b _ a for all elements (commutative)
- **Cyclic Group**: A group generated by a single element; every element is a power of the generator
- **Order of Group**: Number of elements in the group (finite or infinite)
- **Order of Element**: Smallest positive integer n such that aⁿ = e
- **Subgroup**: A subset of a group that is itself a group under the same operation

## Important Formulas and Theorems

- **Order of cyclic group of order n**: n generators exist, equal to φ(n) (Euler's totient)
- **Lagrange's Theorem**: If H is a subgroup of finite group G, then |H| divides |G|
- **Cayley's Theorem**: Every group of order n is isomorphic to a subgroup of Sₙ
- **Euler's Theorem**: a^φ(n) ≡ 1 (mod n) when gcd(a, n) = 1
- **Fermat's Little Theorem**: a^(p-1) ≡ 1 (mod p) for prime p
- **(a⁻¹)⁻¹ = a**: Inverse of inverse returns original element
- **(ab)⁻¹ = b⁻¹a⁻¹**: Inverse of product reverses order

## Key Points

- Identity element in a group is unique
- Inverse of each element is unique
- Cancellation laws hold in all groups: if ax = ay, then x = y
- Every cyclic group is abelian, but not all abelian groups are cyclic
- An element a generates a cyclic group if and only if gcd(a, n) = 1 for group of order n
- {e} (trivial subgroup) and G (whole group) are always subgroups of G
- For subgroup test: nonempty subset H is subgroup if a, b ∈ H implies ab⁻¹ ∈ H

## Common Mistakes to Avoid

- Forgetting to verify closure when checking if a set forms a group
- Confusing the order of an element with the order of the group
- Assuming all abelian groups are cyclic (counterexample: ℤ₂ × ℤ₂ is abelian but not cyclic)
- Not taking modulus when working with modular arithmetic groups
- Mixing up additive and multiplicative notation for inverses

## Revision Tips

1. Practice verifying group axioms with at least 3-4 different examples
2. Memorize standard groups: (ℤ, +), (ℤₙ, +ₙ), (ℚ, +), (ℝ, +)
3. Remember: S₃ and beyond are non-abelian; symmetric groups are your primary non-abelian examples
4. For cyclic groups, always check gcd(a, n) = 1 to find generators
5. Use Lagrange's Theorem to quickly eliminate impossible subgroup orders
6. Review past university questions on group theory properties
