# Cyclic Groups - Summary

## Key Definitions and Concepts

- **Cyclic Group**: A group G is cyclic if ∃a ∈ G such that G = ⟨a⟩ = {a^n : n ∈ Z}. The element a is called the generator.

- **Order of Element**: The smallest positive integer n such that a^n = e (identity). If no such n exists, the element has infinite order.

- **Cyclic Subgroup**: The set ⟨a⟩ = {a^n : n ∈ Z} forms a subgroup generated by element a.

- **Generator**: An element a ∈ G such that every element of G can be expressed as a^n for some integer n.

## Important Formulas and Theorems

- **Number of Generators**: In a cyclic group of order n, the number of generators is φ(n) (Euler's totient function).

- **Order of Element k in Zn**: o(k) = n / gcd(k, n)

- **Subgroup Orders**: For a cyclic group of order n, there is exactly one subgroup for each divisor d of n, and that subgroup has order d, generated by n/d.

- **Isomorphism**: Cyclic group of order n ≅ Zn; Infinite cyclic group ≅ Z

## Key Points

1. Every cyclic group is abelian (commutative): a^m · a^n = a^(m+n) = a^n · a^m

2. In Zn, elements coprime to n are generators (φ(n) of them)

3. The identity element in any cyclic group has order 1

4. A cyclic group of prime order has only two generators (the identity and one other)

5. All subgroups of a cyclic group are cyclic (Fundamental Theorem)

6. The group of integers Z under addition is the only infinite cyclic group up to isomorphism

7. For element a of order m, a^k has order m/gcd(m,k)

8. Cyclic groups are the simplest groups and serve as building blocks for more complex structures

## Common Mistakes to Avoid

1. Confusing the order of an element with the order of the group—they are different concepts.

2. Forgetting that in additive notation, we use multiples (na) rather than powers (a^n).

3. Assuming all groups with one generator are finite—infinite cyclic groups like Z exist.

4. Incorrectly calculating gcd when finding generators; remember gcd(k,n) must equal 1.

## Revision Tips

1. Practice identifying generators in various Zn groups by checking gcd(k,n) = 1.

2. Memorize the relationship: order of subgroup = n/d where d is the divisor, generated by n/d.

3. Remember that cyclic groups are always abelian—this is a key distinguishing property.

4. Solve at least 5-10 problems on finding subgroups and generators to be exam-ready.
