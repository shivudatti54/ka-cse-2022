## Introduction

Cyclic groups represent one of the most fundamental and elegant structures in abstract algebra. A cyclic group is a group that can be generated by a single element through repeated application of the group operation. This means every element in the group can be expressed as some power (or multiple) of a particular element called the generator. The study of cyclic groups is essential in discrete mathematics because they serve as the building blocks for more complex algebraic structures and find applications in cryptography, coding theory, and computer science.

The importance of cyclic groups in the university's Discrete Mathematical Structures course cannot be overstated. They provide a clear and tractable example of how abstract algebraic concepts work, making them ideal for understanding fundamental properties like subgroups, generators, and group isomorphism. Moreover, cyclic groups form the theoretical foundation for understanding finite fields and are extensively used in cryptographic algorithms like RSA and elliptic curve cryptography.

In this topic, we will explore the definition of cyclic groups, their properties, generators, subgroups, and important theorems that govern their behavior. We will examine both finite and infinite cyclic groups with numerous examples to solidify understanding.

## Key Concepts

### Definition of Cyclic Group

A group (G, \*) is called a **cyclic group** if there exists an element a ∈ G such that every element of G can be expressed as a^n for some integer n (in multiplicative notation) or as na for some integer n (in additive notation). The element a is called the **generator** of the group, and we write G = ⟨a⟩.

In additive notation (commonly used for groups of integers modulo n), we write G = {na : n ∈ Z}.

**Example**: The group of integers under addition, (Z, +), is a cyclic group with generator 1 (or -1). Every integer can be written as n(1) for some integer n.

### Order of an Element

The **order** of an element a in a group G, denoted o(a) or |a|, is the smallest positive integer n such that a^n = e (identity element). If no such n exists, the element has infinite order.

**Example**: In the group Z₈ = {0, 1, 2, 3, 4, 5, 6, 7} under addition modulo 8:

- o(2) = 4 because 2+2+2+2 = 8 ≡ 0 (mod 8)
- o(3) = 8 because 3 has order 8
- o(4) = 2 because 4+4 = 8 ≡ 0 (mod 8)

### Cyclic Subgroup

Given any element a in a group G, the set ⟨a⟩ = {a^n : n ∈ Z} (or {na : n ∈ Z} in additive notation) forms a subgroup of G called the **cyclic subgroup generated by a**. The order of this subgroup equals the order of the element a.

**Theorem**: For any group G and element a ∈ G, the set ⟨a⟩ is a subgroup of G.

### Generators of Cyclic Groups

In a cyclic group G = ⟨a⟩, there may be more than one generator. The set of all generators depends on the order of the group:

1. **For infinite cyclic group (Z, +)**: The generators are 1 and -1 only.

2. **For finite cyclic group of order n**: The generators are exactly those elements a^k where gcd(k, n) = 1. The number of generators is φ(n), where φ is Euler's totient function.

**Example**: In Z₁₂ (order 12), the generators are elements coprime to 12: 1, 5, 7, 11. So there are φ(12) = 4 generators.

### Subgroups of Cyclic Groups

**Fundamental Theorem of Cyclic Groups**: Every subgroup of a cyclic group is cyclic. Moreover, for a cyclic group of order n:

- For each divisor d of n, there exists exactly one subgroup of order d.
- The subgroup of order d is generated by n/d.

**Example**: In Z₁₂, the divisors of 12 are 1, 2, 3, 4, 6, 12. The subgroups are:

- Order 12: ⟨1⟩ = Z₁₂
- Order 6: ⟨2⟩ = {0, 2, 4, 6, 8, 10}
- Order 4: ⟨3⟩ = {0, 3, 6, 9}
- Order 3: ⟨4⟩ = {0, 4, 8}
- Order 2: ⟨6⟩ = {0, 6}
- Order 1: ⟨0⟩ = {0}

### Classification of Cyclic Groups

1. **Infinite Cyclic Group**: Isomorphic to (Z, +). Example: The group of all integers under addition.

2. **Finite Cyclic Group of order n**: Isomorphic to (Zₙ, +ₙ). Example: The group of n-th roots of unity under multiplication.

### Properties of Cyclic Groups

1. Every cyclic group is **abelian** (commutative), since a^m · a^n = a^(m+n) = a^(n+m) = a^n · a^m.

2. A finite cyclic group of order n is **isomorphic** to Zₙ under addition modulo n.

3. If G is a cyclic group of order n and d divides n, then there exists exactly one subgroup of order d.

4. In a cyclic group of order n, the equation x^k = e has exactly gcd(k, n) solutions.

## Examples

### Example 1: Identifying Cyclic Groups

**Problem**: Show that the group (Z₆, +₆) is cyclic and find all its generators.

**Solution**:

- Z₆ = {0, 1, 2, 3, 4, 5} under addition modulo 6
- Check if 1 is a generator: ⟨1⟩ = {0·1, 1·1, 2·1, 3·1, 4·1, 5·1} = {0, 1, 2, 3, 4, 5} = Z₆ ✓
- Check if 5 is a generator: ⟨5⟩ = {0·5, 1·5, 2·5, 3·5, 4·5, 5·5} = {0, 5, 4, 3, 2, 1} = Z₆ ✓
- The generators are elements coprime to 6: φ(6) = 2 generators, which are 1 and 5.

### Example 2: Finding Subgroups

**Problem**: Find all subgroups of the cyclic group Z₁₈ and identify their generators.

**Solution**:
Divisors of 18: 1, 2, 3, 6, 9, 18

For each divisor d, the subgroup has order d and is generated by 18/d:

- Order 18: ⟨1⟩ = Z₁₈ (generator: 1)
- Order 9: ⟨2⟩ = {0, 2, 4, 6, 8, 10, 12, 14, 16} (generator: 2, also 16)
- Order 6: ⟨3⟩ = {0, 3, 6, 9, 12, 15} (generator: 3, also 15)
- Order 3: ⟨6⟩ = {0, 6, 12} (generator: 6, also 12)
- Order 2: ⟨9⟩ = {0, 9} (generator: 9)
- Order 1: ⟨0⟩ = {0} (generator: 0)

### Example 3: Order of Elements

**Problem**: In the cyclic group Z₁₂, find the order of element 8 and the cyclic subgroup generated by it.

**Solution**:

- We need smallest n such that 8n ≡ 0 (mod 12)
- 8 × 1 = 8 ≡ 8 (mod 12)
- 8 × 2 = 16 ≡ 4 (mod 12)
- 8 × 3 = 24 ≡ 0 (mod 12)

So o(8) = 3, and the cyclic subgroup is ⟨8⟩ = {0, 4, 8} with order 3.

## Exam Tips

1. **Remember the definition**: A group G is cyclic if ∃a ∈ G such that G = ⟨a⟩ = {a^n : n ∈ Z}.

2. **Generator identification**: In Zn, element k generates a subgroup of order n/gcd(k,n). For k to generate the entire group, gcd(k,n) must equal 1.

3. **Number of generators**: In a cyclic group of order n, the number of generators is φ(n) (Euler's totient function).

4. **Subgroup theorem**: Every subgroup of a cyclic group is cyclic. For each divisor d of n, there is exactly one subgroup of order d.

5. **Cyclic implies abelian**: Always remember that all cyclic groups are abelian. This is a key property used in many proofs.

6. **Isomorphism**: A cyclic group of order n is isomorphic to Zn, and an infinite cyclic group is isomorphic to Z.

7. **Order properties**: If a has order m, then a^k has order m/gcd(m,k).

8. **Additive vs multiplicative notation**: Be comfortable with both notations. In Zn under addition, we use multiples na; in multiplicative groups, we use powers a^n.

9. **Identity element**: Remember that in additive notation, the identity is 0; in multiplicative notation, it's e or 1.

10. **Lagrange's theorem connection**: The order of any subgroup divides the order of the group. In cyclic groups, for each divisor, there's exactly one subgroup.
