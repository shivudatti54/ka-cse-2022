# Lagrange's Theorem - Summary

## Key Definitions and Concepts

- **Order of a group** |G|: Number of elements in group G
- **Order of a subgroup** |H|: Number of elements in subgroup H of G
- **Left coset** gH: Set {gh : h ∈ H} for g ∈ G
- **Index** [G:H]: Number of distinct cosets of H in G

## Important Formulas and Theorems

- **Lagrange's Theorem**: For any finite group G and subgroup H, |H| divides |G|
- **Group-Coset relationship**: |G| = [G:H] × |H|
- **Element order corollary**: Order of any element in a finite group divides |G|
- **Cyclic group property**: In a cyclic group of order n, there exists exactly one subgroup for each divisor of n

## Key Points

1. Lagrange's Theorem provides a necessary condition: subgroup order must divide group order
2. The index [G:H] represents how many cosets are needed to cover G
3. Cosets partition the group - they are disjoint and cover all elements
4. If |G| = p (prime), G is cyclic with no nontrivial proper subgroups
5. Order of identity element is always 1
6. The theorem applies only to finite groups
7. Possible subgroup orders are precisely the divisors of |G|

## Common Mistakes to Avoid

- Confusing coset with subgroup - cosets are not necessarily subgroups
- Forgetting that divisibility is necessary but not sufficient for subgroup existence
- Assuming all divisors correspond to existing subgroups (true only for cyclic groups)
- Using the theorem for infinite groups where it doesn't apply

## Revision Tips

1. Practice finding all subgroups of small groups like Z₁₂ and S₃
2. Verify Lagrange's Theorem by explicitly listing cosets
3. Memorize that prime-order groups are always cyclic
4. Solve problems involving determination of impossible subgroup orders
5. Remember that element orders must divide the group order
