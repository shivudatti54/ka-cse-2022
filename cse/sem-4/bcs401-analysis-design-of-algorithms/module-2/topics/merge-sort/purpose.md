# Learning Objectives

After studying this topic, you should students able to:

1. Explain the divide-and-conquer paradigm as applied to Merge Sort
2. Trace through the complete execution of Merge Sort on a given input array
3. Derive and solve the recurrence relation T(n) = 2T(n/2) + Θ(n) to prove Θ(n log n) time complexity
4. Analyze the space complexity of Merge Sort and explain why O(n) auxiliary space is required
5. Determine whether Merge Sort is stable and explain why the stability property holds
6. Compare Merge Sort with other sorting algorithms (Quick Sort, insertion sort) based on time, space, and stability
7. Identify practical applications where Merge Sort is preferred, particularly in external sorting scenarios
8. Apply the Master Theorem to analyze recursive algorithms with similar structure
