# Exhaustive search (brute-force) systematically examines all solution candidates to guarantee optimality. Key algorithms covered: (1) DFS/BFS for graph traversal with O(|V|+|E|) complexity; (2) Backtracking for constraint satisfaction (N-Queens example); (3) Branch and bound with bounds-based pruning for optimization (Knapsack); (4) TSP exhaustive solution with O(n!) complexity analyzing all permutations. The Travelling Salesman Problem demonstrates why pure exhaustive search becomes infeasible beyond n≈20-25, motivating advanced techniques. Formal proofs establish optimality guarantees, and comparative analysis with branch and bound highlights the practical utility of pruning. This content bridges theoretical foundations to algorithmic design decisions in practice.
