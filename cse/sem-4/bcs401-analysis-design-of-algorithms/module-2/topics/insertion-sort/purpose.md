# Learning Objectives

After studying this topic, you should students able to:

1. Describe the Insertion Sort algorithm and explain its incremental sorting strategy
2. Analyze the time complexity of Insertion Sort for best, average, and worst-case inputs
3. Prove the correctness of Insertion Sort using loop invariant methodology
4. Calculate exact comparison counts for given input arrays
5. Explain why Insertion Sort is considered stable and adaptive
6. Compare Insertion Sort with other sorting algorithms (Selection Sort, Merge Sort, Quick Sort) based on time, space, stability, and adaptivity
7. Identify practical scenarios where Insertion Sort is preferred despite its O(n²) worst-case complexity
8. Determine when Insertion Sort exhibits linear running time and explain the underlying conditions
