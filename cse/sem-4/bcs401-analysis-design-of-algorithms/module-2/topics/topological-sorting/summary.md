# Topological sorting linearizes vertices of a Directed Acyclic Graph (DAG) respecting precedence constraints. A graph admits a topological ordering iff it is acyclic (proved via source vertex existence lemma). Two O(V+E) algorithms exist: (1) Kahn's algorithm using indegree tracking with a queue—processes sources, decrements neighbor indegrees, detects cycles when queue empties prematurely; (2) DFS-based algorithm using post-order traversal with reversal—adds vertices after exploring all descendants, detects cycles via recursion stack. Both use O(V) auxiliary space. The BFS approach builds order forward iteratively; DFS approach builds backward recursively then reverses. Multiple valid orderings may exist for a given DAG.
