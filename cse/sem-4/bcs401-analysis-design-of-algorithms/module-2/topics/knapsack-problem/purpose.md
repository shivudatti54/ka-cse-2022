# Learning Objectives

After studying this topic, you should students able to:

1. Formulate the 0/1 Knapsack Problem mathematically with clear definitions of constraints and objective function.

2. Explain and implement the exhaustive search (brute-force) algorithm for the Knapsack Problem, analyzing its time and space complexity.

3. Prove the correctness of the exhaustive search algorithm using the fact that all possible subsets are enumerated.

4. Prove the optimal substructure property of the 0/1 Knapsack Problem and understand its implications for algorithm design.

5. Derive and explain the dynamic programming recurrence relation for the 0/1 Knapsack Problem.

6. Analyze the time and space complexity of the dynamic programming solution and contrast it with exhaustive search.

7. Explain why the 0/1 Knapsack Problem is NP-hard and understand the implications for algorithmic tractability.

8. Distinguish between 0/1 and Fractional Knapsack Problems, explaining why greedy works for the fractional variant.
