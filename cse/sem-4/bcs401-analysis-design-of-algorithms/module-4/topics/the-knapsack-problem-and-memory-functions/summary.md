The Knapsack Problem is a fundamental combinatorial optimization problem where items with weights and values must be selected to maximize total value within a capacity constraint. The 0/1 variant (each item taken wholly or not at all) exhibits optimal substructure, enabling dynamic programming solutions with O(nW) time complexity. The recurrence relation dp[i][w] = max(dp[i-1][w], dp[i-1][w-w[i]] + v[i]) captures the binary choice at each state. Space optimization reduces memory to O(W) by iterating capacity backward. The Fractional Knapsack Problem permits partial items and is optimally solved by a greedy algorithm selecting by value-to-weight ratio in O(n log n). Memory functions provide a top-down alternative using memoization, computing only necessary subproblems and offering advantages when the solution space is sparse. The study material includes formal proofs, detailed numerical examples, edge case analysis, and hard analytical assessment questions.
