# Learning Objectives

After studying this topic, you should students able to:

1. Define minimum spanning tree and explain its properties in the context of graph theory
2. Describe Kruskal's Algorithm and its step-by-step procedure for finding MST
3. Explain the greedy choice property and the cut property that justify the algorithm's correctness
4. Implement and analyze the Disjoint Set Union (Union-Find) data structure with union-by-rank and path compression
5. Compute the time and space complexity of Kruskal's Algorithm
6. Trace the execution of Kruskal's Algorithm on given weighted graphs to determine the MST
7. Compare and contrast Kruskal's Algorithm with Prim's Algorithm for MST construction
8. Apply Kruskal's Algorithm to solve practical optimization problems involving network design
