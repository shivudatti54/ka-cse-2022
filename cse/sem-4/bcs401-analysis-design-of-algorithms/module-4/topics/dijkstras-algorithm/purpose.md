# Learning Objectives

After studying this topic, you should students able to:

1. Understand the formal problem definition of single-source shortest path in weighted graphs
2. Explain the greedy choice property that makes Dijkstra's algorithm work
3. Trace through Dijkstra's algorithm step-by-step on a given graph
4. Analyze the time complexity of Dijkstra's algorithm using different data structures
5. Prove the correctness of Dijkstra's algorithm using induction and the relaxation invariant
6. Identify the limitations of Dijkstra's algorithm, particularly regarding negative edge weights
7. Implement Dijkstra's algorithm in pseudocode and reconstruct shortest paths using predecessor pointers
8. Compare Dijkstra's algorithm with other shortest path algorithms (Bellman-Ford, Floyd-Warshall)
9. Apply Dijkstra's algorithm to practical problems such as network routing and navigation systems
