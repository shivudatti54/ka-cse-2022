# What Is An Algorithm

An algorithm is a finite sequence of well-defined instructions that transforms inputs into outputs. Five essential characteristics define a valid algorithm: finiteness (must terminate), definiteness (unambiguous steps), input (zero or more inputs), output (one or more outputs), and effectiveness (basic operations). Algorithms exist conceptually independent of their implementations—distinguishing between algorithm (abstract method), pseudocode (intermediate representation), and program (concrete implementation) is crucial. Algorithm analysis focuses on time complexity (Big-O notation) and space complexity to evaluate efficiency. The course will build upon this foundation to cover asymptotic analysis and various design paradigms in subsequent modules.
