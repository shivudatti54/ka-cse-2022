# Sequential Search And Brute Force String Matching

Sequential Search is the simplest search algorithm with Θ(n) worst-case time complexity, applicable to both sorted and unsorted arrays. The algorithm can be optimized for sorted inputs to achieve early termination when elements exceed the search key. Correctness is established through loop invariant proofs. Brute Force String Matching extends sequential search to pattern recognition, achieving Θ(nm) worst-case complexity by comparing patterns against each text position. Despite theoretical suboptimality, brute force remains practical for short patterns and small alphabets. Key analytical techniques include loop invariant construction for correctness proofs, asymptotic complexity analysis, and mathematical derivation of expected comparison counts.
