# Learning Objectives

After studying this topic, you should students able to:

1. Explain the working principle of Counting Sort and its three-phase operation
2. Prove the time complexity of Counting Sort as Θ(n + k) through analysis of each phase
3. Analyze the space complexity of Counting Sort and explain when it becomes impractical
4. Prove the stability property of Counting Sort and explain its importance in Radix Sort
5. Trace through the Counting Sort algorithm step-by-step for any given input
6. Compare Counting Sort with comparison-based sorts in terms of time complexity and practical applicability
7. Identify the limitations of Counting Sort and determine appropriate scenarios for its use
8. Apply the transform-and-conquer paradigm to understand how Counting Sort transforms the sorting problem
