# Learning Objectives

After studying this topic, you should students able to:

1. Define and explain the row space and column space of a matrix as subspaces of the appropriate Euclidean spaces.

2. Compute the rank of a matrix by finding the dimension of its row space or column space using row reduction techniques.

3. Determine whether a set of vectors forms a basis for the row space or column space of a given matrix.

4. Apply elementary row operations to transform a matrix to row echelon form or reduced row echelon form without changing its rank.

5. Identify the pivot columns in reduced row echelon form and use them to construct a basis for the column space of the original matrix.

6. Verify and apply the fundamental theorem that row rank equals column rank for any matrix.

7. Use the concepts of row space and column space to determine if a linear system Ax = b is consistent by checking if b belongs to C(A).
