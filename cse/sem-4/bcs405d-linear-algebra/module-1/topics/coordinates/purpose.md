# Learning Objectives

After studying this topic, you should students able to:

1. Define and explain the concept of an ordered basis and coordinate vector in a vector space.

2. Find the coordinates of a given vector with respect to a specified basis.

3. Construct the transition matrix between two given bases in ℝⁿ.

4. Apply the transition matrix to convert coordinates of vectors from one basis to another.

5. Verify the inverse relationship between transition matrices for opposite coordinate transformations.

6. Convert coordinates between Cartesian and polar coordinate systems in ℝ².

7. Determine the matrix representation of a linear transformation with respect to a given basis.

8. Apply coordinate concepts to solve practical students problems involving vector representations.
