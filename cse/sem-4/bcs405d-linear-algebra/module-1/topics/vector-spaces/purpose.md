# Learning Objectives

After studying this topic, you should students able to:

1. Define a vector space and verify whether a given set with defined operations forms a vector space using the ten axioms.

2. Identify and provide examples of vector spaces including ℝⁿ, polynomial spaces, matrix spaces, and function spaces.

3. Determine if a subset W of a vector space V is a subspace by checking the three necessary conditions (non-empty, closure under addition, closure under scalar multiplication).

4. Express a vector as a linear combination of given vectors and determine the span of a set of vectors.

5. Determine whether a set of vectors is linearly independent or linearly dependent, and identify when vectors form a basis of the vector space.

6. Find the basis and dimension of various vector spaces and subspaces, including those defined by equations.

7. Apply the concepts of vector spaces to solve problems related to linear combinations, independence, and dimensionality in applied contexts.
