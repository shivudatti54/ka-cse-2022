# Learning Objectives

After studying this topic, you should students able to:

1. Define and explain the concept of a linear combination of vectors in ℝⁿ.

2. Define the linear span (or simply span) of a set of vectors and express it mathematically using set notation.

3. Identify and describe the geometric interpretation of spans in ℝ² and ℝ³ (lines and planes through the origin).

4. Determine whether a given vector belongs to the span of a set of vectors by solving appropriate linear equations.

5. Verify if a given set of vectors forms a spanning set for ℝ² or ℝ³.

6. Apply the properties of linear span to solve problems related to subspaces.

7. Distinguish between spanning sets that generate the entire space versus proper subspaces.
