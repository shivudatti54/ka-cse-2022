# Learning Objectives

After studying this topic, you should be able to:

1. Define and explain the concepts of linear independence and spanning sets in vector spaces
2. Define a basis for a vector space and identify whether a given set of vectors forms a basis
3. Explain the concept of dimension and determine the dimension of various vector spaces and subspaces
4. Identify standard bases for common vector spaces such as ℝⁿ, polynomial spaces, and matrix spaces
5. Find a basis for subspaces defined by equations, spans of vectors, and solution spaces of homogeneous systems
6. Apply the theorem that all bases of a finite-dimensional vector space have the same number of elements
7. Extend a linearly independent set to a basis using the Basis Extension Theorem
8. Determine the dimension and find bases for the column space and null space of a matrix
