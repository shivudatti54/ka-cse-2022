# Learning Objectives

After studying this topic, you should students able to:

1. Define and distinguish between the kernel (null space) and image (column space) of a linear transformation, including their properties and geometric interpretations.

2. State the Rank-Nullity Theorem precisely for both linear transformations and matrices, explaining its significance in relating domain dimensions to range and null space dimensions.

3. Compute the rank and nullity of a given matrix by applying row reduction techniques to find the reduced row echelon form and identify pivot columns.

4. Apply the Rank-Nullity Theorem to determine whether a linear transformation is one-to-one (injective) or onto (surjective) based on its rank and nullity values.

5. Solve systems of linear equations using Rank-Nullity concepts, determining consistency conditions and characterizing solution sets in terms of free variables.

6. Identify and describe the four fundamental subspaces of a matrix (column space, null space, row space, left null space) and understand their dimensional relationships.

7. Apply the theorem to determine the dimensions of solution spaces for homogeneous linear systems and to find bases for these subspaces.
