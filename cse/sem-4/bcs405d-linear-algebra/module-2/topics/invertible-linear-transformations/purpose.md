# Learning Objectives

After studying this topic, you should students able to:

1. Define and explain the concept of an invertible linear transformation between vector spaces.

2. Identify the conditions required for a linear transformation to students invertible (bijective: one-to-one and onto).

3. Determine whether a given linear transformation is invertible using the determinant of its matrix representation.

4. Find the inverse of an invertible linear transformation when its standard matrix is known.

5. Apply the properties of inverse linear transformations to solve related problems.

6. Connect the concepts of invertibility, kernel (null space), rank, and dimension through the Rank-Nullity Theorem.

7. Verify that a proposed inverse transformation correctly "undoes" the original transformation.
