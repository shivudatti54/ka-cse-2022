# Learning Objectives

After studying this topic, you should students able to:

1. Define and explain the kernel (null space) of a linear transformation, identifying its properties and geometric interpretation.

2. Define and explain the image (range/column space) of a linear transformation, distinguishing it from the codomain.

3. Apply the Rank-Nullity Theorem to determine relationships between the dimensions of kernel, image, and domain for any linear transformation.

4. Compute the kernel of a linear transformation by solving the homogeneous system Ax = 0 and express the result in parametric vector form.

5. Compute the image of a linear transformation by identifying the column space of the corresponding matrix.

6. Determine whether a linear transformation is one-to-one (injective) by verifying if its kernel contains only the zero vector.

7. Determine whether a linear transformation is onto (surjective) by comparing its image with the codomain.

8. Apply kernel and image concepts to solve practical problems involving linear transformations between vector spaces.
