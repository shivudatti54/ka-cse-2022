# Learning Objectives

After studying this topic, you should students able to:

1. Define a linear transformation and verify whether a given function qualifies as a linear transformation between vector spaces.

2. Explain the concept of matrix representation of a linear transformation with respect to given bases.

3. Find the matrix representation of a linear transformation T: ℝⁿ → ℝᵐ with respect to standard bases.

4. Compute the matrix of composite linear transformations and understand the relationship between transformation composition and matrix multiplication.

5. Apply the change of basis formula to find matrix representations in different coordinate systems.

6. Identify and construct standard matrices for common geometric transformations including rotations, reflections, scalings, projections, and shears.

7. Use matrix representations to evaluate linear transformations at given vectors through matrix-vector multiplication.
