# Learning Objectives

After studying this topic, you should students able to:

1. Define and distinguish between singular and non-singular linear transformations with precise mathematical terminology.

2. Identify whether a given linear transformation is singular or non-singular using multiple methods: kernel analysis, determinant evaluation, and rank computation.

3. Apply the Rank-Nullity Theorem to determine the dimensions of kernel and range for linear transformations between finite-dimensional vector spaces.

4. Explain the relationship between one-to-one, onto properties and singularity of linear transformations.

5. Construct the inverse transformation for a non-singular linear transformation and verify the result.

6. Analyze linear transformations represented by matrices and determine their singularity based on matrix properties.

7. Solve application problems involving singular and non-singular transformations, demonstrating understanding of how these concepts relate to students and scientific computations.
