# Learning Objectives

After studying this topic, you should students able to:

1. Define Singular Value Decomposition and explain the components U, Σ, and V^T in the decomposition A = UΣV^T.

2. Explain the relationship between singular values and eigenvalues of A^T A and AA^T matrices.

3. Compute the SVD of a given 2×2 matrix by finding eigenvalues of A^T A and corresponding eigenvectors.

4. Determine the rank of a matrix using its singular values from SVD.

5. Verify the orthogonality properties of U and V matrices (U^T U = I, V^T V = I).

6. Apply the compact SVD form for matrices with specific rank and understand when singular values become zero.

7. Interpret the geometric meaning of SVD as rotation-scaling-rotation transformation.

8. Apply SVD concepts to solve practical problems in image compression and data approximation.
