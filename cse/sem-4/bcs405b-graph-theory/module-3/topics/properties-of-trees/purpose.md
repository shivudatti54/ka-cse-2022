# Learning Objectives

After studying this topic, you should students able to:

1. Define a tree, forest, leaf vertex, internal vertex, and distinguish between trees and forests in graph theory.

2. Prove and apply the fundamental property that a tree with n vertices contains exactly n-1 edges.

3. Apply the various characterization theorems to determine whether a given graph is a tree.

4. Find the center and radius of a tree using the leaf pruning iterative method.

5. Explain the concept of spanning trees and determine the number of edges to remove from a connected graph to obtain a spanning tree.

6. Apply Cayley's Formula to calculate the number of spanning trees in complete graphs.

7. Understand rooted trees, binary trees, and the relationship between leaves and internal vertices in binary trees.

8. Describe the basic approach for finding minimum spanning trees using Kruskal's and Prim's algorithms.
