# Learning Objectives

After studying this topic, you should students able to:

1. Define and distinguish between trees, labeled trees, and spanning trees in graph theory.

2. Apply Cayley's Formula to calculate the number of labeled trees on n vertices.

3. Construct and interpret Prüfer sequences for labeled trees.

4. Use the Matrix-Tree Theorem to count spanning trees in any connected graph.

5. Compute the number of spanning trees in special graphs including complete graphs, complete bipartite graphs, and cycle graphs.

6. Understand the relationship between Prüfer sequences and tree enumeration.

7. Apply degree counting techniques using Prüfer sequences to reconstruct trees.

8. Analyze and solve problems involving tree counting in various contexts.
