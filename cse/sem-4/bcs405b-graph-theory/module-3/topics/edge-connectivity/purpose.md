# Learning Objectives

After studying this topic, you should students able to:

1. Define edge-cut, edge-connectivity λ(G), and explain what it measures in a graph
2. Identify bridges (cut-edges) in a graph and distinguish between edge and vertex connectivity
3. Apply the inequality κ(G) ≤ λ(G) ≤ δ(G) to solve problems involving edge-connectivity bounds
4. Compute edge-connectivity for special graphs including cycles, complete graphs, and complete bipartite graphs
5. Use Menger's Theorem (edge version) to understand the relationship between edge cuts and edge-disjoint paths
6. Apply the Max-Flow Min-Cut Theorem to compute local edge-connectivity between vertex pairs
7. Determine whether a given graph is k-edge-connected and characterize 2-edge-connected graphs
8. Analyze real-world network reliability problems using edge-connectivity concepts
