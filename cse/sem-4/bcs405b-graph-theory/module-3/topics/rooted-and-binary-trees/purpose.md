# Learning Objectives

After studying this topic, you should students able to:

1. Define and distinguish between trees, rooted trees, and binary trees with their respective properties

2. Identify and explain key tree terminology including parent, child, leaf, level, height, and depth

3. Differentiate between full binary trees, complete binary trees, and perfect binary trees with examples

4. Perform all four binary tree traversal methods (inorder, preorder, postorder, level order) and trace through examples

5. Apply the properties of spanning trees to solve problems related to connected graphs

6. Understand the binary search tree property and its significance in efficient searching operations

7. Analyze the relationship between number of nodes, edges, leaves, and height in binary trees

8. Solve exam problems involving tree counts, traversal sequences, and tree properties
