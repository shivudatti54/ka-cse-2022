# Learning Objectives

After studying this topic, you should students able to:

1. Define and identify pendant vertices (leaf vertices) in a tree with degree 1

2. Compute the distance between any two vertices in a tree by finding the unique path between them

3. Calculate the eccentricity of a vertex as the maximum distance to any other vertex in the tree

4. Determine the radius of a tree as the minimum eccentricity and the diameter as the maximum eccentricity

5. Identify the center(s) of a tree using both the eccentricity method and the pruning algorithm

6. Apply the relationship rad(T) ≤ diam(T) ≤ 2 × rad(T) to verify tree properties

7. Solve problems involving pendant vertices, distances, and centers in various tree configurations

8. Analyze practical applications of tree centers in facility location and network design contexts
