# Learning Objectives

After studying this topic, you should students able to:

1. Define spanning tree and minimum spanning tree with precise mathematical notation
2. State and prove the fundamental properties of spanning trees including edge count and acyclic nature
3. Apply Kruskal's algorithm to construct a minimum spanning tree for a given weighted graph
4. Apply Prim's algorithm to construct a minimum spanning tree for a given weighted graph
5. Explain the cut property and cycle property of minimum spanning trees and use them in proofs
6. Compute the number of spanning trees in a complete graph using Cayley's formula
7. Compare and contrast Kruskal's and Prim's algorithms in terms of time complexity and suitable graph types
8. Identify applications of spanning trees in computer networks and optimization problems
