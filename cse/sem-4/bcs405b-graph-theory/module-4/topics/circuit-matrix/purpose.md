# Learning Objectives

After studying this topic, you should students able to:

1. Define and identify circuits (cycles) in a graph and distinguish between tree edges, chords, and fundamental circuits.

2. Construct the fundamental circuit matrix (F-matrix) for a given graph when provided with a spanning tree.

3. Calculate the number of independent circuits (cyclomatic number) in a graph using the formula c = m - n + 1.

4. Explain the properties of the circuit matrix, including its rank and relationship with the incidence matrix.

5. Apply the orthogonality property FA^T = 0 to verify circuit-cut-set relationships in graph theory problems.

6. Compare and contrast the circuit matrix with other graph matrices like incidence matrix and cut-set matrix.

7. Solve numerical problems involving the construction and analysis of circuit matrices for university examinations.
