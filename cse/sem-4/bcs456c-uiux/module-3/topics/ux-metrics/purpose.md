# Learning Objectives

After studying this topic, you should students able to:

1. Define UX metrics and explain their importance in the user-centered design process

2. Differentiate between behavioral, attitudinal, and composite UX metrics with examples of each

3. Calculate and interpret key UX metrics including SUS, NPS, and CSAT scores

4. Apply TIME metrics (Time, Errors, Efficiency, Memorability) to evaluate task-based user performance

5. Identify appropriate metrics for different evaluation contexts (formative vs. summative testing)

6. Analyze how engagement and business metrics (conversion rate, bounce rate, churn rate) relate to user experience quality

7. Evaluate accessibility compliance using WCAG guidelines and metrics

8. Design a basic UX measurement strategy combining qualitative and quantitative approaches
