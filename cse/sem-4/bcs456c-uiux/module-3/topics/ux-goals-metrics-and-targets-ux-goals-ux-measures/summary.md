# UX Goals, Metrics, and Targets - Summary

## Key Definitions and Concepts

- **UX Goals**: Specific, measurable objectives defining desired user experience outcomes (e.g., task completion, satisfaction levels)
- **UX Metrics**: Quantifiable measures used to evaluate how well designs meet objectives (e.g., task success rate, NPS, SUS)
- **UX Targets**: Specific values representing success for each metric, set using SMART criteria

## Important Formulas and Frameworks

- **SMART Goals**: Specific, Measurable, Achievable, Relevant, Time-bound
- **HEART Framework**: Happiness, Engagement, Adoption, Retention, Task Success
- **SUS Score**: 10-question standardized survey yielding score from 0-100
- **NPS Calculation**: % Promoters (9-10) minus % Detractors (0-6)

## Key Points

- UX goals focus on user outcomes while business goals focus on organizational success
- Behavioral metrics measure user actions (click rates, task completion); attitudinal metrics measure user opinions (surveys, satisfaction scores)
- Setting baselines before design changes is essential for measuring improvement
- Not all metrics are relevant—select metrics that directly align with specific goals
- Effective targets are specific, measurable, and time-bound
- Combining qualitative and quantitative metrics provides comprehensive UX insights
- Common UX metrics: task success rate, time on task, error rate, SUS, NPS, CSAT

## Common Mistakes to Avoid

- Confusing UX goals with business objectives—keep user outcomes separate from revenue targets
- Setting unrealistic targets without baseline measurement or industry benchmarks
- Using only qualitative or only quantitative metrics—both are necessary for complete understanding
- Selecting metrics that don't directly relate to project goals

## Revision Tips

1. Practice creating SMART targets for hypothetical UX scenarios
2. Memorize the HEART framework categories and examples of metrics for each
3. Remember the difference between behavioral and attitudinal metrics with examples
4. Review how to calculate SUS and NPS scores
5. Focus on the relationship between goals, metrics, and targets—they form a connected measurement system
