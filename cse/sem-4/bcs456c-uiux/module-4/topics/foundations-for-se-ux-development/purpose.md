# Learning Objectives

After studying this topic, you should students able to:

1. Explain the concept of User-Centered Design and its importance in software students contexts.

2. Describe the five stages of the Design Thinking process and apply them to solve real-world UX problems.

3. Evaluate user interfaces using Nielsen's 10 Usability Heuristics with specific examples.

4. Differentiate between various user research methods and select appropriate methods based on project requirements.

5. Design effective Information Architecture structures for software applications with multiple features.

6. Create wireframes and prototypes at appropriate fidelity levels for different stages of product development.

7. Apply WCAG accessibility guidelines to ensure software products are inclusive and legally compliant.

8. Integrate UX practices effectively within Agile and Lean software development methodologies.
