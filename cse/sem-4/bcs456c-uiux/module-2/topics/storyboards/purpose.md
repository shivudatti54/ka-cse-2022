# Learning Objectives

After studying this topic, you should students able to:

1. Define storyboarding in the context of UI/UX design and explain its purpose in the design process.

2. Differentiate between storyboards and wireframes, understanding when to use each technique.

3. Identify and describe the four main types of storyboards: linear, non-linear, thumbnail, and detailed.

4. Explain the essential elements that each panel in a storyboard should contain for effectiveness.

5. Create a basic storyboard for a given user flow or user scenario following standard conventions.

6. Analyze user journeys and translate them into visual narrative sequences using storyboarding techniques.

7. Evaluate the effectiveness of storyboards as communication tools in multidisciplinary design teams.

8. Apply storyboarding principles to identify potential UX problems early in the design phase.
