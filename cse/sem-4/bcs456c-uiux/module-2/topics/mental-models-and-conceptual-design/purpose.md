# Learning Objectives

After studying this topic, you should students able to:

1. Define mental models and explain their significance in user experience design
2. Identify the five key characteristics of mental models and explain how each affects user interaction
3. Distinguish between mental models, design models, and system images in the context of UI/UX
4. Apply the concept of mapping to create intuitive relationships between design elements and their functions
5. Explain the conceptual design process and its role in bridging user needs with technical implementation
6. Conduct and interpret card sorting exercises to uncover user mental models for information architecture
7. Create user flows and journey maps to visualize user paths through a product
8. Develop user personas based on research findings to guide design decisions
9. Analyze existing interfaces to identify alignment or misalignment with expected mental models
10. Evaluate the effectiveness of conceptual designs in meeting user goals and expectations
