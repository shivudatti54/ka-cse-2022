# Gradients in Deep Network - Summary

## Key Points

- Vanishing gradients: gradients → 0 (slow learning)
- Exploding gradients: gradients → ∞ (unstable)
- Chain rule links all layers
- Solutions: ReLU, batch norm, residual connections

## Exam Tips

1. Remember: deeper = more gradient problems
2. ReLU helps vanishing gradients
3. Gradient clipping prevents explosion
