# Convergence - Summary

## Critical Point

∇f(x) = 0

## Types

- Minimum (desired)
- Maximum
- Saddle point

## Convergence Conditions

- Learning rate < 2/L
- Lipschitz gradient
- Bounded below function

## Exam Tips

1. Remember gradient = 0 at critical point
2. Saddle points common in deep learning
3. Learning rate crucial for convergence
