# Learning Objectives

After studying this topic, you should students able to:

1. Define and interpret the gradient of a scalar function with respect to a vector variable, understanding its geometric meaning as the direction of steepest ascent.

2. Apply the product rule and chain rule correctly when computing gradients of composite and multiplied functions.

3. Compute gradients of quadratic forms xᵀAx using the appropriate identity, distinguishing between symmetric and non-symmetric matrices.

4. Construct and interpret the Jacobian matrix for vector-valued functions and the Hessian matrix for second-order derivative information.

5. Compute directional derivatives using the gradient identity Dₐf = ∇f · a and interpret their physical meaning.

6. Perform gradient descent iterations using the update formula xₖ₊₁ = xₖ - α∇f(xₖ) with specified learning rates.

7. Apply vector calculus identities to simplify and compute gradients of complex optimization objective functions.

8. Identify critical points where ∇f = 0 and understand their significance in optimization problems.
