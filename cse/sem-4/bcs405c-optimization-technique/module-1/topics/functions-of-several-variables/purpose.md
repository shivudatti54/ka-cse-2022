# Learning Objectives

After studying this topic, you should be able to:

1. Define and distinguish between functions of several variables, their domains, and represent them geometrically using level curves and surfaces.

2. Compute partial derivatives of multivariable functions, including higher-order derivatives, and apply Clairaut's theorem for mixed partials.

3. Calculate and interpret the gradient vector ∇f, understanding its direction (steepest ascent) and magnitude (maximum rate of change).

4. Compute directional derivatives in any specified direction using the formula D_u f = ∇f · u.

5. Construct the Jacobian matrix for vector-valued functions and the Hessian matrix for classifying critical points.

6. Find and classify critical points (local maxima, minima, and saddle points) using the second derivative test with the Hessian determinant.

7. Apply Taylor's theorem for multivariable functions to approximate functions near given points using quadratic forms.

8. Set up optimization problems with constraints using the method of Lagrange multipliers.
