# Convexity - Summary

## Convex Set

Line segment between any two points stays in set

## Convex Function

f(λx+(1-λ)y) ≤ λf(x) + (1-λ)f(y)

## Why Important

- Global minimum guaranteed
- Gradient descent works perfectly

## Examples

- Convex: x², eˣ, ||x||
- Non-convex: sin(x), neural networks

## Exam Tips

1. Second derivative ≥ 0 for convex
2. Local = Global for convex functions
