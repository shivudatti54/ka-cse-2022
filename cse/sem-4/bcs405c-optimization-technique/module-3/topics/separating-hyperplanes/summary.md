# Separating Hyperplane - Summary

## Equation

wᵀx + b = 0

## Distance to Hyperplane

|wᵀx + b| / ||w||

## Key Points

- Divides space into two half-spaces
- SVM finds maximum margin separator
- Support vectors on boundary

## Exam Tips

1. Remember normal vector determines orientation
2. Margin = 2/||w|| for SVM
