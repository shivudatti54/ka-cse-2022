# Learning Objectives

After studying this topic, you should be able to:

1. Explain the concept of exceptions in ARM processors and distinguish between different exception types.

2. Describe the characteristics and differences between IRQ (Interrupt Request) and FIQ (Fast Interrupt Request) exceptions.

3. Analyze the exception handling process including state saving, mode switching, and return mechanisms.

4. Interpret the ARM exception vector table and understand the vector addresses for different exceptions.

5. Configure and implement IRQ and FIQ handlers in assembly language for ARM7TDMI-based microcontrollers.

6. Evaluate the factors affecting interrupt latency and explain why FIQ has lower latency than IRQ.

7. Apply knowledge of CPSR and SPSR registers in exception handling scenarios.

8. Compare exception handling mechanisms between traditional ARM7 and modern Cortex-M processors.
