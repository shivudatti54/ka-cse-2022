# Learning Objectives

After studying this topic, you should students able to:

1. Identify and explain the functions of all ARM general-purpose registers (R0-R12)
2. Describe the roles of special-purpose registers SP (R13), LR (R14), and PC (R15)
3. Explain the structure and flags of the Current Program Status Register (CPSR)
4. Analyze how the Saved Program Status Register (SPSR) preserves processor state during exceptions
5. Compare register banking across different ARM processor modes (User, FIQ, IRQ, Supervisor, Abort, Undefined, System)
6. Apply the ARM Architecture Procedure Call Standard (AAPCS) to analyze function call sequences
7. Evaluate the impact of the instruction pipeline on PC values and branch offset calculations
8. Solve numerical problems involving CPSR flag modifications and register operations
