The Current Program Status Register (CPSR) is a 32-bit fundamental control register in ARM processors that contains condition flags (N, Z, C, V, Q), processor mode bits (M[4:0]), interrupt disable bits (I, F), and state bit (T). The N flag indicates negative results in signed arithmetic, Z flag indicates zero results, C flag indicates unsigned carry/borrow, V flag indicates signed overflow, and Q flag indicates DSP saturation. The processor supports seven modes: User (PL0), FIQ, IRQ, Supervisor, Abort, Undefined, and System (PL1). The SPSR saves CPSR during exceptions. MRS and MSR instructions access CPSR. Assessment requires analyzing flag states after specific instructions and interpreting CPSR bit fields.
