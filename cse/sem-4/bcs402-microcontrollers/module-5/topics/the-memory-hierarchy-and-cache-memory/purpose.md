# Learning Objectives

After studying this topic, you should be able to:

1. Explain the concept of memory hierarchy and describe the different levels from registers to secondary storage.

2. Define temporal and spatial locality principles and explain their importance in cache design.

3. Compare and contrast the three cache mapping techniques: direct mapping, fully associative mapping, and set associative mapping.

4. Analyze cache performance using hit rate, miss rate, and average memory access time (AMAT) calculations.

5. Distinguish between write-through and write-back policies, including their advantages and disadvantages.

6. Understand cache replacement policies such as LRU, FIFO, and Random, and explain their impact on cache performance.

7. Apply cache address mapping concepts to determine cache line index, tag, and offset for given memory addresses.

8. Explain the role of cache memory in microcontroller systems and its impact on real-time performance.
