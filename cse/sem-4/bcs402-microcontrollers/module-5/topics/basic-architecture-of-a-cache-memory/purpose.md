# Learning Objectives

After studying this topic, you should be able to:

1. Explain the concept of memory hierarchy and why cache memory is essential in microcontroller systems.

2. Describe and differentiate between the three primary cache mapping techniques: direct-mapped, fully associative, and set-associative caches.

3. Analyze memory addresses to determine cache line index, tag, and offset fields for different cache configurations.

4. Compare cache write policies (write-through and write-back) with respect to performance and data consistency.

5. Identify and explain the three types of cache misses: compulsory, capacity, and conflict misses.

6. Calculate average memory access time (AMAT) given cache hit time, miss penalty, and hit rate.

7. Understand the principles of locality of reference (temporal and spatial) that make cache memory effective.

8. Evaluate different cache replacement policies (LRU, FIFO, Random) and their impact on cache performance.
