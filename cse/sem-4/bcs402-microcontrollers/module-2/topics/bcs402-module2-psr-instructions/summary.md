# Bcs402 Module2 Psr Instructions

The Program Status Register (PSR) is a 32-bit register in ARM microcontrollers containing: (1) Condition flags (N-bit 31, Z-bit 30, C-bit 29, V-bit 28) that reflect arithmetic/logical operation results; (2) Control bits (I, F for interrupt disable, T for Thumb state); (3) Mode bits (M[4:0]) defining processor state (User, FIQ, IRQ, Supervisor, Abort, Undefined, System). MRS transfers PSR to general-purpose registers for reading; MSR writes to PSR fields using specifiers (\_c, \_x, \_s, \_f). The 16 condition codes (EQ, NE, CS, CC, MI, PL, VS, VC, HI, LS, GE, LT, GT, LE, AL) enable conditional instruction execution based on flag states. The SPSR preserves CPSR during exception handling. Understanding flag manipulation theorems (overflow/carry detection) is essential for writing correct signed/unsigned arithmetic code.
