# Loading constants in ARM requires understanding immediate value encoding: 12-bit fields containing 8-bit values rotated by even positions (0-30). Small constants use MOV/MVN; larger constants require LDR with literal pools, MOVW/MOVT combinations, or ADR/ADRL pseudo-instructions. The theorem proves any 32-bit value can be loaded using MOVW/MOVT, while literal pools provide ±4095 byte reach. Selection depends on constant size, code density, and execution speed requirements.
