# Learning Objectives

After studying this topic, you should students able to:

1. Identify and explain all categories of 8051 data processing instructions including arithmetic, logical, rotate/shift, and boolean operations.
2. Analyze the effect of each instruction on the PSW flags (C, AC, OV, P) and explain their significance in signed and unsigned arithmetic.
3. Apply appropriate addressing modes (register, direct, indirect, immediate) for different data processing instructions.
4. Design and implement multi-byte arithmetic operations using ADD and ADDC instructions.
5. Utilize the DA (Decimal Adjust) instruction correctly for BCD arithmetic operations.
6. Perform bit masking, setting, clearing, and testing operations using logical and boolean instructions.
7. Evaluate the behavior of rotate instructions (RL, RLC, RR, RRC) and their relationship with the Carry flag.
8. Debug and analyze 8051 assembly programs involving data processing by tracing register and flag states.
