# Branch instructions in the 8051 microcontroller enable dynamic program flow control through four categories: unconditional jumps (LJMP, AJMP, SJMP), conditional jumps (JZ, JNZ, JC, JNC, JB, JNB, JBC), compare-and-jump (CJNE), and loop control (DJNZ). Subroutine handling is provided via ACALL, LCALL, RET, and RETI. Unconditional jumps transfer control without condition testing; conditional jumps evaluate accumulator or flag status before branching. The Program Counter (PC) is modified either absolutely (loading a new address) or relatively (adding signed offset to PC). CJNE compares operands without storing results while affecting the carry flag for unsigned comparisons. DJNZ implements counted loops by decrementing and testing. Subroutine calls push the return address onto the stack; RET and RETI pop it back, with RETI additionally restoring interrupt logic. Execution timing varies between 2-4 machine cycles depending on instruction type and branch outcome.
