# Learning Objectives

After studying this topic, you should students able to:

1. Write and execute subqueries (nested queries) in SQL, distinguishing between correlated and non-correlated subqueries

2. Apply set operations (UNION, INTERSECT, EXCEPT) to combine results from multiple queries

3. Use aggregate functions (COUNT, SUM, AVG, MAX, MIN) with GROUP BY and HAVING clauses for data analysis

4. Implement various JOIN operations (INNER, LEFT, RIGHT, FULL) to retrieve data from multiple related tables

5. Utilize EXISTS and NOT EXISTS operators for testing subquery result existence

6. Apply ALL and ANY operators with subqueries for complex comparisons

7. Construct complex SQL queries that integrate multiple concepts for solving real-world database problems

8. Optimize query performance by selecting appropriate techniques for different retrieval scenarios
