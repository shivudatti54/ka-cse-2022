# Learning Objectives

After studying this topic, you should students able to:

1. Explain the purpose and structure of the SELECT statement in SQL for data retrieval.

2. Apply the WHERE clause with various operators (=, <>, <, >, <=, >=) to filter specific rows from database tables.

3. Use pattern matching with the LIKE operator and wildcards (% and \_) for string searches.

4. Implement logical operators (AND, OR, NOT) to combine multiple conditions in retrieval queries.

5. Utilize the DISTINCT keyword to eliminate duplicate rows from query results.

6. Apply the ORDER BY clause to sort query results in ascending or descending order.

7. Write queries using aggregate functions (COUNT, SUM, AVG, MIN, MAX) to perform calculations on data sets.

8. Implement the GROUP BY clause with aggregate functions for grouped data analysis.

9. Apply the HAVING clause to filter groups based on aggregate conditions.

10. Demonstrate proficiency in handling NULL values using IS NULL and IS NOT NULL operators.
