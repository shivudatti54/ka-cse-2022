# Learning Objectives

After studying this topic, you should students able to:

1. Write SQL INSERT statements to add single and multiple rows into database tables with proper syntax and handling of default values, null values, and expressions.

2. Construct UPDATE statements to modify existing data in one or more columns, including the use of arithmetic expressions and conditional updates using CASE expressions.

3. Formulate DELETE statements with appropriate WHERE conditions to remove specific rows or all rows from a table while understanding the implications of omitting conditions.

4. Explain the differences between DELETE and TRUNCATE commands and identify scenarios where each is appropriate.

5. Implement transaction control using COMMIT and ROLLBACK statements to ensure data integrity and atomicity of database operations.

6. Apply subqueries within INSERT, UPDATE, and DELETE statements for advanced data manipulation from related tables.

7. Analyze the impact of DML operations on database tables and determine the appropriate statement to use based on specific requirements.
