This chapter covers the fundamental concepts of computer instructions and instruction sequencing. Students learn about instruction components (opcode, operands), binary encoding formats (R-type, I-type, J-type), various instruction types (data transfer, arithmetic, logic, control), and the complete execution cycle (fetch-decode-execute-writeback). The instruction sequencing portion explains sequential execution, branch mechanisms, subroutine calls, and PC modification. Key comparisons between RISC and CISC architectures, along with the performance equation T = N × CPI × τ, enable quantitative analysis. The chapter includes analytical problems on encoding, execution time calculation, branch addressing, and pipeline hazards, along with multiple choice questions testing application-level understanding.
