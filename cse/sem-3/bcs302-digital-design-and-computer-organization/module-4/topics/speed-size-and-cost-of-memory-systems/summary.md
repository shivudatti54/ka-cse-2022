This chapter established the fundamental speed-size-cost trade-off in memory system design, explaining why no single memory technology can simultaneously satisfy all three requirements. Through quantitative analysis, we demonstrated that SRAM's six-transistor cell provides superior access speed (0.5–2 ns) but at 1000× higher cost-per-bit than DRAM, while DRAM's one-transistor-plus-capacitor cell achieves higher density but requires periodic refresh. The memory hierarchy leverages the principle of locality (temporal and spatial) to approximate ideal memory characteristics: the Average Memory Access Time (AMAT) formula quantifies the performance benefit, showing how a 1 MB cache can achieve 1.9 ns effective access time against 100 ns main memory latency. Cost analysis proves the economic viability of hierarchies: with cache-to-main-memory size ratios of 10⁻⁴ and cost ratios of 10³, the effective cost approaches main memory pricing while delivering cache-level performance. These principles directly enable subsequent study of cache design, virtual memory, and system optimization.
