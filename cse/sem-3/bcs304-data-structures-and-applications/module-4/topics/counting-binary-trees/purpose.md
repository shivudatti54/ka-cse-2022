# Learning Objectives

After studying this topic, you should be able to:

1. Define and distinguish between different types of binary trees including rooted binary trees, full binary trees, complete binary trees, and binary search trees.

2. Explain the concept of Catalan numbers and their relationship to counting binary trees.

3. Apply the formula C(n) = (2n)! / [(n+1)! × n!] to calculate the number of binary trees for a given number of nodes.

4. Use the recurrence relation T(n) = Σ(i=1 to n) T(i-1) × T(n-i) to compute the number of binary trees iteratively.

5. Calculate the number of binary search trees (BST) that can be formed with n distinct keys.

6. Determine the number of full binary trees given the number of internal nodes.

7. Analyze the relationship between the number of nodes and the corresponding tree configurations in various tree types.

8. Apply these counting concepts to solve practical problems related to tree data structures in computing applications.
