# Learning Objectives

After studying this topic, you should be able to:

1. **Define** the adjacency matrix and adjacency list representations of a graph.
2. **Identify** the space complexity of adjacency matrix and adjacency list representations.
3. **Explain** the trade-offs between adjacency matrix and adjacency list representations in terms of time and space complexity.
4. **Construct** an adjacency matrix and adjacency list for a given graph.
5. **Compare** the suitability of adjacency matrix and adjacency list representations for dense and sparse graphs.
6. **Analyze** the time complexity of common graph operations (e.g., edge lookup, neighbor iteration) in both representations.
7. **Justify** the choice of representation for a given graph-based problem or algorithm.
8. **Design** a graph representation (adjacency matrix or adjacency list) for a specific problem or algorithm, considering the trade-offs between time and space complexity.
