# Learning Objectives

After studying this topic, you should be able to:

1. **Define** a Minimum Cost Spanning Tree (MST) and explain its significance in network design.
2. **List** the key properties of an MST, including the number of edges, connectedness, and acyclicity.
3. **Identify** the differences between Kruskal's and Prim's algorithms for finding an MST.
4. **Apply** Kruskal's algorithm to find an MST in a given weighted graph using Union-Find for cycle detection.
5. **Apply** Prim's algorithm to find an MST in a given weighted graph using a priority queue or adjacency matrix.
6. **Analyze** the time complexities of Kruskal's and Prim's algorithms and determine which is more suitable for sparse or dense graphs.
7. **Compare** the advantages and disadvantages of Kruskal's and Prim's algorithms in different scenarios.
8. **Design** an MST-based solution for a real-world problem, such as network design or cluster analysis.
