# Learning Objectives

After studying this topic, you should be able to:

1. **Define** the optimal binary search tree problem and explain its significance in data structure design.

2. **Identify** the components of OBST including keys, dummy keys, and their associated probabilities.

3. **Formulate** the dynamic programming recurrence relation for computing optimal search cost.

4. **Calculate** weights w[i][j] and expected search costs e[i][j] for given probability distributions.

5. **Apply** the OBST algorithm to construct optimal binary search trees for small instances.

6. **Analyze** the time and space complexity of the optimal BST algorithm.

7. **Compare** optimal binary search trees with other tree construction methods and explain when OBST is preferred.

8. **Evaluate** the correctness of a constructed optimal BST by verifying search costs.
