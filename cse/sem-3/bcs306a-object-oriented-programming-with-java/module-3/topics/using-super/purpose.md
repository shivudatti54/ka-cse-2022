# Learning Objectives

After studying this topic, you should students able to:

1. Define the purpose of the `super` keyword in Java and its role in inheritance.
2. Identify the two primary uses of the `super` keyword: calling a superclass constructor and accessing superclass members.
3. Explain how to use `super()` to explicitly invoke a parent class constructor from a subclass constructor.
4. Describe a scenario where using `super` to access an overridden method or hidden field from the parent class is necessary.
5. Implement the `super` keyword in a subclass to call a superclass constructor and access superclass members.
6. Compare the benefits of using `super` to call a superclass constructor versus not using it.
7. Justify the importance of using `super` in maintaining proper object initialization protocols and preventing code duplication.
8. Design a class hierarchy that utilizes the `super` keyword to demonstrate its practical application in Java programming.
