# Learning Objectives

After studying this topic, you should students able to:

1. Understand the syntax and behavior of the simple assignment operator (=) in Java.

2. Identify and correctly use all compound assignment operators (+=, -=, \*=, /=, %=, etc.).

3. Explain the difference between value assignment for primitives and reference assignment for objects.

4. Implement chained assignment statements in Java programs.

5. Apply appropriate type casting when assigning values between different data types.

6. Analyze operator precedence and associativity rules for assignment operators.

7. Write Java code demonstrating proper use of assignment operators in OOP contexts.
