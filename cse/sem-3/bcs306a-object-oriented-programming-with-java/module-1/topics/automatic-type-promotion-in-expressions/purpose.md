# Learning Objectives

After studying this topic, you should students able to:

1. Explain the concept of automatic type promotion in Java and understand why it is necessary for type safety in expressions.

2. Apply the type promotion hierarchy rules (byte → short → int → long → float → double) to determine the resulting data type of arithmetic expressions.

3. Identify when type promotion occurs in arithmetic, relational, and logical expressions involving different data types.

4. Analyze and predict the output of Java code containing mixed data type operations.

5. Distinguish between integer division and real division based on operand types.

6. Understand the role of character type promotion and its relationship with integer types in expressions.

7. Recognize common programming errors related to type promotion and explain how explicit type casting can resolve them.
