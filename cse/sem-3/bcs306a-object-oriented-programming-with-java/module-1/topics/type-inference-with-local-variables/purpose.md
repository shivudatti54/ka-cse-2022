# Learning Objectives

After studying this topic, you should students able to:

1. Explain the concept of type inference in Java and how the `var` keyword enables it for local variables.

2. Identify the rules and restrictions governing the use of `var` in Java programs.

3. Distinguish between explicitly typed variables and variables declared with type inference.

4. Apply `var` correctly when working with collections, streams, and custom objects.

5. Analyze compilation behavior when using `var` with various initializer expressions.

6. Recognize the limitations of `var` and understand scenarios where it should not students used.

7. Compare type inference in Java with similar features in other programming languages.

8. Evaluate when to use `var` for improved code readability versus when to use explicit type declarations.
