# Variables in Java

## Overview

In Java, a variable is a name given to a memory location that stores a value. Variables are used to store and manipulate data in a program. Java has different types of variables, including primitive types, reference types, local variables, instance variables, and static variables.

## Key Points

- Java has eight primitive types: byte, short, int, long, float, double, char, and boolean.
- Reference types are objects that are instances of classes, interfaces, or arrays.
- Local variables are declared inside a method or block and are only accessible within that scope.
- Instance variables are declared inside a class and are associated with an instance of that class.
- Static variables are declared inside a class and are shared by all instances of that class.
- Variables can be declared and initialized using the `type variableName` and `type variableName = value` syntax.
- Java automatically promotes a smaller type to a larger type when necessary.
- Explicit casting is used to convert a larger type to a smaller type.

## Important Definitions

- **Primitive Type**: A basic data type in Java, such as byte, short, int, long, float, double, char, and boolean.
- **Reference Type**: An object that is an instance of a class, interface, or array.
- **Local Variable**: A variable declared inside a method or block and only accessible within that scope.
- **Instance Variable**: A variable declared inside a class and associated with an instance of that class.
- **Static Variable**: A variable declared inside a class and shared by all instances of that class.

## Key Syntax

- Declaring a variable: `type variableName;`
- Initializing a variable: `type variableName = value;`
- Automatic type promotion: `int x = 10; double y = x;`
- Explicit casting: `double x = 10.5; int y = (int) x;`

## Exam Tips

- Be able to identify and explain the different types of variables in Java.
- Understand how to declare and initialize variables.
- Know how to use automatic type promotion and explicit casting.
- Be able to write code that demonstrates the use of variables in Java.
