# Declaring Objects

## Overview

In Java, objects are declared using a specific syntax that includes the class name and a reference variable. This declaration is essential for creating objects that can be used in a Java program. A declared object can be initialized and used to access the class's members.

## Key Points

- A Java object is declared using the `ClassName referenceVariable;` syntax.
- The `ClassName` should be a valid Java class name.
- The `referenceVariable` is the name given to the object.
- The declaration of an object does not create an object in memory.
- An object declaration is a prerequisite for object creation.

## Important Definitions

- **Object Declaration**: The process of declaring an object of a class.
- **Reference Variable**: A variable that holds the memory address of an object.

## Key Formulas / Syntax

- `java ClassName referenceVariable;`

## Exam Tips

- Pay attention to the syntax used for declaring objects in Java.
- Understand the difference between object declaration and object creation.
