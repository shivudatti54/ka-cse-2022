# Learning Objectives

After studying this topic, you should students able to:

1. Define the purpose of declaring objects in Java programming.
2. Recall the basic syntax for declaring objects using reference variables.
3. Explain the difference between declaring a reference variable and creating an object.
4. Describe the process of instantiating objects using the `new` keyword and a class constructor.
5. Demonstrate how to declare and initialize objects of a given class.
6. Compare the roles of reference variables and objects in a Java program.
7. Justify the importance of declaring objects in the context of Object-Oriented Programming (OOP) principles.
8. Design a simple Java program that creates and uses objects of a custom class.
