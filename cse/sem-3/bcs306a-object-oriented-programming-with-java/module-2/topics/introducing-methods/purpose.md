# Learning Objectives

After studying this topic, you should students able to:

1. Define the basic structure of a method in Java, including access modifiers, return types, method names, and parameter lists.
2. Identify the purpose of access modifiers and non-access modifiers in method declarations.
3. Explain the difference between `void` and value-returning methods, and describe the use of the `return` keyword in each case.
4. Implement a simple method with parameters and return values, and demonstrate its invocation and usage.
5. Distinguish between pass-by-value and pass-by-reference in method argument passing, and examine the implications of each approach.
6. Justify the use of methods in achieving encapsulation and code reusability in object-oriented programming.
7. Design and develop a simple Java program that uses multiple methods to solve a real-world problem, such as calculating the area and perimeter of a rectangle.
8. Assess the benefits and trade-offs of using methods in Java programming, including code organization, maintainability, and performance.
