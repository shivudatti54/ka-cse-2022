# Learning Objectives

After studying this topic, you should students able to:

1. List the different types of built-in exceptions in Java, including checked and unchecked exceptions.
2. Identify the common unchecked exceptions (RuntimeException) in Java, such as ArithmeticException, NullPointerException, and ArrayIndexOutOfBoundsException.
3. Explain the difference between checked and unchecked exceptions, including the Java compiler's requirement to handle them.
4. Describe the exception hierarchy in Java, including the Throwable class and its subclasses Error and Exception.
5. Classify exceptions into checked and unchecked categories, providing examples of each.
6. Use try-catch-finally blocks and the throws keyword to manage predefined exceptions effectively.
7. Analyze the importance of handling exceptions in real-world applications, such as file operations, network communication, and user input parsing.
8. Justify the use of specific exception handling mechanisms in a given scenario, considering factors such as program stability, security, and user experience.
