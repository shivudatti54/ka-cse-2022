# Learning Objectives

After studying this topic, you should be able to:

1.  Explain the purpose and outcome of the primary process operations: creation, termination, waiting, and execution.
2.  Differentiate between the `fork()` and `exec()` system calls and describe how they are used together.
3.  Analyze simple code snippets involving `fork()`, `exec()`, and `wait()` to predict the output and process behavior.
4.  Describe the problems of zombie and orphan processes and explain how the OS handles them.
5.  Compare and contrast the two main models of Interprocess Communication (IPC): shared memory and message passing.
6.  Relate process operations to transitions in the process state model (e.g., ready, running, waiting, terminated).
