# Learning Objectives

After studying this topic, you should be able to:

1. Explain the concept of paging and how it solves the problem of external fragmentation.
2. Differentiate between pages, frames, and page tables.
3. Perform logical to physical address translation using a given page table.
4. Describe the structure and purpose of a Page Table Entry (PTE) and its various bits (present, protection, modified, etc.).
5. Explain the role of the Translation Lookaside Buffer (TLB) and calculate effective memory access time.
6. Compare and contrast different page table structures (simple, hierarchical, hashed, inverted) and understand why complex structures are needed for large address spaces.
