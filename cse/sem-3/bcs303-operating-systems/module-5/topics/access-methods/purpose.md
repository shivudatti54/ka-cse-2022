# Learning Objectives

After studying this topic, you should be able to:

1. Define what a file is in the context of an operating system and list its essential attributes.
2. Explain the purpose and process of common file operations (create, read, write, delete, seek).
3. Differentiate between the three fundamental file access methods: Sequential, Direct, and Indexed Sequential.
4. Describe the operational characteristics, advantages, and disadvantages of each access method.
5. Identify real-world scenarios or devices where each access method is most appropriately used.
6. Understand the role of an index in the ISAM method and how it combines sequential and direct access.
