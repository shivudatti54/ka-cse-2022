# Learning Objectives

After studying this topic, you should be able to:

1. Understand and explain the three access specifiers (public, protected, private) in C++ and their effects on member visibility.

2. Differentiate between public, protected, and private inheritance and describe how each affects the accessibility of base class members in derived classes.

3. Analyze and determine the access level of inherited members given the base class member type and inheritance type.

4. Apply the 'using' declaration to modify access levels of inherited members in derived classes.

5. Design class hierarchies with appropriate access control for encapsulation and data hiding.

6. Understand the order of constructor and destructor calls in inheritance hierarchies.

7. Handle multiple inheritance scenarios with different access specifiers for each base class.

8. Apply these concepts to solve university examination problems related to base class access control.
