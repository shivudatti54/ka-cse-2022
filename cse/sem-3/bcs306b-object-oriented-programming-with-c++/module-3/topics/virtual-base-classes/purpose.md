# Learning Objectives

After studying this topic, you should be able to:

1. Explain the diamond problem in multiple inheritance and understand why it occurs in C++ without virtual base classes.

2. Implement virtual inheritance in C++ using the `virtual` keyword in class inheritance declarations.

3. Analyze the memory layout implications of virtual base classes and explain how they eliminate duplicate base class instances.

4. Demonstrate understanding of constructor invocation order in virtual inheritance hierarchies.

5. Differentiate between virtual and non-virtual inheritance and identify scenarios where each is appropriate.

6. Design class hierarchies that properly use virtual base classes to avoid ambiguity in member access.

7. Resolve access ambiguities that may still arise in virtual inheritance scenarios using scope resolution operators.

8. Apply knowledge of virtual base classes to understand real-world C++ library implementations, particularly in the STL.
