# Learning Objectives

After studying this topic, you should be able to:

1. Understand the concept of friend functions and their role in C++ object-oriented programming

2. Declare and define friend functions within a class and explain their access privileges

3. Distinguish between member functions and friend functions based on their characteristics and usage

4. Implement friend functions for operator overloading, particularly for binary operators and stream operators

5. Design and implement friend classes when multiple classes need shared access to private members

6. Analyze when to use friend functions appropriately versus when to prefer encapsulation-friendly alternatives

7. Apply the concept of forward declaration when working with friend functions across multiple classes
