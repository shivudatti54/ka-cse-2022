# Learning Objectives

After studying this topic, you should be able to:

1. Understand the concept of friend functions and their role in C++ object-oriented programming
2. Explain the advantages of using friend functions for operator overloading compared to member functions
3. Implement binary arithmetic operators (+, -, \*, /) using friend functions with proper syntax
4. Implement comparison operators (==, !=, <, >, <=, >=) using friend functions
5. Implement unary operators (-, ++, --) using friend functions
6. Handle commutative operations where the left operand is not a class object (e.g., int + Complex)
7. Apply const correctness in friend function operator overloading
8. Analyze when to choose friend functions versus member functions for operator overloading
