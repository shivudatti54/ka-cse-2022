# Learning Objectives

After studying this topic, you should be able to:

1. Explain the C++ stream class hierarchy and the role of ios_base, istream, and ostream classes in formatted I/O operations.

2. Identify and use various format flags (boolalpha, hex, oct, dec, scientific, fixed, showbase, uppercase, etc.) to control output formatting.

3. Apply stream manipulators (both no-argument and parameterized) to format console output for integers, floating-point numbers, and strings.

4. Distinguish between sticky manipulators (like precision, alignment) and non-sticky manipulators (like setw) and use them appropriately.

5. Write C++ programs that display data in tabular format using field width, alignment, and fill character settings.

6. Create user-defined manipulators for custom output formatting requirements.

7. Demonstrate proper error handling in formatted input operations by checking stream states.
