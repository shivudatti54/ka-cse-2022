# Learning Objectives

After studying this topic, you should be able to:

1. Explain the C++ standard exception class hierarchy and its importance in exception handling.

2. Create custom exception classes by deriving from standard exception classes.

3. Demonstrate proper exception handling by catching exceptions by reference to avoid object slicing.

4. Analyze how virtual functions behave when exceptions are caught through base class references.

5. Implement multi-level exception handling with proper propagation and rethrowing.

6. Design effective exception hierarchies for application-specific error handling scenarios.

7. Apply best practices for exception specifications and noexcept qualifiers.

8. Distinguish between different exception types in the standard library and select appropriate base classes for custom exceptions.
