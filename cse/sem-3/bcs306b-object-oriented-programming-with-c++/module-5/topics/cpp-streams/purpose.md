# Learning Objectives

After studying this topic, you should be able to:

1. Explain the C++ stream class hierarchy and the role of each major class (ios_base, ios, istream, ostream, iostream)

2. Identify the purpose and usage of standard stream objects (cin, cout, cerr, clog) in C++ programs

3. Demonstrate proficiency in file handling operations using ifstream, ofstream, and fstream classes

4. Apply appropriate file opening modes (ios::in, ios::out, ios::app, ios::binary, etc.) for different file operations

5. Utilize string streams (ostringstream, istringstream, stringstream) for type conversion and string parsing

6. Use stream manipulators (endl, setw, setprecision, fixed, hex, etc.) to control output formatting

7. Implement proper error handling for stream operations by checking stream state flags

8. Distinguish between formatted and unformatted I/O operations and choose appropriate methods for specific tasks
