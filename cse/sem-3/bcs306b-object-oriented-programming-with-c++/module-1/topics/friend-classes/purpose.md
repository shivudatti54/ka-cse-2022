# Learning Objectives

After studying this topic, you should be able to:

1. Explain the concept of friend classes and friend functions in C++ and their role in controlled data access.

2. Demonstrate the syntax for declaring friend classes and friend functions within a C++ program.

3. Analyze the properties of friendship in C++, including unidirectionality, non-transitivity, and non-symmetry.

4. Differentiate between friend classes, friend member functions, and friend global functions with appropriate use cases.

5. Design and implement programs that appropriately utilize friend classes to solve practical problems.

6. Evaluate the trade-offs between using friend classes and traditional accessor/mutator methods in terms of encapsulation and code design.

7. Apply the concept of friend functions in operator overloading scenarios involving multiple classes.
