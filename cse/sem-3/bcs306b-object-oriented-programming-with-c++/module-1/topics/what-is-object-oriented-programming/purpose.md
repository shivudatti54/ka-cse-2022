# Learning Objectives

After studying this topic, you should be able to:

1. Define Object-Oriented Programming and explain its fundamental characteristics

2. Distinguish between a class and an object with appropriate examples

3. Explain the four pillars of OOP: Encapsulation, Inheritance, Polymorphism, and Abstraction with practical implementations

4. Compare Object-Oriented Programming with Procedural Programming approaches

5. Identify and apply access specifiers (private, public, protected) in C++ programs

6. Demonstrate inheritance concepts including single, multiple, and multilevel inheritance with C++ code

7. Differentiate between compile-time (function overloading) and runtime (virtual functions) polymorphism

8. Analyze real-world scenarios and model them using OOP principles
