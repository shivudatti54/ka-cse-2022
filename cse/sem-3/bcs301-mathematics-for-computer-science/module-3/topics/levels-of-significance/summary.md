# Levels Of Significance - Summary

## Key Definitions

- **Level of Significance (α)**: The probability of rejecting the null hypothesis when it is actually true (Type I error)
- **Type I Error**: Rejecting H₀ when it is true; probability = α
- **Type II Error**: Failing to reject H₀ when it is false; probability = β
- **Critical Region**: The set of sample values that lead to rejection of H₀
- **Critical Values**: Boundary values that define the critical region
- **P-value**: The probability of obtaining results at least as extreme as observed, assuming H₀ is true
- **Power of Test**: 1 - β, the probability of correctly rejecting a false H₀

## Important Formulas

- Decision Rule (Critical Value Method): Reject H₀ if test statistic falls in critical region
- Decision Rule (P-value Method): Reject H₀ if p-value ≤ α
- Confidence Level = 1 - α
- For two-tailed test: Critical values at ±z_{α/2}
- For one-tailed test: Critical value at ±z_α

## Key Points

1. The significance level α is predetermined before collecting data, typically 0.05, 0.01, or 0.10
2. A lower α reduces Type I error but increases Type II error
3. The choice of α depends on the consequences of making a Type I error
4. Two-tailed tests split α between both tails; one-tailed tests place all α in one tail
5. The p-value provides the exact probability of observing the data given H₀ is true
6. Common significance levels: 0.05 (standard research), 0.01 (confirmatory), 0.10 (exploratory)
7. Statistical significance does not imply practical significance; always consider effect size

## Common Mistakes

1. **Confusing p-value with Type I error**: The p-value is not α; it is the probability of the data given H₀, while α is the threshold set before the test.

2. **Using data to choose test type**: The decision for one-tailed vs two-tailed test must be made before seeing the data based on the research question.

3. **Interpreting "fail to reject" as "accept"**: Failing to reject H₀ does not prove H₀ is true; it only means insufficient evidence to reject it.

4. **Ignoring Type II errors**: Students often focus only on α without considering the implications of β and test power.

5. **Changing α after seeing results**: The significance level must be fixed before the test; modifying it based on data is invalid.
