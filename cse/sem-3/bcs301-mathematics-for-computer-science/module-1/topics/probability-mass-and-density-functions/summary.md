# Probability Mass and Density Functions

## Key Points

- **Probability Mass Function (PMF)**:
  - For discrete random variable \(X\), PMF \(p_X(x)\) gives probability of \(X\) taking value \(x\).
  - Formula: \(p_X(x) = P(X=x)\)
- **Discrete Random Variables**:
  - Take on a countable number of values.
  - Examples include coin flips, dice rolls.

- **Probability Density Function (PDF)**:
  - For continuous random variable \(Y\), PDF \(f_Y(y)\) gives the relative likelihood that \(Y\) takes a value near \(y\).
  - Formula: \(f_Y(y) = P(a < Y \leq b)\) where \(a < y < b\)
- **Continuous Random Variables**:
  - Can take any value within an interval.
  - Examples include height, weight.

## Important Formulas/Definitions

- **Probability Mass Function (PMF)**:
  - For discrete random variable \(X\) with possible values \(\{x_1, x_2, ..., x_n\}\):
    \[
    p_X(x_i) = P(X = x_i)
    \]
- **Probability Density Function (PDF)**:
  - For continuous random variable \(Y\) with probability density function \(f_Y(y)\):
    \[
    f_Y(y) dy = P(a < Y \leq b)
    \]

## Important Theorems

- **Total Probability Rule**:
  - Sum of probabilities for all possible outcomes equals one.
  - For discrete random variable \(X\), if \(\{x*i\}\) are the values it can take, then:
    \[
    \sum*{i} p_X(x_i) = 1
    \]

- **Law of Total Probability**:
  - Summing over all possible events (discrete or continuous).
