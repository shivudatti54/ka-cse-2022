### Learning Purpose: Review of Basic Probability Theory & Random Variables

1.  **Why is this topic important?**
    Probability is the fundamental language of uncertainty, which is central to computer science. It is the backbone of algorithms (e.g., randomized quicksort), machine learning models, network analysis, and performance evaluation of systems. A strong grasp of these basics is non-negotiable for advanced study.

2.  **What will students learn?**
    Students will solidify their understanding of core axioms, rules, and theorems of probability (e.g., Bayes' Theorem). They will formally define discrete random variables, learn to describe them using Probability Mass Functions (PMFs), and calculate key properties like expected value and variance.

3.  **How does it connect to other concepts?**
    This review provides the essential vocabulary and tools for the entire module. It is the direct prerequisite for studying specific discrete distributions (e.g., Binomial, Geometric) next and forms the conceptual bridge to continuous random variables and distributions covered later.

4.  **Real-world applications**
    These concepts are applied everywhere: calculating error rates in data transmission, predicting the load on a web server, designing A/B tests, building spam filters using Naive Bayes classifiers, and analyzing the runtime of probabilistic algorithms.
