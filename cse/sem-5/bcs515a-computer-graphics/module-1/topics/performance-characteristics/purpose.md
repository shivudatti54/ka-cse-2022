# Learning Objectives

After studying this topic, you should be able to:

1. Define and explain the key performance characteristics in computer graphics including frame rate, refresh rate, resolution, color depth, fill rate, and memory bandwidth.

2. Calculate frame rate given rendering time and vice versa, understanding the relationship between frame time and achievable visual complexity.

3. Analyze how resolution and color depth affect memory requirements and computational workload in graphics systems.

4. Determine fill rate requirements for various rendering scenarios including anti-aliased rendering.

5. Evaluate memory bandwidth requirements for different display configurations and understand why bandwidth often limits graphics performance.

6. Identify latency as a critical factor in interactive graphics applications and explain its components.

7. Compare performance characteristics across different graphics applications and determine appropriate targets for various use cases.
