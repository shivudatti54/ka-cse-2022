# Learning Objectives

After studying this topic, you should be able to:

1. Explain the evolution from fixed-function to programmable graphics pipelines and understand the advantages of programmable pipelines.

2. Describe the architecture of the modern programmable graphics pipeline, including all stages from input assembly to output merger.

3. Write basic vertex shaders in GLSL/HLSL that perform coordinate transformations and pass data to fragment shaders.

4. Write basic fragment shaders in GLSL/HLSL that implement per-pixel lighting and texture sampling.

5. Understand the differences between vertex shaders, fragment shaders, geometry shaders, and their respective roles in the rendering pipeline.

6. Apply knowledge of programmable pipelines to implement common rendering effects such as Phong lighting and texture mapping.

7. Analyze graphics programs to identify which shader stage performs specific operations and understand data flow between stages.

8. Design shader-based solutions for practical graphics programming problems, demonstrating competency in shader development for real-time rendering.
