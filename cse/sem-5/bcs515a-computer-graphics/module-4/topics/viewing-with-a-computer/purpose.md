# Learning Objectives

After studying this topic, you should be able to:

1. Explain the complete viewing pipeline in computer graphics and describe the purpose of each transformation stage.

2. Define and apply window-to-viewport transformation to convert world coordinates to screen coordinates with appropriate scaling and translation.

3. Implement various line clipping algorithms, particularly Cohen-Sutherland and Liang-Barsky, to determine visibility of line segments.

4. Distinguish between parallel and perspective projection methods and explain their mathematical foundations.

5. Describe the camera model parameters including View Reference Point, View Plane Normal, View Up Vector, and Projection Reference Point.

6. Apply perspective projection equations to transform 3D coordinates into 2D projected coordinates.

7. Analyze how clipping operations work in conjunction with viewing transformations to optimize rendering.
