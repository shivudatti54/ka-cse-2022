# Learning Objectives

After studying this topic, you should be able to:

1. Identify and explain the different types of light sources used in computer graphics, including ambient, point, directional, spot, and area lights.

2. Understand the mathematical models used to represent light propagation and intensity attenuation in virtual 3D environments.

3. Apply the Phong illumination model to calculate surface illumination from multiple light sources.

4. Compute diffuse reflection using vector mathematics, specifically the dot product between surface normals and light direction vectors.

5. Analyze how light color components interact with surface material properties to produce final rendered colors.

6. Design effective lighting setups for 3D scenes by strategically placing different types of light sources.

7. Evaluate the trade-offs between computational complexity and visual realism when choosing lighting models for real-time versus offline rendering.
