# Learning Objectives

After studying this topic, you should be able to:

1. Explain the concept of homogeneous coordinates and understand why they are essential for computer graphics transformations.

2. Write the transformation matrix for translation, scaling, rotation, and shearing in 2D using homogeneous coordinates.

3. Apply transformation matrices to convert points from one position to another through translation, scaling, and rotation operations.

4. Construct composite transformation matrices by multiplying individual transformation matrices in the correct order.

5. Perform rotation and scaling about arbitrary points (not just the origin) using the translate-rotate-translate approach.

6. Extend the understanding to 3D transformation matrices including rotations about x, y, and z axes.

7. Analyze transformation sequences to determine the final position of geometric objects after multiple transformations.

8. Apply these concepts to solve numerical problems typically asked in examinations.
