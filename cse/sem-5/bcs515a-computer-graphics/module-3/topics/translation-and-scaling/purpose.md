# Learning Objectives

After studying this topic, you should be able to:

1. Explain the concept of homogeneous coordinates and their importance in representing 2D transformations as matrix operations.

2. Apply translation transformation to move points and objects from one position to another using both algebraic and matrix methods.

3. Perform uniform and non-uniform scaling transformations on 2D objects, understanding the effects of different scale factors.

4. Execute scaling about a fixed point (pivot point) using the translate-scale-translate sequence of operations.

5. Construct composite transformation matrices by combining multiple translation and scaling operations in the correct order.

6. Analyze and solve numerical problems involving translation and scaling of geometric objects including points, lines, polygons, and curves.

7. Apply transformation concepts to practical computer graphics scenarios such as windowing, viewport mapping, and object manipulation.
