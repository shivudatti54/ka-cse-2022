# Learning Objectives

After studying this topic, you should be able to:

1. Explain the fundamental problem of converting continuous mathematical lines into discrete pixel representations on a raster display.

2. Describe the DDA (Digital Differential Analyzer) algorithm and demonstrate its step-by-step working with numerical examples.

3. Implement Bresenham's Line Algorithm and explain why it is more efficient than the DDA algorithm in terms of computational complexity.

4. Compare and contrast the DDA, Bresenham's, and Midpoint line drawing algorithms with respect to their computational requirements and accuracy.

5. Apply Bresenham's algorithm to draw lines with positive slope, negative slope, shallow angles, and steep angles.

6. Analyze the decision parameter formulation in Bresenham's algorithm and understand how it determines pixel selection at each step.

7. Evaluate which line drawing algorithm is most suitable for different practical applications in computer graphics.
