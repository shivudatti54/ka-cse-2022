## Learning Objectives (Bloom's Taxonomy)

- **Remember**: Define the four major stages of the graphics pipeline and their sequential order
- **Understand**: Explain how geometric representations (polygons, NURBS, subdivision surfaces) are created and transformed through each pipeline stage
- **Apply**: Implement model, view, and projection transformations using OpenGL/graphics API matrix operations
- **Apply**: Use rasterization algorithms (Bresenham's line drawing, triangle scan conversion) to convert primitives to fragments
- **Analyze**: Compare antialiasing techniques (MSAA, FXAA, TAA) based on quality, performance, and use cases
- **Analyze**: Evaluate fragment processing operations including texturing, blending, depth testing, and their effects on final image quality
- **Create**: Design a simple 3D rendering pipeline that demonstrates all four major tasks working together
- **Evaluate**: Assess the importance of each pipeline stage in producing correct visual output and identify potential issues at each stage
