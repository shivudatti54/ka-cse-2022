# Input Devices - Summary

## Key Definitions and Concepts

- **Input Devices:** Hardware components that convert user data and commands into machine-readable form for processing by the computer
- **Pointing Devices:** Input devices that control cursor position on screen, including mouse, trackball, touchpad, and joystick
- **Scanning Devices:** Convert physical documents and images into digital format through optical scanning
- **OCR (Optical Character Recognition):** Technology converting scanned text images into editable machine-readable text
- **OMR (Optical Mark Recognition):** Reads pre-defined mark positions on forms for surveys and tests
- **Biometric Devices:** Use physiological or behavioral characteristics (fingerprint, iris, face, voice) for identification
- **Touch Screen:** Display/input hybrid allowing direct interaction through touch gestures

## Important Formulas and Theorems

No specific formulas apply to this topic as it is hardware-focused. However, key specifications to remember:

- Standard keyboard: 104-108 keys
- Common keyboard interfaces: USB, Bluetooth, PS/2
- Mouse resolution measured in DPI (dots per inch)—higher DPI means greater sensitivity

## Key Points

- Keyboards remain the primary text input device despite advances in voice and touch input
- Optical mice use LED/laser technology for movement detection, replacing older ball-based designs
- Barcode scanners use laser reflection or camera imaging to read product codes
- Touch screens combine input and output, eliminating separate pointing devices
- Biometric authentication provides higher security than password-based systems
- Graphic tablets enable precise digital drawing for design applications
- Motion sensors (accelerometers, gyroscopes) enable gesture-based interaction
- Input device selection depends on application requirements, accuracy needs, and user preferences

## Common Mistakes to Avoid

- Confusing OCR with OMR—OCR reads text characters while OMR reads marked bubbles/squares
- Thinking all touch screens work the same—resistive requires pressure while capacitive requires conductive touch
- Believing newer devices completely replace older ones—keyboards remain essential for text input
- Overlooking connectivity options—wireless vs. wired affects latency and convenience
