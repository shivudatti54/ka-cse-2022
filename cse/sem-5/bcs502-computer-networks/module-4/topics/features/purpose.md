# Learning Objectives

After studying this topic, you should be able to:

1. Explain the fundamental features of TCP that distinguish it from other transport protocols like UDP, including reliability, connection orientation, and flow control mechanisms.

2. Describe the three-way handshake process used for TCP connection establishment, including the significance of initial sequence numbers and option negotiation.

3. Analyze how TCP ensures reliable data delivery through sequence numbers, acknowledgments, checksums, and retransmission mechanisms.

4. Compare and contrast flow control and congestion control in TCP, explaining how each mechanism operates to optimize network performance.

5. Identify and explain the fields in the TCP header, understanding their roles in connection management and data transmission.

6. Understand the byte-stream service model of TCP and its implications for application development and data transfer.

7. Explain the role of port numbers and socket pairs in TCP communication, including the distinction between well-known, registered, and dynamic ports.

8. Evaluate the importance of TCP features in designing network applications and understanding internet communication protocols.
