# Learning Objectives

After studying this topic, you should be able to:

1. Differentiate between single-bit errors and burst errors in data transmission.

2. Explain the working of parity check, checksum, and CRC error detection techniques.

3. Apply the CRC polynomial division method to generate and verify CRC codes.

4. Design Hamming codes for single-bit error detection and correction.

5. Calculate the number of redundant bits required for a given data word size in Hamming code.

6. Compare and contrast Stop-and-Wait, Go-Back-N, and Selective Repeat ARQ protocols.

7. Analyze the efficiency of different ARQ protocols under various channel conditions.

8. Apply error control mechanisms to solve practical problems in data communication.
