# Learning Objectives

After studying this topic, you should be able to:

1. Explain the four-layer TCP/IP model and differentiate it from the seven-layer OSI model.

2. Identify and describe the function of key protocols at each TCP/IP layer, including IP, TCP, UDP, ICMP, HTTP, FTP, and DNS.

3. Analyze the differences between IPv4 and IPv6 addressing schemes and perform basic IP address calculations.

4. Compare and contrast TCP (reliable, connection-oriented) and UDP (unreliable, connectionless) protocols with respect to their appropriate applications.

5. Understand the TCP three-way handshake process for connection establishment and the four-way handshake for connection termination.

6. Apply subnetting concepts to divide networks and calculate valid host ranges for given CIDR blocks.

7. Explain the data encapsulation and de-encapsulation processes as data moves through TCP/IP layers.
