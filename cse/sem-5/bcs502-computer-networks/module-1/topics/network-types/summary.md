# Network Types - Summary

## Key Definitions and Concepts

- **PAN (Personal Area Network)**: Smallest network type connecting personal devices within 10 meters using Bluetooth or NFC
- **LAN (Local Area Network)**: High-speed privately owned network covering up to 1 km, using Ethernet or Wi-Fi
- **CAN (Campus Area Network)**: Interconnects multiple LANs across a campus area of 1-10 km
- **MAN (Metropolitan Area Network)**: Covers cities spanning 10-100 km using fiber optic technology
- **WAN (Wide Area Network)**: Global-scale network using public infrastructure, exemplified by the Internet

## Important Formulas and Relationships

- Coverage Order (Ascending): PAN < LAN < CAN < MAN < WAN
- Speed Order (Descending): LAN > CAN > MAN > WAN > PAN
- Cost per unit decreases as coverage area increases

## Key Points

1. Network type selection depends on geographical scope, number of users, bandwidth requirements, and budget constraints.

2. LAN offers highest speed (up to 10 Gbps) but minimal coverage; ideal for offices and homes.

3. CAN integrates multiple LANs within a campus under unified administration, typically using fiber optic backbones.

4. MAN serves metropolitan needs like city-wide internet access and cable television networks.

5. WAN enables global connectivity but introduces higher latency and lower speeds compared to LAN.

6. PAN enables personal device connectivity through Bluetooth and NFC technologies.

7. Modern networks often combine multiple network types - LAN connected via WAN for comprehensive coverage.

## Common Mistakes to Avoid

1. Confusing CAN with MAN - CAN is for campus (1-10 km), MAN is for city (10-100 km)

2. Assuming WAN always has lower speed than LAN - Modern fiber-based WANs can match LAN speeds

3. Neglecting security differences - PAN has higher security risks due to open wireless transmissions

4. Ignoring the management aspect - LAN is privately managed while WAN uses service provider infrastructure

5. Forgetting that networks can be hybrid - Most practical networks combine multiple network types
