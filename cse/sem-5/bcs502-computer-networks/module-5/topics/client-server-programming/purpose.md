# Learning Objectives

After studying this topic, you should be able to:

1. Explain the client-server model and its significance in network programming

2. Differentiate between TCP (stream) sockets and UDP (datagram) sockets and identify appropriate use cases for each

3. Implement a TCP server using Java's ServerSocket class with proper accept-listen-connect workflow

4. Implement a TCP client using Java's Socket class to connect to a server and exchange data

5. Utilize Java's InetAddress class to handle IP addresses and hostnames effectively

6. Design and implement a multi-threaded server capable of handling multiple simultaneous client connections

7. Implement UDP client-server communication using DatagramSocket and DatagramPacket classes

8. Apply proper resource management techniques including stream closure and exception handling in socket programs

9. Analyze and debug common issues in client-server applications such as connection failures and port conflicts
