# Learning Objectives

After studying this topic, you should be able to:

1. Explain the client-server architecture of FTP and describe the roles of control and data channels.

2. Distinguish between active and passive FTP modes, and identify which mode is appropriate for different network configurations.

3. Identify and describe at least six common FTP commands and their purposes in file transfer operations.

4. Interpret FTP response codes to determine the status of FTP operations and troubleshoot connection issues.

5. Compare ASCII and Binary transfer modes, and select the appropriate mode based on file types being transferred.

6. Differentiate between FTP, FTPS, and SFTP, understanding their security characteristics and use cases.

7. Explain the concept of anonymous FTP and its typical applications in modern networks.

8. Analyze FTP session traces to understand the sequence of commands and responses during file transfer operations.
