# Learning Objectives

After studying this topic, you should be able to:

1. Explain the concept of error control in data communication and distinguish between single-bit and burst errors.

2. Describe and implement simple parity checking and two-dimensional parity checking for error detection.

3. Understand the checksum method and explain how ones complement arithmetic is used in Internet checksum calculations.

4. Analyze the Cyclic Redundancy Check (CRC) technique, including polynomial representation and binary division for generating CRC codes.

5. Apply the Hamming code technique to encode data with appropriate redundant bits and decode received codewords to detect and correct errors.

6. Compare different error detection and correction techniques based on their efficiency, capability, and applications in real-world systems.

7. Evaluate the effectiveness of various error control methods in terms of types of errors they can detect and correct.

8. Solve numerical problems related to CRC generation, Hamming code encoding, and syndrome calculation for examination success.
