# Learning Objectives

After studying this topic, you should be able to:

1. Explain the HDLC protocol architecture and its three station configurations (Primary, Secondary, Combined)

2. Describe the complete HDLC frame structure including Flag, Address, Control, Information, and FCS fields

3. Differentiate between Information frames, Supervisory frames, and Unnumbered frames with their specific purposes

4. Apply bit stuffing and bit unstuffing techniques to ensure proper frame delimitation

5. Implement error detection using CRC (Cyclic Redundancy Check) with 16-bit and 32-bit FCS

6. Analyze HDLC operational modes (NRM, ARM, ABM) and their appropriate use cases

7. Evaluate flow control mechanisms using RR and RNR commands in HDLC

8. Compare HDLC with other Data Link Layer protocols and understand its practical applications
