# Learning Objectives

After studying this topic, you should be able to:

1. Define Path Vector Routing and explain its fundamental operation in inter-domain routing scenarios

2. Distinguish between Distance Vector, Link State, and Path Vector routing protocols with respect to their characteristics, advantages, and limitations

3. Explain the concept of Autonomous Systems (AS) and their role in Internet routing hierarchy

4. Describe BGP message types (OPEN, UPDATE, KEEPALIVE, NOTIFICATION) and their purposes in BGP session establishment and maintenance

5. Analyze BGP path attributes (AS-PATH, NEXT-HOP, LOCAL-PREF, MED, ORIGIN) and explain their roles in routing decisions

6. Explain the loop prevention mechanism in Path Vector Routing using AS-PATH attribute

7. Differentiate between eBGP and iBGP with respect to their operation, purpose, and key characteristics

8. Apply the BGP route selection process to determine which route would be chosen under given conditions

9. Evaluate the advantages and limitations of Path Vector Routing in large-scale network deployments
