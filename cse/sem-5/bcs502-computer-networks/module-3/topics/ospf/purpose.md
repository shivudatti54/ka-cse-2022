# Learning Objectives

After studying this topic, you should be able to:

1. Explain the working principles of OSPF as a link-state routing protocol and its advantages over distance-vector protocols like RIP.

2. Describe the hierarchical area structure of OSPF, including the role of Area 0, ABRs, ASBRs, and different area types (stub, totally stub, NSSA).

3. Identify and explain the five OSPF packet types and their functions in the neighbor discovery and routing update process.

4. Analyze different Link-State Advertisement (LSA) types and their specific roles in network topology description.

5. Calculate OSPF costs based on interface bandwidth and determine the best path selection using the cost metric.

6. Explain the OSPF adjacency state machine and the conditions required to reach the Full state.

7. Configure basic OSPF on Cisco routers and verify neighbor relationships using show commands.

8. Compare OSPF network types and explain the purpose of Designated Router (DR) and Backup DR (BDR) election in multi-access networks.
