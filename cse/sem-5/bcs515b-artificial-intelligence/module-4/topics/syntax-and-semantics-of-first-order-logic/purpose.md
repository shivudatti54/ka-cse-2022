# Learning Objectives

After studying this topic, you should be able to:

1. Identify and describe all components of the alphabet of First-Order Logic including logical connectives, quantifiers, variables, constants, functions, and predicates.

2. Construct valid terms and atomic formulas using the recursive definitions provided.

3. Determine whether a given string is a well-formed formula (WFF) in First-Order Logic.

4. Identify free and bound variables in formulas and correctly determine the scope of quantifiers.

5. Explain the structure of an interpretation including domain, constant assignments, function assignments, and predicate assignments.

6. Evaluate the truth value of formulas under given interpretations and variable assignments.

7. Distinguish between valid, satisfiable, and unsatisfiable formulas with appropriate examples.

8. Apply the logical equivalences involving quantifiers, particularly De Morgan's laws for quantifiers.
