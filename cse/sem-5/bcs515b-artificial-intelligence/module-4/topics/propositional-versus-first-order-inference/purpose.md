# Learning Objectives

After studying this topic, you should be able to:

1. Explain the fundamental differences between propositional logic and first-order logic in terms of expressive power

2. Identify the key limitations of propositional logic that necessitate first-order logic for real-world knowledge representation

3. Apply universal and existential instantiation rules to derive new facts from quantified statements

4. Perform unification on first-order atomic formulas and determine when unification succeeds or fails

5. Explain how resolution works in first-order logic and apply it to simple proof problems

6. Compare forward chaining and backward chaining inference mechanisms in both logical frameworks

7. Analyze the computational complexity differences between propositional and first-order inference

8. Design appropriate logical representations for simple domain problems using first-order logic
