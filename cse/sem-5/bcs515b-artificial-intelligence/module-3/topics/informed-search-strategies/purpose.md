# Learning Objectives

After studying this topic, you should be able to:

1. Define heuristic function and explain its role in informed search algorithms
2. Describe the Greedy Best-First Search algorithm and analyze its completeness, optimality, and complexity properties
3. Explain the A\* Search algorithm and derive the f(n) = g(n) + h(n) evaluation function
4. Differentiate between admissible and consistent heuristics with examples
5. Prove why A\* Search is optimal when using an admissible heuristic
6. Compare Greedy Best-First Search and A\* Search in terms of solution quality and efficiency
7. Analyze the time and space complexity of informed search algorithms
8. Apply informed search algorithms to solve problems like 8-puzzle and route finding
