# Learning Objectives

After studying this topic, you should be able to:

1. Define heuristic function h(n) and explain its role in informed search algorithms.

2. Distinguish between admissible and consistent (monotonic) heuristic functions with examples.

3. Explain the heuristic evaluation function f(n) = g(n) + h(n) and its components.

4. Design heuristic functions for classic AI problems like 8-puzzle and grid pathfinding.

5. Analyze heuristic dominance and compare the effectiveness of different heuristics.

6. Apply the concept of relaxed problems to derive admissible heuristics.

7. Evaluate the computational complexity of A\* search with different heuristic functions.

8. Identify the optimality conditions for A\* search with respect to heuristic properties.
