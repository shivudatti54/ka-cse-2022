# Learning Objectives

After studying this topic, you should be able to:

1. Define propositions and identify whether a given statement is a proposition or not

2. Construct truth tables for compound propositions using all five logical connectives (¬, ∧, ∨, →, ↔)

3. Distinguish between tautology, contradiction, and contingency with examples

4. Apply De Morgan's laws and other logical equivalences to simplify expressions

5. Translate English statements into propositional and predicate logic formulas

6. Explain the difference between universal and existential quantifiers with appropriate examples

7. Apply rules of inference (modus ponens, modus tollens, etc.) to construct logical proofs

8. Determine the truth value of quantified statements given specific domains
