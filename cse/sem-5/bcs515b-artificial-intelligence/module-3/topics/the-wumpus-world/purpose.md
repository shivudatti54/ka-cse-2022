# Learning Objectives

After studying this topic, you should be able to:

1. Describe the Wumpus World environment including its layout, components, and rules.

2. Identify and explain all six types of percepts received by the agent in the Wumpus World.

3. List and describe all six actions available to the agent in the Wumpus World.

4. Apply propositional logic to represent knowledge about the Wumpus World environment.

5. Perform logical inference to deduce safe and dangerous rooms based on percepts.

6. Analyze the performance measure and calculate scores for different agent behaviors.

7. Design a basic logic-based agent for the Wumpus World problem.

8. Compare forward chaining and backward chaining reasoning approaches in the context of the Wumpus World.
