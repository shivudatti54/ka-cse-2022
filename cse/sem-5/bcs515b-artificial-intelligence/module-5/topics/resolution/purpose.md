# Learning Objectives

After studying this topic, you should be able to:

1. Convert logical formulas from first-order predicate logic to clause form (CNF) through the systematic conversion process

2. Apply the resolution inference rule to derive new clauses from existing clauses in propositional logic

3. Perform unification between literals in first-order logic and compute the Most General Unifier (MGU)

4. Apply resolution in predicate logic by combining unification with the resolution rule

5. Understand and apply different resolution strategies (unit resolution, set of support, linear resolution) to control search complexity

6. Use refutation (proof by contradiction) to prove statements using resolution

7. Explain the significance of Horn clauses in logic programming and their relationship to Prolog

8. Analyze and solve problems requiring derivation of conclusions from given facts using resolution
