# Learning Objectives

After studying this topic, you should be able to:

1. Define classical planning problems formally and explain the components of the planning tuple (S, A, I, G, γ).

2. Represent planning actions using STRIPS formalism with preconditions, add list, and delete list.

3. Apply forward state-space search and backward state-space search to solve simple planning problems.

4. Explain the working of partial-order planning and identify open preconditions and resolvers.

5. Describe the structure of planning graphs and explain how GraphPlan extracts plans from them.

6. Apply heuristic functions in planning, particularly the delete-list relaxation heuristic.

7. Understand PDDL syntax and be able to read and interpret simple planning domain definitions.

8. Analyze the complexity of different planning approaches and select appropriate methods for specific problem characteristics.
