# Learning Objectives

After studying this topic, you should be able to:

1. Define classical planning and explain its formal mathematical formulation as a tuple (Σ, s₀, G)

2. Identify and explain the three fundamental assumptions of classical planning: full observability, determinism, and static world

3. Describe the components of a classical planning problem including states, actions, transition function, and goal conditions

4. Represent actions using preconditions and effects, and determine action applicability in given states

5. Understand how the successor state function (transition function) works in deterministic planning

6. Analyze and solve simple classical planning problems by finding valid action sequences

7. Interpret planning problems expressed in PDDL (Planning Domain Definition Language) format

8. Evaluate whether a given sequence of actions constitutes a valid plan for a planning problem

9. Apply backward planning techniques like goal regression to solve planning problems

10. Appreciate the complexity of classical planning and its relevance to real-world AI applications
