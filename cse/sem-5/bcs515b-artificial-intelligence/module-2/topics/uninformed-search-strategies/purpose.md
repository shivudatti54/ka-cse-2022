# Learning Objectives

After studying this topic, you should be able to:

1. Define and distinguish between informed and uninformed search strategies in AI.

2. Explain the working mechanism of Breadth-First Search (BFS) and implement it using appropriate data structures.

3. Explain the working mechanism of Depth-First Search (DFS) and implement it using appropriate data structures.

4. Analyze the completeness, optimality, time complexity, and space complexity of BFS and DFS.

5. Apply Uniform-Cost Search (UCS) to find minimum-cost paths in weighted graphs.

6. Compare and contrast Depth-Limited Search and Iterative Deepening Search as variants of DFS.

7. Evaluate the practical applications and limitations of each uninformed search strategy.

8. Select appropriate search strategies based on problem characteristics such as solution depth, branching factor, and available memory.
