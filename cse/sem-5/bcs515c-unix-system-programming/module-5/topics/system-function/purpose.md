# Learning Objectives

After studying this topic, you should be able to:

1. Identify and explain the six major categories of system calls in UNIX/Linux operating systems.

2. Differentiate between system calls and library functions, including the mode transition from user mode to kernel mode.

3. Demonstrate the use of process control system calls (fork, exec, wait, exit) for creating and managing processes.

4. Apply file manipulation system calls (open, read, write, close, lseek) for file I/O operations.

5. Understand the concept of file descriptors and their role in system call operations.

6. Analyze the execution flow of system calls and explain how arguments are passed between user space and kernel space.

7. Implement interprocess communication using pipe() and other communication system calls.

8. Evaluate error handling mechanisms in system calls using return values and errno.
