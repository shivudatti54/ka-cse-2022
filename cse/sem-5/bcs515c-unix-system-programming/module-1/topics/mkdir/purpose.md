# Learning Objectives

After studying this topic, you should be able to:

1. Understand the basic syntax and purpose of the `mkdir` command in Linux/Unix systems.

2. Create simple directories using both absolute and relative path specifications.

3. Use the `-p` (parent directories) option to create nested directory structures efficiently.

4. Apply the `-m` (mode) option to set specific permissions while creating directories.

5. Create multiple directories in a single command using space-separated arguments and brace expansion.

6. Understand directory permissions (rwx) and their significance in file system security.

7. Handle error conditions and interpret common error messages generated by `mkdir`.

8. Apply `mkdir` in practical shell scripting scenarios for automation tasks.

9. Verify directory creation using commands like `ls` and `find`.

10. Distinguish between file and directory creation in Unix-like systems.
