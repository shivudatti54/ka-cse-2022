# Learning Objectives

After studying this topic, you should be able to:

1. Explain the Linux hierarchical file system structure and the role of the root directory
2. Distinguish between absolute and relative pathnames with appropriate examples
3. Identify when to use absolute versus relative paths in different scenarios
4. Execute basic directory navigation commands (pwd, cd, ls) with proper syntax and options
5. Create and remove directories using mkdir and rmdir commands with appropriate flags
6. Apply special directory references (., .., ~, -) in path specifications
7. Use pathnames effectively in file and directory operations in Linux environment
8. Demonstrate practical directory management skills through hands-on exercises
