# Learning Objectives

After studying this topic, you should be able to:

1. Explain the complete structure and fields of the /etc/passwd file in Linux/Unix systems

2. Describe the purpose and format of the /etc/shadow file and explain why it improves system security

3. Identify different password hashing algorithms (MD5, SHA-256, SHA-512) and their security characteristics

4. Use the passwd command effectively to manage user passwords, including locking, unlocking, and setting password policies

5. Explain the complete user authentication process from login prompt to shell execution

6. Apply password aging policies using passwd and chage commands to enhance system security

7. Analyze /etc/passwd and /etc/shadow entries to extract user information and security details
