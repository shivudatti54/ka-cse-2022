# Learning Objectives

After studying this topic, you should be able to:

1. Explain the concept of pipes in Unix/Linux and their role in interprocess communication.

2. Describe the purpose and functionality of the popen() function in C programming.

3. Identify and explain the two modes ("r" and "w") of the popen() function and their appropriate use cases.

4. Write C programs that use popen() to read output from shell commands and send input to commands.

5. Properly use the pclose() function to close pipes and obtain command exit status.

6. Implement proper error handling for popen() and pclose() operations.

7. Analyze the internal working of popen() in terms of system calls (pipe, fork, exec).

8. Compare popen() with other IPC mechanisms like pipe(), FIFO, and message queues.
