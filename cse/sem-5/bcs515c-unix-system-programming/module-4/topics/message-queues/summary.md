Message queues enable asynchronous communication between distributed system components through FIFO buffering. Key concepts include producers (message senders), consumers (message receivers), message brokers (middleware managing storage and routing), and various delivery semantics (at-least-once, at-most-once, exactly-once). Common patterns are Point-to-Point (one-to-one delivery) and Publish-Subscribe (one-to-many broadcasting). Message persistence ensures durability against failures. Dead letter queues handle messages that cannot be processed. Queueing theory provides mathematical foundations for performance analysis and capacity planning.
