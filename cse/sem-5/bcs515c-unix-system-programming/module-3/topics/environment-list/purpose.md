# Learning Objectives

After studying this topic, you should be able to:

1. Define environment variables and explain their role in Unix/Linux systems

2. Distinguish between shell variables and environment variables, understanding the export mechanism

3. Identify and explain the purpose of commonly used environment variables such as PATH, HOME, USER, SHELL, and PWD

4. Use the `env` command to display, modify, and manage environment variables

5. Demonstrate how to set, modify, and unset environment variables both temporarily and persistently

6. Understand the concept of process inheritance and how environment variables are passed to child processes

7. Apply the `env` command in practical scenarios including shell scripting and portable program execution

8. Explain the significance of the PATH variable in command execution and search order
