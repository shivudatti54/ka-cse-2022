# Learning Objectives

After studying this topic, you should be able to:

1. Define environment variables and explain their role in Linux/Unix systems
2. Differentiate between shell variables and environment variables with respect to process inheritance
3. Identify and explain the purpose of common environment variables such as PATH, HOME, SHELL, USER, and PWD
4. Demonstrate the use of export command for creating environment variables
5. Configure persistent environment variables through appropriate shell configuration files
6. Apply environment variables in shell scripting for configuration management
7. Troubleshoot common issues related to environment variable settings
8. Understand how environment variables are used by system programs and applications
