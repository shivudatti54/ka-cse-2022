# Learning Objectives

After studying this topic, you should be able to:

1. Explain the concept of current working directory (CWD) and its role in Unix-like operating systems

2. Describe the purpose and functionality of the chdir() system call

3. Implement the chdir() system call in C programs with proper error handling

4. Distinguish between chdir() and fchdir() and explain when to use each

5. Use getcwd() to retrieve and save the current working directory

6. Understand how relative and absolute paths are resolved when using chdir()

7. Analyze error conditions that can occur with chdir() and implement appropriate error handling

8. Explain the inheritance of working directory across fork() and exec() system calls
