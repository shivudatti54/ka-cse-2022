# Learning Objectives

After studying this topic, you should be able to:

1. Explain the purpose and functionality of mkdir() and rmdir() system calls in UNIX/Linux operating systems

2. Identify the correct header files and function prototypes for mkdir() and rmdir()

3. Demonstrate the use of mkdir() to create directories with specified permissions

4. Implement error handling using errno for mkdir() and rmdir() operations

5. Explain the relationship between umask and directory permissions

6. Describe the conditions required for rmdir() to successfully remove a directory

7. Differentiate between various errno values and their meanings in directory operations

8. Write C programs that create and remove directories with proper error checking and validation
