# Learning Objectives

After studying this topic, you should be able to:

1. State and explain the fundamental equivalence theorem between pushdown automata and context-free grammars.

2. Construct a pushdown automaton from a given context-free grammar using the standard construction method.

3. Construct a context-free grammar from a given pushdown automaton using the state-pair variable technique.

4. Distinguish between acceptance by final state and acceptance by empty stack for PDAs, and explain their equivalence.

5. Trace and verify the execution of a PDA on a given input string to demonstrate language acceptance.

6. Explain the connection between deterministic pushdown automata and LR(k) grammars.

7. Apply the equivalence concepts to solve problems involving language recognition and generation.
