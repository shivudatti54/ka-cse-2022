# Learning Objectives

After studying this topic, you should be able to:

1. Define a context-free grammar (CFG) and identify its four essential components: variables, terminals, productions, and start symbol.

2. Construct CFGs for various languages including those with nested structures, matching pairs, and arithmetic expressions.

3. Perform derivations (both leftmost and rightmost) and construct parse trees for given grammar and input strings.

4. Identify and prove ambiguity in context-free grammars by demonstrating multiple parse trees for the same string.

5. Simplify CFGs by removing useless symbols, ε-productions, and unit productions while preserving the language generated.

6. Convert context-free grammars to Chomsky Normal Form (CNF) following the systematic conversion procedure.

7. Convert context-free grammars to Greibach Normal Form (GNF) and understand its applications in parsing.

8. Apply knowledge of CFGs to compiler design, particularly in understanding syntax analysis and parser generation.
