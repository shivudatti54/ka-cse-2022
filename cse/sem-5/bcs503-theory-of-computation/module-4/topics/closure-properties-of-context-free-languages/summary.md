# Closure Properties of Context-Free Languages - Summary

## Key Definitions and Concepts

- **Closure Property**: A class of languages is closed under an operation if applying that operation to languages in the class always produces a language in the same class.
- **Context-Free Language (CFL)**: A language generated by a Context-Free Grammar or recognized by a Pushdown Automaton.
- **Homomorphism**: A function that maps each symbol in an alphabet to a string (possibly empty).
- **Inverse Homomorphism**: Given homomorphism h, h⁻¹(L) = {w : h(w) ∈ L}.

## Important Formulas and Theorems

- **Union**: If L₁, L₂ are CFLs → L₁ ∪ L₂ is CFL
- **Concatenation**: If L₁, L₂ are CFLs → L₁L₂ is CFL
- **Kleene Star**: If L is CFL → L\* is CFL
- **Intersection with Regular**: If L₁ is CFL and L₂ is REG → L₁ ∩ L₂ is CFL
- **Substitution**: If L is CFL and each substitution is CFL → result is CFL
- **Homomorphism**: If L is CFL and h is homomorphism → h(L) is CFL
- **Inverse Homomorphism**: If L is CFL and h is homomorphism → h⁻¹(L) is CFL
- **Reversal**: If L is CFL → Lᴿ is CFL

## Key Points

- Context-free languages are **closed under**: Union, Concatenation, Kleene Star, Intersection with Regular Languages, Substitution, Homomorphism, Inverse Homomorphism, Reversal.

- Context-free languages are **NOT closed under**: General Intersection, Complementation, Set Difference.

- The property CFL ∩ REG = CFL is crucial and distinguishes CFLs from non-CFLs.

- The non-closure under intersection is proven by counterexample: {aⁿbⁿcⁱ} ∩ {aⁱbⁿcⁿ} = {aⁿbⁿcⁿ} (non-CFL).

- Non-closure under complementation follows from non-closure under intersection via De Morgan's law.

## Common Mistakes to Avoid

1. **Assuming closure under general intersection**: Many students mistakenly think CFLs are closed under intersection. Remember this is FALSE - only intersection with regular languages preserves CFL.

2. **Confusing homomorphism with inverse homomorphism**: These are different operations with different closure properties. Both preserve CFL but in different ways.

3. **Forgetting that Kleene star includes ε**: The closure L\* always includes the empty string.

4. **Not verifying the domain of substitution**: For substitution to preserve CFL, each substituted language must itself be context-free.

## Revision Tips

1. **Draw the closure diagram**: Place Regular Languages inside CFL, and remember which operations work at each level.

2. **Practice grammar constructions**: For union, concatenation, and Kleene star, be able to construct combined grammars quickly.

3. **Remember the counterexample**: The language {aⁿbⁿcⁿ} is the classic example showing CFLs are not closed under intersection.

4. **Focus on PDA construction**: For intersection with regular languages, understand how a PDA and DFA can be simulated in parallel.
