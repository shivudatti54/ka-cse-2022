# Learning Objectives

After studying this topic, you should be able to:

1. Define a Non-Deterministic Finite Automaton (NFA) formally using the 5-tuple notation M = (Q, Σ, δ, q₀, F).

2. Explain the key differences between DFA and NFA, particularly the nature of the transition function and acceptance criteria.

3. Compute ε-closure for states in an NFA with ε-transitions and understand its role in language recognition.

4. Determine whether a given string is accepted or rejected by an NFA by tracing all possible computational paths.

5. Construct an NFA for simple languages described in specifications, such as strings ending with particular patterns or containing specific substrings.

6. Apply the subset construction algorithm to convert an NFA to an equivalent DFA, demonstrating understanding of state equivalence.

7. Prove the equivalence between NFA and DFA by explaining that both recognize exactly the class of regular languages.
