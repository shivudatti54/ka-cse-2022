# Learning Objectives

After studying this topic, you should be able to:

1. Define a Turing Machine formally using the 7-tuple notation M = (Q, Σ, Γ, δ, q₀, q_accept, q_reject) and explain the purpose of each component.

2. Describe the working mechanism of a Turing Machine, including the roles of the tape, head, and control unit in computation.

3. Explain the Church-Turing Thesis and its significance in computer science, understanding what it means for a function to be Turing-computable.

4. Differentiate between deterministic and non-deterministic Turing Machines and explain why they have equivalent computational power.

5. Analyze common variants of Turing Machines (multi-tape, multi-head) and understand that all variants are computationally equivalent.

6. Distinguish between decidable and undecidable languages, with the Halting Problem as a prime example of an undecidable problem.

7. Design simple transition functions and state diagrams for Turing Machines that solve basic computational problems.
