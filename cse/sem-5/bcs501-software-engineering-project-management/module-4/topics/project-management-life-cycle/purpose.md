# Learning Objectives

After studying this topic, you should be able to:

1. Describe the five phases of the Project Management Life Cycle and their sequential relationship.

2. Identify the key activities and deliverables associated with each phase of the project life cycle.

3. Explain the importance of project initiation and how feasibility studies influence project decisions.

4. Demonstrate understanding of project planning activities including WBS development, scheduling, and resource allocation.

5. Analyze the role of monitoring and controlling processes in tracking project performance and ensuring successful delivery.

6. Apply Earned Value Management concepts to assess project health using metrics like SPI and CPI.

7. Evaluate the significance of project closure activities including formal acceptance and lessons learned documentation.

8. Compare how the project life cycle applies differently in various software development methodologies such as waterfall and agile.
