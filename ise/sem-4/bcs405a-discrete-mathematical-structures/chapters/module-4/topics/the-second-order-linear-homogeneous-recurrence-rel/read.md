# **The Second Order Linear Homogeneous Recurrence Relation with Constant Coefficients**

## **Introduction**

A recurrence relation is a mathematical relationship between a sequence of numbers, where each term is defined recursively as a function of previous terms. In this topic, we will explore the second order linear homogeneous recurrence relation with constant coefficients, which is a fundamental concept in discrete mathematical structures.

## **Definition**

A second order linear homogeneous recurrence relation with constant coefficients is an equation of the form:

an = pn-1 + qn-2

where:

- an is the nth term of the sequence
- pn-1 and qn-2 are constants
- n is a positive integer

## **Key Concepts**

### Types of Recurrence Relations

- **First order recurrence relations**: Each term depends on only one previous term.
- **Second order recurrence relations**: Each term depends on two previous terms.
- **Higher order recurrence relations**: Each term depends on multiple previous terms.

### Constant Coefficients

- **Constant sequence**: A sequence where each term is constant.
- **Constant coefficient**: A coefficient that remains constant throughout the recurrence relation.

### Homogeneous Recurrence Relations

- **Homogeneous equation**: An equation where all terms are equal to zero.
- **Homogeneous recurrence relation**: A recurrence relation where all terms are homogeneous (i.e., they have the same degree).

## **Examples**

### Example 1: Simple Recurrence Relation

Suppose we have the recurrence relation:

an = 2an-1

This is a first order recurrence relation with constant coefficients.

### Example 2: Second Order Recurrence Relation with Constant Coefficients

Suppose we have the recurrence relation:

an = 3an-1 - 2an-2

This is a second order recurrence relation with constant coefficients.

## **Solving Second Order Linear Homogeneous Recurrence Relations with Constant Coefficients**

To solve these recurrence relations, we can use the following methods:

- **Characteristic Equations**: We can write a characteristic equation based on the recurrence relation and solve for its roots.
- **Tableau Method**: We can use a table to compute the values of the sequence.

## **Characteristic Equations**

The characteristic equation of a second order linear homogeneous recurrence relation with constant coefficients is:

rn^2 + pn + q = 0

where r and p are constants.

**Example: Solving the Characteristic Equation**

Suppose we have the recurrence relation:

an = 3an-1 - 2an-2

The characteristic equation is:

r^2 - 3r + 2 = 0

Solving for r, we get:

r1 = 1
r2 = 2

## **Tableau Method**

The tableau method involves writing a table to compute the values of the sequence.

**Example: Using the Tableau Method**

Suppose we have the recurrence relation:

an = 3an-1 - 2an-2

| n   | an-1 | an-2 | an  |
| --- | ---- | ---- | --- |
| 1   | 0    | 0    | 0   |
| 2   | 0    | 1    | 3   |
| 3   | 1    | 3    | 5   |

We can compute the values of the sequence using the tableau method.
