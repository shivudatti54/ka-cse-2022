# **The Well Ordering Principle – Mathematical Induction**

**Important Definitions:**

- **Well Ordering Principle (WOP):** Every non-empty set of positive integers contains a least element.
- **Well-Ordered Set:** A set is well-ordered if it satisfies the WOP.

**Theorem:** If a set S of positive integers is well-ordered by a total order ≤, then every subset of S is also well-ordered by ≤.

**Mathematical Induction:**

- **Strong Induction (or Complete Induction):** A statement P(n) is true for all n if:
  1. P(1) is true.
  2. For all k, if P(k) is true, then P(k+1) is true.

- **Weak Induction (or Partial Induction):** A statement P(n) is true for all n if:
  1. P(1) is true.
  2. For all k, if P(k) is true and P(k+1) is false, then P(k) is true.

**Important Formulas:**

- **Pascal's Triangle:** A triangular array of binomial coefficients.

**Theorem:**
If P(n) is a statement that depends on n, and:

1. P(1) is true.
2. P(k) implies P(k+1) for all k.
   then P(n) is true for all n.

**Revision Key Points:**

- The Well Ordering Principle states that every non-empty set of positive integers contains a least element.
- Mathematical induction is a technique used to prove statements about positive integers.
- Strong induction is used to prove statements that are true for all positive integers.
- Weak induction is used to prove statements that are true for all positive integers except possibly the last one.
- Pascal's Triangle is an example of a sequence that can be proven by mathematical induction.
