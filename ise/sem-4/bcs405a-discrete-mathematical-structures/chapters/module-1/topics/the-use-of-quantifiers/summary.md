# Quantifiers in Discrete Mathematical Structures

### Key Points

- **Quantifiers**: Symbols or phrases used to indicate the scope of a predicate or property.
- **Domain**: The set of elements over which a predicate or property is defined.
- **Universal Quantifier** (∀):
  - Definition: ∀x P(x) means "P(x) is true for all x in the domain".
  - Example: ∀x x > 0.
- **Existential Quantifier** (∃):
  - Definition: ∃x P(x) means "P(x) is true for at least one x in the domain".
  - Example: ∃x x^2 = 4.
- **Prenex Form**: Formal representation of a sentence using quantifiers and predicates.
- **Monadic Predicate**: A predicate that takes one argument.
- **Poladic Predicate**: A predicate that takes multiple arguments.

### Important Formulas and Theorems

- **Distributive Law**: ∀x P(x) ∧ Q(x) ⇔ ∀x P(x) ∧ ∀x Q(x)
- **Associative Law**: ∀x P(x) ∧ ∀y Q(y) ∧ ∀z R(z) ⇔ ∀x ∀y ∀z P(x) ∧ Q(y) ∧ R(z)
- **Universal Law of Quantification**: ∀x P(x) ⇒ ∃x P(x)

### Definitions and Notations

- **Atomic Formula**: A formula that does not contain any quantifiers.
- **Predicate Symbol**: A symbol that represents a predicate.
- **Propositional Constant**: A symbol that represents a constant proposition (true or false).

### Theorems

- **Löwenheim-Skolem Theorem**: If a first-order theory has an infinite model, then it has models of every infinite cardinality.
- **Compactness Theorem**: If every finite subset of a set of sentences has a model, then the entire set has a model.

### Quick Revision Tips

- Understand the difference between universal and existential quantifiers.
- Learn the prenex form and monadic/poladic predicates.
- Familiarize yourself with the distributive, associative, and universal laws of quantification.
- Review the definitions and notations for atomic formulas, predicate symbols, and propositional constants.
