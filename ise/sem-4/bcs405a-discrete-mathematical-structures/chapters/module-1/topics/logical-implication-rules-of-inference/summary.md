# Logical Implication – Rules of Inference

### Definitions

- **Logical Implication (→)**: A statement of the form A → B, read as "A implies B" or "If A then B"
- **Implication Rule**: A rule of inference that allows us to conclude B from A → B

### Important Formulas and Theorems

- **Modus Ponens**: If A → B and A, then B (Rule of Inference)
- **Modus Tollens**: If A → B and not B, then not A (Rule of Inference)
- **Converse Implication**: A → B if and only if B → A (Theorem)
- **Inverse Implication**: A → B if and only if not B → not A (Theorem)

### Rules of Inference

- **Modus Ponens**: From A → B and A, infer B
- **Modus Tollens**: From A → B and not B, infer not A
- **Disjunctive Syllogism**: From A ∨ B and not A, infer B
- **Hypothetical Syllogism**: From A → B and B → C, infer A → C

### Important Theorems

- **De Morgan's Law**: (A → B) → (not A ∨ not B)
- **Deduction Theorem**: If A → B, then A ¬B ⇒ ¬B

### Other Relevant Rules

- **Addition Rule**: From A → B and A, infer B → B
- **Subtraction Rule**: From A → B and A, infer not B → not B

This summary provides an overview of the key concepts, formulas, and rules of inference related to logical implication. It is designed to be a quick revision guide for exams.
