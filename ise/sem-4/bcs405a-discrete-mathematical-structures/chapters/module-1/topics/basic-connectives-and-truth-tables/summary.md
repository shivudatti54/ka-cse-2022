# **Basic Connectives and Truth Tables**

### Definitions and Formulas

- **Propositional Logic**: A branch of logic that deals with statements that can be either true (T) or false (F).
- **Connectives**:
  - **Conjunction** (∧): True if both propositions are true.
  - **Disjunction** (∨): True if at least one proposition is true.
  - **Negation** (¬): True if the proposition is false, and false if the proposition is true.
  - **Implication** (→): True if the antecedent is false or the consequent is true.
  - **Equivalence** (≡): True if both propositions have the same truth value.
- **Truth Tables**:
  - A table that shows the truth values of a proposition and its connectives under all possible combinations of truth values.

### Important Formulas and Theorems

- **De Morgan's Laws**:
  - ¬(p ∧ q) = ¬p ∨ ¬q
  - ¬(p ∨ q) = ¬p ∧ ¬q
- **Law of Excluded Middle**:
  - p ∨ ¬p = T (True)
- **Law of Non-Contradiction**:
  - p ∧ ¬p = F (False)

### Key Truth Tables

| Connective      | T   | F   | T   | F   |
| --------------- | --- | --- | --- | --- |
| ∧ (Conjunction) | T   | T   | F   | F   |
| ∨ (Disjunction) | F   | F   | T   | T   |
| ¬ (Negation)    | F   | T   | T   | F   |
| → (Implication) | F   | T   | T   | F   |
| ≡ (Equivalence) | T   | F   | F   | T   |

### Theorems

- **Theorem 1**: If p → q and p, then q.
- **Theorem 2**: If p and q, then p ∨ q.

### Quick Revision Tips

- Use truth tables to evaluate complex propositions.
