# **Basic Connectives and Truth Tables**

### Introduction

---

In discrete mathematical structures, logic plays a crucial role in representing and manipulating information. Connectives are operators that combine propositions to form new propositions. Truth tables provide a systematic way to evaluate the truth of these combined propositions. In this section, we will explore the basic connectives and truth tables.

### Basic Connectives

---

Connectives are operators that take propositions as input and produce a new proposition as output. There are several basic connectives, including:

- **Conjunction (AND)**: denoted by ∧
- **Disjunction (OR)**: denoted by ∨
- **Negation (NOT)**: denoted by ¬
- **Implication (IF-THEN)**: denoted by →
- **Equivalence**: denoted by ≡

### Definitions

---

- **Proposition**: a statement that can be either true or false.
- **Conjunct**: a proposition combined with another proposition using conjunction.
- **Disjunct**: a proposition combined with another proposition using disjunction.
- **Negated proposition**: a proposition combined with its negation using negation.

### Truth Tables

---

A truth table is a table that lists all possible combinations of truth values of the propositions involved, along with the resulting truth value of the combined proposition.

#### Truth Table for Conjunction (AND)

| P   | Q   | P ∧ Q |
| --- | --- | ----- |
| T   | T   | T     |
| T   | F   | F     |
| F   | T   | F     |
| F   | F   | F     |

- The proposition P ∧ Q is true only when both P and Q are true.

#### Truth Table for Disjunction (OR)

| P   | Q   | P ∨ Q |
| --- | --- | ----- |
| T   | T   | T     |
| T   | F   | T     |
| F   | T   | T     |
| F   | F   | F     |

- The proposition P ∨ Q is true if either P or Q (or both) is true.

#### Truth Table for Negation (NOT)

| P   | ¬P  |
| --- | --- |
| T   | F   |
| F   | T   |

- The proposition ¬P is true when P is false, and vice versa.

#### Truth Table for Implication (IF-THEN)

| P   | Q   | P → Q |
| --- | --- | ----- |
| T   | T   | T     |
| T   | F   | F     |
| F   | T   | T     |
| F   | F   | T     |

- The proposition P → Q is false only when P is true and Q is false.

#### Truth Table for Equivalence

| P   | Q   | P ≡ Q |
| --- | --- | ----- |
| T   | T   | T     |
| T   | F   | F     |
| F   | T   | F     |
| F   | F   | T     |

- The proposition P ≡ Q is true when both P and Q have the same truth value.

### Key Concepts

---

- **Truth values**: the possible truth values of a proposition (T for true, F for false).
- **Propositional logic**: the study of logical operators and their interactions.
- **Truth tables**: a systematic way to evaluate the truth of combined propositions.

### Conclusion

---

Basic connectives and truth tables provide a fundamental framework for propositional logic. Understanding these concepts is essential for evaluating and manipulating logical expressions. With practice and application, you can develop a deep understanding of logical operators and their interactions.
