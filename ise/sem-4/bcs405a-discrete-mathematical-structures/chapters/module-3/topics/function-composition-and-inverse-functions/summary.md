# **Function Composition and Inverse Functions**

### Definitions and Formulas

- Function Composition: The process of combining two or more functions to create a new function.
- Inverse Function: A function that undoes the action of the original function.
- Domain and Range: The set of input and output values for a function.

### Key Concepts

- **Function Composition:**
  - Formula: `f ∘ g = (f ∘ g)(x) = f(g(x))`
  - Properties:
    - Associative: `(f ∘ g) ∘ h = f ∘ (g ∘ h)`
    - Identity: `f ∘ id = id ∘ f = f`
- **Inverse Functions:**
  - Formula: `f^(-1) = {x | f(x) = y => x = f^(-1)(y)}`
  - Properties:
    - Inverse of inverse: `(f ∘ f^(-1))(x) = x`
    - Inverse of function is unique: `f^(-1)` is unique if `f` is one-to-one
- **One-to-One and Onto Functions:**
  - One-to-One: Each output value is associated with exactly one input value.
  - Onto: Each output value is associated with at least one input value.

### Important Theorems

- **Invertible Functions:**
  - If `f` is one-to-one and onto, then `f^(-1)` is well-defined.
- **Inverse of a Composite Function:**
  - `(f ∘ g)^(-1) = g^(-1) ∘ f^(-1)`

### Key Properties

- **Even and Odd Functions:**
  - Even function: `f(x) = f(-x)`
  - Odd function: `f(-x) = -f(x)`
- **Symmetry of Functions:**
  - Symmetric about the y-axis: `f(-x) = f(x)`
  - Symmetric about the line y = x: `f(x, y) = f(y, x)`
