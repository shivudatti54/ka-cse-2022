# Relations and Functions

### Properties of Relations

- A relation R on a set A is a subset of A x A.
- A relation R on a set A is reflexive if (a, a) ∈ R for all a ∈ A.
- A relation R on a set A is symmetric if (a, b) ∈ R implies (b, a) ∈ R for all a, b ∈ A.
- A relation R on a set A is transitive if (a, b) ∈ R and (b, c) ∈ R implies (a, c) ∈ R for all a, b, c ∈ A.
- A relation R on a set A is antisymmetric if (a, b) ∈ R and (b, a) ∈ R implies a = b for all a, b ∈ A.

### Zero-One Matrices and Directed Graphs

- A zero-one matrix represents a binary relation on a set.
- A directed graph is a 2-tuple (V, E) where V is a set of vertices and E is a set of directed edges.
- A directed path from vertex u to vertex v in a directed graph is a sequence of vertices and edges such that the tail of each edge is the next vertex in the sequence.

### Partial Orders – Hasse Diagrams

- A partial order on a set A is a reflexive, antisymmetric, and transitive relation.
- A Hasse diagram is a graphical representation of a partial order.
- A Hasse diagram is a drawing of vertices and edges such that:
  - Vertices represent elements of the set.
  - Edges represent the partial order relation.
  - Edges connect vertices if the corresponding elements are comparable.

### Equivalence Relations and Partitions

- An equivalence relation on a set A is a reflexive, symmetric, and transitive relation.
- An equivalence partition of a set A is a partition of A into non-empty subsets such that every element of A is in exactly one subset.
- The partition of A induced by an equivalence relation R is the set of equivalence classes of A with respect to R.

Important Formulas and Definitions:

- Reflexive property: (a, a) ∈ R for all a ∈ A.
- Symmetric property: (a, b) ∈ R implies (b, a) ∈ R for all a, b ∈ A.
- Transitive property: (a, b) ∈ R and (b, c) ∈ R implies (a, c) ∈ R for all a, b, c ∈ A.
- Antisymmetric property: (a, b) ∈ R and (b, a) ∈ R implies a = b for all a, b ∈ A.
- Equivalence class: [a] = {x ∈ A | (x, a) ∈ R} for all a ∈ A.
- Partition: A set of non-empty subsets of A such that every element of A is in exactly one subset.
