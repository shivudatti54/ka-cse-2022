# Functions – Plain and One-to-One

**Definitions and Notations**

- A function is a relation between a set of inputs (called the domain) and a set of possible outputs (called the range).
- A relation is a subset of the Cartesian product of two sets.
- A function is denoted by a notation of the form f(x) or f: A → B.
- The domain of a function is the set of all inputs, and the range is the set of all outputs.

**Properties of Functions**

- **one-to-one (injective)**: Each output value corresponds to exactly one input value.
- **onto (surjective)**: Every output value is mapped to by at least one input value.
- **bijective**: A function that is both one-to-one and onto.

**Key Formulas and Theorems**

- **Function notation**: f(x) = y ⇔ (x, y) ∈ f
- **Inverse function**: If f: A → B is one-to-one, then the inverse function f^(-1): B → A is defined by f^(-1)(y) = x if and only if f(x) = y.
- **Bijection theorem**: If f: A → B is bijective, then the cardinality of A and B are equal (|A| = |B|).

**Properties of Plain Functions**

- A function f: A → B is plain if for every x in A, there exists a unique y in B such that f(x) = y.
- A set function f: M → N is a plain function if for every subset M of A, the function f is one-to-one.

**Revision Tips**

- Ensure that the function is one-to-one and onto to prove that it is bijective.
- Use the definition of function notation and inverse functions to analyze functions.
- Recall the bijection theorem to determine the cardinality of sets.
