# Functions – Plain and One-to-One

### Introduction

In the study of discrete mathematical structures, functions play a crucial role in modeling relationships between sets. In this topic, we will explore two types of functions: plain functions and one-to-one functions.

### Plain Functions

A plain function, also known as a relation, is a set of ordered pairs that assign an element from one set to another element from the same set. A plain function can be represented as:

f: A → B

where A is the domain (set of input elements) and B is the codomain (set of output elements).

**Definition:** A function is a relation from A to B that satisfies the following properties:

- Every element of A is related to exactly one element of B.
- No two elements of A are related to the same element of B.

**Examples:**

- f: {1, 2, 3} → {a, b, c}, where f(1) = a, f(2) = b, and f(3) = c.
- g: {x, y} → {0, 1}, where g(x) = 0 and g(y) = 1.

### Key Concepts:

- **Domain:** The set of input elements in a function.
- **Codomain:** The set of output elements in a function.
- **Related element:** An element of the domain that is assigned to an element of the codomain.
- **Range:** The set of all related elements in a function.

### One-to-One Functions

A one-to-one function is a plain function that satisfies the following condition:

- For every element in the domain, there is exactly one related element in the codomain.

**Definition:** A function f: A → B is one-to-one if and only if for every x and y in A, if f(x) = f(y), then x = y.

**Examples:**

- f: {1, 2, 3} → {a, b, c}, where f(1) = a, f(2) = b, and f(3) = c.
- h: {x, y} → {0, 1}, where h(x) = 0 and h(y) = 1.

### Key Concepts:

- **Injective:** A function is injective if it is one-to-one.
- **Onto:** A function is onto if every element in the codomain is related to at least one element in the domain.
- **Bijective:** A function is bijective if it is both injective and onto.

### Conclusion

In conclusion, plain functions are a fundamental concept in discrete mathematical structures, and one-to-one functions are a special type of plain function that satisfies the condition of being injective. Understanding these concepts is essential for modeling relationships between sets and solving problems in various fields.
