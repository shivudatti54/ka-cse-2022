# (8 hours) Revision Notes: Relations and Functions

=============================================

### Important Definitions

- **Relation**: A set of ordered pairs (a, b) where a and b are elements of a set A.
- **Function**: A relation between a set A and a set B, where each element of A is mapped to exactly one element of B.
- **One-to-One (Injective)**: A function where each element of the domain maps to a unique element of the codomain.
- **Many-to-One (Surjective)**: A function where every element of the codomain is mapped to by at least one element of the domain.
- ** Bijection**: A function that is both one-to-one and many-to-one.

### Important Formulas

- **Inverse Function**: f^-1(x) = y if and only if f(y) = x
- **Domain**: The set of all input values for a function.
- **Codomain**: The set of all possible output values for a function.
- **Range**: The set of all output values for a function.

### Important Theorems

- **Horizontal Line Test**: A horizontal line intersects the graph of a function at most once.
- **Vertical Line Test**: A vertical line intersects the graph of a function at most once.
- **Existence of Inverse**: If a function is one-to-one, it has an inverse.

### Important Properties

- **Reflexive**: A relation is reflexive if (a, a) is in the relation for all a in the set.
- **Symmetric**: A relation is symmetric if (a, b) is in the relation if and only if (b, a) is in the relation.
- **Transitive**: A relation is transitive if (a, b) is in the relation and (b, c) is in the relation, then (a, c) is in the relation.
