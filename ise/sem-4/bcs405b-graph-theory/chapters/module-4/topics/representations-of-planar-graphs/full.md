# Representations of Planar Graphs

### Introduction

Planar graphs are a fundamental concept in graph theory, and their representations are crucial in understanding various applications in computer science, engineering, and other fields. In this section, we will delve into the world of planar graphs, exploring their representations, historical context, and modern developments.

### Historical Context

The concept of planar graphs dates back to the 19th century, when mathematician Karl Wilhelm Feuerbach introduced the term "planar graph" in 1860. However, it wasn't until the early 20th century that planar graphs gained significant attention, thanks to the work of mathematicians like Édouard Lucas and Fränz Kirchhoff.

One of the most significant milestones in the study of planar graphs was the development of Kuratowski's theorem, which states that a graph is planar if and only if it does not contain a subgraph that is a subdivision of K5 or K3,3. Kuratowski's theorem was first proved by Anatolii Alexeevich Kurosh in 1930.

### Types of Representations

Planar graphs can be represented in various ways, each with its own strengths and weaknesses. Some of the most common representations include:

#### 1. Cartesian Coordinates

One of the simplest ways to represent planar graphs is using Cartesian coordinates. Each vertex is assigned a set of x and y coordinates, and edges are drawn between vertices that are within a certain distance of each other.

**Example:**

Suppose we have a planar graph with vertices at (0,0), (1,0), (1,1), and (0,1). The edges connecting these vertices can be drawn using Cartesian coordinates.

```
 0     |-----|-----|
 0     |     |     |
 1     |     |     |
 1     |-----|-----|
 1     |     |     |
 0     |-----|-----|
```

#### 2. Planar Embeddings

Planar embeddings are a more sophisticated way to represent planar graphs. In this representation, the graph is drawn in a plane, with each vertex assigned a set of coordinates that define its position in the plane.

**Example:**

Suppose we have a planar graph with vertices at (0,0), (1,0), (1,1), and (0,1). The edges connecting these vertices can be drawn using a planar embedding.

```
  +---+----+----+
  | 0 |   |   |   |
  +---+----+----+
  |   |   |   |
  |   |   |   |
  +---+----+----+
  |   |   |   |
  |   |   |   |
  |   |   |   |
  +---+----+----+
```

#### 3. Topological Embeddings

Topological embeddings are another way to represent planar graphs. In this representation, the graph is drawn in a plane, with each vertex assigned a set of coordinates that define its position in the plane. However, unlike planar embeddings, topological embeddings do not require the graph to be drawn in a way that preserves the edges.

**Example:**

Suppose we have a planar graph with vertices at (0,0), (1,0), (1,1), and (0,1). The edges connecting these vertices can be drawn using a topological embedding.

```
  +---+----+----+
  | 0 |   |   |   |
  +---+----+----+
  |   |   |   |
  |   |   |   |
  +---+----+----+
  |   |   |   |
  |   |   |   |
  |   |   |   |
  +---+----+----+
```

### Applications

Planar graphs have numerous applications in various fields, including:

#### 1. Computer-Aided Design (CAD)

Planar graphs are used in CAD systems to represent complex geometric shapes and structures.

#### 2. Geographic Information Systems (GIS)

Planar graphs are used in GIS to represent geographic data, such as road networks and buildings.

#### 3. VLSI Design

Planar graphs are used in VLSI design to represent digital circuits and chip layouts.

#### 4. Network Analysis

Planar graphs are used in network analysis to represent communication networks and social networks.

### Modern Developments

In recent years, there has been significant progress in the study of planar graphs, including:

#### 1. Random Planar Graphs

Random planar graphs have been studied extensively, and their properties have been used to develop new algorithms and techniques.

#### 2. Planar Graph Algorithms

New algorithms have been developed to solve problems related to planar graphs, such as graph coloring and planar embedding.

#### 3. Planar Graph Applications

Planar graphs have been applied to a wide range of fields, including computer science, engineering, and medicine.
