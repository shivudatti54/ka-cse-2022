# **Circuits**

## **Definitions and Formulas**

- **Circuit**: A closed path in a graph that visits each edge exactly once.
- **Hamiltonian Circuit**: A circuit that visits each vertex exactly once.
- **Eulerian Circuit**: A circuit that uses each edge exactly once.
- **Eulerian Path**: A path that visits each edge exactly once.

## **Key Theorems and Results**

- **Euler's Theorem**: A connected graph has an Eulerian circuit if and only if it has at most two vertices of odd degree.
- **Hamiltonian Theorem**: A connected graph has a Hamiltonian circuit if and only if it is simple and has no isolated vertices.

## **Operations and Properties**

- **Circumference of a Circuit**: The number of edges in a circuit.
- **Circuit sum**: The sum of the lengths of the edges in a circuit.
- **Tightness of a Circuit**: The minimum number of edges required to form a circuit.

## **Important Formulas**

- **Euler's Formula**: V - E + F = 2 (for planar graphs)
- **Handshaking Lemma**: Σdeg(v) = 2E (for any graph)
