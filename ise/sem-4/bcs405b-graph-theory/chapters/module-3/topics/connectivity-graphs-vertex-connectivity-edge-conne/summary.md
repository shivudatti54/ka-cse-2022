# Connectivity Graphs

### Definitions and Notations

- **Graph**: A pair (V, E), where V is a set of vertices and E is a set of edges connecting vertices.
- **Vertex**: An element of the set V.
- **Edge**: A pair of vertices connected by a line segment.
- **Connectivity**: The minimum number of vertices or edges that need to be removed to disconnect the graph.

### Vertex Connectivity

- **Vertex Connectivity (δ(v))**: The minimum number of vertices that need to be removed to disconnect the graph at vertex v.
- **Vertex Cut Set**: A set of vertices whose removal disconnects the graph.
- **Vertex Cut Vertices**: A set of vertices whose removal connects the graph.

### Edge Connectivity

- **Edge Connectivity (δ(e))**: The minimum number of edges that need to be removed to disconnect the edge e.
- **Edge Cut Set**: A set of edges whose removal disconnects the graph.
- **Edge Cut Vertices**: The vertices adjacent to the edges in the edge cut set.

### Cut Set and Cut Vertices

- **Cut Set**: A set of vertices or edges whose removal disconnects the graph.
- **Cut Vertices**: A set of vertices whose removal connects the graph.

### Fundamental Circuits

- **Fundamental Circuit**: A circuit whose removal disconnects the graph.
- **Fundamental Circuit Theorem**: Every connected graph has a fundamental circuit.

### Formulas and Theorems

- **Menger's Theorem**: The minimum number of vertices that need to be removed to disconnect the graph is equal to the minimum number of edges that need to be removed.
- **König's Theorem**: The maximum number of edges in a cut set is equal to the minimum number of vertices in a cut set.

### Key Points

- Vertex connectivity is a measure of how difficult it is to remove a vertex from a graph without disconnecting it.
- Edge connectivity is a measure of how difficult it is to remove an edge from a graph without disconnecting it.
- A cut set is a set of vertices or edges whose removal disconnects the graph.
- A fundamental circuit is a circuit whose removal disconnects the graph.
