# Pendant Vertex and Null Graph

## Introduction

Graph theory is a branch of mathematics that deals with the study of graphs, which are non-linear data structures composed of vertices (or nodes) and edges. In this topic, we will delve into two specific concepts in graph theory: pendant vertices and null graphs.

### Historical Context

The concept of pendant vertices and null graphs dates back to the early days of graph theory. The term "pendant vertex" was first introduced by the German mathematician Konrad Zuse in the 1950s, while the concept of a null graph was introduced by the American mathematician Paul Erdős in the 1940s.

### Definition and Notation

A **pendant vertex** is a vertex in a graph that has degree 1. In other words, a pendant vertex is a vertex that is connected to only one other vertex in the graph. The notation for a pendant vertex is `p`.

A **null graph** is a graph that has no edges. In other words, a null graph is a graph that consists only of vertices, but no edges connect any of these vertices. The notation for a null graph is `∅`.

### Properties of Pendant Vertices and Null Graphs

## Pendant Vertices

- A pendant vertex is a vertex of degree 1.
- Pendant vertices are also known as "leaf" vertices.
- In a tree, a pendant vertex is a vertex that is a leaf node.
- In a graph, a pendant vertex is a vertex that is connected to only one other vertex.

## Null Graphs

- A null graph is a graph that has no edges.
- A null graph is also known as an "empty graph".
- Every null graph is a 1-regular graph, meaning that every vertex has degree 0.

## Types of Pendant Vertices

There are two types of pendant vertices:

- **Internal Pendant Vertex**: An internal pendant vertex is a pendant vertex that is not part of the boundary of the graph.
- **Boundary Pendant Vertex**: A boundary pendant vertex is a pendant vertex that is part of the boundary of the graph.

## Types of Null Graphs

There are two types of null graphs:

- **Complete Null Graph**: A complete null graph is a null graph that contains a vertex of degree 1.
- **Incomplete Null Graph**: An incomplete null graph is a null graph that does not contain a vertex of degree 1.

## Properties of Pendant Vertices and Null Graphs

Pendant Vertices and Null Graphs have the following properties:

- **Degree of Pendant Vertices**: The degree of a pendant vertex is 1.
- **Degree of Vertices in Null Graphs**: The degree of every vertex in a null graph is 0.
- **Number of Edges**: The number of edges in a null graph is 0.

## Applications

Pendant Vertices and Null Graphs have the following applications:

- **Computer Networks**: Pendant vertices can represent leaf nodes in a tree-like network structure, while null graphs can represent the absence of connections between nodes.
- **Social Networks**: Pendant vertices can represent individuals who are connected to only one other person, while null graphs can represent the absence of connections between individuals.
- **Biology**: Pendant vertices can represent terminal branches in a tree-like structure of biological organisms, while null graphs can represent the absence of connections between organisms.
