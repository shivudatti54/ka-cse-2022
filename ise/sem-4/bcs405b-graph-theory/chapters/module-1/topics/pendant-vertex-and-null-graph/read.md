# **Pendant Vertex and Null Graph**

## **Introduction**

In graph theory, a pendant vertex is a vertex with a degree of 1, and a null graph is a graph with no edges. In this study material, we will explore the definitions, properties, and examples of pendant vertices and null graphs.

## **Pendant Vertex**

### Definition

A pendant vertex is a vertex in a graph that has a degree of 1. In other words, it is a vertex that is connected to only one other vertex.

### Examples

- Consider a graph with two vertices, A and B, and a single edge connecting them. The vertex A is a pendant vertex because it has a degree of 1.
- Consider a graph with three vertices, A, B, and C, and edges connecting A to B and C to B. Vertex C is a pendant vertex because it has a degree of 1.

### Properties

- A pendant vertex is also known as an isolated vertex.
- A pendant vertex can be a leaf vertex.
- A pendant vertex can be part of a graph with a degree of 1 for every vertex.

### Important Points

- The sum of the degrees of all vertices in a graph is equal to twice the number of edges.
- A pendant vertex is a vertex with a degree of 1.

## **Null Graph**

### Definition

A null graph is a graph with no edges. In other words, it is a graph with zero edges.

### Examples

- Consider a graph with two vertices, A and B, but no edges connecting them. This graph is a null graph.
- Consider a graph with three vertices, A, B, and C, but only edges connecting A to B and B to C. This graph is not a null graph because it has one edge.

### Properties

- A null graph has no edges.
- A null graph has zero vertices.
- A null graph has no degree.

### Important Points

- A null graph is a graph with no edges.
- A null graph is a special case of an empty graph.

## **Relationship between Pendant Vertex and Null Graph**

- A pendant vertex is a vertex with a degree of 1.
- A null graph is a graph with no edges.
- A pendant vertex can be part of a null graph if the null graph has only one vertex.

### Key Concepts

- Pendant vertex: a vertex with a degree of 1
- Null graph: a graph with no edges
- Leaf vertex: a pendant vertex
- Isolated vertex: a pendant vertex
- Graph with a degree of 1 for every vertex: a graph with all vertices as pendant vertices
