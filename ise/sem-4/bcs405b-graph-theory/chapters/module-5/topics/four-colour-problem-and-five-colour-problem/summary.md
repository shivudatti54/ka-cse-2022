# Four Colour Problem and Five Colour Problem

### Definitions and Key Concepts

- **Graph Colouring**: Assigning colours to vertices of a graph such that no two adjacent vertices have the same colour.
- **Chromatic Number**: The minimum number of colours needed to colour a graph such that no two adjacent vertices have the same colour.

### Four Colour Problem

- **Statement**: Every planar graph can be coloured using 4 colours such that no two adjacent vertices have the same colour.
- **Important Theorem**: Four Colour Theorem (Kuratowski's Theorem) states that a planar graph is 4-colourable if and only if it does not contain a subgraph homeomorphic to K5 (complete graph on 5 vertices) or K3,3 (complete bipartite graph on 6 vertices).
- **Proof**: Uses induction and topological invariance.

### Five Colour Problem

- **Statement**: There is no known efficient algorithm to determine whether a given planar graph can be coloured using 5 colours such that no two adjacent vertices have the same colour.
- **Important Theorem**: No known bound on the chromatic number of a planar graph that can be coloured using 5 colours.

### Important Formulas and Results

- **Chromatic Polynomial**: A polynomial that encodes the number of ways to colour a graph with k colours such that no two adjacent vertices have the same colour.
- **Farkas' Lemma**: A theorem that relates the existence of a linear programme to the colourability of a graph.

### Revision Tips

- Review the Four Colour Theorem and its proof.
- Understand the distinction between the Four Colour Problem and the Five Colour Problem.
- Familiarise yourself with the chromatic polynomial and its relation to graph colouring.
