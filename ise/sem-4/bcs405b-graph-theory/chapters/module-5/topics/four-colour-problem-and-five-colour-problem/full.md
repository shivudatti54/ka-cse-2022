# Four Colour Problem and Five Colour Problem

## Introduction

The Four Colour Problem and Five Colour Problem are two of the most famous unsolved problems in Graph Theory, a branch of mathematics that deals with the study of graphs, which are collections of nodes or vertices connected by edges. These problems have been extensively studied for over a century, and although they remain unsolved, they have led to significant advances in graph theory and its applications.

### Historical Context

The Four Colour Problem was first proposed by Francis Guthrie in 1852, as a conjecture about the number of colours needed to colour a map of the British Isles using four colours. Over the years, the problem gained popularity, and mathematicians such as Alfred Kempe and Percy Heawood made significant contributions to its study. In 1976, Kenneth Appel and Wolfgang Haken proved that the Four Colour Problem has a solution, but their proof was so long and cumbersome that it took over 150 pages to present it.

The Five Colour Problem, on the other hand, was first proposed by Kenneth Appel in 1968. Like the Four Colour Problem, it deals with the minimum number of colours needed to colour a map of the world using five colours.

### Definition of the Problems

The Four Colour Problem is: Given a planar map, can it be coloured using four colours in such a way that no two adjacent regions have the same colour?

The Five Colour Problem is: Given a planar map, can it be coloured using five colours in such a way that no two adjacent regions have the same colour?

### Graph Colouring

Graph colouring is a fundamental problem in graph theory, which deals with assigning colours to the vertices of a graph such that no two adjacent vertices have the same colour. The Four Colour Problem and Five Colour Problem are special cases of graph colouring, where the number of colours is fixed.

### Chromatic Number

The chromatic number of a graph is the minimum number of colours needed to colour the graph such that no two adjacent vertices have the same colour. The Four Colour Problem and Five Colour Problem are equivalent to finding the chromatic number of a planar graph.

### Chromatic Polynomial

The chromatic polynomial of a graph is a polynomial that counts the number of ways to colour the graph using a given number of colours. The chromatic polynomial can be used to study the graph colouring problem and is a powerful tool in graph theory.

### Matchings and Coverings

Matchings and coverings are two important concepts in graph theory that are related to graph colouring. A matching is a subset of edges in a graph such that no two edges share a common vertex. A covering is a subset of vertices in a graph such that every edge is incident to at least one vertex in the subset.
