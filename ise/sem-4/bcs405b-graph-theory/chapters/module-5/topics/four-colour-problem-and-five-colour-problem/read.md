# **Four Colour Problem and Five Colour Problem**

## **Introduction**

The Four Colour Problem and Five Colour Problem are two of the most well-known problems in graph theory. The Four Colour Problem asks whether it is possible to colour the faces of a planar map (a map that can be drawn on a flat surface) using four colours such that no two adjacent faces have the same colour. The Five Colour Problem is a more general version of this problem, where it asks whether it is possible to colour the faces of a planar map using five colours.

## **Definition of Planar Map**

A planar map is a map that can be drawn on a flat surface using a finite number of edges and vertices. The faces of the map are the areas enclosed by the edges. A planar map is said to be planar if it can be drawn in a plane without any edge crossings.

## **Definition of Chromatic Number**

The chromatic number of a graph is the minimum number of colours needed to colour the vertices of the graph such that no two adjacent vertices have the same colour. The chromatic number of a planar map is the minimum number of colours needed to colour the faces of the map such that no two adjacent faces have the same colour.

## **Four Colour Problem**

The Four Colour Problem asks whether it is possible to colour the faces of a planar map using four colours such that no two adjacent faces have the same colour. This problem was first proposed by Francis Guthrie in 1852 and has been the subject of much research and speculation.
