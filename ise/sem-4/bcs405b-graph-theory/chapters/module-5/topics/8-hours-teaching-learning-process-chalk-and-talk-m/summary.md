# Teaching-Learning Process: Chalk and Talk Method / PowerPoint Presentation

=====================================

## **Key Points**

### Chalk and Talk Method

- Advantages:
  - Encourages active participation
  - Allows for immediate clarification and discussion
  - Easy to implement
- Disadvantages:
  - Can be time-consuming
  - Limited to the instructor's ability to convey information

### PowerPoint Presentation

- Advantages:
  - Visual aids can enhance understanding
  - Organized and structured content
  - Easy to review and refer back to
- Disadvantages:
  - May lead to passivity among students
  - Can be overwhelming with too much information

### Key Formulas, Definitions, and Theorems

- **Chromatic Number (χ)**: The minimum number of colours required to colour a graph
- **Chromatic Polynomial**: An algebraic expression that gives the chromatic number for a given graph
- **Matching**: A subset of edges in a graph that do not share a common vertex
- **Covering**: A set of vertices in a graph that covers all edges

### Key Concepts

- **Degree**: The number of edges incident on a vertex
- **Neighbourhood**: The set of vertices adjacent to a given vertex
- **Regular Graph**: A graph where all vertices have the same degree

### Teaching Strategies

- **Guided Practice**: Encourage students to work in groups to apply concepts
- **Independent Practice**: Provide students with problems to work on individually
- **Formative Assessment**: Regularly assess student understanding to adjust instruction
