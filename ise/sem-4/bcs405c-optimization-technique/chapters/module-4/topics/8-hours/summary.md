### Key Concepts

- **Convex Sets**: A set where all line segments connecting any two points in the set lie entirely within the set
- **Convex Functions**: Functions where the line segment connecting any two points in the domain lies entirely within the set of the function's values
- **Convex Optimization**: A field of optimization that deals with convex optimization problems

### Notions and Definitions

- **Convex Cone**: A set of non-negative vectors that is closed and convex
- **Dual Problem**: A problem that involves finding the optimal solution to a given problem, subject to certain constraints
- **Karush-Kuhn-Tucker (KKT) Theorem**: A theorem that provides necessary and sufficient conditions for a point to be a local optimum of a convex optimization problem

### Important Formulas and Theorems

- **Fenchel-Conjugate**: A mathematical concept that provides a way to find the optimal solution to a convex optimization problem
- **Lagrange Multiplier**: A concept used to find the optimal solution to a convex optimization problem subject to constraints
- **Weierstrass Theorem**: A theorem that provides a necessary condition for a function to be convex
