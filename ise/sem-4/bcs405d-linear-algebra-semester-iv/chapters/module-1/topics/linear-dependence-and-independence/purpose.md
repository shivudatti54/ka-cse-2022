# Learning Objectives

After studying this topic, you should be able to:

1. Determine whether a vector is a linear combination of other vectors
2. Find the span of a set of vectors and determine what space it represents
3. Test whether a set of vectors is linearly independent or dependent using various methods
4. Understand the geometric interpretation of linear independence and span in ℝ² and ℝ³
5. Recognize the relationship between linear independence, span, and the concept of basis
6. Apply these concepts to solve problems involving vector spaces and systems of equations
