# Learning Objectives

After studying this topic, you should be able to:

1. Define and identify vector spaces by verifying the ten vector space axioms
2. Determine whether a given subset of a vector space constitutes a subspace using the subspace test
3. Recognize and work with various examples of vector spaces beyond ℝⁿ, including matrix spaces and function spaces
4. Understand and apply the concepts of null space and column space of a matrix
5. Determine the span of a set of vectors and recognize that spans always form subspaces
6. Analyze relationships between different subspaces of a vector space, including intersections and sums
