# Why Study Gradient Descent?

## Big Picture Foundation

Gradient descent is the core optimization algorithm in deep learning. Understanding its variants and hyperparameters is essential for effective training.

## Exam Relevance

| Aspect         | Details                                                   |
| -------------- | --------------------------------------------------------- |
| Frequency      | Core topic in every DL/ML exam                            |
| Marks          | 8-15 marks (comparisons, update rules, convergence)       |
| Question Types | Compare variants, explain momentum, analyze learning rate |

## Practical Value

- **Hyperparameter Tuning**: Set learning rate and batch size effectively
- **Debugging**: Understand why training is slow or unstable
- **Efficiency**: Choose right variant for your hardware

## Learning Outcomes

After this topic, you should be able to:

1. Explain the gradient descent update rule
2. Compare batch, stochastic, and mini-batch variants
3. Describe the effect of learning rate
4. Explain momentum and its benefits
