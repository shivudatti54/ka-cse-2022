# Why Learn Relational Model?

## Exam Relevance

- 15-20% of DBMS exam questions
- Foundation for all database topics
- Relational algebra often tested

## Key Exam Patterns

1. Terminology mapping (relation=table, etc.)
2. Write relational algebra expressions
3. Schema vs instance identification
4. Properties of relations

## Interview Questions

- "What is a relation in databases?"
- "Explain select vs project"
- "What is relational algebra?"

## Real-World Applications

- Database design fundamentals
- Query optimization understanding
- All SQL databases (MySQL, PostgreSQL, etc.)
- Foundation for NoSQL comparison
