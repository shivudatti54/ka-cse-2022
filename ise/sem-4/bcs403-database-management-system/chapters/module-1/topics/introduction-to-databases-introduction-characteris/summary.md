# **Introduction to Databases: Revision Notes**

### Introduction

- A database is a collection of organized data that is stored in a way that allows for efficient retrieval and manipulation.
- A database management system (DBMS) is a software that manages and controls the database.
- The term "database" was first coined by Edgar F. Codd in 1970.

### Characteristics of Database Approach

- **Data Independence**: The database is independent of the application programs that use it.
- **Control Independence**: The database is independent of the operating system and the hardware.
- **Security**: The database ensures that data is only accessible to authorized users.
- **Data Integrity**: The database ensures that data is consistent and accurate.

### Advantages of Using DBMS Approach

- **Data Abstraction**: The DBMS abstracts the underlying data and provides a higher level of abstraction.
- **Data Normalization**: The DBMS ensures that the data is in a normalized form, which reduces data redundancy and improves data integrity.
- **Data Retrieval**: The DBMS provides fast and efficient retrieval of data.
- **Data Manipulation**: The DBMS provides fast and efficient manipulation of data.

### History of Database Applications

- **1960s**: The first databases were developed in the 1960s, primarily for use in government and military organizations.
- **1970s**: The first commercial databases were developed in the 1970s.
- **1980s**: The first relational databases were developed in the 1980s, which revolutionized the field of database management.
- **1990s**: The internet and the web enabled the widespread use of databases.

**Formulas, Definitions, and Theorems**

- **Data Model**: A framework for organizing and representing data.
- **Entity-Relationship Model**: A data model that represents data as entities and relationships.
- **Normalization**: The process of organizing data in a database to minimize data redundancy and improve data integrity.
- **Data Consistency**: A property of a database that ensures that the data is consistent and accurate.

**Important Terms**

- **Database Management System (DBMS)**: A software that manages and controls the database.
- **Database Administrator (DBA)**: A person responsible for the maintenance and management of the database.
- **Database Query**: A request for data to be retrieved from the database.
- **Database Transaction**: A series of operations that are executed as a single, all-or-nothing unit.
