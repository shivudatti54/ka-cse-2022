# Learning Objectives

After studying this topic, you should be able to:

1. Construct a basic SELECT query to retrieve specific columns from a single table.
2. Apply the WHERE clause to filter query results based on one or more conditions.
3. Utilize the ORDER BY clause to sort query results in ascending or descending order.
4. Employ aggregate functions (COUNT, SUM, AVG, MIN, MAX) to calculate summary values from data.
5. Use the GROUP BY clause to arrange identical data into groups for summary analysis.
6. Differentiate between the WHERE and HAVING clauses and apply HAVING to filter groups based on aggregate conditions.
7. Combine all clauses (SELECT, FROM, WHERE, GROUP BY, HAVING, ORDER BY) to write complex queries that solve real-world data retrieval problems.
