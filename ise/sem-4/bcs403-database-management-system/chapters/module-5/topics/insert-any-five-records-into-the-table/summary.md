# **Inserting Records into a Table**

### Key Points

- **Inserting Records:**
  - Used to add new data to a table
  - Can be done using various methods (e.g., SQL INSERT statement)
- **Types of Inserts:**
  - **Single Insert:** Adding a single row to the table
  - **Bulk Insert:** Adding multiple rows to the table at once
- **Insert Constraints:**
  - **Primary Key (PK):** Unique identifier for each row
  - **Foreign Key (FK):** Links a table to another table

### Important Formulas and Definitions

- **Primary Key (PK):** A column or set of columns that uniquely identifies each row in a table.
- **Foreign Key (FK):** A column or set of columns that references the primary key of another table.
- **INSERT INTO:** A SQL statement used to add new data to a table.

### Important Theorems

- **Theorem:** In a relational database, a primary key is unique and non-null.
