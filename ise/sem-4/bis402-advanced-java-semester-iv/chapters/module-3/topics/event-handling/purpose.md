# Learning Objectives
After studying this topic, you should be able to:
1.  Explain the principles and advantages of the Delegation Event Model over older event handling models.
2.  Identify common event types (ActionEvent, MouseEvent, etc.) and their corresponding listener interfaces.
3.  Implement event listeners using anonymous inner classes, named inner classes, and by having a container class implement the listener interface.
4.  Utilize adapter classes (e.g., MouseAdapter) to simplify the implementation of listener interfaces with multiple methods.
5.  Register and unregister listener objects with Swing components to handle user interactions.
6.  Write a complete, functional Swing application that responds to user events such as button clicks and menu selections.
