# Learning Objectives

After studying this topic, you should be able to:

1.  Define key morphological terms such as morpheme, stem, affix, inflection, and derivation.
2.  Explain the role and importance of morphological parsing in the NLP pipeline.
3.  Design a simple Finite-State Automaton (FSA) to recognize a set of morphologically related words.
4.  Describe the function of a Finite-State Transducer (FST) and how it extends the capability of an FSA for morphological analysis and generation.
5.  Identify common orthographic rules (e.g., e-deletion, y-replacement) and provide examples of each.
6.  Discuss the specific challenges involved in morphological parsing of Indian languages and name a relevant framework (Paninian).
