# Learning Objectives

After studying this topic, you should be able to:

1. Explain the Branch and Bound technique and how it applies to the 0/1 Knapsack problem
2. Calculate upper bounds using the fractional knapsack relaxation method
3. Trace through the Branch and Bound algorithm for small problem instances
4. Implement the Branch and Bound algorithm for 0/1 Knapsack
5. Compare and contrast Branch and Bound with Dynamic Programming and Greedy approaches
6. Analyze when Branch and Bound is most appropriate for solving 0/1 Knapsack problems
