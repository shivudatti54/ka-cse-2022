# Chapter 2(Sections 2.1) Analysis & Design of Algorithms

### Key Concepts

- **Big O notation**:
  - Definition: Upper bound of time complexity
  - Notations: O(1), O(log n), O(n), O(n log n), O(n^2), O(2^n), ...
- **Time complexity**: Measure of algorithm's performance
- **Space complexity**: Measure of algorithm's memory usage
- **Big Ω notation**: Lower bound of time complexity
- **Big θ notation**: Tight bound of time complexity (both upper and lower bounds)

### Important Formulas and Definitions

- **Time complexity formulas**:
  - O(1) = constant time complexity
  - O(log n) = logarithmic time complexity
  - O(n) = linear time complexity
  - O(n log n) = linearithmic time complexity
  - O(n^2) = quadratic time complexity
  - O(2^n) = exponential time complexity
- **Space complexity formulas**:
  - O(1) = constant space complexity
  - O(n) = linear space complexity
- **Divide and Conquer**: Algorithmic technique for solving problems by breaking them down into smaller sub-problems

### Important Theorems

- **Master theorem**: Theorem used to solve recurrence relations for time complexity analysis
  - T(n) = aT(n/b) + f(n)
  - Where a, b, and f(n) are constants and the time complexity is T(n)
