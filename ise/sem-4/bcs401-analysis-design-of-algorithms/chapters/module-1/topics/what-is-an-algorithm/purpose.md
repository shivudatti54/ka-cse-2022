# Learning Objectives

After studying this topic, you should be able to:

1.  Define an IoT device and explain its role within the broader IoT ecosystem.
2.  Identify and describe the function of the four core hardware components of a typical IoT device.
3.  Differentiate between a microcontroller (MCU) and a microprocessor (MPU) and suggest appropriate use cases for each.
4.  Compare and contrast common IoT connectivity protocols (e.g., Wi-Fi, BLE, LoRaWAN) based on range, power, and data rate.
5.  Explain the importance of power management in IoT device design.
6.  Classify different types of IoT devices based on their primary function (e.g., sensing, actuation, gateway).
