## What is an Algorithm?

---

- An algorithm is a well-defined procedure that takes some input and produces a corresponding output.
- It is a set of instructions that is used to solve a specific problem or perform a particular task.
- An algorithm must be:
  - Well-defined: It must be clear and unambiguous.
  - Terminate: It must have a stop condition.
  - Finite: It must have a finite number of steps.
  - Correct: It must produce the correct output for a given input.

## Fundamentals of Algorithmic Problem Solving

---

- **Time Complexity**: The amount of time an algorithm takes to complete as a function of the input size.
- **Space Complexity**: The amount of memory an algorithm uses as a function of the input size.
- **Big O Notation**: A way to describe the upper bound of an algorithm's time or space complexity.

## Important Formulas and Definitions

---

- **Big O Notation**:
  - O(1) - constant time complexity
  - O(log n) - logarithmic time complexity
  - O(n) - linear time complexity
  - O(n log n) - linearithmic time complexity
  - O(n^2) - quadratic time complexity
  - O(2^n) - exponential time complexity
- **Recursion**:
  - A function that calls itself in its own definition.
  - Recursion is a technique used to solve problems that have a recursive structure.

## Important Theorems

---

- **Divide and Conquer**: A problem-solving strategy that breaks down a problem into smaller sub-problems.
- **Greedy Algorithm**: A simple algorithm that makes the locally optimal choice at each step.

## Key Terms

---

- **Algorithm**: A set of instructions that is used to solve a specific problem or perform a particular task.
- **Input**: The data that is used to control the algorithm.
- **Output**: The result produced by the algorithm.
- **Time Complexity**: The amount of time an algorithm takes to complete as a function of the input size.
- **Space Complexity**: The amount of memory an algorithm uses as a function of the input size.
