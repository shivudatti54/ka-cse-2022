# Chapter 8 (Sections 8.1) - Analysis & Design of Algorithms

===========================================================

### Key Concepts

- **Dynamic Programming**: A method for solving complex problems by breaking them down into smaller subproblems, solving each subproblem only once, and storing the solutions to subproblems to avoid redundant computation.
- **Optimal Substructure**: A property of problems that allows the optimal solution to be constructed from the optimal solutions of its subproblems.
- **Overlapping Subproblems**: A problem where the optimal solution to the overall problem depends on the optimal solutions of several subproblems that may overlap.

### Important Formulas, Definitions, and Theorems

- **Optimal Substructure Theorem**: If a problem has optimal substructure and the optimal solution to the overall problem can be constructed from the optimal solutions of its subproblems, then the problem can be solved using dynamic programming.
- **Memoization**: A technique used in dynamic programming to store the solutions to subproblems in a memory (e.g., an array or a table) to avoid redundant computation.
- **Bottom-up Dynamic Programming**: A approach to dynamic programming where the solution to the overall problem is constructed from the solutions of its subproblems, starting from the smallest subproblems.
- **Top-down Dynamic Programming**: A approach to dynamic programming where the solution to the overall problem is constructed from the solutions of its subproblems, starting from the largest subproblems.

### Important Examples

- **The Knapsack Problem**: A classic problem in computer science and operations research that involves finding the optimal way to pack a set of items of different weights and values into a knapsack of limited capacity.
- **The Fibonacci Series**: A classic example of an optimal substructure problem where the optimal solution to the overall problem depends on the optimal solutions of its subproblems.

### Key Takeaways

- Understand the concept of optimal substructure and overlapping subproblems.
- Learn how to use memoization and bottom-up dynamic programming to solve problems efficiently.
- Familiarize yourself with the top-down dynamic programming approach.
- Practice solving problems using dynamic programming techniques.
