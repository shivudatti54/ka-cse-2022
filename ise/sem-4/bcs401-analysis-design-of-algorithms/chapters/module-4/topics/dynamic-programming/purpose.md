# Learning Objectives

After studying this topic, you should be able to:

1.  Define the properties of optimal substructure and overlapping subproblems and identify them in a given problem.
2.  Differentiate between the top-down (memoization) and bottom-up (tabulation) approaches to Dynamic Programming, listing the advantages and disadvantages of each.
3.  Design a DP solution for a new problem by correctly defining the subproblems, formulating the recurrence relation, and identifying base cases.
4.  Implement a bottom-up DP solution for the 0/1 Knapsack and Longest Common Subsequence problems, analyzing their time and space complexity.
5.  Reconstruct the actual optimal solution (e.g., the items in the knapsack) from a completed DP table using backtracking.
6.  Contrast the applicability of Dynamic Programming with Greedy algorithms, explaining why DP is necessary for problems like the 0/1 Knapsack.
