# Learning Objectives

After studying this topic, you should be able to:

1.  Define what a GPIO pin is and explain its role as an interface between the microcontroller and external devices.
2.  Identify and describe the function of the key internal components of a GPIO pin (output drivers, Schmitt trigger, pull-up/pull-down resistors).
3.  Differentiate between the key GPIO modes: Digital Input, Digital Output (Push-Pull & Open-Drain), Analog Input, and Alternate Function.
4.  Explain the purpose and procedure for configuring the fundamental GPIO registers: Data Direction, Output, and Input.
5.  Analyze a simple circuit schematic and recommend appropriate GPIO mode and configuration (including the use of pull-up/pull-down resistors) for a given component (e.g., button, LED).
6.  Justify the use of Open-Drain mode for bus protocols like I²C and describe the requirement for an external pull-up resistor.
