# **Mixed Definitions, Cloud Capacity, Use it or lose it (your wallet), Measuring the clouds, Cloud Case Studies, Cloud Use Case: Anonymous Desktop Software**

### Definitions and Key Concepts

- **Cloud**: A model for delivering computed services over the internet.
- **Cloud Capacity**: The amount of resources (CPU, memory, storage, etc.) available in a cloud.
- **Mixed Definitions**: Clouds can be classified into different types (public, private, hybrid) and services (IaaS, PaaS, SaaS).

### Cloud Capacity

- **Cloud Capacity Planning**: The process of estimating and allocating cloud resources to meet business needs.
- **Cloud Scaling**: The ability to adjust cloud resources up or down to meet changing business needs.
- **Cloud Cost Optimization**: Minimizing cloud costs through efficient resource utilization and cost-effective cloud services.

### Measuring the Clouds

- **Cloud Consumption**: The amount of cloud resources used by a business.
- **Cloud Utilization**: The percentage of cloud resources used compared to their available capacity.
- **Cloud Efficiency**: The ratio of cloud resources used to their available capacity.

### Cloud Case Studies

- **Case Study 1: Amazon Web Services (AWS)**: A public cloud provider that offers a wide range of cloud services.
- **Case Study 2: Microsoft Azure**: A public cloud provider that offers a range of cloud services, including IaaS, PaaS, and SaaS.

### Cloud Use Case: Anonymous Desktop Software

- **Use Case**: A software application that allows users to access a cloud-based desktop from anywhere.
- **Benefits**: Increased flexibility, reduced costs, and improved collaboration.

### Important Formulas and Theorems

- **Cloud Cost Formula**: C = R x P x t, where C = cost, R = resources, P = price, and t = time.
- **Cloud Utilization Formula**: U = (R / C) x 100, where U = utilization, R = resources, and C = capacity.
- **Cloud Efficiency Formula**: E = (U / C) x 100, where E = efficiency, U = utilization, and C = capacity.
