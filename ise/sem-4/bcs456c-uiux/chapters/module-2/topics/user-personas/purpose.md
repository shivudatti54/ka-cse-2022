# Learning Objectives

After studying this topic, you should be able to:

1. Define and differentiate between personas and user stories.
2. Explain the purpose and value of creating personas in the user-centered design process.
3. Identify the key components of a well-constructed user persona.
4. Formulate a correct user story using the "As a... I want to... so that..." format.
5. Develop appropriate acceptance criteria for a given user story.
6. Evaluate the synergy between personas and user stories and how they collectively prevent self-referential design.
