# **UX Goals, Metrics, and Targets Revision Notes**

### Introduction

- UX Goals:
  - Define the purpose and objectives of a design project
  - Examples: Increase user engagement, Improve conversion rates, Enhance usability
- UX Measures:
  - Quantifiable outcomes that indicate progress towards UX goals
  - Can be used to evaluate the effectiveness of design decisions

### Measurement Instruments

- Surveys and questionnaires: tools used to gather user feedback and opinions
- Data analytics tools: software used to analyze user behavior and interactions
- usability testing: methods used to evaluate user experience through observation and feedback

### UX Metrics

- **Lifting the Veil Theorem**: "If you can't measure it, you can't design it"
- **Definition**: Quantifiable outcomes that indicate progress towards UX goals
- Examples:
  - User engagement metrics (e.g. time on site, bounce rate)
  - Conversion rate metrics (e.g. form submissions, sales)
  - Satisfaction metrics (e.g. Net Promoter Score, user satisfaction surveys)
- **Formulas:**
  - Conversion rate: (Number of successful outcomes / Total number of attempts) \* 100
  - User satisfaction: (Number of positive responses / Total number of responses) \* 100

### Important Formulas and Definitions

- **A/B Testing Formula**: p-value = (Null Hypothesis Probability - Alternative Hypothesis Probability)
- **Confidence Interval Formula**: CI = (x̄ - SE) ± (Z \* SE)
- **User Engagement Metrics:**
  - Average Session Duration (ASD)
  - Bounce Rate (BR)
  - Time on Site (TOS)

### Key Takeaways

- Clearly define UX goals and objectives
- Use measurements and metrics to evaluate design decisions
- Choose the right measurement instruments and tools
- Analyze and interpret data to inform design decisions
- Use formulas and theorems to calculate and understand metrics.
