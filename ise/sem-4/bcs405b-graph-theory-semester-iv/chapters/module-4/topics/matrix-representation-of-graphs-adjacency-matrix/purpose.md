# Learning Objectives

After studying this topic, you should be able to:

1.  Define the matrix representation of a linear transformation with respect to given bases.
2.  Construct the standard matrix for a linear transformation between Rⁿ and Rᵐ.
3.  Construct the matrix representation of a linear transformation with respect to any arbitrary bases of vector spaces.
4.  Use the matrix representation to compute the image of a vector under a transformation.
5.  Relate the composition of linear transformations to the multiplication of their matrix representations.
6.  Recognize common geometric transformations (rotation, reflection, projection) from their standard matrices.
