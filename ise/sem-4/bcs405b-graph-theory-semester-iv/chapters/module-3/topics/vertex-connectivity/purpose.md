# Learning Objectives

After studying this topic, you should be able to:

1. Define and distinguish between different types of paths (walk, trail, path, cycle)
2. Determine if a graph is connected and identify its connected components
3. Apply Euler's theorem to determine if a graph has an Eulerian path or circuit
4. Understand the concept of Hamiltonian paths and their computational complexity
5. Calculate vertex and edge connectivity of graphs
6. Apply BFS and DFS algorithms to solve connectivity problems
7. Distinguish between strong and weak connectivity in directed graphs
8. Find shortest paths and calculate distance metrics in graphs
