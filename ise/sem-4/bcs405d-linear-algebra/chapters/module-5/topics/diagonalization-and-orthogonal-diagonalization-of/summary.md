# Diagonalization and Orthogonal Diagonalization of Real Symmetric Matrices

## Key Concepts

- **Definition:** A real symmetric matrix is a square matrix that is equal to its transpose.
- **Diagonalization:** A process of transforming a matrix into a diagonal matrix using an orthogonal matrix.
- **Orthogonal Diagonalization:** A specific type of diagonalization where the orthogonal matrix is also orthogonal.

### Key Formulas

- **Diagonalization:** A real symmetric matrix A can be diagonalized as A = PDP^T, where P is an orthogonal matrix and D is a diagonal matrix.
- **Orthogonal Diagonalization:** A real symmetric matrix A can be orthogonally diagonalized as A = QDQ^T, where Q is an orthogonal matrix and D is a diagonal matrix.

### Key Theorems

- **Spectral Theorem:** A real symmetric matrix A can be diagonalized by an orthogonal matrix.
- **Orthogonal Diagonalization Theorem:** A real symmetric matrix A can be orthogonally diagonalized.

### Quadratic Forms and Classifications

- **Definition:** A quadratic form is a polynomial expression of the form Ax^T x, where A is a symmetric matrix.
- **Classifications:** Quadratic forms can be classified into three types:
  - Positive definite
  - Positive semi-definite
  - Negative semi-definite

### Hessian Matrix

- **Definition:** The Hessian matrix is the matrix of second partial derivatives of a function.
- **Key Property:** The Hessian matrix is symmetric.

### Method of Steepest Ascent

- **Definition:** The method of steepest ascent is an optimization technique used to find the maximum of a quadratic function.
- **Key Formula:** The method of steepest ascent updates the current estimate of the maximum as x_new = x_old + α \* ∇f(x_old), where α is a step size and ∇f(x_old) is the gradient of the function at x_old.

### Important Definitions

- **Eigenvalue:** A scalar value that represents how much a linear transformation stretches or shrinks a vector.
- **Eigenvector:** A non-zero vector that, when transformed by a linear transformation, results in a scaled version of itself.
- **Orthogonal Matrix:** A matrix whose rows and columns are orthonormal vectors.
- **Symmetric Matrix:** A matrix that is equal to its transpose.
