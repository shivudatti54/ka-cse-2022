# Learning Objectives

After studying this topic, you should be able to:

1. **Explain** the five values of Extreme Programming
2. **Describe** key XP practices including TDD and pair programming
3. **Apply** the Test-Driven Development cycle
4. **Write** user stories following the INVEST criteria
5. **Identify** when XP is appropriate for a project
6. **Understand** the roles in an XP team
