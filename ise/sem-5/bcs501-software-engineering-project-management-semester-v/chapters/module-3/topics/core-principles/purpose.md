# Learning Objectives

After studying this topic, you should be able to:

1.  Define and explain each component of the CIA Triad (Confidentiality, Integrity, Availability) with relevant examples.
2.  Distinguish between related security principles such as authentication, authorization, and non-repudiation.
3.  Articulate the rationale and benefits of a Defense in Depth (layered defense) security strategy.
4.  Identify and categorize common security controls into their appropriate Defense in Depth layer (e.g., physical, network, data).
5.  Analyze a simple scenario and recommend security controls based on both the CIA Triad and a Defense in Depth approach.
6.  Explain how the principles of least privilege and accountability support core security goals.
