# **1.4 Chapter 2 - 2.1: Artificial Intelligence (AI) Revision Notes**

### Definitions and Notations

- **Artificial Intelligence (AI):** The development of computer systems able to perform tasks that typically require human intelligence, such as learning, problem-solving, and decision-making.
- **Machine Learning (ML):** A subset of AI that involves training algorithms to learn from data and make predictions or decisions.
- **Deep Learning (DL):** A type of ML that uses neural networks with multiple layers to learn complex patterns in data.
- **α (alpha):** The learning rate of an algorithm, which determines how quickly the model learns from the data.
- **β (beta):** The regularization parameter of an algorithm, which helps prevent overfitting.

### Key Concepts

- **Supervised Learning:** A type of ML where the algorithm is trained on labeled data to learn a mapping between inputs and outputs.
- **Unsupervised Learning:** A type of ML where the algorithm is trained on unlabeled data to discover patterns or structure.
- **Reinforcement Learning:** A type of ML where the algorithm learns through trial and error by interacting with an environment.

### Formulas and Theorems

- **Bayes' Theorem:** P(A|B) = P(A ∩ B) / P(B), where P(A|B) is the probability of A given B, and P(A ∩ B) is the probability of A and B.
- **Markov Chain:** A mathematical system that undergoes transitions from one state to another, where the probability of transitioning from one state to another is dependent on the current state.
- **Gradient Descent:** An optimization algorithm used to minimize the loss function of a model, which updates the model's parameters based on the gradient of the loss function.

### Important Theorems

- **No Free Lunch Theorem:** No optimization algorithm can guarantee the optimal solution for all problems, and the best algorithm for one problem may be poor for another.
- **Vapnik-Chervonenkis (VC) Theorem:** A theorem that provides a lower bound on the capacity of a classifier, which is the maximum number of classes it can correctly classify.
