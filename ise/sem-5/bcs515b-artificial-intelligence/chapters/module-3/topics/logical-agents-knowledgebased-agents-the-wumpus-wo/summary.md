# Logical Agents: Knowledge-based Agents

### Key Concepts

- **Knowledge-based agents**: Agents that use knowledge to make decisions and solve problems.
- **The Wumpus world**: A classic problem domain used to test reasoning and decision-making abilities.
- **Propositional logic**: A branch of logic that deals with statements that can be either true or false.

### Propositional Logic

- **Propositions**: Statements that can be either true (T) or false (F).
- **Logical operators**:
  - Conjunction (∧): AND
  - Disjunction (∨): OR
  - Negation (¬): NOT
  - Implication (→): IF-THEN
- **Tautologies**: Statements that are always true.
- **Contradictions**: Statements that are always false.
- **Valid arguments**: Arguments that always lead to a true conclusion.

### Reasoning Patterns in Propositional Logic

- **Deductive reasoning**: Drawing conclusions from premises using logical operators.
- **Inductive reasoning**: Making generalizations based on specific observations.

### The Wumpus World

- **Problemdomain**: A 20x20 grid with a wumpus, pits, and exits.
- **Goals**: Reach the exit while avoiding the wumpus and pitfalls.
- **State space**: The set of all possible grid cells.
- **Actions**: Move up, down, left, right, or stay.

### Important Formulas and Definitions

- **Modus Ponens**: If A → B and A, then B.
- **Modus Tollens**: If A → B and not B, then not A.
- **De Morgan's Law**: ¬(A ∧ B) = ¬A ∨ ¬B and ¬(A ∨ B) = ¬A ∧ ¬B.
- **Dempster-Shafer Theory**: A mathematical framework for reasoning under uncertainty.

### Important Theorems

- **Law of Excluded Middle**: A statement is either true or false.
- **Law of Non-Contradiction**: A statement cannot be both true and false at the same time.
- **Principle of Explosion**: From a contradiction, we can infer anything.
