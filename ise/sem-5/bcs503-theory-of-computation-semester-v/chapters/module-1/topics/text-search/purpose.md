# Learning Objectives

After studying this topic, you should be able to:

1. Explain the primary function of a text search application.
2. Describe the different types of finite automata used to model the text search application.
3. Define the purpose of epsilon-transitions in finite automata.
4. Compare and contrast DFA and NFA.
5. Explain the purpose of the transition function in a finite automaton.
6. Design a finite automaton to model a given text search application.
