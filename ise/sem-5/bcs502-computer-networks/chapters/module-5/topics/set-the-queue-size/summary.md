# Set the Queue Size

### Definition

- Queue: a data structure that follows the FIFO (First-In-First-Out) principle
- Queue size: the number of elements in a queue

### Theorem: Little's Law

- L = λW
- L = average queue length
- λ = arrival rate
- W = average waiting time

### Important Formulas

- Queue size (q): q = λt
- Average queue length (L): L = λW
- Average waiting time (W): W = q/λ

### Set the Queue Size

- Setting the queue size determines the maximum number of elements a queue can hold
- It affects the average waiting time and average queue length
- A higher queue size can lead to longer waiting times and longer queues

### Key Points

- Understand the relationship between queue size, arrival rate, and waiting time
- Recognize the impact of queue size on average queue length and waiting time
- Apply Little's Law to analyze queue performance
- Set the queue size based on network requirements and constraints
