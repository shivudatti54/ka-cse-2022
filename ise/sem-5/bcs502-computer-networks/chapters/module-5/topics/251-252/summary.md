# **Computer Networks Revision Notes**

### 25.1: Network Topology and Network Architecture

- **Definition:** The physical and logical arrangement of devices in a network.
- **Types of Topology:**
  - Bus Topology
  - Star Topology
  - Ring Topology
  - Mesh Topology
- **Network Architecture:**
  - LAN (Local Area Network)
  - WAN (Wide Area Network)
  - MAN (Metropolitan Area Network)
  - Wireless Network

### 25.2: Network Devices and Media

- **Network Devices:**
  - Router
  - Switch
  - Bridge
  - Modem
  - Hub
- **Network Media:**
  - Coaxial Cable
  - Twisted-Pair Cable
  - Fiber Optic Cable
  - Wireless Network

**Important Formulas and Theorems:**

- **Ethernet Frame Format:** `Type | Subtype | Length | Source MAC | Destination MAC | Data`
- **IP Addressing:**
  - IPv4: `A.B.C.D`
  - IPv6: `A:A:A:A:A:A:A:A`
  - Subnet Mask: `/w` (e.g. `/24`)
- **Packet Switching:**
  - Theorem: "Store-and-Forward"

**Key Terms:**

- **Packet:** A small unit of data transmitted over a network.
- **Network Protocol:** A set of rules for communication between devices on a network.
- **Network Interface Card (NIC):** A hardware device that connects a computer to a network.
