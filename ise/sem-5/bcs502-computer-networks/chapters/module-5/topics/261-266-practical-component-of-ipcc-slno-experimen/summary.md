# **Practical Component of IPCC: 26.1-26.6**

## **Experiment 1: Implementing a Point-to-Point Network**

### Key Points

- **Point-to-Point Network**: A network that connects two devices directly using a single logical link.
- **Duplex Links**: Links that allow data to be transmitted in both directions simultaneously (bidirectional).
- **Key Components**:
  - **Nodes** (Devices): Two nodes are required for a point-to-point network.
  - **Links** (Connections): Duplex links are used to connect the nodes.
  - **Transceivers**: Devices that transmit and receive data on the links.
- **Network Topology**: Linear, as there are only two nodes.

### Important Formulas, Definitions, and Theorems

- **Bit Rate**: The rate at which data is transmitted (measured in bits per second, bps).
- **Bandwidth**: The amount of data that can be transmitted over a link per unit time (measured in bits per second, bps).
- **Throughput**: The rate at which data can be transmitted over a network (measured in bits per second, bps).

### Quick Revision Checklist

- Understand the definition of a point-to-point network and duplex links.
- Identify the key components of a point-to-point network (nodes, links, transceivers).
- Recall the network topology of a point-to-point network (linear).
- Be able to calculate bit rate, bandwidth, and throughput.
