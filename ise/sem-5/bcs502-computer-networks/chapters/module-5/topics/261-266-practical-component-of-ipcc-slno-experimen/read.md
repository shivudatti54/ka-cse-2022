## **Experiment 1: Implementing a Three-Node Point-to-Point Network with Duplex Links**

### Introduction

---

In this experiment, we will set up a point-to-point network with three nodes and duplex links between them. We will explore the concept of duplex links and their applications in computer networks.

### Definition of Duplex Links

---

A duplex link is a communication link that allows data to be transmitted in both directions, i.e., from the sender to the receiver and from the receiver to the sender. This allows for simultaneous transmission and reception of data, increasing the overall bandwidth and efficiency of the network.

### Components of the Network

---

For this experiment, we will use the following components:

- **Node 1**: The first node will act as the sender and receiver of data.
- **Node 2**: The second node will act as the receiver of data from Node 1 and the sender of data to Node 3.
- **Node 3**: The third node will act as the receiver of data from Node 2.

### Setting Up the Network

---

To set up the network, we will use the following steps:

1.  Connect Node 1 and Node 2 using a duplex link.
2.  Connect Node 2 and Node 3 using a duplex link.
3.  Configure the nodes to communicate with each other.

### Key Concepts

---

- **Duplex links**: Communication links that allow data to be transmitted in both directions.
- **Point-to-point networks**: Networks that connect two devices directly.
- **Node**: A device that can act as a sender, receiver, or both in a network.

### Practical Exercise

---

1.  Connect Node 1 and Node 2 using a duplex link.
2.  Connect Node 2 and Node 3 using a duplex link.
3.  Configure the nodes to communicate with each other.
4.  Test the network by sending data from Node 1 to Node 3.

### Troubleshooting

---

- **Error 1**: If data is not being transmitted, check the duplex links for any damage or issues.
- **Error 2**: If data is being transmitted, but not receiving, check the configuration of the nodes.

---
