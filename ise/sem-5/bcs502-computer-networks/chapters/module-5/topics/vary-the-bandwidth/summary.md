# **Vary the Bandwidth**

### Definition

- Bandwidth: The rate at which data is transmitted over a network
- Variating bandwidth: Adjusting the data transmission rate to match the available network resources

### Key Points

- Types of bandwidth:
  - Upstream bandwidth: Data sent from the user to the network
  - Downstream bandwidth: Data sent from the network to the user
- Bandwidth limitations:
  - Physical constraints (e.g., cable length, noise)
  - Network congestion
- Bandwidth allocation:
  - Fixed bandwidth allocation (e.g., DSL)
  - Dynamic bandwidth allocation (e.g., Wi-Fi)

### Formulas

- Bandwidth (B) = Data rate (R) x Time (t)
- Bandwidth theorem: The bandwidth of a network is inversely proportional to the latency of the network

### Theorems

- Shannon-Hartley theorem: The maximum data rate (R) that can be transmitted over a communication channel is limited by the bandwidth (B) and the signal-to-noise ratio (SNR)

### Important Concepts

- Quality of Service (QoS): Ensuring that different applications receive sufficient bandwidth and priority
- Bandwidth management: Techniques for managing bandwidth usage and avoiding congestion

### Key Terms

- Latency: The delay between the time data is sent and the time it is received
- Signal-to-noise ratio (SNR): The ratio of the desired signal to the unwanted noise in a communication channel
