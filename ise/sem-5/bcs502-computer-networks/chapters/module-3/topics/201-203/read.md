# \*\*20.1 Network Layer Services

### Overview

The network layer is the fourth layer of the OSI model and is responsible for providing different network services. These services enable devices on different networks to communicate with each other.

### Network Layer Services

- **Routing**: The process of forwarding packets between different networks.
- **Fragmentation**: Breaking down large packets into smaller packets that can be transmitted over networks with different maximum transmission unit (MTU).
- **Reassembly**: Reassembling fragmented packets at the receiving end.

### Internetworking

Internetworking is the process of connecting multiple networks together to form a single network. This is achieved through the use of routers and gateways.

### Network Addressing

Network addressing is the process of assigning a unique address to a device on a network.

### IPv4 Addressing

IPv4 (Internet Protocol version 4) is the original version of the internet protocol used for addressing devices on a network.

- **IPv4 Address Format**: 32-bit address consisting of four octets separated by periods (e.g., 192.168.1.1)
- **IPv4 Address Classes**: A, B, C, D, and E
- **Network ID**: The first 8 bits of the IPv4 address
- **Host ID**: The last 24 bits of the IPv4 address

### IPv4 Datagram

An IPv4 datagram is a packet that contains a logical address (IP address) and a physical address.

- **Source IP Address**: The IP address of the sender.
- **Destination IP Address**: The IP address of the intended recipient.
- **Protocol**: The protocol used to transmit the data (e.g., TCP, UDP).
- **Checksum**: A value used to detect errors during transmission.

### IPv6 Addressing

IPv6 (Internet Protocol version 6) is the latest version of the internet protocol used for addressing devices on a network.

- **IPv6 Address Format**: 128-bit address consisting of 8 groups of 4 hexadecimal digits separated by colons (e.g., 2001:0db8:85a3:0000:0000:8a2e:0370:7334)
- **IPv6 Address Classes**: Not used in IPv6
- **Network ID**: Not applicable in IPv6
- **Host ID**: Applies to all IPv6 addresses.
