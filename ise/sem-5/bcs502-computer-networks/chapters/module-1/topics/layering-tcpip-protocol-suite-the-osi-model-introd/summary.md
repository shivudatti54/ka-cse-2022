# **Revision Notes: Layering, TCP/IP Protocol Suite, OSI Model, Physical Layer**

### Layering

- **Definition:** The process of dividing data into layers to facilitate communication between devices.
- **Key Layers:**
  - Physical (Physical Layer)
  - Data Link (Data Link Layer)
  - Network (Network Layer)
  - Transport (Transport Layer)
  - Session (Session Layer)
  - Presentation (Presentation Layer)
  - Application (Application Layer)
- **TCP/IP Model:**
  - OSI Model: 7 layers
  - TCP/IP Model: 4 layers (Transport, Internet, Application, Presentation)

### TCP/IP Protocol Suite

- **Definition:** A set of protocols used for communication over the internet.
- **Key Protocols:**
  - TCP (Transmission Control Protocol)
  - IP (Internet Protocol)
  - UDP (User Datagram Protocol)
  - ICMP (Internet Control Message Protocol)
- **TCP/IP Model:**
  - Transport Layer: TCP
  - Internet Layer: IP

### The OSI Model

- **Definition:** A 7-layered model for understanding network communication.
- **Layers:**
  1.  Physical Layer
  2.  Data Link Layer
  3.  Network Layer
  4.  Transport Layer
  5.  Session Layer
  6.  Presentation Layer
  7.  Application Layer
- **Theorems:**
  - "A message is sent over the internet when a packet is sent over the network."

### Introduction to Physical Layer: Transmission Media

- **Definition:** The medium through which data is transmitted.
- **Types of Transmission Media:**
  - **Guided Media:**
    - Coaxial cables
    - Twisted pair cables
  - **Unguided Media:**
    - Wireless
    - Fiber optic cables

### Important Formulas and Definitions

- **Bit Rate:** The rate at which data is transmitted (bps)
- **Bandwidth:** The amount of data that can be transmitted per unit of time (bps)
- **Data Rate:** The rate at which data is transmitted (bps)
- **Packet Size:** The size of a data packet (bytes)
