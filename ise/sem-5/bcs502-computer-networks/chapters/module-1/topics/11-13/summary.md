# **Computer Networks Revision Notes**

### 1.1 - Introduction to Computer Networks

- A computer network is a collection of interconnected computers that communicate with each other.
- The primary function of a computer network is to facilitate the exchange of information between devices.
- Examples of computer networks include LANs (Local Area Networks), WANs (Wide Area Networks), and the Internet.

### 1.2 - Network Topology

- **Definition**: The physical and logical arrangement of devices in a network.
- **Types of Topology**:
  - Bus Topology: devices are connected to a single backbone cable.
  - Star Topology: devices are connected to a central hub or switch.
  - Ring Topology: devices are connected in a circular configuration.
  - Mesh Topology: each device is connected to every other device.

### 1.3 - Network Models

- **Definition**: A representation of a network that shows its components and relationships.
- **Types of Network Models**:
  - **Physical Model**: a representation of the physical layout of a network.
  - **Logical Model**: a representation of a network's functions and relationships.

**Important Formulas and Definitions**

- **Packet Switching**: a method of transmitting data in small packets between devices.
- **Network Address**: a unique address assigned to each device in a network.
- **Subnet Mask**: a binary number used to divide an IP address into network and host parts.

**Key Theorems**

- **Theorem of the Network**: states that the number of possible connections between two devices is equal to the number of devices multiplied by the number of devices minus one.
- **Theorem of Network Topology**: states that the number of possible topologies for a given number of devices is equal to the number of possible arrangements of the devices.
