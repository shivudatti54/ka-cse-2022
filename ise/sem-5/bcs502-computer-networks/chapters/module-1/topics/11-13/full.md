# 1.1 - 1.3: Introduction to Computer Networks

=====================================================

## 1.1: Introduction to Computer Networks

---

A computer network is a collection of interconnected devices that can communicate with each other to share resources and exchange data. The term "computer network" can be broadly defined as a set of devices, such as computers, printers, and servers, that are connected together to achieve a common goal.

The concept of a computer network has been around for decades, with the first networks emerging in the 1960s. In the early days, networks were primarily used for government and research purposes, but with the advent of the internet, networks became more widespread and accessible to the general public.

### Key Characteristics of Computer Networks

Before we dive deeper into the topic, let's identify some key characteristics of computer networks:

- **Interconnectedness**: A computer network is a collection of interconnected devices that can communicate with each other.
- **Resource sharing**: Computer networks enable devices to share resources such as files, printers, and internet connections.
- **Data exchange**: Computer networks allow devices to exchange data, such as emails, instant messages, and files.
- **Scalability**: Computer networks can be scaled up or down to meet the needs of the users.

### Types of Computer Networks

There are several types of computer networks, including:

- **Local Area Network (LAN)**: A LAN is a network that spans a small geographical area, such as a home, office building, or campus.
- **Wide Area Network (WAN)**: A WAN is a network that spans a large geographical area, such as a city, state, or country.
- **Metropolitan Area Network (MAN)**: A MAN is a network that spans a metropolitan area, such as a city or town.
- **Wireless Network (WLAN)**: A WLAN is a network that uses wireless communication to connect devices.

### Network Topology

Network topology refers to the physical and logical arrangement of devices in a network. There are several types of network topologies, including:

- **Bus Topology**: A bus topology is a network in which all devices are connected to a single cable or backbone.
- **Star Topology**: A star topology is a network in which all devices are connected to a central device or hub.
- **Ring Topology**: A ring topology is a network in which devices are connected in a circular configuration.
- **Mesh Topology**: A mesh topology is a network in which each device is connected to every other device.

## 1.2: Network Models

---

Network models are used to describe the structure and behavior of a network. There are several network models, including:

### OSI Model

The OSI model is a seven-layered model that describes the communication process between devices in a network. The seven layers of the OSI model are:

1.  **Physical Layer (Layer 1)**: The physical layer defines the physical means of data transmission, such as cable specifications and wireless communication protocols.
2.  **Data Link Layer (Layer 2)**: The data link layer provides error-free transfer of data frames between devices on the same network.
3.  **Network Layer (Layer 3)**: The network layer provides logical addressing and routing of data between devices on different networks.
4.  **Transport Layer (Layer 4)**: The transport layer provides reliable data transfer between devices, including error detection and correction.
5.  **Session Layer (Layer 5)**: The session layer establishes, maintains, and terminates connections between applications.
6.  **Presentation Layer (Layer 6)**: The presentation layer converts data into a format that can be understood by the receiving device.
7.  **Application Layer (Layer 7)**: The application layer provides services to end-user applications, such as email and file transfer.

### TCP/IP Model

The TCP/IP model is a four-layered model that describes the communication process between devices in a network. The four layers of the TCP/IP model are:

1.  **Network Access Layer**: The network access layer provides access to the network, including wireless communication protocols and cable specifications.
2.  **Internet Layer**: The internet layer provides logical addressing and routing of data between devices on different networks.
3.  **Transport Layer**: The transport layer provides reliable data transfer between devices, including error detection and correction.
4.  **Application Layer**: The application layer provides services to end-user applications, such as email and file transfer.

### DNS Model

The DNS model is a hierarchical model that describes the structure of the Domain Name System (DNS). The DNS model consists of three levels:

1.  **Top-Level Domain (TLD)**: The TLD is the highest level of the DNS hierarchy, and it includes the top-level domains such as .com and .org.
2.  **Second-Level Domain (SLD)**: The SLD is a level below the TLD, and it includes the second-level domains such as google.com.
3.  **Domain Name**: The domain name is the lowest level of the DNS hierarchy, and it includes the domain names such as www.google.com.

## 1.3: Network Protocols

---

Network protocols are used to govern the communication process between devices in a network. There are several network protocols, including:

### TCP/IP Protocols

TCP/IP protocols are used to govern the communication process between devices in a network. Some of the most common TCP/IP protocols include:

- **TCP (Transmission Control Protocol)**: TCP is a protocol that provides reliable, connection-oriented data transfer between devices.
- **UDP (User Datagram Protocol)**: UDP is a protocol that provides unreliable, connectionless data transfer between devices.
- **HTTP (Hypertext Transfer Protocol)**: HTTP is a protocol that governs the communication process between web servers and clients.
- **FTP (File Transfer Protocol)**: FTP is a protocol that governs the transfer of files between devices.

### Network Layer Protocols

Network layer protocols are used to govern the communication process between devices on different networks. Some of the most common network layer protocols include:

- **IP (Internet Protocol)**: IP is a protocol that governs the communication process between devices on different networks.
- **ICMP (Internet Control Message Protocol)**: ICMP is a protocol that governs the communication process between devices on different networks.
- **IGMP (Internet Group Management Protocol)**: IGMP is a protocol that governs the communication process between devices on different networks.

### Data Link Layer Protocols

Data link layer protocols are used to govern the communication process between devices on the same network. Some of the most common data link layer protocols include:

- **Ethernet**: Ethernet is a protocol that governs the communication process between devices on the same network.
- **Wi-Fi**: Wi-Fi is a protocol that governs the communication process between devices on the same network.

### Physical Layer Protocols

Physical layer protocols are used to govern the physical means of data transmission between devices. Some of the most common physical layer protocols include:

- **RS-232**: RS-232 is a protocol that governs the physical means of data transmission between devices.
- **Ethernet**: Ethernet is a protocol that governs the physical means of data transmission between devices.
