# **7.1 – 7.3 Revision Notes: Computer Networks**

### 7.1 Network Models

- **Physical Layer**: Defines the physical means of transmitting data between devices, including cable types, wireless standards, and network topologies.
- **Data Link Layer**: Ensures error-free transfer of data frames between two devices on the same network.
- **Network Layer**: Routes data between devices on different networks.
- **Transport Layer**: Provides reliable data transfer between devices, including congestion control and flow control.
- **Session Layer**: Establishes, manages, and terminates connections between applications.
- **Presentation Layer**: Converts data into a format that can be understood by the receiving device.
- **Application Layer**: Provides services to end-user applications, including email, file transfer, and web browsing.

### 7.2 Network Protocols

- **OSI Model**: A 7-layered model that defines the functions of each layer, including physical, data link, network, transport, session, presentation, and application layers.
- **TCP/IP Model**: A 4-layered model that defines the functions of each layer, including network access, internet, transport, and application layers.
- **HTTP/HTTPS**: Hypertext transfer protocols used for web browsing.
- **SMTP**: Simple mail transfer protocol used for email.
- **FTP**: File transfer protocol used for transferring files.

### 7.3 Network Topologies

- **Bus Topology**: A single cable connects all devices, with each device connected to the cable.
- **Star Topology**: All devices connect to a central device, called a hub or switch.
- **Ring Topology**: Devices connect in a circular configuration, with data transmitted in one direction around the ring.
- **Mesh Topology**: Each device connects to every other device, providing multiple paths for data transmission.

### Important Formulas and Theorems

- **Packet Switching Theorem**: States that a packet of data can be routed between two devices on different networks through a series of hops.
- **Error-Correcting Coding (ECC)**: A technique used to detect and correct errors that occur during data transmission.

### Important Definitions

- **Network Interface Card (NIC)**: A hardware component that connects a device to a network.
- **Router**: A device that connects multiple networks together and routes data between them.
- **Switch**: A device that connects multiple devices within a network and forwards data between them.
