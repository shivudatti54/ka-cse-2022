### Revision Notes: Computer Networks - Chapter Ch

#### Key Concepts

- **Data Communications**: The process of sending and receiving data between devices.
- **Network Types**: Local Area Network (LAN), Wide Area Network (WAN), Metropolitan Area Network (MAN), Wireless Network (WLAN), and Internet.
- **Network Models**: Physical, Logical, and Transport Models.

#### Definitions

- **Protocol**: A set of rules and standards for communication between devices.
- **Socket**: An endpoint of a connection between two devices.
- **IP Address**: A unique address assigned to each device on a network.
- **MAC Address**: A unique address assigned to each network interface.

#### Important Formulas and Theorems

- **Packet Switching**: Data is transmitted in packets, each with a header containing destination address.
- **Store-and-Forward**: Data is stored at intermediate nodes before being forwarded to the destination.
- **Layered Model**:
  - Physical Layer (Layer 1): Defines physical connections and signaling.
  - Data Link Layer (Layer 2): Ensures error-free transfer of data frames.
  - Network Layer (Layer 3): Routes data between networks.
  - Transport Layer (Layer 4): Provides reliable data transfer.
  - Session Layer (Layer 5): Establishes, maintains, and terminates connections.
  - Presentation Layer (Layer 6): Converts data into a format compatible with the receiving device.
  - Application Layer (Layer 7): Provides services to end-user applications.

#### Key Terms

- **ARP** (Address Resolution Protocol): Resolves IP addresses to MAC addresses.
- **DNS** (Domain Name System): Maps domain names to IP addresses.
- **TCP/IP** (Transmission Control Protocol/Internet Protocol): A suite of protocols used for communication over the internet.
