# **Computer Networks Revision Notes (2.1 - 2.3)**

## **Section 2.1: Data Communications**

- **Definition:** The process of transmitting data from one device to another.
- **Key Concepts:**
  - Data transmission media (wired, wireless, fiber optics)
  - Data transmission technologies (modulation, encoding, decoding)
  - Data transmission protocols (TCP, UDP, HTTP, FTP)
- **Important Formulas/Equations:**
  - Bandwidth = Data Rate x Time
  - Bit Rate = Number of bits transmitted per second
- **Important Theorems:**
  - Shannon-Hartley Theorem (Channel Capacity)

## **Section 2.2: Network Types**

- **Definition:** A network is a collection of interconnected devices.
- **Types of Networks:**
  - Local Area Network (LAN)
  - Metropolitan Area Network (MAN)
  - Wide Area Network (WAN)
  - Wireless Network (WLAN)
- **Characteristics:**
  - LAN: Small, low-cost, fast data transfer rates
  - MAN: Medium-sized, medium-cost, medium data transfer rates
  - WAN: Large, high-cost, slow data transfer rates
  - WLAN: Portable, low-cost, medium data transfer rates

## **Section 2.3: Network Models**

- **Definition:** A model represents a network's structure and components.
- **Network Models:**
  - Physical Model
  - Logical Model
  - Functional Model
- **Characteristics:**
  - Physical Model: Shows the physical components and connections of the network.
  - Logical Model: Shows the network's logical structure and components.
  - Functional Model: Shows the network's functional components and interactions.
