## Revision Notes: 11.1 - 11.4 - Data Link Layer

=====================================================

### Introduction

- The Data Link Layer is the second layer of the OSI model.
- It provides a means to detect and correct errors in data transmission.

### Key Concepts

- **Block Coding**: a method of error detection and correction.
- **Cyclic Codes**: a type of block code used in error detection and correction.

### 11.1: Error Detection and Correction

- **Error Detection**: the process of identifying errors in data transmission.
- **Error Correction**: the process of correcting errors in data transmission.

### 11.2: Block Coding

- **Definition**: a method of dividing data into blocks to detect and correct errors.
- **Block Code**: a set of codewords used to detect and correct errors.
- **Hamming Code**: a common type of block code used in error detection and correction.

### 11.3: Cyclic Codes

- **Definition**: a type of block code used in error detection and correction.
- **Properties**: cyclic, linear, and irreducible.
- **Generator Polynomial**: a polynomial used to generate cyclic codes.

### 11.4: Data Link Layer Error Correction

- **Forward Error Correction (FEC)**: a method of detecting and correcting errors in data transmission.
- **Check Digit**: a digital code used to detect errors in data transmission.

### Important Formulas and Definitions

- **Hamming Distance**: the number of positions at which two codewords differ.
- **Hamming Weight**: the number of 1's in a binary number.
- **Error-Correcting Code (ECC)**: a code used to detect and correct errors in data transmission.

### Key Theorems

- **Shannon-Fano Theorem**: a theorem used to determine the minimum number of bits required to encode a message.
- **Huffman Coding Theorem**: a theorem used to determine the minimum number of bits required to encode a message.
