# **Controlled Access**

### Overview

Controlled Access is a method of ensuring that only authorized devices can access a network, preventing unauthorized use and collisions.

### Key Points

- **Definition:** Controlled Access is a technique used in wireless networks to limit the access to the network, preventing unauthorized devices from transmitting data.
- **Types:**
  - Time Division Multiple Access (TDMA)
  - Frequency Division Multiple Access (FDMA)
  - Code Division Multiple Access (CDMA)
  - Orthogonal Frequency Division Multiple Access (OFDMA)
- **Advantages:**
  - Reduces interference and collisions
  - Increases network security
  - Supports multiple users
- **Disadvantages:**
  - Requires complex hardware and software
  - Limited flexibility

### Formulas and Definitions

- **Traffic Channel Capacity:** C = B x T, where C is traffic channel capacity, B is bandwidth, and T is transmission time.
- **Traffic Channel Capacity (TDMA):** C = (2 x N) / (2 x T), where N is the number of users.

### Important Theorems

- ** Shannon-Hartley Theorem:** B = C / (2 x log2(1 + (S / N))), where B is bandwidth, C is traffic channel capacity, S is signal-to-noise ratio, and N is noise power spectral density.

### Quick Revision

- Controlled Access techniques include TDMA, FDMA, CDMA, and OFDMA.
- TDMA uses time division to allocate time slots to users.
- FDMA uses frequency division to allocate frequency bands to users.
- CDMA uses code division to allocate code sequences to users.
- OFDMA uses orthogonal frequency division to allocate frequency bands to users.
