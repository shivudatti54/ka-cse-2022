### Introduction

Controlled Access is a type of network access control that restricts the access to a network or a resource to authorized users or devices. It is a mechanism to prevent unauthorized access to a network or a resource, and to ensure that only authorized users or devices can access the network or resource.

Controlled Access is an essential feature of modern computer networks, as it provides a level of security and protection against unauthorized access. It is widely used in various applications, including local area networks (LANs), wide area networks (WANs), and the Internet.

### Historical Context

The concept of Controlled Access dates back to the 1980s, when the need for network security became apparent. In the early days of computer networking, networks were open and unsecured, allowing anyone to access them.

The first controlled access protocol was the Token Ring protocol, which was developed in the late 1980s. Token Ring used a token-passing protocol to grant access to devices on the network, and was widely used in local area networks.

In the 1990s, the need for more secure networks led to the development of protocols such as 802.11 and 802.15, which used protocols such as WEP and WPA to secure wireless networks.
