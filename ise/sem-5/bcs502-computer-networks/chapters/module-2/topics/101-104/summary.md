# **10.1-10.4: Data Link Layer - Error Detection and Correction**

### 10.1 Introduction to Error Detection and Correction

- **Error Detection**: The process of identifying errors in data transmission
- **Error Correction**: The process of correcting errors in data transmission
- **Data Link Layer**: Responsible for error detection and correction

### 10.2 Block Coding

- **Block Coding**: A method of error correction that encodes data into fixed-length blocks
- **Code Words**: Unique encoded blocks
- **Code Rates**: Measure of error correction capability
- **Hamming Code**: A type of block code used for single-bit errors

### 10.3 Cyclic Codes

- **Cyclic Codes**: A type of block code that is invariant under cyclic shifts
- **Generator Polynomials**: Used to generate code words
- **Parity Checks**: Used to detect errors
- **Cyclic Redundancy Checks (CRCs)**: Used to detect errors

### 10.4 Theorems and Formulas

- **Shannon-Hartley Theorem**: Relates signal power to bandwidth
- **Huffman Coding**: A method of variable-length coding
- **Hamming Distance**: Measure of distance between code words

Key Formulas:

- **Block Code Rate**: r = k/n
- **Hamming Code Parity**: p = (x1, x2, ..., xn)
- **CRC Formula**: CRC(x) = g(x) mod m
