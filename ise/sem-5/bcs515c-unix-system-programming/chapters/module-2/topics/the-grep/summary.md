# **UNIX SYSTEM PROGRAMMING: The grep**

## **Key Points**

- **grep** (Global Regular Expression Print) is a command-line utility for searching text.
- **Syntax:** `grep pattern file`
  - `pattern` is the regular expression to match
  - `file` is the file to search in
- **Options:**
  - `-i` (case-insensitive search)
  - `-v` (invert search, show non-matching lines)
  - `-n` (number matching lines)
  - `-o` (output only matching lines)
  - `-r` (search recursively)

## **Definitions and Formulas**

- **Regular Expression:** A pattern used to match characters in strings.
- **grep Pattern Language:** A set of rules for constructing regular expressions.

## **Theorems**

- **Bouyer's Theorem:** The grep command can be used to search for a pattern in a file.
