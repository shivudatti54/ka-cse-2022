# **Reading Dictionaries in UNIX System Programming**

**Key Points:**

- A dictionary in UNIX system programming is a file that stores a collection of key-value pairs, where each key is a word or phrase and the value is its definition or explanation.
- Dictionaries are used to provide information about commands, functions, variables, and other programming concepts.
- There are two types of dictionaries:
  - `/usr/share/dict/words`: contains a list of English words
  - `/usr/share/dict/celebrity`: contains a list of words related to computer science

**Important Formulas and Definitions:**

- None

**Theorems and Concepts:**

- Number of permutations of n objects taken r at a time: `nCr = n! / (r!(n-r)!)`
- Number of combinations of n objects taken r at a time: `nCr = n! / (r!(n-r)!)`

**Commands and Functions:**

- `man`: displays the manual for a command or function
- `dict`: displays the definition of a word or phrase
- `man -d`: displays the manual for a command or function, with definitions

**Revision Tips:**

- Practice using the `man` and `dict` commands to find definitions and manuals for commands and functions.
- Familiarize yourself with the different types of dictionaries available in UNIX.
