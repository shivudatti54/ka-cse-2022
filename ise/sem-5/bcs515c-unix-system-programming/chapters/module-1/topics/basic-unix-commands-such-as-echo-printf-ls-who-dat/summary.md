# **Revision Notes: Basic Unix Commands**

### Overview

- Unix is a multi-user, multi-tasking operating system.
- Basic Unix commands are used for file management, user administration, and system information.

### Key Commands

- **echo**: Prints text to the screen.
  - Syntax: `echo "message"`
- **printf**: Prints formatted text to the screen.
  - Syntax: `printf "%s\n" "message"`
- **ls**: Lists files and directories in the current directory.
  - Syntax: `ls [options] [file_name]`
- **who**: Displays user information, including username and login time.
  - Syntax: `who`
- **date**: Displays the current date and time.
  - Syntax: `date`
- **passwd**: Changes a user's password.
  - Syntax: `passwd [username]`
- **cal**: Displays a calendar for a specified month.
  - Syntax: `cal [month]`

### Combining Commands

- **Pipe**: Redirects the output of one command as the input to another.
  - Syntax: `command1 | command2`
- **Redirect**: Redirects input or output to a file.
  - Syntax: `command > file` or `command < file`
- **Append**: Appends output to a file.
  - Syntax: `command >> file`

### Important Formulas/Definitions/Theorems

- None, this is a list of commands and their syntax.

### Key Concepts

- File permissions and ownership
- User and group identification
- Directory structure and navigation

### Quick Revision Tips

- Practice using basic Unix commands to build muscle memory.
- Learn the syntax and options for each command.
- Practice combining commands to solve real-world problems.
