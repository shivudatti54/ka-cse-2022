# Learning Objectives

After studying this topic, you should be able to:

1. Identify and explain the purpose of each main directory in an Android project structure
2. Differentiate between the res/ and assets/ directories and know when to use each
3. Access various resources (layouts, strings, drawables) programmatically using the R class
4. Understand the role and content of the AndroidManifest.xml file
5. Apply configuration qualifiers to provide alternative resources for different device configurations
6. Navigate and organize files effectively within the Android project structure
