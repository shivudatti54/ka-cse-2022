# **Basic File Types/Categories**

**Important Definitions:**

- **File type**: A classification of a file based on its characteristics, such as its extension, permissions, and contents.
- **File category**: A grouping of files based on their type, purpose, or location.

**File Types:**

- **Regular files**: Contain data, such as text, images, and executables (e.g., `file.txt`, `image.jpg`, `program.exe`)
- **Directories**: Contain other files or directories (e.g., `dir1`, `dir2`, `subdir`)
- **Symbolic links**: Point to a file or directory (e.g., `lnk1`, `lnk2`)
- **Special files**: Represent devices, sockets, or other system resources (e.g., `dev0`, `sock1`)
- **Block devices**: Represent physical devices, such as hard drives or CD-ROMs (e.g., `hd0`, `cd1`)
- **Character devices**: Represent input/output devices, such as keyboards or printers (e.g., `tty0`, `lpt1`)

**File Permissions:**

- **Read**: Allow reading of a file's contents (r)
- **Write**: Allow modification of a file's contents (w)
- **Execute**: Allow execution of a file (x)
- **Owner**: The user who owns the file
- **Group**: The group that owns the file
- **Other**: All users except the owner and group
