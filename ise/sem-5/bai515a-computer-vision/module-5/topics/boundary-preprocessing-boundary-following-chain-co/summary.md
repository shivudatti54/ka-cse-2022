# Boundary Preprocessing

### Definitions and Theorems

- **Chain Code**: A sequence of instructions (up, down, left, right, or stay) that describes the direction of each pixel in a binary image.
- **Boundary Following**: A technique used to obtain a chain code from a binary image.

### Boundary Following

- **Chain Code Generation**:
  - Each pixel in the image is assigned a direction (up, down, left, right, or stay).
  - The chain code is generated by traversing the image from left to right and top to bottom.
- **Boundary Detection**:
  - The chain code is used to detect the boundaries of the image.
  - A pixel is considered a boundary pixel if its direction is different from its neighboring pixels.

### Chain Codes

- **Types of Chain Codes**:
  - 4-way chain code (up, down, left, right)
  - 8-way chain code (up, down, left, right, and four diagonals)
- **Chain Code Notation**:
  - A shorthand notation to represent the direction of each pixel (e.g., u for up, d for down, etc.)

### Important Formulas

- **Chain Code Formula**:
  - The chain code is generated by traversing the image in a zigzag pattern and assigning a direction to each pixel.
  - The direction of each pixel is determined by the following rules:
    - Up: 0
    - Down: 1
    - Left: 2
    - Right: 3
    - Stay: 4

### Important Concepts

- **Boundary Preprocessing**: A technique used to extract the boundaries of an image.
- **Morphological Image Processing**: A technique used to process images using mathematical operations (erosion, dilation, opening, closing, hit-or-miss, etc.).
- **Boundary Detection**: A technique used to detect the boundaries of an image.
