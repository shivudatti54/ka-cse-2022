## **Introduction**

Boundary preprocessing is an important step in morphological image processing. It involves the extraction and processing of image boundaries to produce a binary image where the boundaries are clearly defined. In this study material, we will focus on two techniques used in boundary preprocessing: boundary following and chain codes.

## **Boundary Following**

### Definition

Boundary following is a technique used to trace the boundaries of an object in an image. It involves following the edges of the object to produce a binary image where the boundaries are clearly defined.

### Explanation

The boundary following technique works by starting at a point on the boundary of the object and following the edge until it reaches another boundary point or the edge of the image. At each step, the algorithm checks if the current point is a boundary point or not. If it is, the algorithm marks the point as a boundary point and adds it to the chain code. The chain code is a sequence of numbers that represents the direction of the edge at each point.

### Examples

- An object in an image with a clear boundary.
- An object in an image with a complex boundary.

## **Chain Codes**

### Definition

A chain code is a sequence of numbers that represents the direction of the edge at each point on the boundary of an object.

### Explanation

The chain code is generated by starting at a point on the boundary of the object and following the edge until it reaches another boundary point or the edge of the image. At each step, the algorithm checks if the current point is a boundary point or not. If it is, the algorithm marks the point as a boundary point and adds the corresponding chain code number to the sequence.

### Examples

- A 4-way crossroads with a chain code of 0, 1, 2, 3.
- A 6-way junction with a chain code of 0, 1, 2, 1, 2, 3.

## **Key Concepts**

- **Boundary points**: Points on the boundary of an object where the edge changes direction.
- **Chain code**: A sequence of numbers that represents the direction of the edge at each point on the boundary of an object.
- **Boundary following**: A technique used to trace the boundaries of an object in an image.

## **Code Example**

Here is an example of a Python code snippet that implements the boundary following technique:

```python
import numpy as np

def boundary_following(image):
    # Define the 4-way directions
    directions = [(0, 1), (1, 0), (0, -1), (-1, 0)]

    # Initialize the chain code
    chain_code = []

    # Iterate over the pixels in the image
    for i in range(image.shape[0]):
        for j in range(image.shape[1]):
            # Check if the current pixel is a boundary point
            if image[i, j] == 255 and (i == 0 or image[i - 1, j] == 0) and (j == 0 or image[i, j - 1] == 0):
                # Add the chain code number to the sequence
                chain_code.append(get_direction(i, j, directions))

    return chain_code

def get_direction(i, j, directions):
    # Calculate the direction of the edge at the current point
    for direction in directions:
        if direction == (0, 1) and i > 0 and j == 0:
            return 0
        elif direction == (1, 0) and i == 0 and j > 0:
            return 1
        elif direction == (0, -1) and i > 0 and j == image.shape[1] - 1:
            return 2
        elif direction == (-1, 0) and i == 0 and j == image.shape[1] - 1:
            return 3
        else:
            continue
    return -1
```
