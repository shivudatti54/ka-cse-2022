# Introduction to Computer Vision

## Overview

Computer Vision is an interdisciplinary field enabling computers to gain high-level understanding from digital images or videos, automating tasks performed by the human visual system. The field has progressed through distinct eras from block world recognition to modern deep learning approaches.

## Key Points

- **CV Challenge**: Going from 2D images to 3D world is an ill-posed inverse problem with variability, complexity, and ambiguity
- **Marr's Paradigm (1982)**: Three representation levels - primal sketch (edges/textures), 2.5D sketch (depth/orientation), 3D model
- **Statistical Era (1990s-2000s)**: Eigenfaces (1991), SIFT (1999), Viola-Jones detector (2001), HOG (2005)
- **Deep Learning Revolution**: AlexNet (2012), VGGNet/GoogleNet (2014), ResNet (2015), YOLO/SSD (2016), Transformers (2018+)
- **Key CV Tasks**: Low-level (restoration, enhancement), mid-level (segmentation, optical flow), high-level (detection, recognition, scene understanding)
- **Major Applications**: Autonomous vehicles, medical imaging, surveillance, AR/VR, industrial automation, retail, document analysis, biometrics

## Important Concepts

- CV differs from Computer Graphics (inverse relationship - CV: 2D→3D, CG: 3D→2D)
- Pattern Recognition provides core classification techniques used throughout CV
- Modern trends include foundation models, self-supervised learning, vision-language models, 3D vision
- Current challenges: real-time processing, few-shot learning, explainable AI, privacy preservation

## Notes

- Understand Marr's three-level computational theory as foundational CV framework
- Know the timeline: block world→statistical methods→deep learning
- Be able to explain why CV is harder than it appears (inverse problem complexity)
- Distinguish between low-level, mid-level, and high-level vision tasks
- AlexNet's 2012 ImageNet win marks the start of the deep learning era in CV
