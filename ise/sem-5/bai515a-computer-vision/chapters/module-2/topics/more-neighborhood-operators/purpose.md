# Learning Objectives

After studying this topic, you should be able to:

1. Differentiate between point operators and neighborhood operators
2. Explain the mathematical operations of correlation and convolution
3. Implement common neighborhood operations including smoothing, sharpening, and edge detection filters
4. Select appropriate filters for specific image processing tasks
5. Analyze the effects of different kernel sizes and types on image processing results
6. Handle border issues when applying neighborhood operators
