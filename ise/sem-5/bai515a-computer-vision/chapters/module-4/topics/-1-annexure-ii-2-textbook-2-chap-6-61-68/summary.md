# **Color Image Processing: Chap-6 (6.1-6.8)**

### Key Concepts

- **Color Models:**
  - RGB (Red, Green, Blue) color model
  - CMYK (Cyan, Magenta, Yellow, Black) color model
  - YCbCr (Luminance and Chrominance) color model
- **Color Spaces:**
  - CIE XYZ color space
  - sRGB color space
  - Adobe RGB color space
- **Pseudocolor Image Processing:**
  - Color quantization
  - Color indexing
- **Full Color Image Processing:**
  - Color separation
  - Color mixing

### Important Formulas and Definitions

- **RGB Color Model:**
  - R = red component, G = green component, B = blue component
  - R + G + B = 255 (for 8-bit color)
- **CIE XYZ Color Space:**
  - X = luminance, Y = chrominance, Z = chrominance
  - CIE 1931 color space
- **YCbCr Color Space:**
  - Y = luminance, Cb = blue chrominance, Cr = red chrominance
  - YCbCr conversion: Y = 0.2126R + 0.7152G + 0.0722B, Cb = 128 - 0.2126R - 0.7152G + 0.5236B, Cr = 128 - 0.5374R - 0.7152G + 0.2126B
- **Color Quantization:**
  - Discretization of color values
  - Formula: Q = {(R, G, B) / (255, 255, 255)} \* N, where N = number of quantization levels

### Important Theorems and Concepts

- **Additive Color Mixing:**
  - R + G + B = 255 for full color
- **Subtractive Color Mixing:**
  - CMYK color model for printing
- **Color Separation:**
  - Separation of RGB components for printing
