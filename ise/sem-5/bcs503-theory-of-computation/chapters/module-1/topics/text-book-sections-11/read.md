### Introduction

In this module, we will explore the basics of the theory of computation, which is a fundamental area of computer science. This module will cover the basics of computers, models of computation, and the structure of algorithms.

### What is Computing?

Computing refers to the process of processing information using a computer. It involves input, processing, and output of information using a computer's hardware and software components.

### Models of Computation

A model of computation is a mathematical representation of a computer's behavior. There are several models of computation, including:

- **Sequential Model**: This model assumes that the computer executes statements one at a time, in a linear sequence.
- **Parallel Model**: This model assumes that the computer executes statements simultaneously, in parallel.
- **Finite State Machine (FSM)**: This model assumes that the computer is a finite set of states, with transitions between states based on input.

### Algorithms

An algorithm is a set of instructions that is used to solve a specific problem. It is a well-defined procedure that takes input and produces output.

### Key Concepts

- **Input**: The data that is provided to the computer.
- **Processing**: The operation performed on the input data.
- **Output**: The result of the processing operation.

#### Key Characteristics of Algorithms

- **Well-defined**: The algorithm must have a clear and unambiguous set of instructions.
- **Deterministic**: The algorithm must produce the same output for a given input.
- **Efficient**: The algorithm must use a minimal amount of resources (such as time and memory).

#### Types of Algorithms

- **Sequential Algorithm**: An algorithm that executes statements one at a time, in a linear sequence.
- **Parallel Algorithm**: An algorithm that executes statements simultaneously, in parallel.
- **Divide and Conquer Algorithm**: An algorithm that breaks down a problem into smaller sub-problems, solves each sub-problem, and combines the solutions to solve the original problem.

### Example

Suppose we want to write an algorithm to sort a list of numbers in ascending order. Here is an example of a sequential algorithm:

1.  Read the list of numbers from input.
2.  Initialize an empty list to store the sorted numbers.
3.  Iterate through the list of numbers, comparing each number with the previous number.
4.  If the current number is less than the previous number, add it to the sorted list.
5.  Repeat steps 3-4 until all numbers have been processed.
6.  Output the sorted list of numbers.
