## **What is Computation?**

Computation refers to the process of performing a series of operations on data to produce a desired output. This can be done using various devices, including computers, calculators, and even the human brain.

## **Basic Concepts**

### 1. **Input**

The input of a computational system is the data that is fed into the system, which can be in the form of numbers, characters, or other types of data.

### 2. **Output**

The output of a computational system is the result of the computation, which can be in the form of a numerical answer, a printed message, or another type of data.

### 3. **Algorithm**

An algorithm is a set of instructions that is used to solve a problem or perform a computation. It is a well-defined procedure that takes input data and produces output data.

### 4. **Computer Model**

A computer model is a simplified representation of a computational system, which is used to describe the behavior of the system and its interactions with the environment.

## **Types of Computations**

### 1. **Deterministic Computation**

A deterministic computation is one where the output of the computation is uniquely determined by the input and the algorithm.

### 2. **Non-Deterministic Computation**

A non-deterministic computation is one where the output of the computation is not uniquely determined by the input and the algorithm.

## **Key Concepts**

- **Turing Machine**: A model of computation that is used to describe the behavior of a computational system.
- **State**: A state is a point in the computation where the system has a specific status or configuration.
- **Transition**: A transition is a change in the state of the system based on the current state and input.
- **Accepting and Rejecting States**: Accepting states are those that indicate the system has accepted the input, while rejecting states indicate that the system has rejected the input.

## **Examples**

- A vending machine is a non-deterministic computation, as it is not possible to predict the outcome of a coin insert and selection.
- A simple calculator is a deterministic computation, as the output is uniquely determined by the input and the algorithm.
