# Learning Objectives

After studying this topic, you should be able to:

1. Explain the concept of pipes and FIFOs as IPC mechanisms in Unix
2. Differentiate between regular pipes and FIFOs (named pipes)
3. Use the pipe() system call to create pipes for inter-process communication
4. Implement communication between parent and child processes using pipes
5. Create and use FIFOs for communication between unrelated processes
6. Utilize popen() and pclose() functions for easy pipe creation with shell commands
