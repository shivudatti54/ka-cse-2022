# Learning Objectives

After studying this topic, you should be able to:

1. Explain what signals are and their role in Unix process communication
2. Identify common Unix signals and their default behaviors
3. Implement signal handlers using both signal() and sigaction() functions
4. Differentiate between signal generation, delivery, and handling
5. Use signal masking to control signal delivery during critical sections
6. Understand signal safety considerations and use async-signal-safe functions appropriately
7. Explain signal inheritance behavior across fork() and exec() system calls
