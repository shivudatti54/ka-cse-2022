# Learning Objectives: Behavior Monitoring Tools

After studying this topic, you should be able to:

1.  **Identify** the primary tools used for monitoring malware behavior on a Windows system (e.g., ProcMon, Process Explorer, Wireshark).
2.  **Explain** the core function and key features of each major behavior monitoring tool.
3.  **Execute** a basic dynamic analysis workflow: preparing a sandbox, running a sample while monitoring, and analyzing the resulting logs.
4.  **Filter** monitoring output to isolate malicious activity from normal system noise.
5.  **Extract** key Indicators of Compromise (IOCs) such as created files, modified registry keys, and network connections from behavioral data.
6.  **Correlate** sequences of events across different tools to build a comprehensive narrative of the malware's actions.
