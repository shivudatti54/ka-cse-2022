# Why Learn Decomposition?

## Exam Relevance

- BCNF decomposition is a frequent exam question
- Lossless-join verification problems
- Dependency preservation analysis
- Comparing 3NF vs BCNF decomposition

## Real-World Applications

- Database schema design and optimization
- Data warehouse design
- Legacy database refactoring
- Eliminating data redundancy

## Key Skills

- Apply decomposition algorithm step-by-step
- Verify lossless-join property
- Check dependency preservation
- Choose between 3NF and BCNF
