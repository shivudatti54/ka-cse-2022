# Learning Objectives

After studying this topic, you should be able to:

1. Define and explain the greedy choice property and optimal substructure
2. Design greedy algorithms for optimization problems like activity selection and knapsack
3. Analyze the time and space complexity of greedy algorithms
4. Prove the correctness of greedy algorithms using mathematical reasoning
5. Differentiate between problems that can and cannot be solved using greedy approach
6. Compare and contrast greedy algorithms with dynamic programming approaches
