# Learning Objectives

After studying this topic, you should be able to:

1. Define what constitutes the "environment" for an intelligent agent.
2. List and describe the six fundamental properties used to classify environments in AI.
3. Accurately classify a given task environment (e.g., autonomous vehicle, chess game) based on its properties.
4. Construct a complete PEAS description for a specified agent task.
5. Explain how the nature of an environment influences the design and choice of an intelligent agent.
6. Differentiate between deterministic/stochastic, static/dynamic, and single-agent/multi-agent environments with clear examples.
