# Learning Objectives

After studying this topic, you should be able to:

1. Understand the basic logical operators and how to use them to combine propositions.
2. Construct truth tables for complex statements.
3. Use reasoning patterns, such as modus ponens and modus tollens, to draw conclusions from statements.
4. Apply propositional logic to real-world examples.
5. Explain the difference between propositional logic and first-order logic.
6. Use truth tables to determine the truth value of a statement.
