# Learning Objectives

After studying this topic, you should be able to:

1. Define what a knowledge-based agent is and explain its primary characteristics.
2. Describe the different types of knowledge-based agents and their characteristics.
3. Explain the purpose of a heuristic function and how it is used in a knowledge-based agent.
4. Describe the Wumpus world environment and its purpose.
5. Compare and contrast different types of knowledge-based agents.
6. Apply knowledge-based agent concepts to solve problems.
