# Learning Objectives

After studying this topic, you should be able to:

1. Understand the environment and the hunter's goal in the Wumpus World.
2. Represent the environment using logical statements.
3. Use inference and reasoning to update the agent's knowledge.
4. Compare and contrast different informed search strategies.
5. Apply heuristic functions to guide the agent's search for a solution.
6. Analyze the advantages and disadvantages of different search strategies.
