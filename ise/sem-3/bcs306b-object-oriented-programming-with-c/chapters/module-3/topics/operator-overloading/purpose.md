# Learning Objectives

After studying this topic, you should be able to:

1. Explain the concept and purpose of operator overloading in C++
2. Differentiate between member function and friend function operator overloading
3. Implement overloaded operators for arithmetic, comparison, and stream operations
4. Distinguish between prefix and postfix increment/decrement operators
5. Apply proper memory management techniques in assignment operators
6. Identify which operators cannot be overloaded and understand the reasons why
