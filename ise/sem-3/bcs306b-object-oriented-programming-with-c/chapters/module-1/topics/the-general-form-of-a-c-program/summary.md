# The General Form of a C++ Program

=====================================

### Introduction

---

The general form of a C++ program consists of the following components:

### Program Structure

---

- **Preprocessor Directives**: The first part of the program that are executed before the compilation phase. These directives include `#include`, `#define`, and `#ifdef`.
- **Header Files**: These are pre-written code blocks that provide functionality to the program. They are included using `#include`.
- **Main Function**: This is the entry point of the program and is where the execution begins.
- **Source Code**: This is the actual code that makes up the program.
- **Object-Oriented Code**: This is the actual code that makes up the program.
- **Linker Directives**: These are used to link the object files and libraries together.

### C++ Program Structure

---

- `#include stdio.h` (includes standard input/output library)
- `#include math.h` (includes mathematical library)
- `#define PI 3.14` (defines a constant)
- `int main()` (main function)
  - `int main()` returns an integer
  - It is the entry point of the program
- `return 0;` (returns an integer)
