# Learning Objectives

After studying this topic, you should be able to:

1. Define the ordinary generating function for a given sequence.
2. Recall the generating functions for fundamental sequences (constant, geometric, binomial).
3. Apply the step-by-step method of using generating functions to solve linear recurrence relations with constant coefficients.
4. Perform index shifting accurately when summing recurrence relations.
5. Utilize the method of partial fractions to decompose rational generating functions.
6. Extract the closed-form solution of a sequence from the series expansion of its generating function.
