# Learning Objectives

After studying this topic, you should be able to:

1. Define what an array is in Java and explain why it is classified as a reference type
2. Declare, initialize, and access elements in single-dimensional arrays using both traditional and enhanced for loops
3. Create and manipulate two-dimensional and multi-dimensional arrays in Java
4. Identify and handle common runtime exceptions associated with arrays, such as ArrayIndexOutOfBoundsException and NullPointerException
5. Apply array utility methods from the java.util.Arrays class for sorting, searching, and copying operations
6. Implement fundamental array operations including traversal, searching, and element modification
7. Analyze the time and space complexity of array-based operations
8. Distinguish between shallow copies and deep copies when duplicating arrays
