# **Principles, Using Blocks of Code, Lexical Issues (Whitespace, Identifiers, Literals, Comments, Separators, The Java Keywords)**

## **Principles**

- **Principles of Programming Languages**: A set of rules and guidelines for writing programs.
- **Principles of Object-Oriented Programming (OOP)**:
  - Abstraction
  - Encapsulation
  - Inheritance
  - Polymorphism

## **Using Blocks of Code**

- **Blocks of Code**: A group of statements enclosed in curly brackets `{}`.
- **Code Blocks**: Can be used to group statements that need to be executed together.
- **Nested Blocks**: Can be defined inside other blocks.

## **Lexical Issues**

### Whitespace

- **Whitespace**: Characters that separate tokens, such as spaces, tabs, and newline characters.
- **Importance**: Whitespace is crucial in maintaining the readability and structure of code.

### Identifiers

- **Identifiers**: Names given to variables, methods, and classes.
- **Characteristics**:
  - Must start with a letter or underscore.
  - Can contain letters, digits, and underscore.
  - Cannot be a reserved keyword.

### Literals

- **Literals**: Representations of data, such as numbers, characters, and strings.
- **Types of Literals**:
  - **Integer Literals**: Whole numbers, e.g., 1, 2, 3.
  - **Character Literals**: Single characters, e.g., 'a', 'b', 'c'.
  - **String Literals**: Sequences of characters, e.g., "hello", 'hello'.

### Comments

- **Comments**: Notes added to code to explain its purpose or functionality.
- **Types of Comments**:
  - **Line Comments**: Start with `//` and continue until the end of the line.
  - **Block Comments**: Start with `/*` and end with `*/`, used for multi-line comments.

### Separators

- **Separators**: Characters that separate tokens, such as commas, semicolons, and parentheses.
- **Importance**: Separators are essential for maintaining the syntax and structure of code.

### Java Keywords

- **Java Keywords**: Reserved words that have a specific meaning in Java, e.g., `public`, `class`, `if`, `else`.
- **List of Java Keywords**:
  - `abstract`, `assert`, `boolean`, `break`, `byte`, `case`, `catch`, `char`, `class`, `const`, `continue`, `default`, `do`, `double`, `else`, `enum`, `extends`, `final`, `finally`, `float`, `for`, `goto`, `if`, `implements`, `import`, `instanceof`, `int`, `interface`, `long`, `native`, `new`, `package`, `private`, `protected`, `public`, `return`, `short`, `static`, `strictfp`, `super`, `switch`, `synchronized`, `this`, `throw`, `throws`, `transient`, `try`, `void`, `volatile`, `while`
