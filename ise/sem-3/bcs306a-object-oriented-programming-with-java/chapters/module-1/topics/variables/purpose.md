# Learning Objectives

After studying this topic, you should be able to:

1. Define what a variable is in Java and explain its three essential properties: type, name, and value.

2. Distinguish between the four types of variables in Java: instance variables, static variables, local variables, and parameters.

3. Apply the correct syntax for declaring and initializing variables of different data types.

4. Explain variable scope and lifetime, and determine where a variable can be accessed in a Java program.

5. Follow Java naming conventions while choosing meaningful variable names for programs.

6. Identify default values for instance and static variables versus the requirement to explicitly initialize local variables.

7. Analyze how static variables behave differently from instance variables in terms of memory allocation and sharing.

8. Write Java code that correctly demonstrates the use of variables in various contexts, including object-oriented programs.
