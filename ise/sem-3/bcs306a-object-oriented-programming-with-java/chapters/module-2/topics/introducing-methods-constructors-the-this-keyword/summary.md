# **Introducing Methods, Constructors, The this Keyword, Garbage Collection**

### Class Fundamentals

- **Class**: A blueprint for creating objects
- **Object**: An instance of a class
- **Class Object**: A class that has a reference to itself

### Methods

- **Method**: A block of code that can be executed at a specific time
- **Instance Method**: A method that belongs to an object
- **Static Method**: A method that belongs to a class, not an instance
- **Constructors**: Methods that initialize objects when they are created

### Constructors

- **Constructor**: A special method that initializes objects when they are created
- **Constructor Signature**: `public [accessModifier] ClassName(parameters) { }`
- **Default Constructor**: A constructor without any parameters
- **Parameterized Constructor**: A constructor with one or more parameters

### The this Keyword

- **this**: A keyword used to refer to the current object
- **refers to the object**: The current object that the method is being called on
- **Used to access variables**: The object's variables and methods

### Garbage Collection

- **Garbage Collection**: A process that automatically frees memory occupied by objects no longer needed
- **Reference Counting**: A method that increments or decrements the reference count of an object
- **Heap**: A memory area where objects are stored
- **Mark and Sweep Algorithm**: A garbage collection algorithm that identifies and collects garbage objects

### Important Formulas and Definitions

- **Object Reference Variable**: A variable that holds a reference to an object
- **Null Object Reference**: An object reference variable that does not refer to any object
- **Final**: A keyword that prevents a variable from being reassigned
- **Final Method**: A method that cannot be overridden

### Theorems and Notes

- **Theorem of Inheritance**: A class can inherit properties and methods from another class
- **Polymorphism**: The ability of an object to take on multiple forms
- **Abstraction**: The concept of showing only the necessary information to the outside world
