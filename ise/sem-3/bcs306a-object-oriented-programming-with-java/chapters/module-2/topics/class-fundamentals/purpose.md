# Learning Objectives

After studying this topic, you should be able to:

1. DEFINE what a class is in Java and explain its role as a blueprint for creating objects

2. IDENTIFY and EXPLAIN the three main components of a class: fields (instance variables), constructors, and methods

3. DISTINGUISH between instance variables and class (static) variables, including their memory allocation and accessibility patterns

4. IMPLEMENT encapsulation in classes by using appropriate access modifiers (private, public, protected) and providing public getter and setter methods

5. CREATE and INITIALIZE objects using constructors, including parameterized constructors and constructor overloading

6. APPLY the `this` keyword to resolve variable naming conflicts and to call constructors from other constructors within the same class

7. ANALYZE how static variables and static methods work, including calling them using the class name without object instantiation

8. DESIGN and WRITE complete Java classes that model real-world entities with proper data hiding and behavioral methods for examination problems
