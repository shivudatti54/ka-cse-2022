# Interfaces: Interfaces, Default Interface Methods, Use static Methods in an Interface, Private Interface Methods

===========================================================

## Key Points

---

- **Interfaces**:
  - A abstract class that cannot be instantiated
  - Used to achieve abstraction, modularity, and extensibility
  - Methods in an interface are implicitly abstract and public
- **Default Interface Methods**:
  - Introduced in Java 8 to allow interfaces to provide default implementations for methods
  - Methods in an interface can be overridden using the `@Override` annotation
- **Use static Methods in an Interface**:
  - Static methods in an interface are implicitly static and public
  - Used to group related static methods together
- **Private Interface Methods**:
  - Introduced in Java 9 to allow interfaces to provide private methods
  - Used to hide implementation details and prevent accidental usage
