# Digital Design and Computer Organization

### 1.9: Digital Logic and Combinational Circuits

### Important Definitions

- **Combinational Circuit**: A circuit where the output is a function of the current inputs and there is no memory element.
- **Logic Gate**: A digital circuit that takes one or more input signals and produces an output signal based on a specific logical operation.

### Key Theorems

- **Karnaugh Map (K-Map)**: A method for simplifying Boolean expressions.
- **Boolean Algebra**: A mathematical system for expressing logical operations and simplifying Boolean expressions.

### Important Formulas

- **Complement Law**: A^2 = A \* (A + 1) = A + A'
- **De Morgan's Law**: A + B' = A' \* A + (B') A' = (A + B)(A' + B')
- **Karnaugh Map Formula**:
  - A + B = A + B (if both are 1)
  - A + B = A + B' (if A is 1 and B is 0)
  - A _ B = A _ B (if both are 1)
  - A _ B = A _ B' (if A is 0 and B is 1)

### Important Concepts

- **AND Gate**: Produces an output of 1 only if all inputs are 1.
- **OR Gate**: Produces an output of 1 if any input is 1.
- **NOT Gate**: Inverts the input signal (i.e., 0 becomes 1 and 1 becomes 0).
- **NAND Gate**: The NOT gate and AND gate combined.
- **NOR Gate**: The NOT gate and OR gate combined.
- **XOR Gate**: Produces an output of 1 if the inputs are different.
- **XNOR Gate**: Produces an output of 1 if the inputs are the same.
