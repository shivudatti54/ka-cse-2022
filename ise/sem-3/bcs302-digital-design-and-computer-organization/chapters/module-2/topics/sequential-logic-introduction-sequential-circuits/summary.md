# Sequential Logic: Introduction, Sequential Circuits, Storage Elements: Latches, Flip-Flops

### Introduction

- Sequential logic is a type of digital logic that deals with circuits that change state over time.
- Sequential circuits are used to implement memory and control functions in digital systems.
- Storage elements are used to store binary data in sequential circuits.

### Sequential Circuits

- A sequential circuit is a circuit that has one or more memory elements.
- The state of the circuit changes over time, and the circuit can only be in one of a finite number of states.
- Sequential circuits can be categorized into two types:
  - Combinatorial sequential circuits
  - Sequential Sequential circuits

### Storage Elements

- A storage element is a component that can store a binary value.
- There are two types of storage elements:
  - Latches
  - Flip-Flops

### Latches

- A latch is a type of storage element that can only store one bit of data.
- A latch is a combinational sequential circuit that has a memory element and a set of gates.
- Latches can be classified into two types:
  - S-R Latch
  - T-K Latch

### Flip-Flops

- A flip-flop is a type of storage element that can store one or more bits of data.
- A flip-flop is a sequential sequential circuit that has a memory element and a set of gates.
- Flip-flops are used to store data in sequential circuits.

**Important Formulas and Definitions:**

- **Tautology:** A statement that is always true.
- **Contradiction:** A statement that is always false.
- **K-map:** A truth table that represents the output of a logic function.
- **State table:** A table that represents the states of a sequential circuit.
- **False state:** A state that is not a valid state of the circuit.

**Theorems:**

- **Moore's Law:** The number of transistors on a microchip doubles every two years.
- **Hartley's Law:** The capacity of a storage element is proportional to the square of the number of transistors.

**Revision Tips:**

- Understand the difference between sequential and combinatorial circuits.
- Learn the types of storage elements and their applications.
- Be able to draw and analyze the state table of a sequential circuit.
