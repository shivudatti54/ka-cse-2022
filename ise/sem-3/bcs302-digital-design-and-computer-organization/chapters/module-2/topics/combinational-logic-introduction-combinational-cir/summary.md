# **Combinational Logic Revision Notes**

## **Introduction**

- Combinational logic is a type of digital circuit that produces an output based on a combination of inputs.
- It is a basic building block for digital circuits.
- Combinational logic is used to perform operations such as logic gates, arithmetic, and data encoding.

## **Combinational Circuits**

- A combinational circuit is a digital circuit that uses logic gates to produce an output based on a combination of inputs.
- It does not have any memory and its output is deterministic.

## **Design Procedure**

- Choose a design method: Karnaugh map, Boolean algebra, or truth table.
- Draw the circuit diagram.
- Use a logic simulator or software to test the circuit.

## **Binary Adder-Subtractor**

- A binary adder-subtractor is a combinational circuit that adds or subtracts two binary numbers.
- Formula: `S = A + B - C`
- Theorem: The binary adder-subtractor can be designed using a combination of AND, OR, and NOT gates.

## **Decoders**

- A decoder is a combinational circuit that takes a binary code and produces a specific output.
- Formula: `Y = X * Q`
- Theorem: The decoder can be designed using a combination of AND and OR gates.

## **Encoders**

- An encoder is a combinational circuit that takes a binary input and produces a specific code.
- Formula: `X = Q * Y`
- Theorem: The encoder can be designed using a combination of AND and OR gates.

## **Multiplexers**

- A multiplexer is a combinational circuit that selects one of several input signals based on a binary code.
- Formula: `Y = Q * X`
- Theorem: The multiplexer can be designed using a combination of AND and OR gates.

## **Important Formulas and Definitions**

- Boolean algebra:

* `A + B = A or B`
* `A * B = A and B`
* `not A = A'`

- Karnaugh map:

* A 2-variable Karnaugh map can be used to simplify binary expressions.
* A 3-variable Karnaugh map can be used to simplify ternary expressions.
