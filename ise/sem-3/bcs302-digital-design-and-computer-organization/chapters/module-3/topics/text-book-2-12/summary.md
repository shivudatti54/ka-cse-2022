## **Topic 1.2: Digital Design and Computer Organization**

### Key Concepts

- **Digital Design:**
  - Definition: The process of creating digital systems, including digital circuits and logic gates.
  - Types of Digital Design:
    - Sequential Logic
    - Combinatorial Logic
- **Computer Organization:**
  - Definition: The study of how a computer's hardware and software interact to process information.
  - Components of Computer Organization:
    - Central Processing Unit (CPU)
    - Memory
    - Input/Output (I/O) Devices

### Important Formulas and Definitions

- **Combinatorial Logic:**
  - Boolean Algebra:
    - OR: A + B = 1
    - AND: A ∧ B = 1
    - NOT: A' = 0
  - Karnaugh Maps:
    - Used to simplify Boolean expressions
- **Sequential Logic:**
  - Flip-Flops:
    - Basic building blocks of sequential logic circuits
  - State Machines:
    - Easy to implement and analyze sequential logic circuits

### Important Theorems

- **Boolean Algebra Laws:**
  - Distributive Law: A(B + C) = AB + AC
  - De Morgan's Law: (A + B)' = A'B'
- **Karnaugh Map Theorem:**
  - To minimize Boolean expressions using Karnaugh Maps

### Key Terms

- **Digital Circuit:**
  - A path of interconnected logic gates that perform a specific digital function.
- **Logic Gate:**
  - A basic digital circuit that performs a specific logical operation.

### Quick Revision Tips

- Review Boolean Algebra laws and Karnaugh Maps.
- Understand the difference between combinatorial and sequential logic.
- Familiarize yourself with flip-flops and state machines.
