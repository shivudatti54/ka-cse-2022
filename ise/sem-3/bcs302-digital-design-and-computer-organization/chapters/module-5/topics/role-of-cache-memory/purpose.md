# Learning Objectives

After studying this topic, you should be able to:

1. EXPLAIN the fundamental role of cache memory in bridging the speed gap between processor and main memory

2. DESCRIBE the two types of locality of reference (temporal and spatial) that make caching effective

3. COMPARE the three cache mapping techniques: direct mapping, fully associative mapping, and set-associative mapping, including their advantages and disadvantages

4. CALCULATE cache parameters including line index, tag bits, and address composition for given cache configurations

5. ANALYZE cache performance using Average Memory Access Time (AMAT) and identify factors that affect hit rate and miss penalty

6. DISTINGUISH between write-through and write-back policies, and explain when each is appropriate

7. EXPLAIN common cache replacement algorithms (LRU, FIFO, Random) and their implementation considerations

8. DESCRIBE the purpose and organization of multi-level cache hierarchies (L1, L2, L3) in modern processors
