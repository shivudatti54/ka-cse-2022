# Learning Objectives

After studying this topic, you should be able to:

1. Distinguish between computer organization and computer architecture, explaining the significance of this distinction in system design.

2. Describe the basic structure of a computer system, identifying the roles of CPU, memory, and I/O components.

3. Explain the stored program concept and its implications for modern computing, including the von Neumann architecture.

4. Trace through the instruction cycle phases (fetch, decode, execute, store) and explain what happens in each phase.

5. Identify and differentiate between various addressing modes used in instruction execution.

6. Convert signed integers to and from two's complement representation and perform basic arithmetic in two's complement form.

7. Explain the function of the system bus and its components (address, data, and control buses).

8. Apply knowledge of basic concepts to analyze simple instruction execution sequences and calculate effective addresses.
