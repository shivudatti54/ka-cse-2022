# Learning Objectives

After studying this topic, you should be able to:

1. Explain the structure of memory cells and their organization into larger units like words and banks

2. Differentiate between word-addressable and byte-addressable memory systems

3. Describe the concept of memory banks and explain how bank organization enables interleaving

4. Analyze address decoding methods including linear selection and hierarchical (row-column) decoding

5. Calculate memory capacity given the number of address and data lines

6. Understand big-endian and little-endian byte ordering and their implications

7. Explain how memory interleaving improves effective bandwidth for sequential access patterns

8. Apply memory organization concepts to solve numerical problems related to address mapping and capacity
