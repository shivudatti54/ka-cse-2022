# Learning Objectives

After studying this topic, you should be able to:

1. Define and explain the concept of a Priority Queue and distinguish between Single-Ended and Double-Ended Priority Queues

2. Describe the fundamental operations supported by SEPQ and DEPQ, including INSERT, EXTRACT-MAX, EXTRACT-MIN, GET-MAX, and GET-MIN

3. Implement a Priority Queue using a Binary Heap data structure and explain the heap properties (max-heap and min-heap)

4. Analyze the time and space complexities of various Priority Queue operations in heap-based implementations

5. Compare different implementation approaches for Double-Ended Priority Queues including dual heap, balanced BST, interval heap, and min-max heap

6. Apply Priority Queues in real-world scenarios such as CPU scheduling, Dijkstra's shortest path algorithm, and Huffman coding

7. Trace through heap operations step-by-step to maintain the heap property after insertions and deletions
