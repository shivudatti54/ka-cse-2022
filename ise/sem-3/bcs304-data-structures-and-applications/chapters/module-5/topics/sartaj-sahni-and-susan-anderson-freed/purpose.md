# Learning Purpose: Sartaj Sahni and Susan Anderson-Freed

**1. Why is this topic important?**
This topic is crucial as it introduces students to the foundational figures whose pioneering work has shaped the design, analysis, and teaching of algorithms and data structures. Understanding their contributions provides historical context and a deeper appreciation for the theoretical underpinnings of efficient computation, which is essential for solving complex computational problems.

**2. What will students learn?**
Students will learn about the key research and educational contributions of Sartaj Sahni (e.g., authoring a seminal data structures textbook, work on parallel algorithms) and Susan Anderson-Freed (e.g., foundational work on data structure implementation and pedagogy). They will connect specific algorithms and data structure concepts directly to the innovators behind them.

**3. How does it connect to other concepts?**
Their work is intrinsically linked to core concepts across the curriculum, including algorithm analysis (Big-O notation), data structure implementation (arrays, linked lists, trees, graphs), and algorithm design techniques (e.g., greedy methods, dynamic programming). It provides the "why" behind the "what" students have learned in previous modules.

**4. Real-world applications**
The principles and efficient algorithms developed and taught by these computer scientists are applied everywhere: database indexing, network routing, compilers, operating systems, and genome sequencing. Their textbooks have educated generations of engineers who build these real-world systems.
