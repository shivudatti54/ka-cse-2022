# Learning Objectives

After studying this topic, you should be able to:

1. Distinguish between static and dynamic array allocation, explaining the advantages and use cases of each approach in program design.

2. Apply the malloc() function to allocate memory for one-dimensional arrays at runtime, including proper size calculation and NULL pointer validation.

3. Use calloc() to allocate zero-initialized memory and explain scenarios where zero-initialization provides safer code compared to malloc().

4. Implement dynamic array resizing using realloc(), demonstrating understanding of both memory expansion and new allocation scenarios.

5. Create two-dimensional dynamic arrays using both array-of-pointers and contiguous block approaches, choosing appropriate methods based on performance requirements.

6. Design and implement a self-expanding dynamic array data structure that mimics ArrayList functionality with amortized O(1) append operations.

7. Properly manage memory lifecycle by balancing allocation with deallocation using free(), preventing memory leaks and dangling pointer references.

8. Analyze and debug common dynamic memory errors including NULL dereference, double-free, memory leak, and buffer overflow scenarios.
