# Learning Objectives

After studying this topic, you should be able to:

1. Compare adjacency matrix and adjacency list representations of graphs, including their space and time complexities.

2. Implement basic graph operations: initializing a graph, adding and removing vertices and edges, checking edge existence.

3. Perform breadth-first search (BFS) traversal on a graph and explain its properties including time complexity and shortest-path guarantee.

4. Perform depth-first search (DFS) traversal on a graph using both recursive and iterative approaches.

5. Analyze when to use BFS versus DFS based on the problem requirements.

6. Compute vertex degrees (in-degree and out-degree for directed graphs) and explain their significance.

7. Evaluate the trade-offs between different graph representations for specific application scenarios.

8. Apply elementary graph operations to solve practical problems involving graph manipulation.
