# Learning Objectives

After studying this topic, you should be able to:

1. Define graphs as data structures and explain their components including vertices, edges, and various types of relationships between elements.

2. Distinguish between different types of graphs including directed, undirected, weighted, unweighted, simple, and multigraphs with appropriate examples.

3. Analyze and compare adjacency matrix and adjacency list representations, determining the appropriate representation based on graph characteristics.

4. Implement and trace Breadth-First Search (BFS) and Depth-First Search (DFS) algorithms, understanding their differences and applications.

5. Identify connected components in undirected graphs and strongly connected components in directed graphs.

6. Apply shortest path algorithms including Dijkstra's algorithm to solve problems involving minimum distance calculations.

7. Perform topological sorting on directed acyclic graphs (DAGs) and understand its applications in task scheduling.

8. Explain the concept of spanning trees and compare Kruskal's and Prim's algorithms for finding minimum spanning trees.
