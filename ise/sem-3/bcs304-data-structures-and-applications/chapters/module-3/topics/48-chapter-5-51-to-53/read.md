### 5.1: Sets

**Definition:** A set is an unordered collection of unique elements, known as **members** or **elements**, that can be of any data type, including integers, strings, and objects.

**Key Characteristics:**

• Unordered: Elements in a set do not have a specific order.
• Unique: Each element in a set is distinct and cannot be duplicated.
• Mutable: Sets can be modified by adding or removing elements.

**Example Use Cases:**

• Removing duplicates from a list of names
• Creating a set of unique colors in a painting
• Performing set operations (union, intersection, difference) on two sets

**Set Operations:**

• **Union (A ∪ B):** A set containing all elements from both sets A and B.
• **Intersection (A ∩ B):** A set containing only the elements common to both sets A and B.
• **Difference (A \ B):** A set containing elements in A but not in B.

### 5.2: Maps (Dictionaries)

**Definition:** A map is an unordered collection of key-value pairs, where each key is unique and maps to a specific value.

**Key Characteristics:**

• Unordered: Elements in a map do not have a specific order.
• Key-Value Pairs: Each element in a map consists of a unique key and a specific value.
• Mutable: Maps can be modified by adding or removing key-value pairs.

**Example Use Cases:**

• Storing a person's name and address in a dictionary
• Creating a map of country codes to their corresponding names
• Performing map operations (lookup, insertion, deletion) on a map

### 5.3: Lists

**Definition:** A list is an ordered collection of elements, which can be of any data type, including integers, strings, and objects.

**Key Characteristics:**

• Ordered: Elements in a list have a specific order.
• Mutable: Lists can be modified by adding or removing elements.
• Indexed: Each element in a list has a unique index or position.

**Example Use Cases:**

• Creating a list of favorite foods
• Storing a list of tasks to be completed
• Performing list operations (insertion, deletion, sorting) on a list

**Common List Operations:**

• **Insertion:** Adding an element at a specific position in the list.
• **Deletion:** Removing an element from the list.
• **Sorting:** Rearranging the elements in the list in a specific order.

By understanding these fundamental data structures, you can build more complex programs and solve real-world problems more efficiently.
