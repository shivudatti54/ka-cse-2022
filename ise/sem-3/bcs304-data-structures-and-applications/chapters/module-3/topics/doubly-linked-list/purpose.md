# Why Study Doubly Linked Lists?

## Enhanced Capabilities

Doubly linked lists solve key limitations of singly linked lists:

- Bidirectional traversal
- O(1) delete when node pointer is known
- Efficient LRU cache implementation

## Real-World Applications

1. **Browser History** - Back/forward navigation
2. **LRU Cache** - Used in databases and OS
3. **Text Editors** - Undo/redo functionality
4. **Music Players** - Previous/next track

## Key Interview Question

"How would you implement an LRU Cache?"

- Uses hash map + doubly linked list
- O(1) access and O(1) eviction
- Classic system design question
