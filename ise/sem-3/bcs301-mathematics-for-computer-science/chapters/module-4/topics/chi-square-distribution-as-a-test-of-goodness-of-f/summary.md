# **Chi-square Distribution as a Test of Goodness of Fit**

## **Key Points**

- **Definition**: Chi-square distribution is a discrete probability distribution used to test the goodness of fit of a set of observed values against a hypothesized distribution.
- **Formula**: Chi-square statistic = Σ [(observed frequency - expected frequency)^2 / expected frequency]
- **Null Hypothesis**: The observed frequencies follow a specific distribution (e.g. binomial, Poisson).
- **Alternative Hypothesis**: The observed frequencies do not follow the specific distribution.
- **Test Statistic**: Chi-square statistic is used to determine the test statistic value.
- **Critical Region**: Chi-square distribution table is used to determine the critical region value.

## **Important Formulas and Theorems**

- **Chi-square distribution**: A discrete probability distribution with a probability mass function given by f(x) = (1 / (2^(n/2))) \* (x^(n/2-1)) \* e^(-x/2) for x = 0, 1, 2, ..., ∞
- **Poisson Distribution**: A discrete probability distribution with a probability mass function given by f(x) = (e^(-λ) \* (λ^x)) / x! for x = 0, 1, 2, ...
- **Binomial Distribution**: A discrete probability distribution with a probability mass function given by f(x) = (nCx \* p^x \* q^(n-x)) for x = 0, 1, 2, ..., n
- **Independence Test**: Chi-square test of independence is used to test if two categorical variables are independent.

## **Important Theorems**

- **Lagrange's Theorem**: The chi-square distribution is related to the Poisson distribution.
- **Maxwell's Theorem**: The chi-square distribution is related to the binomial distribution.
