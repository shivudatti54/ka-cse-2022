## **Introduction**

A random variable is a mathematical concept used to model a variable that can take on different values with varying probabilities. Random variables can be either discrete or continuous, and they play a crucial role in probability theory.

## **Discrete Random Variables**

### Definition

A discrete random variable is a variable that can take on a countable number of distinct values. These values are usually represented as integers or other countable numbers.

### Types of Discrete Random Variables

- **Bernoulli Distribution**: A random variable that can take on two values, 0 and 1, with a probability of 1 - p and p, respectively.
- **Binomial Distribution**: A random variable that can take on a non-negative integer value, representing the number of successes in a fixed number of trials.
- **Poisson Distribution**: A random variable that can take on a non-negative integer value, representing the number of events in a fixed interval of time or space.

### Properties of Discrete Random Variables

- **Uniqueness**: Each value of a discrete random variable is unique.
- **Countability**: A discrete random variable can take on a countable number of values.
- **Probability**: Each value of a discrete random variable has a non-negative probability.

### Examples

- A coin toss can be modeled as a Bernoulli random variable, where heads is 1 and tails is 0.
- The number of heads in a deck of cards can be modeled as a binomial random variable.
- The number of phone calls received in a given time period can be modeled as a Poisson random variable.

## **Continuous Random Variables**

### Definition

A continuous random variable is a variable that can take on any value within a given range or interval. These values are usually represented as real numbers.

### Types of Continuous Random Variables

- **Uniform Distribution**: A random variable that can take on any value within a given range, with a probability density function (pdf) that is constant over the interval.
- **Normal Distribution**: A random variable that can take on any value within a given range, with a pdf that follows a bell-shaped curve.

### Properties of Continuous Random Variables

- **Continuity**: A continuous random variable can take on any value within a given range or interval.
- **Density**: A continuous random variable has a probability density function (pdf) that describes the probability of each value.
- **Area**: The area under the pdf curve represents the probability of the random variable taking on values within a given range.

### Examples

- The height of a person can be modeled as a continuous random variable, with a normal distribution.
- The time it takes to complete a task can be modeled as a continuous random variable, with a uniform distribution.

## **Random Variables and Their Distributions**

### Discrete Uniform Distribution

A discrete random variable that can take on any value within a given range, with a probability of 1/n for each value, where n is the number of values.

### Discrete Bernoulli Distribution

### Continuous Uniform Distribution

A random variable that can take on any value within a given range, with a probability density function (pdf) that is constant over the interval.

### Normal Distribution

A random variable that can take on any value within a given range, with a pdf that follows a bell-shaped curve.

## **Expected Value and Variance**

### Expected Value

The expected value of a random variable is the sum of each possible value multiplied by its probability. It represents the long-run average value of the random variable.

### Variance

The variance of a random variable is a measure of its spread or dispersion. It represents the average squared difference between each possible value and the expected value.
