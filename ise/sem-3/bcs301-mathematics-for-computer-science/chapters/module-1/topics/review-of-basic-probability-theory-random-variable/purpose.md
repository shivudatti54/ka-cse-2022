### Learning Purpose: Review of Basic Probability Theory & Random Variables

**1. Why is this topic important?**
Probability theory is the fundamental language of uncertainty, which is central to nearly every domain of computer science. From designing randomized algorithms and machine learning models to analyzing network traffic and system performance, a rigorous understanding of probability is essential for building efficient and intelligent computational systems.

**2. What will students learn?**
Students will review core probability concepts, including sample spaces, events, and axioms. They will then formally define and differentiate between discrete and continuous random variables. This includes learning to calculate and interpret key properties such as probability mass functions (PMFs), probability density functions (PDFs), cumulative distribution functions (CDFs), expected value, and variance.

**3. How does it connect to other concepts?**
This foundational review is a direct prerequisite for the subsequent topics in this module, such as specific distributions (e.g., Binomial, Poisson, Normal). It provides the necessary tools for later courses in statistics, machine learning (e.g., Bayesian methods), data science, randomized algorithms, queuing theory, and cryptography.

**4. Real-world applications**
These concepts are applied to model real-world stochastic processes. Examples include predicting user behavior on a website (discrete), simulating noise in a sensor's signal (continuous), calculating the expected runtime of a probabilistic algorithm, and assessing risk in financial or security models.
