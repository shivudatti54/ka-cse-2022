# Learning Objectives

After studying this topic, you should be able to:

1. Explain the three levels of scheduling (long-term, short-term, medium-term) and describe their roles in process management.

2. Analyze and compare scheduling algorithms using the five key criteria: CPU utilization, throughput, turnaround time, waiting time, and response time.

3. Apply the First-Come-First-Served (FCFS) scheduling algorithm to calculate waiting times, turnaround times, and completion times for a given set of processes.

4. Solve problems using Shortest Job First (SJF) and Shortest Remaining Time First (SRTF) algorithms, demonstrating understanding of both preemptive and non-preemptive variants.

5. Execute Round Robin scheduling with a given time quantum, tracking process execution through multiple cycles and calculating performance metrics.

6. Evaluate the trade-offs between different scheduling algorithms including their advantages, disadvantages, and suitability for various system types (batch, interactive, real-time).

7. Describe the challenges and techniques involved in multiple-processor scheduling including processor affinity and load balancing.

8. Distinguish between thread scheduling at user-level and kernel-level, and explain how this affects overall system performance.
