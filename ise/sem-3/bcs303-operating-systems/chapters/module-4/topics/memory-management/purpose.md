# Learning Objectives

After studying this topic, you should be able to:

1. Explain the fundamental objectives of memory management including relocation, protection, sharing, and logical organization

2. Compare and contrast contiguous memory allocation with paging and segmentation techniques

3. Analyze the working of page tables and translate logical addresses to physical addresses using page table entries

4. Evaluate different page replacement algorithms including FIFO, LRU, and OPTIMAL, and calculate page fault counts for given reference strings

5. Describe the concept of virtual memory and explain how demand paging enables execution of programs larger than physical memory

6. Identify the causes of thrashing and propose solutions based on the working set model

7. Understand copy-on-write technique and its role in efficient memory management during process creation

8. Differentiate between internal and external fragmentation and explain how paging addresses external fragmentation
