# System Structures: What Operating Systems Do

### Overview

This topic covers the main system structures that operating systems are involved in.

### Key Points

- **Process Structure**:
  - Process: A program in execution
  - Process control block (PCB): Stores information about a process
  - Process scheduling: Allocating time slices to processes
- **Job Structure**:
  - Job: A set of processes that share resources
  - Job scheduling: Allocating resources to jobs
- **Thread Structure**:
  - Thread: A lightweight process
  - Thread scheduling: Scheduling threads within a process
- **Memory Structure**:
  - Virtual memory: Combining physical and secondary storage
  - Memory management: Allocating and deallocating memory
- **File System Structure**:
  - File system: Organizing and managing files
  - File management: Creating, deleting, and accessing files
- **Network Structure**:
  - Network interface: Connecting to a network
  - Network protocol: Establishing communication between devices

### Important Formulas, Definitions, and Theorems

- **Priority scheduling algorithm**: Schedules processes based on their priority
- **Round-robin scheduling algorithm**: Schedules processes in a circular order
- **First-Come-First-Served (FCFS) scheduling algorithm**: Schedules processes in the order they arrive
- **Memory hierarchy**: A layered structure of memory levels (e.g., L1, L2, L3 cache, main memory)
