# Learning Objectives

After studying this topic, you should be able to:

1. Identify and describe the four fundamental components of a computer system: CPU, memory, I/O devices, and system bus

2. Explain the Fetch-Decode-Execute instruction cycle and its significance in CPU operation

3. Understand the interrupt-driven architecture and describe the sequence of events during interrupt handling

4. Compare and contrast programmed I/O, interrupt-driven I/O, and DMA communication methods

5. Analyze memory hierarchy and calculate effective memory access time considering cache hit rates

6. Describe multiprocessor and multi-core architectures and their implications for operating system design

7. Explain the relationship between hardware architecture and operating system resource management

8. Understand memory addressing schemes including physical and logical addresses
