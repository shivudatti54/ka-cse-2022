# Learning Objectives

After studying this topic, you should be able to:

1. Explain the role of an operating system as a resource manager and describe how it allocates CPU, memory, and I/O resources among competing processes.

2. Compare and contrast the three main types of user interfaces: command-line interface (CLI), graphical user interface (GUI), and touch-based interface, with their respective advantages and disadvantages.

3. Define system calls and explain their role as the interface between user programs and the operating system kernel.

4. Categorize system calls into their major types and provide examples of each category used in modern operating systems.

5. Trace the evolution of operating systems from early batch processing through time-sharing to modern multiprocessing systems.

6. Identify and explain the key goals of an operating system including convenience, efficiency, throughput, reliability, and security.

7. Analyze real-world scenarios to determine how the operating system performs its functions as both a resource manager and service provider.
