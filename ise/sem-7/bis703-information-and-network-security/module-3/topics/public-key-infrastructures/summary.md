# Public Key Infrastructures

Summary

- Public Key Infrastructure (PKI) is a set of policies, procedures, and technologies used to manage public-private key pairs and digital certificates.
- PKI consists of components such as Certificate Authority (CA), Registration Authority (RA), certificate repository, and private key generator.
- The working of PKI involves key pair generation, certificate request, certificate issuance, certificate deployment, and certificate verification.

### Important Formulas, Definitions, and Theorems

- Public Key Infrastructure (PKI): A set of policies, procedures, and technologies used to manage public-private key pairs and digital certificates.
- Certificate Authority (CA): A trusted entity that issues digital certificates to individuals, devices, and organizations.
- X.509 certificates: A type of digital certificate that uses the X.509 standard.

### Key Points

- PKI is used to secure web browsing, email encryption, and virtual private networks (VPNs).
- PKI management involves managing the lifecycle of digital certificates, including issuance, deployment, verification, and revocation.
- Symmetric key distribution uses the same key for encryption and decryption, while asymmetric key distribution uses a pair of keys.
- Digital signatures authenticate the sender and ensure data integrity.
