# Man-in-the-middle Attack

Summary

- A man-in-the-middle (MITM) attack is a type of cyber attack where an attacker intercepts and alters the communication between two parties.
- MITM attacks can compromise the confidentiality, integrity, and authenticity of data.
- There are several types of MITM attacks, including active eavesdropping, passive eavesdropping, SSL stripping, and Wi-Fi eavesdropping.

### Important Formulas, Definitions, and Theorems

- **Man-in-the-middle attack**: A type of cyber attack where an attacker intercepts and alters the communication between two parties.
- **Active eavesdropping**: A type of MITM attack where the attacker actively intercepts and modifies the data being transmitted.
- **Passive eavesdropping**: A type of MITM attack where the attacker only intercepts and reads the data being transmitted, without modifying it.

### Key Points

- MITM attacks can be prevented using secure communication protocols, encryption, and digital signatures.
- Verifying the authenticity of a recipient's identity is essential to prevent MITM attacks.
- Certificate pinning, public key pinning, and hash-based authentication are detection techniques used to identify MITM attacks.
- MITM attacks can have a significant impact on the security of a system or network.
- Examples of MITM attacks include SSL stripping and Wi-Fi eavesdropping.
- MITM attacks can be used to steal sensitive information, inject malware, or modify data.
