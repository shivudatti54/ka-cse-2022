# Learning Objectives

After studying this topic, you should students able to:

1. **Explain** the mathematical foundations required for RSA, including prime numbers, greatest common divisor (GCD), and Euler's totient function φ(n).

2. **Describe** the complete RSA key generation process, including the selection of two large primes, computation of modulus n and φ(n), and selection of public and private exponents.

3. **Apply** the RSA encryption algorithm to convert plaintext into ciphertext using the public key (e, n).

4. **Apply** the RSA decryption algorithm to recover plaintext from ciphertext using the private key (d, n).

5. **Analyze** why RSA decryption correctly recovers the original message using Euler's theorem and the properties of modular exponentiation.

6. **Implement** the extended Euclidean algorithm to compute the private exponent d as the modular multiplicative inverse of the public exponent e modulo φ(n).

7. **Solve** numerical problems involving RSA key generation, encryption, and decryption with small prime numbers.

8. **Evaluate** the security implications of RSA parameters and explain why large prime numbers are necessary for practical security.
