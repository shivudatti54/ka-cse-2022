# Learning Objectives

After studying this topic, you should students able to:

1. Define elliptic curves over finite fields and explain their mathematical properties essential for cryptographic applications.

2. Explain the arithmetic operations of point addition and scalar multiplication on elliptic curves.

3. Apply the Elliptic Curve Diffie-Hellman (ECDH) key exchange protocol to establish shared secrets between two parties.

4. Analyze the key generation process in ECC and compute public/private key pairs for secure communication.

5. Compare the security strength and efficiency of ECC with RSA and other public-key cryptosystems.

6. Apply ECDSA (Elliptic Curve Digital Signature Algorithm) to generate and verify digital signatures.

7. Explain why ECC achieves comparable security to RSA with significantly smaller key sizes.

8. Identify real-world applications of ECC in secure web browsing, cryptocurrencies, and mobile communications.
