# Learning Objectives
After studying this topic, you should be able to:
1. Define and distinguish between the core principles of cryptography: confidentiality, integrity, authentication, and non-repudiation
2. Explain the components of the CIA triad and how cryptography supports each principle
3. Compare and contrast symmetric and asymmetric cryptography in terms of keys, performance, and use cases
4. Identify real-world applications of cryptographic principles in technologies like TLS, digital signatures, and password storage
5. Describe the fundamental concepts of plaintext, ciphertext, encryption, decryption, and cryptographic keys
6. Understand the historical evolution of cryptography and its impact on modern security practices
