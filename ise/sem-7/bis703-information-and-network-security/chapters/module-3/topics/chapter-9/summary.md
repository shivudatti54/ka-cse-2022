### Chapter 9 Revision Notes

#### Information and Network Security

#### Module: 10 hours

### Key Topics

- **Types of Attacks**
  - Types of malware: viruses, worms, trojans, spyware, adware
  - Types of phishing: email phishing, SMS phishing, smishing
  - Denial of Service (DoS) and Distributed Denial of Service (DDoS) attacks
- **Network Security Measures**
  - Firewalls
  - Intrusion Detection and Prevention Systems (IDPS)
  - Virtual Private Networks (VPNs)
- **Cryptography**
  - Types of encryption: symmetric, asymmetric, hash
  - Encryption algorithms: AES, RSA, SHA
- **Secure Communication**
  - Secure Sockets Layer/Transport Layer Security (TLS) protocol
  - Secure Hypertext Transfer Protocol (HTTPS)

### Important Formulas and Definitions

- **Ciphertext**: encrypted data
- **Plaintext**: decrypted data
- **Key**: piece of information used to encrypt/decrypt data
- **Cipher**: algorithm used for encryption
- **Hash function**: one-way function that takes input data of any size and produces a fixed-size output
- **Digital signature**: unique code that verifies authenticity of data
- **Public key**: key used for encryption, private key for decryption
- **RSA Algorithm**: asymmetric encryption algorithm
- **AES Algorithm**: symmetric encryption algorithm

### Theorems and Concepts

- **Kerberos**: authentication protocol
- **SSL/TLS Handshake**: process of establishing secure connection
- **Elliptic Curve Cryptography (ECC)**: asymmetric encryption algorithm

- Review types of attacks and network security measures
