# Learning Objectives
After studying this topic, you should be able to:
1.  Define cryptography and articulate its four core principles: confidentiality, integrity, authentication, and non-repudiation.
2.  Explain the mechanics of historical ciphers like the Caesar and Vigenère ciphers, and demonstrate how they can be broken (e.g., via brute-force or frequency analysis).
3.  Differentiate between monoalphabetic and polyalphabetic substitution ciphers and state the cryptographic weakness addressed by the latter.
4.  Describe the significance of mechanical encryption devices like the Enigma machine and the factors that led to their cryptanalysis.
5.  Contrast symmetric and asymmetric cryptographic systems, identifying the key distribution problem and how public-key cryptography solves it.
6.  Identify the major modern cryptographic algorithms (DES, AES, RSA) and the fundamental mathematical problem that provides security for each (e.g., factoring for RSA).
