# Learning Objectives

After studying this topic, you should be able to:

1.  Define a cryptographic hash function and list its core properties.
2.  Differentiate between pre-image resistance, second pre-image resistance, and collision resistance.
3.  Explain the Merkle-Damgård iterated structure used in building many hash functions.
4.  Describe common applications of hash functions, including data integrity verification, password storage, and digital signatures.
5.  Explain the role of a 'salt' in secure password hashing and its security benefits.
6.  Compare and contrast hashing with encryption.
