# Textbook: 2- Chapter 12 (12.1 to 12.6)

### Key Points

- **Cryptographic Hash Functions**
  - Definition: One-way function that maps input to fixed-size output (hash value)
  - Properties: Deterministic, Non-invertible, Fixed-size output
  - Formula: `h(x) = hash(x)`
- **Digital Signatures**
  - Definition: Cryptographic signature that authenticates the signer and data
  - Components: Public key, Private key, Hash function
  - Formula: `signature = signing_key \* hash(data)`
- **Data Integrity and Authentication**
  - Definition: Ensuring data is accurate and not tampered with
  - Techniques: Hash functions, Digital signatures, Message Authentication Codes (MACs)
- **Cryptography Basics**
  - Definition: Algorithms for secure communication
  - Types: Symmetric, Asymmetric, Public-key
- **Key Exchange Protocols**
  - Definition: Secure exchange of cryptographic keys
  - Protocols: RSA, Diffie-Hellman, Elliptic Curve Cryptography (ECC)
- **Secure Sockets Layer/Transport Layer Security (SSL/TLS)**
  - Definition: Secure communication between client and server
  - Components: Handshake, Key exchange, Data encryption
- **Key Management**
  - Definition: Secure storage and distribution of cryptographic keys
  - Techniques: Key storage, Key distribution, Key revocation
