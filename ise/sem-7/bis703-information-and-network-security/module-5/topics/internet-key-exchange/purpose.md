# Learning Objectives

After studying this topic, you should students able to:

1. Explain the basics of Internet Key Exchange (IKE) and its role in establishing IPsec Security Associations (SAs).
2. Describe the differences between Main Mode and Aggressive Mode in IKE.
3. Configure IKE to use preshared keys instead of public key authentication.
4. Explain the importance of authentication and identity protection in IKE.
5. Describe the role of Diffie-Hellman key exchange in IKE.
6. Understand the security implications of using IKE to establish IPsec SAs.
7. Apply IKE to establish IPsec SAs in a real-world scenario.
