## **Introduction**

Parallel computing is a technique of solving complex problems by dividing them into smaller sub-problems and processing them simultaneously using multiple processing units. In this topic, we will explore the concept of recording the difference in execution time, which is a crucial aspect of parallel computing.

## **Historical Context**

The concept of parallel computing dates back to the 1950s, when the first parallel computers were developed. These early computers used multiple processing units to perform calculations in parallel, but the execution time was often dominated by the serial execution of the control flow. In the 1960s, the first multi-processors were developed, which allowed multiple processes to run concurrently on a single processor.

In the 1980s, the first parallel computing architectures were developed, including the Maspar Machine and the Cray-2. These architectures used multiple processors to perform calculations in parallel, but the execution time was often dominated by the communication overhead between processors.

In the 1990s, the development of the Internet and the widespread use of the World Wide Web led to the development of distributed computing, where multiple computers were connected via the Internet to form a network. This led to the development of parallel computing frameworks, such as PVM (Parallel Virtual Machine) and MPI (Message Passing Interface).

## **Modern Developments**

In recent years, the development of parallel computing has accelerated with the advent of new technologies, including:

- **GPU Computing**: Graphics Processing Units (GPUs) are now being used for parallel computing, thanks to the development of CUDA and OpenCL.
- **Cloud Computing**: Cloud computing platforms, such as Amazon Web Services (AWS) and Microsoft Azure, provide scalable parallel computing resources.
- **Hadoop**: Hadoop is an open-source framework for distributed computing, which allows for the processing of large datasets in parallel.
- **DNNs**: Deep Neural Networks (DNNs) are being used for parallel computing, thanks to the development of frameworks, such as TensorFlow and PyTorch.
