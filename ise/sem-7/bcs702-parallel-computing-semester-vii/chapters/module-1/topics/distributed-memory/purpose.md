# Learning Objectives
After studying this topic, you should be able to:
1.  Define and distinguish between shared-memory and distributed-memory parallel architectures.
2.  Explain the hardware organization of both architectures, including UMA/NUMA and cluster configurations.
3.  Identify the key advantages and disadvantages of each architecture, particularly regarding scalability, programming complexity, and cost.
4.  Associate the correct programming model (OpenMP vs. MPI) with each architecture and explain why it is appropriate.
5.  Describe the challenges of cache coherence in shared-memory systems and communication overhead in distributed-memory systems.
6.  Recognize a hybrid architecture and suggest an appropriate hybrid programming approach (e.g., MPI + OpenMP).
