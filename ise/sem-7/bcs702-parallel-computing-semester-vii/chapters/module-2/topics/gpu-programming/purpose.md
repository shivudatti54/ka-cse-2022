# Learning Objectives
After studying this topic, you should be able to:
1. Explain the fundamental architectural differences between CPUs and GPUs
2. Describe the GPU memory hierarchy and identify appropriate use cases for each memory type
3. Understand and apply the thread hierarchy model (threads, blocks, grids) in GPU programming
4. Calculate theoretical speedup using Amdahl's Law for parallel programs
5. Differentiate between CUDA and OpenCL programming models
6. Implement basic GPU programs using appropriate memory management techniques
