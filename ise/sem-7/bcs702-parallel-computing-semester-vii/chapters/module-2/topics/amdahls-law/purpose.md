# Learning Objectives
After studying this topic, you should be able to:
1. State Amdahl's Law and explain its significance in parallel computing
2. Calculate theoretical speedup using Amdahl's Law formula given parallel fraction and processor count
3. Identify sequential components in parallel programs that limit speedup
4. Analyze the diminishing returns of adding more processors to a parallel system
5. Apply Amdahl's Law to GPU programming scenarios considering data transfer overheads
6. Compare Amdahl's Law with Gustafson's Law and understand their different perspectives
