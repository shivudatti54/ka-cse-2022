# Learning Objectives
After studying this topic, you should be able to:
1.  Explain the challenges of IoT data (Volume, Velocity, Variety) that necessitate a platform like Apache Hadoop.
2.  Describe the architecture and core components of Hadoop: HDFS for storage and YARN for resource management.
3.  Explain the MapReduce programming model and its application in batch processing historical IoT data.
4.  Differentiate between batch processing (MapReduce) and real-time processing (Spark Streaming/Storm) in the context of IoT.
5.  Identify other components of the Hadoop ecosystem (e.g., Kafka, Spark, HBase) and their roles in a complete IoT data pipeline.
6.  Diagram a typical IoT data analytics pipeline, showing the flow of data from ingestion to storage, processing, and analysis.
