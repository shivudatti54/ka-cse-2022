# Learning Objectives
After studying this topic, you should be able to:
1. Classify optimization problems based on variable types (continuous, discrete, mixed)
2. Distinguish between linear and nonlinear optimization problems
3. Identify convex and non-convex optimization problems and understand their properties
4. Differentiate between local and global optima in optimization problems
5. Select appropriate solution methods based on problem classification
6. Formulate real-world problems using proper optimization problem structures
