# **TB1: Ch 6: 6.1-6.5 - Big Data Analytics**

### Overview

- Big Data Analytics is the process of extracting insights and knowledge from large and complex datasets.
- MongoDB is a NoSQL database that is used for big data analytics.

### Key Concepts

- **Data Warehousing**: a centralized repository of data that is used for business intelligence and analytics.
- **ETL (Extract, Transform, Load)**: a process of extracting data from various sources, transforming it into a standardized format, and loading it into a data warehouse.
- **Big Data Analytics**: a process of extracting insights and knowledge from large and complex datasets.

### MongoDB Concepts

- **Document-based database**: a database that stores data in the form of documents, each with a unique identifier.
- **Schema-less database**: a database that does not require a predefined schema to store data.
- **NoSQL database**: a database that does not follow the traditional structured query language (SQL) rules.

### Big Data Analytics Concepts

- **Hadoop**: an open-source framework for big data processing and storage.
- **MapReduce**: a programming model for processing large data sets in parallel across a cluster of computers.
- **NoSQL databases**: databases that are designed to handle large amounts of unstructured or semi-structured data.

### Important Formulas and Definitions

- **Data volume**: the amount of data being stored or processed.
- **Data velocity**: the speed at which data is generated and processed.
- **Data variety**: the different types of data being stored or processed.
- **Data veracity**: the accuracy and reliability of the data being stored or processed.

### Important Theorems

- **Bloom Filter**: a data structure that is used to quickly determine whether an element is a member of a set or not.
- **Hash Function**: a mathematical function that takes an input and produces a fixed-size output.

### Quick Revision Notes

- Big Data Analytics is the process of extracting insights and knowledge from large and complex datasets.
- MongoDB is a NoSQL database that is used for big data analytics.
- Data Warehousing and ETL are processes that are used to extract, transform, and load data into a data warehouse.
- Hadoop and MapReduce are tools that are used for big data processing and storage.
