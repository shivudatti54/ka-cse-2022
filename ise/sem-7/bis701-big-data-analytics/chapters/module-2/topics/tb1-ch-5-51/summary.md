# TB1: Ch 5: 5.1- Big Data Analytics

====================================================================

## Overview

- Big Data Analytics refers to the process of examining data sets that are large, complex, and varied.
- Hadoop is a popular framework used for big data analytics due to its scalability and ability to handle large data sets.

## Why Hadoop?

---

- **Scalability**: Hadoop can handle large data sets and scale horizontally.
- **Flexibility**: Hadoop can handle different data formats and types.
- **Cost-effectiveness**: Hadoop is an open-source framework, making it cost-effective.

## Why Not RDBMS?

---

- **Scalability limitations**: RDBMS have limitations in terms of scalability and handling large data sets.
- **Data structure limitations**: RDBMS are designed for structured data and may not be suitable for unstructured or semi-structured data.
- **Data consistency limitations**: RDBMS ensure data consistency, but may not be able to handle large amounts of data.

## RDBMS vs Hadoop

---

|                      | RDBMS      | Hadoop                       |
| -------------------- | ---------- | ---------------------------- |
| **Data Structure**   | Structured | Unstructured/Semi-structured |
| **Scalability**      | Limited    | Scalable                     |
| **Data Consistency** | High       | Low                          |
| **Cost**             | High       | Low                          |

## Important Formulas/Definitions/ Theorems

---

- **Hadoop Ecosystem**: Hadoop ecosystem consists of Hadoop Distributed File System (HDFS), MapReduce, and other related tools.
- **MapReduce**: MapReduce is a programming model used in Hadoop for processing large data sets.
- **Big Data**: Big data refers to the large, complex, and varied data sets that require new forms of processing and analysis.
- **NoSQL Database**: NoSQL database is a type of database that does not use the traditional structured query language (SQL) to manage and retrieve data.

## Key Points

---

- Hadoop is a popular framework for big data analytics due to its scalability and flexibility.
- RDBMS have limitations in terms of scalability, data structure, and data consistency.
- Hadoop ecosystem consists of HDFS, MapReduce, and other related tools.
- Big data refers to large, complex, and varied data sets that require new forms of processing and analysis.
