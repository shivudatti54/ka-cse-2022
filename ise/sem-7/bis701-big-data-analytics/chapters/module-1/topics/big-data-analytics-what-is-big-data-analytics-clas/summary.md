# **Big Data Analytics Summary**

### What is Big Data Analytics?

- Big Data Analytics: The process of extracting insights and knowledge from large, complex, and diverse sets of data.
- Definition: "Big Data" refers to the large volume, velocity, variety, and veracity of data generated by various sources.

### Classification of Analytics

- **Descriptive Analytics**: The process of analyzing historical data to understand what happened. (e.g., Sales Analysis)
- **Diagnostic Analytics**: The process of analyzing data to identify the cause of a problem. (e.g., Root Cause Analysis)
- **Predictive Analytics**: The process of analyzing data to predict future outcomes. (e.g., Forecasting)
- **Prescriptive Analytics**: The process of analyzing data to provide recommendations for action. (e.g., Optimization)

### Importance of Big Data Analytics

- **Improved Decision Making**: Provides insights that inform business decisions.
- **Increased Efficiency**: Automates processes and reduces manual effort.
- **Enhanced Customer Experience**: Offers personalized experiences and recommendations.
- **Competitive Advantage**: Provides a unique competitive advantage through data-driven insights.

### Technologies used in Big Data Environment

- **Hadoop**: An open-source framework for processing large data sets.
- **Spark**: An open-source framework for processing large data sets.
- **NoSQL Databases**: Designed for handling large amounts of unstructured data.
- **Cloud Computing**: Provides scalable infrastructure for processing large data sets.

### Key Formulas and Definitions

- **Volume**: The amount of data in a dataset.
- **Velocity**: The speed at which data is generated.
- **Variety**: The types of data in a dataset (e.g., structured, unstructured, semi-structured).
- **Veracity**: The accuracy of the data.
- **Data Warehousing**: A centralized repository for storing and managing data.
- **ETL (Extract, Transform, Load)**: The process of extracting data from multiple sources, transforming it into a standardized format, and loading it into a target system.

### Important Theorems

- **No Free Lunch Theorem**: States that all optimization algorithms have a trade-off between computational complexity and performance.
- **Curse of Dimensionality**: States that as the number of features in a dataset increases, the accuracy of machine learning models decreases.
