# **TB1: Big Data Analytics Revision Notes**

### Chapter 1: Definition and Characteristics

- **Definition of Big Data:**
  - Large volume of data
  - High velocity of data
  - Variety of data (structured, semi-structured, unstructured)
  - Veracity of data (quality and accuracy)
- **Characteristics of Big Data:**
  - Scalability
  - Interconnectedness
  - Real-time processing
  - Real-time analytics

### Chapter 2: Classification and Evolution of Big Data

- **Classification of Big Data:**
  - Big Data 1.0: Small-scale, structured data analysis
  - Big Data 2.0: Large-scale, semi-structured data analysis
  - Big Data 3.0: Real-time, unstructured data analysis
- **Evolution of Big Data:**
  - Hadoop (2005)
  - NoSQL databases (2009)
  - Cloud computing (2010)
  - Artificial intelligence (2015)

### Chapter 3: Big Data Technologies

- **Hadoop:**
  - MapReduce programming model
  - Distributed file system (HDFS)
  - YARN (Yet Another Resource Negotiator)
- **Spark:**
  - In-memory data processing
  - Stream processing
  - Machine learning
- **NoSQL Databases:**
  - Key-Value Stores (e.g., Riak)
  - Column-Family Stores (e.g., Cassandra)
  - Document-Oriented Stores (e.g., MongoDB)

### Chapter 4: Big Data Analytics

- **Data Mining:**
  - Definition: Extraction of knowledge from large datasets
  - Types: Supervised, Unsupervised, Semi-Supervised
- **Machine Learning:**
  - Definition: Development of algorithms to learn from data
  - Types: Supervised, Unsupervised, Semi-Supervised

### Important Formulas and Definitions

- **Hadoop MapReduce Formula:**
  - `map(key, value) -> key, value`
  - `reduce(key, values) -> output`
- **Big Data Analytics Theorem:**
  - "With enough data, any problem can be solved."
  - "Data is the new oil."

### Important Theorems

- **No-Surplus Theorem:**
  - If data is abundant, it can be stored and processed.
- **Scalability Theorem:**
  - If data is scalable, it can be processed in real-time.

Note: This summary is a concise overview of the key points and formulas related to Big Data Analytics. It is not an exhaustive list and is intended to serve as a quick revision guide for exams.
