# Blockchain Technology

**Ethereum 101, Introduction, Ethereum clients and releases, The Ethereum stack, Ethereum blockchain, 235-269**

### Key Concepts

- **Blockchain Definition**: A distributed digital ledger technology that records transactions across a network of computers in a secure and transparent manner.
- **Hash Functions**: One-way mathematical functions that take input data of any size and produce a fixed-size output (hash value).
- **Cryptography**: The practice of using mathematical algorithms to secure data and protect against unauthorized access.
- **Smart Contracts**: Self-executing contracts with the terms of the agreement written directly into code.

### Ethereum Stack

- **Client-Server Architecture**: The Ethereum blockchain is a decentralized network of nodes that communicate with each other to validate and record transactions.
- **Protocol Layer**: The layer responsible for the underlying communication protocol between nodes.
- **Network Layer**: The layer responsible for the overall network topology and connectivity.

### Ethereum Blockchain

- **Block Time**: The time it takes for a new block to be added to the blockchain (approximately 15 seconds).
- **Block Reward**: The reward given to miners for validating a new block and adding it to the blockchain (currently 2 ETH).
- **Gas**: The unit of measurement for the computational effort required to execute a transaction or contract.

### Important Formulas and Theorems

- **Hash Function Formula**: `h(x) = p(x) || s(x)`
- **Block Reward Formula**: `R = 2 * 2^21`
- **Gas Limit Formula**: `L = t * 17`

### Important Definitions

- **Miner**: A node on the Ethereum network that validates transactions and adds them to blocks.
- **Node**: A computer on the Ethereum network that participates in the validation and verification of transactions.
- **Transaction**: A message sent from one node to another on the Ethereum network.

### Important Theorems

- **Shannon's Source-Channel Coding Theorem**: A fundamental theorem in information theory that describes the relationship between the capacity of a channel and the rate of a source.
