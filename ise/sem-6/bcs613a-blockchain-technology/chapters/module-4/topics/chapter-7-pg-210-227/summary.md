# Blockchain Technology

**Chapter 7: pg 210-227**

### Key Concepts

- **Blockchain Architecture**
  - Decentralized network of nodes
  - Distributed ledger technology
  - Cryptographic algorithms
- **Ethereum Blockchain Components**
  - Gas: unit of account for transaction fees
  - Block: group of transactions
  - Block Header: contains metadata for block
  - Block Number: unique identifier for each block
  - Transaction: data stored on the blockchain
- **Consensus Mechanisms**
  - Proof-of-Work (PoW): energy-intensive algorithm
  - Proof-of-Stake (PoS): energy-efficient algorithm
  - Byzantine Fault Tolerance (BFT): consensus algorithm
- **Smart Contracts**
  - Self-executing contracts with the terms of the agreement
  - Written in Solidity programming language
  - Run on the Ethereum Virtual Machine (EVM)
- **Gas Limit and Gas Price**
  - Gas limit: maximum amount of gas allowed for transaction
  - Gas price: price paid per unit of gas used

### Important Formulas and Definitions

- **Block Reward**: amount of new Ether created and distributed to miners
- **Transaction Fee**: fee paid to miners for processing transactions
- **Hash Function**: one-way mathematical function that generates a unique string of characters
- **Public-Key Cryptography**: cryptographic technique used for secure communication
- **Elliptic Curve Cryptography (ECC)**: cryptographic technique used for secure communication

### Theorems and Principles

- **Shannon's Source Coding Theorem**: provides a theoretical basis for blockchain security
- **Kolmogorov Complexity**: measures the complexity of data
- **Information Theory**: provides a mathematical framework for understanding information and entropy

### Quick Revision Points

- Blockchain is a decentralized, distributed ledger technology
- Ethereum blockchain uses a proof-of-work consensus mechanism
- Smart contracts are self-executing contracts with the terms of the agreement
- Gas limit and gas price are used to regulate transaction fees
- Hash functions are used to secure transactions and maintain the integrity of the blockchain
