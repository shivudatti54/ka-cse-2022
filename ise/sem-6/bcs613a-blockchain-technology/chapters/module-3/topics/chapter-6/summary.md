# **Chapter 6 Revision Notes**

## **Blockchain Technology**

- Blockchain is a decentralized, distributed ledger technology.
- It consists of a chain of blocks, each containing a set of transactions.
- Each block is linked to the previous block through a unique code called a "hash."
- The blockchain is maintained by a network of computers (nodes) that verify and record transactions.

## **Key Concepts**

- **Consensus Mechanisms**:
  - Proof of Work (PoW): Energy-intensive process that requires solving complex mathematical problems.
  - Proof of Stake (PoS): Requires validators to "stake" their own cryptocurrency to participate.
- **Smart Contracts**: Self-executing contracts with the terms of the agreement written directly into code.
- **Cryptographic Hash Functions**: One-way mathematical functions that produce a fixed-size string of characters.

## **Theories and Formulas**

- **Shannon's Capacity Theorem**: Calculates the maximum data transfer rate over a communication channel.
- **Golomb-Rachmaninoff Theorem**: Establishes the minimum number of bits required to encode a binary message.
- **Public-Key Cryptography**:
  - **Encryption**: Transforms plaintext into ciphertext using a secret key.
  - **Decryption**: Transforms ciphertext back into plaintext using a corresponding public key.

## **Important Definitions**

- **Blockchain Architecture**: A distributed ledger technology that enables secure, transparent, and tamper-proof transactions.
- **Cryptocurrency**: A digital or virtual currency that uses cryptography for security and is decentralized and trustless.
- **Mining**: The process of verifying transactions and adding them to the blockchain.
