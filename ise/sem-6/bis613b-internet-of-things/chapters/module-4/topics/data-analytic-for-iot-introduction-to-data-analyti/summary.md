# Data Analytics for IoT

### Introduction

- Data analytics for IoT (Internet of Things) involves analyzing and interpreting data generated by connected devices.
- The increasing volume, velocity, and variety of IoT data present new challenges for data analytics.

### Machine Learning

- **Definition:** Machine learning is a subset of AI that enables devices to learn from data and make predictions or decisions.
- **Types:**
  - Supervised learning
  - Unsupervised learning
  - Reinforcement learning
- **Formulas:**
  - Probability Theory: Bayes' theorem, Conditional probability
  - Regression Analysis: Linear regression, Logistic regression

### Big Data Analytics Tools and Technology

- **Definition:** Big data refers to large, complex, and structured data that requires specialized tools and techniques for analysis.
- **Tools:**
  - Hadoop
  - Spark
  - NoSQL databases (e.g., MongoDB, Cassandra)
  - Data warehousing (e.g., Amazon Redshift, Google BigQuery)

### Edge Streaming Analytics

- **Definition:** Edge streaming analytics involves processing and analyzing data in real-time, at the edge of the network.
- **Benefits:**
  - Reduced latency
  - Improved real-time decision-making
- **Technologies:**
  - Edge computing platforms (e.g., AWS IoT Greengrass, Google Cloud IoT Core)
  - Edge analytics platforms (e.g., AWS Lake Formation, Google Cloud AI Platform)

### Key Concepts

- **IoT Data Model:** A standardized framework for representing IoT data.
- **Data Quality:** Ensuring the accuracy, completeness, and consistency of IoT data.
- **Data Governance:** Establishing policies and procedures for managing IoT data.

### Important Formulas and Theorems

- **Bayes' Theorem:** P(A|B) = P(A ∩ B) / P(B)
- **Conditional Probability:** P(A|B) = P(A ∩ B) / P(B)
- **Linear Regression:** y = β0 + β1x + ε
