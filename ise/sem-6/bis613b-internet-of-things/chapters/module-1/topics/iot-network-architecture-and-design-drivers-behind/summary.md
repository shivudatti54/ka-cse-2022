# **IoT Network Architecture and Design: Revision Notes**

### Drivers behind new architecture

- **Increased connectivity**: Growing number of devices and the need for seamless communication between them
- **Enhanced security**: Growing concerns about data security and the need for robust security measures
- **Data analytics**: Growing demand for real-time data analysis and insights
- **Scalability**: Need for IoT networks to scale up or down according to changing requirements

### Comparing IoT Architectures

- **Device-to-Device (D2D)**: Direct communication between devices
- **Device-to-Cloud (D2C)**: Data transmitted to cloud servers for processing and analysis
- **Device-to-Gateway (D2G)**: Data transmitted to a central gateway for processing and distribution

### Simplified IoT Architecture

- **3-Layer Architecture**:
  - **Device Layer**: Devices that make up the IoT network
  - **Communication Layer**: Network infrastructure and protocols
  - **Application Layer**: Software applications that use data from the IoT network
- **4-Layer Architecture**:
  - **Physical Layer**: Devices and connections
  - **Data Link Layer**: Network protocols and communication
  - **Network Layer**: Data transmission and routing
  - **Application Layer**: Software applications that use data from the IoT network

### Core IoT Functional

- **Data Collection**: Gathering data from IoT devices
- **Data Processing**: Analyzing and processing data in real-time
- **Data Storage**: Storing data for later analysis or use
- **Data Analytics**: Extracting insights and knowledge from data

### Important Formulas and Definitions

- **M2M (Machine-to-Machine)**: Communication between devices without human intervention
- **IoT (Internet of Things)**: Network of physical devices, vehicles, home appliances, and other items embedded with sensors, software, and connectivity, allowing them to collect and exchange data.
- **IoT Protocol**: Set of rules and standards for communication between devices and the internet
- **IoT Security**: Measures to prevent unauthorized access, data theft, and other security breaches in IoT networks

### Theorems

- **Moore's Law**: Exponential increase in computing power and reduction in cost over time
- **Bailey's Law**: Exponential increase in the number of devices connected to the internet over time
