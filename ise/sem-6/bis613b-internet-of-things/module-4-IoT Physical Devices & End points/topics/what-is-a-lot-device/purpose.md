# Learning Purpose: What is an IoT Device

**1. Why this topic matters**
IoT devices are the fundamental building blocks of every IoT system, serving as the physical endpoints that sense the environment and perform actions. Understanding what constitutes an IoT device, its core components, and its role within the broader ecosystem is essential before exploring specific hardware platforms like Raspberry Pi and designing complete IoT solutions.

**2. What you will learn**
You will learn the formal definition and classification of IoT devices, including their four core hardware components: sensors/actuators, processing units, communication modules, and power supply. You will understand the difference between microcontrollers (MCU) and microprocessors (MPU), and learn to classify devices based on their primary function such as sensing, actuation, or gateway operations.

**3. How it connects to other topics**
This topic builds on Module 1's physical design concepts and provides the foundation for the Raspberry Pi topics that follow in Module 4. Understanding device characteristics informs the IoT levels selection from Module 1 and the management requirements from Module 2. The device types discussed here are programmed using the Python skills from Module 3.

**4. Real-world relevance**
IoT devices range from simple temperature sensors in smart homes to complex industrial controllers in manufacturing plants. Understanding device capabilities and constraints helps engineers select the right hardware for specific applications, whether designing a low-power agricultural sensor that runs on batteries for years or a high-performance edge computing gateway.
