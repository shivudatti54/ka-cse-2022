# **Cloud Computing: Virtual Machines and Virtualization of Clusters and Data Centers**

**Chapter 3:**

- **3.1: Introduction to Virtualization**
  - Definition of Virtualization: creation of a virtual environment to support multiple operating systems on a single physical machine
  - Types of Virtualization: Server Virtualization, Desktop Virtualization, Network Virtualization
- **3.2: Virtual Machine (VM) Architecture**
  - Components of a VM: Hypervisor, VM Monitor, Virtual Hardware
  - VM Life Cycle: Creation, Configuration, Execution, Termination
- **3.3: Types of Hypervisors**
  - Bare-Metal Hypervisors: VMware, XenServer
  - Hosted Hypervisors: VirtualBox, Hyper-V
  - Bare-Metal and Hosted Hypervisors: Comparison
- **3.4: Virtual Machine Monitor (VMM)**
  - Definition of VMM: software component that executes a hypervisor
  - Functions of VMM: Resource Allocation, Process Management
- **3.5: Clustering and Data Center Virtualization**
  - Definition of Clustering: collection of multiple servers connected to a single node
  - Clustering Types: Shared Disk Clustering, Shared Nothing Clustering
  - Data Center Virtualization: Network Virtualization, Storage Virtualization

**Important Formulas and Definitions:**

- **Definition of Virtualization:** The process of dividing a physical hardware resource into multiple virtual resources.
- **Hypervisor Definition:** A piece of software that creates and manages virtual machines.
- **VM Monitor Definition:** A software component that executes a hypervisor and manages virtual machines.

**Important Theorems:**

- **Ultimate Virtualization Theorem:** A virtual machine can run a single operating system on a single physical machine.
- **Hypervisor Theorem:** A hypervisor can create and manage multiple virtual machines on a single physical machine.
