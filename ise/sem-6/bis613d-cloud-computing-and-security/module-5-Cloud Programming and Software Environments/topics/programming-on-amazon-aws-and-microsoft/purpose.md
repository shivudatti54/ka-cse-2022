# Learning Objectives

After studying this topic, you should students able to:

1.  Explain the shared responsibility model and its implications for cloud application development.
2.  Differentiate between the core compute, storage, and database services offered by AWS and Azure.
3.  Describe the event-driven programming paradigm and identify use cases for serverless functions (AWS Lambda, Azure Functions).
4.  Articulate the purpose and benefits of Infrastructure as Code (IaC) and name the primary IaC tools for AWS and Azure.
5.  Compare and contrast the programming models and service ecosystems of AWS and Azure.
6.  Implement basic interactions with cloud services (e.g., listing resources) using provided SDK code examples.
