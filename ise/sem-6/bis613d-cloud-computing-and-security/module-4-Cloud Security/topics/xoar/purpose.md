# Learning Objectives

After studying this topic, you should students able to:

1. Explain the security problems of monolithic server applications (large attack surface, excessive privileges, no internal isolation).
2. Describe the XOAR approach of decomposing applications into isolated, least-privilege compartments.
3. Explain the two core principles: privilege separation (decomposition) and principle of least privilege.
4. Identify the key components of XOAR: Compartment Manager, Policy Engine, and Inter-Compartment Communication.
5. Discuss the benefits of XOAR including reduced blast radius, smaller attack surface, and easier auditing.
6. Explain how XOAR complements VM and OS security by providing application-level defense in depth.
