# Learning Objectives

After studying this topic, you should students able to:

1. Explain why cloud security is the top concern for organizations adopting cloud computing.
2. Identify and describe the major cloud security risks and threats, including the CSA Treacherous Twelve.
3. Explain the shared responsibility model and how it divides security duties between provider and customer.
4. Define Privacy Impact Assessment (PIA) and explain its importance in cloud deployments.
5. Describe the steps involved in conducting a PIA for a cloud-based system.
6. Discuss the role of regulatory frameworks (GDPR, HIPAA, ISO 27001) in cloud security and privacy.
