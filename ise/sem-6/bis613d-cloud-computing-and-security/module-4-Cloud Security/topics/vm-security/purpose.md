# Learning Objectives

After studying this topic, you should students able to:

1. Explain the virtualization stack and identify the attack surfaces at each layer.
2. Differentiate between Type 1 and Type 2 hypervisors and their security implications.
3. Describe VM isolation mechanisms (CPU, memory, I/O, network) and explain VM Escape attacks.
4. Apply best practices for VM lifecycle security including image management, sprawl prevention, and secrets handling.
5. Analyze the security risks of live migration and snapshots and describe mitigation strategies.
6. Explain micro-segmentation and east-west monitoring for securing inter-VM communication.
