# Learning Objectives

After studying this topic, you should students able to:

1.  Define cloud data encryption and explain its critical role within the cloud shared responsibility model.
2.  Differentiate between symmetric and asymmetric encryption, including their mechanisms, advantages, disadvantages, and common use cases.
3.  Identify and describe the three states of data (at rest, in transit, in use) and the appropriate encryption strategies for each state.
4.  Explain the importance of cryptographic key management and compare cloud-based key management solutions (KMS, Cloud-HSM, Customer-Managed Keys).
5.  Analyze and recommend encryption features for common storage and compute services on major cloud platforms (AWS, Azure, GCP).
6.  Formulate a set of best practices for implementing a comprehensive data encryption strategy in a cloud environment.
