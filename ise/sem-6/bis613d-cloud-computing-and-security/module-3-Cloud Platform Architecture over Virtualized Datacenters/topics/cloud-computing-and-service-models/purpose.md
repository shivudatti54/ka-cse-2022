# Learning Objectives

After studying this topic, you should students able to:

1. Define cloud computing according to the NIST model and list its five essential characteristics.
2. Differentiate between the three primary cloud service models: IaaS, PaaS, and SaaS, based on the division of management responsibilities.
3. Identify real-world examples of each service model (e.g., AWS EC2 for IaaS, GAE for PaaS, Office 365 for SaaS).
4. Explain the business and technical benefits and trade-offs of each service model.
5. Describe emerging service models like Function-as-a-Service (FaaS) and how they differ from traditional PaaS.
6. Compare and contrast the four cloud deployment models: public, private, hybrid, and community.
