# Learning Objectives
After studying this topic, you should be able to:
1. Define informed search and explain how it differs from uninformed search
2. Explain the concept of heuristic functions and their properties
3. Implement and trace the execution of Greedy Best-First Search algorithm
4. Implement and trace the execution of A* search algorithm
5. Evaluate whether a heuristic function is admissible and/or consistent
6. Compare different informed search algorithms based on completeness, optimality, and efficiency
7. Design appropriate heuristic functions for given problem domains
8. Explain memory-bounded variants of A* and their tradeoffs
