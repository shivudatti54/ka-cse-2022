## **Key Concepts**

- **Uncertainty**: The inability to assign a precise value to a variable
- **Symbolic Reasoning**: The use of symbols and rules to reason about a problem
- **Statistical Reasoning**: The use of data and probability to make predictions or decisions

## **Bayes' Theorem**

- Definition: P(A|B) = P(A and B) / P(B)
- Formula: P(A|B) = P(B|A) \* P(A) / P(B)
- Theorem: Used to update probabilities based on new information

## **Bayes' Theorem Examples**

- **Classical Bayes**: P(A|B) = P(A and B) / P(B)
- **Conditional Bayes**: P(A|B) = P(B|A) \* P(A) / P(B)

## **Probability Theory**

- ** probability**: A numerical value between 0 and 1, representing the likelihood of an event
- **Random Variable**: A variable that can take on different values depending on the outcome of an experiment
- **Expected Value**: E(X) = ∑xP(x)
