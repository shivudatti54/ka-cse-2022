# **Text Book 1: Ch 4 Revision Notes**

## **Key Concepts**

- **Predicate Logic**: A formal language for representing knowledge using logical statements.
- **Knowledge Representation**: The process of representing knowledge in a way that can be used by machines.
- **Predicate Variables**: Symbols that represent objects or concepts in a predicate.
- **Predicate Functions**: Functions that return a value based on the input predicate variables.

## **Important Formulas and Definitions**

- **Definition of a Predicate**: A predicate is a function that assigns a value to a predicate variable.
- **Predicate Logic Formulas**:
  - `P(x)`: Read as "P is true for x".
  - `P(x) ∧ Q(x)`: Read as "P is true for x and Q is true for x".
  - `P(x) ∨ Q(x)`: Read as "P is true for x or Q is true for x".
  - `¬P(x)`: Read as "P is not true for x".
- **Predicate Calculus**:
  - **Universal Quantification**: ∀x (P(x) → Q(x))
  - **Existential Quantification**: ∃x (P(x) ∧ Q(x))

## **Theorems**

- **Law of Excluded Middle**: P(x) ∨ ¬P(x) for all x
- **Law of Non-Contradiction**: P(x) ∧ ¬P(x) for all x (leads to a contradiction)

## **Key Takeaways**

- Understand the basics of predicate logic and knowledge representation.
- Be able to represent knowledge using logical statements.
- Familiarize yourself with predicate variables and functions.
- Review the formulas and definitions of predicate logic.
- Understand the theorems of predicate calculus.
