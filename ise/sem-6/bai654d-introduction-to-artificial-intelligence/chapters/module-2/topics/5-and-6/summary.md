# revision notes for knowledge representation issues using predicate logic

### Topic 5: Propositional Logic and Predicate Logic

- **Propositional Logic:**
  - Definition: A branch of logic that deals with statements that can be either true or false.
  - Propositional variables: P, Q, R, etc.
  - Connectives: ∧ (and), ∨ (or), ¬ (not), → (implies)
  - Tautologies: Statements that are always true, regardless of the values of the variables.
  - Contradictions: Statements that are always false, regardless of the values of the variables.

- **Predicate Logic:**
  - Definition: An extension of propositional logic that deals with statements that contain variables and predicates.
  - Predicates: R(P(Q)), where P is a predicate and Q is a variable.
  - Quantifiers: ∀ (for all), ∃ (there exists)
  - Logical rules: Modus ponens, modus tollens, hypothetical syllogism

- **Formulas:**
  - Atomic formulas: P(Q), R(P)
  - Compound formulas: ¬P, P ∧ Q, P ∨ Q
  - Well-formed formulas (wffs): Formulas that are grammatically correct and follow the rules of predicate logic.

### Topic 6: Knowledge Representation Using Predicate Logic

- **Knowledge Representation:**
  - Definition: The process of representing knowledge in a way that can be understood and used by machines.
  - Types of knowledge representation: Description logics, ontologies, expert systems.
- **Description Logics:**
  - A family of formal languages used for knowledge representation.
  - Include concepts, properties, and relationships.
  - Use to represent knowledge in a formal and structured way.
- **Ontologies:**
  - A formal representation of knowledge that defines a conceptualization of the world.
  - Use to represent knowledge in a structured and organized way.
  - Include classes, properties, and relationships.

### Important Formulas and Theorems

- **Modus ponens:**
  - From P ⊢ Q and P, ∴ Q
- **Modus tollens:**
  - From P ⊢ Q and ¬Q, ∴ ¬P
- **Hypothetical syllogism:**
  - From P ⊢ Q and Q ⊢ R, ∴ P ⊢ R
