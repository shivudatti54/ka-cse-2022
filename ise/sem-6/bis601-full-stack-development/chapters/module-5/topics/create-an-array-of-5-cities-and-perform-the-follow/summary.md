# **Create an Array of Cities and Log Total Number of Cities**

**Summary:**

- Create an array of 5 cities in MongoDB.
- Perform the following operations:
  - Log the total number of cities.

**Revision Notes:**

- **Create an Array of Cities:**
  _ Use the `Array` keyword in MongoDB to create an array of cities.
  _ Example:
  ```javascript
  var cities = ["New York", "Los Angeles", "Chicago", "Houston", "Seattle"];

````
*   **Log the Total Number of Cities:**
    *   Use the `length` property to get the total number of cities in the array.
    *   Example:
        ```javascript
var totalCities = cities.length;
print("Total number of cities:", totalCities);
````

**Important Formulas and Definitions:**

- **Array:**
  - A collection of values of the same data type stored in a single variable.
- **Length Property:**
  - Returns the number of elements in an array.

**Key Points:**

- Use the `Array` keyword to create an array of cities.
- Use the `length` property to get the total number of cities.
- Use the `print` statement to log the total number of cities.

**Theorem/Statement:**

- "An array is a collection of values of the same data type stored in a single variable."
