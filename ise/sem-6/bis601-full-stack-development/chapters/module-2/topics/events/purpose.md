# Learning Objectives

After studying this topic, you should be able to:

1.  Define and apply the happened-before relation (→) to order events in a distributed system execution.
2.  Differentiate between causally related events and concurrent events.
3.  Explain the rules and implement the algorithm for assigning Lamport timestamps to events.
4.  Explain the rules and implement the algorithm for assigning Vector clock timestamps to events.
5.  Compare and contrast Lamport and Vector clocks, highlighting their strengths, weaknesses, and complexity.
6.  Identify consistent and inconsistent global states and explain why inconsistency arises.
