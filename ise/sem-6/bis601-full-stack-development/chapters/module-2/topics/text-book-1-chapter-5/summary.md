# **Text Book 1: Chapter 5 - Document Object Model: DOM Manipulation, Selecting Elements, Working with DOM Nodes, Updating**

## **Key Points**

- **DOM Structure**
  - DOM is a tree-like structure
  - Document Object Model represents the structure of a document
  - Each element in the DOM has a unique ID and properties

- **DOM Nodes**
  - Dom nodes are the building blocks of the DOM
  - Types of DOM nodes: element, attribute, text node, comment, document node
  - DOM nodes can be accessed using the `document` object

- **Selecting Elements**
  - Selecting elements using CSS selectors
  - Types of CSS selectors: tag name, class name, ID selector, attribute selector
  - Using `querySelector` and `querySelectorAll` methods to select elements

- **Updating Elements**
  - Updating element properties using dot notation
  - Updating element text content using the `innerHTML` and `textContent` properties
  - Removing elements using the `remove()` method

- **Formulas and Definitions**
  - No specific formulas, but definitions:
  * **DOM traversal**: traversing the DOM tree to move from one node to another
  * **DOM manipulation**: modifying the structure and content of the DOM

- **Theorems**
  - **Theorem 1:** The DOM is a tree-like structure, where each element has a unique ID and properties.
  - **Theorem 2:** DOM nodes can be accessed using the `document` object, and can be updated using the `innerHTML` and `textContent` properties.

## **Important Terms**

- DOM (Document Object Model)
- DOM nodes
- CSS selectors
- DOM traversal
- DOM manipulation
