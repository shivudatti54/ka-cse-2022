# **Basic JavaScript Instructions, Statements, Comments, Variables, Data Types, Arrays, Strings**

### Instructions and Statements

- **Instructions**: A set of statements that define a sequence of operations.
- **Statements**: A single instruction or a group of instructions that together form an instruction.
- **Empty Statement**: A statement that does nothing (`;`).
- **Label**: A statement that associates a name with the location of a control in a jump statement.

### Comments

- **Single-Line Comment**: Starts with `//`.
- **Multi-Line Comment**: Starts with `/*` and ends with `*/`.

### Variables and Data Types

- **Variables**: Store and manipulate data.
- **Data Types**: Define the type of data stored in a variable.
  - **Number**: A number value (`let num = 10;`)
  - **String**: A sequence of characters (`let name = 'John Doe';`)
  - **Boolean**: A true or false value (`let isAdmin = true;`)
  - **Null**: A special value that represents the absence of any object value (`let age = null;`)
  - **Undefined**: A special value that represents an uninitialized variable (`let age;`)

### Arrays

- **Arrays**: Collections of values of the same data type.
- **Declaring Arrays**: `let arr = [1, 2, 3, 4, 5];`
- **Accessing Array Elements**: `arr[0]` (returns the first element)

### Strings

- **Strings**: Sequences of characters.
- **String Concatenation**: Using the `+` operator to join strings (`let name = 'John'; let age = 30; let greeting = 'Hello, ' + name + ' you are ' + age + ' years old.';`)
- **String Methods**: Various methods that operate on strings (`let str = 'Hello'; str.length; str.toUpperCase();`)
