# Learning Objectives

After studying this topic, you should be able to:

1.  Differentiate between Statement, PreparedStatement, and CallableStatement and choose the appropriate one for a given scenario.
2.  Write code to execute SQL SELECT queries using Statement and PreparedStatement and process the returned ResultSet.
3.  Write code to execute SQL INSERT, UPDATE, and DELETE statements using the appropriate execute method and handle the return value.
4.  Explain the importance of using PreparedStatement for security (preventing SQL injection).
5.  Navigate a ResultSet using methods like next(), previous(), and absolute(), understanding the cursor's initial position.
6.  Correctly retrieve data from a ResultSet using getter methods (getInt, getString, etc.) with both column names and indexes.
