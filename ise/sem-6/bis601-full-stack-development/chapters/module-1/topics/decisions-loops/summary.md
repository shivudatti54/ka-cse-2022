# **Decisions & Loops**

### Overview

This topic covers the fundamental concepts of decision-making and looping in programming, specifically in JavaScript.

### Key Concepts

- **Decision-Making:**
  - Conditional Statements (if/else, switch)
  - Conditional Expressions (ternary operator)
- **Loops:**
  - For Loops
  - While Loops
  - Do-While Loops
  - Looping Control Statements (break, continue, exit)

### Important Formulas and Definitions

- **Conditional Statements:**
  - `if` statement: `if (condition) { code to execute }`
  - `else` statement: `if (condition) { code to execute } else { alternative code }`
  - `switch` statement: `switch (expression) { case value1: code to execute; break; case value2: code to execute; break; }`
- **Loops:**
  - For Loop: `for (initialization; condition; increment) { code to execute }`
  - While Loop: `while (condition) { code to execute }`
  - Do-While Loop: `do { code to execute } while (condition)`
- **Looping Control Statements:**
  - `break`: exits a loop and jumps to the next statement
  - `continue`: skips to the next iteration of the loop
  - `exit`: exits a loop and jumps to the next statement

### Important Theorems and Principles

- **The Loop Invariant Principle:** A loop invariant is a property that remains true before and after each iteration of the loop.
- **The Avoidance Principle:** Minimize the use of loops and conditional statements, and instead use functions to abstract away complex logic.
