## **1.1: Cloud Computing Overview**

### Definition:

Cloud computing is a model for delivering computing services over the internet, where resources such as servers, storage, databases, software, and applications are provided as a service to users on-demand.

### Key Characteristics:

- **On-demand self-service**: Users can provision and de-provision resources as needed.
- **Broad network access**: Resources are accessible over the internet or a private network.
- **Resource pooling**: Resources are pooled together to provide a multi-tenant environment.
- **Rapid elasticity**: Resources can be quickly scaled up or down to match changing demands.
- **Measured service**: Users only pay for the resources they use.

### Example:

A company uses a cloud-based email service like Google Workspace or Microsoft 365 to provide email, calendar, and collaboration tools to its employees. The company does not need to maintain its own email infrastructure or hardware.

## **1.2: Distributed System Models**

### Definition:

A distributed system is a collection of independent computers that work together to provide a shared service or functionality.

### Types of Distributed Systems:

- **Client-server model**: A client requests a service from a server, which provides the requested service.
- **Peer-to-peer model**: Each node in the system can act as both a client and a server.
- **Client-server-pub-sub model**: A client requests a service from a server, which then sends the request to a publisher-subscriber system to retrieve the requested data.

### Example:

A social network like Facebook uses a peer-to-peer model to connect users and facilitate communication. Each user's device acts as both a client and a server, allowing them to connect with other users and share data.

## **1.3: Scalable Computing**

### Definition:

Scalable computing refers to the ability of a system to automatically increase or decrease resources as needed to match changing demands.

### Types of Scalability:

- **Horizontal scalability**: Adding more nodes to the system to increase processing power or storage.
- **Vertical scalability**: Increasing the power of individual nodes to improve performance.

### Example:

A company uses a cloud-based database to store customer data. As the company grows, it can scale up its database by adding more nodes to increase storage capacity and improve performance.

## **1.4: Cloud Computing Service Models**

### Definition:

A cloud computing service model refers to the way in which cloud computing services are delivered to users.

### Types of Cloud Computing Service Models:

- **Infrastructure as a Service (IaaS)**: Provides virtualized computing resources over the internet.
- **Platform as a Service (PaaS)**: Provides a complete development and deployment environment for applications.
- **Software as a Service (SaaS)**: Provides software applications over the internet.

### Example:

A company uses a SaaS-based accounting software to manage its financial operations. The software is provided over the internet and can be accessed by multiple users.

## **1.5: Cloud Computing Deployment Models**

### Definition:

A cloud computing deployment model refers to the way in which cloud computing resources are provisioned and managed.

### Types of Cloud Computing Deployment Models:

- **Public Cloud**: A public cloud is a cloud computing environment that is open to the general public and can be accessed over the internet.
- **Private Cloud**: A private cloud is a cloud computing environment that is provisioned and managed within an organization's premises.
- **Hybrid Cloud**: A hybrid cloud is a cloud computing environment that combines public and private cloud resources.

### Example:

A company uses a public cloud-based email service to provide email to its employees. The company does not need to provision or manage its own email infrastructure.

## Key Concepts:

- Cloud computing is a model for delivering computing services over the internet.
- Distributed systems are collections of independent computers that work together to provide a shared service or functionality.
- Scalable computing refers to the ability of a system to automatically increase or decrease resources as needed.
- Cloud computing service models include IaaS, PaaS, and SaaS.
- Cloud computing deployment models include public, private, and hybrid clouds.
