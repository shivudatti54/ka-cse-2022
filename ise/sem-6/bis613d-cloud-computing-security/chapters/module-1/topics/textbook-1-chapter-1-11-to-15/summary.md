# Textbook 1: Chapter 1: 1.1 to 1.5

### Overview

- Distributed computing models:
  - Client-server model
  - Peer-to-peer (P2P) model
  - Cloud computing model
- Scalable computing technologies:
  - Grid computing
  - Utility computing
  - Cloud computing

### Definitions

- **Cloud computing**: A model for delivering computing services over the internet.
- **Grid computing**: A system that enables organizations to share resources and services over a network.
- **Utility computing**: A model for providing computing resources on-demand, like public utilities.
- **Scalable computing**: The ability of a system to scale up or down to meet changing demands.

### Theorems and Formulas

- **Scalability law**: The relationship between the number of clients and the resources required to support them.
- **Cloud computing formula**: C = I x P x T, where:
  - C = Total cost
  - I = Initial investment
  - P = Pay-as-you-go pricing model
  - T = Time
- **Grid computing formula**: G = A x B, where:
  - G = Grid size
  - A = Available resources
  - B = Required resources

### Key Concepts

- **Cloud security**: The protection of cloud computing resources and data from unauthorized access, use, disclosure, modification, or destruction.
- **Distributed system security**: The security of a distributed system, which is a system consisting of multiple computers that work together to achieve a common goal.

### Important Terms

- **Cloud provider**: A company that provides cloud computing services.
- **Cloud consumer**: A user who consumes cloud computing services.
- **Infrastructure as a Service (IaaS)**: A cloud computing model that provides virtualized computing resources.
- **Platform as a Service (PaaS)**: A cloud computing model that provides a platform for developing and deploying applications.
