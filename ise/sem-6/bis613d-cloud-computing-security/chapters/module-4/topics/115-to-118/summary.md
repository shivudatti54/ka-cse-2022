# **Cloud Computing & Security Revision Notes: 11.5 to 11.8**

### Risk Management in Cloud Computing

- **Cloud Risk Management Framework**: A structured approach to identify, assess, and mitigate cloud risks.
- **Risk Assessment Matrix**: A tool used to categorize risks based on their likelihood and impact.
- **Cloud Risk Taxonomy**: A classification of risks into various categories, such as data loss, unauthorized access, and compliance risks.

### Data Security in the Cloud

- **Data Encryption**: The process of converting plaintext data into unreadable ciphertext to protect it from unauthorized access.
- **Data Masking**: A technique used to hide sensitive data, such as credit card numbers, to prevent unauthorized access.
- **Data Loss Prevention (DLP)**: A strategy used to protect sensitive data from unauthorized access, disclosure, or loss.

### Cloud Data Privacy

- **Cloud Data Privacy Act**: A legislative framework that regulates the handling of personal data in the cloud.
- **GDPR**: The General Data Protection Regulation, a European Union regulation that sets standards for data protection.
- **Cloud Data Protection Framework**: A set of standards and guidelines for protecting cloud data, including data encryption, access control, and auditing.

### Cloud Security Theorems and Formulas

- **Bake Sale Model**: A framework used to evaluate the trade-off between the cost of security measures and the cost of potential security breaches.
- **Cloud Security Architecture Diagram (CSAD)**: A visual representation of a cloud security architecture, including security controls and protocols.
- **Cloud Security Risks and Threats**: A list of potential security risks and threats in the cloud, including data breaches, unauthorized access, and compliance risks.

### Important Definitions

- **Cloud Security**: The practice of protecting cloud computing resources, data, and applications from unauthorized access, use, disclosure, disruption, modification, or destruction.
- **Cloud Risk**: A potential threat or vulnerability in the cloud that could compromise the security, availability, or confidentiality of cloud resources and data.
- **Cloud Security Architecture**: A design and organizational structure that enables the secure operation of cloud computing resources and applications.
