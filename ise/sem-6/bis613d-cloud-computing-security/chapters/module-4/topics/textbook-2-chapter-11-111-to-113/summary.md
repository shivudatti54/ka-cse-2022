# Textbook 2: Chapter 11: 11.1 to 11.3

## Cloud Security: Top Concerns and Risks

- **Cloud Security Risks:**
  - Unauthorized access to cloud resources
  - Data breaches and eavesdropping
  - Malware and ransomware attacks
  - Denial of Service (DoS) and Distributed Denial of Service (DDoS) attacks
  - Lack of control over data and applications

## Privacy Impact Assessment

- **Privacy Impact Assessment (PIA) Process:**
  - Identify stakeholders and their interests
  - Assess the risks and impact of cloud services on stakeholders
  - Evaluate the benefits and risks of cloud services
- **PIA Framework:**
  - Federal Information Security Management Act (FISMA)
  - National Institute of Standards and Technology (NIST) Framework

## Cloud Data

- **Cloud Data Storage:**
  - Types: structured, semi-structured, unstructured
  - Characteristics: scalability, flexibility, and reliability
- **Cloud Data Security:**
  - Data encryption and access controls
  - Data backup and recovery
  - Data loss prevention and incident response

## Important Definitions and Formulas

- **Definition:** Cloud Security
  - The practice of protecting cloud computing resources and data from unauthorized access, use, disclosure, disruption, modification, or destruction.
- **Theorem:** Cloud Security Theory
  - A theoretical framework for evaluating the security of cloud computing resources and data.
- **Formula:** Cloud Security Risk Assessment (CSRA)
  - CSRA = (Risk X Vulnerability) / (Asset Value)
