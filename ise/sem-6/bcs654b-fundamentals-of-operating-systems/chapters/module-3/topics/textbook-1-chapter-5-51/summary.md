# Revision Notes: Fundamentals of Operating Systems (Chapter 5.1)

## Key Concepts

- **Definition:** An operating system (OS) is a software that manages computer hardware resources and provides common services to computer programs.
- **Components:**
  - System software
  - Application software
  - Input/Output devices
  - Hardware

## System Software

- **Definition:** System software is a type of software that manages computer hardware resources and provides common services to computer programs.
- **Types:**
  - Operating system (OS)
  - Utilities
  - Drivers
- **Functions:**
  - Process management
  - Memory management
  - File management
  - Input/Output management

## Process Management

- **Definition:** Process management refers to the ability of an operating system to manage processes (programs) running on the computer.
- **Process States:**
  - Running
  - Waiting
  - Ready
  - Zombie
  - Dead
- **Process Scheduling:**
  - First-Come-First-Served (FCFS)
  - Shortest Job First (SJF)
  - Priority Scheduling

## Memory Management

- **Definition:** Memory management refers to the allocation and deallocation of memory for processes.
- **Types:**
  - Virtual Memory
  - Real Memory
- **Memory Allocation:**
  - Paged Memory
  - Segmented Memory
- **Memory Protection:**
  - Page Protection
  - Segment Protection

## Formulae

- **Page Fault Formula:** P = (P + Q) / (P + Q + R)
- **Page Replacement Algorithm:** Ψ = LRU / LFU

## Important Definitions

- **CPU:** Central Processing Unit
- **I/O:** Input/Output
- **PS:** Process Status
- **PDS:** Process Descriptor Structure
