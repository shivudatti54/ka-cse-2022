# **Revision Notes: Fundamentals of Operating Systems**

## **Topic: 5.4 Chapter 6: 6.1**

### Overview

- Operating Systems (OS) manage computer hardware resources
- OS provides a platform for running applications

### Key Concepts

- **Process**: Independent execution of a program
- **Thread**: Lightweight process within a process
- **Memory Management**: Allocating and deallocating memory for processes
- **Resource Allocation**: Allocating hardware resources (CPU, I/O devices)

### Important Formulas and Definitions

- **CPU Time**: Time spent by CPU executing a process
- **Turnaround Time**: Time spent by CPU executing a process from arrival to completion
- **Waiting Time**: Time spent by a process waiting for CPU access
- **Context Switching**: Swapping execution from one process to another
- **Process Scheduling Algorithm**: Determining the order of process execution

### Theoretical Background

- **First-Come-First-Served (FCFS)**: Scheduling algorithm where the first process in the ready queue is executed first
- **Shortest Job First (SJF)**: Scheduling algorithm where the shortest process in the ready queue is executed first
- **Priority Scheduling**: Scheduling algorithm where processes are assigned priority based on their characteristics

### Important Theorems

- **Theorem of Invariance of the CPU Time**: CPU time is a deterministic function of input and output
- **Theorem of the Distribution of Waiting Time**: Waiting time is a function of input and output

### Key Terms

- **Resource Contention**: A situation where two or more processes contend for the same resource
- **Resource Multiplexing**: The ability of one process to share one resource with other processes

### Revision Tips

- Review the definitions and formulas of key concepts
- Practice solving problems related to process scheduling algorithms
- Focus on understanding the theoretical background and theorems related to CPU time and waiting time
