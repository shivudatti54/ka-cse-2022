# Chapter 4: Process Management

### 4.1 Process Concept

- A process is a program in execution, containing a sequence of instructions to be executed by the CPU.
- Processes are the basic execution units of a computer.
- Each process has its own program counter, stack, and memory space.

### Key Concepts

- **Process Scheduling**: The selection of a process to be executed next by the operating system.
- **Process Creation**: The process of creating a new process in the operating system.
- **Process Termination**: The process of terminating a process in the operating system.

### Important Formulas and Definitions

- **CPU Time**: The amount of time a process spends executing on the CPU.
- **Context Switching**: The process of switching from one process to another.
- **Process ID (PID)**: A unique identifier for a process.
- **Process Virtual Memory (PVM)**: A virtual memory space shared by multiple processes.

### Important Theorems

- **Theorem: Process Scheduling Theorem**: The operating system must schedule processes based on their priority and availability.
- **Theorem: Process Termination Theorem**: The operating system must terminate a process when it is finished executing.

### Quick Revision Key Points

- A process is a program in execution with its own program counter, stack, and memory space.
- Process scheduling is the selection of a process to be executed next by the operating system.
- Process creation and termination are essential operating system functions.
- CPU time, context switching, PID, and PVM are important concepts in process management.
