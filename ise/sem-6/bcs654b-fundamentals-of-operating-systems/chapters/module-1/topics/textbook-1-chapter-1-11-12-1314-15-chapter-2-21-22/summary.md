# **Fundamentals of Operating Systems Revision Notes**

## **Chapter 1: Introduction to Operating Systems**

- **Definition**: An operating system (OS) is a software that manages computer hardware resources and provides common services to computer programs.
- **Functions of an Operating System**:
  - Process Management
  - Memory Management
  - File Management
  - Input/Output Management
  - Security
  - Networking
- **Computer System Organization**:
  - Central Processing Unit (CPU)
  - Memory (RAM)
  - Input/Output Devices
  - Storage Devices
  - Power Supply
- **Types of Operating Systems**:
  - Single-user, single-tasking
  - Single-user, multi-tasking
  - Multi-user, single-tasking
  - Multi-user, multi-tasking

## **Chapter 2: Characteristics of Operating Systems**

- **Definition**: An operating system (OS) is a software that manages computer hardware resources and provides common services to computer programs.
- **Types of Operating Systems**:
  - **Batch Operating Systems**: Non-interactive, process scheduling
  - **Time-Sharing Operating Systems**: Interactive, process scheduling
- **Process Scheduling**:
  - **FCFS (First-Come-First-Served)**: First process to arrive gets the CPU
  - **SJF (Shortest Job First)**: Shortest job gets the CPU
  - **Priority Scheduling**: Processes are assigned a priority
- **Memory Management**:
  - ** paging**: Divide memory into fixed-size blocks (pages)
  - **Segmentation**: Divide memory into variable-size blocks (segments)
