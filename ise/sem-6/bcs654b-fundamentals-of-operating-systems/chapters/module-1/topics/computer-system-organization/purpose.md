# Learning Objectives
After studying this topic, you should be able to:
1. Identify and describe the four main structural components of a computer system
2. Explain the fetch-decode-execute cycle and the role of CPU registers
3. Differentiate between types of memory in the memory hierarchy and their characteristics
4. Describe how interrupts work and the interrupt handling process
5. Compare and contrast different I/O techniques including programmed I/O, interrupt-driven I/O, and DMA
6. Explain the concepts of multiprocessing and clustered systems
