# **Revision Notes: Android Description, Open Handset Alliance, and Android (Chapters 1 & 2)**

### Overview

- The Open Handset Alliance (OHA) is a consortium of companies that includes Google, HTC, Motorola, Samsung, and others.
- Android is an open-source operating system developed by the OHA.

### Key Points

#### Android Description

- Android is a mobile operating system (OS) designed for touchscreen devices.
- It uses a Linux kernel and is based on the Linux operating system.
- Android has a modular architecture, allowing users to customize and extend the OS.

#### Open Handset Alliance (OHA)

- The OHA was established in 2003 to develop and promote the Android OS.
- Members of the OHA agree to implement the Android OS on their devices.
- The OHA is responsible for setting standards for Android devices.

#### Android Architecture

- Android is a client-server architecture.
- The Android OS consists of:
  - Android Runtime (ART)
  - Dalvik Virtual Machine (DVM)
  - Linux kernel
  - Android Framework
  - Android Services

### Definitions

- **Open-source**: Software that is freely available for anyone to use, modify, and distribute.
- **Operating System (OS)**: Software that manages and controls computer hardware and software resources.
- **Linux kernel**: The core of the Linux operating system, responsible for managing computer hardware resources.

### Theorems and Formulas

- None specific to this topic, but relevant to Android development:
  - The Linux kernel uses the **Monolithic** architecture, where the kernel and user-space applications share the same memory space.
  - Android uses the **Hybrid** architecture, which combines elements of both monolithic and microkernel architectures.

### Important Concepts

- **Android App**: A software application that runs on the Android OS.
- **Activity**: A fundamental component of the Android OS, responsible for managing user interactions.
- **Intent**: A message sent from one application to another, used for communication between applications.
