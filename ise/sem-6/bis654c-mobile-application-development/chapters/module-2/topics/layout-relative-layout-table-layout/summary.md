# **Layout Relative Layout – Table Layout**

## **Key Points**

### Definitions

- **Layout Relative Layout**: A layout that arranges its child views relative to each other.
- **Table Layout**: A layout that arranges its child views in a table structure.

### Key Concepts

- **Root View**: The top-level view in the layout hierarchy.
- **Child Views**: Views that are nested inside the root view.
- **Parent View**: A view that contains child views.
- **Layout Parameters**: Attributes that define the size and position of a view.

### Properties and Methods

- **android:layout_width** and **android:layout_height**: Define the width and height of a view.
- **android:layout_weight**: Defines the relative weight of a view in a table layout.
- **android:layout_gravity**: Defines the gravity of a view within its parent.
- **setPadding** and **setMargins**: Set padding and margins for a view.

### Table Layout

- **TableLayout**: A layout that arranges its child views in a table structure.
- **TableRow**: A view that contains one or more child views.
- **TableCellView**: A view that represents a cell in a table.
