# Learning Objectives

After studying this topic, you should be able to:

1.  Explain the core components of a decision tree (root, internal, and leaf nodes) and interpret a simple tree diagram.
2.  Calculate key metrics used for splitting, including Entropy, Gini Impurity, Information Gain, and Gain Ratio, for a given dataset.
3.  Compare and contrast the ID3, C4.5, and CART algorithms, highlighting their differences in splitting criteria, handling of data types, and pruning methods.
4.  Describe the problem of overfitting in the context of decision trees and explain how pruning and other stopping criteria help mitigate it.
5.  List the advantages and disadvantages of using decision tree models.
6.  Outline the step-by-step process a tree induction algorithm follows to build a model from data.
