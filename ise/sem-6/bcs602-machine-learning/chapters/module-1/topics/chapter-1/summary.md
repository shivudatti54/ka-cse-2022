# Machine Learning

### Chapter-1: Introduction

#### Overview

- Machine learning is a subset of artificial intelligence that enables systems to learn from data.
- The ultimate goal of machine learning is to enable computers to learn from data and make decisions or predictions without being explicitly programmed.

#### Key Concepts

- **Data**: Input provided to a machine learning algorithm, such as images, text, or sensor readings.
- **Model**: A mathematical representation of a system or process that an algorithm can learn from data.
- **Algorithms**: A set of instructions that an algorithm follows to learn from data and make predictions or decisions.

#### Important Formulas and Definitions

- **Probability**: A measure of the likelihood of an event occurring.
  - **Bayes' Theorem**: P(A|B) = P(A and B) / P(B)
- **Artificial Neural Network (ANN)**: A model inspired by the structure and function of the human brain.
- **Supervised Learning**: A type of machine learning where the algorithm is trained on labeled data to learn a mapping between inputs and outputs.
- **Unsupervised Learning**: A type of machine learning where the algorithm is trained on unlabeled data to discover patterns or relationships.

#### Important Theorems

- **No Free Lunch Theorem**: There is no algorithm that can perform any task better than random guessing on all tasks.
- **Vapnik-Chervonenkis (VC) Dimension**: A measure of the capacity of a classifier to distinguish between two classes.

#### Revision Tips

- Understand the difference between supervised and unsupervised learning.
- Familiarize yourself with artificial neural networks and their applications.
- Practice applying Bayes' Theorem in different scenarios.
