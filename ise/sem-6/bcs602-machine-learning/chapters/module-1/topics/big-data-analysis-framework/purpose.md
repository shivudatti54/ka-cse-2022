# Learning Objectives
After studying this topic, you should be able to:
1.  Define Big Data by explaining each of the 5 V's (Volume, Velocity, Variety, Veracity, Value).
2.  Describe the core components of a Big Data framework: Data Ingestion, Storage, and Processing.
3.  Differentiate between batch processing and stream processing, including their use cases.
4.  Explain the basic flow of the MapReduce programming model (Map -> Shuffle -> Reduce).
5.  Identify the role of key components in the Hadoop ecosystem (HDFS, NameNode, DataNode, YARN).
6.  Articulate the critical relationship between Big Data infrastructure and the Machine Learning process, particularly in data preparation and model training.
