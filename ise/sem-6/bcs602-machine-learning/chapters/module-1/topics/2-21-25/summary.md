# **Revision Notes: 2 (2.1-2.5) - Machine Learning**

### Overview

- Machine learning is a subset of artificial intelligence that enables systems to learn from data without being explicitly programmed.

### Definitions

- **Machine Learning (ML)**: The study of developing algorithms and statistical models that enable machines to learn from data, make decisions, and improve their performance over time.
- **Supervised Learning**: A type of ML where the algorithm learns from labeled data to make predictions.
- **Unsupervised Learning**: A type of ML where the algorithm learns from unlabeled data to identify patterns and structure.

### Key Concepts

- **Regression**: Predicting a continuous output variable.
  - Formula: y = f(x, β)
- **Classification**: Predicting a categorical output variable.
  - Formula: y = f(x, β)
- **Logistic Regression**: A type of regression for classification problems.
- **Decision Trees**: A simple, interpretable model for classification and regression.
- **Overfitting**: When a model is too complex and performs well on the training data but poorly on new data.
- **Cross-Validation**: A technique to evaluate model performance using multiple subsets of data.

### Important Theorems

- **Bayes' Theorem**: P(A|B) = P(A and B) / P(B)
- **Bayes' Rule**: P(A) = P(A|B) \* P(B) / P(B)

### Key Formulas

- **Cost Function**: J(θ) = (1/2) \* ∑(y_i - f(x_i, θ))^2
- **Gradient Descent**: θ = θ - α \* ∇J(θ)

### Quick Revision Tips

- Understand the difference between supervised and unsupervised learning.
- Know the formulas for regression and classification.
- Be aware of the concept of overfitting and how to use cross-validation to evaluate model performance.
