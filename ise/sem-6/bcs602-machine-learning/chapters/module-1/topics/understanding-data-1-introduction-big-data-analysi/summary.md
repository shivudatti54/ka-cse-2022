# **Understanding Data – 1: Revision Notes**

### Introduction

---

- Big Data refers to the vast amount of data being generated at an unprecedented rate.
- Machine learning is a subset of artificial intelligence that involves training algorithms on data to make predictions or decisions.
- Data quality, accuracy, and relevance are crucial for successful machine learning applications.

### Big Data Analysis Framework

---

- 5 V's of Big Data:
  - Volume: Large amount of data
  - Velocity: High speed of data generation
  - Variety: Different types of data
  - Veracity: Data accuracy and reliability
  - Value: Extracting insights and business value
- Data processing frameworks: Hadoop, Spark, NoSQL databases

### Descriptive Statistics

---

- Definition: Descriptive statistics provide a summary of the basic features of a dataset.
- Key concepts:
  - Measures of central tendency: Mean, Median, Mode
  - Measures of variability: Range, Variance, Standard Deviation
  - Data visualization: Histograms, Box plots, Scatter plots

### Univariate Data Analysis and Visualization

---

- Definition: Univariate analysis involves analyzing individual variables or features of a dataset.
- Key concepts:
  - Correlation analysis: Pearson correlation coefficient, Spearman rank correlation coefficient
  - Probability distributions: Normal distribution, Poisson distribution, Binomial distribution
  - Data visualization: Scatter plots, Bar charts, Histograms

### Important Formulas and Definitions

---

- **Mean**: μ = (Σx) / n
- **Median**: m = (x1 + x2 + ... + xn) / 2
- **Mode**: x̄ = most frequent value
- **Variance**: σ^2 = Σ(x - μ)^2 / (n - 1)
- **Standard Deviation**: σ = √(σ^2)
- **Correlation coefficient**: r = Σ[(xi - μ) * (yi - μ)] / (√(Σ(xi - μ)^2) \* √(Σ(yi - μ)^2))
