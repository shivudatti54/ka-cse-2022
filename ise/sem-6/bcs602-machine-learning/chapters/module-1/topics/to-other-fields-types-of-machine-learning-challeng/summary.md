### Types of Machine Learning

- **Supervised Learning**:
  - Definition: Training algorithm learns from labeled data to make predictions on new data
  - Algorithms: Linear Regression, Logistic Regression, Decision Trees, Random Forests
  - Formula: $y = f(x) + \epsilon$
- **Unsupervised Learning**:
  - Definition: Training algorithm learns from unlabeled data to identify patterns and structure
  - Algorithms: K-Means Clustering, Hierarchical Clustering, Principal Component Analysis (PCA)
  - Formula: $\min_{\mathbf{\theta}} D(\mathbf{X}, \mathbf{y})$
- **Reinforcement Learning**:
  - Definition: Training algorithm learns through trial and error to take actions in an environment
  - Algorithms: Q-Learning, SARSA, Deep Q-Networks (DQN)
  - Formula: $\max_{\pi} V_{\pi}(\mathbf{s})$

### Challenges of Machine Learning

- **Overfitting**: Model becomes too complex and performs well on training data but poorly on new data
- **Underfitting**: Model is too simple and fails to capture underlying patterns in data
- **Data Quality**: Noisy, missing, or biased data can lead to poor model performance
- **Explainability**: Difficulty in interpreting and understanding model decisions

### Machine Learning Process

1.  **Problem Formulation**: Define problem and identify relevant data
2.  **Data Collection**: Gather and preprocess data
3.  **Data Preprocessing**: Clean, transform, and normalize data
4.  **Model Selection**: Choose suitable machine learning algorithm
5.  **Model Training**: Train model on labeled data
6.  **Model Evaluation**: Evaluate model performance on unseen data
7.  **Model Deployment**: Deploy model in production environment
