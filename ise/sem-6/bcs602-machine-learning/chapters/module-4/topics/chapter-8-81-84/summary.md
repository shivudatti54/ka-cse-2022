### Chapter-8 Revision Notes (8.1-8.4)

#### 8.1 Introduction to Probability-based Learning

- **Probability Theory**: Branch of mathematics dealing with chance events and their likelihood of occurrence.
- **Random Variables**: Variables that can take on different values based on chance events.
- **Expected Value (E(X))**: Average value of a random variable, calculated as the sum of each possible value multiplied by its probability.

#### 8.2 Fundamentals of Bayes Theorem

- **Bayes' Theorem**: Mathematical formula for updating the probability of a hypothesis based on new evidence.
- **Bayes' Theorem Formula**: P(H|E) = P(E|H) \* P(H) / P(E)
- **Conditional Probability**: Probability of an event occurring given that another event has occurred.

#### 8.3 Bayesian Learning

- **Bayesian Learning**: Method of learning that updates probabilities based on new data using Bayes' Theorem.
- **Bayesian Networks**: Directed acyclic graphs representing relationships between random variables.
- **Conditional Independence**: Relationship between two variables given a third variable.

#### 8.4 Applications of Bayesian Learning

- **Naive Bayes**: Simplified version of Bayesian learning that assumes conditional independence between features.
- **Kalman Filter**: Recursive algorithm for estimating the state of a system from noisy measurements.
- **Gaussian Mixture Models**: Statistical model for clustering data into different classes.

#### Important Formulas and Definitions

- **Probability Density Function (PDF)**: Function describing the probability distribution of a random variable.
- **Cumulative Distribution Function (CDF)**: Function describing the cumulative probability of a random variable.
- **Entropy**: Measure of uncertainty in a probability distribution.
