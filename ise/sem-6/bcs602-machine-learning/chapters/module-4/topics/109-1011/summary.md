# **Bayesian Learning Revision Notes: 10.9-10.11**

### Key Concepts

- **Bayes' Theorem**: updates the prior probability distribution based on new data to obtain the posterior distribution
- **Conditional Probability**: the probability of an event occurring given that another event has occurred
- **Joint Probability**: the probability of two or more events occurring together
- **Independence**: the events are not affected by each other

### Important Formulas

- **Bayes' Theorem**:
  \[ P(A|B) = \frac{P(B|A) \cdot P(A)}{P(B)} \]
- **Conditional Probability**:
  \[ P(A|B) = \frac{P(A \cap B)}{P(B)} \]
- **Law of Total Probability**:
  \[ P(B) = \sum\_{i=1}^{n} P(B|A_i) \cdot P(A_i) \]

### Definitions

- **Prior Distribution**: the probability distribution of a variable before observing new data
- **Likelihood**: the probability of observing new data given a set of parameters
- **Posterior Distribution**: the probability distribution of a variable after observing new data

### Theorems

- **Bayes' Theorem**: a fundamental theorem in probability theory that describes how to update probabilities based on new data
- **Bayesian Risk**: the expected loss when using a Bayesian model, which is a key concept in Bayesian learning
