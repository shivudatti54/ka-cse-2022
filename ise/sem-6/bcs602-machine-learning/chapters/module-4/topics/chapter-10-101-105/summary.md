# Chapter-10 (10.1-10.5) Bayesian Learning

### Introduction

- Bayesian learning is a probability-based approach to machine learning.
- It uses Bayes' theorem to update probabilities based on new data.

### Key Concepts

- **Bayes' Theorem**:
  - P(H|D) = P(D|H) \* P(H) / P(D)
  - P(H) = Prior probability of the hypothesis
  - P(D|H) = Likelihood of the data given the hypothesis
  - P(D) = Marginal probability of the data
- **Conditional Probability**: P(A|B) = P(A and B) / P(B)
- **Decision Theory**: Map posterior probabilities to decisions.

### Important Formulas

- **Bayes' Theorem**:
  - P(H|D) = P(D|H) \* P(H) / P(D)
- **Decision Rule**:
  - Choose class with highest posterior probability

### Key Definitions

- **Prior Probability**: Probability of an event before observing data
- **Likelihood**: Probability of data given an event
- **Posterior Probability**: Probability of an event after observing data

### Important Theorems

- **Bayes' Theorem** is a fundamental tool for updating probabilities
- **Decision Theory** provides a framework for making decisions based on posterior probabilities

- Understand Bayes' theorem and its application
- Familiarize yourself with conditional probability and decision theory
- Review prior probabilities, likelihoods, and posterior probabilities
