# **Chapter-3 (3.3): Bivariate and Multivariate Data**

## **Introduction**

In machine learning, data is a fundamental concept that enables the development of predictive models. Data can be broadly classified into two categories: bivariate data and multivariate data. Understanding these concepts is essential for data analysis, modeling, and visualization.

## **Bivariate Data**

Bivariate data refers to a dataset that contains two variables. These variables are related to each other, and the relationship between them can be explored using statistical methods. Bivariate data can be visualized using scatter plots, which are useful for identifying patterns, trends, and correlations between the variables.

## **Types of Bivariate Data**

- **Linear Relationship**: A linear relationship between two variables indicates a direct proportional relationship. For example, the relationship between the number of hours studied and the score obtained on an exam.
- **Non-Linear Relationship**: A non-linear relationship between two variables indicates an inverse or curvilinear relationship. For example, the relationship between the amount of sugar consumed and the risk of diabetes.

## **Multivariate Data**

Multivariate data, on the other hand, refers to a dataset that contains multiple variables. These variables are related to each other, and the relationship between them can be explored using statistical methods. Multivariate data can be visualized using dimensionality reduction techniques, such as principal component analysis (PCA) and t-SNE (t-distributed Stochastic Neighbor Embedding).

## **Types of Multivariate Data**

- **Categorical Variables**: Variables that take on categorical values, such as labels or categories. For example, a variable that represents the color of a car (red, blue, green).
- **Numerical Variables**: Variables that take on numerical values, such as scores or measurements. For example, a variable that represents the height of a person.

## **Key Concepts**

- **Correlation Coefficient**: A statistical measure that calculates the strength and direction of the relationship between two variables.
- **Regression Analysis**: A statistical method that models the relationship between a dependent variable and one or more independent variables.
- **Dimensionality Reduction**: A technique that reduces the number of variables in a dataset while preserving the most important information.

## **Example**

Suppose we have a dataset that contains information about the age and income of a group of people. We can use bivariate data to analyze the relationship between these two variables.

| Age | Income |
| --- | ------ |
| 25  | 50000  |
| 30  | 60000  |
| 35  | 70000  |
| 40  | 80000  |

Using a scatter plot, we can visualize the relationship between age and income. We can see that as age increases, income also increases. This indicates a positive correlation between the two variables.
