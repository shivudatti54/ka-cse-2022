### 2.6: Bivariate Data and Descriptive Statistics

- **Bivariate Data**: Data that describes two variables
- **Correlation Coefficient (r)**:
  - Range: -1 to 1
  - Interpretation:
    - r = 1: Perfect positive correlation
    - r = 0: No correlation
    - r = -1: Perfect negative correlation
- **Coefficient of Determination (R-squared)**:
  - Measures proportion of variance explained by one variable
  - Formula: R-squared = r^2
- **Regression Analysis**:
  - Used to model relationship between two variables
  - Goal: Predict value of one variable based on value of other variable

### 2.7: Multivariate Data and Descriptive Statistics

- **Multivariate Data**: Data that describes multiple variables
- **Multivariate Statistics**:
  - Includes correlation analysis, regression analysis, and principal component analysis (PCA)
- **Variance-Inflation Factor (VIF)**:
  - Measures effect of multicollinearity on regression coefficients
  - Formula: VIF = (1 / (1 - R^2))

### 2.8: Introduction to Multivariate Statistics

- **Multivariate Regression**:
  - Generalizes linear regression to multiple variables
  - Formula: y = β0 + β1x1 + β2x2 + … + βnxn + ε
- **Principal Component Analysis (PCA)**:
  - Reduces dimensionality of multivariate data
  - Goal: Identify underlying patterns and relationships in data
