# **Pointers: Introduction, Pointer Concepts, Accessing Variables through Pointers, Pointer Applications, Dynamic Memory Allocation Functions**

### Introduction to Pointers

- **Definition:** A pointer is a variable that stores the memory address of another variable.
- **Purpose:** To indirectly access variables and perform operations on them.

### Pointer Concepts

- **Pointer Variables:** Variables that store memory addresses.
- **Dereferencing:** Accessing the value stored at the memory address held by a pointer.
- **Pointer Arithmetic:** Incrementing or decrementing a pointer to move to the next or previous memory location.

### Accessing Variables through Pointers

- **Direct Addressing:** Accessing a variable using its memory address.
- **Indirect Addressing:** Accessing a variable using a pointer.

### Pointer Applications

- **Arrays:** Pointers can be used to implement arrays.
- **Dynamic Memory Allocation:** Pointers are used to allocate and deallocate memory dynamically.
- **String Handling:** Pointers are used to manipulate strings.

### Dynamic Memory Allocation Functions

- **`malloc()`:** Allocates a block of memory and returns its address.
- **`calloc()`:** Allocates a block of memory and initializes it to zero.
- **`realloc()`:** Reallocates a block of memory.
- **`free()`:** Deallocates a block of memory.

Important Formulas and Theorems:

- **Pointer Arithmetic Formula:** `ptr += 1` increments the pointer by 1, `ptr -= 1` decrements the pointer by 1.
- **Memory Address Formula:** `&var` returns the memory address of a variable `var`.
