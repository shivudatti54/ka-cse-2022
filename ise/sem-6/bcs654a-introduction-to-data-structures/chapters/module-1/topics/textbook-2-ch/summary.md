# Textbook 2: Ch - Arrays: Introduction, One-Dimensional Arrays, Two-Dimensional Arrays, Initializing

## Introduction

- An array is a collection of elements of the same data type stored in contiguous memory locations.
- Arrays are used to store and manipulate large amounts of data efficiently.

## One-Dimensional Arrays

- A one-dimensional array is an array with elements of the same data type stored in a single row or column.
- Notation: `arr[0] = element at index 0`, `arr[n] = element at index n`
- Formula: `size of array = number of elements`
- Definition: An array with a fixed number of elements of the same data type.

## Two-Dimensional Arrays

- A two-dimensional array is an array with elements of the same data type stored in multiple rows and columns.
- Notation: `arr[i][j] = element at row i, column j`
- Formula: `size of array = number of rows * number of columns`
- Definition: An array with a fixed number of rows and columns, each containing elements of the same data type.

## Initializing Two-Dimensional Arrays

- Initialization can be done using the following methods:
  - Direct initialization: `arr[i][j] = value`
  - Previous element initialization: `arr[i][j] = arr[i-1][j]`
  - Loop initialization: `for (i = 0; i < rows; i++) { for (j = 0; j < columns; j++) { arr[i][j] = value; }}`

## Important Formulas and Definitions

- **Array Index**: The index of an element in an array is its position in the sequence of elements, starting from 0.
- **Array Size**: The number of elements in an array.
- **Array Dimension**: The number of rows and columns in a two-dimensional array.

## Important Theorems

- **Array Initialization Theorem**: An array can be initialized using one of the methods mentioned above.
- **Array Access Theorem**: An element in an array can be accessed using its index.
