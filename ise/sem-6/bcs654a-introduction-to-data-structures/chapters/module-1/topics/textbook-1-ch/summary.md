# Textbook 1: Ch

## Introduction to Data Structures: Arrays

### Module: Arrays - Introduction, One-Dimensional Arrays, Two-Dimensional Arrays

#### Key Concepts

- **Definition of an Array**: A collection of elements of the same data type stored in contiguous memory locations.
- **Types of Arrays**:
  - **One-Dimensional (1D) Arrays**: Elements are stored in a single row or column.
  - **Two-Dimensional (2D) Arrays**: Elements are stored in rows and columns.
- **Array Initialization**:
  - **Static Initialization**: Elements are initialized during array declaration.
  - **Dynamic Initialization**: Elements are initialized before use.

#### Important Formulas and Definitions

- **Array Indexing**:
  - `arr[i]`: Accessing the element at index `i`.
  - `arr.length`: Getting the number of elements in the array.
- **Array Slicing**:
  - `arr[start:stop]`: Extracting a subset of elements from the array.
- **Array Operations**:
  - `arr.push(element)`: Adding an element to the end of the array.
  - `arr.pop()`: Removing the last element from the array.

#### Theorems

- **Array Inequality Theorem**: If `arr1` and `arr2` are two arrays of the same size, and `arr1[i] < arr2[i] for all `i`, then `arr1 <= arr2`.
- **Array Search Algorithm**: If `arr` is a sorted array and `x` is an element of the array, then there exists a unique `i` such that `arr[i] == x`.

#### Key Formulas

- **Array Length Formula**: `length = n`, where `n` is the number of elements in the array.
- **Array Index Formula**: `index = i`, where `i` is the index of the element.

#### Quick Revision Tips

- Familiarize yourself with array indexing and slicing.
- Understand the difference between static and dynamic initialization.
- Practice solving array-related problems to improve your coding skills.
