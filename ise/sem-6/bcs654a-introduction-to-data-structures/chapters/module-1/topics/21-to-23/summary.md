**Revision Notes: Arrays (2.1 to 2.3)**

### Introduction to Arrays

- An array is a collection of elements of the same data type stored in contiguous memory locations.
- Arrays are used to store and manipulate large amounts of data.
- Arrays are declared using the data type followed by the size of the array in square brackets.

### One-Dimensional Arrays

- A one-dimensional array is an array that stores elements in a single row or column.
- Example: `int arr[5] = {1, 2, 3, 4, 5};`
- Initializations: `arr[i] = value;`
- Accessing elements: `arr[i];`

### Two-Dimensional Arrays

- A two-dimensional array is an array of arrays.
- Example: `int arr[2][3] = {{1, 2, 3}, {4, 5, 6}};`
- Initializations: `arr[i][j] = value;`
- Accessing elements: `arr[i][j];`

### Important Formulas and Definitions

- **Array Indexing Formula**: `arr[i]` (one-dimensional) and `arr[i][j]` (two-dimensional)
- **Array Size Formula**: `size = number_of_elements`

### Important Theorems and Concepts

- **Array Number System**: System of discrete values used to represent array indices.
- **Array Traversal**: Method of traversing an array from start to end.

### Key Takeaways

- Understand the concepts of one-dimensional and two-dimensional arrays.
- Learn how to initialize and access array elements.
- Familiarize yourself with the array indexing formula and size formula.
