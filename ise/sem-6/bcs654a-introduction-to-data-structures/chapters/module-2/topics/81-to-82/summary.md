# **Revision Notes: 8.1 to 8.2 - Introduction to Data Structures**

### Key Concepts

- **Data Structures:** A way to organize and store data in a computer so that it can be efficiently accessed and modified.
- **Types of Data Structures:** Arrays, Linked Lists, Stacks, Queues, Trees (Binary Search Trees, AVL Trees, etc.), Graphs.
- **Advantages and Disadvantages:** Each data structure has its own advantages and disadvantages.

### Important Formulas and Definitions

- **Time Complexity:**
  - O(1) - Constant Time Complexity
  - O(log n) - Logarithmic Time Complexity
  - O(n) - Linear Time Complexity
  - O(n log n) - Linearithmic Time Complexity
  - O(n^2) - Quadratic Time Complexity
- **Space Complexity:**
  - O(1) - Constant Space Complexity
  - O(n) - Linear Space Complexity
- **Big O Notation:**
  - Used to describe the upper bound of an algorithm's time or space complexity.

### Important Theorems

- **Heapsort Algorithm:**
  - A comparison-based sorting algorithm that uses a binary heap data structure.
- **Merge Sort Algorithm:**
  - A divide-and-conquer algorithm that splits the input into smaller subarrays and merges them.

### Key Terms

- **Top-Down Approach:** A method for solving problems by breaking them down into smaller subproblems.
- **Bottom-Up Approach:** A method for solving problems by combining the solutions of smaller subproblems.
- **Dynamic Programming:** A method for solving problems by breaking them down into smaller subproblems and solving each subproblem only once.

### Quick Revision Tips

- Practice solving problems on paper to understand the time and space complexities of different algorithms.
- Learn the trade-offs between different data structures and algorithms.
- Review the key concepts and formulas regularly to reinforce your understanding.
