# **6.1 to 6.3 Revision Notes**

## **6.1 Sets and Operations**

- **Definition:** A set is a collection of distinct objects, called elements or members, that can be anything (objects, numbers, characters, etc.).
- **Types of Sets:**
  - **Union:** A set that contains all elements from two or more sets.
  - **Intersection:** A set that contains only the elements common to two or more sets.
  - **Difference:** A set that contains elements that are in one set but not in another.
- **Set Operations Formulas:**
  - Union: A ∪ B = {x | x ∈ A or x ∈ B}
  - Intersection: A ∩ B = {x | x ∈ A and x ∈ B}
  - Difference: A \ B = {x | x ∈ A and x ∉ B}

## **6.2 relations and Functions**

- **Definition:** A relation between two sets A and B is a subset of the Cartesian product A × B.
- **Function:** A relation from A to B is a subset of A × B such that each element of A is related to exactly one element of B.
- **Function Notation:** f: A → B, where f(x) is the output value for input x.

## **6.3 Equivalence Relations and Partitions**

- **Definition:** An equivalence relation is a binary relation that is reflexive, symmetric, and transitive.
- **Equivalence Class:** A set of elements that are related to each other under an equivalence relation.
- **Partition:** A division of a set into non-empty subsets such that every element is in exactly one subset.

**Important Formulas and Theorems:**

- **De Morgan's Law:** (A ∩ B)' = A' ∪ B'
- **Distributive Law:** A ∪ (B ∩ C) = (A ∪ B) ∩ (A ∪ C)
