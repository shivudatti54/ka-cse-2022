# Learning Objectives

After studying this topic, you should be able to:

1. Define a queue and explain the First-In-First-Out (FIFO) principle.
2. Identify and describe the basic operations performed on a queue (enqueue, dequeue, peek, isEmpty, isFull).
3. Implement a basic linear queue using an array in the C programming language.
4. Analyze the limitation of memory wastage in a simple linear queue implementation.
5. Differentiate between different types of queues: linear, circular, double-ended, and priority queues.
6. List and explain common real-world and computer science applications of queues.
