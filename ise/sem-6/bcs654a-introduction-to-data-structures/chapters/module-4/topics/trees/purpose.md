# Learning Objectives
After studying this topic, you should be able to:
1.  State and verify the multiple equivalent definitions of a tree.
2.  Calculate basic properties of trees (number of leaves, height) and forests.
3.  Prove that a connected graph with n vertices must have at least n-1 edges, and exactly n-1 if it is a tree.
4.  Apply Breadth-First Search (BFS) or Depth-First Search (DFS) to find a spanning tree for a given graph.
5.  Execute Kruskal's and Prim's algorithms step-by-step on a small weighted graph to find its Minimum Spanning Tree (MST).
6.  Compare and contrast Kruskal's and Prim's algorithms, including their data structures and time complexities.
