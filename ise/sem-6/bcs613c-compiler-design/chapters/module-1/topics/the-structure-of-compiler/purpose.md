# Learning Objectives

After studying this topic, you should be able to:

1.  Identify and list the major phases involved in the compilation process.
2.  Describe the specific function and key output of each phase (Lexical Analysis, Syntax Analysis, Semantic Analysis, Intermediate Code Generation, Code Optimization, Code Generation).
3.  Explain the role of the symbol table and error handling routines throughout compilation.
4.  Differentiate between the compiler's front end (analysis) and back end (synthesis), including which phases belong to each and their respective dependencies.
5.  Trace a simple programming language statement through each phase of the compiler, predicting the output form at each stage.
6.  Justify the use of an Intermediate Representation (IR) in compiler design and state its key advantages.
