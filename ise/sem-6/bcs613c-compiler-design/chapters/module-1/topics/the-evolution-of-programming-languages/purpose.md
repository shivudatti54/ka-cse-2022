# Learning Objectives

After studying this topic, you should be able to:

1.  Identify and distinguish between the four primary generations of programming languages (1GL, 2GL, 3GL, 4GL) based on their key characteristics and examples.
2.  Explain the impact of increased language abstraction on the complexity and required phases of a compiler, specifically linking high-level features to the need for syntax-directed translation.
3.  Compare and contrast the imperative (procedural) and declarative programming paradigms, providing clear examples of each.
4.  Describe the core principles of major programming paradigms such as Object-Oriented and Functional programming.
5.  Analyze how modern language trends (e.g., managed runtimes, multi-paradigm designs) are a continuation of the historical evolution towards greater abstraction and portability.
6.  Evaluate the trade-offs between low-level languages (e.g., control, speed) and high-level languages (e.g., productivity, portability) in the context of compiler operation.
