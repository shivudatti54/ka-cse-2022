# Learning Objectives

After studying this topic, you should be able to:

1. Define Syntax Directed Definitions and explain their role in compiler design
2. Differentiate between synthesized and inherited attributes with examples
3. Construct S-attributed and L-attributed definitions for given grammar productions
4. Analyze and create dependency graphs for semantic rules
5. Implement SDDs with appropriate parsing techniques (LR for S-attributed, LL for L-attributed)
6. Evaluate attribute values for given input strings using SDD rules
