# Learning Objectives

After studying this topic, you should be able to:

1. Explain the difference between top-down and bottom-up parsing and the role of LR parsers.
2. Define and interpret LR(0) items for a given grammar production.
3. Compute the closure and goto of a set of LR(0) items.
4. Construct the canonical collection of LR(0) sets for a small grammar.
5. Build the ACTION and GOTO tables for an SLR(1) parser.
6. Simulate the steps of the SLR parsing algorithm for a given input string using the parsing tables.
