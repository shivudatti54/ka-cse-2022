# Learning Objectives

After studying this topic, you should be able to:

1. Explain the working principle of Linear Search algorithm and describe its step-by-step execution process.

2. Implement Linear Search in C programming using both iterative and recursive approaches.

3. Analyze the time and space complexity of Linear Search, distinguishing between best, worst, and average case scenarios.

4. Compare Linear Search with other searching algorithms, particularly Binary Search, and determine appropriate use cases for each.

5. Apply Linear Search to solve practical problems involving searching in one-dimensional arrays.

6. Identify and explain the sentinel optimization technique for improving Linear Search performance.

7. Trace through the algorithm manually to determine the output for given inputs, a skill essential for examination success.
