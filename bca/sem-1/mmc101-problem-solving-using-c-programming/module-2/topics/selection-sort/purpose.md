# Learning Objectives

After studying this topic, you should be able to:

1. Explain the working principle of Selection Sort and how it divides the array into sorted and unsorted portions.

2. Implement Selection Sort in C programming language with correct syntax and logic.

3. Analyze and determine the time complexity of Selection Sort for best, average, and worst-case scenarios.

4. Identify the space complexity of Selection Sort and explain why it qualifies as an in-place sorting algorithm.

5. Trace through given arrays manually to demonstrate how Selection Sort processes elements pass by pass.

6. Compare Selection Sort with other sorting algorithms like Bubble Sort and Insertion Sort based on time complexity, space complexity, and stability.

7. Modify the Selection Sort algorithm to sort in descending order by changing the comparison operator.

8. Explain the concept of stability in sorting algorithms and determine whether Selection Sort is stable or unstable with justification.
