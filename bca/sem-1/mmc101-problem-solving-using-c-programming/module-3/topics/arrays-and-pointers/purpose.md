# Learning Objectives

After studying this topic, you should be able to:

1. DECLARE AND INITIALIZE one-dimensional and two-dimensional arrays with appropriate data types

2. ACCESS AND MANIPULATE array elements using both subscript notation and pointer arithmetic

3. EXPLAIN THE RELATIONSHIP between array names and pointers in C

4. PERFORM POINTER ARITHMETIC operations including addition, subtraction, and comparison

5. WRITE FUNCTIONS that accept arrays as parameters using both array notation and pointer notation

6. IMPLEMENT PROGRAMS using arrays and pointers to solve common problems like searching and matrix operations

7. DISTINGUISH between arrays and pointers, including their memory characteristics and modifiability
