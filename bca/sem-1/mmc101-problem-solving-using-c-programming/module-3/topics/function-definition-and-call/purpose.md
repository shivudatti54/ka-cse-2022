# Learning Objectives

After studying this topic, you should be able to:

1. Define a function in C with correct syntax, including return type, function name, parameter list, and function body.

2. Explain the difference between function definition and function call, and describe the sequence of events that occurs during a function call.

3. Distinguish between actual parameters (arguments) and formal parameters, and explain how parameters are passed by value in C.

4. Write and use functions that return values using the return statement, and functions that do not return values using void return type.

5. Demonstrate understanding of variable scope and lifetime within functions, including automatic and static storage classes.

6. Trace the execution of programs involving multiple function calls and predict the output accurately.

7. Identify and correct common errors related to function definition and function calls in C programs.
