# Learning Objectives

After studying this topic, you should be able to:

1. Understand the fundamental principles of binary search algorithm and how it achieves O(log n) time complexity through divide and conquer approach.

2. Explain the working of recursion in C programming and identify the base case and recursive case in recursive functions.

3. Implement binary search using recursion in C, including proper function definition with appropriate parameters (array, low, high, key).

4. Analyze the time and space complexity of recursive binary search and compare it with iterative binary search.

5. Trace the execution of recursive binary search for given inputs, showing the sequence of recursive calls and their outcomes.

6. Modify binary search to handle variations such as finding first occurrence, last occurrence, and counting duplicates in a sorted array.

7. Recognize common pitfalls in binary search implementation, including integer overflow and incorrect base case conditions.

8. Apply binary search using recursion to solve practical searching problems in C programming assignments and examination questions.
