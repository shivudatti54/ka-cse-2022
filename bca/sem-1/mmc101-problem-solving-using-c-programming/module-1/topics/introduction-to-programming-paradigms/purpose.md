# Learning Objectives

After studying this topic, you should be able to:

1. Define what a programming paradigm is and explain its significance in computer programming

2. Identify and distinguish between the major programming paradigms: procedural, imperative, declarative, functional, and object-oriented

3. Explain why C is classified as a procedural programming language and list its key characteristics

4. Compare the imperative and declarative paradigms, understanding the difference between describing "how" versus "what"

5. Understand the relationship between programming paradigms and problem-solving approaches

6. Recognize that modern languages can support multiple paradigms and explain the implications

7. Apply knowledge of paradigms to make informed decisions about appropriate programming approaches for different problem types
