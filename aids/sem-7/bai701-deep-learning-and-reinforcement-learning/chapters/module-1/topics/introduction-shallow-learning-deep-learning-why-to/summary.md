# **Deep Learning and Reinforcement Learning Revision Notes**

## **Introduction**

- Deep learning is a subset of machine learning that uses neural networks with multiple layers to analyze data.
- Inspired by the structure and function of the human brain.
- Also known as neural networks or neural architecture.

## **Shallow Learning**

- Shallow learning uses a single layer of weights and biases to learn patterns in data.
- Examples: decision trees, linear regression.
- Limitation: cannot learn complex patterns in data.

## **Deep Learning**

- Deep learning uses multiple layers of weights and biases to learn complex patterns in data.
- Examples: convolutional neural networks (CNNs), recurrent neural networks (RNNs).
- Key features:
  - Multi-layer perceptron (MLP)
  - Convolutional neural network (CNN)
  - Recurrent neural network (RNN)
  - Long short-term memory (LSTM) network

## **Why to use Deep Learning**

- Can learn complex patterns in data.
- Can be used for image and speech recognition, natural language processing, and more.
- Key benefits:
  - Improved accuracy
  - Increased efficiency

## **How Deep Learning Works**

- Data is fed into the network, where it is processed layer by layer.
- Each layer applies a non-linear transformation to the data, allowing the network to learn complex patterns.
- Output is generated by the final layer.

## **Deep Learning Challenges**

- **Vanishing gradients**: gradients become smaller, making it harder for the network to learn.
- **Exploding gradients**: gradients become larger, making the network unstable.
- **Overfitting**: network becomes too specialized to the training data.
- **Underfitting**: network fails to capture important patterns in the data.
- **Regularization techniques**: dropout, L1, and L2 regularization can help mitigate these challenges.

## **Key Formulas and Definitions**

- **Activation function**: introduces non-linearity to the model (e.g. ReLU, sigmoid).
- **Backpropagation**: algorithm used to compute gradients.
- **Loss function**: measures the difference between predicted and actual outputs (e.g. mean squared error, cross-entropy).
- **Regularization**: techniques used to prevent overfitting (e.g. dropout, L1, L2 regularization).
