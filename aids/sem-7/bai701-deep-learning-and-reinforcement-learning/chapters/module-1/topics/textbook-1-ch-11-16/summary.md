**Textbook 1: Ch 1.1 – 1.6 Revision Notes - Deep Learning and Reinforcement Learning**
==========================================================================

**Definition and Introduction**
-----------------------------

* Deep learning is a subfield of machine learning that focuses on neural networks with multiple layers.
* Deep learning techniques have achieved state-of-the-art performance in many applications, including image classification, natural language processing, and speech recognition.

**Key Concepts**
----------------

* **Neural Networks**:
	+ definition: a collection of interconnected nodes (neurons) that process and transmit information.
	+ types: feedforward, convolutional, recurrent
* **Activation Functions**:
	+ definition: a function that maps the input to an output, introducing non-linearity in the model.
	+ examples: sigmoid, ReLU, tanh
* **Loss Functions**: definition: a function that measures the difference between predicted and actual outputs.

**Formulas and Theorems**
-------------------------

* **Forward Pass**: the process of propagating inputs through the network to produce an output.
* **Backward Pass**: the process of computing the gradients of the loss function with respect to the model's parameters.
* **Stochastic Gradient Descent (SGD)**: an optimization algorithm for minimizing the loss function.
* **Batch Normalization**: a technique that normalizes the inputs to each layer to improve training.

**Important Definitions**
-------------------------

* **Activation Function**: a function that maps the input to an output, introducing non-linearity in the model.
* **Recurrent Neural Network (RNN)**: a type of neural network that processes sequences of data.
* **Convolutional Neural Network (CNN)**: a type of neural network that processes images.

**Key Theorems**
-----------------

* **The Fundamental Theorem of Calculus**: a theorem that establishes the relationship between the derivative and the integral.
* **The Chain Rule**: a theorem that establishes the relationship between the derivative of a composite function and the derivatives of its components.

**Revision Tips**
-----------------

* Review the definitions and formulas of key concepts.
* Practice computing the forward and backward passes for a simple neural network.
* Familiarize yourself with common optimization algorithms, such as SGD and Adam.
