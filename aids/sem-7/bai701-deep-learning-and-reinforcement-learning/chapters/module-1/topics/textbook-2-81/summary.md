# **Textbook 2: 8.1 - Deep Learning and Reinforcement Learning**

## **Key Concepts**

- **Deep Learning**:
  - A subset of Machine Learning that uses neural networks with multiple layers
  - Inspired by the structure and function of the human brain
  - Used for tasks such as image and speech recognition, natural language processing, and decision-making

## **Types of Deep Learning**

- **Feedforward Neural Networks**: simplest type of neural network
- **Convolutional Neural Networks (CNNs)**: for image recognition and processing
- **Recurrent Neural Networks (RNNs)**: for sequential data such as speech and text

## **Reinforcement Learning**

- **Definition**: an agent learns to take actions to maximize a reward signal
- **Key Concepts**:
  - **Agent**: the entity that interacts with the environment
  - **Environment**: the external world that the agent interacts with
  - **Policy**: the strategy used by the agent to select actions
  - **Value Function**: a function that estimates the expected return of a particular action

## **Important Formulas and Theorems**

- **Bellman Equation**: a mathematical equation that updates the value function
- **Q-Learning**: an algorithm for learning the value function
- **Policy Gradient Methods**: a class of algorithms for learning the policy

## **Key Definitions**

- **Markov Decision Process (MDP)**: a mathematical framework for reinforcement learning
- **State**: a description of the current situation
- **Action**: a decision made by the agent
- **Reward**: a feedback signal that indicates the quality of the action

## **Important Theorems**

- **Bellman's Inequality**: a bound on the expected return of an action
- **Optimality Implication**: a theorem that states that a policy is optimal if it satisfies certain conditions.
