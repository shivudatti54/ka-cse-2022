# Learning Objectives

After studying this topic, you should be able to:

1. Explain the fundamental nature of learning in Reinforcement Learning and how it differs from supervised and unsupervised learning paradigms.

2. Understand and articulate the exploration-exploitation dilemma and its significance in RL learning problems.

3. Describe the role of value functions (state-value and action-value) in learning and explain how they are updated through experience.

4. Apply temporal-difference learning methods, particularly TD(0), and explain how the TD error drives the learning process.

5. Implement and analyze Q-learning algorithm, understanding its off-policy nature and convergence properties.

6. Compare and contrast on-policy methods (SARSA) with off-policy methods (Q-learning) and explain scenarios where each is appropriate.

7. Explain the influence of learning rate and discount factor parameters on learning behavior and convergence.

8. Design and evaluate basic exploration strategies for RL learning problems.
