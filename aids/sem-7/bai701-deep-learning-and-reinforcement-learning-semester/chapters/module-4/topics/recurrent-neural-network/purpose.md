# Learning Objectives
After studying this topic, you should be able to:
1.  Explain the limitation of feedforward networks for sequential data and the need for Recurrent Neural Networks (RNNs).
2.  Describe the core architecture of a simple RNN, including the concept of a hidden state and unfolding through time.
3.  Write the mathematical equations governing the update of the hidden state in a vanilla RNN.
4.  Identify and differentiate between the main RNN architectural types (One-to-One, One-to-Many, Many-to-One, Many-to-Many).
5.  Explain the problem of vanishing and exploding gradients in deep and recurrent networks.
6.  Detail the structure of an LSTM cell, including the function of its forget, input, and output gates, and explain how it mitigates the vanishing gradient problem.
7.  Compare and contrast LSTMs and GRUs, listing one key advantage of each.
