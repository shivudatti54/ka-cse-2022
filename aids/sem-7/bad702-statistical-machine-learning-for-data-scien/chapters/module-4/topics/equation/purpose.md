# Learning Objectives

After studying this topic, you should be able to:

1. Explain the importance of evaluating community detection algorithms
2. Calculate and interpret modularity scores for given network partitions
3. Apply external evaluation metrics (NMI, ARI) when ground truth is available
4. Identify the limitations and biases of different evaluation metrics
5. Select appropriate evaluation methods for different community detection scenarios
6. Compare and contrast the performance of different community detection algorithms using multiple metrics
