# Learning Objectives

After studying this topic, you should be able to:

1.  Differentiate between the use cases for regression and classification algorithms.
2.  Explain the limitation of linear models and the motivation for using Polynomial Regression.
3.  Formulate the equation for a polynomial regression model of a given degree.
4.  Describe the concept of the sigmoid function and how it enables Logistic Regression to output probabilities.
5.  Interpret the coefficients of a logistic regression model in terms of odds ratios.
6.  Compare and contrast the cost functions (MSE vs. Log Loss) and fitting methods (OLS vs. MLE) used in linear/polynomial and logistic regression.
7.  Identify strategies to prevent overfitting in Polynomial Regression and apply techniques to use Logistic Regression for non-linear classification.
