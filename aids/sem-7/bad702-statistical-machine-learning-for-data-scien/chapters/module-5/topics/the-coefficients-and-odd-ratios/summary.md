# **The Coefficients and Odd Ratios**

## **Definition and Notation**

- **Coefficient**: a numerical value that represents the change in the response variable for a one-unit change in the predictor variable, while holding all other predictor variables constant.
- **Odd Ratio (OR)**: the ratio of the coefficient to its standard error.

## **Key Formulas and Theorems**

- **Coefficient**: β = (SXY - SX\* Sy) / Sy^2
- **Standard Error of Coefficient**: SE(β) = sqrt((SXY^2 / n) + (SX\* Sy^2 / n))
- **Odd Ratio**: OR = exp(β / SE(β))
- **Theorem**: The odd ratio is equivalent to the exponentiation of the coefficient, i.e., OR = exp(β), when β is the coefficient and SE(β) is the standard error.

## **Key Points**

- Coefficients represent the change in the response variable for a one-unit change in the predictor variable.
- Odd ratios represent the change in the odds of the response variable for a one-unit change in the predictor variable.
- The odd ratio is equivalent to the exponentiation of the coefficient.
- The standard error of the coefficient is used to calculate the odd ratio.

## **Importance**

- Coefficients are used to interpret the relationships between predictor variables and the response variable.
- Odd ratios are used to estimate the effect of a predictor variable on the odds of the response variable.
