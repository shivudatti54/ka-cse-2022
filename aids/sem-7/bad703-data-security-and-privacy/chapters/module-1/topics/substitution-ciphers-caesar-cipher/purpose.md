# Learning Objectives
After studying this topic, you should be able to:
1.  Define and explain the principle of a substitution cipher.
2.  Perform encryption and decryption using the Caesar Cipher given a key.
3.  Calculate the keyspace for both Caesar and Monoalphabetic ciphers.
4.  Explain the vulnerability of monoalphabetic ciphers to frequency analysis.
5.  Demonstrate a basic frequency analysis attack on a monoalphabetic ciphertext.
6.  Compare and contrast the Caesar and Monoalphabetic substitution ciphers in terms of key, strength, and weaknesses.
