# Learning Objectives
After studying this topic, you should be able to:
1. Explain the purpose and importance of Encapsulating Security Payload (ESP) in network security
2. Describe the structure and components of an ESP packet
3. Differentiate between transport mode and tunnel mode operations of ESP
4. Identify the security services provided by ESP and how they are implemented
5. Compare and contrast ESP with Authentication Header (AH)
6. Understand the role of ESP in the broader IPsec framework
