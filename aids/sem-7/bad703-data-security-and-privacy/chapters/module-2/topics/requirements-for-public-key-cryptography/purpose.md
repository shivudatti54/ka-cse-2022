# Learning Objectives

After studying this topic, you should be able to:

1. Explain the fundamental difference between symmetric and asymmetric (public key) cryptography and articulate why public key cryptography was invented.

2. List and describe the six fundamental requirements for public key cryptosystems: key generation efficiency, encryption/decryption efficiency, one-way function property, trapdoor information, semantic security, and forward security.

3. Distinguish between a one-way function and a trapdoor function with appropriate examples from cryptography.

4. Understand the concept of semantic security (IND-CPA) and explain why probabilistic encryption is necessary for modern cryptographic systems.

5. Explain perfect forward secrecy (PFS) and distinguish between key exchange protocols that provide PFS and those that do not.

6. Discuss practical implementation considerations including key sizes, bandwidth requirements, and resistance to side-channel attacks.

7. Analyze the computational hardness assumptions underlying popular public key cryptosystems and explain the future threat posed by quantum computers.
