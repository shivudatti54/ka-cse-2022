# Learning Objectives
After studying this topic, you should be able to:
1.  Define a Pseudorandom Number Generator (PRNG) and distinguish it from a True Random Number Generator (TRNG).
2.  Explain the two fundamental properties that make a PRNG cryptographically secure (CSPRNG).
3.  Describe the common internal structure of a PRNG, including the roles of the internal state, output function, and update function.
4.  Identify why common non-cryptographic PRNGs (like Linear Congruential Generators) are insecure for cryptographic applications.
5.  Recognize secure PRNG constructions based on cryptographic primitives like hash functions and block ciphers.
6.  Articulate the critical importance of seed management and entropy in the security of a PRNG.
