# Learning Objectives
After studying this topic, you should be able to:
1.  Explain the purpose and necessity of the Transport Layer Security (TLS) protocol in web security.
2.  Describe the step-by-step process of the TLS handshake protocol, identifying the purpose of each key message exchanged.
3.  Differentiate between the roles of the TLS Handshake Protocol and the TLS Record Protocol.
4.  Analyze how TLS integrates cryptographic concepts from symmetric encryption, public-key cryptography, and hash functions to achieve its security goals.
5.  Compare and contrast the features and security improvements of TLS 1.2 and TLS 1.3.
6.  Identify common attacks on TLS/SSL (e.g., BEAST, POODLE) and explain the corresponding mitigation strategies.
