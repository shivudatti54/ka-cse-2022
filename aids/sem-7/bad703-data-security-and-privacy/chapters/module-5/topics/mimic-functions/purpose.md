# Learning Objectives
After studying this topic, you should be able to:
1.  Explain the fundamental concepts of MPI, including communicators, groups, and ranks.
2.  Implement point-to-point communication using `MPI_Send` and `MPI_Recv`, and describe their blocking semantics.
3.  Utilize key collective communication operations (`MPI_Bcast`, `MPI_Scatter`, `MPI_Gather`, `MPI_Reduce`) to implement common parallel patterns.
4.  Design and implement a parallel algorithm using data decomposition, as demonstrated by the trapezoidal rule example.
5.  Create and use MPI derived datatypes to communicate non-contiguous or complex data structures.
6.  Measure and evaluate the performance of MPI programs using `MPI_Wtime`.
