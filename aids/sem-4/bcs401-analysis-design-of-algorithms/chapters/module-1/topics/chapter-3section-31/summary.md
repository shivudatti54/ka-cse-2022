### Algorithm Fundamentals

- **Definition**: A set of instructions that a computer follows to solve a problem or perform a task.
- **Importance**: Algorithms are the building blocks of computer science and are used in a wide range of applications, including operating systems, databases, and machine learning.

### Types of Algorithms

- **Determine the type of problem**: Algorithmic problems can be classified into two main categories:
  - **Decide problems**: These problems can be solved in a finite number of steps. Examples include sorting and searching.
  - **Search problems**: These problems require an unbounded amount of time to solve. Examples include finding a path in a maze.

### Characterization of Algorithms

- **Pseudocode**: A high-level representation of an algorithm that uses a sequence of steps to solve a problem. It is easier to read and maintain than machine code.
- **Time complexity**: A measure of the amount of time an algorithm takes to complete. It is expressed using Big O notation (e.g., O(n), O(n log n)).
- **Space complexity**: A measure of the amount of memory an algorithm uses. It is also expressed using Big O notation (e.g., O(n), O(log n)).

### Big O Notation

- **O-notation**: A way to describe the upper bound of an algorithm's time or space complexity.
- **Common time complexities**:
  - O(1) - constant time complexity (e.g., accessing an array by index)
  - O(log n) - logarithmic time complexity (e.g., binary search)
  - O(n) - linear time complexity (e.g., searching an array)
  - O(n log n) - linearithmic time complexity (e.g., sorting algorithms)
  - O(n^2) - quadratic time complexity (e.g., bubble sort)
  - O(2^n) - exponential time complexity (e.g., recursive algorithms)
- **Common space complexities**:
  - O(1) - constant space complexity (e.g., arrays)
  - O(n) - linear space complexity (e.g., linked lists)

### Important Formulas and Theorems

- **Graham's Scan Algorithm**: An algorithm for finding the convex hull of a set of points in 2D space.
- **Hermite Interpolation Formula**: A mathematical formula for interpolating values in a function.
- **Sorting algorithms**: Algorithms for arranging elements in a specific order (e.g., merge sort, quick sort).
