# **Chapter 2(Sections 2.1) Analysis & Design of Algorithms**

**Key Concepts:**

- **Algorithm Analysis**
  - Big O notation (Upper bound, lower bound, tight bound)
  - Time complexity (O(1), O(log n), O(n), O(n log n), O(n^2), O(2^n), etc.)
  - Space complexity (O(1), O(log n), O(n), O(n log n), etc.)
- **Algorithm Design**
  - Divide and Conquer algorithm
  - Greedy algorithm
  - Dynamic Programming
- **Importance of Algorithm Analysis**
  - Measuring algorithm efficiency
  - Identifying time and space complexities
  - Predicting algorithm performance
- **Important Formulas and Definitions:**
  - Big O notation: O(f(n)) = upper bound
  - Big Ω notation: Ω(f(n)) = lower bound
  - Big θ notation: θ(f(n)) = tight bound
  - Time complexity: O(1), O(log n), O(n), O(n log n), O(n^2), O(2^n), etc.
  - Space complexity: O(1), O(log n), O(n), O(n log n), etc.
- **Important Theorems:**
  - Master Theorem for solving recurrence relations
  - Recurrence relation formulas (e.g., Fibonacci sequence)
