# Space-Time Tradeoffs: Sorting by Counting - Comparison Counting Sort

==============================================

### Key Points

- **Comparison Counting Sort**: A comparison-based sorting algorithm that uses counting sort as a subroutine to sort smaller subarrays.
- **Time Complexity**: O(nk), where n is the number of elements and k is the range of input.
- **Space Complexity**: O(n + k)
- **Stability**: Stable sorting algorithm
- **Comparison-Based**: Uses comparisons to sort elements

### Important Formulas and Definitions

- **Stability**: A sorting algorithm is stable if it maintains the relative order of equal elements.
- **Counting Sort**: A sorting algorithm that uses the range of input to create an auxiliary array to store the counts of individual elements.
- **Comparison Counting Sort Formula**: `time = O(nk)`, `space = O(n + k)`

### Theorem

- **Comparison Counting Sort Theorem**: If input size is n and range is k, comparison counting sort has a time complexity of O(nk) and a space complexity of O(n + k).
