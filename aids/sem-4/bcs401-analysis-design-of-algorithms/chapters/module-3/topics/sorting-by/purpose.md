# Learning Objectives

After studying this topic, you should be able to:

1. Explain the fundamental difference between distribution-based sorting and comparison-based sorting algorithms
2. Describe how Counting Sort achieves linear time complexity and identify its optimal use cases
3. Implement Counting Sort with proper handling of range determination and cumulative count calculation
4. Analyze the time and space complexity of Radix Sort and explain its dependency on digit count and radix
5. Distinguish between LSD and MSD Radix Sort approaches and their respective applications
6. Apply Bucket Sort to uniformly distributed data and analyze its expected performance
7. Evaluate when to use distribution sorting versus comparison sorting based on input characteristics
8. Demonstrate understanding of stability in sorting algorithms and its practical importance
