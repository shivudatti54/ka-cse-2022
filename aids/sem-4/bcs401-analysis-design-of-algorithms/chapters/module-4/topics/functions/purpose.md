# Learning Objectives

After studying this topic, you should be able to:

1. Define and identify different types of functions (injective, surjective, bijective, constant, identity)
2. Determine whether a given function is injective, surjective, bijective, or neither
3. Perform function composition and understand its properties
4. Find inverse functions when they exist
5. Apply floor and ceiling functions correctly
6. Calculate the number of possible functions between finite sets
