# Learning Objectives

After studying this topic, you should be able to:

1. Define and explain the concept of a Minimum Spanning Tree (MST) and its properties in the context of weighted undirected graphs.

2. Describe the step-by-step procedure of Kruskal's Algorithm, including edge sorting, cycle detection, and union operations.

3. Apply Kruskal's Algorithm to find the minimum spanning tree for a given weighted graph by tracing through all steps manually.

4. Implement cycle detection using the Disjoint Set Union (Union-Find) data structure and explain how FIND and UNION operations work.

5. Analyze the time and space complexity of Kruskal's Algorithm and relate it to the sorting and DSU operations.

6. Compare and contrast Kruskal's Algorithm with Prim's Algorithm, understanding when each algorithm is more suitable.

7. Prove the correctness of Kruskal's Algorithm using the cut property of minimum spanning trees.

8. Apply Kruskal's Algorithm to solve real-world problems such as network design, connecting cities with minimum cost, and cluster analysis.
