# Learning Objectives

After studying this topic, you should be able to:

1. Explain the working principle of Dijkstra's Algorithm and its relationship with the greedy method paradigm.

2. Trace through Dijkstra's Algorithm step-by-step on a given weighted graph and determine shortest paths from a source vertex.

3. Apply the relaxation operation correctly to update distance estimates during algorithm execution.

4. Analyze the time and space complexity of Dijkstra's Algorithm under different implementations (array vs. priority queue).

5. Identify why Dijkstra's Algorithm fails with negative edge weights and explain when to use alternative algorithms.

6. Implement Dijkstra's Algorithm in a programming language using appropriate data structures.

7. Compare Dijkstra's Algorithm with other shortest path algorithms like BFS and Bellman-Ford.
