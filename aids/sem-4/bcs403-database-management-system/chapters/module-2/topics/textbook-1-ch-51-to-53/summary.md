### Key Concepts

- **Definition:** A database management system (DBMS) is a software system that allows you to define, create, maintain, and manipulate databases.
- **DBMS Functions:**
  - Data definition: defining database structures and relationships
  - Data control: controlling access to data
  - Data protection: protecting data from unauthorized access
  - Data retrieval: retrieving data from the database
  - Data manipulation: modifying data in the database
- **Types of DBMS:**
  - Relational DBMS (RDBMS)
  - Object-Oriented DBMS (OODB)
  - Hierarchical DBMS
  - Network DBMS

### Important Formulas and Definitions

- **Normal Form (NF):** A database is in NF if it is free from redundancy and depends on the entire primary key.
- **Primary Key (PK):** A unique identifier for each record in a table.
- **Foreign Key (FK):** A field in a table that refers to the primary key of another table.
- **Database Normalization:**
  - First Normal Form (1NF): Each column contains a single value.
  - Second Normal Form (2NF): Each non-key attribute depends on the entire primary key.
  - Third Normal Form (3NF): If a table is in 2NF, and a non-key attribute depends on another non-key attribute, then it should be moved to a separate table.

### Important Theorems

- **Entity-Relationship Model (ERM):** A theoretical model that represents the relationships between entities in a database.
- **Universal Relation Model (URM):** A theoretical model that represents the relationships between entities and attributes in a database.
