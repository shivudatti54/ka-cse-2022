# **3.1 to 3.10 RBT: L1 - Database Management System Revision Notes**

### Introduction

- Database management system (DBMS) is a software that enables organizations to store, manage, and retrieve data efficiently.
- DBMS provides a platform for data storage, retrieval, and manipulation.

### Definitions

- **Database**: A collection of organized data that is stored in a way that allows for efficient retrieval and manipulation.
- **Database management system (DBMS)**: A software that manages and controls access to a database.

### Key Concepts

- **Data Entities**: The objects or concepts that are represented in a database, such as customers, orders, and products.
- **Attributes**: The characteristics or features of data entities, such as name, address, and phone number.
- **Relationships**: The connections between data entities, such as one-to-one, one-to-many, and many-to-many.

### Theories and Formulas

- **First Normal Form (1NF)**: A database is in 1NF if each table cell contains only one value.
- **Second Normal Form (2NF)**: A database is in 2NF if each non-key attribute depends on the entire primary key.
- **Third Normal Form (3NF)**: A database is in 3NF if each non-key attribute depends on the entire primary key and not on another non-key attribute.
- **Boyce-Codd Normal Form (BCNF)**: A database is in BCNF if it is in 3NF and there are no transitive dependencies.

### Important Formulas

- **Relational Algebra**: A formal language for expressing queries and operations on relational databases.
- **Tuple Relational Model**: A model that represents data as a collection of tuples, each containing a set of attributes.

### Important Theorems

- **Database Theorem**: A theorem that states that a relational database can be decomposed into a set of relations.

### Important Concepts

- **Database Design**: The process of designing a database to meet the needs of an organization.
- **Database Implementation**: The process of implementing a database in a computer system.

### Quick Revision Tips

- Review the definitions of data entities, attributes, and relationships.
- Practice applying the theories and formulas of normal forms.
- Review the important concepts of relational algebra and tuple relational model.
- Practice applying the database theorem.
