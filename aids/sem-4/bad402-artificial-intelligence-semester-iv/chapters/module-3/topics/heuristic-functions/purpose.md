# Learning Objectives

After studying this topic, you should be able to:

1. Define heuristic functions and explain their role in informed search algorithms
2. Differentiate between admissible and consistent heuristic functions
3. Design appropriate heuristic functions for given problem domains
4. Calculate heuristic values for problems like the 8-puzzle using different methods
5. Analyze the dominance relationship between multiple heuristic functions
6. Explain how heuristic functions affect the performance of A\* and other informed search algorithms
