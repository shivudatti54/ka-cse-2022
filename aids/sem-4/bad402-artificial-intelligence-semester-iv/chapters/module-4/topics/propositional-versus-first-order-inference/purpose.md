# Learning Objectives

After studying this topic, you should be able to:

1.  Explain the fundamental limitations of Propositional Logic that motivate the use of First-Order Logic.
2.  Define and perform unification on two simple First-Order Logic expressions.
3.  Apply the Generalized Modus Ponens rule to derive a conclusion from a given rule and set of facts using a unifying substitution.
4.  Differentiate between the forward chaining and backward chaining inference strategies, listing one advantage of each.
5.  Compare and contrast Propositional and First-Order Logic inference using a table of key characteristics (expressiveness, elements, complexity).
6.  Explain the terms "undecidable" and "semi-decidable" in the context of FOL inference.
