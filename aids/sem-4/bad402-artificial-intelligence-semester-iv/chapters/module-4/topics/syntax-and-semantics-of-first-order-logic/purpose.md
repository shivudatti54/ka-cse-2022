# Learning Objectives

After studying this topic, you should be able to:

1. Identify and construct valid terms and well-formed formulas in First Order Logic
2. Translate natural language statements into First Order Logic formulas and vice versa
3. Explain the semantic concepts of interpretation, variable assignment, and satisfaction
4. Distinguish between universal and existential quantifiers and use them correctly
5. Analyze the scope of quantifiers and identify bound and free variables
6. Evaluate the truth value of FOL formulas in given interpretations
