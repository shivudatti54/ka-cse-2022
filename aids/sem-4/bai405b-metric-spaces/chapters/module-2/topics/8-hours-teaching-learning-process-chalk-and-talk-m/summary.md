# **Teaching-Learning Process: Chalk and Talk Method / PowerPoint Presentation**

## **Definition of Chalk and Talk Method**

- A traditional teaching method where the lecturer uses a chalkboard to write equations and formulas
- Lecturer writes on the board, and students take notes

## **Definition of PowerPoint Presentation**

- A modern teaching method using computer software to create interactive presentations
- Includes text, images, diagrams, and animations

## **Key Points: Chalk and Talk Method**

- Advantages:
  • Easy to create and modify content
  • Can be used to create interactive presentations
  • Can be used to create multimedia presentations
- Disadvantages:
  • Dependent on technology
  • Can be distracting for students
  • Can lead to a passive learning experience

## **Key Points: PowerPoint Presentation**

- Advantages:
  • Can be easily shared with students
  • Can be easily modified and updated
  • Can include multimedia content
- Disadvantages:
  • Can be distracting for students
  • Requires technical expertise
  • Can lead to a passive learning experience

## **Important Formulas and Definitions**

- **Distance Formula**: $d = \sqrt{(x_2 - x_1)^2 + (y_2 - y_1)^2}$
- **Metric Space**: A set with a distance function, satisfying the properties of symmetry and non-negativity
- **Metric Property**: A set with a distance function that satisfies the properties of symmetry and non-negativity

## **Important Theorems**

- **Triangle Inequality**: For any metric space, $d(x, z) \leq d(x, y) + d(y, z)$
- **Parallelogram Law**: For any metric space, $d(x, z)^2 + d(y, z)^2 = d(x, y)^2 + d(y, z)^2$
