## **Introduction**

In this topic, we will explore the concept of metric spaces and their applications. We will discuss the definition, properties, and examples of metric spaces. The teaching-learning process will be described using the chalk and talk method and PowerPoint presentation techniques.

## **Definition of a Metric Space**

A metric space is a set X together with a metric or distance function d that satisfies the following properties:

- **Non-negativity**: d(x, y) ≥ 0 for all x, y in X
- **Identity of Indiscernibles**: d(x, y) = 0 if and only if x = y
- **Symmetry**: d(x, y) = d(y, x) for all x, y in X
- **Triangle Inequality**: d(x, z) ≤ d(x, y) + d(y, z) for all x, y, z in X

## **Properties of Metric Spaces**

- **Completeness**: A metric space is said to be complete if every Cauchy sequence in the space converges to a point in the space.
- **Compactness**: A metric space is said to be compact if every open cover of the space contains a finite subcover.

## **Examples of Metric Spaces**

- **Real Numbers**: The set of real numbers with the standard Euclidean metric is a metric space.
- **Discrete Metric Space**: A metric space with a distance function that assigns 0 to all points and 1 to any two distinct points is called a discrete metric space.

## **Applications of Metric Spaces**

- **Geometry**: Metric spaces are used to describe geometric shapes and spaces.
- **Topology**: Metric spaces are used to study the properties of topological spaces.
- **Measure Theory**: Metric spaces are used to define and study measures on a space.
