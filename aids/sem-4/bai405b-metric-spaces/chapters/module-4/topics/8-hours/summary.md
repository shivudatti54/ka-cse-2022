# Revision Notes: Compactness in Metric Spaces (8 hours)

==============================================

## Definitions and Notations

- A metric space is a set X equipped with a metric d that satisfies the following properties:
  - d(x, y) ≥ 0 for all x, y in X (non-negativity)
  - d(x, y) = 0 if and only if x = y (identity of indiscernibles)
  - d(x, y) = d(y, x) for all x, y in X (symmetry)
  - d(x, z) ≤ d(x, y) + d(y, z) for all x, y, z in X (triangle inequality)
- A set X in a metric space (X, d) is said to be compact if every open cover of X has a finite subcover.

## Important Formulas and Theorems

- The open ball B(x, r) = {y in X | d(x, y) < r} for a point x in X and a real number r > 0.
- The open cover is a collection of open sets that cover the entire space X.
- The finite subcover is a finite subset of the open cover that still covers X.

## Key Results and Theorems

- Heine-Borel Theorem: A set X in a metric space (X, d) is compact if and only if it is closed and bounded.
- Bolzano-Weierstrass Theorem: Every bounded sequence in a metric space has a convergent subsequence.
- Open Mapping Theorem: If X is a compact metric space and F is a continuous open mapping from X to a metric space Y, then F(X) is open in Y.

## Important Concepts

- Boundedness: A set X in a metric space is bounded if there exists a real number M such that d(x, y) < M for all x, y in X.
- Closedness: A set X in a metric space is closed if its complement is open.
- Compactness: A set X in a metric space is compact if every open cover of X has a finite subcover.

## Key Properties of Compact Sets

- Compact sets are bounded and closed.
- Compact sets have the property that every sequence has a convergent subsequence.
- Compact sets are closed and bounded.
