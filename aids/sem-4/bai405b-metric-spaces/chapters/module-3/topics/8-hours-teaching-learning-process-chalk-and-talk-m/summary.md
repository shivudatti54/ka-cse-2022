# **Teaching-Learning Process: Chalk and Talk Method / PowerPoint Presentation**

**Module:** Complete Metric Spaces and Continuous Functions
**Topic:** (8 hours) Teaching-Learning Process Chalk and Talk Method / PowerPoint Presentation

### Key Points

- **Chalk and Talk Method:**
  - Advantages:
    - Encourages interactive learning
    - Allows for real-time feedback
    - Helps to clarify doubts
  - Disadvantages:
    - Time-consuming
    - Limited to the lecturer's pace
- **PowerPoint Presentation:**
  - Advantages:
    - Visual aids help to illustrate complex concepts
    - Easier to organize and review material
    - Can be used for remote or group learning
  - Disadvantages:
    - May not engage learners as much as a chalk and talk method
    - Requires technical skills to create effective presentations

### Definitions

- **Metric Space:** A set X together with a metric (or distance function) d: X × X → ℝ, satisfying:
  - d(x, y) ≥ 0 for all x, y in X
  - d(x, y) = 0 if and only if x = y
  - d(x, y) = d(y, x) for all x, y in X
  - d(x, z) ≤ d(x, y) + d(y, z) for all x, y, z in X

### Important Formulas and Theorems

- **Bolzano-Weierstrass Theorem:** Every bounded sequence in a metric space has a convergent subsequence.
- **Heine-Borel Theorem:** Every closed and bounded subset of a metric space is compact.
- **Extreme Value Theorem:** Every continuous function on a closed and bounded interval attains its maximum and minimum values.
