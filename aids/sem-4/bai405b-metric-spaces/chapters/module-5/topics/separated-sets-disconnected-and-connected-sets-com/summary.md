# Separated Sets, Disconnected and Connected Sets, Components, Connected Subsets of R, Continuous Functions on Connected Sets

=====================================================

## Definitions and Theorems

- **Metric Space**: A set X equipped with a metric d such that (i) d(x,y) ≥ 0 for all x, y in X (non-negativity) and (ii) d(x,y) = 0 if and only if x = y (identity of indiscernibles).
- **Separated Sets**: A set X is said to be separated if it can be represented as the union of two disjoint non-empty open sets.
- **Disconnected Set**: A set X is said to be disconnected if it can be represented as the union of two disjoint non-empty open sets.
- **Connected Set**: A set X is said to be connected if it cannot be represented as the union of two disjoint non-empty open sets.
- **Component**: The component of a point x in a connected set X is the largest connected subset of X that contains x.
- **Connected Subset of R**: A subset S of R is said to be connected if it cannot be represented as the union of two disjoint non-empty open intervals.

## Key Formulas and Results

- **Baire Category Theorem**: Every complete metric space contains a connected subset of every non-empty open subset.
- **Heine-Borel Theorem**: A set in R is compact if and only if it is closed and bounded.
- **Definition of Connectedness**: A set X is connected if and only if it cannot be represented as the union of two disjoint non-empty open sets.

## Important Theorems

- **Leray's Theorem**: If X is a metric space and Z is a closed subset of X, then Z is a connected subset of X if and only if it is a closed connected subset of a complete metric space.
- **Theorem of Riesz**: If X is a metric space and Z is a connected subset of X, then there exists a continuous function f: X → R such that f(x) = 0 for all x in Z.
