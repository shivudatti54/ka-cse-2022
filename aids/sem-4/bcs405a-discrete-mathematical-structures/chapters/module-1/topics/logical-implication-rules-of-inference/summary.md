# Logical Implication – Rules of Inference

### Definitions

- **Logical Implication**: A statement of the form "If P, then Q" (P → Q) where P is the antecedent and Q is the consequent.
- **Rule of Inference**: A set of rules used to derive conclusions from premises.

### Important Formulas and Theorems

- **Modus Ponens**: From P and P → Q, we can infer Q.
  - Formula: P, (P → Q) ⊢ Q
- **Modus Tollens**: From P and (P → ¬Q), we can infer ¬P.
  - Formula: P, (P → ¬Q) ⊢ ¬P
- **Converse Modus Ponens**: From P, (Q → P), we can infer Q.
  - Formula: P, (Q → P) ⊢ Q
- **Hypothetical Syllogism**: From P, Q, and (P → Q), we can infer Q.
  - Formula: P, Q, (P → Q) ⊢ Q

### Rules of Inference

- **Presumption**: If we assume a hypothesis, we can derive conclusions based on that assumption.
- **Deduction**: If we have a set of premises, we can derive a conclusion from those premises using the rules of inference.
- **Validity**: A set of premises is valid if it always leads to a true conclusion.

### Important Theorems

- **Darley's Theorem**: A set of premises is valid if and only if every possible interpretation of the premises leads to a true conclusion.
- **Hilbert's Theorem**: Every valid set of premises can be derived using the rules of inference.

### Key Points

- Logical implication is a fundamental concept in logic.
- Modus ponens and modus tollens are two important rules of inference.
- Converse modus ponens and hypothetical syllogism are also important rules of inference.
- Validity is a crucial concept in logic, and it is related to the rules of inference.
- Darley's theorem and Hilbert's theorem are important theorems that relate validity to the rules of inference.
