# **Properties of Relations, Computer Recognition – Zero-One Matrices and Directed Graphs, Partial Orders – Hasse Diagrams, Equivalence Relations and Part**

## **Relations**

A relation R in a set A is a subset of the Cartesian product A x A.

- Definition: A relation R from A to B is a subset of A x B.
- Types of Relations:
  - **Equivalence Relation**: A relation that is reflexive, symmetric, and transitive.
  - **Partial Order**: A relation that is reflexive, antisymmetric, and transitive.
  - **Function**: A relation that is both injective and surjective.

## **Computer Recognition – Zero-One Matrices and Directed Graphs**

A zero-one matrix is a matrix where each element is either 0 or 1.

- Definition: A zero-one matrix is a matrix where each element is either 0 or 1.
- Representation of Relations using Zero-One Matrices:
  - **Equivalent Relations**: If two relations are equivalent, their corresponding zero-one matrices will be identical.
  - **Partial Orders**: The zero-one matrix of a partial order relation will have a specific structure, with 1s on the diagonal and 0s elsewhere.

## **Directed Graphs**

A directed graph is a graph where edges have direction.

- Definition: A directed graph is a graph where edges have a direction.
- Types of Directed Graphs:
  - **Digraph**: A directed graph where there is a directed edge between every pair of vertices.
  - **Simple Digraph**: A directed graph where there is no multiple edge between the same pair of vertices.

## **Partial Orders – Hasse Diagrams**

A Hasse diagram is a graphical representation of a partial order relation.

- Definition: A Hasse diagram is a graphical representation of a partial order relation, where elements are drawn below those they are less than.
- Properties of Hasse Diagrams:
  - **Reflexivity**: Elements are drawn on the same level as themselves.
  - **Antisymmetry**: Elements are not drawn on the same level if one is less than the other.
  - **Transitivity**: Elements are drawn on the same level if one is less than another, which is less than a third.

## **Equivalence Relations and Part**

An equivalence relation is a relation that is reflexive, symmetric, and transitive.

- Definition: An equivalence relation is a relation that is reflexive, symmetric, and transitive.
- Types of Equivalence Relations:
  - **Equivalence Classes**: Elements in an equivalence relation are grouped into classes based on their equivalence.

## **Key Concepts**

- **Reflexivity**: A relation is reflexive if every element is related to itself.
- **Symmetry**: A relation is symmetric if whenever element a is related to element b, then element b is also related to element a.
- **Transitivity**: A relation is transitive if whenever element a is related to element b and element b is related to element c, then element a is also related to element c.
- **Partial Order**: A relation is a partial order if it is reflexive, antisymmetric, and transitive.
- **Equivalence Relation**: A relation is an equivalence relation if it is reflexive, symmetric, and transitive.
