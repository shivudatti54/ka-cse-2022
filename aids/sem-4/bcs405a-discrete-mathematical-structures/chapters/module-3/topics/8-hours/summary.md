# **Discrete Mathematical Structures: Relations and Functions Revision Notes**

## **Hours 1-2: Relations**

- **Definition:** A binary relation R on a set A is a subset of A × A, containing ordered pairs (a, b) such that a ∈ A and b ∈ A.
- **Types of Relations:**
  - **Equivalence Relation:** Symmetric, Transitive, and Reflexive.
  - **Partial Order:** Reflexive, Antisymmetric, and Transitive.
  - **Order Relation:** Reflexive, Transitive, and Antisymmetric.
- **Important Formulas:**
  - Reflexive Law: (a, a) ∈ R for all a ∈ A
  - Symmetric Law: (a, b) ∈ R -> (b, a) ∈ R for all a, b ∈ A
  - Transitive Law: (a, b) ∈ R and (b, c) ∈ R -> (a, c) ∈ R for all a, b, c ∈ A
- **Theorem:** The reflexive, symmetric, and transitive laws are equivalent to the definition of an equivalence relation.

## **Hours 3-4: Functions**

- **Definition:** A function f from a set A to a set B is a relation from A to B such that for each a ∈ A, there exists a unique b ∈ B such that (a, b) ∈ f.
- **Types of Functions:**
  - **Injections (One-to-One):** Each element in the range corresponds to at most one element in the domain.
  - **Surjections (Onto):** Each element in the range corresponds to at least one element in the domain.
  - **Bijections (One-to-One and Onto):** Each element in the range corresponds to exactly one element in the domain.
- **Important Formulas:**
  - Injection: If (a, b) ∈ f and (c, d) ∈ f, then a = c
  - Surjection: For every b ∈ B, there exists an a ∈ A such that (a, b) ∈ f
  - Bijection: A function f from A to B is bijective if and only if f is both injective and surjective.

## **Hours 5-6: Properties and Theorems**

- **Properties of Functions:**
  - **Domain and Codomain:** The sets of all first components and second components of the ordered pairs in the function.
  - **Range:** The set of all second components of the ordered pairs in the function.
- **Theorem:** The sum of the measures of the angles in a triangle is always 180°.

## **Hours 7-8: Introduction to Graph Theory**

- **Definition:** A graph is a non-empty set of vertices connected by edges.
- **Types of Graphs:**
  - **Simple Graph:** No multiple edges between any two vertices.
  - **Directed Graph:** Edges have direction.
  - **Weighted Graph:** Edges have weights.
- **Important Concepts:**
  - **Adjacency Matrix:** A matrix representing the adjacency of vertices in a graph.
  - **Adjacency List:** A list representing the adjacency of vertices in a graph.
