# **Discrete Mathematical Structures: Relations and Functions**

## **Topic Overview**

In this topic, we will explore the fundamental concepts of relations and functions in discrete mathematics. Relations and Functions are crucial in mathematics and computer science, and understanding them is essential for problem-solving and critical thinking.

## **What is a Relation?**

A relation between two sets A and B is a subset of the Cartesian product A × B. In other words, a relation R is a set of ordered pairs (a, b) where a ∈ A and b ∈ B.

## **Types of Relations**

- **Reflexive Relation**: A relation R is reflexive if for every element a in A, (a, a) is in R.
- **Symmetric Relation**: A relation R is symmetric if for every pair (a, b) in R, (b, a) is also in R.
- **Transitive Relation**: A relation R is transitive if for every pair (a, b) in R and (b, c) in R, (a, c) is also in R.

## **Types of Relations (continued)**

- **Equivalence Relation**: A relation R is an equivalence relation if it is reflexive, symmetric, and transitive.
- **Partial Order**: A relation R is a partial order if it is reflexive, anti-symmetric, and transitive.

## **What is a Function?**

A function from a set A to a set B is a relation from A to B that satisfies the following properties:

- **Single-Valued**: For every element a in A, there is at most one element b in B such that (a, b) is in the function.
- **Well-Defined**: For every element b in B, there is at least one element a in A such that (a, b) is in the function.

## **Types of Functions**

- **Constant Function**: A function that maps every element to a fixed element in the codomain.
- **Identity Function**: A function that maps every element to itself.
- **One-to-One Function**: A function that maps distinct elements to distinct elements.
- **Onto Function**: A function that maps every element in the codomain to at least one element in the domain.

## **Examples of Functions**

- f: {1, 2, 3} → {a, b} defined by f(1) = a, f(2) = b, f(3) = a
- g: {1, 2, 3} → {x, y} defined by g(1) = x, g(2) = y, g(3) = x
- h: {a, b, c} → {1, 2} defined by h(a) = 1, h(b) = 2, h(c) = 1

## **Key Concepts**

- **Domain**: The set of input elements for a function.
- **Codomain**: The set of possible output elements for a function.
- **Range**: The set of actual output elements for a function.
- **Image**: The set of all possible output elements for a function.
