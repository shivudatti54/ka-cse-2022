## Function Composition and Inverse Functions

### Revision Notes

#### Definitions

- **Function Composition**: The process of applying one function to the output of another function.
- **Inverse Function**: A function that undoes the action of another function.

#### Key Formulas and Theorems

- **Function Composition Formula**: $(f \circ g)(x) = f(g(x))$
- **Inverse Function Formula**: $f^{-1}(y) = x \iff f(x) = y$
- **Inverse Function Theorem**: If $f$ is one-to-one, then the inverse function $f^{-1}$ exists.

#### Important Points

- **Composition of Functions**: The composition of two functions $f$ and $g$ is denoted as $(f \circ g)$ and is defined as $(f \circ g)(x) = f(g(x))$.
- **Inverse of a Function**: The inverse of a function $f$ is denoted as $f^{-1}$ and satisfies the property $f(f^{-1}(y)) = y$.
- **One-to-One Functions**: A function is one-to-one if it assigns distinct outputs to distinct inputs.
- **Bijective Functions**: A function is bijective if it is both one-to-one and onto.

#### Theorems

- **Inverse Function Theorem**: If $f$ is one-to-one, then the inverse function $f^{-1}$ exists and is unique.
- **Composition Property**: The composition of functions satisfies the property $(f \circ g) \circ f = f \circ (g \circ f)$.

#### Examples

- **Example 1**: Find the inverse of the function $f(x) = 2x + 1$.

* Solution: Let $y = 2x + 1$, then $x = \frac{y-1}{2}$. Switch $x$ and $y$ to get $y = \frac{x-1}{2}$. Therefore, $f^{-1}(x) = \frac{x-1}{2}$.

- **Example 2**: Find the composition of the functions $f(x) = x^2$ and $g(x) = 2x$.

* Solution: $(f \circ g)(x) = f(g(x)) = f(2x) = (2x)^2 = 4x^2$.
