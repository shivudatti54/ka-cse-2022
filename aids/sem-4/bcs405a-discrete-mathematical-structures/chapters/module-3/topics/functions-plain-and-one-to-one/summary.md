# Functions – Plain and One-to-One

### Definitions

- **Function**: A relation between a set of inputs (called the domain) and a set of possible outputs (called the range) with the property that each input is related to exactly one output.
- **Plain Function**: A function where every element in the domain maps to exactly one element in the range.
- **One-to-One Function**: A function where every element in the range is mapped to by at most one element in the domain.

### Important Formulas

- **Function Notation**: f(x)
- **Inverse Function**: f^(-1)(x)

### Theorems

- **One-to-One Theorem**: If a function f is one-to-one, then its inverse function f^(-1) is also one-to-one.
- **Constant Function**: If f(x) = c, where c is a constant, then the function is one-to-one.

### Key Points

- A function can be either plain or one-to-one.
- A plain function is one-to-one, but not necessarily the other way around.
- The vertical line test can be used to determine if a relation is a function.
- The horizontal line test can be used to determine if a function is one-to-one.
- The range of a function can be empty.
- A function can have any number of inputs (domain) and any number of outputs (range).
- Functions can be composed together.
- Inverse functions can be composed together.

### Important Rules

- If f(x1) = f(x2), then x1 = x2.
- If x1 ≠ x2, then f(x1) ≠ f(x2).
- If f(x) = f(y), then x = y.
