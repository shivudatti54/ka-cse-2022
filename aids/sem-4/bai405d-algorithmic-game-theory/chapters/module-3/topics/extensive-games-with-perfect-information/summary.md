# **Extensive Games with Perfect Information**

### Definition

- An extensive game is a two-player game with perfect information, where the game tree is finite and the players have complete knowledge of the game state at each node.
- Perfect information implies that players have complete knowledge of the game history and can make optimal decisions.

### Key Concepts

- **Game Tree**: A tree representation of all possible game states and their successors.
- **Node**: A node in the game tree represents a game state.
- **Player's Turn Node**: A node representing a player's turn.
- **Subgame**: A subgame is a smaller game tree that represents a subset of the original game tree.

### Important Formulas and Definitions

- **Value of a Node**: The maximum expected payoff that a player can achieve from a node.
- **Payoff Function**: A function that assigns a payoff to each node or subset of nodes.
- **Value of the Game**: The maximum expected payoff that a player can achieve from the entire game tree.
- **Minimax Theorem**: A theorem that states the optimal strategy for a player in a perfect information game is to minimize the maximum payoff of the opponent's perfect play.

### Theorems

- **Nash Equilibrium**: A concept that describes a stable state where no player can improve their expected payoff by unilaterally changing their strategy.
- **Prisoner's Dilemma**: A classic game that illustrates the conflict between individual and group rationality.

### Important Results

- **Minimax Formula**: A formula that provides a closed-form solution to the minimax problem.
- **Backward Induction**: A technique for solving extensive games by iteratively working backward from the terminal node to the root node.

### Important Theorems (continued)

- **Fermat's Criterion**: A theorem that states that a node is a Nash equilibrium if and only if the player's value at that node is greater than or equal to the value of any other node in the opponent's subgame.
- **Perrin's Theorem**: A theorem that states that a game with perfect information has a unique Nash equilibrium if and only if it satisfies a certain condition.

### Important Games

- **Prisoner's Dilemma**: A classic game that illustrates the conflict between individual and group rationality.
- **Zermelo's Dice Game**: A game that demonstrates the minimax theorem and the concept of Nash equilibrium.
