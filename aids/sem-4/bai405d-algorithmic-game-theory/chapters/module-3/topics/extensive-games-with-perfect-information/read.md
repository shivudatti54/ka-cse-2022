# **Extensive Games with Perfect Information**

## **Introduction**

In game theory, an extensive game is a game that involves multiple stages or periods, where players take turns making decisions. In this section, we will focus on extensive games with perfect information, where players have access to all the information about the game state at every stage.

## **Definition**

An extensive game is a game that involves multiple stages, where each stage represents a decision-making period for the players. The game can be represented as a tree or a flowchart, where each node represents a decision point, and the edges represent the possible actions taken by the players.

## **Perfect Information**

In an extensive game with perfect information, players have access to all the information about the game state at every stage. This means that players can see the current state of the game and make decisions based on that information.

## **Key Characteristics**

- **Multiple stages**: Extensive games involve multiple stages, where players take turns making decisions.
- **Perfect information**: Players have access to all the information about the game state at every stage.
- **Tree or flowchart representation**: Extensive games can be represented as a tree or a flowchart, where each node represents a decision point, and the edges represent the possible actions taken by the players.

## **Example**

Consider a simple game called "Rock, Paper, Scissors". This game involves two players, Alice and Bob, who take turns playing either rock, paper, or scissors. The game can be represented as a tree:

1.  Alice's first move:
    - Rock (R)
    - Paper (P)
    - Scissors (S)
2.  Bob's response:
    - Rock (R)
    - Paper (P)
    - Scissors (S)
3.  Alice's second move:

In this example, Alice and Bob take turns playing the game, and the game state is updated after each move. This is an extensive game with perfect information, where players have access to all the information about the game state at every stage.

## **Game Theory Approach**

In game theory, extensive games with perfect information can be analyzed using backward induction. Backward induction is a method for solving extensive games by working backwards from the final stage to the initial stage.

Here's how it works:

1.  Start at the final stage and work backwards.
2.  At each stage, consider the possible actions taken by the players and the payoffs associated with each action.
3.  Use the information from the previous stages to make decisions about the current stage.
4.  Repeat the process until you reach the initial stage.
