# Learning Objectives

After studying this topic, you should be able to:

1. Define and distinguish between matchings, vertex covers, and edge covers in graphs
2. Apply Hall's marriage theorem to determine if a bipartite graph has a perfect matching
3. Use Kőnig's theorem to relate maximum matchings and minimum vertex covers in bipartite graphs
4. Implement basic algorithms for finding maximum matchings in bipartite graphs
5. Solve assignment problems using the Hungarian algorithm
6. Analyze the approximation ratio of greedy algorithms for vertex cover problems
