# **Equilibrium Revision Notes**

**Definition:** A state in a game where no player can improve their payoff by unilaterally changing their strategy, assuming all other players keep their strategies unchanged.

**Key Concepts:**

- **Pure Strategy:** A player's chosen action or strategy.
- **Mixed Strategy:** A probability distribution over a set of pure strategies.
- **Nash Equilibrium:** A state where no player can improve their payoff by changing their strategy, assuming all other players keep their strategies unchanged.

**Important Formulas and Theorems:**

- **Nash Equilibrium Theorem:** A game has a Nash equilibrium if and only if it is a finite game with a finite number of players and a finite number of pure strategies.
- **Minimax Theorem:** The value of a game is equal to the minimum value of the maximum payoff that any player can achieve.
- **Pareto Optimality:** A state where no player can improve their payoff without making another player worse off.

**Types of Equilibrium:**

- **Subgame Perfect Nash Equilibrium:** A Nash equilibrium that holds in every subgame of the original game.
- **Mixed Strategy Nash Equilibrium:** A Nash equilibrium where players use mixed strategies.

**Important Theorems:**

- **Nash's Second Theorem:** If a game has a Nash equilibrium, then it is a game with a finite number of players and a finite number of pure strategies.
- **Aumann's Agreement Theorem:** If a game has a Nash equilibrium, then all players agree on the Nash equilibrium.

**Key Players:**

- **John Nash:** Developed the concept of Nash Equilibrium and Nash's Second Theorem.
- **Robert Aumann:** Developed Aumann's Agreement Theorem.
