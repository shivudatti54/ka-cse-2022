# Nash Equilibrium of an Infinitely Repeated Prisoner's Dilemma

### Definition

- The Prisoner's Dilemma is a game theory paradox where two players, Alice and Bob, must decide whether to cooperate or defect.
- An infinitely repeated Prisoner's Dilemma is a repeated game where the same game is played multiple times.

### Key Concepts

- **Nash Equilibrium**: A state where no player can improve their payoff by unilaterally changing their strategy, assuming the other player's strategy remains the same.
- **Infinitely Repeated Game**: The game is repeated infinitely, with each player having the option to defect or cooperate in each period.

### Formulas and Theorems

- **Nash Equilibrium**: A state where the following conditions are met:
  - No player can improve their payoff by changing their strategy, assuming the other player's strategy remains the same.
  - Both players choose the same strategy.
- **Payoff Matrix**:
  - Cooperate (C): both players receive 3 points
  - Defect (D): both players receive 1 point
- **Fermat's Dilemma**: The optimal strategy for both players when the game is repeated infinitely is to defect in each period.
- **Nash Equilibrium for Infinitely Repeated Prisoner's Dilemma**: The Nash equilibrium is to defect in every period, as this maximizes the expected payoff for each player.

### Important Results

- **Nash Equilibrium in Infinitely Repeated Prisoner's Dilemma**: The Nash equilibrium is not a Pareto optimum, as both players can improve their payoff by defecting in the first period.
- **Iterated Prisoner's Dilemma**: The repeated game can be played iteratively, allowing players to learn from each other's strategies and adapt their behavior.

### Key Points

- The Nash equilibrium of an infinitely repeated Prisoner's Dilemma is to defect in every period.
- The optimal strategy for both players is to defect, as this maximizes the expected payoff.
- The Nash equilibrium is not a Pareto optimum, as both players can improve their payoff by defecting in the first period.
