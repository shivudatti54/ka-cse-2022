# Repeated Games

### Definition

A repeated game is a game played multiple times over a finite number of periods, where the same players, payoffs, and strategies are repeated.

### Key Concepts

- **Strategic Subgame Perfectness (SGP)**: A strategy is SGP if it is a Nash equilibrium in every subgame.
- **Pareto Optimality**: A Nash equilibrium is Pareto optimal if there is no other Nash equilibrium that yields a higher payoff for at least one player.

### Important Formulas and Theorems

- **Nash Equilibrium (NE)**: A Nash equilibrium is a state where no player can improve their payoff by unilaterally changing their strategy.
- **Ponzi's Theorem**: If the game is zero-sum and the players have rational expectations, then the Nash equilibrium can be obtained by backward induction.
- **Ferber's Theorem**: If the game is strictly competitive and the players have rational expectations, then the Nash equilibrium can be obtained by backward induction.

### Repeated Games Theorems

- **Folk Theorem**: Every rational equilibrium can be achieved in a repeated game with a finite horizon.
- **Robbins' Theorem**: If the game is zero-sum and the players have rational expectations, then the Nash equilibrium can be obtained by backward induction.

### Important Definitions

- **Finite Horizon**: A repeated game with a finite number of periods.
- **Zero-Sum Game**: A game where the sum of the payoffs is always zero.
- **Rational Expectations**: Players expect other players to follow rational strategies.

### Key Points

- Repeated games can lead to more stable outcomes than single-shot games.
- Nash equilibrium can be obtained by backward induction in zero-sum games with rational expectations.
- Folk theorem states that every rational equilibrium can be achieved in a repeated game with a finite horizon.
- Robbins' theorem states that Nash equilibrium can be obtained by backward induction in zero-sum games with rational expectations.
