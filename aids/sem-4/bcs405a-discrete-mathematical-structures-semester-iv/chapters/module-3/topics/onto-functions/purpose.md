# Learning Objectives

After studying this topic, you should be able to:

1. Define and explain the concept of onto (surjective) functions with precise mathematical notation

2. Determine whether a given function is onto by verifying that every codomain element has a preimage

3. Construct proofs demonstrating that a function is onto by solving f(x) = y for arbitrary y in the codomain

4. Identify when a function is not onto by finding a counterexample element in the codomain

5. Distinguish between injective, surjective, and bijective functions and understand their relationships

6. Apply the inclusion-exclusion principle to count the number of onto functions between finite sets of given sizes

7. Connect the concept of onto functions to the pigeonhole principle and understand when surjectivity is impossible

8. Apply knowledge of onto functions in solving practical problems related to function composition and inverses
