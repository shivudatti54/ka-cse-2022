### Revision Notes: Optimization Techniques (8 hours)

#### Module: Applications of Vector Calculus

The following are key points, formulas, definitions, and theorems related to optimization techniques, condensed for quick revision:

### Definitions and Formulas

- **Optimization**: Finding the maximum or minimum value of a function subject to constraints
- **Gradient**: A vector of partial derivatives of a function with respect to each variable
- **Gradient Theorem**: The rate of change of a function in the direction of a vector is equal to the dot product of the gradient and the direction vector
- **Multivariable Optimization**: Finding the maximum or minimum value of a function of multiple variables subject to constraints

### Theorems

- **Weierstrass Theorem**: Every continuous function on a closed and bounded interval has a maximum and minimum value
- **Lagrange Multiplier Method**: Finding the maximum or minimum of a function subject to constraints using the gradient and Lagrange multipliers
- **Cauchy-Schwarz Inequality**: A useful inequality for bounding the norm of a vector

### Optimization Techniques

- **Unconstrained Optimization**: Finding the maximum or minimum value of a function without constraints
- **Constrained Optimization**: Finding the maximum or minimum value of a function subject to constraints using techniques such as Lagrange multipliers
- **Multiple Criteria Optimization**: Finding the optimal solution among multiple conflicting objectives

### Notation

- **∇f(x)**: Gradient of f(x)
- **f(x)**: Function to be optimized
- **g(x)**: Constraint functions
- **λ**: Lagrange multipliers
- **x**: Variables of the function

### Important Results

- **First Derivative Test**: Determining whether a function has a local maximum or minimum using the first derivative
- **Second Derivative Test**: Determining whether a function has a local maximum or minimum using the second derivative
- **Saddle Point**: A point where a function has a local maximum in one direction and a local minimum in another direction

This summary covers key points, definitions, formulas, theorems, and techniques related to optimization, providing a concise revision guide for the topic.
