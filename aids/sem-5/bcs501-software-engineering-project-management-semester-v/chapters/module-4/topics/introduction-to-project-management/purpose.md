# Learning Objectives

After studying this topic, you should be able to:

1. **Define** what constitutes a project and its characteristics
2. **Explain** why project management is important for software
3. **Describe** different contract types and their risk profiles
4. **Apply** the W5HH principle for project initiation
5. **Categorize** software projects by size, type, and objectives
6. **List** the four core management functions (POMC)
