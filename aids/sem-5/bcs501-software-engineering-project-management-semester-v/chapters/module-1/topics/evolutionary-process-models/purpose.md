# Learning Objectives

After studying this topic, you should be able to:

1. **Describe** the Waterfall Model and its phases
2. **Explain** incremental process models and when to use them
3. **Compare** evolutionary process models (Prototyping, Spiral)
4. **Identify** the four quadrants of the Spiral Model
5. **Evaluate** when to use each process model based on project characteristics
6. **Discuss** advantages and disadvantages of each model
