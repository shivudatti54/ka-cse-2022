### CLOUD COMPUTING Revision Notes: Chapter 4

#### 4.1: Cloud Computing Overview

- Definition: Cloud computing is a model of delivering computing services over the internet, on-demand and pay-as-you-go.
- Key characteristics:
  - Scalability
  - On-demand self-service
  - Broad network access
  - Resource pooling
  - Rapid elasticity

#### 4.2: Cloud Service Models

- Definition: A cloud service model refers to the way in which cloud computing services are delivered.
- Types:
  - Infrastructure as a Service (IaaS)
  - Platform as a Service (PaaS)
  - Software as a Service (SaaS)

#### 4.3: Cloud Deployment Models

- Definition: A cloud deployment model refers to the way in which cloud computing services are delivered.
- Types:
  - Public Cloud
  - Private Cloud
  - Community Cloud
  - Hybrid Cloud

#### 4.4: Cloud Computing Security

- Definition: Cloud computing security refers to the measures taken to protect cloud computing resources and data.
- Key security concerns:
  - Data encryption
  - Authentication and authorization
  - Access control
  - Compliance

#### 4.5: Benefits and Challenges of Cloud Computing

- Benefits:
  - Cost savings
  - Scalability
  - Increased agility
  - Improved reliability
- Challenges:
  - Security concerns
  - Dependence on internet connectivity
  - Complexity

### Important Formulas and Definitions

- Cloud Service Level Agreement (SLA): A contract between the cloud provider and customer that outlines the expected level of service.
- Cloud computing lifecycle: The stages of cloud computing, including planning, procurement, deployment, and monitoring.
- Cloud security framework: A set of guidelines and standards for ensuring the security of cloud computing resources and data.
