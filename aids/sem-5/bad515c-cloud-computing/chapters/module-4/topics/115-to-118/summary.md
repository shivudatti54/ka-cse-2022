# **Cloud Security Revision Notes (11.5-11.8)**

### Key Concepts

- **Cloud Security**: Protection of cloud-based resources and data from unauthorized access, use, disclosure, disruption, modification, or destruction.
- **Cloud Security Risks**: Data breaches, insider threats, malware, phishing, DDoS attacks, and infrastructure vulnerabilities.
- **Privacy Impact Assessment**: Evaluation of potential risks to personal data in cloud computing.

### Important Formulas and Definitions

- **Risk Assessment Framework**:
  - Risk = (Probability of Occurrence x Impact) / (Probability of Mitigation x Mitigation Effectiveness)
- **Cloud Security Controls**:
  - Access Control: Authorization, Authentication, and Accounting (AAA)
  - Data Protection: Encryption, Access Control, and Data Loss Prevention (DLP)
  - Incident Response: Detection, Containment, Eradication, and Recovery (DCER)
- **Cloud Security Theorems**:
  - The God's Dice Theorem: Assuming that all data is equally likely to be accessed, no data should be considered confidential.
  - The Burleson Theorem: Assuming that all data is equally likely to be incorrect, no data should be considered sensitive.
