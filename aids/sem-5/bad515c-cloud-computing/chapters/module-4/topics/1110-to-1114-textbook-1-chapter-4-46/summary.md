# Cloud Security

### 11.10 to 11.14: Chapter 4.6 - Cloud Security Concerns

**Key Points:**

- **Cloud Security Risks**:
  - Data breaches
  - Unauthorized access
  - Malware and viruses
  - Denial of Service (DoS) and Distributed Denial of Service (DDoS) attacks
- **Privacy Impact Assessment**:
  - Identify risks to personal data
  - Evaluate data sensitivity
  - Determine necessary security measures
- **Cloud Security Models**:
  - IaaS (Infrastructure as a Service)
  - PaaS (Platform as a Service)
  - SaaS (Software as a Service)

**Definitions:**

- **Cloud Security**: Protecting cloud computing resources from unauthorized access, use, disclosure, disruption, modification, or destruction.
- **Cloud Computing**: A model for delivering computing services over the internet.

**Theorem:**

- **Kerckhoffs' Principle**: A secure system is one that is designed so that even if it is compromised, the attacker will not be able to exploit the vulnerability without knowing the key.

**Important Formulas:**

- **Probabilistic Risk Assessment (PRA) Formula**: PRA = Σ (Risk x Probability) / Σ (Risk x Probability)
- **Risk Evaluation Formula**: Risk = (Probability of Loss x Value of Loss) / (Probability of Loss + Cost of Mitigation)

**Key Takeaways:**

- Cloud security is a top concern for cloud users
- Assessing privacy impact is crucial in cloud computing
- Understanding cloud security models and principles can help mitigate risks.
