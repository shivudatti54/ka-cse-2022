# Data Warehousing And the Web Revision Notes

=====================================================

### Overview

- Data warehousing is a key component of business intelligence (BI)
- Web-enabled data warehouses provide real-time access to data, supporting decision-making and business operations

### Web-Enabled Data Warehouse

- A web-enabled data warehouse is a data warehouse that provides access to data through the web
- Characterized by pre-built reports, data mining, and OLAP capabilities
- Key features:
  - Pre-built reports and dashboards
  - Data mining and predictive analytics
  - OLAP (Online Analytical Processing) capabilities
  - Web-based access and data extraction capabilities

### Web-Based Information Delivery

- Web-based information delivery refers to the delivery of information to users through the web
- Key features:
  - Easy access to information from anywhere
  - Real-time updates and refreshes
  - Personalization and filtering capabilities
  - Integration with other BI tools and applications

### OLAP And The Web

- OLAP (Online Analytical Processing) is a technology that enables analytical processing of data
- Web-based OLAP provides a web-based interface for OLAP capabilities
- Key features:
  - Drill-down and roll-up capabilities
  - Data summarization and aggregation
  - Data slicing and dicing
  - Personalization and filtering capabilities

### Building A Web-Enabled Data Warehouse

- Key steps:
  - Plan and design the data warehouse
  - Develop and implement the data warehouse
  - Integrate with other BI tools and applications
  - Test and deploy the web-enabled data warehouse
- Key considerations:
  - Data quality and integrity
  - Data security and access control
  - Scalability and performance

### Formulas, Definitions, Theorems

- **Data Warehousing Formula**: Data warehouse = (Data + Reports + OLAP) / (Users + Budget)
- **Data Quality Formula**: Data quality = (Accuracy + Completeness + Consistency) / (Precision + Recall)
- **Data Warehouse Definition**: A data warehouse is a repository of integrated, transformed, and aggregated data used to support business intelligence and decision-making.
- **OLAP Definition**: OLAP is a technology that enables analytical processing of data, providing a web-based interface for data analysis and reporting.

### Important Terms

- **Business Intelligence (BI)**: The process of using data and analytics to support business decision-making
- **Data Mining**: The process of discovering patterns and relationships in data
- **Data Warehousing**: The process of designing, building, and maintaining a repository of integrated, transformed, and aggregated data used to support business intelligence and decision-making.
