# Learning Objectives

After studying this topic on LP Problem Formulation, you should be able to:

1.  Identify situations where Linear Programming is an appropriate optimization tool.
2.  Correctly define decision variables for a given problem scenario.
3.  Construct a linear objective function to be either maximized (e.g., profit) or minimized (e.g., cost).
4.  Translate problem descriptions regarding limited resources and minimum requirements into linear inequality constraints.
5.  Recognize and apply the non-negativity restriction to decision variables.
6.  Synthesize all components into a complete and standard-form LP model.
