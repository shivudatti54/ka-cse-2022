# **Periodic Noise Reduction by Frequency Domain Filtering**

## **Introduction**

Noise is a significant issue in image processing, as it can affect the quality and accuracy of the reconstructed image. Periodic noise, in particular, can be challenging to remove, as it can be difficult to distinguish from the original signal. Frequency domain filtering is a powerful technique for reducing periodic noise in images. In this study material, we will explore the concept of periodic noise and the frequency domain filtering technique.

## **What is Noise?**

Noise is random fluctuations in the signal that can occur due to various factors, such as:

- **Thermal noise**: generated by random thermal motion of particles
- **Shot noise**: generated by random arrival of particles
- **Quantization noise**: generated by the finite number of quantization levels
- **Systematic noise**: generated by non-random factors, such as non-linear systems

## **Types of Noise in Images**

- **Gaussian noise**: a type of noise with a normal distribution
- **Salt and pepper noise**: a type of noise characterized by random, isolated pixels
- **Impulse noise**: a type of noise characterized by random, isolated pixels with a high amplitude

## **Periodic Noise**

Periodic noise is a type of noise that has a regular, periodic pattern. This type of noise can be difficult to remove, as it can be mistaken for the original signal. Periodic noise can be caused by various factors, such as:

- **Sample rate mismatch**: when the sampling rate of the image is not synchronized with the true data rate
- **Aliasing**: when the image data is sampled at a rate that is too low to capture the high-frequency components

## **Frequency Domain Filtering**

Frequency domain filtering is a technique for reducing noise in images by filtering the frequency components of the image. This technique is based on the fact that noise has a specific frequency signature, which can be removed by filtering the image in the frequency domain.

## **Key Concepts**

- **Discrete Fourier Transform (DFT)**: a mathematical operation that converts a discrete-time signal into its frequency domain representation
- **Fast Fourier Transform (FFT)**: an efficient algorithm for computing the DFT
- **Filtering**: the process of modifying the frequency components of an image to reduce noise

## **Frequency Domain Filtering Techniques**

- **Low-pass filtering**: filtering the high-frequency components of the image to reduce noise
- **Band-pass filtering**: filtering the specific frequency range of the image to retain the desired signal while removing noise
- **High-pass filtering**: filtering the low-frequency components of the image to retain the high-frequency details while removing noise

## **Example**

Suppose we have an image with periodic noise, which is represented by the following equation:

`x(t) = A * sin(2 * pi * f * t) + n(t)`

where `x(t)` is the original signal, `A` is the amplitude, `f` is the frequency, and `n(t)` is the noise signal.

To remove the periodic noise, we can apply a low-pass filter in the frequency domain. The transfer function of the low-pass filter is:

`H(f) = 1 / (1 + (f / f_c)^2)`

where `f_c` is the cutoff frequency.

By multiplying the original signal with the transfer function of the low-pass filter, we can remove the periodic noise:

`y(t) = x(t) * H(f)`

The resulting image `y(t)` will have reduced periodic noise, resulting in a cleaner and more accurate reconstruction.

## **Code Implementation**

Here is an example code implementation of frequency domain filtering using Python and the `scipy` library:

```python
import numpy as np
from scipy.fft import fft, ifft

# Generate a noisy signal
A = 1.0
f = 10.0
t = np.linspace(0, 1, 1000)
x = A * np.sin(2 * np.pi * f * t) + 0.1 * np.random.randn(1000)

# Apply a low-pass filter in the frequency domain
def low_pass_filter(f):
    return 1 / (1 + (f / 50.0)**2)

H = np.vectorize(low_pass_filter)
y = x * H(np.fft.fftfreq(len(x)))

# Inverse FFT
y = np.real(ifft(y))
```

This code generates a noisy signal, applies a low-pass filter in the frequency domain, and then applies an inverse FFT to obtain the filtered signal.

## **Conclusion**

Frequency domain filtering is a powerful technique for reducing periodic noise in images. By filtering the frequency components of the image, we can remove the noise and obtain a cleaner and more accurate reconstruction. This technique is widely used in image processing, computer vision, and signal processing applications.
