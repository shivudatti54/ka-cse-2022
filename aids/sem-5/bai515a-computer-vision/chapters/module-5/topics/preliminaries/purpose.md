# Learning Objectives
After studying this topic, you should be able to:
1. Define and distinguish key network components: nodes, edges, graphs, directed/undirected networks, and weighted/unweighted networks.
2. Calculate basic node-level metrics, including degree, in-degree, and out-degree, from a network diagram or adjacency matrix.
3. Compute fundamental network-level properties, such as size, number of edges, and density, for a given network.
4. Explain the concept of a path, identify the shortest path between nodes, and define the network diameter.
5. Construct an adjacency matrix from a simple network diagram and reconstruct a network from its adjacency matrix.
6. Provide real-world examples for different types of networks (e.g., directed, undirected, weighted) and justify the classification.
