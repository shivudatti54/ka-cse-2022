### Revision Notes: 25.1-25.2 Computer Networks

#### Introduction

- Definition: A computer network is a collection of interconnected devices that communicate with each other to share resources.
- Types of Computer Networks:
  - Local Area Network (LAN)
  - Metropolitan Area Network (MAN)
  - Wide Area Network (WAN)
  - Wireless Network (WLAN)

#### Network Topology

- Definition: The physical or logical arrangement of devices in a network.
- Types of Network Topology:
  - Bus Topology
  - Star Topology
  - Ring Topology
  - Mesh Topology

#### Network Devices

- Definition: Devices that connect and manage networks.
- Types of Network Devices:
  - Router
  - Switch
  - Hub
  - Bridge

#### Network Protocols

- Definition: Sets of rules that govern data communication between devices.
- Types of Network Protocols:
  - TCP/IP (Transmission Control Protocol/Internet Protocol)
  - DNS (Domain Name System)
  - HTTP (Hypertext Transfer Protocol)

#### Network Fundamentals

- ** OSI Model**:
  - 7 layers:
    1. Physical
    2. Data Link
    3. Network
    4. Transport
    5. Session
    6. Presentation
    7. Application
- **TCP Handshake**:
  - 3 phases:
    1. SYN (Synchronize)
    2. SYN-ACK (Synchronize-Acknowledgment)
    3. ACK (Acknowledgment)

#### Important Formulas and Definitions

- **Packet Switching**: A method of transmitting data in small packets between networks.
- **IP Address**: A unique address assigned to each device on a network.
- **MAC Address**: A unique address assigned to each network interface card (NIC).

#### Theorems and Concepts

- **Store-and-Forward**: A method of data transmission where data is stored in a buffer before being transmitted.
- **Error Detection and Correction**: Techniques used to detect and correct errors that occur during data transmission.
