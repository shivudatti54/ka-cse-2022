# **7.1 - 7.3: Computer Networks**

### 7.1: Basic Network Concepts

- **Definition:** A collection of interconnected devices that communicate with each other.
- **Key Components:**
  - Devices (computers, servers, etc.)
  - Communication media (cables, wireless, etc.)
  - Network interface cards (NICs)
  - Network protocols (TCP/IP, HTTP, etc.)
- **Network Topology:** Physical and logical arrangement of devices in a network.

### 7.2: Network Models

- **Definition:** A conceptual representation of a network.
- **Types of Network Models:**
  - **Physical Model:** Shows the physical components of a network.
  - **Logical Model:** Shows the communication between devices in a network.
  - **Network Topology Model:** Shows the arrangement of devices in a network.
- **Theorems:**
  - **First Theorem of Communication:** A message must have a sender, receiver, and channel to be communicated.
  - **Second Theorem of Communication:** A message must be encoded and decoded to be transmitted and received.

### 7.3: Network Protocols

- **Definition:** A set of rules and procedures for communication between devices.
- **Types of Network Protocols:**
  - **Transport Layer Protocols:** TCP/IP, UDP, etc.
  - **Application Layer Protocols:** HTTP, FTP, etc.
- **Important Formulas:**
  - **Bit Rate Formula:** Bit rate = Data rate x Time
  - **Bandwidth Formula:** Bandwidth = Data rate x Time
