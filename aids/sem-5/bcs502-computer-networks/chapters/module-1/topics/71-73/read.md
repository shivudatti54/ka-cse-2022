# **7.1-7.3: Network Models and Protocols**

### 7.1: Network Models

A network model is a conceptual representation of a network, describing its structure, components, and interactions. There are several network models, each with its strengths and weaknesses.

#### 7.1.1: Physical Network Model

The physical network model is a representation of the physical components of a network, such as cables, routers, and switches.

- **Definition:** A network model that describes the physical components and connections of a network.
- **Example:** A cable with a router and switch connected to it.

#### 7.1.2: Data Link Layer Model

The data link layer model is a representation of the data link layer of a network, which is responsible for framing, error detection, and error correction.

- **Definition:** A network model that describes the functions and protocols of the data link layer.
- **Example:** Ethernet, Wi-Fi, and PPP (Point-to-Point Protocol) are examples of data link layer protocols.

#### 7.1.3: Network Layer Model

The network layer model is a representation of the network layer of a network, which is responsible for routing and forwarding data between nodes.

- **Definition:** A network model that describes the functions and protocols of the network layer.
- **Example:** IP (Internet Protocol), ICMP (Internet Control Message Protocol), and IGMP (Internet Group Management Protocol) are examples of network layer protocols.

### 7.2: Network Protocols

A network protocol is a set of rules and procedures that govern the communication between devices on a network.

#### 7.2.1: OSI Model

The OSI (Open Systems Interconnection) model is a 7-layered model that describes the functions and protocols of a network.

- **Layers:**
  - Physical Layer (Layer 1)
  - Data Link Layer (Layer 2)
  - Network Layer (Layer 3)
  - Transport Layer (Layer 4)
  - Session Layer (Layer 5)
  - Presentation Layer (Layer 6)
  - Application Layer (Layer 7)
- **Example:** TCP/IP (Transmission Control Protocol/Internet Protocol) is an example of a protocol that follows the OSI model.

#### 7.2.2: TCP/IP Model

The TCP/IP model is a 4-layered model that describes the functions and protocols of a network.

- **Layers:**
  - Network Access Layer
  - Internet Layer
  - Transport Layer
  - Application Layer
- **Example:** TCP/IP is an example of a protocol that follows the TCP/IP model.

### 7.3: Network Topologies

A network topology is a representation of the physical or logical arrangement of devices in a network.

#### 7.3.1: Network Topology Types

There are several types of network topologies, including:

- **Bus Topology:** A single cable connects all devices.
- **Star Topology:** All devices are connected to a central device.
- **Ring Topology:** Devices are connected in a circular configuration.
- **Mesh Topology:** Each device is connected to every other device.

- **Definition:** A network topology is a representation of the physical or logical arrangement of devices in a network.
- **Example:** A network with a central switch connected to multiple devices is an example of a star topology.

By understanding the different network models, protocols, and topologies, you can design and implement more efficient and effective network systems.
