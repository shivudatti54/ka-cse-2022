# **2.1 - Data Representation in Computer Networks**

### Introduction

Data representation is a fundamental concept in computer networks, as it plays a crucial role in the transmission and reception of data over a network. In this section, we will explore the different ways data is represented in computer networks.

### Types of Data Representation

There are two primary types of data representation:

- **Binary Representation**: This is the most common method of data representation, where data is represented using only two digits: 0 and 1. Binary representation is used in computers to store and transmit data.
- **Text Representation**: This method of data representation uses a combination of binary digits and control characters to represent text data.

### Character Encoding Schemes

Character encoding schemes are used to convert text data into binary format. Some common character encoding schemes include:

- **ASCII (American Standard Code for Information Interchange)**: This is a 7-bit character encoding scheme that represents 128 unique characters, including letters, numbers, and special characters.
- **Unicode**: This is a 16-bit character encoding scheme that represents over 1 million unique characters from different languages and scripts.

### Key Concepts

- **Bit**: A binary digit that represents a single value, either 0 or 1.
- **Byte**: A group of 8 bits that represent a single character or number.
- **Character Encoding Scheme**: A method of representing text data in binary format.

# **2.2 - Network Topology**

### Introduction

Network topology refers to the physical and logical arrangement of devices in a network. In this section, we will explore the different types of network topologies.

### Types of Network Topology

There are several types of network topology, including:

- **Bus Topology**: In a bus topology, all devices are connected to a single cable or backbone.
- **Star Topology**: In a star topology, all devices are connected to a central device or hub.
- **Ring Topology**: In a ring topology, devices are connected in a circular configuration, and data travels in one direction around the ring.
- **Mesh Topology**: In a mesh topology, each device is connected to every other device, providing multiple paths for data to travel.

### Key Concepts

- **Network Topology**: The physical and logical arrangement of devices in a network.
- **Connectivity**: The method by which devices are connected to each other.
- **Scalability**: The ability of a network to accommodate growing numbers of devices.

# **2.3 - Network Models**

### Introduction

Network models are used to describe the structure and behavior of a network. In this section, we will explore the different types of network models.

### Types of Network Models

There are several types of network models, including:

- **Physical Model**: This model describes the physical structure of a network, including the types of cables and devices used.
- **Logical Model**: This model describes the logical structure of a network, including the devices and connections that make up the network.
- **Conceptual Model**: This model describes the overall function and behavior of a network.

### Key Concepts

- **Network Model**: A representation of the structure and behavior of a network.
- **Layered Model**: A model that divides a network into layers, each with its own specific function and responsibility.
- **Open Systems Interconnection (OSI) Model**: A layered model that describes the structure and behavior of a network.
