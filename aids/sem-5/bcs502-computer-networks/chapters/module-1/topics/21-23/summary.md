# **Revision Notes: 2.1-2.3 - Computer Networks**

### 2.1 Network Fundamentals

- **Definition:** A network of interconnected devices (computers, servers, printers, etc.) that can exchange data between them.
- **Characteristics:**
  - Interconnected devices
  - Ability to communicate with each other
  - Ability to exchange data
- **Types of Networks:**
  - Local Area Network (LAN)
  - Metropolitan Area Network (MAN)
  - Wide Area Network (WAN)
  - Wireless Network (WLAN)

### 2.2 Network Topologies

- **Definition:** The physical and logical arrangement of devices in a network.
- **Types of Topologies:**
  - **Bus Topology:**
    - Single cable with devices connected to it
    - Central hub for data transmission
  - **Star Topology:**
    - Central device connects devices
    - Data transmitted through the central device
  - **Ring Topology:**
    - Devices connected in a circular configuration
    - Data transmitted in one direction around the ring
  - **Mesh Topology:**
    - Each device connected to every other device
    - Data transmitted through multiple paths

### 2.3 Network Models

- **Definition:** A theoretical representation of a network.
- **Types of Models:**
  - **Physical Model:** Actual physical components of the network
  - **Logical Model:** Representation of the network without considering physical components
  - **Data Model:** Representation of the data being transmitted over the network

### Important Formulas and Definitions

- **Network Topology Factors:**
  - **Diameter:** Greatest distance between two devices
  - **Spanning Tree:** Minimum number of paths required to connect all devices
- **Network Diseases:**
  - **Black Hole:** Device that drops packets without forwarding
  - **Police Man in the Middle:** Third-party device intercepts and modifies data
- **Network Theorems:**
  - **Reilly-White Theorem:** Minimum number of paths required to connect all devices
  - **Lindgren Theorem:** Maximum number of devices that can be connected using a single path
