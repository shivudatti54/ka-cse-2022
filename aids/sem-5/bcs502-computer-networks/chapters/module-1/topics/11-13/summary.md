# **Computer Networks Revision Notes: 1.1 - 1.3**

### 1.1 Data Communication

- **Definition:** The process of exchanging data between devices using a common language and protocol.
- **Key Concepts:**
  - Data transmission
  - Data reception
  - Error detection and correction
  - Data compression and decompression
- **Important Formulas:**
  - Bit rate = Data rate / Time period
  - Bandwidth = Data rate / Time period
- **Theorem:** Shannon-Hartley Theorem (Capacity of a communication channel)

### 1.2 Network Types

- **Definition:** A network is a collection of interconnected devices that communicate with each other.
- **Types of Networks:**
  - Local Area Network (LAN)
  - Metropolitan Area Network (MAN)
  - Wide Area Network (WAN)
  - Wireless Network (WLAN)
- **Key Characteristics:**
  - LAN: Small, limited geographical area
  - MAN: Medium-sized, limited geographical area
  - WAN: Large, extended geographical area
  - WLAN: Wireless communication

### 1.3 Network Models

- **Definition:** A network model is a conceptual representation of a network.
- **Types of Network Models:**
  - Physical Model (Layer 1-2)
  - Data Link Model (Layer 3-4)
  - Network Model (Layer 5-7)
  - OSI Model (Layer 1-7)
  - TCP/IP Model (Layer 1-4)
- **Key Concepts:**
  - OSI Model: 7 layers
  - TCP/IP Model: 4 layers
  - Network Model: 7 layers

Note: This summary is a concise revision guide and covers the key points for 1.1 - 1.3. It is not a comprehensive study guide.
