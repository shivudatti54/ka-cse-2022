# **Computer Networks: Introduction to Physical Layer**

## **Introduction**

In this topic, we will explore the physical layer of computer networks, which is the foundation of data communication. The physical layer is responsible for transmitting data between devices over a physical medium. In this study material, we will cover the following topics:

- **Transmission Media**
- **Guided Media**
- **Unguided Media: Wireless**

## **Transmission Media**

Transmission media refers to the physical medium through which data is sent between devices. Examples of transmission media include:

- **Cable**
- **Wire**
- **Fiber Optic Cables**
- **Radio Waves**
- **Infrared Waves**

Transmission media can be classified into two categories:

- **Dedicated Media**: Dedicated media is a transmission medium where each device has a dedicated connection. Examples of dedicated media include coaxial cables and fiber optic cables.
- **Shared Media**: Shared media is a transmission medium where multiple devices share the same connection. Examples of shared media include wireless networks and Ethernet cables.

## **Guided Media**

Guided media is a type of transmission media where data is transmitted through a physical path or guide. Examples of guided media include:

- **Coaxial Cables**: Coaxial cables use a central wire surrounded by insulation and a braided shield to transmit data.
- **Twisted Pair Cables**: Twisted pair cables use two insulated wires twisted together to transmit data.
- **Fiber Optic Cables**: Fiber optic cables use light to transmit data through thin glass or plastic fibers.

Guided media has the following advantages:

- **Higher Bandwidth**: Guided media has a higher bandwidth than unguided media.
- **Higher Reliability**: Guided media has a higher reliability than unguided media.
- **Easier Installation**: Guided media is easier to install than unguided media.

## **Unguided Media: Wireless**

Unguided media, also known as wireless media, is a type of transmission media where data is transmitted without a physical path or guide. Examples of unguided media include:

- **Radio Waves**: Radio waves use radio frequencies to transmit data between devices.
- **Infrared Waves**: Infrared waves use infrared light to transmit data between devices.
- **Bluetooth**: Bluetooth is a type of wireless media that uses radio frequencies to transmit data between devices.

Unguided media has the following advantages:

- **Portability**: Unguided media is portable and can be used in a variety of locations.
- **Ease of Installation**: Unguided media is easy to install and requires no physical infrastructure.
- **Flexibility**: Unguided media offers flexibility in terms of network topology and device placement.

## **TCP/IP Protocol Suite**

The TCP/IP protocol suite is a set of protocols used to transmit data over the internet. The TCP/IP suite includes the following protocols:

- **TCP (Transmission Control Protocol)**: TCP is a connection-oriented protocol that ensures reliable data transfer.
- **IP (Internet Protocol)**: IP is a connectionless protocol that routes data between devices on the internet.
- **UDP (User Datagram Protocol)**: UDP is a connectionless protocol that provides best-effort delivery of data.

## **The OSI Model**

The OSI model is a 7-layered model that describes the functionality of a network. The OSI model includes the following layers:

- **Physical Layer**: The physical layer is responsible for transmitting data between devices over a physical medium.
- **Data Link Layer**: The data link layer is responsible for framing and error detection.
- **Network Layer**: The network layer is responsible for routing data between devices on different networks.
- **Transport Layer**: The transport layer is responsible for providing reliability and error detection.
- **Session Layer**: The session layer is responsible for establishing and terminating connections.
- **Presentation Layer**: The presentation layer is responsible for data formatting and compression.
- **Application Layer**: The application layer is responsible for providing services to end-users.

## **Key Concepts**

- **Transmission Media**: The physical medium through which data is sent between devices.
- **Guided Media**: A type of transmission media where data is transmitted through a physical path or guide.
- **Unguided Media: Wireless**: A type of transmission media where data is transmitted without a physical path or guide.
- **TCP/IP Protocol Suite**: A set of protocols used to transmit data over the internet.
- **The OSI Model**: A 7-layered model that describes the functionality of a network.
