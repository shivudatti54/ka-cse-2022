# Layering, TCP/IP Protocol Suite, OSI Model, and Physical Layer

### Layering

- **Layering**: Dividing data into manageable parts, each handled by a specific layer
- **7-Layer Model**: OSI (Open Systems Interconnection) model
- **TCP/IP Model**: 4-layer model (Network Access, Internet, Transport, Application)

### TCP/IP Protocol Suite

- **TCP/IP Model**:
  - Network Access Layer
  - Internet Layer
  - Transport Layer
  - Application Layer
- **Key Protocols**:
  - TCP (Transmission Control Protocol)
  - IP (Internet Protocol)
  - UDP (User Datagram Protocol)
  - DNS (Domain Name System)

### The OSI Model

- **Open Systems Interconnection Model**
- **7 Layers**:
  1.  Physical Layer (PHY)
  2.  Data Link Layer (DLL)
  3.  Network Layer (NL)
  4.  Transport Layer (TL)
  5.  Session Layer (SL)
  6.  Presentation Layer (PL)
  7.  Application Layer (AL)
- **Key Concepts**:
  - Layering
  - Framing
  - Error detection and correction

### Introduction to Physical Layer: Transmission Media

- **Physical Layer (PHY)**
- **Transmission Media**:
  - **Guided Media**:
    - Coaxial Cable
    - Twisted Pair
  - **Unguided Media**:
    - Wireless
    - Fiber Optic
- **Key Concepts**:
  - Bandwidth
  - Bit rate
  - Data rate

### Guided Media

- **Guided Media**:
  - **Coaxial Cable**:
    - Advantages: high bandwidth, low cost
    - Disadvantages: limited installation, heat buildup
  - **Twisted Pair**:
    - Advantages: low cost, easy installation
    - Disadvantages: limited bandwidth, prone to interference

### Unguided Media

- **Unguided Media**:
  - **Wireless**:
    - Advantages: flexibility, mobility
    - Disadvantages: security concerns, interference
  - **Fiber Optic**:
    - Advantages: high bandwidth, low latency
    - Disadvantages: high cost, installation challenges

## Important Formulas, Definitions, and Theorems

- **Bit Rate**: amount of data transmitted per unit time
- **Bandwidth**: range of frequencies available for data transmission
- **Data Rate**: rate at which data is transmitted
