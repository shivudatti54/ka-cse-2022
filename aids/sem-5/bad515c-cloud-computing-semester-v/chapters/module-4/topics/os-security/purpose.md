# Learning Objectives
After studying this topic, you should be able to:
1.  Identify the key components of a virtualized environment and their associated security risks.
2.  Differentiate between Type 1 and Type 2 hypervisors and explain their security implications.
3.  Apply best practices for hardening a hypervisor and guest virtual machines.
4.  Develop a strategy for managing VM image libraries to prevent deployment of vulnerable systems.
5.  Analyze the security risks introduced by cloud features like live migration and snapshots.
6.  Design a layered defense strategy for securing virtualized workloads in a cloud environment.
