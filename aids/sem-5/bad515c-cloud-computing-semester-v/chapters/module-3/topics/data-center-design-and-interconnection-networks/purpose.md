# Learning Objectives
After studying this topic, you should be able to:
1.  Explain the core design principles (scalability, reliability, efficiency) that guide modern cloud data center architecture.
2.  Identify and describe the key physical and logical components of a data center (e.g., server racks, ToR switches, PDUs, spine-leaf fabric).
3.  Compare and contrast traditional three-tier and modern spine-leaf network architectures, highlighting the advantages of the latter for cloud environments.
4.  Define Power Usage Effectiveness (PUE) and explain its significance in measuring data center efficiency.
5.  Articulate the concept of Software-Defined Networking (SDN) and its benefits for automating and managing data center networks.
6.  Describe the purpose and importance of interconnecting multiple data centers to form a global cloud platform.
