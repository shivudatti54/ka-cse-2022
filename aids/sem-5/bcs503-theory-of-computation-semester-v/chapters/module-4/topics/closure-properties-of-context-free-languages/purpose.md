# Learning Objectives
After studying this topic, you should be able to:
1. Define the union, intersection, concatenation, and Kleene star of context-free languages
2. Construct new grammars that recognize the result of these operations
3. Use the product construction to prove the intersection of a context-free language and a regular language is context-free
4. Identify the result of each operation using the table
5. Answer questions about closure properties of context-free languages
6. Apply the closure properties to solve problems in the theory of computation
