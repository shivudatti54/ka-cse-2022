# **Software and Software Engineering: Revision Notes**

## **The Nature of Software**

- **Definition of Software**: Intangible product of human creativity, executed through electronic means.
- **Characteristics of Software**:
  - Intangibility
  - Flexibility
  - Reusability
  - Complexity
  - Dynamic nature

## **The Unique Nature of WebApps**

- **Definition of WebApp**: Software application designed to run on web browsers or servers.
- **Characteristics of WebApps**:
  - Client-server architecture
  - Browser-based
  - Online accessibility
  - Dynamic content

## **Software Engineering**

- **Definition of Software Engineering**: Application of engineering principles and methods to develop software.
- **Key Concepts**:
  - Requirements gathering
  - Design
  - Implementation
  - Testing
  - Maintenance

## **The Software Process**

- **Waterfall Model**: Linear, sequential approach to software development.
- **V-Model**: Iterative, incremental approach to software development.
- **Agile Methodology**: Flexible, adaptive approach to software development.

## **Software Engineer**

- **Definition of Software Engineer**: Professional responsible for designing, developing, and maintaining software systems.
- **Key Skills**:
  - Programming languages
  - Software development methodologies
  - Project management
  - Problem-solving
  - Communication

## **Important Formulas and Definitions**

- **Software Development Life Cycle (SDLC)**: Model for planning, designing, developing, testing, and maintaining software systems.
- **COCOMO Model**: Method for estimating the effort required to develop software.
- **Kanban Methodology**: Visual system for managing work and improving workflows.

## **Theorems and Concepts**

- **Floyd's Cycle Detection Algorithm**: Algorithm for detecting cycles in linked lists.
- **Dijkstra's Shortest Path Algorithm**: Algorithm for finding the shortest path in a graph.
