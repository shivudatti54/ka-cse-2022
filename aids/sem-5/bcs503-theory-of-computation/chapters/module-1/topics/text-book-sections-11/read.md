### Introduction

**What is Theory of Computation?**

The Theory of Computation is a branch of computer science that deals with the study of the fundamental limits of computation. It examines the resources required to solve computational problems and the time and space complexities of algorithms.

### What is Computability?

Computability is the study of the set of problems that can be solved by a computer. A problem is said to be computable if there exists an algorithm (a set of instructions) that can solve it. In other words, a problem is computable if it can be solved by a computer with a finite number of steps.

**Key Concepts:**

- **Computer:** A machine that can store and process information.
- **Algorithm:** A set of instructions that solves a problem.
- **Computable problem:** A problem that can be solved by a computer.

### Types of Computability

There are two types of computability:

- **Deterministic Computability:** A problem is computable if there exists an algorithm that can solve it in a finite number of steps. Deterministic computability is based on the idea that the algorithm will always produce the same output for a given input.
- **Nondeterministic Computability:** A problem is computable if there exists an algorithm that can solve it in a finite number of steps, but the algorithm may produce different outputs for the same input.

### Examples of Computable Problems

- **Sorting a list of numbers:** A list of numbers can be sorted in ascending order using a simple algorithm.
- **Checking if a number is prime:** A number can be checked to see if it is prime using a simple algorithm.
- **Solving a linear equation:** A linear equation can be solved using a simple algorithm.

### Examples of Non-Computable Problems

- **The Halting Problem:** There is no algorithm that can determine whether a given program will run forever or eventually halt.
- **The Decision Problem:** There is no algorithm that can determine whether a given problem has a solution or not.
- **The Traveling Salesman Problem:** There is no known algorithm that can solve this problem exactly, although approximation algorithms exist.

### Conclusion

In conclusion, computability is a fundamental concept in the theory of computation. It deals with the study of the set of problems that can be solved by a computer. The theory of computation provides a framework for understanding the resources required to solve computational problems and the time and space complexities of algorithms.
