# The Pumping Lemma for Context-Free Languages

### Key Concepts

- **Context-Free Language (CFL):** A language that can be generated by a context-free grammar.
- **Pumping Lemma:** A lemma that proves no regular language can be deterministic.
- **Pumping Property:** Given a string `w` in a CFL, you can find a substring `x` and an integer `i > 0` such that `x` is a factor of `w`, `|x| ≥ 1`, and `x` appears at least `i` times in `w`.

### Definitions

- **Context-Free Grammar (CFG):** A 4-tuple `G = (V, Σ, P, S)` where:
  - `V` is a set of non-terminal symbols.
  - `Σ` is a set of terminal symbols.
  - `P` is a set of production rules.
  - `S` is the start symbol.
- **Regular Language:** A language that can be recognized by a deterministic finite automaton (DFA).

### Pumping Lemma Theorem

If a language `L` is context-free, then there exists an integer `p` such that:

`∀w ∈ L, ∀i ≥ 1, ∀x, |x| = p, w = x \* u \* v \* x \* w'`

where:

- `w` is a string in `L`.
- `i` is an integer greater than or equal to 1.
- `x` is a substring of `w`.
- `u` is a substring of `w` that does not contain `x`.
- `v` is a string that contains only `x`.
- `w'` is the remaining part of `w` after removing `x \* v`.

### Important Formulas

- Chomsky Normal Form (CNF):
  - A CFG is in CNF if:
    - The grammar has no epsilon productions.
    - Each production is of the form `A → aB` or `A → a`.
- Pumping Lemma for CFLs:
  - `L` is context-free if and only if `L` is regular.
  - The pumping lemma is also known as the "pumping lemma for regular languages".

### Key Theorems

- Pumping Lemma for CFLs:
  - If a language is context-free, then it is regular.
  - If a language is regular, then it is context-free.
