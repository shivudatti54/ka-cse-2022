# Learning Objectives

After studying this topic, you should be able to:

1. Explain the key features provided by the Transport Layer, including multiplexing, demultiplexing, and port-based communication.

2. Distinguish between connection-oriented (TCP) and connectionless (UDP) services and identify use cases for each protocol.

3. Analyze the flow control mechanism in TCP using the sliding window protocol and receive window advertisements.

4. Evaluate the error control mechanisms employed by TCP, including checksums, sequence numbers, acknowledgments, and retransmissions.

5. Compare and contrast the various congestion control algorithms: Slow Start, Congestion Avoidance, Fast Retransmit, and Fast Recovery.

6. Interpret TCP header fields and understand their roles in reliable data transfer.

7. Calculate practical values related to TCP performance, including throughput, window sizes, and timeout periods.

8. Design appropriate Transport Layer solutions based on application requirements, choosing between TCP and UDP.
