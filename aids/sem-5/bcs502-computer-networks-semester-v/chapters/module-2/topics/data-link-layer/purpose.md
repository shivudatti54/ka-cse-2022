# Learning Objectives

After studying this topic, you should be able to:

1.  Explain the primary function of the Data Link Layer.
2.  Describe the different DLC services, including framing, flow control, error control, and connectionless and connection-oriented services.
3.  Explain the difference between connectionless and connection-oriented services.
4.  Describe the purpose of the trailer in a data frame.
5.  Explain the purpose of flow control in the Data Link Layer.
6.  Describe the difference between HDLC and PPP.
