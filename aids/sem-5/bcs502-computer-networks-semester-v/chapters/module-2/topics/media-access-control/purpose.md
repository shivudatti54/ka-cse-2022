# Learning Objectives

After studying this topic, you should be able to:

1. Explain the primary function of a MAC protocol.
2. Describe the different types of MAC protocols and their functions.
3. Identify the advantages and disadvantages of each MAC protocol.
4. Explain the purpose of framing in MAC protocols.
5. Describe the operation of CSMA/CD and Token Ring protocols.
6. Compare and contrast random access and controlled access protocols.
