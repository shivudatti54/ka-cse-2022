# Learning Objectives

After studying this topic, you should be able to:

1. Describe the seven layers of the OSI model and their functions in network communication.

2. Explain the concept of data encapsulation and de-encapsulation in network communication.

3. Identify the protocol data units (PDUs) associated with each OSI layer.

4. Compare the functions of each layer and explain how they work together to enable communication.

5. Analyze network problems and determine which OSI layer is responsible for troubleshooting.

6. Differentiate between the OSI model and the TCP/IP protocol suite architecture.

7. Classify network devices (repeaters, hubs, switches, routers, gateways) according to the OSI layer at which they operate.

8. Evaluate the importance of layering in network design and understand the benefits of modular network architecture.
