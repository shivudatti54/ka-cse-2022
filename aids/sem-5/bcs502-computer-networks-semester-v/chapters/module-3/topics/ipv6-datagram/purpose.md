# Learning Objectives

After studying this topic, you should be able to:

1.  Understand the IPv6 address format and the different types of IPv6 addresses.
2.  Explain the IPv6 datagram format and the different fields in the header.
3.  Describe the different types of extension headers and their uses.
4.  Compare and contrast IPv4 and IPv6 datagrams.
5.  Identify the main differences between IPv4 and IPv6 datagrams.
6.  Explain the purpose of the Flow Label field in the IPv6 header.
