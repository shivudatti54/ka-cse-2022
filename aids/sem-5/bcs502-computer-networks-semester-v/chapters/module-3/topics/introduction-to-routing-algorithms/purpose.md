# Learning Objectives

After studying this topic, you should be able to:

1. Understand the basics of Distance Vector Routing (DVR), Link State Routing (LSR), and Path Vector Routing (PVR)
2. Compare and contrast the advantages and disadvantages of each routing algorithm
3. Explain how each routing algorithm works
4. Identify the main advantage and disadvantage of each routing algorithm
5. Solve problems involving routing algorithms
