# Types Of System Calls
**Why this topic matters:**
Understanding the different categories of system calls gives you a complete picture of the services an operating system provides to programs. Every action your program takes -- creating files, launching processes, communicating over the network -- maps to one of these six categories.

**Real-world applications:**
System programmers, DevOps engineers, and security professionals use these system calls daily. File management calls power every application that saves data, process control calls enable multi-process architectures, and communication calls underpin all networked applications from web servers to chat apps.

**Connection to other concepts:**
This topic directly extends the system calls topic by classifying them. It connects to process management (fork, exec, wait), file systems (open, read, write), inter-process communication (pipes, shared memory, sockets), and protection/security (chmod, chown).
