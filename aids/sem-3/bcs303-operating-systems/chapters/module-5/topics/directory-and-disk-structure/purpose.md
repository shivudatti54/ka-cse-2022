# Learning Objectives

After studying this topic, you should be able to:

1. Explain the purpose and importance of directory structures in file systems
2. Compare and contrast different directory structures (single-level, two-level, tree-structured, acyclic-graph, general graph)
3. Differentiate between absolute and relative pathnames with examples
4. Describe common directory operations and their purposes
5. Explain file protection mechanisms including ACLs and permission bits
6. Convert between different permission representations (symbolic, octal, binary)
