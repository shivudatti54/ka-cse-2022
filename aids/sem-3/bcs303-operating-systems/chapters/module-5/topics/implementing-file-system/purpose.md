# Learning Objectives

After studying this topic, you should be able to:

1.  Describe the hierarchical, tree-like structure of the Unix filesystem and explain the significance of the root directory (/).
2.  Identify and differentiate between the basic Unix file types (regular, directory, link, etc.) using the output of the `ls -l` command.
3.  Explain the purpose of the major standard directories as defined by the Filesystem Hierarchy Standard (FHS), such as /bin, /etc, /home, /var, and /usr.
4.  Define the role of environment variables $HOME and $PATH and demonstrate how to use the ~ (tilde) shortcut.
5.  Construct both absolute and relative pathnames to navigate the filesystem and reference files from any given location.
6.  Execute basic file and directory management commands, including `pwd`, `cd`, `ls`, `mkdir`, and `rmdir`.
