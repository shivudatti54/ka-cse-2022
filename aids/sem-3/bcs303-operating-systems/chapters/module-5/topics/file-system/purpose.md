# Learning Objectives

After studying this topic, you should be able to:

1.  Describe the core structures and layout of the FAT, NTFS, and ext4 file systems.
2.  Explain the process of file creation, reading, and deletion for each file system and the forensic artifacts each action leaves behind.
3.  Identify and interpret key metadata, including timestamps, from each file system and explain potential discrepancies.
4.  Compare and contrast the capabilities, limitations, and forensic implications of FAT, NTFS, and ext4.
5.  Apply knowledge of file system internals to common forensic scenarios, such as data recovery and timeline analysis.
6.  Recognize advanced features like NTFS Alternate Data Streams (ADS) and ext4 extents and explain their relevance to an investigation.
