# **Graphics using R Exploratory Data Analysis, Main Graphical Packages**

## **Revision Notes**

### Main Graphical Packages

- **ggplot2**: A powerful and flexible system for creating beautiful, publication-quality graphics.
- **base graphics**: Built-in R graphics system for creating simple plots.
- **plotly**: Interactive visualization library for creating web-based interactive plots.

### Pie Charts

- **Pie Chart Formula**: `pie(x, main, col, radius)`, where:
  - `x`: Values to be plotted.
  - `main`: Title of the pie chart.
  - `col`: Colors for each slice.
  - `radius`: Radius of the pie chart.
- **Pie Chart Definition**: A circular statistical graphic divided into slices to illustrate numerical proportion.

### Scatter Plots

- **Scatter Plot Formula**: `plot(x, y, main, xlab, ylab, col)`, where:
  - `x`: X-axis values.
  - `y`: Y-axis values.
  - `main`: Title of the plot.
  - `xlab`: Label for the X-axis.
  - `ylab`: Label for the Y-axis.
  - `col`: Color for the points.
- **Scatter Plot Definition**: A graphical representation of the relationship between two variables.

### Line Plots

- **Line Plot Formula**: `plot(x, y, main, xlab, ylab, col)`, where:
  - `x`: X-axis values.
  - `y`: Y-axis values.
  - `main`: Title of the plot.
  - `xlab`: Label for the X-axis.
  - `ylab`: Label for the Y-axis.
  - `col`: Color for the line.
- **Line Plot Definition**: A graphical representation of the relationship between two variables.

### Histograms

- **Histogram Formula**: `hist(x, main, xlab, col)`, where:
  - `x`: Values to be plotted.
  - `main`: Title of the histogram.
  - `xlab`: Label for the X-axis.
  - `col`: Color for the bars.
- **Histogram Definition**: A graphical representation of the distribution of a set of data.

### Box Plots

- **Box Plot Formula**: `boxplot(x, main, xlab)`, where:
  - `x`: Values to be plotted.
  - `main`: Title of the box plot.
  - `xlab`: Label for the X-axis.
- **Box Plot Definition**: A graphical representation of the distribution of a set of data.

### Bar Plots

- **Bar Plot Formula**: `bar(x, main, xlab, col)`, where:
  - `x`: Values to be plotted.
  - `main`: Title of the bar plot.
  - `xlab`: Label for the X-axis.
  - `col`: Color for the bars.
- **Bar Plot Definition**: A graphical representation of the comparison of categorical variables.

### Other Graphical Plots

- **Heatmap**: A graphical representation of matrix data.
- **Tree Map**: A graphical representation of hierarchical data.
- **Violin Plot**: A graphical representation of the distribution of a set of data.

### Important Formulas and Definitions

- **Mean**: μ = (Σx) / n, where μ is the mean and x is the value of each observation.
- **Variance**: σ^2 = Σ(x - μ)^2 / (n - 1), where σ^2 is the variance and x is the value of each observation.
- **Standard Deviation**: σ = √(σ^2), where σ is the standard deviation and σ^2 is the variance.

### Theorems

- **Central Limit Theorem (CLT)**: The distribution of the sample mean will be approximately normal, regardless of the underlying distribution of the data.
- **Law of Large Numbers (LLN)**: The average of the results obtained from a large number of trials will be close to the expected value and will converge to the expected value as the number of trials increases.
