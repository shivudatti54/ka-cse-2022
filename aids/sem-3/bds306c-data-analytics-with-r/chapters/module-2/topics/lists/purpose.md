# Learning Objectives

After studying this topic, you should be able to:

1. Define what a list is in R and explain how it differs from vectors and arrays

2. Create lists with both named and unnamed elements using the list() function

3. Access individual elements from lists using position indexing ([[]]) and name indexing ($)

4. Distinguish between single bracket ([]) and double bracket ([[]]) subsetting of lists

5. Modify lists by adding, updating, and removing elements

6. Apply functions to list elements using lapply() and sapply()

7. Convert lists to vectors using unlist() and understand the implications of type coercion

8. Work with nested lists to represent hierarchical data structures

9. Extract components from list objects returned by standard R functions like lm() and summary()
