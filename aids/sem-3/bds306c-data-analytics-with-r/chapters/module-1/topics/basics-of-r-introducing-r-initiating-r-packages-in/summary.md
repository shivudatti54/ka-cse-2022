# **Basics of R Revision Notes**

### Introduction to R

- R is a programming language for statistical computing and graphics.
- R is particularly well-suited for data analysis and visualization.
- R is available for free and open-source.

### Initiating R

- To start R, open a terminal or command prompt and type `R`.
- R can also be accessed through an IDE (Integrated Development Environment) such as R Studio.
- R can be installed on most operating systems, including Windows, macOS, and Linux.

### Packages in R

- Packages are libraries of R functions that can be easily installed and loaded into R.
- There are over 15,000 packages available on the Comprehensive R Archive Network (CRAN).
- To install a package, use the `install.packages()` function.

### Environments and Functions

- In R, a function is a block of code that can be executed multiple times with different inputs.
- A function can take arguments, which are inputs passed to the function.
- R has several types of functions, including:
  - Built-in functions: functions that are part of the R language.
  - User-defined functions: functions created by the user.
  - Package functions: functions from packages.

### Flow Controls

- Flow control statements determine the order in which code is executed.
- R has several flow control statements, including:
  - Conditional statements (if, ifelse, switch).
  - Loops (for, while, repeat).
  - Jump statements (break, next, return).

### Loops

- Loops execute a block of code repeatedly for a specified number of times.
- R has several types of loops, including:
  - For loops: iterate over a sequence of values.
  - While loops: iterate while a condition is true.
  - Repeat loops: iterate indefinitely until a condition is false.

### Basic Data Types in R

- R has several built-in data types, including:
  - Numeric: whole numbers and decimal numbers.
  - Character: strings of text.
  - Logical: true or false values.
  - Complex: complex numbers.
  - List: ordered collections of values.

### Vectors

- Vectors are ordered collections of values in R.
- Vectors can be created using the `c()` function.
- Vectors can be manipulated using various functions, including `mean()`, `median()`, and `sum()`.

## **Key Concepts**

- Variables: store and manipulate data in R.
- Data types: determine the type of data stored in a variable.
- Functions: blocks of code that can be executed multiple times with different inputs.
- Flow control: determines the order in which code is executed.
- Loops: execute a block of code repeatedly for a specified number of times.
- Vectors: ordered collections of values in R.
