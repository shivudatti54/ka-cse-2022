# Learning Objectives

After studying this topic, you should be able to:

1. Describe the purpose, origin, and voluntary nature of the NIST Cybersecurity Framework.
2. List, define, and sequence the five core Functions of the NIST CSF (Identify, Protect, Detect, Respond, Recover).
3. Differentiate between the four Implementation Tiers (Partial, Risk-Informed, Repeatable, Adaptive) and explain their purpose.
4. Explain the concept of a Framework Profile and how Current and Target Profiles are used for gap analysis.
5. Map common cybersecurity activities (e.g., risk assessment, access control, incident response) to the appropriate NIST CSF Function and Category.
6. Outline the seven-step process recommended for implementing the NIST CSF within an organization.
