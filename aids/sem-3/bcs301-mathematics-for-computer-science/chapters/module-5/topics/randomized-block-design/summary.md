# Randomized Block Design

### Definition

A randomized block design is a type of experimental design used to reduce the effects of extraneous variables.

### Key Points

- **Definition**: A randomized block design is a type of experimental design where all treatments are assigned to blocks, and within each block, the treatments are assigned randomly.
- **Purpose**: To reduce the effects of extraneous variables (e.g. differences in observations due to location, time, or equipment).
- **Assumptions**:

* Independence of observations
* Homogeneity of variance within blocks
* Randomization within blocks

- **Types of Randomized Block Designs**:

* Completely Randomized Block Design (CRBD)
* Balanced Incomplete Block Design (BIBD)
* Unbalanced Block Design

### Important Formulas

- **Number of Replicates (n)**: The number of times each treatment is replicated.
- **Number of Blocks (b)**: The number of blocks in the design.
- **Number of Treatments (t)**: The number of treatments in the design.
- **Replication Factor (r)**: The number of replicates per treatment.

### Theorems

- **Yates' Theorem**: Used to calculate the number of replicates required for a randomized block design.
- **Blom's Theorem**: Used to calculate the number of blocks required for a randomized block design.

### Important Concepts

- **Blocking Variable**: A variable that affects the observations and needs to be controlled.
- **Treatments**: The variable being tested.
- **Replicates**: The number of times each treatment is measured.

### Formula for Yates' Theorem

- **n = (t \* r) / b**: The number of replicates required for a randomized block design.
