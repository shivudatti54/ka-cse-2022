# Learning Objectives

After studying this topic, you should be able to:

1. Differentiate between local shell variables and environment variables
2. Create, modify, and delete both types of variables using appropriate syntax
3. Use the export command to make variables available to child processes
4. Understand and utilize special shell variables ($?, $#, $@, etc.)
5. Apply variable manipulation techniques for string operations and default values
6. Explain how environment variables are accessed in C programs using getenv() and environ
