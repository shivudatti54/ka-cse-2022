# Mathematical Expectation

### Definition

- The mathematical expectation, also known as the expected value, is a measure of the central tendency of a probability distribution.
- It represents the long-run average value of a random variable.

### Key Formulas

- **Mean** (or **Expected Value**): E(X) = ∑xP(x)
- **Variance**: σ^2 = E(X^2) - (E(X))^2
- **Standard Deviation**: σ = √(σ^2) = √(E(X^2) - (E(X))^2)

### Important Theorems

- **Linearity of Expectation**: E(aX + b) = aE(X) + b for constants a and b
- **Law of Total Expectation**: E(X) = E(E(X|Y)) for random variables X and Y

### Other Key Points

- **Expected Value of a Constant**: E(c) = c for any constant c
- **Expected Value of a Sum**: E(X1 + X2 + ... + Xn) = E(X1) + E(X2) + ... + E(Xn)
- **Expected Value of a Product**: E(XY) = E(X)E(Y) for independent random variables X and Y

### Formula Derivation

- The mean is calculated by summing the product of each possible value of X and its probability, weighted by its occurrence.

### Key Concepts

- **Random variable**: A variable whose possible values are determined by chance events.
- **Probability distribution**: A function that assigns a probability to each possible value of a random variable.
- **Independent events**: Events whose occurrence or non-occurrence does not affect the probability of other events.
