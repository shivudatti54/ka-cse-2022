# Mathematical Expectation

=========================

## Introduction

---

Mathematical expectation, also known as expected value, is a fundamental concept in probability theory and statistics. It represents the long-run average value of a random variable or a random process. In this section, we will explore the concept of mathematical expectation, its definition, and its applications in probability distributions.

## Definition

---

Mathematical expectation is defined as the sum of the products of each possible outcome and its probability. It is denoted by the symbol E(X) and is calculated as follows:

E(X) = ∑xP(X=x)

where:

- x is the possible outcome
- P(X=x) is the probability of the outcome x

## Explanation

---

Mathematical expectation represents the average value of a random variable. It takes into account the probability of each outcome and its magnitude. The higher the expected value, the more likely the variable will take on large values.

For example, consider a fair coin toss. The possible outcomes are {heads, tails} with equal probabilities. The expected value of the random variable X (which represents the outcome of the coin toss) is:

E(X) = (1/2) × 1 + (1/2) × 0 = 1/2

This means that if we were to repeat the coin toss many times, the average outcome would be 1/2.

## Types of Mathematical Expectation

---

There are two types of mathematical expectation:

- **Discrete Expectation**: This type of expectation is used for discrete random variables, where the random variable can take on only specific values.
- **Continuous Expectation**: This type of expectation is used for continuous random variables, where the random variable can take on any value within a given range.

## Properties of Mathematical Expectation

---

- **Linearity**: The expected value of a linear combination of random variables is equal to the linear combination of their expected values.
- **Translation**: The expected value of a random variable shifted by a constant is equal to the expected value of the original random variable plus the constant.
- **Independence**: The expected value of the sum of independent random variables is equal to the sum of their expected values.

## Examples

---

- **Example 1**: Consider a random variable X that represents the number of heads in a fair coin toss. The possible outcomes are {0, 1, 2} with probabilities P(X=0) = 1/4, P(X=1) = 1/2, and P(X=2) = 1/4. The expected value of X is:
  E(X) = 0 × 1/4 + 1 × 1/2 + 2 × 1/4 = 3/4
- **Example 2**: Consider a random variable X that represents the value of a fair dice roll. The possible outcomes are {1, 2, 3, 4, 5, 6} with equal probabilities. The expected value of X is:
  E(X) = (1 + 2 + 3 + 4 + 5 + 6) / 6 = 21/6 = 3.5

## Key Concepts

---

- **Random variable**: A variable that can take on different values based on chance.
- **Probability distribution**: A function that describes the probability of each possible outcome of a random variable.
- **Expected value**: The long-run average value of a random variable or a random process.
