# Memory Location And Addresses
Memory addressing provides the fundamental mechanism for processors to uniquely identify and access storage locations. For an n-bit address bus, the system can address 2^n distinct locations. In byte-addressable architectures (standard), each address corresponds to one byte; in word-addressable systems, capacity increases by a factor equal to word size. Endianness determines byte ordering: big-endian stores MSB at lowest address (network protocols, IBM mainframes), while little-endian stores LSB at lowest address (Intel x86). Advanced concepts include physical versus logical addresses (translated by MMU), address binding methods, and memory alignment requirements. Understanding these principles is critical for system programming, cache design, and virtual memory implementation.
