# Learning Objectives
After studying this topic, you should be able to:
1.  Define Direct Memory Access (DMA) and explain its primary purpose in a computer system.
2.  Identify the key components of a DMA Controller (DMAC) and describe the function of its internal registers.
3.  Sequence the steps involved in a typical DMA transfer, from initiation to termination.
4.  Compare and contrast the different modes of DMA operation (Burst, Cycle Stealing, Transparent).
5.  Differentiate between DMA and Interrupt-Driven I/O, highlighting the scenarios where each is most appropriate.
6.  Analyze the advantages and disadvantages of using DMA, including real-world applications and potential issues like cache coherency.
