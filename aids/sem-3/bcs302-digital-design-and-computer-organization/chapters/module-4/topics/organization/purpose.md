# Learning Objectives
After studying this topic, you should be able to:
1.  Describe the four main components of the Von Neumann architecture and their interconnections via the system bus.
2.  Identify the core internal components of a CPU (ALU, CU, Register File) and explain their distinct roles in program execution.
3.  Differentiate between the functions of key CPU registers, including the Program Counter (PC), Instruction Register (IR), Memory Address Register (MAR), and Memory Data Register (MDR).
4.  Trace the sequence of events (micro-operations) that occur during the Fetch, Decode, and Execute phases of the instruction cycle for a given simple instruction.
5.  Explain how the width of the data bus and address bus impacts system performance and memory capacity.
6.  Contrast the unified memory approach of Von Neumann architecture with the separate memory approach of Harvard architecture.
