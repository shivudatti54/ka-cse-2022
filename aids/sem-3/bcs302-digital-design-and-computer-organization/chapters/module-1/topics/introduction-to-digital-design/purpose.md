# Learning Objectives
After studying this topic, you should be able to:
1.  Differentiate between the physical design and logical design of an IoT system.
2.  Identify and describe the four core functional blocks of a generic IoT architecture.
3.  Explain the role and importance of communication protocols, specifically describing the publish-subscribe model of MQTT.
4.  Classify a described IoT system into one of the six IoT levels based on its configuration and complexity.
5.  Justify the selection of a specific communication protocol (e.g., Wi-Fi, LoRaWAN, Bluetooth) for a given IoT application scenario.
6.  Define the purpose of information modeling and standards like YANG in achieving IoT interoperability.
