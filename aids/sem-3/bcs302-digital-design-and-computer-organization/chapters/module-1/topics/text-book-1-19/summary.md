# Text Book 1: 1.9

### Digital Design and Computer Organization

### Topic 1.9

#### Key Points

- **Binary Number System:**
  - Definition: Base-2 number system using only 0s and 1s
  - Number representation: 2^0, 2^1, 2^2, ...

- **Data Types:**
  - Definition: Categories of data (integers, floating-point numbers, characters, etc.)
  - Examples:
    - Integer: 000, 0000, 00000
    - Floating-point: 000.000000, 000.0000000
    - Character: A, @, !

- **Bit and Bit Sequence:**
  - Definition: Group of 0s and 1s used to represent binary data
  - Bit sequence: sequence of bits (e.g., 10101010)

- **Binary to Decimal Conversion:**
  - Formula: (b*n \* 2^n) + (b*(n-1) \* 2^(n-1)) + ... + (b_0 \* 2^0)

- **Decimal to Binary Conversion:**
  - Formula: divide decimal number by 2 and keep track of remainders

#### Important Formulas and Theorems

- **Binary Arithmetic:**
  - Addition: binary AND of the sum and carry
  - Subtraction: binary OR of the difference and borrow
  - Multiplication: binary shift of the product and carry
  - Division: binary shift of the quotient and remainder

- **Boolean Algebra:**
  - Definition: algebraic system used to represent digital logic
  - Laws:
    - Commutative Law: a + b = b + a
    - Associative Law: (a + b) + c = a + (b + c)
    - Distributive Law: a(b + c) = ab + ac

#### Important Definitions

- **Boolean Variable:** variable that can take on only two values (0 or 1)
- **Logic Gate:** digital circuit that implements a Boolean operation (AND, OR, NOT)
- **Bitwise Operation:** operation performed on individual bits of a binary number
