# Why Study Multiplexers?

## Fundamental Data Selection Circuit

Multiplexers are one of the most versatile and commonly used combinational circuits. They appear in virtually every digital system from simple circuits to complex processors.

## Exam Relevance

| Aspect         | Details                                                  |
| -------------- | -------------------------------------------------------- |
| Frequency      | Asked in 90%+ of Digital Design exams                    |
| Marks          | 8-15 marks typically                                     |
| Question Types | Design, function implementation, cascading, Verilog/VHDL |

## Common Exam Questions

1. Design 4:1 MUX using basic gates
2. Implement Boolean function using MUX
3. Build 8:1 MUX from 4:1 MUX
4. Design 4:1 MUX using 2:1 MUX
5. Write Verilog/VHDL code for MUX
6. Implement n-variable function using (n-1) select MUX

## Real-World Applications

1. **CPU Data Path** - Register selection
2. **Communication Systems** - Channel selection
3. **Memory Systems** - Bank selection
4. **Display Systems** - Pixel data routing
5. **Network Routers** - Packet routing

## Connection to Other Topics

After mastering MUX, you'll understand:

- **Demultiplexers** - Reverse operation
- **Decoders** - MUX internally uses decoder
- **ALU Design** - Operation selection
- **Memory Design** - Address decoding
- **FPGA Architecture** - LUT implementation

## Learning Outcomes

After studying this topic, you should be able to:

1. Design any size MUX using gates
2. Implement Boolean functions using MUX
3. Build larger MUX from smaller ones
4. Write Verilog/VHDL for MUX
5. Calculate gate count and delay
6. Apply MUX in practical designs
