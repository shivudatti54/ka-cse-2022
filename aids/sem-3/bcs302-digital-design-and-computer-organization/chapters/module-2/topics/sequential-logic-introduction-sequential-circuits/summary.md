# Sequential Logic: Introduction, Sequential Circuits, Storage Elements: Latches, Flip-Flops

### Introduction

- Sequential logic is a branch of digital logic that deals with systems that have memory and can change state over time.
- It's used in digital circuits, computers, and other electronic devices to control and store information.
- Important concepts:
  - Sequential circuits
  - Storage elements
  - Latches
  - Flip-flops

### Sequential Circuits

- Sequential circuits have memory elements that can change state over time.
- They are used to implement functions that depend on past and present inputs.
- Characteristics:
  - Memory elements
  - Feedback loops
  - Sequential outputs

### Storage Elements

- Storage elements are used to store information in sequential circuits.
- There are two main types:
  - Latches
  - Flip-flops

### Latches

- A latch is a basic storage element that can store a single bit of information.
- It has two inputs and one output.
- Characteristics:
  - Synchronous or asynchronous
  - Unijunction or dual-junction
  - D-latch and JK-latch

### Flip-Flops

- A flip-flop is a more advanced storage element that can store multiple bits of information.
- It has two inputs and one output.
- Characteristics:
  - Master slave configuration
  - T-flip-flop and D-flip-flop
  - Clocked and unclocked

### Important Formulas and Definitions

- **Tautology**: A statement that is always true.
- **Contradiction**: A statement that is always false.
- **Conjunctive Normal Form (CNF)**: A logical statement that can be represented as a conjunction of clauses.
- **Karnaugh map**: A graphical tool used to simplify Boolean expressions.
- **Truth table**: A table used to determine the output of a Boolean expression.

### Important Theorems

- **De Morgan's laws**:
  - The complement of the conjunction of two statements is the disjunction of their complements.
  - The complement of the disjunction of two statements is the conjunction of their complements.
- **Karnaugh map simplification**:
  - A set of adjacent cells with the same value can be simplified to a single cell.
