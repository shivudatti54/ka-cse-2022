# Learning Objectives
After studying this topic, you should be able to:
1. Define what a Security Operations Center (SOC) is and state its core objectives.
2. Compare and contrast different SOC models (In-house, MSSP, Hybrid) and identify scenarios where each is appropriate.
3. Describe the tiered structure of a typical SOC and explain the distinct responsibilities of each tier (1, 2, 3).
4. Identify key roles within a SOC (e.g., Analyst, Incident Responder, Threat Hunter, Manager) and summarize their functions.
5. Explain how the SOC interacts with other internal and external teams (IT, Legal, MSSPs).
6. List the essential tooling that enables SOC operations (SIEM, EDR, SOAR).
