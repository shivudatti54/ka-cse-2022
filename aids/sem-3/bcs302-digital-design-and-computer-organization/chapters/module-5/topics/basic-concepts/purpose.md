# Learning Objectives

After studying this topic, you should be able to:

1. Explain the structure and components of the Central Processing Unit (CPU), including the Control Unit, ALU, and various registers.

2. Describe the three types of buses (data, address, control) and their roles in computer communication.

3. Trace the complete instruction cycle and explain each phase: Fetch, Decode, Execute, and Store.

4. Use Register Transfer Notation (RTL) to express micro-operations and data transfers within the processor.

5. Explain how the ALU performs arithmetic and logical operations and describe the status flags it generates.

6. Calculate memory address space based on address bus width and word size.

7. Analyze the data flow between CPU components during instruction execution.

8. Apply these fundamental concepts to understand more advanced topics like pipelining and cache memory.
