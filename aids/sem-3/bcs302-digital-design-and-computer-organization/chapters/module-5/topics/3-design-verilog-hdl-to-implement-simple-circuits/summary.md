# 3 Design Verilog HDL to implement simple circuits using structural

## Module: 8 Hr, Digital Design and Computer Organization

### Key Points

- **What is Structural HDL?**
  - HDL (Hardware Description Language) used for designing digital circuits
  - Focuses on the structural description of a circuit
- **Types of Structural HDL**
  - **Combinational**: Describes a circuit as a combination of AND, OR, and NOT gates
  - **Sequential**: Describes a circuit with memory elements (e.g., flip-flops)
- **Verilog Syntax**
  - **Module**: Defines a circuit using a unique name
  - **Wire**: Represents a signal connection between modules
  - **Instance**: Instantiates a module in the design
  - **Assign**: Assigns a value to a signal
- **Design Flow**
  - **Top-Level Module**: The main module that contains all other modules
  - **Submodules**: Modules that are instantiated in the top-level module
  - **Netlist**: The final representation of the design

### Important Formulas, Definitions, and Theorems

- **Karnaugh Map (K-Map)**: A method for simplifying Boolean expressions
- **Truth Table**: A table that shows all possible input combinations and their corresponding output values
- **Hazard Detection**: Identifies hazards in a digital circuit to ensure correct behavior
- **Simplification Rules**: Rules for simplifying Boolean expressions (e.g., De Morgan's laws)

### Revision Tips

- Understand the structural description of a circuit
- Familiarize yourself with Verilog syntax and design flow
- Practice designing simple circuits using Verilog HDL
