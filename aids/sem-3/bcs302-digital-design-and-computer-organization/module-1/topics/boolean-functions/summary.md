# Boolean Functions
A Boolean function f: {0,1}ⁿ → {0,1} maps binary inputs to binary output. Key representations include truth tables, Boolean expressions (SOP/POS), and canonical forms (minterms/maxterms). Canonical SOP expresses F as Σ of minterms where output=1; Canonical POS expresses F as Π of maxterms where output=0. Simplification using Boolean algebra theorems (absorption, consensus, De Morgan's) and Karnaugh maps reduces gate count. The duality principle enables deriving dual identities. Functional completeness of {AND, OR, NOT} allows implementing any Boolean function. Don't care conditions enable further optimization. These concepts are essential for combinational circuit design in digital systems.
