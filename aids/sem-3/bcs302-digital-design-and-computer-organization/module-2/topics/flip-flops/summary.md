# Module Summary: Flip-Flops

## Key Learning Outcomes
Upon completion of this module, students will be able to:
- Explain the fundamental difference between combinational and sequential logic circuits
- Describe the operation of SR, JK, D, and T flip-flops using truth tables
- Derive and apply characteristic equations for each flip-flop type
- Utilize excitation tables for flip-flop implementation in sequential circuits
- Analyze timing diagrams showing edge-triggered behavior
- Understand critical timing parameters including setup time, hold time, and propagation delay

## Core Concepts
Flip-flops are 1-bit memory elements with two stable states. They are clock-synchronized devices that update their output only at specific clock transitions (edges). The four fundamental types—SR, JK, D, and T—differ in their input configurations and behavioral characteristics. The D flip-flop is most commonly used in practice due to its simplicity and absence of invalid states, while JK and T flip-flops provide toggle functionality essential for counter design. Understanding these components is essential for digital system design and computer architecture.
