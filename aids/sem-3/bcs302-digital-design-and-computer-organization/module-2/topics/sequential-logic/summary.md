# Sequential Logic
### Topic Summary: Sequential Logic

**Core Concept:** Sequential logic circuits have outputs dependent on both current inputs and stored previous states, introducing memory into digital systems through feedback mechanisms.

**Key Points:**
- Sequential circuits differ from combinational circuits by incorporating memory elements and feedback paths
- Synchronous circuits use clock signals for controlled state transitions; asynchronous circuits respond immediately to input changes
- Storage elements include level-sensitive latches (SR, D) and edge-triggered flip-flops (D, JK, T), each with distinct characteristic equations
- State behavior is formally represented using state tables and state diagrams
- Moore machines produce outputs based on state alone; Mealy machines output based on both state and inputs
- Timing analysis ensures setup/hold constraints are met and determines maximum operating frequency
