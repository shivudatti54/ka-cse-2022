# Chapter 5 (5.1 to 5.4) Summary

### 5.1 Introduction to Probability

- Probability is a measure of the likelihood of an event occurring
- Probability is denoted by the letter P
- The probability of an event A is represented as P(A)
- Rule of Product: P(A and B) = P(A) \* P(B)

### 5.2 Random Variables

- A random variable is a variable that can take on different values
- Discrete random variable: can take on distinct, finite values
- Continuous random variable: can take on any value within a given range
- Example: rolling a die

### 5.3 Probability Distributions

- Probability distribution: a function that describes the probability of a random variable taking on specific values
- Types:
  - Discrete uniform distribution
  - Discrete Poisson distribution
  - Continuous uniform distribution
  - Continuous normal distribution

### 5.4 Bayes' Theorem

- Bayes' theorem: P(A|B) = P(B|A) \* P(A) / P(B)
- Bayes' theorem: used to update probabilities based on new information
- Example: disease diagnosis
