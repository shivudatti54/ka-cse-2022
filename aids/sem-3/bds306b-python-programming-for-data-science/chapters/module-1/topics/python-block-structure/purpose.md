# Learning Objectives: Bitcoin Block Structure
After studying this topic, you should be able to:
1.  Identify and describe the function of each of the six fields in the Bitcoin block header.
2.  Explain the process of Bitcoin mining (proof-of-work) and how the Nonce and Target (Bits) fields are used within it.
3.  Describe the structure and purpose of a Merkle Tree and how it generates the Merkle Root for a block.
4.  Differentiate between the concepts of block size and block weight in the context of the SegWit upgrade.
5.  Articulate how the 'Previous Block Hash' field creates an immutable cryptographic chain of blocks.
6.  Recognize the unique properties of the Genesis Block.
