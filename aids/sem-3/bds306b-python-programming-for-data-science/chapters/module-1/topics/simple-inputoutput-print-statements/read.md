# Module 1: Simple Input/Output with `print()` Statements

## Introduction

In Python, the simplest form of a program often involves interacting with the user. This interaction is facilitated by **Input** (receiving data) and **Output** (displaying data) operations. For data science, this is the foundational step for reading raw data and presenting your results. The `print()` function is the primary tool for output, while the `input()` function handles user input. Mastering these is crucial for building interactive scripts and debugging your code.
