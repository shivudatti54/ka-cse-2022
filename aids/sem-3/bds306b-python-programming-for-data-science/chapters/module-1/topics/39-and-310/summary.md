# **3.9 and 3.10 Revision Notes**

## **3.9: Probability Theory**

### Key Concepts

- Probability distribution:
  - Discrete uniform distribution
  - Continuous uniform distribution
- Expected value (mean)
- Variance
- Standard deviation
- Bayes' theorem

### Formulas

- Expected value: `E(X) = ∑xP(x)`
- Variance: `σ^2 = E(X^2) - (E(X))^2`
- Standard deviation: `σ = √σ^2`

### Important Theorems

- **Chebyshev's Inequality**: `P(|X - E(X)| ≥ kσ) ≤ 1/k^2`

### 3.10: Regression Analysis

---

### Key Concepts

- Simple linear regression
- Multiple linear regression
- Coefficient of determination (R-squared)
- Residual analysis

### Formulas

- Simple linear regression:
  - Regression equation: `y = β0 + β1x + ε`
  - Coefficient of determination: `R^2 = 1 - (SS_res / SS_tot)`
- Multiple linear regression:
  - Regression equation: `y = β0 + β1x1 + … + βnxn + ε`

### Important Theorems

- **Ordinary Least Squares (OLS)**: regression coefficients are obtained by minimizing the sum of squared errors

- Review probability distributions and expected values
- Practice calculating variance and standard deviation
- Understand simple and multiple linear regression concepts
- Familiarize yourself with R-squared and residual analysis
