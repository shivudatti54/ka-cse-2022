### 3.2: Data Types and Variables

In this chapter, we will learn about the different data types in Python and how to use variables to store and manipulate data.

### Data Types

Python has several built-in data types that can be used to store and manipulate data.

#### 1. Integers (int)

Integers are whole numbers that can be either positive or negative.

- Example: `x = 10` or `y = -5`
- Operations: addition, subtraction, multiplication, and division

#### 2. Floating Point Numbers (float)

Floating point numbers are decimal numbers that can have a fractional part.

- Example: `x = 10.5` or `y = -3.14`
- Operations: addition, subtraction, multiplication, and division

#### 3. Strings (str)

Strings are sequences of characters that can be enclosed in quotes.

- Example: `x = "Hello, World!"` or `y = 'Hello, World!'`
- Operations: concatenation, slicing, and indexing

#### 4. Boolean Values (bool)

Boolean values are true or false values that can be used in conditional statements.

- Example: `x = True` or `y = False`
- Operations: logical operations (AND, OR, NOT)

#### 5. Lists (list)

Lists are ordered collections of values that can be of any data type.

- Example: `x = [1, 2, 3, 4, 5]` or `y = ["apple", "banana", "cherry"]`
- Operations: indexing, slicing, concatenation, and methods (append, insert, remove)

#### 6. Tuples (tuple)

Tuples are ordered, immutable collections of values that can be of any data type.

- Example: `x = (1, 2, 3, 4, 5)` or `y = ("apple", "banana", "cherry")`
- Operations: indexing, slicing

#### 7. Dictionaries (dict)

Dictionaries are unordered collections of key-value pairs that can be of any data type.

- Example: `x = {"name": "John", "age": 30}` or `y = {"apple": "red", "banana": "yellow"}`
- Operations: key-value access, indexing, and methods (keys, values, items)

#### 8. Sets (set)

Sets are unordered collections of unique values that can be of any data type.

- Example: `x = {1, 2, 3, 4, 5}` or `y = {"apple", "banana", "cherry"}`
- Operations: union, intersection, difference, and methods (add, remove, discard)

### Variables

Variables are used to store and manipulate data in Python.

- Example: `x = 10` or `y = "Hello, World!"`
- Types of variables: local, global, and nonlocal

### Key Concepts

- Data types and variables are used to store and manipulate data in Python.
- Python has several built-in data types, including integers, floating point numbers, strings, boolean values, lists, tuples, dictionaries, and sets.
- Variables are used to store and manipulate data in Python.
- Types of variables: local, global, and nonlocal.

### Code Examples

```python
# Example 1: Creating variables and printing values
x = 10
y = "Hello, World!"

print("y =", y)

# Example 2: Creating lists and dictionaries
numbers = [1, 2, 3, 4, 5]
person = {"name": "John", "age": 30}

print("Person =", person)
```
