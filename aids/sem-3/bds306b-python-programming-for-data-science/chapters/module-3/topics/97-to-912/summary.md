# **9.7 to 9.12) Python Programming for Data Science Revision Notes**

### 9.7) Data Preprocessing

- Data cleaning and preprocessing are essential steps in data science
- Handling missing values:
  - Mean, median, and mode imputation
  - Imputation using Python libraries (e.g., Pandas, Scikit-learn)
- Data normalization and scaling:
  - Min-Max Scaling
  - Standardization
  - Feature Scaling
- Data transformation:
  - Log transformation
  - Box-Cox transformation

### 9.8) Data Visualization

- Importance of data visualization in data science
- Types of plots:
  - Scatter plots
  - Line plots
  - Bar plots
  - Histograms
- Best practices for data visualization:
  - Choose the right plot type for the data
  - Use color and size effectively
  - Label axes and add titles

### 9.9) Model Evaluation

- Importance of model evaluation in data science
- Metrics for evaluating model performance:
  - Accuracy
  - Precision
  - Recall
  - F1-score
  - Mean Squared Error (MSE)
  - Mean Absolute Error (MAE)
- Cross-validation:
  - K-fold cross-validation
  - Stratified cross-validation

### 9.10) Model Selection

- Types of models:
  - Linear Regression
  - Logistic Regression
  - Decision Trees
  - Random Forests
  - Support Vector Machines (SVMs)
- Model selection criteria:
  - Accuracy
  - Precision
  - Recall
  - F1-score
  - AUC-ROC
  - Cross-validation

### 9.11) Hyperparameter Tuning

- Importance of hyperparameter tuning in data science
- Techniques for hyperparameter tuning:
  - Grid search
  - Random search
  - Bayesian optimization
  - Cross-validation
- Hyperparameter tuning libraries:
  - Scikit-learn
  - Hyperopt

### 9.12) Model Deployment

- Importance of model deployment in data science
- Techniques for model deployment:
  - Model serving

  - TensorFlow Serving
