## **Key Concepts**

- **Data Types**
  - Integer: whole numbers, e.g., 1, 2, 3
  - Float: decimal numbers, e.g., 3.14, -0.5
  - String: sequences of characters, e.g., 'hello', "hello"
  - Boolean: true or false values
  - List: ordered collections of items, e.g., [1, 2, 3], ['a', 'b', 'c']
  - Dictionary: unordered collections of key-value pairs, e.g., {1: 'one', 2: 'two'}
- **Functions**
  - Defining functions: `def function_name(parameters):`
  - Function calls: `function_name(arguments)`
- **Control Structures**
  - Conditional statements: if-else statements
  - Loops: for loops, while loops
