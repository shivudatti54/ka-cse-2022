# Data Structure Operations
This chapter comprehensively examined the six fundamental operations on data structures: traversal, searching, insertion, deletion, sorting, and merging. Traversal accesses all elements in Θ(n) time for both arrays and linked lists. Searching was analyzed through linear search (O(n) worst case) and binary search (Θ(log n) using recurrence relation T(n) = T(n/2) + 1). Insertion and deletion operations were contrasted between arrays (O(1) at end, O(n) at beginning) and linked lists (O(1) at beginning, O(n) at end). Sorting algorithms covered bubble sort (Θ(n²) average, Θ(n) best with optimization) and merge sort (Θ(n log n) proven via Master Theorem). The merging operation combines sorted structures in Θ(n₁ + n₂) time. A critical insight is that the choice of data structure significantly impacts operation efficiency—for instance, array insertion at the end is O(1) while linked list insertion at the end is O(n), demonstrating the importance of understanding both theoretical complexity and practical trade-offs in data structure selection.
