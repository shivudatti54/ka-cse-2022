# Arrays are contiguous-memory linear data structures that provide O(1) random access through the address formula Address(arr[i]) = B + (i × w). This module covers: (1) formal mathematical definition and Array ADT specification; (2) C implementation with static stack allocation and dynamic heap allocation including resize operations; (3) proof of address calculation theorem by mathematical induction; (4) comprehensive complexity analysis of all operations—traversal O(n), linear search O(n), binary search O(log n), insertion O(n), deletion O(n); (5) multidimensional array representation in row-major order with derived address formulas; (6) amortized O(1) insertion cost proof for dynamically resizing arrays with geometric growth. Arrays serve as the foundational data structure for understanding more complex structures and enable cache-efficient algorithm implementations.
