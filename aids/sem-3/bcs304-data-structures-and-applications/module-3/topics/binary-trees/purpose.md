# Learning Objectives

After studying this topic, you should students able to:

1. Define and explain binary tree terminology including root, parent, child, leaf, depth, height, and level
2. Distinguish between different types of binary trees: full, complete, perfect, degenerate, and balanced
3. Prove the relationship between internal nodes and leaf nodes in a full binary tree
4. Derive and apply the formula for maximum number of nodes in a binary tree of given height
5. Analyze the relationship between number of nodes and height in binary trees
6. Compare and contrast linked representation and array representation of binary trees
7. Calculate time and space complexity for basic binary tree operations
8. Apply binary tree properties to solve numerical problems in examinations
