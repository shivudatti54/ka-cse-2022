# Learning Objectives

After studying this topic, you should be able to:

1.  Differentiate between a standard queue, a double-ended queue (deque), and a priority queue based on their principles of operation.
2.  Describe and implement the core operations (insertFront, insertRear, deleteFront, deleteRear) of a deque using a circular array.
3.  Analyze the time complexity of enqueue and dequeue operations for different implementations of a priority queue (unsorted array, sorted array, linked list).
4.  Identify real-world scenarios where a deque would be a more suitable choice than a standard queue or stack.
5.  Explain the concept of priority in a queue and justify the use of a priority queue in applications like CPU scheduling.
6.  Solve problems, such as checking for palindromes, by leveraging the properties of a deque.
