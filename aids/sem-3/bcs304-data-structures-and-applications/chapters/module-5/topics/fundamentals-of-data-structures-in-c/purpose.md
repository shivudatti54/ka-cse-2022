# Learning Objectives
After studying this topic, you should be able to:
1.  Identify and categorize different data types generated by IoT devices, including sensor data, actuator data, metadata, and multimedia data.
2.  Differentiate between structured, semi-structured, and unstructured data and provide examples of each in an IoT context.
3.  Evaluate and select appropriate data structures (e.g., arrays, structs, queues) for efficient data handling on resource-constrained IoT devices.
4.  Compare and contrast data interchange formats like JSON, Protocol Buffers, CSV, and Avro based on characteristics like size, speed, flexibility, and use case.
5.  Map the journey of IoT data from the edge to the cloud, identifying the typical data structures and formats used at each stage of the pipeline.
6.  Apply this knowledge to design the data payload for a simple IoT application, such as the weather monitoring system case study.
