# Learning Objectives
After studying this topic, you should be able to:
1.  Define and correctly apply the three types of elementary row operations to a given matrix.
2.  Determine the elementary matrix corresponding to a given row operation and explain its effect through multiplication.
3.  Use a sequence of elementary row operations to transform a matrix into its Row Echelon Form (REF) and Reduced Row Echelon Form (RREF).
4.  Employ elementary row operations to solve systems of linear equations using Gaussian and Gauss-Jordan elimination.
5.  Find the inverse of an invertible matrix using the algorithm that involves elementary row operations on the augmented matrix [A | I].
6.  Explain why elementary row operations preserve the rank of a matrix and the solution set of a linear system.
