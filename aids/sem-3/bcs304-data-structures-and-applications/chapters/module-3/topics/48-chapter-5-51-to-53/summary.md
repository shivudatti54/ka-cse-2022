# **4.8 Chapter-5: 5.1 to 5.3 Revision Notes**

## **5.1 Stack**

- Definition: A Last-In-First-Out (LIFO) data structure
- Operations:
  - Push: adds an element to the top of the stack
  - Pop: removes the top element from the stack
  - Peek: returns the top element without removing it
  - isEmpty: checks if the stack is empty
- Examples: evaluating postfix expressions, parsing syntax

## **5.2 Queue**

- Definition: A First-In-First-Out (FIFO) data structure
- Operations:
  - enqueue: adds an element to the end of the queue
  - dequeue: removes the front element from the queue
  - peek: returns the front element without removing it
  - isEmpty: checks if the queue is empty
- Examples: job scheduling, print queues

## **5.3 Linked List**

- Definition: A dynamic collection of elements, each with a reference to the next element
- Operations:
  - insert: adds an element at a specific position
  - delete: removes an element at a specific position
  - traverse: visits each element in the list
- Types:
  - Singly Linked List
  - Doubly Linked List
- Applications: dynamic memory allocation, browser history
