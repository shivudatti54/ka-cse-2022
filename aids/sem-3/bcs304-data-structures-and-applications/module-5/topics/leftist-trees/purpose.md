# Learning Objectives

After studying this topic, you should students able to:

1. Define leftist trees and explain their relationship to priority queues
2. Define and compute the null path length (NPL) for any node in a binary tree
3. State and explain the leftist property and its significance in tree operations
4. Trace through the merge algorithm step-by-step for given leftist trees
5. Analyze and prove the O(log n) time complexity of the merge operation
6. Implement the insert, delete-min, and find-min operations using merge
7. Construct a leftist tree from a given set of keys using the build-heap operation
8. Compare leftist trees with binary heaps and explain the advantages of leftist trees for merge operations
9. Apply leftist trees to solve practical problems involving meldable priority queues
