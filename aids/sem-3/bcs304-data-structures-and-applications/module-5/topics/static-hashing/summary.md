# Static hashing employs fixed-size tables with deterministic hash functions to achieve O(1) average-case operations. Key topics include hash function design (division, mid-square, folding, digit extraction), collision resolution through separate chaining and open addressing (linear probing, quadratic probing, double hashing), and rigorous performance analysis using load factor mathematics. The module emphasizes mathematical proofs for expected probe sequences, addresses clustering phenomena, and includes worked numerical examples and application-level assessment questions to reinforce learning.
