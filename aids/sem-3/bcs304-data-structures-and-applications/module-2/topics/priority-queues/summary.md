# Priority Queues
A priority queue is an abstract data type where elements are served by priority rather than insertion order, unlike FIFO queues. Two primary types exist: ascending (min-priority) queues serve smallest priority first, while descending (max-priority) queues serve largest priority first. The binary heap provides optimal O(log n) implementation using complete binary trees with array representation and the heap property. Insert operation percolates up from leaf to root, while delete-min/max percolates down from root to leaf. Build-heap achieves O(n) time through bottom-up construction. Alternative implementations include unordered/ordered arrays and Fibonacci heaps with amortized analysis. Critical applications include Dijkstra's algorithm, Prim's MST, CPU scheduling, and Huffman coding. The module emphasizes formal complexity proofs, algorithmic reasoning, and application-based problem solving for students-level competency.
