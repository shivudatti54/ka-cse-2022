# Queues
A queue is a fundamental FIFO (First-In-First-Out) linear data structure with operations at two ends: rear for insertion (enqueue) and front for deletion (dequeue). Key operations include enqueue, dequeue, peek, isEmpty, and isFull. Array-based implementation provides O(1) time complexity for all operations but suffers from memory wastage due to front index drift. Linked list implementation offers dynamic sizing with O(1) enqueue/dequeue. Circular queues resolve the memory wastage problem using modular arithmetic. Queues are essential in operating system scheduling (CPU, disk), graph algorithms (BFS), network packet handling, and real-world waiting line systems. The module includes detailed C implementations, complexity proofs, and application scenarios.
