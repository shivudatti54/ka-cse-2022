# Learning Objectives

After studying this topic, you should students able to:

1. Define disjoint sets and explain their significance in equivalence relation handling and graph algorithms.
2. Explain the forest-based (array) representation of disjoint sets with parent pointers.
3. Implement MakeSet, Find (with path compression), and Union (with union by rank) operations.
4. Analyze the amortized time complexity of union-find operations using the inverse Ackermann function.
5. Compare array-based and linked list representations in terms of time, space, and cache efficiency.
6. Apply disjoint sets in Kruskal's algorithm for Minimum Spanning Tree construction.
7. Solve problems involving cycle detection and connectivity verification using union-find.
8. Prove the correctness of union-find optimizations and their time complexity bounds.
