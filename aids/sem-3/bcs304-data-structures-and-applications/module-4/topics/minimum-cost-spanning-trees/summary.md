# Minimum Cost Spanning Trees
A Minimum Cost Spanning Tree (MST) is a spanning tree with minimum total edge weight in a weighted connected undirected graph. This topic covers: (1) formal definitions and properties including cut property and cycle property; (2) theoretical proofs establishing correctness of greedy strategies; (3) Kruskal's algorithm using Union-Find for cycle detection with O(E log V) complexity; (4) Prim's algorithm growing MST from a source vertex using priority queues with O(E log V) or O(V²) complexity; (5) comparison between algorithms for sparse vs dense graphs; (6) hard-level MCQs testing numerical computation and algorithm analysis. The cut property states the minimum edge crossing any cut belongs to all MSTs, while the cycle property states the maximum edge in any cycle belongs to no MST—both form the theoretical foundation for greedy MST algorithms.
