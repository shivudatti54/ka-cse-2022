# Module 1: The Java Keywords & Data Types

## Introduction

Java is a strongly typed, object-oriented programming language. This means two things are fundamental to writing any Java program: understanding the reserved words that form the language's syntax (keywords) and the different kinds of data you can work with (data types). This module provides a foundational overview of these two critical concepts, which are the building blocks for all Java applications.

---

## 1. Java Keywords

Keywords are predefined, reserved words in Java that have a specific meaning and purpose to the compiler. They cannot be used as identifiers (e.g., for variable names, class names, or method names).

Java has approximately 50 keywords. Some of the most common ones you will encounter early on include:

*   **Class and Object Management:** `class`, `new`, `this`, `super`, `extends`, `implements`
*   **Access Modifiers:** `public`, `private`, `protected`
*   **Data Types:** `int`, `char`, `double`, `float`, `boolean`, `byte`, `long`, `short`
*   **Flow Control:** `if`, `else`, `switch`, `case`, `for`, `while`, `do`, `break`, `continue`
*   **Exception Handling:** `try`, `catch`, `finally`, `throw`, `throws`
*   **Modifiers:** `static`, `final`, `abstract`, `synchronized`, `volatile`
*   **Package Control:** `package`, `import`
