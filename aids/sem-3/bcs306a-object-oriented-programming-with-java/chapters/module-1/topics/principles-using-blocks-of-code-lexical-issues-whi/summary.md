# \*\*Principles), Using Blocks of Code, Lexical Issues (Whitespace, Identifiers, Literals, Comments, Separators, The Java Keywords)

## **Principles of Programming**

- **Modularity**: Divide a program into smaller, independent modules.
- **Abstraction**: Hide implementation details and show only necessary information.
- **Encapsulation**: Bundling data and methods that manipulate it into a single unit.
- **Inheritance**: Creating a new class based on an existing class.
- **Polymorphism**: Ability of an object to take multiple forms.

## **Using Blocks of Code**

- **Blocks of Code**: Code enclosed in curly brackets `{}`.
- **Code Blocks**: Can contain statements, expressions, or blocks of code.
- ** Nesting**: Blocks of code can be nested within each other.

## **Lexical Issues**

- **Whitespace**: Spaces, tabs, and line breaks used to separate tokens.
- **Identifiers**: Names given to variables, methods, and classes.
  - **Keyword Identifiers**: Reserved words in Java (e.g., `public`, `class`).
  - **Variable Identifiers**: Names given to variables.
- **Literals**: Values represented exactly as written in code.
  - **Integer Literals**: Whole numbers (e.g., `123`).
  - **String Literals**: Sequences of characters enclosed in quotes (e.g., `"hello"`).
- **Comments**: Notes added to code to explain its purpose or behavior.
- **Separators**: Symbols used to separate tokens (e.g., commas, semicolons).
- **Java Keywords**: Reserved words in Java (e.g., `if`, `else`, `return`).

## **Java Keywords**

- `abstract`: Declaring a class as abstract.
- `assert`: Used to verify a condition.
- `boolean`: A primitive data type representing true or false.
- `break`: Used to exit a loop or switch statement.
- `byte`: A primitive data type representing an 8-bit signed integer.
- `case`: Used in switch statements to specify a value.
- `catch`: Used to handle exceptions.
- `class`: Declaring a new class.
- `const`: Used to declare constants.
- `continue`: Used to skip to the next iteration.
- `default`: Used in switch statements to specify a default value.
- `do`: Used to start a block of code that is executed repeatedly.
- `double`: A primitive data type representing a floating-point number.
- `else`: Used to specify an alternative value.
- `enum`: Used to declare an enumeration.
- `extends`: Used to inherit from a class.
- `final`: Used to declare final variables or methods.
- `finally`: Used to specify a block of code that is executed regardless.
- `float`: A primitive data type representing a floating-point number.
- `for`: Used to start a loop that iterates over a range.
- `goto`: Used to transfer control to a labeled statement.
- `if`: Used to specify a condition.
- `implements`: Used to implement an interface.
- `import`: Used to import packages.
- `instanceof`: Used to check if an object is an instance of a class.
- `int`: A primitive data type representing an integer.
- `interface`: Declaring a new interface.
- `long`: A primitive data type representing a signed integer.
- `native`: Used to declare native methods.
- `new`: Used to create a new object.
- `package`: Used to declare a package.
- `private`: Used to declare private variables or methods.
- `protected`: Used to declare protected variables or methods.
- `public`: Used to declare public variables or methods.
- `return`: Used to specify a value to be returned from a method.
- `short`: A primitive data type representing an 16-bit signed integer.
- `static`: Used to declare static variables or methods.
- `strictfp`: Used to specify a strictfp class.
- `super`: Used to access a superclass's members.
- `switch`: Used to execute different blocks of code based on a value.
- `synchronized`: Used to synchronize access to a shared resource.
- `this`: Used to access a current object's members.
- `throw`: Used to throw an exception.
- `throws`: Used to specify exceptions that a method may throw.
- `transient`: Used to declare transient variables.
- `try`: Used to start a block of code that may throw an exception.
- `void`: Used to declare methods that do not return a value.
- `volatile`: Used to declare volatile variables.
- `while`: Used to start a loop that iterates until a condition is met.
