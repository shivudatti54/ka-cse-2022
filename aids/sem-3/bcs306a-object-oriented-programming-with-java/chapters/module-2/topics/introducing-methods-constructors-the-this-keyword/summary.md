# Introducing Methods, Constructors, The `this` Keyword, Garbage Collection

=====================================

## Object Oriented Programming with JAVA

=====================================

### Module: Introducing Classes

---

### Topic: Introducing Methods, Constructors, The `this` Keyword, Garbage Collection

---

### Key Points

---

- **Methods**
  - A block of code that performs a specific task
  - Can return values using `return` statement
  - Can take arguments using `param` keyword
- **Constructors**
  - Special method that is called when an object is created
  - No return type and no parameters
  - Used to initialize objects
- **The `this` Keyword**
  - Refers to the current object
  - Used to access variables and methods of the current class
  - Used to create a new object within the scope of a method
- **Garbage Collection**
  - Process of automatically freeing up memory occupied by objects that are no longer in use
  - Ensures memory safety and efficiency

### Important Formulas, Definitions, Theorems

---

- **Method Overloading**: When two or more methods have the same method name but different parameters.
- **Method Overriding**: When a subclass provides a different implementation of a method that is already defined in its superclass.
- **Constructors**: `public`, `private`, and `protected` access modifiers can be used to control access to constructors.
- **Garbage Collection**: `Minor GC` and `Major GC` are the two types of garbage collection.

### Revision Tips

---

- Understand the difference between `this` keyword and object references.
- Know when to use `public` and `private` access modifiers for constructors.
- Be able to explain the concept of method overloading and overriding.
- Understand the importance of garbage collection in ensuring memory safety and efficiency.
