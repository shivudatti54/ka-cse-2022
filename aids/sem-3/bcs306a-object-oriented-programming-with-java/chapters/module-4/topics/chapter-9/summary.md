# **Chapter 9: Packages in Java**

## **Key Points**

- **What is a Package?**: A package in Java is a collection of related classes, interfaces, and nested classes that share a common purpose or applicability domain.
- **Importing Packages**:
  - `import` statement is used to import a package.
  - Package name is used to import a specific class or interface.
  - Wildcard `*` can be used to import all classes in a package.
- **Package Declaration**:
  - `package` keyword is used to declare a package.
  - Package name is specified after the `package` keyword.
  - Package declaration is used inside the class file.
- **Member Access Modifiers**: Member access modifiers (`public`, `private`, `protected`) can be used to access members of a class.
- **Access Modifiers**:
  - `public`: Accessible from anywhere.
  - `private`: Accessible only within the class.
  - `protected`: Accessible within the class and its subclasses.

## **Important Formulas and Definitions**

### Package Declaration

`package package_name;`

### Importing Packages

`import package_name.class_name;`

### Access Modifiers

- `public`: Accessible from anywhere.
- `private`: Accessible only within the class.
- `protected`: Accessible within the class and its subclasses.

### Key Theorems

- **Package Hierarchy**: A package can have multiple sub-packages.
- **Sub-packages**: A sub-package is a package that is a subclass of another package.
