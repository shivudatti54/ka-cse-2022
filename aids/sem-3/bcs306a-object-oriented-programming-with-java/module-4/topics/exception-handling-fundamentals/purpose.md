# Learning Objectives

After studying this topic, you should students able to:

1. Understand the concept of exceptions and their role in Java program execution

2. Explain the Java exception class hierarchy including the distinction between Error and Exception

3. Differentiate between checked exceptions and unchecked exceptions with examples

4. Write proper try-catch-finally blocks to handle exceptions in Java programs

5. Use the throw keyword to explicitly throw exceptions from methods

6. Declare exceptions using the throws keyword in method signatures

7. Determine the appropriate exception handling strategy for different scenarios

8. Understand exception propagation and the call stack mechanism
