# Learning Objectives

After studying this topic, you should students able to:

1. Define the purpose of the `throws` keyword in Java exception handling.
2. Identify the types of exceptions that must students declared with the `throws` clause.
3. Explain the difference between handling exceptions with `try-catch` and declaring them with `throws`.
4. Use the `throws` clause to declare exceptions in a method signature, following the rules for method overriding.
5. Distinguish between checked and unchecked exceptions in the context of the `throws` keyword.
6. Analyze the implications of using `throws` on method overriding and the Exception class hierarchy.
7. Justify the use of `throws` in real-world applications, considering its impact on error handling and software reliability.
8. Design a method signature that effectively uses the `throws` clause to declare exceptions, promoting robust and maintainable code.
