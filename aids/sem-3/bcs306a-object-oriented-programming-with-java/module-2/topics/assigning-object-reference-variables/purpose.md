# Learning Objectives

After studying this topic, you should students able to:

1. Explain the difference between primitive variables and reference variables in Java
2. Describe what happens in memory when object reference variables are assigned
3. Demonstrate how changes to one reference variable affect another when both point to the same object
4. Differentiate between shallow copy and deep copy of objects
5. Compare the use of == operator and equals() method for object comparison
6. Explain the role of the this reference variable in Java
7. Analyze how references are passed to methods and understand pass-by-value semantics
8. Apply the concept of object reference assignment to solve practical programming problems
