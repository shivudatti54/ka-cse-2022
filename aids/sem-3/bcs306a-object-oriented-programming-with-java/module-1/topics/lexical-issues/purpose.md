# Learning Objectives

After studying this topic, you should students able to:

1. Explain the role of Unicode in Java and describe how Java handles characters from different writing systems.

2. Compare and contrast the different input mechanisms in Java (Scanner, BufferedReader, Console) and select the appropriate one for different scenarios.

3. Demonstrate the process of tokenization and explain how Java's lexical analyzer converts source code into tokens.

4. Identify and correctly use different types of literals in Java, including integer, floating-point, character, string, and boolean literals.

5. Apply the rules for forming valid identifiers and differentiate between identifiers and reserved keywords in Java.

6. Write Java programs that effectively read and process user input using Scanner and BufferedReader classes.

7. Understand the significance of white space and comments in Java and explain why proper code documentation is essential.

8. Demonstrate knowledge of operators and separators in Java and apply correct operator precedence rules in complex expressions.
