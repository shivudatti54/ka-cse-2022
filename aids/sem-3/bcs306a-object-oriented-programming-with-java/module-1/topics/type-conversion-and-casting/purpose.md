# Learning Objectives

After studying this topic, you should students able to:

1. Explain the difference between implicit (widening) and explicit (narrowing) type conversion in Java
2. Identify the data type widening hierarchy and determine when implicit conversions occur automatically
3. Apply explicit casting to convert between incompatible or narrower primitive types
4. Analyze how binary numeric promotion works in arithmetic expressions with mixed data types
5. Implement reference type casting between classes in an inheritance hierarchy using upcasting and downcasting
6. Use the instanceof operator to perform safe type checking before downcasting
7. Convert String values to numeric types using wrapper class methods like parseInt() and parseDouble()
8. Write Java programs that handle type conversion errors using appropriate exception handling
