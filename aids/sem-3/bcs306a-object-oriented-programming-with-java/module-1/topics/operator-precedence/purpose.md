# Learning Objectives

After studying this topic, you should students able to:

1. Define operator precedence and explain why it is necessary in Java programming.

2. Recall the hierarchy of Java operators from highest to lowest precedence.

3. Apply the precedence rules to evaluate complex arithmetic expressions correctly.

4. Understand the associativity of operators and how it affects evaluation order when operators have the same precedence.

5. Use parentheses effectively to override default precedence and make expressions clearer.

6. Evaluate expressions involving logical and relational operators using proper precedence rules.

7. Distinguish between prefix and postfix increment/decrement operators and their effects in expressions.

8. Analyze and debug expressions that produce unexpected results due to precedence issues.
