## **Sorting Algorithms**

- **Bubble Sort:**
  - Works by repeatedly swapping the adjacent elements if they are in the wrong order
  - Time complexity: O(n^2)
  - Space complexity: O(1)
- **Selection Sort:**
  - Works by selecting the smallest element from the unsorted portion and swapping it with the first element
  - Time complexity: O(n^2)
  - Space complexity: O(1)
- **Insertion Sort:**
  - Works by inserting elements into the sorted portion one by one
  - Time complexity: O(n^2) in the worst case
  - Space complexity: O(1)

## **Big O Notation**

- **Big O**:
  - Worst-case time complexity of an algorithm
  - Used to describe the performance of an algorithm
- **Little o**:
  - Best-case time complexity of an algorithm
  - Used to describe the optimal performance of an algorithm

## **Important Formulas and Definitions**

- **Time Complexity:**
  - T(n) = Ω(n log n)
  - T(n) = O(n^2) (worst-case for Bubble Sort, Selection Sort)
  - T(n) = Ω(1) (best-case for Insertion Sort)
- **Space Complexity:**
  - S(n) = Ω(1) (constant space complexity)

**Theorem:**

- **Stirling's Accumulation Formula:**
  - n! = (n-1)! \* n
  - Used to calculate the factorial of a number
