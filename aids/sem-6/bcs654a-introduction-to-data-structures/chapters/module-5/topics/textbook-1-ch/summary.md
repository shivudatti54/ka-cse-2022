## **Introduction to Data Structures**

- Data structures are collections of data elements, each of which represents a value or set of values
- Types of data structures: arrays, linked lists, stacks, queues, trees, graphs

## **Sorting**

### Introduction

- Sorting: the process of arranging data in a specific order
- Importance of sorting: efficient searching, data analysis, and visualization

### Algorithms

- **Bubble Sort**
  - Simple, intuitive algorithm
  - Works by repeatedly swapping adjacent elements if they are in the wrong order
  - Time complexity: O(n^2)
  - Space complexity: O(1)
- **Selection Sort**
  - Works by selecting the smallest (or largest) element from the unsorted portion of the array
  - Time complexity: O(n^2)
  - Space complexity: O(1)
- **Insertion Sort**
  - Works by iterating through the array one element at a time
  - Time complexity: O(n^2)
  - Space complexity: O(1)

### Theorems

- **Bubble Sort Theorem**: If the input array is already sorted, no swaps will be made during the bubble sort process.
- **Insertion Sort Theorem**: If the input array is already sorted, insertion sort will have a time complexity of O(n).

### Important Formulas

- Time complexity: O(n) = linear time complexity
- Space complexity: O(1) = constant space complexity
