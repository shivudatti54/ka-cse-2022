### 9.1 to 9.2, 9.3 (Only 9.3.1 to 9.3.5, 9.3.11 to 9.3.12), 9.4 to 9.5

#### 9.1

- **Definition:** A singly linked list is a linear data structure where each element is a separate object, and each element (called a node) points to the next node in the sequence.
- **Types of Singly Linked Lists:**
  - **Linear Linked List:** A basic linked list where each node points to the next node.
  - **Circular Linked List:** A linked list where the last node points back to the first node.

#### 9.2

- **Operations on Singly Linked Lists:**
  - **Insertion:** Insert a new node at the beginning or end of the list.
  - **Deletion:** Delete a node from the list.
  - **Traversing:** Traverse the list in a forward or backward direction.

#### 9.3

- **Self-Referential Structures:**
  - **Circular Linked List:** A linked list where the last node points back to the first node.
  - **Ring Buffer:** A data structure that uses a circular buffer to store data.

#### 9.3.1

- **Ring Buffer Operations:**
  - **Insert:** Add an element to the buffer.
  - **Delete:** Remove an element from the buffer.

#### 9.3.11

- **Circular Linked List Operations:**
  - **Insert at Head:** Insert a new node at the beginning of the list.
  - **Insert at Tail:** Insert a new node at the end of the list.

#### 9.3.12

- **Circular Linked List Traversals:**
  - **Forward Traversal:** Traverse the list in a forward direction.
  - **Backward Traversal:** Traverse the list in a backward direction.

#### 9.4

- **Operations on Self-Referential Structures:**
  - **Insertion:** Insert a new node into the self-referential structure.
  - **Deletion:** Delete a node from the self-referential structure.

#### 9.5

- **Theoretical Background:**
  - **Function Definition:** A function that takes a linked list as input and returns a linked list as output.
  - **Theorem:** Theorem 1: A self-referential structure can be used to implement a ring buffer.

**Important Formulas and Definitions:**

- **Node:** A single element in a linked list.
- **Linked List:** A data structure where each element points to the next element.
- **Ring Buffer:** A data structure that uses a circular buffer to store data.

**Important Theorems:**

- **Theorem 1:** A self-referential structure can be used to implement a ring buffer.
