# Learning Objectives

After studying this topic, you should be able to:

1. Define what constitutes a tree data structure and explain its hierarchical nature
2. Identify and explain all fundamental tree terminology including root, parent, child, sibling, leaf, and internal nodes
3. Differentiate between degree of a node and degree of a tree, and calculate these values for given trees
4. Understand and compute height and depth of trees and individual nodes
5. Explain the concepts of ancestors and descendants and identify them in a given tree
6. Distinguish between various types of trees including general trees, binary trees, and binary search trees
7. Compare full, complete, and perfect binary trees with examples
8. Represent trees using both linked representation and array representation methods
9. Apply the properties of trees to solve computational problems involving node counts and edge calculations
