# **Textbook 1: Ch - Introduction to Data Structures**

## **Arrays: Introduction**

- An array is a collection of elements of the same data type stored in contiguous memory locations.
- Arrays are used to store and manipulate large amounts of data.
- A one-dimensional array is an array containing elements of the same data type stored in sequential memory locations.

## **One-Dimensional Arrays**

- Declaration: `int arr[5];` or `float arr[10]`
- Indexing: `arr[0]` to `arr[4]`
- Initialization: `arr[i] = value;`
- Formula for Finding the Element at Index `i`: `arr[i]`

## **Two-Dimensional Arrays**

- Declaration: `int arr[2][3];` or `float arr[3][4]`
- Indexing: `arr[0][0]` to `arr[1][2]`
- Initialization: `arr[i][j] = value;`
- Formula for Finding the Element at Index `(i, j)`: `arr[i][j]`

## **Important Formulas and Definitions**

- **Array Index**: The position of an element in an array, represented by `i`.
- **Lower Bound**: The smallest index that can be used to access an array element.
- **Upper Bound**: The largest index that can be used to access an array element.
- **Array Size**: The number of elements in an array, represented by `n`.

## **Important Theorems**

- **Array Indexing Theorem**: `arr[i]` is equal to the value stored at the memory location `base + i * size`, where `base` is the base address of the array and `size` is the size of each element.
- **Array Iteration Theorem**: `i = 0` to `n-1`, where `n` is the size of the array.

## **Revision Key Points**

- Arrays are used to store and manipulate large amounts of data.
- One-dimensional arrays are used to store elements in sequential memory locations.
- Two-dimensional arrays are used to store elements in a grid-like structure.
- Array indexing is used to access elements in an array.
- Array size is the number of elements in an array.
