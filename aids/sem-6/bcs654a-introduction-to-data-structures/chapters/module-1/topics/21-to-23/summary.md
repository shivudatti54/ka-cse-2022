# **2.1 to 2.3: Arrays**

### 2.1: Types of Arrays

- **One-Dimensional Arrays**: An array of elements of the same data type stored in contiguous memory locations.
- **Two-Dimensional Arrays**: An array of arrays, also known as matrices, where each element is a one-dimensional array.

### 2.2: Initializing Arrays

- **Static Initialization**: Elements are initialized when the array is declared.
- **Dynamic Initialization**: Elements are initialized using a loop or function.

### 2.3: Array Operations

- **Accessing Elements**: Elements can be accessed using their index (row and column for 2D arrays).
- **Assigning Values**: Values can be assigned to elements using their index.

**Important Formulas and Definitions**

- **Indexing Formula**: `arr[i][j]` to access the element at row `i` and column `j` in a 2D array.
- **Array Size**: The number of elements in an array, calculated using the formula `size = number_of_elements`.

**Key Theorems and Concepts**

- **Array Bounds Checking**: A technique used to prevent out-of-bounds array access.
- **Array Overlap**: When two or more arrays overlap in memory, leading to unexpected behavior.
