# **8.1 to 8.5: INTRODUCTION TO DATA STRUCTURES**

### 8.1: Introduction to Data Structures

- A data structure is a way to organize and store data in a computer so that it can be efficiently accessed and modified.
- Data structures are used to improve the efficiency and scalability of computer programs.
- Key characteristics of data structures:
  - Organizes data in a way that facilitates efficient access and modification.
  - Optimizes data retrieval and manipulation operations.
  - Scalability: can handle large volumes of data.

### 8.2: One-Dimensional Arrays

- An array is a collection of elements of the same data type stored in contiguous memory locations.
- Key features:
  - One-dimensional (1D): elements are stored in a single row or column.
  - Fixed size: cannot be dynamically resized.
  - Indexing: elements are accessed using an index or subscript.

### 8.3: Two-Dimensional Arrays

- A two-dimensional array is a collection of elements of the same data type stored in a table or grid format.
- Key features:
  - Two-dimensional (2D): elements are stored in rows and columns.
  - Fixed size: cannot be dynamically resized.
  - Indexing: elements are accessed using row and column indices.

### 8.4: Initializing Two-Dimensional Arrays

- Initializing a two-dimensional array involves setting the values of each element to a specific value or pattern.
- Methods of initialization:
  - Rectangle initialization: sets a rectangular region of the array to a specific value.
  - Uniform initialization: sets all elements of the array to a specific value.

### 8.5: Important Formulas, Definitions, and Theorems

- **Array Indexing Formula**: `arr[i] = elements at index i in array arr`
- **Array Size Formula**: `size of array = number of elements in array`
- **Definition of a Data Structure**: A data structure is a way to organize and store data in a computer so that it can be efficiently accessed and modified.
- **Theorem of Array Operations**: For any array `arr` and indices `i` and `j`, `arr[i] = arr[j]` implies `i = j`.
