# Learning Objectives

After studying this topic, you should be able to:

1. Declare, initialize, and access elements in a one-dimensional array.
2. Describe how arrays are stored in contiguous memory and calculate element addresses.
3. Traverse an array using loops to perform operations like printing, summing, or finding max/min values.
4. Implement basic algorithms such as linear search and bubble sort on one-dimensional arrays.
5. Identify and avoid common errors associated with array usage, such as out-of-bounds indexing.
6. Explain the relationship between array names and pointers in the C language.
