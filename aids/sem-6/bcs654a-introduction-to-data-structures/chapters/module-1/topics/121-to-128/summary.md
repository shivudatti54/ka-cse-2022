# 12.1 to 12.8: Introduction to Arrays

=====================================================

### 12.1: Introduction to Arrays

- An array is a collection of elements of the same data type stored in contiguous memory locations.
- Arrays are used to store a fixed number of elements.
- Arrays have a fixed size.

### 12.2: One-Dimensional Arrays

- One-dimensional arrays store elements in a single row or column.
- Each element is identified by an index or subscript.
- Example: `int arr[5] = {1, 2, 3, 4, 5};`

### 12.3: Two-Dimensional Arrays

- Two-dimensional arrays store elements in a table or matrix.
- Each element is identified by two indices or subscripts (row and column).
- Example: `int arr[2][3] = {{1, 2, 3}, {4, 5, 6}};`

### 12.4: Initializing Two-Dimensional Arrays

- Array initialization is done using the assignment operator (`=`).
- Each element of the array is initialized with a value.
- Example: `int arr[2][3] = {{1, 2, 3}, {4, 5, 6}};`

### 12.5: Declaring Arrays

- Arrays can be declared using the `int arr[n]` syntax.
- `n` is the number of elements in the array.
- Example: `int arr[5] = {1, 2, 3, 4, 5};`

### 12.6: Accessing Array Elements

- Array elements are accessed using the index or subscript operator (`arr[index]`).
- The index starts from 0.

### 12.7: Modifying Array Elements

- Array elements can be modified using the assignment operator (`arr[index] = value`).
- Example: `arr[0] = 10;`

### 12.8: Important Formulas and Definitions

- **Array Size**: The number of elements in an array is called its size.
- **Array Index**: The index or subscript of an array element.
- **Array Declaration**: The syntax for declaring an array (`int arr[n]`).

### Theorems

- **Theorem of Array Initialization**: An array can be initialized using the assignment operator (`=`).
- **Theorem of Array Access**: Array elements can be accessed using the index or subscript operator (`arr[index]`).
