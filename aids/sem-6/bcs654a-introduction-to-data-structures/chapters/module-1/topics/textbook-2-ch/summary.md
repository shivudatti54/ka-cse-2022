## Textbook 2: Ch - Arrays: Introduction, One-Dimensional Arrays, Two-Dimensional Arrays

### Introduction

- Arrays are a collection of elements of the same data type stored in contiguous memory locations.
- Arrays provide efficient storage and retrieval of data.
- Arrays are declared using the `[]` syntax.

### One-Dimensional Arrays

- A single row of elements in an array.
- Elements are accessed using the array name and an index.
- Formula: `arr[i]` to access the `i`-th element.
- Example: `int arr[5] = {1, 2, 3, 4, 5};`

### Two-Dimensional Arrays

- A table of rows and columns of elements in an array.
- Elements are accessed using the array name and row and column indices.
- Formula: `arr[row][column]` to access the element at row `row` and column `column`.
- Example: `int arr[2][3] = {{1, 2, 3}, {4, 5, 6}};`

### Initializing Two-Dimensional Arrays

- Arrays can be initialized using curly brackets `{}`.
- Elements are specified using a comma-separated list.
- Example: `int arr[2][3] = {{1, 2, 3}, {4, 5, 6}};`

### Important Formulas and Definitions

- **Array Index**: The position of an element within an array, starting from 0.
- **Array Length**: The number of elements in an array.
- **Array Size**: The total number of bytes required to store an array.

### Theorem: Array Operations

- **Addition**: `arr[i] += 5` increments the `i`-th element by 5.
- **Subtraction**: `arr[i] -= 5` decrements the `i`-th element by 5.
- **Multiplication**: `arr[i] *= 2` multiplies the `i`-th element by 2.
- **Division**: `arr[i] /= 2` divides the `i`-th element by 2.
