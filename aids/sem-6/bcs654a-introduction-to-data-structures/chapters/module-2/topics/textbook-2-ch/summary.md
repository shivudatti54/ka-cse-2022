# Textbook 2: Chapter Ch - INTRODUCTION TO DATA STRUCTURES

=====================================

### Overview

- Introduction to data structures and their importance in computer science
- Brief history and evolution of data structures

### Key Concepts

- **Definition**: A collection of data elements, each of which can store multiple values, accessed by a unique index
- **Types of Data Structures**
  - Arrays
  - Linked Lists
  - Stacks
  - Queues
  - Trees
  - Graphs

### Important Formulas and Definitions

- **Array**:
  - Definition: A collection of elements, each identified by an index
  - Formula: `A[i] = elements[i]`
- **Linked List**:
  - Definition: A dynamic collection of elements, each of which points to the next element
  - Formula: `node->next = new_node`
- **Stack**:
  - Definition: A Last-In-First-Out (LIFO) data structure
  - Formula: `push(element)`, `pop()`
- **Queue**:
  - Definition: A First-In-First-Out (FIFO) data structure
  - Formula: `enqueue(element)`, `dequeue()`

### Important Theorems

- **Big O Notation**:
  - Definition: A measure of an algorithm's complexity
  - Formula: `O(n^2)`, `O(n log n)`, etc.
- **Divide and Conquer**:
  - Definition: A problem-solving strategy that breaks down a problem into smaller sub-problems

### Key Terms

- **Time Complexity**: The amount of time an algorithm takes to complete
- **Space Complexity**: The amount of memory an algorithm uses
- **Big O**: A notation used to describe the upper bound of an algorithm's time complexity
