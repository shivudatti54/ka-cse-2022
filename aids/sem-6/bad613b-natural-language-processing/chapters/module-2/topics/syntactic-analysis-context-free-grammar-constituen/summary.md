# Syntactic Analysis: Context-Free Grammar, Constituency, Top-down and Bottom-up Parsing, CYK Parsing

=====================================================

### Context-Free Grammar (CFG)

- Definition: A set of production rules for generating an infinite number of strings in a specific format
- Notation: A \(\rightarrow\) b, where A is a non-terminal and b is a string
- Example: S \(\rightarrow\) NP VP, where S is the sentence, NP is a noun phrase, and VP is a verb phrase

### Constituency

- Definition: The hierarchical structure of a sentence, where each phrase is a constituent of another phrase
- Types:
  - Phrase structure grammar (PSG)
  - Constituent-based parsing

### Top-down Parsing

- Definition: A parsing algorithm that starts with the overall sentence and recursively breaks it down into smaller phrases
- Algorithm:
  1.  Analyze the sentence as a whole
  2.  Identify the non-terminal phrase
  3.  Replace the non-terminal phrase with its constituent phrases
  4.  Repeat steps 1-3 until the sentence is parsed

### Bottom-up Parsing

- Definition: A parsing algorithm that starts with the individual words and combines them to form phrases
- Algorithm:
  1.  Analyze each word individually
  2.  Identify the phrase boundaries
  3.  Combine the words to form phrases
  4.  Repeat steps 1-3 until the sentence is parsed

### CYK Parsing

- Definition: A parsing algorithm that uses dynamic programming to parse the sentence
- Formulas:
  - α(k) = max(α(k-1), k \* Cα(k-1)(k))
  - α(k) = argmax(α(k-1), k \* Cα(k-1)(k))
- Where α(k) is the maximum score and Cα(k)(k) is the number of ways to parse the sentence up to the k-th position

### Theorems

- **Chomsky's Universal Grammar**: A set of production rules that can generate an infinite number of strings in a specific format
- **Context-Free Grammar Theorem**: A sentence can be parsed in multiple ways using a context-free grammar

### Important Formulas and Definitions

- **First Set (α1)**: The set of all strings generated by the leftmost derivation of a non-terminal
- **Follow Set (α2)**: The set of all strings generated by the rightmost derivation of a non-terminal
- **Derivation Tree**: A tree representing the sequence of production rules applied to generate a sentence
