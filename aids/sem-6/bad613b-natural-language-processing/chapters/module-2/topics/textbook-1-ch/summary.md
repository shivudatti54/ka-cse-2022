# Textbook 1: Ch

## Natural Language Processing

### Word Level Analysis

### 1. Regular Expressions

- Definition: A pattern used to match character combinations
- Types:
  - String matching
  - Position-based matching
- Example: `^hello$` matches "hello" from start to end
- Important Formulas:
  - Regular Expression Matching
- Key Concepts:
  - Character classes
  - Quantifiers
  - Anchors

### 2. Finite-State Automata (FSA)

- Definition: A mathematical model for recognizing patterns
- Types:
  - Deterministic FSA (DFA)
  - Nondeterministic FSA (NFA)
- Important Formulas:
  - DFA transition function
  - NFA transition function
- Key Concepts:
  - States
  - Transitions
  - Accepting states

### 3. Morphological Parsing

- Definition: Decomposition of words into morphemes
- Types:
  - Part-of-speech tagging
  - Dependency parsing
- Important Formulas:
  - Part-of-speech tagging algorithms
  - Dependency parsing algorithms
- Key Concepts:
  - Lexical entries
  - Syntactic categories

### 4. Theoretical Background

- Definition: Mathematical foundation for NLP
- Important Theorems:
  - Turing Machine
  - Chomsky Hierarchy
- Key Concepts:
  - Formal languages
  - Context-free grammars

- Review regular expression syntax
