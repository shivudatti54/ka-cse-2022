# **Layout Relative Layout – Table Layout**

### Introduction

- A Layout Relative Layout is used to position child views relative to the parent view using relative layout parameters.
- A Table Layout is a type of layout that is used to arrange views in a table-like structure.

### Key Features and Benefits

- **Flexibility**: Layout Relative Layout allows you to position views relative to each other, the parent view, or other views.
- **Easy Layout Management**: Table Layout makes it easy to arrange views in a grid-like structure.
- **Simple Code**: Table Layout has a simple code structure, making it easy to implement.

### Important Formulas and Definitions

- **Relative Layout Parameters**: `android:layout_width`, `android:layout_height`, `android:layout_marginLeft`, `android:layout_marginTop`, etc.
- **Table Layout Parameters**: `android:layout_width`, `android:layout_height`, `android:layout_span`, `android:layout_column`, etc.

### Important Theorems

- **Theorem of Relative Layout**: The position of a view is determined by its relative layout parameters.
- **Theorem of Table Layout**: The position of a view is determined by its table layout parameters.

### Revision Points

- Use `android:layout_width` and `android:layout_height` to set the size of a view.
- Use `android:layout_marginLeft` and `android:layout_marginTop` to set the margin of a view.
- Use `android:layout_span` and `android:layout_column` to arrange views in a table-like structure.

### Example Use Cases

- Use Layout Relative Layout to create a simple layout with relative positioning.
- Use Table Layout to create a grid-like layout with easy layout management.

By following these revision points, you'll be able to recall the key features, formulas, and theorems of Layout Relative Layout – Table Layout.
