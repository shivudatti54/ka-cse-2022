# **Mobile Application Development: Introduction to Android OS (Chapters 1 & 2)**

## **Overview**

- The Android operating system is developed by the Open Handset Alliance (OHA), a consortium of companies including Google, HTC, Motorola, and Samsung.
- Android is an open-source OS, allowing customization and modification by device manufacturers.

## **Key Concepts**

### Android Architecture

- **Linux kernel**: The core operating system that provides the foundation for Android.
- **Dalvik Virtual Machine (DVM)**: A virtual machine that runs Android applications.
- **Java**: The primary programming language used for Android app development.

### Android Description

- **API (Application Programming Interface)**: A set of programming interfaces that allow apps to interact with the OS.
- **SDK (Software Development Kit)**: A set of tools, libraries, and documentation that developers use to build Android apps.

## **Important Formulas and Definitions**

- **API level**: The version of the Android API used by an app.
- **Manifest file**: A file that declares an app's components, permissions, and features.
- **Intent**: A message that requests an action from an app component.

### Theorems and Principles

- **KISS (Keep it Simple, Stupid)**: A principle that suggests simplicity and ease of use are essential for good design.
- **Separation of Concerns**: A principle that suggests separating concerns into different components to improve maintainability and scalability.
