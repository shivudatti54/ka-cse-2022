# **Ecosystem, Android Versions, Android Activity, Features of Android, and Android Architecture Stack Linux Kernel**

## **Ecosystem**

- The Open Handset Alliance (OHA) was established in 2008 to promote the development and certification of Android-based devices.
- The Android ecosystem includes:
  - Devices (smartphones, tablets, wearables, etc.)
  - Applications (Apps)
  - Services (Google Play Store, Google Play Services, etc.)
  - Operating System (Android OS)

## **Android Versions**

- Major Android versions:
  - Android 1.0 (2008)
  - Android 2.0 (Eclair, 2009)
  - Android 3.0 (Honeycomb, 2011)
  - Android 4.0 (Ice Cream Sandwich, 2011)
  - Android 5.0 (Lollipop, 2014)
  - Android 6.0 (Marshmallow, 2015)
  - Android 7.0 (Nougat, 2016)
  - Android 8.0 (Oreo, 2017)
  - Android 9.0 (Pie, 2018)
  - Android 10 (Q, 2019)
  - Android 11 (R, 2020)
  - Android 12 (S, 2021)

## **Android Activity**

- Android Activity:
  - A single screen or window in an Android application.
  - Can be a fragment of a larger application.
  - Has a unique instance ID (Activity ID).
  - Has a life cycle (e.g., onCreate(), onStart(), onResume(), onPause(), onStop(), onDestroy())

## **Features of Android**

- Key features:
  - Touchscreen interface
  - Multi-tasking
  - Virtual keyboard
  - Google Play Store
  - Android Pay
  - Google Maps
  - Google Search
  - Android Wear
  - Android Things

## **Android Architecture Stack**

- Android Architecture Components:
  - Room persistence library
  - LiveData
  - ViewModel
  - Navigation Components
  - Data Binding Library
- Android Architecture Stack:
  - MVC (Model-View-Controller)
  - MVVM (Model-View-ViewModel)
  - MVP (Model-View-Presenter)

## **Linux Kernel**

- Key concepts:
  - Monolithic kernel
  - Kernel modules
  - Device drivers
  - Process management
  - Memory management
- Formula: `pid = (ppid * 16) + pid_no`
- Definition: A process ID (PID) is a unique identifier for a process.
- Theorem: The Linux kernel provides a process scheduling algorithm that ensures fairness and efficiency.
