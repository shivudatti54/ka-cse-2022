# Hands-on: Clients and wallets – Geth

## **Introduction**

- Geth is a popular Ethereum client used for development and testing.
- It is a command-line tool that allows users to interact with the Ethereum blockchain.

## **Key Components**

- **Geth**: The Ethereum client used for development and testing.
- **Wallets**: Software programs that allow users to interact with the Ethereum blockchain.
- **Keystore**: A password-protected database that stores user Ethereum accounts.

## **Geth Architecture**

- **Web3.py**: A Python library that provides a high-level API for interacting with the Ethereum blockchain.
- **Web3.js**: A JavaScript library that provides a high-level API for interacting with the Ethereum blockchain.
- **JSON-RPC**: A protocol that allows clients to send requests to the Ethereum blockchain.

## **Key Commands**

- `geth --datadir <path>`: Starts Geth with a specified data directory.
- `web3.py`: Provides a high-level API for interacting with the Ethereum blockchain.
- `json-rpc`: Allows clients to send requests to the Ethereum blockchain.

## **Important Formulas and Definitions**

- **Private Key**: A unique numerical value used to secure an Ethereum account.
- **Public Key**: A unique numerical value used to identify an Ethereum account.
- **Hash**: A unique numerical value used to identify a block on the Ethereum blockchain.
