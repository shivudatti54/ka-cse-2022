# Chapter 4 Revision Notes

### Introduction

- Blockchain is a distributed digital ledger technology
- Decentralized, secure, and transparent record-keeping system

### Types of Blockchain

- **Public Blockchain**: Open to anyone, e.g., Bitcoin, Ethereum
- **Private Blockchain**: Restricted to specific organizations or users
- **Consortium Blockchain**: Limited to a group of organizations or users

### Blockchain Architecture

- **Block**: A collection of transactions
- **Chain**: A sequence of blocks
- **Network**: A group of nodes verifying transactions

### Consensus Mechanisms

- **Proof of Work (PoW)**:
  - Formula: Energy consumption = Hash rate x Block time
  - Theorem: Halving Period
- **Proof of Stake (PoS)**:
  - Formula: Energy consumption = Stake x Block time
  - Theorem: Byzantine Generals' Problem

### Smart Contracts

- **Definition**: Self-executing contracts with the terms of the agreement written directly into code
- **Characteristics**: Immutable, transparent, and tamper-proof

### Cryptographic Techniques

- **Hash Functions**: One-way encryption, e.g., SHA-256
- **Digital Signatures**: Verification of sender's identity and message authenticity

### Important Formulas and Definitions

- **Hash Function**: h(x) = x → a fixed-size string of characters
- **Digital Signature**: σ = (m, r, s) where r = h(m) and s = private key
- **Smart Contract**: self-executing contract with the terms of the agreement written directly into code

### Important Theorems

- **Byzantine Generals' Theorem**: Ensures the consistency of the network
- **Halving Period**: Reduction in block reward every 210,000 blocks
