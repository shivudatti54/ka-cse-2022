# **Summary: 5 and 6 - Knowledge Representation using Predicate Logic**

### Definitions and Notations

- **Predicate Logic**: A formal system for representing knowledge using logical statements and symbols.
- **Predicate**: A function that takes arguments and returns a truth value (True or False).
- **Domain**: The set of all possible values for the arguments of a predicate.
- **Ontology**: A formal representation of knowledge in a specific domain.

### Important Formulas and Theorems

- **Universal Quantification**: ∀x (P(x) → Q(x))
- **Existential Quantification**: ∃x (P(x) ∧ Q(x))
- **Symbolic Representation**: ∀x R(x, y) ∧ ∀x S(x, z) → R(a, z)
- **Modus Ponens**: From P and P → Q, infer Q

### Key Points

- **Predicate Logic Representation**:
  - Representing knowledge using predicate logic formulas.
  - Using predicates, quantifiers, and logical operators to express knowledge.
- **Knowledge Representation Theories**:
  - Description Logics (DL).
  - First-Order Logic (FOL).
- **Key Concepts**:
  - Tautologies and contradictions.
  - Validity and satisfiability.
  - Inference rules and logical operators.

### Important Theorems

- **Löwenheim-Skolem Theorem**: Every countable model of a first-order theory has a countable model.
- **Compactness Theorem**: Every finite subset of a set of first-order sentences has a model.
