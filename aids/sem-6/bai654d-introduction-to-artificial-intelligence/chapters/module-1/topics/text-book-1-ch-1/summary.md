# Text Book 1: Ch 1 - Introduction to Artificial Intelligence

### Module: What is artificial intelligence? Problems, Problem Spaces, and search

**Definitions and Formulas:**

- **Artificial Intelligence (AI):** The development of computer systems able to perform tasks that typically require human intelligence, such as learning, problem-solving, and decision-making.
- **Problem Space:** A set of possible solutions to a problem, represented as a graph or network.
- **Search:** A method for finding a solution to a problem by exploring the problem space.

**Key Concepts:**

- **Artificial Intelligence (AI) Problem Solving:**
  - **Single-Goal Problem:** A problem with a single optimal solution.
  - **Multi-Goal Problem:** A problem with multiple optimal solutions.
- **Types of Problems:**
  - **NP-Complete Problems:** Problems that are computationally intractable.
  - **NP Problems:** Problems for which a solution can be verified in polynomial time.
- **Search Algorithms:**
  - **Breadth-First Search (BFS):** Explores all nodes at a given depth before moving to the next depth.
  - **Depth-First Search (DFS):** Explores as far as possible along each branch before backtracking.

**Important Theorems and Notations:**

- **NP-Completeness Theorem:** A problem is NP-complete if it is in NP and every problem in NP can be reduced to it in polynomial time.
- **Time Complexity Notation:** O(n), Ω(n), and Θ(n) represent time complexity in Big O notation.

**Revision Key Points:**

- AI involves problem-solving, learning, and decision-making.
- Problem spaces are represented as graphs or networks.
- Search algorithms are used to find solutions in problem spaces.
- NP-complete problems are computationally intractable.
