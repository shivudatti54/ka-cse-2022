# Learning Objectives
After studying this topic, you should be able to:
1. Define Context-Free Grammar and identify its four formal components
2. Differentiate between terminal and non-terminal symbols in a CFG
3. Derive sentences using production rules and construct parse trees
4. Identify and explain different types of grammatical ambiguity
5. Convert CFGs to Chomsky Normal Form
6. Apply the CYK algorithm to determine if a string belongs to a CFG language
7. Compare and contrast top-down, bottom-up, and CYK parsing approaches
8. Evaluate the limitations of CFGs for modeling natural language phenomena
