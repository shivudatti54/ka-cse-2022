# Learning Objectives
After studying this topic, you should be able to:
1.  Define a statistical language model and explain its purpose in NLP.
2.  Explain the chain rule of probability and how the Markov assumption simplifies it for N-gram models.
3.  Calculate bigram and trigram probabilities using Maximum Likelihood Estimation (MLE) from a given corpus.
4.  Identify the problem of data sparsity and explain the need for smoothing techniques like Add-One and Backoff.
5.  Define perplexity and interpret what a high or low value implies about a model's quality.
6.  Describe the relevance of the Paninian framework to modeling morphologically rich languages like many Indian languages.
