## **8.1: Introduction to Hadoop**

### What is Hadoop?

Hadoop is an open-source, distributed computing framework that enables processing and storing large amounts of data across a cluster of computers. It is designed to handle massive amounts of data across multiple servers, making it ideal for big data analytics.

### Key Features of Hadoop:

- **Distributed Processing**: Hadoop allows for parallel processing of large datasets across multiple nodes.
- **Scalability**: Hadoop is designed to scale horizontally, making it suitable for large datasets.
- **Flexibility**: Hadoop supports various data formats and can be used with different programming languages.

### Advantages of Hadoop:

- **Cost-Effective**: Hadoop is open-source and free, making it an affordable solution for big data analytics.
- **Scalability**: Hadoop can handle large datasets and scale horizontally.
- **Flexibility**: Hadoop can be used with various data formats and programming languages.

### Disadvantages of Hadoop:

- **Complexity**: Hadoop can be complex to set up and manage.
- **Security**: Hadoop requires proper security measures to protect sensitive data.
- **Performance**: Hadoop can be slower than traditional databases for small datasets.

### Example Use Case:

- Analyzing social media data to understand customer behavior.
- Processing large datasets for data science and machine learning applications.

## **8.3: Why Not RDBMS?**

### Comparison of Hadoop and RDBMS

Hadoop and RDBMS (Relational Database Management System) are both used for data storage and analysis. However, they have different design goals and are suited for different use cases:

- **RDBMS**:
  - Optimized for small to medium-sized datasets.
  - Provides fast query performance and transaction support.
  - Limited scalability and flexibility.
- **Hadoop**:
  - Optimized for large-scale data processing and storage.
  - Provides high scalability and flexibility.
  - Limited query performance and transaction support.

## **8.5: History of Hadoop**

### Evolution of Hadoop

Hadoop has undergone significant evolution since its inception:

- **Hadoop 0.1**: Hadoop's first version, released in 2006.
- **Hadoop 0.9**: Released in 2007, this version introduced MapReduce and a distributed processing framework.
- **Hadoop 1.0**: Released in 2009, this version introduced HDFS (Hadoop Distributed File System) and improved performance.
- **Hadoop 2.0**: Released in 2012, this version introduced YARN (Yet Another Resource Negotiator) and improved scalability.
- **Hadoop 3.0**: Released in 2016, this version introduced improved performance and scalability.

### Key Milestones in Hadoop's Evolution:

- **Apache Hadoop**: Hadoop was donated to the Apache Software Foundation in 2006.
- **Open Source**: Hadoop is open-source, allowing for community contributions and collaboration.
- **Commercial Support**: Hadoop offers commercial support options for businesses and organizations.
