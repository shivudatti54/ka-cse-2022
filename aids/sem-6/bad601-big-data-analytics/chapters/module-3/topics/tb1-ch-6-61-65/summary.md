# **TB1: Ch 6: 6.1-6.5 - Big Data Analytics**

### Key Points

- **What is Big Data Analytics?**
  - The process of extracting insights and knowledge from large and complex datasets.
  - Involves various techniques, including data mining, predictive analytics, and data visualization.

- **What is MongoDB?**
  - NoSQL database designed for handling large amounts of data.
  - Documents-based database, where each document represents a single record.
  - MongoDB is a popular choice for big data analytics due to its scalability and high performance.

- **Terms used in RDBMS and MongoDB**
  - **ACID Compliance**: Atomicity, Consistency, Isolation, Durability (ensures database transactions are processed reliably).
  - **Data Modeling**: Creation of database schema to represent data structure and relationships.
  - **Indexing**: Optimizes query performance by creating a data structure that facilitates fast data retrieval.

- **Data Types in MongoDB**
  - **Document**: A single record in MongoDB, represented as a JSON-like object.
  - **Field**: A single key-value pair within a document.
  - **Collection**: A group of related documents in MongoDB.

- **Data Storage in MongoDB**
  - **Document-Oriented**: Each document is stored as a single unit.
  - **Schemaless**: No predefined schema; documents can have varying fields.
  - **Distributed Storage**: Data is spread across multiple servers for high scalability.

- **Data Retrieval in MongoDB**
  - **Query Language**: MongoDB Query Language (MQL) is used to retrieve data.
  - **Cursors**: A mechanism to retrieve data in a streaming fashion.

- **Big Data Analytics with MongoDB**
  - **Data Ingestion**: The process of collecting and processing large amounts of data.
  - **Data Processing**: The application of algorithms and techniques to extract insights from data.
  - **Data Storage**: The storage of processed data for future analysis and insights.

- **Important Formulas, Definitions, and Theorems**
  - **NoSQL Database Formula**: NoSQL databases use a variety of data models, including NoSQL, semi-structured, and semi-relational databases.
  - **Big Data Analytics Definition**: Big data analytics is the process of extracting insights and knowledge from large and complex datasets.
  - **ACID Compliance Theorem**: Ensures database transactions are processed reliably and securely.
