# Learning Objectives
After studying this topic, you should be able to:
1. Identify and describe the six core components of a SIEM system and their functions
2. Compare and contrast different SIEM deployment models (on-premises, cloud, hybrid)
3. Explain the process of log collection, normalization, and correlation in SIEM architecture
4. Analyze the scalability and performance considerations in SIEM design
5. Describe how SIEM systems integrate with other security tools in a modern SOC
6. Evaluate appropriate SIEM architectural choices based on organizational requirements
