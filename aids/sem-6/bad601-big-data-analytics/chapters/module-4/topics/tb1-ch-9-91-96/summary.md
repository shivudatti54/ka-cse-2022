# **TB1: Ch 9: 9.1-9.6 - Big Data Analytics**

### Introduction to Hive

- Hive is a data warehousing and SQL-like query language for Hadoop.
- It allows users to analyze data stored in Hadoop using SQL-like syntax.

### Hive Architecture

- **Architecture**: Hive consists of a metastore, a query compiler, and a execution engine.
- **Components**:
  - **Metastore**: stores metadata about the data in Hadoop.
  - **Query Compiler**: compiles Hive queries into MapReduce jobs.
  - **Execution Engine**: executes the MapReduce jobs on Hadoop.

### Hive Data Types

- **Data Types**:
  - **Integer**: 32-bit integer.
  - **String**: string values.
  - **Boolean**: boolean values.
  - **Timestamp**: timestamp values.
  - **Date**: date values.
  - **Time**: time values.
  - **Decimal**: decimal values.
  - **Array**: array values.
  - **Map**: map values.

### Hive File Formats

- **File Formats**:
  - **Text**: plain text files.
  - **ORC**: optimized row columnar format.
  - **PARQUET**: parallelized row columnar format.
  - **RCFILE**: row columnar format.

### Hive Query

- **Query Syntax**: similar to SQL syntax.
- **Query Types**:
  - **SELECT**: selects data from tables.
  - **INSERT**: inserts data into tables.
  - **UPDATE**: updates data in tables.
  - **DELETE**: deletes data from tables.

### Important Formulas, Definitions, Theorems

- **DBSCAN (Density-Based Spatial Clustering of Applications with Noise)**: a clustering algorithm.
- **K-Means (K-Means Clustering)**: a clustering algorithm.
- **Principal Component Analysis (PCA)**: a dimensionality reduction technique.
- **Hive Query Optimization**: optimizes Hive queries for better performance.
