#### Introduction to Hive

- **What is Hive?**
  - Hive is a data warehousing and SQL-like query language for Hadoop.
  - It provides a way to analyze data stored in Hadoop Distributed File System (HDFS).
- **Hive Architecture**
  - Hive Metastore: stores metadata about the data in HDFS.
  - Hive Executive: executes queries and manages the execution of queries.
  - Hive Query Compiler: compiles Hive queries into MapReduce jobs.
  - Hive Query Optimizer: optimizes the compiled queries.
  - Hive Query Executor: executes the optimized queries.
