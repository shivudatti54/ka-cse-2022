## Introduction

---

MapReduce is a programming model used for processing large data sets in parallel across a cluster of computers. It was designed to handle the challenges of large-scale data processing, particularly in the context of big data analytics. In this topic, we will focus on developing a MapReduce program to find the tags associated with each movie by analyzing MovieLens data.

## Historical Context

---

The concept of big data analytics dates back to the early 2000s, when the amount of data generated by humans and machines began to grow exponentially. As the data volume increased, traditional data processing systems became inadequate to handle the complexity of the data. MapReduce was introduced in 2004 by Google to address these challenges.

MapReduce was designed to work with large data sets by breaking them down into smaller chunks, processing each chunk in parallel, and then combining the results. This approach enabled Google to efficiently process vast amounts of data and improve its search results.

## Big Data Analytics with MapReduce

---

MapReduce is a programming model that consists of two main components:

1.  **Mapper**: The mapper is responsible for processing the input data in parallel across a cluster of computers. It breaks down the data into smaller chunks, processes each chunk, and generates a key-value pair for each chunk.
2.  **Reducer**: The reducer is responsible for combining the key-value pairs generated by the mapper. It aggregates the data, performs calculations, and generates the final output.

## MovieLens Data

---

MovieLens is a publicly available dataset that contains information about movies, ratings, and users. It is commonly used for movie recommendation systems and other big data analytics applications.

The MovieLens dataset consists of the following columns:

- `movieId`: Unique identifier for each movie
- `rating`: Rating given by a user to a movie
- `userId`: Unique identifier for each user
- `timestamp`: Timestamp of the rating

## Tag Association Task

---

The tag association task involves finding the tags associated with each movie by analyzing the MovieLens data. In this task, we need to:

- Extract the movie IDs and corresponding ratings from the data
- Identify the tags (e.g., genres, directors, actors) associated with each movie
- Group the movies by their tags and count the number of movies associated with each tag

## MapReduce Program Implementation

---

Here is a sample MapReduce program implementation in Java:

```java
import org.apache.hadoop.conf.Configuration;
import org.apache.hadoop.fs.Path;
import org.apache.hadoop.io.IntWritable;
import org.apache.hadoop.io.Text;
import org.apache.hadoop.mapreduce.Job;
import org.apache.hadoop.mapreduce.Mapper;
import org.apache.hadoop.mapreduce.Reducer;
import org.apache.hadoop.mapreduce.lib.input.FileInputFormat;
import org.apache.hadoop.mapreduce.lib.output.FileOutputFormat;

public class TagAssociation {
    public static void main(String[] args) throws Exception {
        // Create a new Hadoop configuration
        Configuration conf = new Configuration();

        // Set the input and output paths
        String inputPath = "input/";
        String outputPath = "output/";

        // Create a new job
        Job job = Job.getInstance(conf, "TagAssociation");

        // Set the input and output paths
        FileInputFormat.addInputPath(job, new Path(inputPath));
        FileOutputFormat.setOutputPath(job, new Path(outputPath));

        // Define the mapper class
        job.setMapperClass(Mapper.class);

        // Define the reducer class
        job.setReducerClass(Reducer.class);

        // Define the key and value classes
        job.setOutputKeyClass(Text.class);
        job.setOutputValueClass(IntWritable.class);

        // Run the job
        System.exit(job.waitForCompletion(true) ? 0 : 1);
    }
}

class Mapper extends Mapper<Text, Text, Text, IntWritable> {
    @Override
    public void map(Text key, Text value, Context context) throws IOException, InterruptedException {
        // Extract the movie ID and rating
        String[] movieIdAndRating = value.toString().split(" ");
        int movieId = Integer.parseInt(movieIdAndRating[0]);
        int rating = Integer.parseInt(movieIdAndRating[1]);

        // Generate the key-value pair
        context.write(new Text(movieId), new IntWritable(rating));
    }
}

class Reducer extends Reducer<Text, IntWritable, Text, IntWritable> {
    @Override
    public void reduce(Text key, Iterable<IntWritable> values, Context context) throws IOException, InterruptedException {
        // Group the movies by their tags
        int tagCount = 0;
        for (IntWritable value : values) {
            tagCount++;
        }

        // Generate the output
        context.write(key, new IntWritable(tagCount));
    }
}
```
