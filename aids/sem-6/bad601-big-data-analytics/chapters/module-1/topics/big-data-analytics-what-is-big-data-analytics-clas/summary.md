# **Big Data Analytics: Revision Notes**

### Definition and Characteristics of Big Data

- **Definition:** Big Data refers to the large, complex, and diverse sets of data that are difficult to process and analyze using traditional data processing tools.
- **Characteristics:**
  - **Volume:** Large amounts of data
  - **Velocity:** High speed of data generation and processing
  - **Variety:** Different types of data (structured, semi-structured, unstructured)
  - **Veracity:** Low accuracy and quality of data

### Classification of Analytics

- **Descriptive Analytics:** Analyzes past data to understand what happened (e.g., sales data)
- **Diagnostic Analytics:** Analyzes past data to identify the reason for a problem (e.g., identifying causes of low sales)
- **Predictive Analytics:** Analyzes past data to predict future outcomes (e.g., forecasting sales)
- **Prescriptive Analytics:** Analyzes past data and predicts future outcomes to recommend actions (e.g., recommending marketing campaigns)

### Importance of Big Data Analytics

- **Improved decision-making:** Enables organizations to make data-driven decisions
- **Increased revenue:** Helps organizations identify new revenue streams and opportunities
- **Competitive advantage:** Provides organizations with a competitive advantage in the market
- **Enhanced customer experience:** Enables organizations to personalize customer experiences

### Technologies Used in Big Data Environment

- **Hadoop:** An open-source framework for processing large datasets
- **NoSQL databases:** Databases that store data in a flexible, scalable manner (e.g., MongoDB, Cassandra)
- **Cloud computing:** Cloud-based infrastructure for storing and processing large datasets (e.g., Amazon Web Services, Microsoft Azure)
- **Machine learning algorithms:** Algorithms that enable organizations to automatically learn from data (e.g., decision trees, clustering algorithms)

### Formulas and Theorems

- **Bayes' theorem:** A mathematical formula used in predictive analytics to update probabilities based on new data
- **Information theory:** A branch of mathematics that deals with the quantification of information
- **Data compression:** A technique used to reduce the size of data while preserving its content
