# **Big Data Analytics Study Material**

## **Module: Classification of Data**

### 1.1: Characteristics of Big Data

- **Volume**: Large amount of data
- **Velocity**: High speed of data generation and consumption
- **Variety**: Different types of data such as structured, semi-structured, and unstructured
- **Veracity**: Uncertainty and variability in data quality
- **Value**: Potential to extract valuable insights

### 2.1-2.5: Classification of Big Data

- **Structured Data**: Organized and formatted data (e.g., databases, spreadsheets)
  - **Example**: Customer information, orders, and inventory
- **semi-structured Data**: Partially organized data (e.g., XML, JSON)
  - **Example**: Log files, configuration files, and sensor data
- **Unstructured Data**: Unorganized and unformatted data (e.g., text, images, videos)
  - **Example**: Social media posts, emails, and audio recordings
- **Sensed Data**: Data from sensors and IoT devices
  - **Example**: Temperature, humidity, and motion sensor data
- **Social Data**: Data from social media and online interactions
  - **Example**: Social media posts, tweets, and online reviews

### 2.7: Types of Big Data

- **Relational Data**: Structured data stored in relational databases
- **Document-Oriented Data**: Semi-structured data stored in document-oriented databases
- **Key-Value Data**: NoSQL databases that store data as key-value pairs
- **Column-Oriented Data**: Data stored in column-oriented databases
- **Graph Data**: Data stored in graph databases

### 2.9-2.11: Big Data Sources

- **Log Files**: Data from application logs, server logs, and network logs
- **Sensor Data**: Data from sensors, IoT devices, and wearables
- **Social Media**: Data from social media platforms, online interactions, and community forums
- **Transactional Data**: Data from e-commerce websites, online stores, and financial transactions
- **IoT Data**: Data from Internet of Things devices, smart homes, and smart cities

### 2.12: Big Data Venues

- **Cloud Storage**: Cloud-based storage services (e.g., AWS S3, Google Cloud Storage)
- **Data Warehouses**: Centralized repositories for storing and analyzing data (e.g., Amazon Redshift, Google BigQuery)
- **NoSQL Databases**: Non-relational databases (e.g., MongoDB, Cassandra)
- **Big Data Platforms**: Platforms for processing and analyzing big data (e.g., Hadoop, Spark)
- **Data Lakes**: Central repositories for storing raw, unprocessed data

### 3.2: Big Data Analytics

- **Descriptive Analytics**: Analyzing historical data to understand business patterns
- **Diagnostic Analytics**: Analyzing data to identify root causes of problems
- **Predictive Analytics**: Analyzing data to predict future trends and outcomes
- **Prescriptive Analytics**: Analyzing data to provide recommendations and actions

### 3.5: Big Data Mining Techniques

- **Supervised Learning**: Training models on labeled data to make predictions
- **Unsupervised Learning**: Discovering patterns and relationships in unlabeled data
- **Reinforcement Learning**: Training models to make decisions based on rewards and penalties
- **Clustering**: Grouping similar data points together
- **Association Rule Mining**: Discovering relationships between data points

### 3.8: Big Data Data Visualization

- **Types of Visualizations**: Bar charts, pie charts, scatter plots, heat maps
- **Best Practices**: Using color, size, and position to convey insights
- **Storytelling**: Using visualizations to tell a story and convey insights

### 3.12: Cloud Computing for Big Data

- **Public Cloud**: Sharing resources with other users (e.g., AWS, Google Cloud)
- **Private Cloud**: Housing resources for a single organization
- **Hybrid Cloud**: Combining public and private clouds
- **Cloud-based Big Data**: Running big data workloads on cloud infrastructure

## **Module: Evolution of Big Data**

### 4.1: History of Big Data

- **1950s-1960s**: The origins of big data in mainframe computing
- **1970s-1980s**: The rise of relational databases
- **1990s-2000s**: The emergence of NoSQL databases and data warehousing
- **2010s-present**: The growth of big data and its applications

### 4.2: Current Trends and Future Directions

- **Artificial Intelligence**: Using AI to extract insights from big data
- **Internet of Things**: Using IoT data to build smart cities and homes
- **Blockchain**: Using blockchain to secure and verify big data
- **Edge Computing**: Processing big data at the edge of the network
- **Data Governance**: Ensuring the quality, security, and compliance of big data
