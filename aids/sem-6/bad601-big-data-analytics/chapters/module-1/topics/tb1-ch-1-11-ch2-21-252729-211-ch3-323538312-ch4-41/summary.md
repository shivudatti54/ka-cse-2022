# **BIG DATA ANALYTICS REVISION NOTES**

## **CLASSIFICATION OF DATA**

- **Definition**: Classification of data is the process of organizing data into categories or groups based on shared characteristics.
- **Types of Classification**:
  - Hierarchical classification
  - Non-hierarchical classification
  - Categorical classification

## **CHARACTERISTICS OF BIG DATA**

- **Definition**: Big data refers to the large volume, variety, and velocity of data that organizations collect and analyze.
- **Characteristics**:
  - Volume: Large amounts of data
  - Variety: Different types of data sources (structured, semi-structured, unstructured)
  - Velocity: High-speed data generation and processing

## **DEFINITION AND EVOLUTION OF BIG DATA**

- **Definition**: Big data is a term used to describe the large amounts of structured, semi-structured, and unstructured data that organizations collect and analyze.
- **Evolution**:
  - Early days: Focus on structured data
  - Mid-period: Emergence of unstructured data
  - Present day: Big data encompasses all types of data

## **FORMULAS AND DEFINITIONS**

- **Data Volume Formula**: V = Size of data
- **Data Variety Formula**: V = Variety of data sources
- **Data Velocity Formula**: V = Speed of data generation and processing
- **Big Data Definition**: Big data refers to the large volume, variety, and velocity of data that organizations collect and analyze.

## **THEOREMS**

- **No Free Lunch Theorem**: No algorithm can achieve optimal performance across all possible input data distributions.
- **Law of Accelerating Returns**: Exponential growth in computing power and data storage will lead to exponential growth in innovation.
