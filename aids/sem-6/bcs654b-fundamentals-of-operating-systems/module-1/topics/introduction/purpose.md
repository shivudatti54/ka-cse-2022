# Learning Objectives

After studying this topic, you should students able to:

1. Define an operating system and explain its role as an intermediary between users and hardware
2. Describe what an operating system does from both the user view and system view perspectives
3. Explain the basic computer system organization including CPU, memory, and I/O devices
4. Understand how interrupt-driven I/O and DMA work in computer system organization
5. Identify the different types of operating systems (batch, time-sharing, distributed, real-time)
6. Explain the goals of an operating system: convenience, efficiency, and ability to evolve
