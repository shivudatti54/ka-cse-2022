# **Directory Structure, Protection, File System Implementation**

## **File System Structure**

- A file system is a collection of files and directories that are stored on a storage device.
- A file system is organized into:
  - Root directory (also known as the boot directory)
  - Directories (also known as subdirectories)
  - Files
- Types of file systems:
  - Hierarchical file system
  - Amortized file system

## **File System Operations**

- File system operations:
  - Creation (create a new file or directory)
  - Deletion (delete a file or directory)
  - Reading (read the contents of a file)
  - Writing (write to a file)
  - Modification (modify the contents of a file)
  - Rename (rename a file or directory)
  - Move (move a file or directory to a different location)

- File system internals:
  - File system hierarchy
  - File system layouts
  - File system structures
  - File system algorithms

## **Important Definitions and Formulas**

- **Inode**: a data structure that contains information about a file or directory
- **Block**: a unit of storage on a storage device
- **File allocation table (FAT)**: a table that maps file names to inode numbers
- **Directory entry**: a data structure that contains information about a file or directory
- **File system metric**: a measure of the performance or efficiency of a file system (e.g. throughput, IOPS)

## **Important Theorems**

- **Kilworth's Theorem**: describes the relationship between the size of a file system and the number of files it can support.
- **Garbage collection theorem**: describes the relationship between the frequency of garbage collection and the overall performance of a file system.

## **Key Concepts**

- **File system consistency**: the property of a file system that ensures that all files and directories are in a consistent state.
- **File system redundancy**: the property of a file system that ensures that data can be recovered in case of failures or errors.
- **File system performance**: the measure of how efficiently a file system can handle requests and operations.
