### 5.1 Fundamentals of Operating Systems

#### Revision Notes

**Key Concepts:**

- **Definition of an Operating System (OS):**
  - A software that manages computer hardware resources
  - Provides common services to computer programs
- **Operating System Components:**
  - Kernel (central part)
  - Device drivers
  - System libraries
  - Utilities
- **Operating System Functions:**
  - Process management
  - Memory management
  - File management
  - Input/Output management

**Important Formulas and Definitions:**

- **CPU Time:** Time spent by CPU in executing a process
- **System Call:** Request for a service from an OS
- **Context Switching:** Switching between two or more processes
- **Page Fault:** Occurs when a process accesses a page that is not in memory

**Important Theorems:**

- **Thomson's Lamp Post Theorem:** A CPU must be able to execute a sequence of operations in the order that the program specifies
- **Goldberg's Theorem:** A CPU must be able to handle interrupts in a predictable manner

**Revision Tips:**

- Understand the differences between various OS components
- Familiarize yourself with key OS functions and their importance
- Practice calculating CPU time and understanding context switching
- Review the context of page faults and system calls
