# **FUNDAMENTALS OF OPERATING SYSTEMS: REVISION NOTES**

## **Chapter 1: Introduction to Operating Systems**

- **1.1: Definition of an Operating System**
  - An operating system (OS) is a software that manages computer hardware resources and provides common services to computer programs.
  - Examples: Windows, macOS, Linux

- **1.2: Characteristics of an Operating System**
  - 1.2.1: Organization
    - Hierarchical organization
    - Modular design
  - 1.2.2: Functionality
    - Process management
    - Memory management
    - Input/Output management
    - File management

- **1.3: History of Operating Systems**
  - Early operating systems: CP-67, CTSS
  - Development of modern operating systems: Unix, Windows

- **1.4: Types of Operating Systems**
  - 1.4.1: Single-user, single-tasking
  - 1.4.2: Single-user, multi-tasking
  - 1.4.3: Multi-user, single-tasking
  - 1.4.4: Multi-user, multi-tasking

- **1.5: Operating System Interface**
  - Command-line interface (CLI)
  - Graphical user interface (GUI)
  - 1.5.1: Command-line interface (CLI)
    - Syntax
    - Piping and redirection
  - 1.5.2: Graphical user interface (GUI)
    - Windows, icons, menus
