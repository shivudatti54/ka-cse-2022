# **Textbook 1: Chapter 1: Fundamentals of Operating Systems**

## **1.1: What is an Operating System?**

An operating system (OS) is a software that manages computer hardware resources and provides services to computer programs. It acts as an intermediary between computer hardware and user-level applications, controlling the allocation of system resources such as CPU time, memory, and storage.

## **History of Operating Systems**

The first operating system was CTSS (Compatible Time-Sharing System), developed in the 1960s at the Massachusetts Institute of Technology (MIT). It was a time-sharing system that allowed multiple users to access the computer simultaneously.

In the 1970s, the first commercial operating systems were developed, including CP-40 and Kelvin. The first widely used operating system was MS-DOS, released in 1981 for IBM PCs.

## **Modern Operating Systems**

Today, operating systems are highly complex software systems that provide a wide range of services, including process management, memory management, file systems, and input/output operations.

## **1.2: Computer System Organization**

A computer system is composed of several components, including:

- **Hardware**: The physical components of the computer system, such as the central processing unit (CPU), memory, and storage devices.
- **Software**: The programs that run on the computer system, including the operating system and application software.

The operating system plays a critical role in managing the computer system's hardware resources and providing services to application software.

## **System Components**

1. **Central Processing Unit (CPU)**: The CPU is the brain of the computer system, responsible for executing instructions and performing calculations.
2. **Memory (RAM)**: Random Access Memory (RAM) is a type of memory that temporarily stores data and program instructions while the CPU is executing them.
3. **Storage Devices**: Storage devices, such as hard drives and solid-state drives, provide permanent storage for data and programs.
4. **Input/Output Devices**: Input/output devices, such as keyboards, mice, and monitors, allow users to interact with the computer system.

## **1.3: Operating System Functions**

An operating system performs several critical functions, including:

- **Process Management**: The operating system manages the creation, execution, and termination of processes (programs) running on the computer system.
- **Memory Management**: The operating system manages the allocation and deallocation of memory for processes.
- **File Systems**: The operating system provides a file system that allows programs to store and retrieve data.
- **Input/Output Operations**: The operating system manages input/output operations between devices and processes.

## **1.4: Operating System Concepts**

Several key concepts are essential to understanding operating systems, including:

- **Process**: A process is a program in execution, including its program counter, registers, and memory.
- **Thread**: A thread is a lightweight process that shares the same memory space and resources.
- **Context Switching**: Context switching is the process of switching from one process to another, requiring the operating system to save and restore the state of each process.
- **Deadlock**: Deadlock is a situation where two or more processes are blocked indefinitely, waiting for each other to release resources.

## **1.5: Operating System Types**

Several types of operating systems exist, including:

- **Single-User, Single-Tasking**: This type of operating system allows only one user to run one program at a time.
- **Multi-User, Multi-Tasking**: This type of operating system allows multiple users to run multiple programs simultaneously.
- **Real-Time Operating System**: This type of operating system is designed for embedded systems that require predictable and fast responses to events.
- **Mobile Operating System**: This type of operating system is designed for mobile devices, such as smartphones and tablets.
