# Learning Objectives

After studying this topic, you should students able to:

1. Explain the need for CPU scheduling and the role of the scheduler and dispatcher.
2. Differentiate between preemptive and non-preemptive scheduling and identify the circumstances that cause scheduling decisions.
3. Calculate key performance metrics (CPU utilization, throughput, turnaround time, waiting time, response time) for a given set of processes and schedule.
4. Describe the operation, advantages, and disadvantages of common scheduling algorithms: FCFS, SJF, Priority, and Round Robin.
5. Construct Gantt charts and compute performance metrics for various scheduling scenarios.
6. Explain the concept of thread scheduling and the difference between Process Contention Scope (PCS) and System Contention Scope (SCS).
