# Learning Objectives

After studying this topic, you should be able to:

1.  Define and explain the purpose of a structure as a composite data type.
2.  Declare a structure type and create variables of that type using various methods.
3.  Access and manipulate the members of a structure using the correct operator.
4.  Initialize a structure variable correctly at the time of declaration.
5.  Differentiate between structures and unions based on their memory usage and purpose.
6.  Create and use an array of structures to manage multiple records.
7.  Explain why the size of a structure might not equal the sum of its members' sizes.
