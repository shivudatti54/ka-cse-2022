# Machine Learning

### 10.9-10.11: Bayesian Learning

#### Key Points

- **Bayesian Learning**: A probabilistic approach to machine learning, based on Bayes' theorem.
- **Bayes' Theorem**: P(A|B) = P(B|A) \* P(A) / P(B)
- **Conditional Probability**: P(A|B) = P(A and B) / P(B)
- **Posterior Distribution**: P(A|B), represents the updated probability of A given B.
- **Prior Distribution**: P(A), represents the initial probability of A.
- **Likelihood Function**: P(B|A), represents the probability of B given A.

#### Important Formulas

- Bayes' Theorem: P(A|B) = P(B|A) \* P(A) / P(B)

#### Definitions

- **Bayesian Model**: A probabilistic model that incorporates prior knowledge and updates it with new data.
- **Model Uncertainty**: The uncertainty in the model's parameters or predictions.

#### Theorems

- **Bayes' Theorem**: A mathematical formula for updating the probability of a hypothesis based on new data.
- **Kalman Filter**: A mathematical algorithm for estimating the state of a system from noisy data.

#### Key Concepts

- **Inference**: The process of drawing conclusions from data.
- **Model Selection**: The process of selecting the best model for a given problem.
- **Hyperparameter Tuning**: The process of adjusting model hyperparameters for better performance.
