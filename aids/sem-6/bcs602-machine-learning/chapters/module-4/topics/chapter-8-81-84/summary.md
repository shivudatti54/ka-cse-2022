### Machine Learning: Bayesian Learning

#### 8.1 Introduction to Bayesian Learning

- Bayesian learning is a probabilistic approach to machine learning
- It uses Bayes' theorem to update the probability of a hypothesis based on new data

#### 8.2 Fundamentals of Bayes Theorem

- **Bayes' Theorem**: P(H|D) = P(D|H) \* P(H) / P(D)
  - P(H): Prior probability of hypothesis H
  - P(D|H): Likelihood of data D given hypothesis H
  - P(H|D): Posterior probability of hypothesis H given data D
  - P(D): Marginal probability of data D

#### 8.3 Conditional Probability

- **Conditional Probability Formula**: P(A|B) = P(A and B) / P(B)
- **De Morgan's Law**: P(A or B) = 1 - P(A and B)

#### 8.4 Bayes' Theorem Variants

- **Naive Bayes**: Assumes independence between features
- **Kalman Filter**: A recursive algorithm for estimating the state of a system from noisy measurements
