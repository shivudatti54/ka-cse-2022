# Learning Objectives
After studying this topic, you should be able to:
1. Explain the fundamental concepts behind weighted KNN and nearest centroid classifiers
2. Differentiate between standard KNN, weighted KNN, and nearest centroid approaches
3. Apply inverse distance weighting and other weighting schemes in KNN
4. Calculate class centroids and implement nearest centroid classification
5. Analyze the time and space complexity of both algorithms
6. Evaluate when to use each algorithm based on problem characteristics and constraints
7. Implement feature scaling techniques appropriate for distance-based algorithms
8. Compare and contrast the advantages and limitations of both approaches
