# Learning Objectives

After studying this topic, you should be able to:

1. Define regression analysis and distinguish it from classification in machine learning
2. Identify the dependent and independent variables in a regression problem
3. Explain the concept of the line of best fit and the Ordinary Least Squares method
4. Describe the different types of regression including simple, multiple, and polynomial regression
5. Interpret the coefficient of determination (R²) and understand its role in model evaluation
6. State and explain the four key assumptions underlying linear regression
7. Apply regression analysis to solve simple numerical problems
8. Evaluate regression models using residual analysis and identify signs of overfitting
