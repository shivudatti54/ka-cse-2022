# Learning Objectives

After studying this topic, you should be able to:

1. DEFINE linear regression and distinguish between simple and multiple linear regression variants, explaining when each is appropriate in practical applications.

2. DERIVE the ordinary least squares solution for simple linear regression, calculating the slope and intercept using the appropriate formulas with provided data.

3. EXPLAIN the concept of the cost function in linear regression, specifically understanding Mean Squared Error (MSE) and its role in model optimization.

4. IMPLEMENT gradient descent algorithm for linear regression, including the computation of gradients and parameter updates in each iteration.

5. VERIFY whether the assumptions of linear regression (linearity, independence, homoscedasticity, normality) hold for a given dataset through visual and statistical analysis.

6. EVALUATE linear regression models using appropriate metrics including R-squared, adjusted R-squared, RMSE, and MAE, interpreting what each metric reveals about model performance.

7. APPLY multiple linear regression to problems with multiple predictors, constructing the model equation and making predictions for new input values.

8. ANALYZE real-world datasets to determine when linear regression is an appropriate modeling technique and when alternative approaches might be necessary.
