# Learning Objectives

After studying this topic, you should be able to:

1. Define and explain the basic components of a decision tree, including root node, internal nodes, and leaf nodes
2. Understand the concept of node splitting and identify the criteria used to determine the best split (Entropy, Information Gain, Gini Impurity)
3. Apply the Information Gain formula to calculate which feature should be used for splitting at each node
4. Compare and contrast major decision tree algorithms: ID3, C4.5, and CART
5. Explain the problem of overfitting in decision trees and describe techniques to prevent it (pruning, setting depth limits)
6. Analyze how decision trees handle both categorical and continuous features
7. Build a simple decision tree for a given dataset and trace the classification process for new instances
8. Evaluate the advantages and limitations of decision tree learning in practical applications
