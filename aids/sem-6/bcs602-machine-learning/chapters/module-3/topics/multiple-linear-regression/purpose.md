# Learning Objectives

After studying this topic, you should be able to:

1. Define multiple linear regression and explain its difference from simple linear regression
2. Formulate the mathematical model for multiple linear regression with k predictor variables
3. List and explain the five classical assumptions of multiple linear regression
4. Apply the Ordinary Least Squares (OLS) method to estimate regression coefficients
5. Interpret regression coefficients in context, including the ceteris paribus assumption
6. Calculate and interpret R² and adjusted R² to evaluate model fit
7. Detect multicollinearity using Variance Inflation Factor (VIF)
8. Conduct hypothesis tests for individual coefficients and overall model significance
9. Identify limitations of multiple linear regression and appropriate use cases
10. Implement multiple linear regression using software tools for real-world predictive modeling
