### Machine Learning Revision Notes

#### Chapter-4 (4.2-4.5)

- **4.2: Supervised Learning**
  - Definition: Learning from labeled data
  - Types: Regression, Classification
  - Cost Function: Mean Squared Error (MSE)
  - Optimization: Gradient Descent
- **4.3: Unsupervised Learning**
  - Definition: Learning without labeled data
  - Types: Clustering, Dimensionality Reduction
  - Algorithms: K-Means, Principal Component Analysis (PCA)
- **4.4: Model Evaluation**
  - Metrics: Accuracy, Precision, Recall, F1 Score
  - Evaluation Methods: Cross-Validation, Confusion Matrix
- **4.5: Model Selection**
  - Criteria: Bias-Variance Tradeoff, Overfitting, Underfitting

#### Chapter-5 (5.1-5.3, 5.5-5.7)

- **5.1: Linear Regression**
  - Definition: Simple linear regression model
  - Cost Function: Mean Squared Error (MSE)
  - Optimization: Ordinary Least Squares (OLS)
- **5.2: Logistic Regression**
  - Definition: Binary classification model
  - Cost Function: Cross-Entropy Loss
  - Optimization: Gradient Descent
- **5.3: Neural Networks**
  - Definition: Multi-layered feedforward network
  - Cost Function: Mean Squared Error (MSE) or Cross-Entropy Loss
  - Optimization: Backpropagation
- **5.5: Decision Trees**
  - Definition: Tree-based classification model
  - Algorithms: ID3, CART
- **5.6: Random Forests**
  - Definition: Ensemble learning technique
  - Algorithms: Bagging, Boosting
- **5.7: Support Vector Machines (SVMs)**
  - Definition: Non-linear classification model
  - Cost Function: Soft Margin Loss

#### Chapter-6 (6.1, 6.2)

- **6.1: Overfitting vs Underfitting**
  - Definition: Overfitting: model fits noise, Underfitting: model fails to fit data
  - Techniques: Regularization, Dimensionality Reduction
- **6.2: Model Selection Criteria**
  - Criteria: Akaike Information Criterion (AIC), Bayesian Information Criterion (BIC)
