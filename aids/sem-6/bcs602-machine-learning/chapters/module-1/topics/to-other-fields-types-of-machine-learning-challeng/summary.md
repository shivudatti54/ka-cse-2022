# **Machine Learning Revision Notes**

### To Other Fields

- Machine learning is applied in various fields, including:
  - **Computer Vision**: Image recognition, Object detection, Facial recognition
  - **Natural Language Processing (NLP)**: Text classification, Sentiment analysis, Language translation
  - **Recommendation Systems**: Product recommendations, Content recommendation
  - **Robotics**: Motion planning, Control systems
  - **Healthcare**: Disease diagnosis, Predictive analytics

### Types of Machine Learning

- **Supervised Learning**: Training model on labeled data to predict outcomes
  - **Regression**: Continuous output, e.g., predicting house prices
  - **Classification**: Categorical output, e.g., spam vs. non-spam emails
- **Unsupervised Learning**: Training model on unlabeled data to identify patterns
  - **Clustering**: Grouping similar data points, e.g., customer segmentation
  - **Dimensionality Reduction**: Reducing feature space, e.g., PCA
- **Reinforcement Learning**: Training model through trial and error to take actions
  - **Q-learning**: Learning to take actions that maximize rewards
- **Deep Learning**: Using neural networks to learn hierarchical representations
  - **Convolutional Neural Networks (CNNs)**: Image recognition
  - **Recurrent Neural Networks (RNNs)**: Sequence data, e.g., speech recognition

### Challenges of Machine Learning

- **Overfitting**: Model fits training data too well, fails on new data
- **Underfitting**: Model is too simple, fails to capture underlying patterns
- **Data Quality Issues**: Noisy, missing, or biased data can lead to poor model performance
- **Explainability**: Understanding why model made certain predictions can be difficult

### Machine Learning Process

1. **Problem Formulation**: Define problem and identify relevant data
2. **Data Collection**: Gather and preprocess data
3. **Data Preprocessing**: Clean, transform, and normalize data
4. **Model Selection**: Choose suitable algorithm and hyperparameters
5. **Model Training**: Train model on labeled data
6. **Model Evaluation**: Evaluate model performance on unseen data
7. **Model Deployment**: Deploy model in production environment

### Machine Learning Applications

- **Predictive Maintenance**: Predicting equipment failures to reduce downtime
- **Customer Segmentation**: Identifying high-value customers for targeted marketing
- **Image Classification**: Classifying medical images for diagnosis
- **Recommendation Systems**: Suggesting products based on user behavior

**Formulas and Definitions**

- **Bias-Variance Tradeoff**: Measuring model performance on training and test data
- **Mean Squared Error (MSE)**: Measuring regression model performance
- **Confusion Matrix**: Evaluating classification model performance
- **Gradient Descent**: Optimization algorithm for minimizing loss function

**Theorems**

- **No Free Lunch Theorem**: No algorithm can perform better than average on all problems
- **Vapnik-Chervonenkis (VC) Theorem**: Measures model capacity and generalizability
