### Introduction

- Machine learning (ML) is a subset of artificial intelligence (AI) that enables systems to learn from data without being explicitly programmed.
- Need for ML: Automation, data analysis, pattern recognition, and decision-making.

### Definition of Machine Learning

- A field of study that gives computers the ability to learn without being explicitly programmed.
- Machine learning algorithms enable computers to automatically improve their performance on a task based on data.

### Types of Machine Learning

- **Supervised Learning**: Learning from labeled data to make predictions on new, unseen data.
- **Unsupervised Learning**: Discovering patterns and relationships in unlabeled data.
- **Reinforcement Learning**: Learning through trial and error by interacting with an environment to maximize rewards.

### Key Formulas and Theorems

- **Bayes' Theorem**: P(A|B) = P(B|A) \* P(A) / P(B)
- **Perceptron Learning Rule**: Δw = α \* (y - (w \* x + b)) \* x
- **Gradient Descent**: Minimize loss function L(w) = (1/2) \* ||w - y||^2

### Important Definitions

- **Bias**: Average error of a model or algorithm.
- **Variance**: Measure of how much a model or algorithm's predictions vary.
- **Overfitting**: When a model is too complex and performs well on training data but poorly on new data.
- **Underfitting**: When a model is too simple and fails to capture important patterns in the data.

### Important Terms

- **Feature Engineering**: Selecting and preprocessing features to improve model performance.
- **Regularization**: Techniques to prevent overfitting, such as L1 and L2 regularization.
- **Hyperparameter Tuning**: Adjusting model hyperparameters to optimize performance.
