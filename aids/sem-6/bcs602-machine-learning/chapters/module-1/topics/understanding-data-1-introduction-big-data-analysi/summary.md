# **Understanding Data – 1: Revision Notes**

## **Introduction**

- Machine learning requires high-quality data
- Data quality and preprocessing are crucial for successful machine learning models

## **Big Data Analysis Framework**

- Big data characteristics: large volume, variety, velocity, and veracity
- Big data processing: batch processing, stream processing, and real-time processing
- Big data storage: relational databases, NoSQL databases, and data warehouses

## **Descriptive Statistics**

- Measures of central tendency: mean, median, and mode
- Measures of variability: range, variance, and standard deviation
- Formulas:
  - Mean: μ = (Σx) / n
  - Median: median = (x1 + x2 + ... + xn) / 2
  - Mode: mode = most frequent value
  - Variance: σ² = Σ(x - μ)² / (n - 1)
  - Standard deviation: σ = √(σ²)

## **Univariate Data Analysis and Visualization**

- Univariate analysis: summary statistics, visualizations, and data distribution
- Visualizations: histograms, box plots, scatter plots, and bar charts
- Theorems:
  - Central Limit Theorem (CLT): states that sample means converge to population means
  - Law of Large Numbers (LLN): states that sample averages converge to population means

## **Important Terms**

- **Data preprocessing**: cleaning, transforming, and feature engineering of data
- **Data visualization**: graphical representation of data to understand patterns and trends
- **Overfitting**: model fits training data too closely, resulting in poor generalization performance
- **Underfitting**: model fails to capture important patterns in training data
