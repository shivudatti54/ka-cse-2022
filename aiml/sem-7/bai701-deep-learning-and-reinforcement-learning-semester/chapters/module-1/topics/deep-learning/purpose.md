# Why Study What is Deep Learning?

## Big Picture Foundation
Understanding what deep learning is and how it differs from traditional machine learning provides the conceptual foundation for all subsequent deep learning topics. This is the **entry point** to modern AI.

## Exam Relevance
| Aspect | Details |
|--------|---------|
| Frequency | Appears in virtually every AI/ML exam |
| Marks | 5-10 marks (definitions, comparisons, history) |
| Question Types | Short definitions, compare/contrast essays, timeline questions |

## Practical Value
- **Correct problem framing**: Know when deep learning is appropriate vs. traditional ML
- **Career relevance**: Deep learning powers most cutting-edge AI applications
- **Communication**: Explain deep learning concepts to stakeholders and colleagues

## Learning Outcomes
After this topic, you should be able to:
1. Define deep learning and distinguish it from machine learning
2. Explain why deep learning has become successful recently
3. Describe the hierarchical feature learning process
4. Identify suitable applications for deep learning
