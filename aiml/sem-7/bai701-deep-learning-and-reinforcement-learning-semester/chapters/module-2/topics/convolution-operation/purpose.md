# Learning Objectives
After studying this topic, you should be able to:
1.  Define the convolution operation mathematically and explain its purpose within a CNN.
2.  Calculate the spatial dimensions of a feature map given the input size, kernel size, stride, and padding.
3.  Explain the concepts of parameter sharing and sparse connectivity and their advantages over fully-connected layers.
4.  Describe the role of the kernel as a feature detector and provide examples of simple hand-crafted kernels.
5.  Manually compute the output of a convolution operation on a small, sample input matrix.
6.  Differentiate between standard convolution and its common variants (e.g., 1x1, dilated).
