# Why Study Neural Network Basics?

## Big Picture Foundation
Understanding neurons, layers, and forward propagation is essential for grasping how deep learning models work. This knowledge forms the basis for understanding training, architectures, and debugging.

## Exam Relevance
| Aspect | Details |
|--------|---------|
| Frequency | Core topic in every DL/ML exam |
| Marks | 8-15 marks (diagrams, computations, architecture design) |
| Question Types | Draw network diagrams, compute forward pass, design architectures |

## Practical Value
- **Model Design**: Choose appropriate layer sizes and depths
- **Debugging**: Understand parameter flow and dimension mismatches
- **Optimization**: Calculate memory and compute requirements

## Learning Outcomes
After this topic, you should be able to:
1. Explain the computation in a single neuron
2. Describe the role of weights, biases, and activations
3. Perform forward propagation calculations
4. Calculate the number of parameters in a network
