# Learning Objectives
After studying this topic, you should be able to:
1.  Explain the vanishing gradient problem and its detrimental effect on standard Recurrent Neural Networks (RNNs).
2.  Identify and describe the function of each core component of an LSTM cell: the cell state, forget gate, input gate, and output gate.
3.  Write down the mathematical equations that define the computations within a single LSTM time step.
4.  Articulate how the LSTM's additive update rule for the cell state mitigates the vanishing gradient problem.
5.  Compare and contrast the architecture, capabilities, and computational cost of simple RNNs and LSTMs.
6.  List at least three practical applications where LSTMs are effectively used.
