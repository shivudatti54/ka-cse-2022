# Learning Objectives
After studying this topic, you should be able to:
1.  Explain the fundamental difference between symmetric and asymmetric cryptography and the problem RSA solves.
2.  Describe the mathematical principles underlying the RSA algorithm, including the integer factorization problem and Euler's theorem.
3.  Perform RSA key generation, encryption, and decryption using a small-scale numerical example.
4.  Differentiate between the use of RSA for encryption/decryption and for creating/verifying digital signatures.
5.  Identify practical considerations for implementing RSA, such as key size and the use of padding schemes.
6.  Explain the rationale behind using RSA in a hybrid cryptosystem with a symmetric cipher like AES.
