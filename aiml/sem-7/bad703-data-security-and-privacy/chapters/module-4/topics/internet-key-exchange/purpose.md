# Learning Objectives
After studying this topic, you should be able to:
1.  Define Internet Key Exchange (IKE) and explain its critical role within the IPsec framework.
2.  Differentiate between an IKE Security Association (SA) and an IPsec SA, stating the purpose of each.
3.  Describe the two-phase operation of IKEv1, including the goals and message flows of Main Mode and Aggressive Mode.
4.  Explain the key improvements and simplified message flow of IKEv2 compared to IKEv1.
5.  Identify and define core IKE concepts such as Perfect Forward Secrecy (PFS) and Diffie-Hellman groups.
6.  Compare and contrast the authentication methods used by IKE (Pre-Shared Keys vs. Digital Certificates).
