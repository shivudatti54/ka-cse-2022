# **Revision Notes: Chapter 17.1 - Data Security and Privacy**

**Definition:** Data security and privacy refer to the measures taken to protect sensitive information from unauthorized access, use, or disclosure.

**Key Concepts:**

- **Data Security:** Protecting data from unauthorized access, use, or disclosure.
- **Data Privacy:** Protecting personal information from unauthorized access, use, or disclosure.

**Important Formulas:**

- **Confidentiality, Integrity, and Availability (CIA) Triad:**
  - Confidentiality: Protecting data from unauthorized access.
  - Integrity: Ensuring data accuracy and completeness.
  - Availability: Ensuring data accessibility.

**Important Theorems:**

- **Shannon's Source and Sink Model:**
  - Involves data transmission channels, noise, and interference.
  - Used to study the reliability of data transmission.

**Important Definitions:**

- **Encryption:** Converting plaintext into unreadable ciphertext using an algorithm and key.
- **Decryption:** Converting ciphertext back into plaintext using an algorithm and key.
- **Digital Signature:** A cryptographic technique used to verify the authenticity of a message or document.

**Key Concepts and Formulas:**

- **Data Encryption:**
  - Example formula: `C = E(P, K)`, where C is ciphertext, P is plaintext, K is key, and E is encryption algorithm.
- **Data Decryption:**
  - Example formula: `P = D(C, K)`, where P is plaintext, C is ciphertext, K is key, and D is decryption algorithm.

**Quick Revision Tips:**

- Review the CIA triad and its components.
- Understand the concepts of encryption and decryption.
- Familiarize yourself with the Shannon source and sink model.
- Review the definitions of confidentiality, integrity, and availability.
