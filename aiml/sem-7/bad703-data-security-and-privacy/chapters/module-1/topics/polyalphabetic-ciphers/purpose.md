# Learning Objectives
After studying this topic, you should be able to:
1. Construct a Playfair matrix from a given keyword and use it to encrypt and decrypt a message.
2. Perform encryption and decryption using the Hill cipher for a given key matrix and plaintext.
3. Differentiate between monoalphabetic and polyalphabetic substitution ciphers.
4. Encrypt and decrypt a message using the Vigenère cipher with a given keyword.
5. Explain the theoretical security of the One-Time Pad and the practical reasons for its limited use.
6. Identify the main cryptanalytic techniques used to break the Playfair, Hill, and Vigenère ciphers.
