# **Deep Learning and Reinforcement Learning Revision Notes**

## **Introduction**

- Deep learning is a subset of machine learning that uses neural networks with multiple layers to analyze data.
- Inspired by the structure and function of the human brain.
- Enables machines to learn from data and improve their performance over time.

## **Shallow Learning**

- Also known as traditional machine learning.
- Uses a single layer or a few layers of neural networks.
- Limited to learning linear decision boundaries.
- Examples: linear regression, logistic regression.

## **Deep Learning**

- Uses neural networks with multiple hidden layers.
- Enables the learning of complex decision boundaries.
- Can learn non-linear relationships in data.
- Examples: convolutional neural networks (CNNs), recurrent neural networks (RNNs).

## **Why to use Deep Learning**

- **Handling complex data**: Deep learning can learn non-linear relationships in high-dimensional data.
- **Improved performance**: Deep learning models can achieve state-of-the-art performance on many tasks.
- **Transfer learning**: Pre-trained deep learning models can be fine-tuned for specific tasks.

## **How Deep Learning Works**

- **Forward pass**: Input data is propagated through the network to generate an output.
- **Backward pass**: Error is calculated and propagated backwards through the network to adjust weights and biases.
- **Optimization**: Loss function is minimized using an optimization algorithm (e.g. stochastic gradient descent).

## **Deep Learning Challenges**

- **Overfitting**: Models may memorize training data and fail to generalize.
- **Vanishing gradients**: Backpropagation may not learn effectively due to gradients becoming very small.
- **Exploration-exploitation trade-off**: Models may not know when to explore new possibilities or exploit known knowledge.

## **Important Formulas and Definitions**

- **Activation function**: Non-linear function applied to the output of a layer (e.g. sigmoid, ReLU).
- **Loss function**: Measures the difference between predicted and actual outputs (e.g. mean squared error, cross-entropy).
- **Optimization algorithm**: Technique used to minimize the loss function (e.g. stochastic gradient descent, Adam).
- **Regularization**: Techniques used to prevent overfitting (e.g. dropout, L1/L2 regularization).
- **Convolutional neural network (CNN)**: Neural network using convolutional and pooling layers to extract features from images.
- **Recurrent neural network (RNN)**: Neural network using recurrent connections to model temporal relationships.
