# Learning Objectives
After studying this topic, you should be able to:
1.  Formulate a simple two-variable linear programming problem from a word problem, identifying decision variables, the objective function, and constraints.
2.  Accurately plot constraint inequalities and identify the feasible region on a graph.
3.  Identify the coordinates of all corner points of the feasible region by solving the relevant equations.
4.  Determine the optimal solution by evaluating the objective function at each corner point for both maximization and minimization problems.
5.  Recognize and explain the three special cases: multiple optimal solutions, an unbounded solution, and an infeasible problem.
6.  Use the iso-profit line method to visually verify the optimal solution found by the corner point method.
