### Experiment 1: Read a dataset from the user and implement the following:

#### Key Points

- **Dataset Input**: Read a dataset from the user in a structured format.
- **Data Representation**: Represent the dataset as a graph or network.
- **Node Identification**: Identify the nodes (variables) and their relationships in the network.
- **Edge Representation**: Represent the edges (conditional dependencies) between nodes.
- **Query Implementation**: Implement a query to make predictions or inference based on the network.

#### Important Formulas, Definitions, and Theorems

- **Bayesian Networks**: A probabilistic graphical model that represents a set of variables and their conditional dependencies.
  - Definition: A Bayesian network is a directed acyclic graph (DAG) where each node represents a random variable, and the edges represent conditional dependencies.
  - Formula: P(X) = ∑ P(X|Pa(X))

- **Markov Chains**: A mathematical system that undergoes transitions from one state to another according to certain probabilistic rules.
  - Definition: A Markov chain is a sequence of random states that obey the Markov property, which states that the next state depends only on the current state.

- **Approximate Inference**: A method for approximating the marginal probability or joint probability distribution of a set of variables in a Bayesian network.
  - Definition: Approximate inference is a technique used to approximate the marginal probability or joint probability distribution of a set of variables in a Bayesian network.
