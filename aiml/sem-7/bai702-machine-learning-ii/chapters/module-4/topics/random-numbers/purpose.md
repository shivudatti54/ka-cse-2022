# Learning Objectives
After studying this topic, you should be able to:
1. Define and explain the Random Network Model and its two main formulations (G(n, M) and G(n, p))
2. Derive and interpret the degree distribution of random networks and understand its Poisson approximation
3. Calculate key properties of random networks including clustering coefficient and average path length
4. Explain the phase transition and giant component phenomenon in random networks
5. Compare and contrast random networks with real-world networks and identify their limitations
6. Apply the random network model as a null hypothesis for network analysis
