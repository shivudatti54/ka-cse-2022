# **Markov Chain Monte Carlo (MCMC) Methods: Sampling**

### Definitions and Formulas

- **Markov Chain**: a sequence of random states, where the probability of transitioning from one state to another depends only on the current state.
- **Markov Chain Monte Carlo (MCMC)**: a method for generating samples from a target distribution using a Markov chain.
- **Proposal Distribution**: a probability distribution used to propose new states to the Markov chain.
- **Target Distribution**: the distribution we want to sample from.
- **Acceptance Probability**: the probability that a proposed state is accepted.

### Key Concepts

- **Random Numbers**: used to generate new states in the Markov chain.
- **Gaussian Random Numbers**: a type of random number generator used in MCMC.
- **Monte Carlo Or Bust**: the concept that MCMC methods are only as good as the proposal distribution.
- **Theorem**: Metropolis-Hastings Theorem:
  - If the target distribution is a probability distribution, and the proposal distribution is a valid probability distribution,
  - then the Markov chain generated by the MCMC method will converge to the target distribution.
- **Importance Sampling**: a technique used to improve the efficiency of MCMC methods.

### MCMC Algorithms

- **Metropolis-Hastings Algorithm**:
  - Choose a proposal distribution
  - Generate a new state using the proposal distribution
  - Calculate the acceptance probability
  - Accept or reject the new state based on the acceptance probability
- **Gibbs Sampling Algorithm**:
  - Initialize a state vector
  - Iterate through each component of the state vector
  - Generate a new component using conditional distributions
  - Update the state vector with the new component
- **Hamiltonian Monte Carlo Algorithm**:
  - Choose a proposal distribution
  - Generate a new state using the proposal distribution
  - Calculate the Hamiltonian of the state
  - Update the state vector using the Hamiltonian

### Important Theorems

- **Markov Chain Convergence Theorem**: if the Markov chain generated by the MCMC method converges to the target distribution, then the sample mean of the chain will converge to the expected value of the target distribution.
- **Law of Large Numbers**: if the Markov chain generated by the MCMC method converges to the target distribution, then the sample mean of the chain will converge to the expected value of the target distribution.
