# **Explanation-Based Learning of Search Control Knowledge**

## **Definition**

Explanation-Based Learning (EBL) is a type of learning approach that focuses on acquiring knowledge by explaining why a solution is correct.

## **Key Points**

- **Definition of EBL**: EBL is a learning approach that involves acquiring knowledge by explaining why a solution is correct, rather than just memorizing examples.
- **Components of EBL**: Explanation, Evaluation, and Generalization.
- **Types of EBL**:
  - **Inductive EBL**: Learning from examples and explanations.
  - **Top-Down EBL**: Learning by explaining why a solution is correct.
- **Search Control Knowledge (SCK)**: Knowledge used to control the search process in EBL.

## **Key Concepts**

- **Explanation**: A statement that explains why a solution is correct.
- **Evaluation**: The process of assessing the correctness of the explanation.
- **Generalization**: The process of applying the learned knowledge to new situations.
- **Covering Set**: A set of examples that are used to explain a solution.
- **Search Control Knowledge (SCK)**: Knowledge used to control the search process in EBL.
