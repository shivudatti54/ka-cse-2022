## **Introduction**

Ensemble learning is a powerful technique used in machine learning to improve the accuracy of classification models. It involves combining the predictions of multiple models to produce a more accurate result. In this chapter, we will delve into the world of boosting, stumping, bagging, and random forests, four popular ensemble learning techniques.

## **Historical Context**

The concept of ensemble learning dates back to the 1960s, when the first ensemble methods were proposed. However, it wasn't until the 1990s that boosting and bagging became popular techniques in machine learning.

- **Boosting**: The concept of boosting was first introduced by Yoav Freund and Robert Schapire in 1995. They proposed the first boosting algorithm, AdaBoost, which uses a weighted voting scheme to combine the predictions of multiple models.
- **Bagging**: Bagging, short for "bagging" or "bootstrap aggregating," was first introduced by Leo Breiman in 1994. He proposed the first bagging algorithm, called "bagging," which uses multiple datasets to train multiple models.

## **Boosting: Adaboost, Stumping**

### Adaboost

Adaboost is a popular boosting algorithm that uses a weighted voting scheme to combine the predictions of multiple models. The algorithm works as follows:

1.  **Initialization**: The algorithm starts with a single model and a weight of 1/2.
2.  **Iteration**: The algorithm iterates through the training data, updating the weights of the models based on their performance.
3.  **Weight Update**: The weights of the models are updated using the following formula:
    - `w_t+1 = w_t * exp(-L_t)`
    - `L_t = -log(w_t * y_t)`
    - `y_t` is the true label of the data point, and `w_t` is the weight of the model at iteration `t`

4.  **Prediction**: The final prediction is made by combining the predictions of all the models using the following formula:
    - `y_pred = sign(sum(w_t \* y_t))`

### Stumping

Stumping is a simple boosting algorithm that uses a single decision tree to make predictions. The algorithm works as follows:

1.  **Decision Tree**: A decision tree is built to make predictions on the training data.
2.  **Weight Update**: The weights of the models are updated based on their performance:
    - `w_t+1 = w_t * exp(-L_t)`
    - `L_t = -log(w_t * y_t)`
3.  **Prediction**: The final prediction is made by combining the predictions of all the models using the following formula:
    - `y_pred = sign(sum(w_t \* y_t))`

### Example

Suppose we have a binary classification problem with 10 features and 1000 samples. We can use Adaboost to train a model using the following code:

```python
from sklearn.ensemble import AdaBoostClassifier
from sklearn.metrics import accuracy_score

# Create a dataset
X = np.random.rand(1000, 10)
y = np.random.randint(0, 2, 1000)

# Train an Adaboost model
model = AdaBoostClassifier(n_estimators=100, learning_rate=0.1)
model.fit(X, y)

# Make predictions
y_pred = model.predict(X)

# Evaluate the model
accuracy = accuracy_score(y, y_pred)
print("Accuracy:", accuracy)
```

## **Bagging: Subagging**

### Bagging

Bagging is a popular ensemble learning technique that uses multiple datasets to train multiple models. The algorithm works as follows:

1.  **Bootstrap Sampling**: A bootstrap sample of the training data is generated by randomly selecting a subset of samples with replacement.
2.  **Model Training**: Multiple models are trained on the bootstrap samples.
3.  **Prediction**: The final prediction is made by combining the predictions of all the models using the following formula:
    - `y_pred = mode(y_t)`
    - `y_t` is the true label of the data point

### Subagging

Subagging is a variant of bagging that uses a weighted voting scheme to combine the predictions of multiple models. The algorithm works as follows:

1.  **Bootstrap Sampling**: A bootstrap sample of the training data is generated by randomly selecting a subset of samples with replacement.
2.  **Model Training**: Multiple models are trained on the bootstrap samples.
3.  **Weight Update**: The weights of the models are updated based on their performance:
    - `w_t+1 = w_t * exp(-L_t)`
    - `L_t = -log(w_t * y_t)`
4.  **Prediction**: The final prediction is made by combining the predictions of all the models using the following formula:
    - `y_pred = sign(sum(w_t \* y_t))`

### Example

Suppose we have a binary classification problem with 10 features and 1000 samples. We can use bagging to train a model using the following code:

```python
from sklearn.ensemble import BaggingClassifier
from sklearn.metrics import accuracy_score

# Create a dataset
X = np.random.rand(1000, 10)
y = np.random.randint(0, 2, 1000)

# Train a bagging model
model = BaggingClassifier(n_estimators=100, random_state=42)
model.fit(X, y)

# Make predictions
y_pred = model.predict(X)

# Evaluate the model
accuracy = accuracy_score(y, y_pred)
print("Accuracy:", accuracy)
```

## **Random Forests**

Random forests are an ensemble learning technique that uses multiple decision trees to make predictions. The algorithm works as follows:

1.  **Bootstrap Sampling**: A bootstrap sample of the training data is generated by randomly selecting a subset of samples with replacement.
2.  **Decision Tree Training**: Multiple decision trees are trained on the bootstrap samples.
3.  **Prediction**: The final prediction is made by combining the predictions of all the decision trees using the following formula:
    - `y_pred = mode(y_t)`
    - `y_t` is the true label of the data point

### Example

Suppose we have a binary classification problem with 10 features and 1000 samples. We can use random forests to train a model using the following code:

```python
from sklearn.ensemble import RandomForestClassifier
from sklearn.metrics import accuracy_score

# Create a dataset
X = np.random.rand(1000, 10)
y = np.random.randint(0, 2, 1000)

# Train a random forest model
model = RandomForestClassifier(n_estimators=100, random_state=42)
model.fit(X, y)

# Make predictions
y_pred = model.predict(X)

# Evaluate the model
accuracy = accuracy_score(y, y_pred)
print("Accuracy:", accuracy)
```
