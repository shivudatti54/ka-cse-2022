# Why Study Q-Learning?

## Big Picture Foundation

Q-Learning is one of the most important algorithms in RL history. It demonstrated that optimal behavior can be learned without knowing the environment model, making RL practical for real-world problems.

## Exam Relevance

| Aspect         | Details                                                   |
| -------------- | --------------------------------------------------------- |
| Frequency      | Appears in virtually every RL exam                        |
| Marks          | 8-15 marks (algorithm, derivation, comparison with SARSA) |
| Question Types | Write algorithm, explain update rule, numerical examples  |

## Practical Value

- **Foundation for DQN**: Deep Q-Networks are Q-learning with neural networks
- **Widely Used**: Standard baseline algorithm for many RL problems
- **Off-Policy Benefits**: Can reuse experience, more sample efficient
- **Simple Implementation**: Easy to code and understand

## Learning Outcomes

After this topic, you should be able to:

1. Write the Q-learning update rule and explain each component
2. Implement Q-learning for simple environments
3. Explain why Q-learning is off-policy
4. Compare Q-learning with other TD methods
5. Understand convergence conditions and hyperparameter effects
