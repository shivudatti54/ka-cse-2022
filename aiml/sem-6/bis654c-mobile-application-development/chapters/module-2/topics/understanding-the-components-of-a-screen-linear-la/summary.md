# Understanding the Components of a Screen in Android Development

### Linear Layout

- A user interface layout where elements are arranged sequentially in a single row or column.
- `LinearLayout` class is used to create a linear layout.
- Key attributes:
  - `orientation`: specifies the orientation of the layout (horizontal or vertical).
  - `gravity`: sets the alignment of elements within the layout.
- Advantages:
  - Easy to manage and maintain.
  - Suitable for simple layouts.

### Absolute Layout

- A user interface layout where elements are positioned relative to their parent element.
- `RelativeLayout` class is used to create an absolute layout.
- Key attributes:
  - `layout_width` and `layout_height`: specify the width and height of the element.
  - `layout_margin`: sets the margin of the element.
- Advantages:
  - Allows for complex layouts with relative positioning.
  - Suitable for custom layouts.

### Frame Layout

- A user interface layout that uses a FrameLayout to display a single element.
- `FrameLayout` class is used to create a frame layout.
- Key attributes:
  - `layout_width` and `layout_height`: specify the width and height of the child element.
- Advantages:
  - Suitable for layouts that require a single element to be displayed.
  - Easy to use and manage.

## Key Formulas, Definitions, and Theorems

- **Definition of Linear Layout**: A layout where elements are arranged sequentially in a single row or column.
- **Definition of Absolute Layout**: A layout where elements are positioned relative to their parent element.
- **Definition of Frame Layout**: A layout that uses a FrameLayout to display a single element.
- **Theorem of Layout**: The theorem states that a layout can be described as a combination of linear layouts, absolute layouts, and frame layouts.
