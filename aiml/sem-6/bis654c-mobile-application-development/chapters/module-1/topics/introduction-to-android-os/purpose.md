# Learning Objectives
After studying this topic, you should be able to:
1.  Define the Android SDK and list its core components.
2.  Differentiate between the roles of the JDK, SDK Manager, and AVD Manager.
3.  Explain the purpose and common uses of the Android Debug Bridge (adb).
4.  Describe the key steps involved in the Android application build process, naming the tools involved at each stage (e.g., javac, d8, aapt2).
5.  Configure a basic Android development environment by installing necessary packages via the SDK Manager.
6.  Create and launch an Android Virtual Device (AVD) using the AVD Manager.
