# Ecosystem – Android Versions – Android Activity – Features of Android – Android Architecture Stack Linux Kernel

### Ecosystem

- Android ecosystem refers to the collection of hardware, software, and services that support the Android operating system.
- Key players in the Android ecosystem include:
  - Google (hardware and software development)
  - Android Open Handset Alliance (OHAA)
  - Device manufacturers (e.g. Samsung, Huawei, Xiaomi)

### Android Versions

- Major Android versions:
  - Android 1.0 (released in 2008)
  - Android 2.0 (released in 2009)
  - Android 4.0 (released in 2011)
  - Android 8.0 (released in 2017)
  - Android 11 (released in 2020)
- Android versioning scheme:
  - Minor updates (e.g. Android 4.1, Android 4.2)
  - Major updates (e.g. Android 4.0, Android 5.0)

### Android Activity

- Android activity is a fundamental component of the Android architecture.
- Definition: A single screen or window that users interact with directly.
- Life cycle:
  - Created
  - Started
  - Resumed
  - Paused
  - Stopped
  - Destroyed

### Features of Android

- Multi-touch support
- Gesture recognition
- Google Play Store for apps
- Integrated email and messaging clients
- Bluetooth and Wi-Fi connectivity
- Camera and microphone capabilities

### Android Architecture Stack

- **Application**: The top-level component of the Android architecture.
- **Service**: A component that runs in the background, performing tasks such as networking or data storage.
- **BroadcastReceiver**: A component that listens for system-wide broadcasts.
- **ContentProvider**: A component that manages access to shared data.
- **Thread**: A fundamental concept in multi-threading, allowing for concurrent execution of tasks.

### Linux Kernel

- Definition: The core operating system kernel responsible for managing hardware resources.
- Key components:
  - Process scheduler
  - Memory manager
  - File system manager
  - Networking manager
- Relationship with Android:
  - Android uses the Linux kernel as its foundation
  - Android kernel modifications are made to support Android-specific features

## Formulas, Definitions, and Theorems

- **Android versioning formula**: Android [major version].xx (e.g. Android 8.0.0)
- **Activity life cycle theorem**: An activity can be in one of five states: Created, Started, Resumed, Paused, or Stopped.
