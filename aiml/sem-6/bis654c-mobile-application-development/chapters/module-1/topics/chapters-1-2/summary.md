# **Mobile Application Development: Android OS (Chapters 1 & 2) Revision Notes**

### Introduction

- The Android operating system is an open-source mobile operating system developed by the Open Handset Alliance (OHA).
- Android was created by Google in collaboration with various technology companies.

### Android Description

- **Android Architecture**
  - Model View Presenter (MVP) architecture
  - Android Architecture Components (e.g., Data Binding, LiveData)
- **Android Components**
  - Activities, Fragments, Services, BroadcastReceivers
  - Intents, IntentFilters, IntentActions

### Open Handset Alliance (OHA)

- **History**: Formed in 2003 by Google, HTC, Motorola, Samsung, and other companies to promote open standards for mobile devices.
- **Goals**: Encourage competition, innovation, and consumer choice in the mobile market.

### Android OS Features

- **Hardware Abstraction Layer (HAL)**: Provides a layer of abstraction between the Android OS and the underlying hardware.
- **Java Development Kit (JDK)**: Android apps are written in Java or Kotlin.
- **Dalvik Virtual Machine (VM)**: Runs Android apps in a virtual machine.

### Important Formulas and Definitions

- **API**: Application Programming Interface - a set of rules and protocols that allow software components to interact with each other.
- **SDK**: Software Development Kit - a set of tools and libraries that allow developers to build Android apps.
- **Java Virtual Machine (JVM)**: A virtual machine that runs Java code.

### Important Theorems

- **Android's Decentralized Architecture**: Android's architecture is decentralized, meaning that there is no single point of failure.
- **Android's Open-Source Nature**: Android's open-source nature allows for community-driven development and customization.

### Key Points

- Android is an open-source mobile operating system developed by the Open Handset Alliance (OHA).
- Android's architecture is based on the Model View Presenter (MVP) pattern.
- Android components include activities, fragments, services, and broadcast receivers.
- Android apps are written in Java or Kotlin and run on the Dalvik Virtual Machine (VM).
