# **Textbook 1: Chapter 3: 3.1 to 3.5 Revision Notes**

### 3.1: Introduction to Virtualization

- **Definition:** Virtualization is the process of creating a virtual environment or layer on top of existing physical infrastructure.
- **Types of Virtualization:**
  - Host-Guest Virtualization (e.g., VMware, VirtualBox)
  - Server Virtualization (e.g., KVM, Xen)
  - Desktop Virtualization (e.g., Citrix, Terminal Services)

### 3.2: Types of Virtual Machines

- **Definition:** A virtual machine is a software emulation of a physical computer.
- **Types of Virtual Machines:**
  - **Type 1 (bare-metal):** Runs directly on physical hardware (e.g., Xen, KVM)
  - **Type 2 (hosted):** Runs on top of an existing host operating system (e.g., VirtualBox, VMware Workstation)

### 3.3: Virtualization Architecture

- **Definition:** The infrastructure that supports virtualization.
- **Components:**
  - **Hypervisor:** The software layer that creates virtual machines (e.g., VMware ESXi, KVM)
  - **Virtual Machine Monitor:** A software layer that manages virtual machines (e.g., Xen)
  - **Physical Servers:** The underlying hardware infrastructure

### 3.4: Virtualization Benefits

- **Definition:** The advantages of virtualization.
- **Benefits:**
  - **Hardware Consolidation:** Multiple virtual machines run on a single physical server
  - **Improved Mobility:** Virtual machines can be easily moved between physical hosts
  - **Enhanced Security:** Isolation of virtual machines from each other and the host

### 3.5: Virtualization Implementation Levels

- **Definition:** The different levels of virtualization implementation.
- **Levels:**
  - **Level 1:** Hypervisor-based virtualization (e.g., VMware ESXi)
  - **Level 2:** Virtual machine monitor-based virtualization (e.g., Xen)
  - **Level 3:** Hosted virtualization (e.g., VirtualBox)

### Important Formulas, Definitions, and Theorems

- **VLAN (Virtual Local Area Network):** A virtual network that uses etiqueting to separate traffic on a physical network.
- **NIC (Network Interface Card):** A hardware component that provides network connectivity.
- **VMID (Virtual Machine Identifier):** A unique identifier assigned to each virtual machine.
