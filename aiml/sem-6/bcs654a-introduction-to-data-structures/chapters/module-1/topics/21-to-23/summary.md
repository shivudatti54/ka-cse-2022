# 2.1 to 2.3: Introduction to Data Structures

=============================================

### 2.1: Introduction to Arrays

---

- An array is a collection of elements of the same data type stored in contiguous memory locations.
- Arrays are used to store and manipulate large amounts of data efficiently.
- Notation: `arr[]` or `arr(n)` where `n` is the size of the array.

### 2.2: Characteristics of Arrays

---

- **Homogeneity**: All elements in an array must be of the same data type.
- **Continuity**: Elements in an array are stored in contiguous memory locations.
- **Fixed Size**: The size of an array is fixed at the time of declaration.
- **Indexing**: Elements in an array are accessed using an index or subscript.

### 2.3: Declaring and Initializing Arrays

---

- Declaration: `int arr[5]` or `double arr[5]`
- Initialization: `arr[i] = value;` for individual elements or `for (int i = 0; i < n; i++) { arr[i] = value; }` for entire array

### Important Formulas and Definitions

---

- **Array Index**: `i` or `j` (for 2D arrays)
- **Array Size**: `n` (for 1D arrays) or `m x n` (for 2D arrays)
- **Array Access**: `arr[i]` or `arr[i][j]` (for 2D arrays)

### Theorems and Properties

---

- **Array Properties**:
  - **Accessible**: All elements in an array can be accessed directly using their index.
  - **Indexable**: All elements in an array can be indexed using their index.
  - **Mutability**: Elements in an array can be modified after initialization.
