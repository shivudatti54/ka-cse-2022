# **Revision Notes: Arrays (8.1-8.5)**

### 8.1: Introduction to Arrays

- An array is a collection of elements of the same data type stored in contiguous memory locations.
- Arrays are used to store a fixed number of values in a single variable.
- Arrays are declared using the `[]` symbol.

### 8.2: One-Dimensional Arrays

- A one-dimensional array is an array with a single dimension.
- It can store elements of the same data type.
- Example: `int arr[5]`
- Size of the array is determined by the number of elements it can hold.

### 8.3: Two-Dimensional Arrays

- A two-dimensional array is an array with two dimensions.
- It can store elements of the same data type.
- Example: `int arr[2][3]`
- Size of the array is determined by the number of rows and columns.

### 8.4: Initializing Two-Dimensional Arrays

- An array can be initialized using the following syntax:

```c
int arr[2][3] = {{1, 2, 3}, {4, 5, 6}};
```

- This initializes the array with the given values.

### 8.5: Important Formulas and Definitions

- **Rank of an array**: The number of dimensions in an array.
- **Size of an array**: The number of elements it can hold.
- **Pointer to an array**: A pointer that points to the first element of an array.

### Important Theorems

- **Theorem:**
  If `A` is a `m \times n` matrix and `B` is an `n \times p` matrix, then the product `AB` is defined and is an `m \times p` matrix.
  This theorem can be used to perform matrix multiplication.
