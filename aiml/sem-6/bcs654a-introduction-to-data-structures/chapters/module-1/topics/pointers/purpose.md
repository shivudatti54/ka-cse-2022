# Learning Objectives

After studying this topic, you should be able to:

1. Define what a pointer is and explain how it differs from a regular variable.
2. Correctly use the address-of (`&`) and dereference (`*`) operators.
3. Declare and initialize pointer variables to point to data of various types.
4. Explain the relationship between arrays and pointers, including using pointer arithmetic to traverse arrays.
5. Describe the purpose and use of dynamic memory allocation functions (`malloc`, `calloc`, `realloc`, `free`).
6. Identify and avoid common pointer-related errors such as uninitialized (wild) pointers, dangling pointers, and memory leaks.
