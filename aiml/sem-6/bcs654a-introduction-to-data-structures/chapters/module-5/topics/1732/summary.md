# 17.3.2 Introduction

## Key Points

- **Definition:** Insertion Sort is a simple sorting algorithm that works by dividing the input into a sorted and an unsorted region.
- **How it works:**
  - Start with the first element as the sorted region.
  - Iterate over the unsorted region, inserting each element into its proper position in the sorted region.
- **Time complexity:**
  - Best case: O(n)
  - Average case: O(n^2)
  - Worst case: O(n^2)

## Important Formulas and Definitions

- **Insertion Sort Formula:** `arr[i] = min(arr[0..i-1])`
- **Definition of Insertion Sort:** A sorting algorithm that works by dividing the input into a sorted and an unsorted region and then inserting the elements of the unsorted region into the sorted region.

## Theorem

- **Insertion Sort Theorem:** Insertion Sort is an in-place sorting algorithm, meaning it does not require any additional storage space.
