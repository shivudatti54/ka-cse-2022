# **8.1 to 8.2 Revision Notes**

### Introduction to Data Structures

- **Data Structure:** A collection of data elements, each of which represents a value or a relationship between values.
- **Types of Data Structures:** Linear (arrays, linked lists), Non-Linear (stacks, queues, trees, graphs), Composite (nested data structures)

### 8.1: Arrays

- **Definition:** A collection of elements of the same data type stored in contiguous memory locations.
- **Importance:** Used for efficient storage and retrieval of data.
- **Operations:**
  - Indexing (accessing elements by their indices)
  - Assigning values to elements
  - Retrieving values from elements
- **Formulas:** None
- **Theorems:** None

### 8.2: Linked Lists

- **Definition:** A dynamic collection of elements, where each element points to the next element.
- **Importance:** Used for efficient insertion, deletion, and searching of elements.
- **Operations:**
  - Insertion (at beginning or end of the list)
  - Deletion (from the list)
  - Searching (finding a specific element)
- **Types of Linked Lists:** Singly Linked Lists, Doubly Linked Lists
- **Formulas:** None
- **Theorems:** None
