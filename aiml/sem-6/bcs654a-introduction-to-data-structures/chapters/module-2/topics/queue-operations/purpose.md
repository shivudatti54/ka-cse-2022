# Learning Objectives

After studying this topic, you should be able to:

1.  Define a queue and explain its First-In-First-Out (FIFO) principle.
2.  List and describe the standard operations performed on a queue (enqueue, dequeue, peek, isEmpty, isFull).
3.  Implement a queue using a linear array and identify its limitation regarding memory usage.
4.  Implement a circular queue using an array to efficiently utilize memory space.
5.  Differentiate between various types of queues (Simple, Circular, Priority, Deque).
6.  Identify real-world and computer science applications where a queue is the appropriate data structure.
