# **Textbook 2: Ch - Trees: Introduction, Basic Concepts, Representation of Binary Trees, Operations on Binary Trees**

## **Important Definitions and Formulas**

- **Binary Tree**: A tree where each node has at most two children (i.e., left child and right child).
- **Node**: A single element in the tree.
- **Root Node**: The topmost node of the tree.
- **Leaf Node**: A node with no children.
- **Height of a Tree**: The number of edges in the longest path from the root node to a leaf node.
- **Balanced Tree**: A tree where the height of the two subtrees of every node differs at most by one.

## **Basic Concepts**

- **Tree Traversal**: A process of visiting each node in the tree exactly once.
- **Inorder Traversal**: Left-Root-Right (L-R-R).
- **Preorder Traversal**: Root-Left-Right (R-L-R).
- **Postorder Traversal**: Left-Right-Root (L-R-R).

## **Representation of Binary Trees**

- **Inorder Representation**: A string representation of the tree in inorder traversal order.
- **Preorder Representation**: A string representation of the tree in preorder traversal order.
- **Postorder Representation**: A string representation of the tree in postorder traversal order.

## **Operations on Binary Trees**

- **Insertion**: Adding a new node to the tree.
- **Deletion**: Removing a node from the tree.
- **Search**: Finding a node with a given key in the tree.
- **Minimum/Maximum Finding**: Finding the node with the minimum/maximum key in the tree.

## **Important Theorems**

- **Breadth-First Search (BFS) Theorem**: A tree with n nodes has at most n-1 edges.
- **Depth-First Search (DFS) Theorem**: A tree with n nodes has at least n-1 edges.
