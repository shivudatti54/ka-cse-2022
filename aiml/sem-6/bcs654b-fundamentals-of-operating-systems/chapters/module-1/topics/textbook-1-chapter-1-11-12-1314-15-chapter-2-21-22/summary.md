# Fundamentals of Operating Systems: Revision Notes

==============================================

### Chapter 1: Introduction to Operating Systems

- **1.1: What is an Operating System (OS)?**:
  - A software application that manages computer hardware resources
  - Provides common services to computer programs running on the hardware
- **1.2: Computer System Organization**:
  - A hierarchical structure consisting of hardware and software components
  - Includes the Central Processing Unit (CPU), Memory, Storage Devices, and Input/Output Devices
- **1.3: Types of Operating Systems**:
  - Batch Operating Systems
  - Time-Sharing Operating Systems
  - Multi-User Operating Systems
  - Multi-Tasking Operating Systems
- **1.4: Operating System Characteristics**:
  - Efficiency
  - Reliability
  - Portability
  - Security
- **1.5: Advantages and Disadvantages of Operating Systems**:
  - Advantages:
    - Improved productivity
    - Simplified software development
    - Increased system reliability
  - Disadvantages:
    - Complexity
    - Security risks

### Chapter 2: Basic Concepts of Operating Systems

- **2.1: Process Concept**:
  - A program in execution
  - Includes program code, data, and control structures
- **2.2: Process Scheduling**:
  - Allocating the CPU to a process
  - Types of Scheduling:
    - First-Come-First-Served (FCFS)
    - Shortest Job First (SJF)
    - Priority Scheduling
- **2.2.1: Process Scheduling Algorithms**:
  - FCFS
  - SJF
  - Priority Scheduling
- **2.2.2: Deadlocks**:
  - A situation where two or more processes are unable to proceed
  - Types of Deadlocks:
    - Living Deadlock
    - Starvation Deadlock
- **2.3: Memory Management**:
  - Managing the allocation and deallocation of memory
  - Types of Memory Management:
    - Virtual Memory
    - Segmentation
- **2.3.2: Page Replacement Algorithms**:
  - First-In-First-Out (FIFO)
  - Least Recently Used (LRU)
  - Optimal Page Replacement (OPR)
- **2.3.3: Memory Protection**:
  - Preventing a process from accessing unauthorized memory regions
  - Types of Memory Protection:
    - Hardware-based
    - Software-based
