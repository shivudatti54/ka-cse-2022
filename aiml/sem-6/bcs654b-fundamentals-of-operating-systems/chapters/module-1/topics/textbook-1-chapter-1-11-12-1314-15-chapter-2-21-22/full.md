# **FUNDAMENTALS OF OPERATING SYSTEMS**

## **Module: Introduction**

### Section 1.1: What are Operating Systems?

**Definition:** An operating system (OS) is a software that manages computer hardware resources and provides common services to computer programs. It acts as an intermediary between computer hardware and user-level applications.

**Historical Context:** The first operating system, CP-40, was developed in 1961 for the IBM 7090 computer. However, it was not until the development of the IBM System/360 in the late 1960s that operating systems became more sophisticated and widely used.

**Key Characteristics:** Operating systems have several key characteristics, including:

- **Process Management:** The ability to manage multiple programs simultaneously.
- **Memory Management:** The ability to manage computer memory and allocate it to programs.
- **File System Management:** The ability to manage files and directories.
- **Input/Output Management:** The ability to manage input/output operations.

### Section 1.2: Computer System Organization

**Definition:** A computer system is a collection of hardware and software components that work together to perform tasks.

**Components:** A computer system typically consists of:

- **Central Processing Unit (CPU):** The brain of the computer, responsible for executing instructions.
- **Memory:** The storage device that holds data and programs.
- **Input/Output Devices:** Devices that allow users to interact with the computer.
- **Storage Devices:** Devices that store data, such as hard drives and solid-state drives.

**Organization:** Computer systems can be organized in different ways, including:

- **Single-User, Single-Tasking:** One user can use the computer at a time, and only one program can run at a time.
- **Multi-User, Multi-Tasking:** Multiple users can use the computer at the same time, and multiple programs can run at the same time.

### Section 1.3: Operating System Functions

**Definition:** Operating system functions are the tasks that an operating system performs to manage computer hardware resources and provide common services to computer programs.

**Functions:** Operating system functions include:

- **Process Management:** Creating, scheduling, and terminating processes.
- **Memory Management:** Allocating and deallocating memory to programs.
- **File System Management:** Creating, deleting, and managing files and directories.
- **Input/Output Management:** Managing input/output operations.

### Section 1.4: Operating System Architecture

**Definition:** Operating system architecture refers to the design and organization of an operating system.

**Components:** Operating system architecture typically consists of:

- **Kernel:** The core of the operating system, responsible for managing computer hardware resources.
- **Device Drivers:** Programs that manage input/output devices.
- **System Libraries:** Collections of pre-written code that provide common functions.

### Section 1.5: Operating System Types

**Definition:** Operating system types refer to the different types of operating systems that exist.

**Types:** There are several types of operating systems, including:

- **Single-User Operating System:** Designed for one user.
- **Multi-User Operating System:** Designed for multiple users.
- **Real-Time Operating System:** Designed for applications that require predictable responses.
- **Embedded Operating System:** Designed for embedded systems, such as traffic lights and microwave ovens.

### Section 2.1: Operating System Services

**Definition:** Operating system services are the functions that an operating system provides to computer programs.

**Services:** Operating system services include:

- **Process Creation:** Creating new processes.
- **Process Scheduling:** Scheduling processes to run.
- **Memory Allocation:** Allocating memory to programs.
- **File System Operations:** Performing file system operations, such as creating and deleting files.

### Section 2.2.1: File System Management

**Definition:** File system management refers to the management of files and directories.

**Functions:** File system management functions include:

- **File Creation:** Creating new files.
- **File Deletion:** Deleting existing files.
- **File Copying:** Copying files.
- **File Moving:** Moving files.

### Section 2.2.2: File System Hierarchies

**Definition:** File system hierarchies refer to the organization of files and directories.

**Types:** There are several types of file system hierarchies, including:

- **Flat File System:** A simple file system with no directory structure.
- **Hierarchical File System:** A file system with a directory structure.
- **Network File System:** A file system that allows multiple computers to share files.

### Section 2.3.1: Input/Output Management

**Definition:** Input/output management refers to the management of input/output operations.

**Functions:** Input/output management functions include:

- **Input Operations:** Reading data from input devices, such as keyboards and mice.
- **Output Operations:** Writing data to output devices, such as printers and displays.

### Section 2.3.2: I/O Modes

**Definition:** I/O modes refer to the different modes in which input/output operations can occur.

**Modes:** There are several modes of input/output operations, including:

- **Bounded I/O:** I/O operations are bounded by a buffer.
- **Unbounded I/O:** I/O operations are not bounded by a buffer.
- **Interrupt-Driven I/O:** I/O operations are triggered by interrupts.
