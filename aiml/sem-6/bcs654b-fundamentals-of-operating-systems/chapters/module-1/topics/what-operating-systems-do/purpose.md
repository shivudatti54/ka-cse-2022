# Learning Objectives
After studying this topic, you should be able to:
1.  Explain the two fundamental roles of an operating system: as an extended machine and as a resource manager.
2.  Identify and describe the four main management functions of an operating system: process, memory, file system, and device management.
3.  Differentiate between the concepts of spooling, buffering, and caching in the context of I/O management.
4.  Explain how the operating system creates the illusion of multitasking through process scheduling.
5.  Describe the purpose of memory protection and how it contributes to system stability.
6.  List at least five essential services provided by a typical operating system.
