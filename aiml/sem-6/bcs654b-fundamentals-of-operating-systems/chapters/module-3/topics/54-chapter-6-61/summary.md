**5.4 Chapter 6: 6.1 - Fundamentals of Operating Systems**

### Overview

- Operating System (OS) is a software that manages computer hardware resources
- Provides a platform for running applications and utilities

### Key Concepts

- **Process**: A program in execution
  - Process ID (PID)
  - Process Status
- **Thread**: A lightweight process
  - Thread ID (TID)
  - Thread Status
- **Memory Management**: Allocation and deallocation of memory
  - Virtual Memory
  - Page Replacement Algorithm

### Important Formulas and Definitions

- **CPU Time**: Time spent by a process in executing instructions
- **I/O Time**: Time spent waiting for input/output operations
- **Context Switching**: Switching from one process to another
- **Page Fault**: When a page is not found in memory
- **Cache Memory**: A small, fast memory for frequently used data

### Important Theorems and Concepts

- **Theorem:**
  - "A process requires only one memory location, and each process has its own memory space."
- **Concept:**
  - Multiprogramming: Multiple processes run simultaneously
  - Multitasking: Multiple processes run simultaneously, but only one is visible to the user

### Revision Tips

- Review process and thread concepts
- Understand memory management techniques (virtual memory, page replacement algorithm)
- Familiarize yourself with CPU time, I/O time, context switching, page faults, and cache memory
