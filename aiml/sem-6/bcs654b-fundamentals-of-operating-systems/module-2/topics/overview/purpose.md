# Learning Objectives

After studying this topic, you should students able to:

1. Define a process and explain how it differs from a program
2. Describe the components of a process: code, data, stack, heap, and Process Control Block (PCB)
3. Explain the five process states and the transitions between them
4. Understand process scheduling concepts: job queue, ready queue, and device queues
5. Describe operations on processes including creation (fork) and termination (exit)
6. Explain inter-process communication (IPC) using shared memory and message passing
