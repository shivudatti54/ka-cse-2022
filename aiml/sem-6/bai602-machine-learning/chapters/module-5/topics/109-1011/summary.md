# **10.9-10.11: Artificial Neural Networks**

### Key Concepts

- **Artificial Neural Networks (ANNs)**: Modelled after biological neural networks, ANNs are composed of interconnected nodes (neurons) that process and transmit information.
- **Perceptron**: A type of feedforward neural network that can learn linearly separable patterns.
- **Learning**: The process of updating ANNs to improve performance.

### Important Formulas and Definitions

- **Activation Function**: A mathematical function that introduces non-linearity into the model, e.g. sigmoid, ReLU.
- **Weight Update**: The process of adjusting the connection strengths between neurons based on the error.
  - **Delta Rule**: Δw = -η \* dw
  - **Stochastic Gradient Descent (SGD)**: Update weights using the gradient of the loss function.
- **Loss Function**: A mathematical function that measures the difference between predicted and actual outputs, e.g. Mean Squared Error (MSE), Cross-Entropy Loss (CE).

### Important Theorems

- **Universal Approximation Theorem**: ANNs can approximate any continuous function.
- **No Free Lunch Theorem**: ANNs can be optimized using various algorithms, but no algorithm is universally optimal.

### Key Terms

- **Neuron**: A single node in the neural network that receives input and produces output.
- **Synapse**: The connection between two neurons that allows them to communicate.
- **Backpropagation**: An algorithm used to train ANNs by propagating the error backwards.
