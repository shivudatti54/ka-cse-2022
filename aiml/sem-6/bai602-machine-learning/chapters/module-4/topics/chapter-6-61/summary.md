# **Decision Tree Learning: Revision Notes (Chapter-6, 6.1)**

### Introduction

- Decision Tree Learning is a supervised learning algorithm used for classification and regression tasks.
- It involves creating a tree-like model of decisions to solve a problem.

### Decision Tree Induction

- Decision Tree Induction is the process of creating a decision tree from a dataset.
- The process involves:
  - **Splitting**: Divide the data into subsets based on a feature.
  - **Purification**: Remove impurities from the data.
  - **Stopping**: Stop splitting when a stopping criterion is met.

### Decision Tree Model

- A decision tree model consists of:
  - **Root Node**: The starting point of the tree.
  - **Decision Nodes**: Nodes that split the data.
  - **Leaf Nodes**: Nodes that represent a class or target value.
  - **Edges**: Connections between nodes.

### Important Formulas and Definitions

- **Entropy (H)**: A measure of the amount of uncertainty in a dataset.
- **Information Gain (IG)**: A measure of the reduction in uncertainty after splitting the data.
- **Decision Tree Induction Formula**: `IDT = (DT, P)` where `DT` is the decision tree and `P` is the probability of the class label.

### Important Theorems

- **Theorem 1**: If `IDT` is a decision tree, then `IDT` is a good inducer if and only if `IDT` minimizes the entropy of the data.

### Key Concepts

- **Overfitting**: When a decision tree is too complex and performs well on the training data but poorly on the testing data.
- **Underfitting**: When a decision tree is too simple and performs poorly on both the training and testing data.

### Quick Revision Tips

- Understand the concept of entropy and information gain.
- Know how to create a decision tree using a dataset.
- Be aware of overfitting and underfitting.
