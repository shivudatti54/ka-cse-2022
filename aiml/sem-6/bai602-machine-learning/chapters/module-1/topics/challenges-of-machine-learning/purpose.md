# Why Study Types of Machine Learning?

## Big Picture Foundation
Understanding the three core learning types helps you quickly classify any ML problem and choose the right algorithm family. This is a **first principle** in machine learning curricula.

## Exam Relevance
| Aspect | Details |
|--------|---------|
| Frequency | Appears in most introductory ML exams |
| Marks | 5-10 marks (definitions, comparisons, examples) |
| Question Types | Short answers, comparative essays, application-based questions |

## Practical Value
- **Correct framing** prevents wasted effort (e.g., using supervised methods on unlabeled data).
- **Better model selection** and evaluation metrics.
- **Clear communication** with stakeholders about data requirements and outcomes.

## Learning Outcomes
After this topic, you should be able to:
1. Distinguish supervised, unsupervised, and reinforcement learning
2. Map real problems to the correct learning type
3. Select appropriate evaluation metrics for each type
