### 2.1: Introduction to Machine Learning

- Machine learning is a subset of artificial intelligence that enables systems to learn from data without being explicitly programmed.
- Definition: Machine learning is a field of study that gives computers the ability to automatically learn and improve from experience without being explicitly programmed.
- Types of machine learning:
  - Supervised learning
  - Unsupervised learning
  - Reinforcement learning

### 2.2: Supervised Learning

- Definition: Supervised learning is a type of machine learning where the algorithm learns from labeled data.
- Goals:
  - Classification: predict a categorical label
  - Regression: predict a continuous value
- Examples:
  - Image classification
  - Sentiment analysis
  - Predicting house prices

### 2.3: Unsupervised Learning

- Definition: Unsupervised learning is a type of machine learning where the algorithm learns from unlabeled data.
- Goals:
  - Clustering: group similar data points
  - Dimensionality reduction: reduce the number of features
- Examples:
  - Customer segmentation
  - Anomaly detection
  - Image compression

### 2.4: Reinforcement Learning

- Definition: Reinforcement learning is a type of machine learning where the algorithm learns through trial and error by interacting with an environment.
- Goals:
  - Maximize a reward function
  - Minimize a penalty function
- Examples:
  - Robotics
  - Game playing
  - Autonomous vehicles

### 2.5: Key Formulas and Theorems

- **Definition of a Binary Classifier**:
  - P(y=1|X=x) = Pr класса 1 в заданном данных x
  - P(y=0|X=x) = Pr класса 0 в заданном данных x
- **Bayes' Theorem**:
  - P(A|B) = P(B|A) \* P(A) / P(B)
- **Perceptron Learning Rule**:
  - Update w = w - α \* (y - (w^T \* x)) \* x
