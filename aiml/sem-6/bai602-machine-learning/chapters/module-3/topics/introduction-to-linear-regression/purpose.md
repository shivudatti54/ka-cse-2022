# Why Study Linear Regression?

## Big Picture Foundation
Linear regression is the **foundational supervised learning algorithm**. Understanding it deeply provides insight into optimization, loss functions, regularization, and evaluation metrics that apply to all ML models.

## Exam Relevance
| Aspect | Details |
|--------|---------|
| Frequency | Appears in every ML exam |
| Marks | 8-15 marks (derivations, comparisons) |
| Question Types | Derive gradients, compare regularization, explain assumptions |

## Practical Value
- **Interpretability**: Coefficients directly show feature importance
- **Baseline Model**: Often the first model to try; sets performance baseline
- **Industry Standard**: Widely used in finance, economics, and healthcare
- **Building Block**: Foundation for logistic regression, neural networks

## Learning Outcomes
After this topic, you should be able to:
1. Write the linear regression model equation
2. Derive gradient descent updates for MSE
3. Explain the normal equation and when to use it
4. Compare Ridge, Lasso, and Elastic Net regularization
5. Interpret R^2 and other evaluation metrics
