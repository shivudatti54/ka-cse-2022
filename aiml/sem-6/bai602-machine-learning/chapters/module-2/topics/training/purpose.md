# Learning Objectives
After studying this topic, you should be able to:
1. Explain the fundamental concepts behind Word2Vec and the Distributional Hypothesis
2. Differentiate between CBOW and Skip-Gram architectures and their respective use cases
3. Describe the training process of Word2Vec models including negative sampling and hierarchical softmax
4. Implement a basic Word2Vec model using libraries like Gensim
5. Evaluate word embeddings using both intrinsic and extrinsic evaluation methods
6. Identify limitations of Word2Vec and understand how newer approaches address these limitations
