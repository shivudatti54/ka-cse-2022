### Textbook 2: Ch - Machine Translation: Language Divergences and Typology

#### Key Concepts

- **Language Typology**: Study of language structures and their relationships
- **Language Divergences**: Differences between languages, including phonology, morphology, syntax, and semantics
- **Encoder-based Machine Translation**: Use of neural networks to learn language representations

#### Important Formulas and Definitions

- **Word Embeddings**: Representing words as vectors in a high-dimensional space (e.g., Word2Vec, GloVe)
- **Encoder**: Neural network layer that extracts linguistic features from input text
- **Decoder**: Neural network layer that generates output text based on encoded features
- **Attention Mechanism**: Mechanism to focus on specific parts of the input text when generating output
- **Cross-Entropy Loss**: Objective function used to evaluate machine translation models

#### Key Theorems and Results

- **Ullmann's Hypothesis**: Language differences are due to linguistic and historical factors, not just random chance
- **Chomsky's Poverty of the Stimulus**: Human language acquisition is impossible without innate universal grammar
- **Language Typology**: Classification of languages into linguistic families and types (e.g., isolates, agglutinative, fusional)

#### Important Algorithms and Techniques

- **Transformer Architecture**: Neural network architecture for sequence-to-sequence tasks, including machine translation
- **Pre-training**: Training models on large datasets before fine-tuning on specific tasks
- **Fine-tuning**: Adjusting pre-trained models for specific tasks or domains

#### Important Notions and Concepts

- **Neural Machine Translation**: Use of neural networks to translate text from one language to another
- **Statistical Machine Translation**: Use of statistical models to translate text, including n-gram models and machine learning algorithms
- **Hybrid Approaches**: Combining different techniques, such as rule-based and statistical methods, to improve machine translation performance
