# Learning Objectives

After studying this topic, you should be able to:

1.  Understand the concepts of error detection and correction.
2.  Know the different techniques used for error detection and correction.
3.  Understand the concepts of block coding and cyclic codes.
4.  Know the different types of block coding and cyclic codes.
5.  Understand the concept of checksum and its types.
6.  Understand the purpose of PPP and MAC.
