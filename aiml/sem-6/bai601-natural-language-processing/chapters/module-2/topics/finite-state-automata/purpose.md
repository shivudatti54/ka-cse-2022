# Learning Objectives
After studying this topic, you should be able to:
1.  Define and write basic regular expressions using literals, character classes, quantifiers, and anchors.
2.  Explain the core components of a Finite-State Automaton (FSA) and draw a state diagram from a simple description.
3.  Trace the operation of a DFA or NFA on a given input string to determine acceptance or rejection.
4.  Articulate the formal equivalence between regular expressions and finite-state automata.
5.  Convert a simple regular expression into an equivalent FSA and vice-versa.
6.  Describe practical applications of regex and FSA in NLP, specifically for morphological parsing and spelling error detection.
