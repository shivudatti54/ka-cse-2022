# Learning Objectives
After studying this topic, you should be able to:
1.  Define Chomsky Normal Form (CNF) and convert a simple Context-Free Grammar (CFG) into CNF.
2.  Explain the core principle of dynamic programming as applied in the CYK algorithm.
3.  Construct a CYK parsing table for a given sentence and a CFG in CNF.
4.  Determine whether a sentence is syntactically valid according to a grammar by checking for the presence of the start symbol in the top-right cell of the CYK table.
5.  Analyze the time and space complexity of the CYK algorithm and justify the reasoning.
6.  Discuss the advantages (efficiency) and disadvantages (CNF requirement) of the CYK algorithm compared to other parsing methods.
