# **Text Book 1: Ch 1 - Introduction to Artificial Intelligence**

### Key Concepts

- **Definition of Artificial Intelligence (AI):**
  - A branch of computer science that enables machines to perform tasks that typically require human intelligence
- **Types of AI:**
  - Narrow or Weak AI: designed to perform a specific task
  - General or Strong AI: capable of performing any intellectual task
- **Problem Spaces:**
  - A set of possible solutions to a problem
  - Can be represented as a graph or matrix

### Important Formulas and Definitions

- **Problem Space:** A set of possible solutions to a problem, represented as a graph or matrix
- **Solution:** A possible outcome of a problem, represented as a node in the problem space
- **Heuristic:** A method or technique used to find a solution to a problem
- **Greedy Algorithm:** A heuristic search algorithm that chooses the next state based on a local criterion

### Important Theorems

- **Brute Force Theorem:** A problem can be solved in a finite number of steps if there are a finite number of possible solutions
- **P versus NP Problem:** A problem can be solved in polynomial time if there is a polynomial-time algorithm for solving it

### Important Terms

- **Search Space:** The set of all possible states of a problem
- **State:** A point in the search space
- **Action:** A possible next state in the search space

### Quick Revision Notes

- AI is a branch of computer science that enables machines to perform tasks that typically require human intelligence
- Problem spaces are sets of possible solutions to a problem, represented as graphs or matrices
- Heuristics and greedy algorithms are used to find solutions to problems
- The P versus NP problem is a famous problem in computer science that determines the complexity of solving certain problems.
