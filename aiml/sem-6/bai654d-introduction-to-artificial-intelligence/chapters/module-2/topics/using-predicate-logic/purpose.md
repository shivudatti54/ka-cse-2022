# Learning Objectives
After studying this topic, you should be able to:
1. Differentiate between propositional logic and predicate logic
2. Represent knowledge using predicates, quantifiers, and logical connectives
3. Convert natural language statements into predicate logic formulas and vice versa
4. Apply universal and existential instantiation rules for inference
5. Convert predicate logic formulas to clause form for resolution
6. Perform unification between predicate expressions
7. Identify the limitations of predicate logic for knowledge representation
