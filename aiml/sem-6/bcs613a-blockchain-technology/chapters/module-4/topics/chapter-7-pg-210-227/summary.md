# Blockchain Technology

### Chapter 7: pg: 210-227

### Key Points

#### Ethereum Stack and Blockchain

- Definition: A blockchain is a distributed digital ledger technology.
- The Ethereum stack consists of:
  - Virtual Machine (EVM): executes smart contracts.
  - Consensus Algorithm: ensures network agreement (e.g., Proof of Work (PoW)).
  - Network: interconnected nodes (computers) that validate and add blocks.
  - Blockchain: the distributed ledger containing blocks.

#### Consensus Algorithms

- **Proof of Work (PoW)**:
  - Definition: Requires miners to solve complex mathematical puzzles (computationally expensive).
  - Goal: Validate transactions and add new blocks to the blockchain.
  - Formula: Energy consumption (e.g., Bitcoin) or computational power (e.g., Ethereum).
- **Proof of Stake (PoS)**:
  - Definition: Validators (nodes) "stake" their own cryptocurrency to participate in validation.
  - Goal: Encourages honest behavior and reduces energy consumption.
  - Formula: No specific formula, but validators are chosen based on the size of their "stake".

#### Smart Contracts

- Definition: Self-executing contracts with the terms of the agreement written directly into lines of code.
- Characteristics:
  - Immutable: once deployed, cannot be altered.
  - Transparent: all transactions are public and visible.
  - Autonomous: executes automatically when conditions are met.

#### Hash Functions

- Definition: One-way mathematical functions that take input data (e.g., string, number) and produce a fixed-size output (hash value).
- Purpose: Ensures data integrity and facilitates secure transactions.
- Formula: Not applicable, as hash functions are deterministic and not invertible.

### Important Formulas, Definitions, and Theorems

- **Sha256 (Secure Hash Algorithm 256)**:
  - Formula: Not applicable, as it's a one-way hash function.
  - Definition: A widely used cryptographic hash function.
- **Keccak-256**:
  - Formula: Not applicable, as it's a one-way hash function.
  - Definition: A cryptographic hash function used in Ethereum's consensus algorithm (PoW).
