# Learning Objectives
After studying this topic, you should be able to:
1.  Define and differentiate between the three main types of machine learning: supervised, unsupervised, and reinforcement learning.
2.  Identify real-world applications and examples for each type of learning.
3.  Explain the role of feedback (labels, rewards, none) in each learning paradigm.
4.  Describe the core components and process of a reinforcement learning agent interacting with an environment.
5.  Understand how machine learning techniques can be applied to overcome the knowledge acquisition bottleneck in expert systems.
6.  Compare and contrast the different learning types using a structured table of features.
