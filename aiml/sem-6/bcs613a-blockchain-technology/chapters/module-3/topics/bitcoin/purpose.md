# Learning Objectives
After studying this topic, you should be able to:
1.  Define Bitcoin and explain its core purpose as a decentralized digital currency.
2.  Describe the key components of a Bitcoin transaction and explain the transaction lifecycle using the UTXO model.
3.  Identify and explain the function of each field in a Bitcoin block header.
4.  Differentiate between full nodes and SPV nodes and explain their roles in the Bitcoin network.
5.  Compare and contrast the various types of Bitcoin wallets, including their security trade-offs.
6.  Explain the role of mining and the Proof-of-Work consensus mechanism in securing the network and achieving distributed consensus.
