# Chapter 6 Revision Notes

### Blockchain Technology

- **Definition:** A distributed digital ledger technology that records transactions across a network of computers in a secure and transparent manner.
- **Key Components:**
  - Blocks: collections of transactions
  - Chain: sequence of blocks
  - Nodes: computers that validate transactions and add new blocks
  - Consensus Mechanism: algorithm that verifies transactions and adds new blocks

### Cryptography

- **Definition:** The practice of using algorithms to secure data and communications.
- **Key Concepts:**
  - Symmetric Encryption: uses the same key for encryption and decryption.
  - Asymmetric Encryption: uses a pair of keys (public and private) for encryption and decryption.

### Consensus Mechanisms

- **Definition:** Algorithm that verifies transactions and adds new blocks to the blockchain.
- **Types:**
  - Proof of Work (PoW): energy-intensive algorithm that requires significant computational power.
  - Proof of Stake (PoS): algorithm that rewards validators with tokens or coins.
  - Delegated Proof of Stake (DPoS): variant of PoS that allows users to vote for validators.

### Smart Contracts

- **Definition:** Self-executing contracts with the terms of the agreement written directly into lines of code.
- **Key Concepts:**
  - Self-executing: contracts execute automatically when conditions are met.
  - Turing-Complete: can be used to execute complex logic.

### Blockchain Applications

- **Definition:** The use of blockchain technology to solve real-world problems.
- **Examples:**
  - Supply Chain Management
  - Identity Verification
  - Healthcare Record Management

### Important Formulas and Definitions

- **Hash Function:** a mathematical function that takes input data of any size and produces a fixed-size string of characters, known as a hash.
- **Public-Key Cryptography:** a type of asymmetric encryption that uses a pair of keys (public and private) for encryption and decryption.

### Important Theorems

- **Shannon's Theorem:** states that the capacity of a communication channel is directly proportional to the logarithm of the signal-to-noise ratio.
- **Turing's Halting Problem:** states that there cannot exist an algorithm that can determine, given an arbitrary program and input, whether the program will run forever or eventually halt.
