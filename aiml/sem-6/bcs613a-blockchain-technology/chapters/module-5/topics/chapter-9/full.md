## **Hyperledger Fabric**

Hyperledger Fabric is a distributed ledger technology framework that supports the development of blockchain networks. It is based on a consensus algorithm called the Byzantine Fault Tolerance (BFT) algorithm, which ensures that all nodes on the network agree on the state of the ledger.

Hyperledger Fabric consists of a number of components, including:

- **Peer Nodes**: These are the nodes that make up the blockchain network. Each peer node has a copy of the ledger and can propose new blocks of transactions to the network.
- **Orderer Nodes**: These nodes are responsible for validating and ordering the transactions on the network. They use the BFT algorithm to ensure that all nodes on the network agree on the state of the ledger.
- **CA (Certificate Authority)**: This is a node that issues digital certificates to the peer nodes on the network. These certificates are used to authenticate the identity of the peer nodes.
