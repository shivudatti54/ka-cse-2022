# Learning Objectives
After studying this topic, you should be able to:
1. Differentiate between Type 1 and Type 2 hypervisors and identify examples of each
2. Explain the mechanisms of CPU, memory, and I/O device virtualization
3. Compare and contrast full virtualization, paravirtualization, and hardware-assisted virtualization
4. Identify major virtualization tools and platforms and their appropriate use cases
5. Describe how virtualization enables data center automation and resource management
6. Understand the role of virtualization in cloud computing architecture
