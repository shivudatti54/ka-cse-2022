# **Chap-3 (3.1) Computer Vision: Image Formation**

## **Key Concepts**

- **Photometric Image**: An image formed by the combination of light intensity and spatial coordinates.
- **Image Formation**: The process of converting 3D scene into 2D image.
- **Pinhole Camera Model**: A simplified model of a camera that assumes a small aperture (pinhole) and a flat image plane.

## **Definitions**

- **Image**: A 2D representation of a 3D scene.
- **Scene**: The 3D environment in which the image is formed.
- **Viewpoint**: The location of the observer (camera) in the scene.

## **Formulas**

- **Camera Lens Equation**: 1/f = 1/do + 1/di (where f = focal length, do = object distance, di = image distance)
- **Pinhole Camera Model**: y = (x \* f) / L (where y = image coordinate, x = object coordinate, f = focal length, L = image distance)

## **Theorems**

- **Perspective Projection**: The projection of a 3D scene onto a 2D plane using linear perspective.
- **Orthographic Projection**: The projection of a 3D scene onto a 2D plane using parallel lines.

## **Important Results**

- **Camera Calibration**: The process of determining the intrinsic and extrinsic parameters of a camera.
- **Image Formation Equations**: The equations that describe the relationship between the 3D scene and the resulting 2D image.

## **Revision Tips**

- Understand the difference between photometric image and image formation.
- Familiarize yourself with the pinhole camera model and its applications.
- Practice calculating the image coordinates using the camera lens equation and pinhole camera model.
