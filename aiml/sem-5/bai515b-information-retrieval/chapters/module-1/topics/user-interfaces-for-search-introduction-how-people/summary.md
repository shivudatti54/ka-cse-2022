# **User Interfaces for Search: Revision Notes**

## **Introduction**

- User interfaces (UIs) play a crucial role in information retrieval (IR) systems.
- Effective UI design can improve search experience and user satisfaction.

## **How People Search**

- **Cognitive search theory**: users search for information using mental representations of the search task.
- **Information retrieval model**: users search for information using a query term or phrase.
- **User behavior**: users exhibit variability in search behavior, including search intent, query strings, and search history.

## **Search Interfaces Today**

- **Text-based interfaces**: simple text-based interfaces, such as search boxes or menus.
- **Graphical interfaces**: visual interfaces, such as search bars or navigation menus.
- **Hybrid interfaces**: combine text-based and graphical interfaces.

## **Visualization on Search Interfaces**

- **Faceted search**: uses visual interfaces to filter search results based on attributes.
- **Faceted categories**: use visual interfaces to organize search results by categories.
- **Ranking visualizations**: use visual interfaces to display search results ranking.

## **Design and Evaluation of Search Interfaces**

- **Formative evaluation**: evaluate UI design during development.
- **Summative evaluation**: evaluate UI design after completion.
- **Usability metrics**: use metrics, such as click-through rates and user satisfaction.

## **Important Formulas, Definitions, and Theorems**

- **Query expansion**: formula to expand query terms to improve search accuracy.
- **Term frequency-inverse document frequency (TF-IDF)**: definition of a relevance ranking formula.
- **Latent Semantic Analysis (LSA)**: theorem to analyze the semantic meaning of words.
