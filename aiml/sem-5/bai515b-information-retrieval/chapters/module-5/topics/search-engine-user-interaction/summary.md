# Search Engine User Interaction

### Overview

Search engine user interaction refers to the process by which users interact with search engines to retrieve relevant information.

### Key Concepts

- **Search Query**: A user's input to a search engine, typically a keyword or phrase.
- **Search Engine Algorithm**: The formulas and techniques used to rank search results, such as PageRank, TF-IDF, and cosine similarity.
- **User Behavior**: How users interact with search engines, including query types, search frequency, and click-through rates.

### Important Formulas and Definitions

- **PageRank**: A formula developed by Google to rank web pages. It uses the concept of link analysis to determine the importance of a page.
  - $PR(A) = \frac{\sum_{i=1}^{n} \frac{1}{C_i}}{n}$
  - $C_i = \text{number of incoming links to page A from page i}$
- **TF-IDF (Term Frequency-Inverse Document Frequency)**: A formula used to weigh the importance of words in a document.
  - $TF(X) = \text{frequency of word X in document}$
  - $IDF(X) = \log \left( \frac{N}{n(X)} \right)$
  - $TF-IDF(X) = TF(X) \cdot IDF(X)$
- **Cosine Similarity**: A measure of similarity between two vectors.
  - $\cos(\theta) = \frac{u \cdot v}{\|u\| \|v\|}$

### Theorems

- **Relevance Theorem**: States that the most relevant search results should be ranked highest.
- **User Satisfaction Theorem**: States that users are more likely to return to a website that meets their information needs.

### Key Theorems in Search Engine User Interaction

- **Relevance**: The relevance of search results to the user's query.
- **Precision**: The number of relevant search results among all the search results.
- **Recall**: The number of relevant search results among all the relevant results.
- **Click-Through Rate (CTR)**: The percentage of users who click on a search result after seeing it.
- **Bounce Rate**: The percentage of users who leave a website after visiting it.
