# **Web Data**

### Definition

- Web data refers to the large amount of structured and unstructured data available on the World Wide Web.

### Types of Web Data

- **Structured data**: Organized and formatted data, such as HTML, XML, and JSON.
- **Unstructured data**: Free-form data, such as text, images, and videos.

### Characteristics of Web Data

- **Volume**: Large amounts of data are generated daily.
- **Velocity**: Data is constantly being created and updated.
- **Variety**: Data comes in different formats and structures.
- **Veracity**: Data quality and accuracy vary.

### Web Data Sources

- **Web pages**: HTML, CSS, JavaScript, and other web page elements.
- **Social media**: User-generated content, such as tweets, posts, and comments.
- **Databases**: Structured data stored in relational databases.
- **APIs**: Application programming interfaces providing access to data.

### Important Formulas and Theorems

- **PageRank**: A algorithm for ranking web pages based on link popularity.
- **TF-IDF**: A technique for calculating the importance of words in a document.
- **Latent Semantic Analysis (LSA)**: A method for analyzing relationships between words and topics.

### Definitions

- **Web crawl**: The process of periodically scanning and indexing web pages.
- **Web scraping**: The process of extracting data from web pages using automated tools.
- **Data mining**: The process of discovering patterns and relationships in large datasets.

### Key Concepts

- **Data normalization**: Standardizing data format for easier processing.
- **Data integration**: Combining data from multiple sources.
- **Data storage**: Storing data in a format that can be retrieved efficiently.

### Theories and Models

- **IR (Information Retrieval) models**: Models that describe how to retrieve relevant documents from a large collection.
- **Semantic Web**: A vision for the web that enables machines to understand the meaning of web data.
- **Web 2.0**: A term that describes the shift towards user-generated content and social media.
