# Learning Objectives
After studying this topic, you should be able to:
1. Explain the fundamental principles of probabilistic approaches to link prediction
2. Differentiate between probabilistic and heuristic methods for link prediction
3. Describe the assumptions and mechanics of Stochastic Block Models
4. Understand how latent feature models work in continuous space
5. Implement basic probability calculations for different probabilistic models
6. Evaluate probabilistic link prediction models using appropriate metrics
