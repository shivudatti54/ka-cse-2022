# Learning Objectives
After studying this topic, you should be able to:
1.  Describe the architecture of a standard Information Retrieval (IR) system and the function of each component.
2.  Explain the structure and purpose of an inverted index and construct one from a small sample corpus.
3.  Compare and contrast the Boolean, Vector Space, and Probabilistic IR models, highlighting the advantages and disadvantages of each.
4.  Calculate TF-IDF weights and cosine similarity for simple document and query vectors.
5.  Evaluate the effectiveness of an IR system using the concepts of precision and recall.
6.  Explain how lexical resources like WordNet and FrameNet, as well as techniques like stemming, can be used to improve IR performance.
