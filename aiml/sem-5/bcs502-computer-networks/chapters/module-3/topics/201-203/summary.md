# **20.1-20.3 Revision Notes**

## **Network Layer Services**

- **Connectionless vs. Connection-Oriented**: Connectionless (e.g., IP, UDP) vs. Connection-Oriented (e.g., TCP)
- **Packet Switching**: Data broken into small packets, transmitted independently, and reassembled at the destination
- **Virtual Circuit**: Establishment of a temporary connection between two endpoints for data transmission

## **Packet Switching**

- **Packet Switching Theorem**: The packet switching theorem states that it is possible to send data from any point to any other point in a network using packet switching.
- **Packet Switching Network**: A network where data is transmitted in packets, and the packets are routed between nodes based on their IP addresses.

## **IP Addressing**

- **IPv4 Address**: A 32-bit address consisting of four octets (e.g., 192.168.1.1)
- **IPv6 Address**: A 128-bit address consisting of eight groups of four hexadecimal digits (e.g., 2001:0db8:85a3:0000:0000:8a2e:0370:7334)
- **Subnetting**: Division of a network into smaller subnets using subnet masks.

## **IPv4 Datagram**

- **Datagram**: A self-contained unit of data transmitted over a network
- **Header**: The portion of the datagram that contains control information (e.g., source and destination IP addresses)
- **Payload**: The portion of the datagram that contains the actual data being transmitted

## **Important Formulas and Definitions**

- **Packet Loss**: The number of packets that are lost during transmission
- **Packet Delay**: The time it takes for a packet to travel from the source to the destination
- **Jitter**: The variation in packet delay
- **Throughput**: The rate at which data is transmitted over a network

## **Theorems**

- **Store-and-Forward**: A packet is stored at each node before being forwarded to the next node.
- **Connectionless Routing**: Packets are routed independently based on their IP addresses.
