# Controlled Access

## Overview

Controlled Access is a method of managing multiple users within a computer network to prevent network congestion and ensure efficient use of network resources.

### Key Points

- **Definition:** Controlled Access is a technique used to manage multiple users within a network to prevent network congestion and ensure efficient use of network resources.
- **Types of Controlled Access:**
  - Token Ring
  - CSMA/CA (Carrier Sense Multiple Access/Collision Avoidance)
  - CSMA/CD (Carrier Sense Multiple Access/Collision Detect)
  - Priority Queuing
- **Key Protocols:**
  - Ethernet (CSMA/CD)
  - Token Ring
  - Wi-Fi (CSMA/CA)
- **Advantages:**
  - Prevents network congestion
  - Ensures efficient use of network resources
  - Reduces collisions
- **Disadvantages:**
  - Increased complexity
  - Higher costs

### Formulas and Theorems

- **Probability of Collision:** P(c) = (1 - P(non-collision)) \* (number of nodes - 1)
- **Collision Detection Time:** t_c = 1 / (2 \* P(non-collision))

### Important Concepts

- **Token:** A token is a unique identifier used to grant access to the network.
- **Priority:** Priority is assigned to users based on their importance or urgency.
- **Queueing Theory:** Queueing theory is used to analyze and model the behavior of multiple users within a network.

### Definitions

- **Access Point (AP):** An access point is a device that connects users to a network.
- **Collision:** A collision occurs when two or more users transmit data simultaneously, causing interference.
- **Collision Avoidance:** Collision avoidance is a technique used to prevent collisions by detecting the presence of other users before transmitting data.
