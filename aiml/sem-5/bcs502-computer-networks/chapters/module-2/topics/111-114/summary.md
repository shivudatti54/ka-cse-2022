# **11.1 -11.4: Quick Revision Notes**

## **11.1: Introduction to Error Detection and Correction**

- Error detection and correction are crucial in computer networks to ensure reliable data transmission.
- Errors can occur due to signal degradation, interference, or transmission errors.
- Types of errors:
  - Bit errors (single-bit errors, double-bit errors)
  - Frame errors (loss of data frames)
- Error detection techniques:
  - Parity bits
  - Checksums
  - Cyclic redundancy checks (CRCs)

## **11.2: Block Coding**

- Block coding is a technique used to detect and correct errors in digital data.
- Block codes divide data into fixed-length blocks.
- Types of block codes:
  - Cyclic codes
  - Non-cyclic codes
  - Convolutional codes

## **11.3: Cyclic Codes**

- Cyclic codes are a type of block code that can detect and correct errors.
- Properties of cyclic codes:
  - Periodicity
  - Shift-invariance
  - Divisibility
- Cyclic code formulas:
  - Generator polynomial (G(x))
  - Feedback polynomial (E(x))

## **11.4: Data Link Layer Error Control**

- Data link layer error control techniques:
  - Error detection and correction using block codes and cyclic codes
  - Forward error correction (FEC)
  - Automatic repeat request (ARQ)

**Important Formulas and Definitions:**

- Hamming distance: minimum number of positions at which two code words differ
- Hamming bound: upper limit on the minimum distance of a code
- Error-correcting code (ECC): code that can detect and correct errors
- Forward error correction (FEC): technique used to detect and correct errors
- Automatic repeat request (ARQ): protocol used to detect and correct errors
