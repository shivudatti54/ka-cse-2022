# Physical Layer

### Transmission Media

- **Guided Media:** Cable (twisted pair, coaxial, fiber optic)
  - Advantages: Increased bandwidth, Fast transmission speed
  - Disadvantages: Limited flexibility, Higher installation cost
- **Unguided Media:** Wireless (RF, Microwaves, Infrared)
  - Advantages: Flexibility, Easy installation
  - Disadvantages: Interference, Limited bandwidth, Security concerns

### Key Concepts

- **Bandwidth:** Measured in bits per second (bps)
- **Frequency:** Measured in Hertz (Hz)
- **Wavelength:** Measured in meters (m)

# TCP/IP Protocol Suite and OSI Model

### TCP/IP Model

- **Layer 1: Physical Layer:** Deals with transmission media
- **Layer 2: Data Link Layer:** Frames and error detection
- **Layer 3: Network Layer:** Routing and fragmentation
- **Layer 4: Transport Layer:** Segmentation and reassembly
- **Layer 5: Session Layer:** Establishes connections
- **Layer 6: Presentation Layer:** Data compression and encryption
- **Layer 7: Application Layer:** Provides services to end-user

### TCP/IP Protocol Suite

- **TCP (Transmission Control Protocol):**
  - Connection-oriented
  - Reliable, error-checked, and sequence-numbered packets
- **IP (Internet Protocol):**
  - Connectionless
  - Routes packets between networks

## Important Formulas and Definitions

- **Bandwidth:** B = bw × t (bits per second = bandwidth × time)
- **Frequency:** f = c / λ (frequency = speed of light / wavelength)
- **Wavelength:** λ = c / f (wavelength = speed of light / frequency)

## The OSI Model Theorem

- **The OSI Model is a seven-layered framework for understanding and designing computer networks.**
