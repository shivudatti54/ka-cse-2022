# **Revision Notes: 7.1-7.3 - Computer Networks**

### 7.1: Network Topologies

- **Definition:** Physical arrangement of devices in a network
- **Types:**
  - **Bus Topology:** all devices connect to a single cable
  - **Star Topology:** all devices connect to a central device (hub or switch)
  - **Ring Topology:** devices connect in a circular configuration
  - **Mesh Topology:** each device connects to every other device
- **Advantages/Disadvantages:**
  - Bus: easy to install, but difficult to troubleshoot
  - Star: easy to troubleshoot, but expensive
  - Ring: resilient, but difficult to install
  - Mesh: resilient, but expensive

### 7.2: Network Models

- **Definition:** conceptual representation of a network
- **Types:**
  - **Physical Model:** represents the physical layout of devices
  - **Functional Model:** represents the functions performed by devices
  - **Logical Model:** represents the data and processes transmitted between devices
- **Functions:**
  - **Packet Switching:** breaks data into packets and transmits them independently
  - **Circuit Switching:** establishes a dedicated circuit for data transmission

### 7.3: Protocol

- **Definition:** set of rules governing data transmission and communication
- **Types:**
  - **Packet Switching:** protocols such as TCP/IP, DNS, and DHCP
  - **Circuit Switching:** protocols such as T1 and T3
- **Key Protocols:**
  - **TCP (Transmission Control Protocol):** ensures data is delivered in the correct order
  - **IP (Internet Protocol):** ensures data is delivered over IP networks
  - **HTTP (Hypertext Transfer Protocol):** governs data transmission over the web

### Important Formulas/Definitions/Theorems:

- ** OSI Model:** 7 layers (Physical, Data Link, Network, Transport, Session, Presentation, Application)
- ** TCP Three-Way Handshake:** SYN, SYN-ACK, ACK
- ** IP Address Classes:** A, B, C, D
