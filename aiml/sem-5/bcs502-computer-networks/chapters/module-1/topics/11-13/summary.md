### 1.1 - 1.3: Computer Networks

#### Introduction

- A computer network is a group of interconnected devices that communicate with each other.
- The primary functions of a computer network are:
  - Data transmission
  - Data storage
  - Data sharing

#### Key Concepts

- **Network Topology**: The physical or logical arrangement of devices in a network.
  - Examples: Bus, Star, Ring, Mesh
  - Key concepts:
    - **Node**: A device in a network.
    - **Link**: A connection between two nodes.
- **Network Models**: Theoretical frameworks used to analyze and design networks.
  - Examples: OSI (Open Systems Interconnection), TCP/IP (Transmission Control Protocol/Internet Protocol)
- **Network Protocols**: Sets of rules that govern data communication between devices.
  - Examples: HTTP (Hypertext Transfer Protocol), FTP (File Transfer Protocol)

#### Theorems and Formulas

- **Theorem of Network Reliability**: The probability of a network being reliable is equal to the probability of each link being reliable.
- **Dilworth's Theorem**: The maximum number of disjoint dominating sets in a graph is equal to the minimum number of vertices that must be removed to make the graph a tree.

### Key Definitions

- **Node**: A device in a network.
- **Link**: A connection between two nodes.
- **Network Interface**: A device that connects a computer to a network.
- **Router**: A device that connects multiple networks together.
- **Switch**: A device that connects multiple devices within a network.
