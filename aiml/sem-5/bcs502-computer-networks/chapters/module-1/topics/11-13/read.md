# **1.1 - 1.3: Introduction to Computer Networks**

### 1.1: Data Communications

#### Definition:

Data communication refers to the process of sending and receiving data between devices over a communication medium, such as a physical or wireless link.

#### Explanation:

Data communication is the primary function of computer networks. It involves the transfer of data between devices, such as computers, printers, and servers, over a network. The data can be in the form of text, images, audio, or video.

#### Key Concepts:

- **Data transmission**: The process of sending data from one device to another.
- **Data reception**: The process of receiving data from one device to another.
- **Data link**: A segment of a network that connects two devices and enables data transmission.
- **Packet switching**: A method of data transmission where data is broken into small packets and transmitted over the network.

#### Example:

Suppose you have a printer connected to a computer via a network cable. When you print a document, the computer sends the data to the printer using packet switching. The printer receives the packets and reassembles the data to print the document.

### 1.2: Networks

#### Definition:

A network is a collection of interconnected devices that communicate with each other to share resources and exchange data.

#### Explanation:

A network is a fundamental concept in computer science that enables devices to communicate with each other. Networks can be categorized into two main types: wired networks and wireless networks.

#### Key Concepts:

- **Network topology**: The physical or logical arrangement of devices in a network.
- **Network architecture**: The design and structure of a network.
- **Network protocols**: Sets of rules and standards that govern data communication between devices.
- **Network devices**: Devices that connect to a network, such as routers, switches, and hubs.

#### Example:

Imagine a home network with multiple computers, a printer, and a router. The computers are connected to the router via Ethernet cables, and the router connects to the internet via a broadband connection. When you print a document from one computer, the data is transmitted to the printer through the network.

### 1.3: Network Models

#### Definition:

A network model is a conceptual representation of a network, used to describe its structure, functions, and relationships.

#### Explanation:

Network models are used to design, analyze, and understand the behavior of networks. There are several network models, each with its own strengths and weaknesses.

#### Key Concepts:

- **Physical model**: A model that describes the physical layout of a network.
- **Logical model**: A model that describes the functional and organizational structure of a network.
- **Communication model**: A model that describes the communication between devices in a network.
- **Applicability model**: A model that describes the network's ability to meet the user's requirements.

#### Example:

Suppose we are designing a network for a small business. We can use a physical model to visualize the layout of the network, including the location of devices and cables. We can also use a logical model to describe the network's architecture and functions. This will help us to identify the network's strengths and weaknesses and to ensure that it meets the business's needs.
