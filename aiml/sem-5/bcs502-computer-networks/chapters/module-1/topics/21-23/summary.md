# **Computer Networks Revision Notes (2.1 - 2.3)**

### 2.1 Network Models

- **Physical Model**: Describes the physical components of a network (e.g., cables, routers, switches)
- **Data Link Layer Model**: Focuses on the data link layer protocols (e.g., Ethernet, Wi-Fi)
- **Network Layer Model**: Concerned with routing and addressing (e.g., IP, ICMP)
- **Transport Layer Model**: Deals with data transfer and reliability (e.g., TCP, UDP)
- **Application Layer Model**: Covers network services and protocols (e.g., HTTP, FTP)

### 2.2 Network Topologies

- **Bus Topology**: A single cable connects all devices
- **Star Topology**: Devices connect to a central hub
- **Ring Topology**: Devices form a circular connection
- **Mesh Topology**: Each device connects to every other device

### 2.3 Network Protocols

- **OSI Model**:
  - Physical Layer (Layer 1): Defines physical network interfaces
  - Data Link Layer (Layer 2): Ensures error-free data transfer
  - Network Layer (Layer 3): Routes data between networks
  - Transport Layer (Layer 4): Ensures reliable data transfer
  - Session Layer (Layer 5): Establishes, manages sessions
  - Presentation Layer (Layer 6): Converts data into a usable format
  - Application Layer (Layer 7): Provides network services
- **TCP/IP Model**:
  - Network Access Layer (Layer 1-3): Similar to OSI model
  - Internet Layer (Layer 4-5): Routes data between networks
  - Transport Layer (Layer 6): Ensures reliable data transfer

### Important Formulas and Definitions

- **Packet Switching**: Breaking data into small packets for efficient transmission
- **Routing Table**: A table used by routers to determine the best path for data transmission
- **IP Address**: A unique address assigned to each device on a network
- **MAC Address**: A unique hardware address assigned to each network interface
- **TCP (Transmission Control Protocol)**: A protocol that ensures reliable data transfer over IP networks
- **UDP (User Datagram Protocol)**: A protocol that provides best-effort delivery of data over IP networks
