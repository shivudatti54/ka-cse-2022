## **What is a Point-to-Point Network?**

A point-to-point network is a type of network that connects two devices directly, without any intermediate devices. In a point-to-point network, each device is connected to another device using a physical link, such as a cable or wireless link. The two devices are connected in a one-to-one manner, and there is no intermediate device between them.

## **What are Duplex Links?**

A duplex link is a type of link that allows data to be transmitted in both directions, simultaneously. Duplex links have two transmission paths, one for transmitting data and another for receiving data. This allows for faster data transfer rates and improved network performance.

## **Components of a Point-to-Point Network**

A point-to-point network consists of the following components:

- **Nodes**: These are the devices that make up the network, such as computers, routers, or switches.
- **Links**: These are the physical connections between nodes, such as cables or wireless links.
- **Network Interface Cards (NICs)**: These are the cards installed in each node that allow it to connect to the network.

4.  **Configure the Router**: Configure the router to connect to the nodes.

- **Links**: Physical connections between devices in a network.
