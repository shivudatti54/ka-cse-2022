# **Practical Component of IPCC Sl.No Experiments 1: Implementing a Point-to-Point Network**

## **Key Points**

- **Point-to-Point Network**: A network where data is transmitted directly between two devices.
- **Duplex Links**: A link that can transmit data in both directions simultaneously.
- **Node**: A device in a network that can communicate with other devices.

## **Network Topology**

- **Three-Node Point-to-Point Network**: A network with three nodes (A, B, and C) connected with duplex links.

## **Network Configuration**

- **Link Configuration**:
  - A: B ( duplex link)
  - B: C (duplex link)
  - C: A (duplex link)
- **Node Configuration**:
  - Node A: Has a connection to Node B and Node C
  - Node B: Has a connection to Node A and Node C
  - Node C: Has a connection to Node A and Node B

## **Key Formulas and Definitions**

- **Bit Error Rate (BER)**: The probability of a single bit being transmitted incorrectly.
- **Packet Error Rate (PER)**: The probability of a single packet being transmitted incorrectly.
- **Network Throughput**: The rate at which data is transmitted over the network.

## **Important Theorems**

- **Theorem of Network Congestion**: The network congestion is determined by the number of nodes and links in the network.
- **Theorem of Network Capacity**: The network capacity is determined by the bandwidth of the links and the number of nodes in the network.
