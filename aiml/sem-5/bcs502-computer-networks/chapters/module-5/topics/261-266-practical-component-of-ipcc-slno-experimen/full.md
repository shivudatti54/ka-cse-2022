# **Practical Component of IPCC Sl.No Experiments 1: Implementing a Point-to-Point Network with Duplex Links**

## **Introduction**

In this practical component, we will focus on implementing a point-to-point network with duplex links between three nodes. A point-to-point network is a type of network where data is transmitted between two devices, typically using a dedicated communication channel. Duplex links, also known as full-duplex links, allow for simultaneous transmission and reception of data in both directions.

## **Historical Context**

The concept of point-to-point networks dates back to the early days of telecommunications, where telegraph lines and telephone cables connected two points directly. With the advent of computer networks in the 1960s and 1970s, point-to-point networks became a fundamental building block for computer communication.

In the 1980s, the introduction of Ethernet technology enabled the creation of point-to-point networks using coaxial cables. However, with the rise of wireless technologies and the need for higher bandwidth, point-to-point networks evolved to use wireless links, such as Wi-Fi and wireless local area networks (WLANs).

## **Technical Overview**

A point-to-point network typically consists of three components:

1. **Sender Node**: The device that sends data to the receiver node.
2. **Receiver Node**: The device that receives data from the sender node.
3. **Link**: The communication channel between the sender and receiver nodes, which can be a physical medium (e.g., coaxial cable, fiber optic cable) or a wireless link (e.g., Wi-Fi).

In this experiment, we will focus on implementing a point-to-point network using duplex links between three nodes. The duplex link allows for simultaneous transmission and reception of data in both directions.

## **Components and Materials**

- Three nodes (computers or devices with Ethernet or Wi-Fi connectivity)
- Three duplex links (Ethernet cables or Wi-Fi access points)
- Network switches or routers (optional)
- Cables and connectors (e.g., Ethernet cables, Wi-Fi antennas)

## **Setup and Configuration**

1. **Node 1 (Sender Node)**:
   - Connect the sender node to the first duplex link using an Ethernet cable or Wi-Fi antenna.
   - Configure the sender node to transmit data to the receiver node.
2. **Node 2 (Receiver Node)**:
   - Connect the receiver node to the second duplex link using an Ethernet cable or Wi-Fi antenna.

3. **Node 3 (Node 3)**:
   - Connect node 3 to the third duplex link using an Ethernet cable or Wi-Fi antenna.

## **Network Topology**

The network topology of a point-to-point network is typically linear, meaning that the nodes are connected in a straight line. In this experiment, we will use a linear topology with three nodes and three duplex links.

## **Data Transmission**

To demonstrate data transmission between the nodes, we can use a simple protocol such as TCP/IP. The sender node will send data packets to the receiver node, which will receive and forward the packets to node 3.
