# **Computer Networks Revision Notes (25.1-25.2)**

### Key Concepts

- **Network Topology**: Physical or logical arrangement of devices in a network
  - Types:
    - Bus
    - Star
    - Ring
    - Mesh
    - Hybrid
- **Network Devices**:
  - Host
  - Repeater
  - Router
  - Switch
  - Bridge
  - Hub
- **Network Communication**:
  - Data transmission
  - Packet switching
  - Error detection and correction

### Important Formulas and Definitions

- **Packet Size**: 1500 bytes (default for Ethernet)
- **Jitter**: Variation in packet delay
- **Latency**: Time delay between packet transmission and reception
- **Throughput**: Amount of data transmitted per unit time
- **Bandwidth**: Amount of data that can be transmitted per unit time
- **Packet Error Rate (PER)**: Probability of packet error during transmission

### Important Theorems

- **Store-and-Forward**: Data is stored in a buffer before being transmitted
- **Cut-Through**: Data is transmitted without buffering
