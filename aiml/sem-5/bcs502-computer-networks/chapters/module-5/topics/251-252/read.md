# 25.1-25.2: Computer Networks Fundamentals

=====================================================

## 25.1: Network Fundamentals

### Definition

A computer network is a collection of interconnected devices that can communicate with each other to share resources and exchange information.

### Types of Computer Networks

- **Local Area Network (LAN):** A network that spans a small geographical area, such as a home, office building, or campus.
- **Wide Area Network (WAN):** A network that covers a larger geographical area, such as a city, state, or country.
- **Metropolitan Area Network (MAN):** A network that spans a metropolitan area, such as a city or town.
- **Wireless Network (WLAN):** A network that uses wireless communication to connect devices.

### Network Topology

- **Bus Topology:** A network where all devices are connected to a single cable or backbone.
- **Star Topology:** A network where all devices are connected to a central device or hub.
- **Ring Topology:** A network where devices are connected in a circular configuration, with each device connected to two neighbors.
- **Mesh Topology:** A network where each device is connected to every other device.

### Network Devices

- **Router:** A device that connects multiple networks together and routes traffic between them.
- **Switch:** A device that connects multiple devices within a network and forwards data between them.
- **Hub:** A device that connects multiple devices within a network and repeats incoming data to all devices.
- **Bridge:** A device that connects two or more networks and forwards data between them.

## 25.2: Network Protocols

### Definition

A protocol is a set of rules and standards that govern how data is communicated between devices on a network.

### Types of Network Protocols

- **Transport Layer Protocols:** Protocols that ensure reliable data transfer between devices, such as TCP (Transmission Control Protocol) and UDP (User Datagram Protocol).
- **Internet Layer Protocols:** Protocols that route data between networks, such as IP (Internet Protocol).
- **Link Layer Protocols:** Protocols that connect devices within a network, such as Ethernet and Wi-Fi.
- **Application Layer Protocols:** Protocols that provide services to applications, such as HTTP (Hypertext Transfer Protocol) and FTP (File Transfer Protocol).

### Key Network Protocols

- **TCP (Transmission Control Protocol):** A protocol that ensures reliable data transfer between devices.
- **UDP (User Datagram Protocol):** A protocol that provides best-effort delivery of data.
- **IP (Internet Protocol):** A protocol that routes data between networks.
- **HTTP (Hypertext Transfer Protocol):** A protocol that provides services for web browsing.
- **FTP (File Transfer Protocol):** A protocol that provides services for file transfer.

### Key Network Protocols (Continued)

- **DNS (Domain Name System):** A protocol that translates domain names to IP addresses.
- **DHCP (Dynamic Host Configuration Protocol):** A protocol that assigns IP addresses to devices on a network.
- **SNMP (Simple Network Management Protocol):** A protocol that provides management information about a network.
