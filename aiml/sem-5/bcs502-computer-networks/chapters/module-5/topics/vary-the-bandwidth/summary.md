# Vary the Bandwidth

**Definition:** Bandwidth is the rate at which data is transferred between two devices over a network.

**Theorems:**

- **Bandwidth-Theorem:** The bandwidth of a network is determined by the number of available bandwidth channels.
- **Network Bandwidth Formula:** B = L/C, where B is bandwidth, L is data transmission rate, and C is the duration of transmission.

**Key Points:**

- **Bandwidth Types:**
  - Downlink bandwidth (DL): The rate at which data is received from the internet.
  - Uplink bandwidth (UL): The rate at which data is transmitted to the internet.
- **Bandwidth Allocation:**
  - Dynamic bandwidth allocation: Allocates bandwidth in real-time based on network demand.
  - Static bandwidth allocation: Allocates a fixed amount of bandwidth to a device.
- **Bandwidth Management:**
  - Quality of Service (QoS): Ensures that critical applications receive sufficient bandwidth.
  - Bandwidth shaping: Regulates bandwidth usage to prevent network congestion.
- **Bandwidth Measurement:**
  - Bitrate: Measured in bits per second (bps).
  - Packet loss: Measured in percentage.

**Important Formulas:**

- **Bandwidth Formula:** B = L/C, where B is bandwidth, L is data transmission rate, and C is the duration of transmission.
- **Packet Loss Formula:** P = Δt / T, where P is packet loss, Δt is packet delay, and T is transmission time.

**Revision Tips:**

- Understand the difference between downlink and uplink bandwidth.
- Know how bandwidth allocation and management techniques can impact network performance.
- Be able to calculate bandwidth and packet loss using the provided formulas.
