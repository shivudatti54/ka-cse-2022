# Learning Objectives
After studying this topic, you should be able to:
1.  Explain the necessity of recovery mechanisms in distributed transactions.
2.  Describe the purpose and mechanics of Write-Ahead Logging (WAL).
3.  Differentiate between the Undo and Redo operations in recovery algorithms.
4.  Explain the role of checkpointing in optimizing the recovery process.
5.  Analyze how the Two-Phase Commit (2PC) protocol uses logging to enable recovery from failures.
6.  Identify the 'blocking' problem in 2PC and contrast it with the non-blocking property of 3PC.
