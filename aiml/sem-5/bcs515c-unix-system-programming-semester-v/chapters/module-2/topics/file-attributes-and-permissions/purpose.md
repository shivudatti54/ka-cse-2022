# Learning Objectives

After studying this topic, you should be able to:

1.  Interpret the detailed output of the `ls -l` command, identifying the file type, permissions, ownership, and other attributes.
2.  Differentiate between the meaning of read, write, and execute permissions for both regular files and directories.
3.  Change file permissions using both the symbolic (e.g., `g+w`) and absolute/octal (e.g., `755`) modes with the `chmod` command.
4.  Explain the purpose and effect of the special permission bits: setuid, setgid (on files and directories), and the sticky bit.
5.  Change the ownership and group of a file using the `chown` and `chgrp` commands.
6.  Calculate the default permissions for new files and directories based on a given `umask` value.
