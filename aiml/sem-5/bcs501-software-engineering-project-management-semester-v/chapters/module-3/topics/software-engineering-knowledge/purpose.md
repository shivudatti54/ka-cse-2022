# Learning Objectives

After studying this topic, you should be able to:

1. **Define** software engineering and explain its systematic approach
2. **List** the five generic framework activities and their purposes
3. **Identify** umbrella activities that span the software process
4. **Apply** Polya's problem-solving approach to software development
5. **Explain** the ACM/IEEE Code of Ethics for software engineers
6. **Describe** CMMI levels and their characteristics
