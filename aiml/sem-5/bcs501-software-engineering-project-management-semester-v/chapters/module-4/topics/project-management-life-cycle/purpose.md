# Learning Objectives

After studying this topic, you should be able to:

1. **Describe** the five phases of the project life cycle
2. **List** key activities and deliverables for each phase
3. **Compare** traditional and modern PM approaches
4. **Explain** PMBOK and PRINCE2 frameworks
5. **Understand** the purpose of stage gates in governance
6. **Select** appropriate PM approach based on project needs
