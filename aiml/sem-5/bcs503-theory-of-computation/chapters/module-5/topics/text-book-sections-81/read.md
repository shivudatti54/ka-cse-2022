## **Introduction**

In this section, we will explore the concept of computation and the basic components of a computation. This section is essential to understanding the fundamentals of computer science and the theory of computation.

### What is Computation?

Computation is the process of performing a series of operations on data using a set of rules and algorithms. This process can be performed using a computer, and it involves input, processing, and output of data.

### Basic Components of a Computation

A computation consists of the following basic components:

- **Input**: The input is the data that is used to start the computation.
- **Processing**: The processing phase involves applying a set of rules and algorithms to the input data to produce the output.
- **Output**: The output is the result of the computation, which is the final data produced by the processing phase.

### Types of Computations

There are two types of computations:

- **Deterministic Computation**: In a deterministic computation, the output is uniquely determined by the input and the rules and algorithms used.
- **Non-Deterministic Computation**: In a non-deterministic computation, there can be multiple possible outputs for a given input.

## **Key Concepts**

- **Algorithm**: An algorithm is a set of rules and procedures used to solve a problem or perform a computation.
- **Programming Language**: A programming language is a formal system used to express algorithms and computations.
- **Computer**: A computer is a machine that can perform computations and execute programs.

## **Examples**

- **Simple Calculator**: A simple calculator is a deterministic computation that takes input numbers and performs arithmetic operations.
- **Chess Game**: A chess game is a non-deterministic computation that involves making a series of moves based on a set of rules and algorithms.

## **Key Terminology**

- **Turing Machine**: A Turing machine is a mathematical model of a computation that consists of a tape, a read/write head, and a set of rules.
- **Algorithmic Complexity**: Algorithmic complexity is a measure of the computational resources required to execute an algorithm.
