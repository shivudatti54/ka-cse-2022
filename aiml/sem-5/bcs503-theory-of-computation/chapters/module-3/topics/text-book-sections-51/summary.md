# **Theory of Computation Revision Notes: Sections 5.1**

**Definitions:**

- **Turing Machine (TM):** A mathematical model for computation that consists of a tape divided into cells, each of which can hold a symbol from a finite alphabet.
- **State:** A set of symbols that the TM can be in at a given time.
- **Transition:** A rule that defines the next state and symbol to write to the tape, based on the current state and symbol.

**Key Concepts:**

- **Halting Problem:** The question of whether a given TM will ever halt on a given input.
- **Decidability:** The ability to determine whether a problem is solvable in a finite number of steps.
- **Undecidability:** The inability to determine whether a problem is solvable in a finite number of steps.

**Important Formulas and Theorems:**

- **Church-Turing Thesis:** Any effectively calculable function can be computed by a TM.
- **Turing Machine Theory:** The study of the behavior of TMs and their limitations.
- **König's Theorem:** A TM will eventually halt on an infinite input if it is possible to decide whether the TM will ever halt.

**Other Key Points:**

- **Pushdown Automata (PDA):** A type of TM that uses a stack to store symbols.
- **Context-Free Grammars:** A way of specifying the syntax of a language using a set of production rules.
- **Regular Expressions:** A way of specifying patterns in strings using a formal language.
