# The Pumping Lemma for Context-Free Languages

### Definition

- A context-free language is a language that can be generated by a pushdown automaton (PDA).
- The Pumping Lemma states that for any context-free language L, there exists a string p ∈ L such that |p| ≥ p's pumping length.

### Pumping Length

- The pumping length of a string p is the smallest integer k such that:
  - p can be divided into three substrings p = xyz.
  - |y| ≥ 1.
  - For all i ≥ 0, xy^iz is a valid string in L.

### Theorems

- **Pumping Lemma Theorem**: If L is a context-free language, then L is context-sensitive.
- **Rice's Theorem**: If L is a non-trivial context-free language, then it is undecidable to determine whether a given string is in L.

### Formulas and Definitions

- **Context-free grammar**: A grammar G = (V, Σ, P, S) where:
  - V is a set of non-terminal symbols.
  - Σ is a set of terminal symbols.
  - P is a set of production rules.
  - S is the start symbol.
- **Pushdown automaton (PDA)**: A 5-tuple A = (Q, Σ, Γ, δ, q0, F) where:
  - Q is a finite set of states.
  - Σ is a set of input symbols.
  - Γ is a set of stack symbols.
  - δ is a transition function.
  - q0 is the initial state.
  - F is a set of final states.

### Important Formulas

- **Church-Rosser Theorem**: If G is a context-free grammar, then it is possible to rewrite any string generated by G using a sequence of production rules.
- **Pumping Lemma Formula**: If L is a context-free language and p ∈ L has pumping length k, then:
  - |p| ≥ k.
  - p = xyz.
  - For all i ≥ 0, xy^iz is a valid string in L.

### Revision Tips

- Context-free languages are equivalent to context-sensitive languages using the Pumping Lemma.
- The Pumping Lemma can be used to prove the undecidability of certain languages.
- Context-free grammars and PDAs are key tools for studying context-free languages.
