# Learning Objectives: Context-Free Grammars

After studying this topic, you should be able to:

1.  Define a Context-Free Grammar using the formal 4-tuple notation `(V, T, P, S)`.
2.  Differentiate between terminal symbols (tokens) and non-terminal symbols (syntactic variables).
3.  Read and interpret production rules of a CFG.
4.  Perform both leftmost and rightmost derivations for a given string using a provided grammar.
5.  Construct a parse tree for a given string based on a provided grammar.
6.  Identify an ambiguous grammar by demonstrating two possible parse trees for the same input string.
7.  Rewrite a simple ambiguous grammar (e.g., for arithmetic expressions) into an unambiguous one that enforces standard operator precedence and associativity.
