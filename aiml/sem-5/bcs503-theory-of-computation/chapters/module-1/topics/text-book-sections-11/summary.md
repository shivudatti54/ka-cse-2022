# **Theory of Computation Revision Notes**

## **Section 1.1: Introduction to Theory of Computation**

### Definitions and Notations

- **Turing Machine**: A mathematical model for computation that consists of a read/write head that can move left or right on an infinite tape.
- **State**: A finite set of symbols that the machine can be in.
- **Transition**: A rule that determines the next state and symbol to read/write.
- **Input**: The data that the machine is processing.
- **Output**: The result of the computation.

### Important Formulas and Theorems

- **Turing Machine Acceptance Criterion**: A string w is accepted by M if there exists a computation of M on w that ends in a halting state.
- **Halting Problem**: There is no general algorithm that can determine whether a given Turing machine will halt on a given input.

### Key Points

- The Turing Machine model is a fundamental concept in the Theory of Computation.
- The machine can be in one of a finite number of states, and the transition function determines the next state and symbol to read/write.
- The acceptance criterion for a Turing machine is based on whether the machine halts on a given input.
- The halting problem is an undecidable problem, meaning that there is no general algorithm to determine whether a machine will halt.
