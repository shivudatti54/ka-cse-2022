# **Theory of Computation**

## **Section 1.1: Introduction**

### What is Theory of Computation?

The theory of computation is the study of the fundamental limits of computation and the design of algorithms. It provides a framework for understanding the behavior of computational systems, including their efficiency, reliability, and scalability.

### Importance of Theory of Computation

The theory of computation has numerous applications in various fields, including:

- Computer science: It provides a foundation for designing algorithms and data structures.
- Mathematics: It helps in understanding the properties of computational systems.
- Engineering: It is used in designing and optimizing computational systems.

### Key Concepts

- **Computation**: The process of performing tasks using a computer.
- **Algorithms**: Step-by-step procedures for solving problems.
- **Computational complexity**: The study of the resources required to solve computational problems.

## **Definition of Computation**

Computation is the process of performing tasks using a computer. It involves executing a sequence of operations, such as reading input, processing data, and producing output.

### Types of Computation

- **Sequential computation**: The process of performing tasks one at a time.
- **Parallel computation**: The process of performing tasks simultaneously.

## **Definition of Algorithm**

An algorithm is a step-by-step procedure for solving a problem. It involves a sequence of operations, such as reading input, processing data, and producing output.

### Characteristics of an Algorithm

- **Finite**: An algorithm must have a finite number of steps.
- **Deterministic**: An algorithm must produce the same output for the same input.
- **Efficient**: An algorithm must use a minimal number of resources.

## **Example of an Algorithm**

Example 1: Sorting a list of numbers.

1. Read the input list of numbers.
2. Initialize an empty list.
3. Iterate through the input list and add each number to the empty list.
4. Return the sorted list.

### Computational Complexity

---

Computational complexity is the study of the resources required to solve computational problems. It involves analyzing the time, space, and other resources required to solve a problem.

### Types of Computational Complexity

- **Time complexity**: The study of the time required to solve a problem.
- **Space complexity**: The study of the space required to solve a problem.

## **Big O Notation**

Big O notation is a mathematical notation used to describe the computational complexity of an algorithm. It is a way of expressing the upper bound of an algorithm's time or space complexity.

- **O(1)**: Constant time complexity.
- **O(log n)**: Logarithmic time complexity.
- **O(n)**: Linear time complexity.
- **O(n log n)**: Linearithmic time complexity.

## **Key Points to Remember**

- Computation is the process of performing tasks using a computer.
- Algorithms are step-by-step procedures for solving problems.
- Computational complexity is the study of the resources required to solve computational problems.
- Big O notation is a way of expressing the upper bound of an algorithm's time or space complexity.
