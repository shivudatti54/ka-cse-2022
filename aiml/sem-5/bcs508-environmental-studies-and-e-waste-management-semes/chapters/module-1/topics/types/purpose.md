# Learning Objectives
After studying this topic, you should be able to:
1.  Define and distinguish between the three primary types of machine learning: Supervised, Unsupervised, and Reinforcement Learning.
2.  Identify real-world problems that are best suited for each type of machine learning.
3.  List common algorithms associated with each learning type (e.g., Regression and Classification for Supervised; Clustering for Unsupervised; Q-Learning for RL).
4.  Explain the key components and workflow of a Reinforcement Learning system (Agent, Environment, Action, State, Reward).
5.  Describe the rationale and typical use cases for Semi-supervised Learning.
6.  Analyze the challenges and limitations inherent to each learning paradigm.
