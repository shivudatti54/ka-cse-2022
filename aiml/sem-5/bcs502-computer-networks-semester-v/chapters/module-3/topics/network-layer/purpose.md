# Learning Objectives

After studying this topic, you should be able to:

1. Understand the services provided by the Network Layer.
2. Know the IPv4 address format and the different classes of addresses.
3. Understand the IPv4 datagram format and the different fields in the header.
4. Know the different types of routing algorithms and how they work.
5. Understand the purpose of the Type of Service field in the IPv4 header.
6. Understand the purpose of the Fragment Offset field in the IPv4 header.
