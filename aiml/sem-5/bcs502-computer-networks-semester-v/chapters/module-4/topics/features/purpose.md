# Why Study HOG Features?

## Classic Object Detection Method

HOG is the foundation of traditional object detection methods and remains important for understanding how handcrafted features work before deep learning.

## Exam Relevance

| Aspect | Details |
|--------|---------|
| Frequency | High - common CV exam topic |
| Marks | 10-15 marks (algorithm, calculation, applications) |
| Question Types | Algorithm explanation, descriptor size calculation, comparison |

## Historical Importance

- CVPR 2005 paper is one of the most cited in CV
- Dominated pedestrian detection for nearly a decade
- Conceptual foundation for understanding CNN features
- Still used in resource-constrained applications

## Learning Outcomes

After studying this topic, you should be able to:
1. Explain the HOG feature extraction pipeline
2. Calculate HOG descriptor dimensions
3. Understand the role of block normalization
4. Compare HOG with SIFT and deep learning features
5. Apply HOG for object detection
