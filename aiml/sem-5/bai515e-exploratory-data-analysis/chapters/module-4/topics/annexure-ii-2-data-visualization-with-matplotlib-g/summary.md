# Annexure-II 2 Data Visualization with MatPlotlib: Revision Notes

## General Matplotlib Tips

- The Matplotlib library is a popular data visualization tool in Python.
- Importing Matplotlib: `import matplotlib.pyplot as plt`
- Common axes limits: `plt.xlim()`, `plt.ylim()`
- Adding titles and labels: `plt.title()`, `plt.xlabel()`, `plt.ylabel()`
- Showing grid: `plt.grid()`

## Simple Line Plots

- Creating a simple line plot: `plt.plot(x, y)`
- Adding labels and title: `plt.xlabel()`, `plt.ylabel()`, `plt.title()`
- Changing line style and color: `plt.plot(x, y, 'ro-')`

## Simple Scatter Plots

- Creating a simple scatter plot: `plt.scatter(x, y)`
- Adding labels and title: `plt.xlabel()`, `plt.ylabel()`, `plt.title()`
- Changing marker style and size: `plt.scatter(x, y, marker='o', s=100)`

## Visualization with Seaborn

- Importing Seaborn: `import seaborn as sns`
- Creating a heatmap: `sns.heatmap()`
- Creating a joint plot: `sns.jointplot()`
- Creating a boxplot: `sns.boxplot()`

## Important Formulas, Definitions, and Theorems

- Definition of a data point: `(x, y)`
- Definition of a dataset: `Set of data points`
- Theorem of least squares regression: `y = a + bx + ε`
