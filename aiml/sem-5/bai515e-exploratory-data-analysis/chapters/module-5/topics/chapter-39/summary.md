# Chapter 39: Scikit-Learn Basics

### Important Concepts

- **Scikit-Learn**: A popular Python library for machine learning.
- **Supervised Learning**: Training a model on labeled data to make predictions on new data.
- **Unsupervised Learning**: Training a model on unlabeled data to discover patterns or structure.
- **Overfitting**: When a model is too complex and performs well on training data but poorly on new data.
- **Underfitting**: When a model is too simple and fails to capture important patterns in the data.

### Scikit-Learn Basics

- **Importing Scikit-Learn**: `from sklearn import datasets, svm, metrics`
- **Loading Datasets**: `load_iris()`, `load_breast_cancer()`
- **Data Preprocessing**: `FeatureScaling()`, `StandardScaler()`
- **Model Evaluation**: `accuracy_score()`, `classification_report()`

### Important Formulas and Definitions

- **Accuracy**: `TP + TN / (TP + TN + FP + FN)`
- **Precision**: `TP / (TP + FP)`
- **Recall**: `TP / (TP + FN)`
- **F1-Score**: `2 \* (Precision \* Recall) / (Precision + Recall)`

### Important Theorems

- **Bayes' Theorem**: `P(H|E) = P(E|H) \* P(H) / P(E)`
- **Decision Boundaries**: `w^T \* x + b = 0`
