# Learning Objectives
After studying this topic, you should be able to:
1. Define the terms PDA and CFG and explain their differences.
2. Construct a CFG from a PDA and vice versa.
3. Explain the purpose of the start symbol in a CFG.
4. Determine if a language is context-free.
5. Apply the pumping lemma to prove that a language is not context-free.
6. Compare and contrast the features of PDAs and CFGs.
