# Why Learn Regular Expressions?

## Exam Relevance
- 15-20% of TOC exam questions
- Regex to NFA conversion common
- Writing regex for languages

## Key Exam Patterns
1. Write regex for given language
2. Convert regex to NFA (Thompson's)
3. Apply algebraic laws
4. Determine if two regex are equivalent

## Interview Questions
- "Write regex for email validation"
- "How do regex engines work?"
- "Regex for pattern matching problem"

## Real-World Applications
- Text search and filtering
- Input validation
- Log parsing
- Compiler lexical analysis
- grep, sed, awk tools
