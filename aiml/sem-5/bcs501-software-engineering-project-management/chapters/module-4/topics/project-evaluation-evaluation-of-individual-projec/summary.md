# **Project Evaluation Revision Notes**

**1.1 Introduction to Project Evaluation**

- Definition: The process of assessing the performance of a project against its objectives and stakeholder expectations.
- Importance: Evaluates project success, identifies areas for improvement, and informs future project decisions.

**1.2 Types of Project Evaluations**

- Formative Evaluation: Ongoing monitoring and feedback during project execution.
- Summative Evaluation: Final assessment of project performance after completion.

**1.3 Cost-Benefit Evaluation Techniques**

- Definition: Methods used to compare the costs and benefits of a project.
- Techniques:
  - Net Present Value (NPV)
  - Internal Rate of Return (IRR)
  - Payback Period
  - Benefit-Cost Ratio

**1.4 Cost-Benefit Evaluation Formulas**

- NPV = ∑(CFt / (1 + r)^t) - ∑(CFt)
- IRR = r when NPV = 0
- Payback Period = Time taken for costs to be recovered
- Benefit-Cost Ratio = Benefits / Costs

**1.5 Risk Evaluation Techniques**

- Definition: Methods used to identify, assess, and mitigate project risks.
- Techniques:
  - SWOT Analysis
  - Risk Matrix
  - Decision Trees

**1.6 Risk Evaluation Theorems**

- Probability-Expected Value Theorem: A = P(E) \* v(E)
- Expected Utility Theory: v(E) = p(E) \* u(E)

**Key Definitions**

- **Expected Value**: The average value of a random variable.
- **Utility**: The value or magnitude of a benefit or cost.
- **Risk**: An uncertain event that may impact the project.

**Important Formulas**

- Net Present Value (NPV) formula
- Internal Rate of Return (IRR) formula
- Payback Period formula
