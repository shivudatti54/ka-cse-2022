# Three Standard Files and Redirection

### Overview

- Three standard files created during the execution of a program:
  - Standard Input (STDIN): `0`
  - Standard Output (STDOUT): `1`
  - Standard Error (STDERR): `2`

### Redirection Operators

- `>` : Redirects output to a file
- `>>` : Appends output to a file (new file is created if none exists)
- `<` : Redirects input from a file
- `>>=` : Appends input from a file
- `2>`, `2>>` : Redirects standard error output

### Examples

- `cat file1.txt > file2.txt` : Copies contents of file1.txt to file2.txt
- `ls -l > output.txt` : Redirects output of `ls -l` to output.txt
- `2> error.log` : Redirects standard error output to error.log
