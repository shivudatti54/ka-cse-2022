# **The Root Login**

### Overview

- The root login is the primary account for a Unix system administrator.
- It is used to access and manage the system as the highest privileged user.

### Key Points

- **Definition:** The root user account, also known as "root," is the primary account for a Unix system administrator.
- **Purpose:** To access and manage the system as the highest privileged user.
- **Privileges:** Root has all privileges and can perform all actions on the system.
- **Password:** The root password is not displayed on the login prompt.

### Important Formulas and Definitions

- **UID (User ID):** A unique identifier for each user on the system.
- **GID (Group ID):** A unique identifier for each group on the system.
- **Group membership:** A user can belong to multiple groups.

### Key Features

- **Root login:** The root user can log in to the system without a password.
- **Sudo command:** Used to execute commands with superuser privileges.
- **su command:** Used to switch to the root user account.

### Important Theorems

- **The Unix theorem:** A Unix system is a hierarchical, multi-user, multi-tasking operating system.
- **The Unix principle:** A Unix system is designed to be secure, reliable, and efficient.
