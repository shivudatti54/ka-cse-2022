# Reading Dictionaries

### Overview

- A dictionary is a data structure used in programming to store and retrieve mappings between keys and values.
- Dictionaries are also known as associative arrays, hash tables, or maps.

### Key Concepts

- **Key**: A unique identifier for a value in a dictionary.
- **Value**: The data stored in a dictionary, associated with a key.
- **Dictionary operations**:
  - **Insert**: Add a new key-value pair to a dictionary.
  - **Search**: Retrieve a value from a dictionary by its key.
  - **Delete**: Remove a key-value pair from a dictionary.
  - **Update**: Modify the value associated with a key.

### Dictionary Data Structures

- **Hash Table**: A dictionary implemented using a hash function to map keys to indices of an array.
- **Linked List**: A dictionary implemented using a linked list to store key-value pairs.

### Important Formulas and Definitions

- **Hash Function**: A function that takes a key as input and returns an index at which the corresponding value is stored in the dictionary.
- **Collision**: A situation where two keys hash to the same index in a hash table.
- **Resolving Collision**: A technique used to handle collisions in hash tables.

### Important Theorems and Properties

- **Hash Function Theorem**: A good hash function should minimize collisions and ensure efficient lookup times.
- **Average Case Analysis**: A measure of the expected time complexity of a hash function in the worst-case scenario.

### Examples and Applications

- **Real-world applications**: Dictionaries are used in databases, file systems, and web browsers to store and retrieve data.
- **Programming languages**: Many programming languages, such as Python, Java, and C++, use dictionaries as a fundamental data structure.
