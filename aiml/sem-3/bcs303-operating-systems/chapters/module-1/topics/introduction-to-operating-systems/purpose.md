# Learning Objectives

After studying this topic, you should be able to:

1. DEFINE what an operating system is and explain its role as an intermediary between hardware and users.

2. EXPLAIN the two primary perspectives of an operating system—resource manager and user interface provider.

3. DESCRIBE the evolution of operating systems from early batch processing to modern multi-purpose systems.

4. DIFFERENTIATE between various types of operating systems including batch, time-sharing, real-time, network, and distributed systems.

5. IDENTIFY and EXPLAIN the five main functions of an operating system: process management, memory management, file management, device management, and security/protection.

6. ANALYZE how an operating system manages CPU scheduling, memory allocation, and I/O operations as a resource manager.

7. UNDERSTAND the concept of virtual memory and explain how it allows programs to use more memory than physically available.

8. APPLY knowledge of operating system concepts to explain real-world computing scenarios and system behaviors.
