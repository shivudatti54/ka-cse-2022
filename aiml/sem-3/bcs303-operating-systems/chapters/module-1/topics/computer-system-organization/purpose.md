# Learning Objectives

After studying this topic, you should be able to:

1. Describe the three main subsystems of a computer system and explain how they are interconnected through the system bus.

2. Explain the architecture of the Central Processing Unit (CPU), including the functions of the Control Unit, Arithmetic Logic Unit (ALU), and various registers.

3. Trace the fetch-decode-execute cycle for a given instruction and identify the role of each register involved in the process.

4. Analyze the memory hierarchy (cache, primary, secondary) and explain how locality principles improve system performance.

5. Compare and contrast different I/O techniques (programmed I/O, interrupt-driven I/O, DMA) and evaluate their efficiency.

6. Explain the structure and function of the system bus, including data, address, and control buses.

7. Describe the interrupt mechanism in computer systems, including types of interrupts and the interrupt handling process.

8. Apply knowledge of computer system organization to understand how operating systems manage hardware resources effectively.
