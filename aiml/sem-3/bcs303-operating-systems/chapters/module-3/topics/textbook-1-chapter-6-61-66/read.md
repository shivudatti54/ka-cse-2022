# **Operating Systems: Chapter 6 (6.1-6.6) Study Material**

## **6.1 Introduction to Operating Systems**

### Definition

An operating system (OS) is a software that manages computer hardware resources and provides common services to computer programs.

### Characteristics

- Acts as an intermediary between computer hardware and application programs
- Manages computer hardware components (CPU, memory, storage, input/output devices)
- Provides a platform for running applications
- Provides common services (file management, process management, input/output management)

### Importance of Operating Systems

- Enables multiple programs to run simultaneously
- Provides a secure environment for running applications
- Manages computer resources efficiently
- Provides a user-friendly interface for interacting with the computer

### Examples of Operating Systems

- Windows
- macOS
- Linux
- Android
- iOS

## **6.2 System Resource Management**

### Definition

System resource management refers to the process of managing computer hardware resources (CPU, memory, storage, input/output devices) to ensure efficient use and optimal performance.

### Types of System Resources

- **CPU (Central Processing Unit)**: executes instructions and performs calculations
- **Memory (RAM)**: temporary storage for data and applications
- **Storage (Hard Disk)**: permanent storage for data and applications
- **Input/Output Devices**: devices that input data into the computer (keyboard, mouse) and output data from the computer (monitor, printer)

**Key Concepts:**

- **Resource Allocation**: allocating resources to applications and processes
- **Resource Allocation Algorithms**: algorithms used to allocate resources to applications and processes (e.g. First-Come-First-Served, Shortest Job First)
- **Resource Utilization**: monitoring and optimizing the use of system resources

## **6.3 Process Management**

### Definition

Process management refers to the process of creating, scheduling, and terminating processes (programs) that run on the computer.

### Types of Process Management

- **Batch Processing**: processing multiple jobs sequentially
- **Interactive Processing**: processing multiple jobs interactively (e.g. command-line interface)
- **Multitasking**: running multiple processes simultaneously

### Process Scheduling Algorithms

- **First-Come-First-Served**: scheduling processes in the order they arrive
- **Shortest Job First**: scheduling processes with the shortest execution time first
- **Priority Scheduling**: scheduling processes based on priority (e.g. high-priority processes get priority over low-priority processes)

**Key Concepts:**

- **Process Creation**: creating a new process
- **Process Scheduling**: scheduling a process to run on the computer
- **Process Termination**: terminating a process

## **6.4 Memory Management**

### Definition

Memory management refers to the process of managing the computer's memory (RAM) to ensure efficient use and optimal performance.

### Types of Memory Management

- **Virtual Memory**: using secondary storage (hard disk) to supplement primary memory (RAM)
- **Paging**: dividing memory into fixed-size blocks (pages) and swapping pages in and out of RAM
- **Segmentation**: dividing memory into smaller segments and assigning each segment to a program

**Key Concepts:**

- **Memory Address Space**: the range of addresses that a program can access
- **Page Replacement Algorithms**: algorithms used to replace pages in virtual memory (e.g. First-In-First-Out, Optimal)
- **Memory Allocation**: allocating memory to programs and processes

## **6.5 File Management**

### Definition

File management refers to the process of managing files and directories on the computer.

### Types of File Management

- **File Organization**: organizing files into directories and subdirectories
- **File Access Control**: controlling access to files (e.g. permissions, access control lists)
- **File Sharing**: sharing files between users and applications

**Key Concepts:**

- **File Structure**: the organization of files and directories on the computer
- **File Permissions**: controlling access to files based on user identity and group membership
- **File Sharing Protocols**: protocols used to share files between users and applications (e.g. SFTP, FTP)

## **6.6 Security and Protection**

### Definition

Security and protection refer to the measures taken to prevent unauthorized access to the computer and its data.

### Types of Security Measures

- **Access Control**: controlling access to the computer and its data based on user identity and group membership
- **Authentication**: verifying the identity of users and applications
- **Encryption**: encrypting data to prevent unauthorized access
- **Intrusion Detection and Prevention**: detecting and preventing unauthorized access to the computer and its data

**Key Concepts:**

- **Security Threats**: potential threats to the security of the computer and its data (e.g. malware, phishing)
- **Security Protocols**: protocols used to protect the computer and its data (e.g. firewalls, antivirus software)
- **Security Policies**: policies used to govern security practices on the computer (e.g. password policies, access control policies)
