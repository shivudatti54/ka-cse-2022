# Process Management: Process Concept

### Overview

Process Management is a crucial component of Operating Systems, focusing on the execution of programs. Here are the key points to revise for the topic:

### Definitions and Concepts

- **Process**: A self-contained unit of execution that runs in its own memory space, with its own program counter, registers, and memory.
- **Process Scheduling**: The allocation and execution of processes by the Operating System.
- **Process Management**: The set of activities involved in creating, executing, and terminating processes.

### Process States

- **New**: A process that has been created but has not yet been scheduled.
- **Ready**: A process that is waiting to be scheduled and executed.
- **Running**: A process that is currently being executed.
- **Waiting**: A process that is waiting for some event to occur (e.g., I/O completion).
- **Terminated**: A process that has completed execution.

### Process Control Structures

- **Process Table**: A data structure used to manage and store information about processes.
- **Process ID (PID)**: A unique identifier assigned to each process.
- **Process Status**: The current state of a process (e.g., running, waiting, terminated).

### Process Scheduling Algorithms

- **First-Come-First-Served (FCFS)**: Schedules processes in the order they arrive.
- **Shortest Job First (SJF)**: Schedules processes with the shortest execution time first.
- **Priority Scheduling**: Assigns priority to processes based on their characteristics (e.g., burst time, priority value).

### Important Formulas and Theorems

- **Burst Time**: The time required to execute a process.
- **Turnaround Time**: The time a process spends waiting in the queue before being executed.
- **Waiting Time**: The time a process spends waiting for I/O operations to complete.
- **Response Time**: The time from when a process requests I/O to when it receives the response.

### Revision Tips

- Understand the different process states and their implications on process management.
- Familiarize yourself with process scheduling algorithms and their advantages/disadvantages.
- Know the key concepts and formulas related to process management.

By revising these key points, you'll be well-prepared to tackle the topic of Process Management: Process Concept in your Operating Systems course.
