# Learning Objectives

After studying this topic, you should students able to:

1. Compare and contrast contiguous, linked, and indexed file allocation methods, including their advantages and disadvantages
2. Explain how UNIX i-nodes work and calculate maximum file size given block and pointer sizes
3. Describe different free space management techniques (bit vector, linked list, grouping, counting) and their appropriate use cases
4. Explain the purpose and operation of journaling file systems and how they improve reliability
5. Analyze directory implementation approaches and their impact on file system performance
6. Identify optimization techniques used in file systems, including caching, read-ahead, and delayed writes
