# Learning Objectives

After studying this topic, you should students able to:

1. Define the purpose of methods and classes in Java programming.
2. Identify the components of a method declaration, including return type, method name, and parameter list.
3. Explain the concept of pass-by-value in Java and how it applies to primitive and object parameters.
4. Describe the difference between static and instance methods, including their usage and access restrictions.
5. Classify methods as either static or instance methods based on their characteristics.
6. Implement a simple class with methods in Java, demonstrating understanding of method declaration and usage.
7. Analyze the role of methods and classes in a Java application, including their relationship to other concepts such as overloading, constructors, and access control.
8. Design a basic Java program that utilizes classes and methods to solve a real-world problem, demonstrating understanding of their practical applications.
