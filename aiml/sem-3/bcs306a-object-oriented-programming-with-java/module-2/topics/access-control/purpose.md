# Learning Objectives

After studying this topic, you should students able to:

1. Define access control in Java and explain its importance in implementing encapsulation.
2. List the four access levels in Java and identify the access modifier associated with each level.
3. Describe the visibility of classes, methods, and variables for each access level, including public, private, protected, and default.
4. Explain how access control supports encapsulation through the use of private fields and public getters/setters.
5. Apply access modifiers to control the visibility of classes, methods, and variables in a Java program.
6. Compare and contrast the different access levels, including their restrictions and use cases.
7. Justify the use of access control in a Java program to ensure internal details are hidden and to prevent unauthorized modifications.
8. Design a Java class that demonstrates proper use of access control to implement encapsulation.
