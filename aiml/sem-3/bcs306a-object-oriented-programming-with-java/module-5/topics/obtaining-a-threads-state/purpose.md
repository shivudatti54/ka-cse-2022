# Learning Objectives

After studying this topic, you should students able to:

1. Explain the complete lifecycle of a Java thread from creation to termination

2. Identify and describe all six states of a thread as defined by the Thread.State enum

3. Use the getState() method to obtain the current state of a thread programmatically

4. Differentiate between BLOCKED, WAITING, and TIMED_WAITING states with appropriate examples

5. Understand what causes a thread to transition between different states

6. Apply thread state monitoring for debugging and synchronization purposes

7. Recognize common scenarios that lead to each thread state in concurrent Java applications
