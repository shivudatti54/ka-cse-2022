# Learning Objectives

After studying this topic, you should students able to:

1. Define Local Variable Type Inference (LVTI) and its purpose in Java programming.
2. Identify the benefits and limitations of using the `var` keyword for local variable declarations.
3. Explain how type inference works with the `var` keyword, including its interaction with polymorphic assignments and method calls.
4. Describe how LVTI affects method resolution in the context of inheritance and polymorphism.
5. Implement the `var` keyword in Java code to improve readability and maintainability, especially with generic class names.
6. Analyze how LVTI influences the design of inheritance hierarchies and polymorphic method calls.
7. Evaluate the impact of using `var` on code readability, maintainability, and type safety in object-oriented programming.
8. Design and develop Java code that effectively utilizes LVTI and inheritance to achieve robust, scalable software architectures.
