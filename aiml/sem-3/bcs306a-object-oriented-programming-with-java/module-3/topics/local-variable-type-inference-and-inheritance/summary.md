# Local Variable Type Inference and Inheritance

## Overview

Local variable type inference is a feature in Java that allows developers to declare local variables without specifying their type, using the `var` keyword instead. This feature simplifies code and improves readability. Inheritance is also an important concept in Java that allows one class to inherit properties and behavior from another class.

## Key Points

- Local variable type inference is used to declare local variables without specifying their type.
- The `var` keyword is used to declare local variables with inferred types.
- The type of the variable is inferred from the assigned value.
- Local variable type inference is only applicable to local variables, not to instance or static variables.
- It simplifies code and improves readability.

## Important Definitions

- **Local Variable Type Inference**: A feature in Java that allows developers to declare local variables without specifying their type.
- **Inheritance**: A mechanism in Java that allows one class to inherit properties and behavior from another class.

## Key Syntax

- `var variableName = value;`
