### Interfaces in Object Oriented Programming with JAVA

#### Revision Notes

**Definitions and Key Concepts**

- **Interface**: A abstract class that defines a contract or a set of methods that must be implemented by any class that implements it.
- **Default Interface Methods**: Methods defined in an interface that have a default implementation.
- **Static Interface Methods**: Methods defined in an interface that belong to the interface itself, not any class implementing it.
- **Private Interface Methods**: Methods defined in an interface that can only be accessed within the interface itself.

**Interface Implementations**

- **Interface vs Abstract Class**: An interface is a abstract class that can have multiple inheritance, while an abstract class cannot.
- **Multiple Inheritance**: A class can implement multiple interfaces.
- **Interface Methods**: Must be implemented by any class implementing the interface.

**Default Interface Methods**

- **Default Method**: A method defined in an interface that has a default implementation.
- **Method Overriding**: A subclass can override a method defined in its superclass or interface.
- **Default Interface Method**: A method defined in an interface that has a default implementation.

**Static Interface Methods**

- **Static Method**: A method that belongs to the interface itself, not any class implementing it.
- **Static Interface Method**: A method defined in an interface that belongs to the interface itself.

**Private Interface Methods**

- **Private Method**: A method that can only be accessed within the interface itself.
- **Private Interface Method**: A method defined in an interface that can only be accessed within the interface itself.

**Important Formulas and Theorems**

- **Interface Inheritance**: A class can implement multiple interfaces, but can only extend one class.
- **Interface Method Implementation**: Any class implementing an interface must implement all methods defined in the interface.

### Important Formulas:

- `interface A extends interface B { }` - Class A implements interface B
- `interface A extends interface B, interface C { }` - Class A implements interface B and C
- `public static void main(String[] args) { }` - Main method of a class
