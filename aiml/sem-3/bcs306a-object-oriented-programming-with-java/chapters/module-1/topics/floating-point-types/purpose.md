# Learning Objectives

After studying this topic, you should be able to:

1. Identify and differentiate between the two floating-point primitive types in Java: float and double, including their memory sizes and precision characteristics.

2. Write floating-point literals using both decimal notation and scientific notation, with proper suffix usage (f/F for float).

3. Explain how floating-point numbers are represented internally according to the IEEE 754 standard, including the concepts of sign, exponent, and mantissa.

4. Recognize and handle special floating-point values such as Infinity (positive and negative) and NaN (Not a Number) in Java programs.

5. Understand the precision limitations of floating-point arithmetic and explain why certain decimal values cannot be represented exactly in binary format.

6. Compare floating-point values appropriately using tolerance-based comparison instead of exact equality checks.

7. Perform type conversions between float and double, understanding when explicit casting is necessary and the potential for data loss.
