# Learning Objectives

After studying this topic, you should be able to:

1. Define what a variable is in Java and explain its role in memory storage and data manipulation
2. Distinguish between the four types of variables in Java: instance, static, local, and parameter variables
3. Correctly declare and initialize variables using appropriate data types
4. Apply the `final` keyword to create constants and understand naming conventions for constants
5. Utilize the `var` keyword for local variable type inference introduced in Java 10
6. Identify the scope and lifetime of different types of variables in Java programs
7. Explain default values for instance and static variables versus the requirement to initialize local variables
8. Write Java code that demonstrates proper variable declaration, initialization, and usage following naming conventions
