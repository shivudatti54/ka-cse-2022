# Operators in Java

=====================================

## Arithmetic Operators

---

- **Addition**: `a + b`
- **Subtraction**: `a - b`
- **Multiplication**: `a * b`
- **Division**: `a / b`
- **Modulus**: `a % b`
- **Exponentiation**: `a ^ b`

## Relational Operators

---

- **Equal**: `a == b`
- **Not Equal**: `a != b`
- **Greater Than**: `a > b`
- **Less Than**: `a < b`
- **Greater Than or Equal**: `a >= b`
- **Less Than or Equal**: `a <= b`

## Boolean Logical Operators

---

- **AND**: `a && b`
- **OR**: `a || b`
- **NOT**: `!a`

## Assignment Operator

---

- **Simple Assignment**: `a = b`
- **Addition Assignment**: `a += b`
- **Subtraction Assignment**: `a -= b`
- **Multiplication Assignment**: `a *= b`
- **Division Assignment**: `a /= b`
- **Modulus Assignment**: `a %= b`

## The ? Operator

---

- **Null Coalescing Operator**: `a ? b : c` (returns `b` if `a` is `null`, otherwise returns `c`)
- **Null Conditional Operator**: `a ? b : c` (returns `b` if `a` is not `null`, otherwise returns `c`)

## Operator Precedence

---

| Operator      | Precedence |
| ------------- | ---------- | --- | --- |
| `^`           | 3          |
| `*` and `/`   | 2          |
| `+` and `-`   | 1          |
| `==` and `!=` | 2          |
| `>` and `<`   | 2          |
| `&&` and `    |            | `   | 3   |
| `?`           | 4          |

## Important Formulas, Definitions, and Theorems

---

- **The Distributive Law**: `a * (b + c) = a * b + a * c`
- **The Associative Law**: `(a + b) + c = a + (b + c)`
- **The Commutative Law**: `a + b = b + a`

These formulas, definitions, and theorems are fundamental to understanding the behavior of operators in Java.
