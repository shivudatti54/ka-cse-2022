# Learning Objectives
After studying this topic, you should be able to:
1.  Define symmetric encryption and asymmetric encryption in your own words.
2.  Identify the core architectural difference between symmetric and asymmetric cryptosystems (single key vs. key pair).
3.  Explain the key distribution problem and how asymmetric encryption solves it.
4.  Compare and contrast the performance characteristics (speed, efficiency) of symmetric and asymmetric algorithms.
5.  Describe the purpose of the public key and the private key in an asymmetric key pair.
6.  Explain the concept of a hybrid cryptosystem and provide a real-world example of its use (e.g., TLS/SSL).
