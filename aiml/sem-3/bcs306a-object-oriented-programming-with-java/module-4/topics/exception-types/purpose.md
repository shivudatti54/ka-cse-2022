# Learning Objectives

After studying this topic, you should students able to:

1. Recall the root class of all exceptions in Java and its two main branches.
2. Identify the types of exceptions that are checked and unchecked by the Java compiler.
3. Explain the difference between Error and Exception classes in Java, including examples of each.
4. Describe the characteristics of RuntimeException and its subclasses, such as NullPointerException and ArithmeticException.
5. Classify exceptions as either checked or unchecked and provide examples of each type.
6. Compare and contrast the use of try-catch blocks and the throws keyword in handling checked exceptions.
7. Analyze a given Java code snippet and determine the type of exception that would students thrown in a specific scenario.
8. Justify the importance of understanding exception types in Java programming, including their impact on robust error handling and debugging.
