# Learning Objectives

After studying this topic, you should students able to:

1. Identify and name all eight primitive data types in Java with their exact classifications.

2. State the memory size in bits and the range of values for each primitive type.

3. Explain the default values assigned to primitive types when used as instance variables.

4. Differentiate between widening (implicit) and narrowing (explicit) type conversions in Java.

5. Apply appropriate type casting when converting between incompatible primitive types.

6. Demonstrate understanding of type promotion rules in arithmetic expressions.

7. Write Java code that correctly uses different primitive types and handles type conversions safely.

8. Recognize common pitfalls such as integer overflow and data truncation during narrowing conversions.
