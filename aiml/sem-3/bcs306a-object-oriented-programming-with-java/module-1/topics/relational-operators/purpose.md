# Learning Objectives

After studying this topic, you should students able to:

1. Identify and list all six relational operators in Java: ==, !=, >, <, >=, and <=

2. Understand that relational operators always return a boolean value (true or false)

3. Apply relational operators in comparing primitive data types including integers, floating-point numbers, characters, and booleans

4. Differentiate between reference comparison (==) and content comparison (equals() method) for objects in Java

5. Implement relational operators within conditional statements (if-else) and loop structures (while, for)

6. Recognize and avoid common mistakes such as using assignment (=) instead of equality (==), and comparing floating-point numbers directly

7. Explain the behavior of relational operators when comparing characters using ASCII/Unicode values

8. Solve practical programming problems using relational operators for validation, decision-making, and flow control
