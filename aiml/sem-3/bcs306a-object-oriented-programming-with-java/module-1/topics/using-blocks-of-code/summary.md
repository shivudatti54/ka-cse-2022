# Using Blocks of Code

## Overview

In Java, a block of code is a group of statements enclosed within curly brackets `{}`. Blocks of code are used to organize and structure the program's logic, making it easier to read, write, and maintain. There are three main types of blocks of code: method blocks, conditional blocks, and loop blocks.

## Key Points

- A block of code is a group of statements enclosed within curly brackets `{}`.
- Method blocks define the body of a method.
- Conditional blocks execute code based on a condition.
- Loop blocks execute code repeatedly.
- Blocks of code are used to organize and structure the program's logic.
- Use meaningful variable names and comments to make the code readable.
- Keep the block of code concise and focused on a single task.

## Important Definitions

- **Method Block**: A block of code that defines the body of a method.
- **Conditional Block**: A block of code that executes based on a condition.
- **Loop Block**: A block of code that executes repeatedly.

## Key Syntax

- Method block: `public static void methodName() { // code }`
- Conditional block: `if (condition) { // code }`
- Loop block: `for (int i = 0; i < 5; i++) { // code }`

## Comparisons

| Block Type        | Description                        | Example                           |
| ----------------- | ---------------------------------- | --------------------------------- |
| Method Block      | Defines the body of a method       | `public static void printHello()` |
| Conditional Block | Executes code based on a condition | `if (x > 10)`                     |
| Loop Block        | Executes code repeatedly           | `for (int i = 0; i < 5; i++)`     |

## Exam Tips

- Understand the different types of blocks of code and their uses.
- Practice writing clear and concise code using blocks.
- Pay attention to indentation, whitespace, and curly brackets.
