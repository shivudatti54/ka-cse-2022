# Connected components partition graph vertices into maximal connected subgraphs where every pair of vertices has a path between them. For undirected graphs, DFS/BFS algorithms find components in O(V+E) time, with Union-Find offering near-constant amortized time for dynamic connectivity. Strongly connected components (SCCs) in directed graphs require mutual reachability, and Kosaraju's algorithm computes them using two DFS passes on the transposed graphs original and. The condensation graph of SCCs is always a DAG. Applications span network analysis, social networks, image processing, compiler design, and web crawling.
