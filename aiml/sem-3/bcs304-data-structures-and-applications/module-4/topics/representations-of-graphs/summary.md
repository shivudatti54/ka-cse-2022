# Representations Of Graphs
Three primary graph representations were examined: (1) Adjacency Matrix - a V×V binary matrix offering O(1) edge operations but requiring O(V²) space, ideal for dense graphs; (2) Adjacency List - an array of linked lists requiring O(V+E) space, optimal for sparse graphs with O(deg(v)) edge operations; (3) Adjacency Multilist - combines list space efficiency with edge object pointers for efficient deletions. Key theorems established symmetry properties of undirected graph matrices and proved space-time trade-offs determining representation optimality based on graph density. The assessment validated understanding of representation selection criteria for graphs with varying vertex and edge counts.
