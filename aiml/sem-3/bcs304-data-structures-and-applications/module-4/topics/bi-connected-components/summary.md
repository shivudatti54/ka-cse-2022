# Bi Connected Components
Bi-connected components represent maximal 2-connected subgraphs in a graph. An articulation point (cut vertex) is a vertex whose removal increases the number of connected components. A bridge (cut edge) is an edge with the same property. The key theorem states that vertex u is an articulation point if and only if it has a child v in the DFS tree such that low[v] ≥ disc[u]. The root is an articulation point if and only if it has at least two children. Tarjan's algorithm finds articulation points in O(V+E) time using DFS with discovery times and low values. Biconnected components are extracted using a stack-based approach during DFS, where edges are pushed and popped when low[v] ≥ disc[u]. Bridges are identified when low[v] > disc[u] for tree edges. The content includes complete C implementation, worked examples with step-by-step DFS traces, and hard MCQs testing application-level understanding.
