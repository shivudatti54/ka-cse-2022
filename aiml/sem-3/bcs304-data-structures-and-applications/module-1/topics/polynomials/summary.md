# Polynomials
A polynomial P(x) = aₙxⁿ + ... + a₀ can be represented using three primary methods: (1) Dense array stores all n+1 coefficients in indices 0 to n, requiring Θ(n) space; (2) Sparse array stores only k non-zero terms as (coeff, expo) pairs, requiring Θ(k) space ideal for sparse polynomials where k << n; (3) Linked list provides dynamic insertion with O(1) head insertion. Key operations include addition (O(max(m,n)) for dense, O(k₁+k₂) for sparse), and evaluation using Horner's method in Θ(n) time. Selection depends on polynomial density: dense for dense polynomials (better cache locality), sparse/linked for sparse polynomials (better space efficiency).
