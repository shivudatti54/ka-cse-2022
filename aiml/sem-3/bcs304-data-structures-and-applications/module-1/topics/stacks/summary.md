# Stacks
A stack is a linear LIFO (Last In, First Out) data structure where insertion and deletion occur exclusively at one end called the top. This document establishes the mathematical foundations through formal proof of the LIFO property using mathematical induction, defines the Stack Abstract Data Type with complete axiomatic specifications including time complexity O(1) for all fundamental operations, presents two distinct implementation approaches—array-based (static, fixed capacity O(n) space) and linked list-based (dynamic, O(n) space)—with complete C implementations, analyzes time complexity (O(1) for push/pop/peek) and space complexity for both approaches, and demonstrates practical applications in function call management, expression evaluation (infix-to-postfix conversion), parenthesis matching, undo mechanisms, and backtracking algorithms. Five hard-level MCQs test application-based understanding including stack operation tracing, complexity analysis, overflow/underflow detection, expression conversion, and recursion stack usage.
