# Dynamic Allocated Arrays
Dynamic arrays are resizable random-access structures using heap memory. They maintain an invariant that size ≤ capacity, with resize triggered when appending to a full array. Using aggregate analysis, we prove append has O(1) amortized cost with a doubling strategy: total work for n operations is O(n) because geometric series of copy costs is bounded by 2n. Key operations include append O(1) amortized, insertAt O(n), and deleteAt O(n). The structure provides O(1) access time with superior cache locality compared to linked structures. Advanced considerations include shrinking strategy (50% threshold prevents thrashing), growth factor trade-offs (α=2 balances space/time), and cache benefits from contiguous allocation.
