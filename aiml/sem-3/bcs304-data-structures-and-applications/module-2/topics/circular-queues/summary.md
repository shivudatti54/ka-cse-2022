# Circular Queues
A Circular Queue (Ring Buffer) is a linear data structure implementing FIFO principle using circular boundary traversal via modular arithmetic: index = (index + 1) % capacity. It solves the space utilization problem of linear queues where dequeued positions become inaccessible. Key state variables are front (head index, initialized to -1) and rear (tail index, initialized to -1). The "one-slot-sacrifice" convention distinguishes empty (front=-1 AND rear=-1) from full ((rear+1)%capacity == front) states. All primitive operations—enqueue, dequeue, isEmpty, isFull, peek—execute in O(1) time with O(N) space. Formal proofs establish correctness via loop invariants. Applications include CPU scheduling (round-robin), buffer management, BFS traversal, and concurrent message passing. Alternative implementations include linked-lists (dynamic capacity) and dynamic array resizing (amortized analysis).
