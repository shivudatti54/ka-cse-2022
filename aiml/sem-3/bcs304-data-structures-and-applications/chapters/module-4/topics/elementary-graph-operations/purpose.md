# Learning Objectives

After studying this topic, you should be able to:

1. Compare and contrast adjacency matrix and adjacency list representations of graphs, including their space and time complexities.

2. Implement the Breadth-First Search (BFS) algorithm and explain its working principle using a queue data structure.

3. Implement the Depth-First Search (DFS) algorithm using both recursive and iterative approaches.

4. Perform basic vertex operations including adding vertices, removing vertices, and checking vertex existence.

5. Execute edge operations such as adding edges, removing edges, and checking edge existence in both directed and undirected graphs.

6. Calculate vertex degrees (in-degree and out-degree for directed graphs) and retrieve neighbors of a given vertex.

7. Analyze when to apply BFS versus DFS based on the problem requirements and understand the applications of each traversal method.

8. Apply graph traversal operations to solve practical problems like finding connected components and detecting cycles.
