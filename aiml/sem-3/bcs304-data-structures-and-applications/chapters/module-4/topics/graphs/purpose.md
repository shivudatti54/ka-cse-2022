# Learning Objectives

After studying this topic, you should be able to:

1. DEFINE and distinguish between different types of graphs including undirected, directed, simple, multigraph, weighted, and unweighted graphs.

2. EXPLAIN fundamental graph terminology including vertices, edges, degree, in-degree, out-degree, path, cycle, and connectivity.

3. IMPLEMENT graph representations using both adjacency matrix and adjacency list data structures, analyzing their space and time complexities.

4. COMPARE adjacency matrix and adjacency list representations, selecting the appropriate representation based on graph density and application requirements.

5. APPLY Breadth-First Search (BFS) algorithm for graph traversal, understanding its properties and applications in finding shortest paths.

6. APPLY Depth-First Search (DFS) algorithm for graph traversal, understanding recursion-based and iterative implementations.

7. UTILIZE graph traversal algorithms to solve problems such as cycle detection, connected component identification, and path finding.

8. ANALYZE the time and space complexities of elementary graph operations and represent these in big-O notation.
