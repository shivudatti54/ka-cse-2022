# Learning Objectives

After studying this topic, you should be able to:

1. Implement iterative and recursive algorithms for reversing a singly linked list, explaining the role of each pointer variable.

2. Apply the two-pointer (tortoise and hare) technique to find the middle element of a linked list with optimal time and space complexity.

3. Determine the nth node from the end of a linked list without computing the list length, using the pointer gap method.

4. Detect cycles in linked lists using Floyd's algorithm and identify the starting point of the cycle mathematically.

5. Merge two sorted linked lists into a single sorted list by rearranging existing node links, achieving linear time complexity.

6. Remove duplicate nodes from sorted linked lists while maintaining the sorted order of remaining elements.

7. Perform concatenation and splitting operations on linked lists, managing pointer adjustments at boundary points.

8. Analyze the time and space complexity of each additional operation and select appropriate algorithms for different scenarios.
