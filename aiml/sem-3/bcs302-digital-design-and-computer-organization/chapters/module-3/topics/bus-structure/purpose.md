# Learning Objectives

After studying this topic, you should be able to:

1. Define what a bus is and explain its role in interconnecting computer system components
2. Distinguish between the three main types of buses: data bus, address bus, and control bus
3. Explain why the data bus is bidirectional while the address bus is unidirectional
4. Describe the concept of bus arbitration and compare centralized, distributed, and daisy-chain methods
5. Analyze the trade-offs between single-bus and multi-bus computer structures
6. Calculate bus bandwidth given bus width and clock frequency
7. Determine maximum addressable memory based on the number of address lines
8. Explain how bus timing, wait states, and burst transfers affect system performance
