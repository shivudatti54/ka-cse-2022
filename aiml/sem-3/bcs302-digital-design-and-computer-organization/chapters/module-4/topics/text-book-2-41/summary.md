# **Text Book 2: 4.1 Revision Notes**

## **Definition and Key Points**

- **Digital Design**: The process of designing and building digital systems using electronic components.
- **Computer Organization**: The study of the structure, behavior, and performance of computer systems.

- **Digital Circuit**: A circuit that performs a specific digital function using electronic components.
- **Digital Signal**: A discrete-time signal that represents information in binary form.

## **Important Formulas and Equations**

- **Boolean Algebra**:
  - Complement law: A + A' = 1, AA' = 0
  - Distributive law: A(B + C) = AB + AC
  - De Morgan's law: A' + B' = (AB)'
- **Karnaugh Map**:
  - A 2-variable K-map is an 4x4 grid with 2^2 = 4 possible combinations.
  - The 3-variable K-map is a 8x8 grid with 2^3 = 8 possible combinations.

## **Theorems and Concepts**

- **The Boolean Algebra Laws**:
  - idempotent law: AA = A
  - commutative law: AB = BA
  - associative law: (AB)C = A(BC)
- **The Theorem of Countable Regions**:
  - For n variables, there are 2^n possible combinations.

## **Key Concepts and Terms**

- **Binary Counter**: A counter that can count in binary form (base 2).
- **Sequential Logic**: Logic circuits that have memory and can change state over time.
- **Combinational Logic**: Logic circuits that do not have memory and produce a single output for a given input.
