# Sequential Logic: Introduction, Sequential Circuits, Storage Elements: Latches, Flip-Flops

### Introduction

- Sequential logic deals with digital circuits that have memory and can store and retrieve information.
- Sequential circuits are composed of storage elements that can retain information for a period of time.
- Sequential logic is used in computers, calculators, and other digital devices.

### Sequential Circuits

- A sequential circuit is a digital circuit that has a memory element.
- Sequential circuits can store and retrieve information.
- Characteristics:
  - Memory elements
  - Feedback loops
  - Combinational logic

### Storage Elements: Latches

- A latch is a storage element that can retain information for a period of time.
- There are two types of latches:
  - Transparent latch (TL)
  - Invertible latch (IL)
- Characteristics:
  - TL: stores data and produces output when input changes
  - IL: stores data and produces inverted output when input changes
- Formulas:
  - TL: Q = (Q', Q) (input changes)
  - IL: Q = (Q', Q') (input changes)

### Storage Elements: Flip-Flops

- A flip-flop is a storage element that can retain information for a period of time.
- There are two types of flip-flops:
  - SR flip-flop
  - D flip-flop
- Characteristics:
  - SR flip-flop: stores data using two inputs (S, R)
  - D flip-flop: stores data using one input (D)
- Formulas:
  - SR flip-flop: Q = (S, R) (input changes)
  - D flip-flop: Q = D (input changes)

### Important Formulas and Definitions

- Truth table: a table that shows all possible input combinations and their corresponding outputs
- Karnaugh map: a graphical representation of a Boolean function
- Gates:
  - AND gate
  - OR gate
  - NOT gate
- Theorems:
  - Complement law
  - Idempotent law
  - Commutative law
