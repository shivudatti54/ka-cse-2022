# **Revision Notes: Text book 1: 4.1 - Digital Design and Computer Organization**

**Key Concepts:**

- **Binary Numbers and Representation**
  - Definition: Binary number is a base-2 number system that uses only two digits: 0 and 1.
  - Important Formulas:
    - Decimal to binary conversion: `Binary = (Decimal / 2)^0 + (Decimal % 2)^1`
  - **Binary Arithmetic Operations**
  - **Binary Multiplication and Division**
  - **Shift Operations**
  - **Addition and Subtraction in Binary**

**Digital Logic:**

- **Boolean Algebra**
  - Definition: Boolean algebra is a mathematical system that deals with logical operations.
  - Important Formulas:
    - `A + A = A` (Idempotent Law)
    - `A * A = A` (Idempotent Law)
- **Logical Operations**
  - AND (`A AND B` = 1 only if both A and B are 1)
  - OR (`A OR B` = 1 if either A or B is 1)
  - NOT (`NOT A` = 1 if A is 0, and 0 if A is 1)

**Digital Design:**

- **Binary Counter**
  - Definition: Binary counter is a digital circuit that counts in binary.
- **Flip-Flop**
  - Definition: Flip-flop is a digital circuit that can store a binary value.

**Important Theorems:**

- **De Morgan's Law**
  - `NOT (A AND B) = NOT A OR NOT B`
  - `NOT (A OR B) = NOT A AND NOT B`

**Key Terms:**

- **Binary System**
- **Boolean Algebra**
- **Digital Logic**
- **Digital Design**
