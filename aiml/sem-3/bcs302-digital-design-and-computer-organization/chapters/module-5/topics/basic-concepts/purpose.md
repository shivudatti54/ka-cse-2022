# Learning Objectives

After studying this topic, you should be able to:

1. Distinguish between computer architecture and computer organization, explaining the implications of each on system design and programmer perspective.

2. Explain the von Neumann architecture and the Harvard architecture, including their advantages and disadvantages, and understand the concept of the von Neumann bottleneck.

3. Describe the components of the CPU (control unit, ALU, registers) and their roles in instruction execution.

4. Trace through the steps of the instruction cycle, including both fetch and execute phases, for a simple machine instruction.

5. Identify and explain the three types of buses (data, address, control) and understand how bus width affects system performance.

6. Perform calculations related to memory addressing, including determining maximum addressable memory from address and data bus specifications.

7. Understand different data representation schemes, particularly two's complement representation for signed integers, and perform conversions.

8. Explain various addressing modes (immediate, direct, indirect, register, indexed) with examples.

9. Compare RISC and CISC architectures, understanding the design philosophy and trade-offs of each approach.
