# Learning Objectives

After studying this topic, you should be able to:

1. EXPLAIN the concept of pipelining and its role in improving processor throughput without reducing individual instruction execution time.

2. DESCRIBE the five stages of the classic RISC pipeline and the function of each stage in instruction execution.

3. CALCULATE the theoretical and actual speedup achieved by a pipelined processor compared to a non-pipelined processor.

4. IDENTIFY and DISTINGUISH between the three main types of pipeline hazards: structural, data, and control hazards.

5. EXPLAIN how data forwarding and compiler scheduling can resolve data hazards in pipeline processors.

6. ANALYZE the critical role of cache memory in pipeline performance and calculate average CPI considering cache miss rates.

7. INTERPRET pipeline timing diagrams to understand instruction overlap and identify potential stall cycles.
