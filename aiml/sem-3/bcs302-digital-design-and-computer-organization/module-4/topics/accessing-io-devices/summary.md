# Accessing Io Devices
This study material covers three primary methods of I/O access: Programmed I/O (polling), Interrupt-Driven I/O, and Direct Memory Access (DMA). It introduces I/O device classification and interface architecture, contrasting memory-mapped I/O with port-mapped I/O addressing schemes. Mathematical performance models are developed to analyze CPU overhead in polling operations and interrupt latency components. The DMA section provides detailed analysis of transfer modes (burst, cycle stealing, transparent) with comparative throughput formulas. Upon completion, students will be able to select appropriate I/O access methods based on device characteristics and performance requirements.
