# Speed Size And Cost Of Memory Systems
### Learning Purpose: Speed, Size and Cost of Memory Systems

**1. Why is this topic important?**

Memory system design represents one of the most critical bottlenecks in modern computer architecture. The processor-memory speed gap (known as the "memory wall") has widened exponentially since the 1980s, with processor speeds improving at approximately 55% annually while memory latency improves at only 7% annually. Understanding the fundamental trade-offs between speed, size, and cost equips engineers to design efficient memory hierarchies that maximize performance within economic constraints.

**2. How does this topic connect to the curriculum?**

This topic provides foundational context for subsequent modules on cache memory (mapping functions, replacement policies), virtual memory (address translation, page replacement), and I/O organization. The principles of locality directly inform cache design decisions, while the cost-performance analysis framework applies to all memory hierarchy levels.

**3. What will students learn?**

Students will understand the physical and economic reasons behind memory trade-offs, analyze memory technologies quantitatively using access time and cost-per-bit formulas, calculate average memory access time for hierarchical systems, and apply locality principles to predict cache performance. This analytical capability is essential for system architects and embedded systems designers.
