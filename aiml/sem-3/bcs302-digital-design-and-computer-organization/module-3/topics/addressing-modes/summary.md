# Addressing modes define how operands are specified in machine instructions, with the effective address (EA) representing the location from which data is accessed. Nine major addressing modes were examined: Immediate (operand embedded in instruction), Register (operand in CPU register), Direct (absolute memory address), Indirect (pointer in memory), Register Indirect (pointer in register), Indexed (base + index), Base-Register (base + displacement), Relative (PC + offset), and Stack (implicit top-of-stack). Each mode involves trade-offs between execution speed, instruction complexity, code density, and programming flexibility. Performance analysis reveals register operations as fastest (single cycle), while indirect modes incur multiple memory accesses. MCQ assessment requires effective address calculation, comparative analysis of modes for specific programming tasks, and numerical problems involving instruction encoding and addressable memory space calculations.
