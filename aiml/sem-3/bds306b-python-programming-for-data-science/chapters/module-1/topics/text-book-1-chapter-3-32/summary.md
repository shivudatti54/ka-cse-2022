# **Text Book 1: Chapter 3 (3.2) Revision Notes**

**Module:** Python Programming for Data Science
**Duration:** 6 hours

**Key Points:**

- **Data Types:**
  - Variables: `x = 5` (integer), `y = 3.14` (float), `name = "John"` (string)
  - Data types: `int`, `float`, `str`, `bool`
- **Operators:**
  - Arithmetic operators: `+`, `-`, `*`, `/`, `**` (exponentiation)
  - Comparison operators: `==`, `!=`, `>`, `<`, `>=` , `<=`
  - Logical operators: `and`, `or`, `not`
- **Control Structures:**
  - If-else statements: `if condition: action`, `elif condition: action`, `else: action`
  - Loops: `for loop`, `while loop`
- **Functions:**
  - Defining a function: `def function_name(parameters):`
  - Calling a function: `function_name(arguments)`
- **List Operations:**
  - Indexing: `my_list[0]` (accessing an element)
  - Slicing: `my_list[1:3]` (accessing a subset of elements)
  - Append: `my_list.append(element)`
  - Insert: `my_list.insert(index, element)`
- **Important Formulas:**
  - Sum of a list: `sum(my_list)`
  - Length of a list: `len(my_list)`
- **Important Definitions:**
  - Variable: a name given to a value
  - Data type: the type of value stored in a variable
- **Important Theorems:**
  - None (this chapter focuses on programming concepts)

**Quick Revision Tips:**

- Practice writing simple Python programs to reinforce your understanding of the concepts.
- Focus on common errors and pitfalls when working with data types, operators, and control structures.
- Review the basics of functions, lists, and data structures before moving on to more advanced topics.
