# Learning Objectives
After studying this topic, you should be able to:
1.  Define a hash index and explain its core purpose in a DBMS.
2.  Describe the process of inserting and looking up a record using a hash index.
3.  Explain the problem of hash collisions and compare the separate chaining and open addressing resolution techniques.
4.  Differentiate between static and dynamic hashing, explaining the limitations of static hashing.
5.  Explain the structure and operation of the extendible hashing technique, including the roles of the directory, global depth, and local depth.
6.  Compare and contrast hash indexes with B+ tree indexes, identifying the appropriate use cases for each.
