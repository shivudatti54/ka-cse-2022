## **4.2 Introduction to Probability**

- Probability is a measure of the likelihood of an event occurring
- Probability is always between 0 and 1 (0 ≤ P(A) ≤ 1)
- Events can be mutually exclusive or not mutually exclusive

## **4.3 Basic Probability Rules**

- **Complement Rule**: P(A') + P(A) = 1 (where A' is the complement of A)
- **Addition Rule**: P(A ∪ B) = P(A) + P(B) - P(A ∩ B)
- **Multiplication Rule**: P(A ∩ B) = P(A) × P(B)

## **4.4 Conditional Probability**

- Conditional probability is the probability of an event occurring given that another event has occurred
- Definition: P(A|B) = P(A ∩ B) / P(B)
- **Rule of Total Probability**: P(A) = Σ P(A|B) × P(B)

## **4.5 Bayes' Theorem**

- Bayes' Theorem is a formula for updating the probability of a hypothesis based on new evidence
- Definition: P(H|E) = P(E|H) × P(H) / P(E)

## **4.6 Applications of Probability**

- **Random Variables**: A random variable is a function that assigns a numerical value to each outcome of an experiment
- **Expected Value**: E(X) = ∑ xP(X=x)
