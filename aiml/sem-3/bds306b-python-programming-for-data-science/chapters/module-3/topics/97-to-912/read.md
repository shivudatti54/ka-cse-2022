# **9.7 to 9.12: Introduction to Data Science with Python**

### 9.7: What is Data Science?

**Definition:** Data science is a field that involves extracting insights and knowledge from data using various techniques such as machine learning, statistical modeling, and data visualization.

**Key Concepts:**

- Data science is a multidisciplinary field that combines computer science, statistics, and domain knowledge.
- Data science involves four main steps: data collection, data cleaning, data analysis, and data visualization.
- Data science requires strong programming skills, data manipulation, and visualization techniques.

### 9.8: Why Use Python for Data Science?

**Why Python?**

- Python is a high-level, general-purpose programming language that is easy to learn and use.
- Python has a vast collection of libraries and frameworks that make it ideal for data science tasks.
- Python is widely used in industry and academia for data science applications.

**Key Libraries:**

- NumPy and Pandas for data manipulation and analysis.
- Matplotlib and Seaborn for data visualization.
- Scikit-learn for machine learning.

### 9.9: Data Types in Python

**Integer vs. Float**

- Integers are whole numbers, e.g., 1, 2, 3, etc.
- Floats are decimal numbers, e.g., 3.14, -0.5, etc.

**String**

- Strings are sequences of characters, e.g., "hello", "world", etc.

**Boolean**

- Booleans are true or false values.

**Key Concepts:**

- Use the `type()` function to check the data type of a variable.
- Use the `int()` and `float()` functions to convert variables to integers and floats.
- Use the `str()` function to convert variables to strings.

### 9.10: Basic Data Structures in Python

**Lists**

- Lists are ordered collections of items, e.g., [1, 2, 3], ["hello", "world"], etc.

**Tuples**

- Tuples are ordered collections of items that cannot be changed, e.g., (1, 2, 3), ("hello", "world"), etc.

**Dictionaries**

- Dictionaries are unordered collections of key-value pairs, e.g., {"name": "John", "age": 30}, etc.

**Key Concepts:**

- Use the `append()` method to add items to a list.
- Use the `index()` method to find the index of an item in a list.
- Use the `get()` method to retrieve a value from a dictionary by its key.

### 9.11: Data Manipulation with Pandas

**Importing Pandas**

- Import the Pandas library using `import pandas as pd`.

**Creating a DataFrame**

- Create a DataFrame from a dictionary using `pd.DataFrame()`.
- Create a DataFrame from a list of lists using `pd.DataFrame()`.
- Create a DataFrame from a CSV file using `pd.read_csv()`.

**Key Concepts:**

- Use the `head()` method to display the first few rows of a DataFrame.
- Use the `tail()` method to display the last few rows of a DataFrame.
- Use the `sort_values()` method to sort a DataFrame by a column.

### 9.12: Data Visualization with Matplotlib and Seaborn

**Importing Matplotlib and Seaborn**

- Import the Matplotlib library using `import matplotlib.pyplot as plt`.
- Import the Seaborn library using `import seaborn as sns`.

**Creating a Plot**

- Create a line plot using `plt.plot()`.
- Create a scatter plot using `plt.scatter()`.
- Create a bar plot using `plt.bar()`.

**Key Concepts:**

- Use the `title()` method to set the title of a plot.
- Use the `xlabel()` method to set the x-axis label.
- Use the `ylabel()` method to set the y-axis label.
