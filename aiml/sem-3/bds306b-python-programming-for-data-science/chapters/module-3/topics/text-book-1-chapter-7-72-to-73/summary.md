# **Text Book 1: Chapter 7 (7.2 to 7.3) Revision Notes**

### Key Concepts

- **Univariate Data**
  - One-dimensional data
  - Descriptive statistics: mean, median, mode, range
  - Visualizations: histograms, box plots
- **Bivariate Data**
  - Two-dimensional data
  - Correlation: Pearson's r, Spearman's rho
  - Regression: simple linear regression, multiple linear regression

### Import Formulas and Definitions

- **Mean**
  - Formula: μ = (Σx) / n
  - Definition: Average value of a dataset
- **Median**
  - Formula: Q = (x*n + x*{n+1}) / 2
  - Definition: Middle value of a dataset when it is sorted
- **Mode**
  - Formula: mode = x_j
  - Definition: Value that appears most frequently in a dataset
- **Range**
  - Formula: R = max(x) - min(x)
  - Definition: Difference between largest and smallest values in a dataset
- **Correlation Coefficient**
  - Formula: r = Σ[(xi - x̄)(yi - ȳ)] / sqrt[Σ(xi - x̄)^2 * Σ(yi - ȳ)^2]
  - Definition: Measures strength and direction of linear relationship between two variables

### Important Theorems

- **Central Limit Theorem (CLT)**
  - States that sample means will be approximately normally distributed for large sample sizes
  - Implication: Parametric inference can be used with large sample sizes

### Quick Review

- Univariate data: descriptive statistics, visualizations
- Bivariate data: correlation, regression
- Formulae and definitions: mean, median, mode, range, correlation coefficient
- Theorems: CLT
