### Chapter 8 Key Points

- **8.1: Introduction to Pandas**
  - Pandas is a Python library for data manipulation and analysis.
  - It provides data structures and functions to efficiently handle structured data, including tabular data such as spreadsheets and SQL tables.
- **8.2: Data Structures in Pandas**
  - Series: one-dimensional labeled array of values.
  - DataFrame: two-dimensional labeled data structure with columns of potentially different types.
- **8.3: Data Manipulation in Pandas**
  - Selecting and filtering data using `loc`, `iloc`, and `ix`.
  - Grouping and aggregating data using `groupby` and `pivot_table`.
- **8.4: Merging and Joining Data**
  - Merging data from multiple sources using `merge` and `join`.
  - Inner, left, right, and full outer joins.

### Chapter 9 Key Points

- **9.1: Introduction to Matplotlib and Seaborn**
  - Matplotlib is a plotting library for creating static, animated, and interactive visualizations.
  - Seaborn is a visualization library built on top of Matplotlib, providing a high-level interface for creating attractive statistical graphics.
- **9.2: Visualization with Matplotlib**
  - Creating lines, scatter, and bar plots using `plot`, `scatter`, and `bar`.
  - Customizing plot appearance using `title`, `xlabel`, `ylabel`, and `legend`.
- **9.3: Visualization with Seaborn**
  - Creating informative and attractive statistical graphics using `pairplot`, `heatmap`, and `boxplot`.
  - Customizing plot appearance using `palette`, `hue`, and `scale`.

### Important Formulas, Definitions, and Theorems

- **Mean**: μ = (Σx) / n (where μ is the mean, x is each data point, and n is the number of data points)
- **Standard Deviation**: σ = √[(Σ(x - μ)^2) / (n - 1)] (where σ is the standard deviation, x is each data point, μ is the mean, and n is the number of data points)
- **Variance**: σ^2 = (Σ(x - μ)^2) / (n - 1) (where σ^2 is the variance, x is each data point, μ is the mean, and n is the number of data points)
