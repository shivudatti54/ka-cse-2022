# Graphics using R Exploratory Data Analysis, Main Graphical Packages

## Revision Notes

### **Exploratory Data Analysis (EDA) in R**

- **What is EDA?**: The process of summarizing and visualizing datasets to understand the relationships and patterns.
- **R Packages for EDA:**
  - `base graphics` (built-in)
  - `ggplot2`
  - `plotly`
- **Key Plot Types for EDA:**
  - Histograms
  - Box Plots
  - Scatter Plots
  - Bar Plots

### **Main Graphical Packages in R**

- **`graphics` package:**
  - Basic line plots
  - Basic scatter plots
- **`ggplot2` package:**
  - High-level graphics for data visualization
  - Supports various plot types (e.g., line plots, scatter plots, bar plots)
- **`plotly` package:**
  - Interactive 2D and 3D plots

### **Pie Charts**

- **Definition**: A circular graph showing proportions of different categories.
- **Formula:** Not applicable (visual representation only)
- **Theorem:** None (visual representation only)

### **Scatter Plots**

- **Definition**: A graph showing the relationship between two variables.
- **Formula:** Not applicable (visual representation only)
- **Theorem:** Not applicable (visual representation only)

### **Line Plots**

- **Definition**: A graph showing the trend of a variable over time or across categories.
- **Formula:** `plot(x, y, type = "l")`
- **Theorem:** Not applicable (visual representation only)

### **Histograms**

- **Definition**: A graphical representation of the distribution of a variable.
- **Formula:** `hist(x, main = "", xlab = "", ylab = "")`
- **Theorem:** None (visual representation only)

### **Box Plots**

- **Definition**: A graphical representation of the distribution of a variable.
- **Formula:** `boxplot(x, main = "", xlab = "", ylab = "")`
- **Theorem:** None (visual representation only)

### **Bar Plots**

- **Definition**: A graphical representation of the distribution of a variable.
- **Formula:** `barplot(x, main = "", xlab = "", ylab = "")`
- **Theorem:** None (visual representation only)

### **Other Graphical Types**

- **Heatmaps:** Visual representation of data density (e.g., gene expression).
- **Network plots:** Visual representation of relationships between variables (e.g., social networks).
- **Violin plots:** Visual representation of the distribution of a variable (similar to box plots).
