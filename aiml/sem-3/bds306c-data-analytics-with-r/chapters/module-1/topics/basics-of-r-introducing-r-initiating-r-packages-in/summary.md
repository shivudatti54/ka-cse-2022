# **Basics of R Revision Notes**

### Introduction to R

- R is a free and open-source programming language for statistical computing and graphics.
- R is widely used in data analysis, machine learning, and data visualization.
- R is available for Windows, macOS, and Linux operating systems.

### Initiating R

- To initiate R, simply type `R` in the command prompt or terminal.
- You can also install R from the official R website: <https://www.r-project.org/>

### Packages in R

- Packages are libraries of pre-written R code that can enhance the functionality of R.
- R has over 15,000 packages available on CRAN (Comprehensive R Archive Network).
- To install a package, use the `install.packages()` function.

### Environments and Functions

- An R environment is a workspace where R code is executed.
- Functions in R are blocks of code that perform a specific task.
- R has built-in functions, and users can also create their own functions.

### Flow Controls

- Flow controls are used to control the flow of R code based on conditions.
- R has the following flow control statements:
  - `if()`: executes a block of code if a condition is true.
  - `else`: executes a block of code if a condition is false.
  - `for()`: repeats a block of code for a specified number of iterations.
  - `while()`: repeats a block of code while a condition is true.

### Loops

- Loops are used to repeat a block of code for a specified number of iterations.
- R has two types of loops:
  - `for()`: loops over a sequence of values.
  - `while()`: loops while a condition is true.

### Basic Data Types in R

- R has the following basic data types:
  - `character`: text data (e.g., strings).
  - `numeric`: numerical data (e.g., numbers).
  - `integer`: whole numbers.
  - `logical`: boolean data (e.g., true or false).
  - `complex`: complex numbers.

### Vectors

- Vectors are a type of data structure in R that store a sequence of values.
- Vectors are created using the `c()` function or the comma operator.
- Vectors have several methods:
  - `length()`: returns the length of a vector.
  - `sum()`: returns the sum of a vector.
  - `mean()`: returns the mean of a vector.
