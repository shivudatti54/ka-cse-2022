# Probability Distributions: Review of Basic Probability Theory

### Definitions and Notations

- **Random Variable**: A function that assigns a numerical value to each outcome of a random experiment.
- **Probability Distribution**: A function that assigns a probability value to each possible outcome of a random experiment.
- **Random Experiment**: An event whose outcome is unknown until observed.
- **Sample Space**: The set of all possible outcomes of a random experiment.
- **Event**: A subset of the sample space.

### Basic Concepts

- **Probability**: A number between 0 and 1 that represents the likelihood of an event occurring.
- **Experiment**: A random procedure with a finite number of outcomes.
- **Random Variable**: A variable that takes on different values based on the outcome of a random experiment.

### Probability Rules

- **Complement Rule**: P(E) = 1 - P(E')
- **Addition Rule**: P(E ∪ F) = P(E) + P(F) - P(E ∩ F)
- **Multiplication Rule**: P(E ∩ F) = P(E)P(F)
- **Rule of Total Probability**: P(E) = ∑P(E|H_i)P(H_i)

### Probability Distributions

- **Discrete Random Variable**: A random variable that takes on only countable values.
- **Continuous Random Variable**: A random variable that takes on uncountable values.
- **Probability Mass Function (PMF)**: A function that assigns a probability value to each possible outcome of a discrete random variable.
- **Probability Density Function (PDF)**: A function that assigns a probability value to each possible outcome of a continuous random variable.

### Important Formulas

- **Bernoulli's Equation**: P(X = k) = (p^k)(q^(1-k))
- **Binomial Distribution**: P(X = k) = (nCk)(p^k)(q^(n-k))
- **Normal Distribution**: f(x) = (1/σ√(2π)) e^(-((x-μ)^2) / (2σ^2))

### Important Theorems

- **Law of Large Numbers (LLN)**: The average of a large number of independent and identically distributed random variables converges to the population mean.
- **Central Limit Theorem (CLT)**: The distribution of the mean of a large sample of independent and identically distributed random variables converges to a normal distribution.
