# Mathematical Expectation

### Definitions

- **Expected Value (or Mathematical Expectation)**: The long-run average value of a random variable.
- **Random Variable**: A function that assigns a numerical value to each outcome of a probability experiment.
- **Probability Distribution**: A function that assigns a probability to each possible value of a random variable.

### Formulas

- **Expected Value (E(X))**: `E(X) = ∑xP(x)`, where x is the value of the random variable and P(x) is the probability of x.
- **Linearity of Expectation**: `E(aX + b) = aE(X) + b`, where a and b are constants.

### Theorems

- **Law of Iterated Expectation**: `E(E(X|Y)) = E(X)`, where X and Y are random variables.
- **No-Free-Lunch Theorem**: It is impossible to build a strategy that is guaranteed to win against every possible opponent.

### Key Points

- **Linearity of Expectation**: Can be used to simplify complex problems by breaking them down into smaller, more manageable parts.
- **Law of Iterated Expectation**: Can be used to simplify problems involving conditional expectations.
- **No-Free-Lunch Theorem**: Highlights the importance of understanding the underlying probability distribution when making decisions.

### Important Formulas

- **Borel-Cantelli Lemma**: If `∑P(A_i) < ∞`, then `P(∞ ∑ A_i) = 0`.
- **Chebyshev's Inequality**: `P(|X - E(X)| ≥ k) ≤ σ^2 / k^2`, where σ is the standard deviation of X.

### Notes

- **Standard Deviation (σ)**: A measure of the spread or dispersion of a probability distribution.
- **Variance (σ^2)**: The square of the standard deviation and a measure of the average squared deviation from the mean.
