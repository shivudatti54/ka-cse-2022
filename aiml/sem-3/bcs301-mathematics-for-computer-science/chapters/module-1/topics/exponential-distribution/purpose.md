# Learning Objectives

After studying this topic, you should be able to:

1. Define the Exponential Distribution and identify its probability density function (PDF) and cumulative distribution function (CDF).

2. Calculate the mean, variance, and standard deviation of an exponentially distributed random variable.

3. Apply the memoryless property to solve conditional probability problems involving exponential random variables.

4. Compute probabilities using the CDF formula P(X ≤ x) = 1 - e^(-λx) and survival function P(X > x) = e^(-λx).

5. Recognize the relationship between the Poisson process (discrete event counting) and the Exponential Distribution (continuous waiting time).

6. Solve real-world problems involving inter-arrival times, reliability analysis, and queuing scenarios.

7. Distinguish the Exponential Distribution from other continuous distributions and identify scenarios where it is the appropriate model.
