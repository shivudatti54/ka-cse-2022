# Introduction to Statistical Inference - Summary

## Key Definitions and Concepts

- **Statistical Inference**: The process of drawing conclusions about a population based on sample data using probability theory

- **Population**: The complete set of all items or observations under study; characterized by parameters (μ, σ², P)

- **Sample**: A subset of the population selected for study; characterized by statistics (x̄, s², p̂)

- **Parameter**: A numerical measure that describes a characteristic of the population (fixed but unknown)

- **Statistic**: A numerical measure computed from sample data (random variable that varies)

- **Estimation**: Using sample data to estimate population parameters (point estimates and interval estimates)

- **Hypothesis Testing**: Making and testing claims about population parameters using sample evidence

- **Simple Random Sample**: A sample where every population member has equal chance of selection

## Important Formulas and Theorems

- Sample mean: x̄ = Σxᵢ / n
- Sample proportion: p̂ = x/n (where x is number of successes in n trials)
- Population mean: μ (parameter)
- Population variance: σ² (parameter)

## Key Points

1. Statistical inference bridges sample data and population conclusions through probability

2. Parameters (population) are fixed and unknown; statistics (sample) are random and known

3. The two branches of inference are estimation and hypothesis testing

4. Descriptive statistics summarizes data; inferential statistics makes generalizations

5. Random sampling ensures representativeness and validity of inferences

6. The sampling distribution describes how statistics vary across different samples

7. Probability quantifies uncertainty in inferential conclusions

8. Computer science applications include algorithm comparison, A/B testing, and quality assurance

## Common Mistakes to Avoid

1. Confusing parameters with statistics—remember Greek letters for parameters, Roman letters for statistics

2. Assuming a sample statistic exactly equals the population parameter—always acknowledge sampling error

3. Using non-random samples and then applying inferential methods—invalid sampling invalidates inference

4. Confusing descriptive statistics (what the sample shows) with inferential statistics (what the population likely has)
