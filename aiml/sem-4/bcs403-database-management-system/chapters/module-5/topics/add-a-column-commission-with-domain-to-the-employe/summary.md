# **Add a Column Commission with Domain to the Employee Table**

## **Key Points**

- **Commission Column**: Add a new column to the `Employee` table to store commission data.
- **Domain**: Define the domain for the commission column to ensure data integrity.
- **Data Type**: Choose an appropriate data type for the commission column (e.g., `DECIMAL`, `FLOAT`).
- **Formula**: Create a formula to calculate commission based on the `Salary` and `Commission_Pct` columns.
- **Theoretical Background**:
  - **Definition of Commission**: Commission is a payment made to an employee based on their sales or performance.
  - **Commission Formula**: Commission = Salary \* Commission_Pct

## **Important Formulas and Definitions**

- **Commission Formula**: Commission = Salary \* Commission_Pct
- **Domain**: A set of values that a column can take, ensuring data integrity and preventing invalid data entry.
