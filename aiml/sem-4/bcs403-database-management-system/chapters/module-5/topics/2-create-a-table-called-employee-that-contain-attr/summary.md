# **Database Management System: Creating an Employee Table**

### Key Points

- A table is a fundamental concept in database management systems.
- The `EMPNO` attribute is a primary key, unique identifier for each employee.
- The following attributes are included in the Employee table:

- `EMPNO` (Employee Number)
- `ENAME` (Employee Name)
- `JOOID` (Job ID)
- `MGRNO` (Manager's Employee Number)
- `DEPTNO` (Department Number)
- `SAL` (Salary)
- `COMM` (Commission)

### Important Formulas

- None

### Definitions

- **Primary Key (PK):** A unique identifier for each record in a table, ensuring data integrity and preventing duplicate entries.
- **Foreign Key (FK):** A field in a table that references the primary key of another table, establishing relationships between tables.

### Theorems

- **Data Invariance Theorem:** A database management system maintains data consistency, ensuring that data remains unchanged over time.
- **Data Integrity Theorem:** A database management system ensures that data is accurate, complete, and consistent.
