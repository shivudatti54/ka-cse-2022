# Check the Result

### Overview

- A check the result operation is used to verify the accuracy of data stored in a database.
- It involves comparing the expected result with the actual result.

### Key Points

- **Definition:** Check the result operation is used to verify the accuracy of data stored in a database.
- **Purpose:** To detect any errors or inconsistencies in the data.
- **Types:**
  - **Check the result:** Verifies the accuracy of a single record.
  - **Check the result on multiple records:** Verifies the accuracy of multiple records.

### Important Formulas

- None

### Important Definitions

- **Expected result:** The expected value of a record or set of records.
- **Actual result:** The value of the record or set of records actually stored in the database.
- **Error:** A discrepancy between the expected result and the actual result.

### Important Theorems

- **Check the result theorem:** If the check the result operation returns false, then there is an error in the data.
