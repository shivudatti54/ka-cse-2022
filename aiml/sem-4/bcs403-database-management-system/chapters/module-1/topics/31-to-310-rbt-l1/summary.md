# **DBMS Revision Notes: 3.1 to 3.10 RBT L1**

### Introduction

---

- Database Management System (DBMS) is a software system that allows to define, create, maintain, and manipulate databases.
- It provides a interface for accessing and managing data.

### Key Concepts

---

- **Database**: A collection of organized data.
- **Relational Database**: A database that stores data in the form of tables with defined relationships.
- **DBMS**: Software that manages a database.

### RDBMS (Relational Database Management System)

---

### Key Features

---

- **Atomicity**: A database transaction can be treated as a single, indivisible unit.
- **Consistency**: The database must maintain consistency throughout.
- **Isolation**: Multiple transactions can be processed concurrently without interfering with each other.
- **Durability**: Once a transaction is committed, its effects are permanent.

### Important Formulas

---

- **ACID Theorem**:
  - Atomicity: A database transaction can be treated as a single, indivisible unit.
  - Consistency: The database must maintain consistency throughout.
  - Isolation: Multiple transactions can be processed concurrently without interfering with each other.
  - Durability: Once a transaction is committed, its effects are permanent.
- **Relational Algebra**: A mathematical language used to manipulate relational databases.
- **Database Normalization**: A process of organizing data in a database to minimize data redundancy.

### Important Definitions

---

- **Primary Key**: A unique identifier for each record in a table.
- **Foreign Key**: A field in a table that refers to the primary key of another table.
- **Index**: A data structure that improves the speed of data retrieval.

### Important Theorems

---

- **Database Independence**: A DBMS should work on any hardware platform.
- **Data Independence**: A DBMS should work on any hardware platform, regardless of the data definition.

### Important SQL Commands

---

- **CREATE**: Creates a new database or table.
- **INSERT**: Inserts new data into a table.
- **SELECT**: Retrieves data from a table.
- **UPDATE**: Updates existing data in a table.
- **DELETE**: Deletes data from a table.
