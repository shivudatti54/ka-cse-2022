# Module 5: Introduction to Group Theory - A Semester-End Guide

## 1. Introduction

Group Theory is a fundamental branch of **abstract algebra** with profound applications in engineering, particularly in computer science (coding theory, cryptography), computer graphics (transformations, symmetries), and quantum physics. For your Discrete Mathematical Structures course, understanding the core definitions and properties of groups is essential. This guide breaks down the key concepts you must know for your semester-end examination.

## 2. Core Concepts

### What is a Group?

A **group** is an algebraic structure consisting of a non-empty set, G, together with a binary operation (let's denote it by *) that combines any two elements *a* and *b* to form another element, written *a * b*.

For the pair (G, *) to be considered a group, it must satisfy the following four axioms (properties):

1.  **Closure:** For all *a, b* in G, the result of the operation *a * b* is also in G.
2.  **Associativity:** For all *a, b, c* in G, the equation *(a * b) * c = a * (b * c)* holds.
3.  **Identity Element:** There exists an element *e* in G such that for every element *a* in G, the equations *e * a = a* and *a * e = a* hold.
4.  **Inverse Element:** For each *a* in G, there exists an element *b* in G (denoted as *a⁻¹*), such that *a * b = e* and *b * a = e*, where *e* is the identity element.

### Key Types of Groups

*   **Finite vs. Infinite Group:** A group (G, *) is **finite** if the set G has a finite number of elements. The number of elements is called the **order** of the group (denoted |G|). Otherwise, it is an infinite group.
*   **Abelian Group:** A group is called **Abelian** (or commutative) if, in addition to the four group axioms, it also satisfies *commutativity*: *a * b = b * a* for all *a, b* in G.
*   **Cyclic Group:** A group is **cyclic** if it can be generated by a single element. This means every element in G can be written as *aⁿ* (if the operation is multiplication) or *n*a* (if the operation is addition) for some integer *n*, where *a* is the **generator**.

## 3. Examples

Let's solidify these definitions with common examples you will likely see on an exam.

**Example 1: The Set of Integers under Addition, (ℤ, +)**
*   **Closure:** The sum of any two integers is an integer. ✅
*   **Associativity:** (a + b) + c = a + (b + c). ✅
*   **Identity Element:** The integer 0, since a + 0 = 0 + a = a. ✅
*   **Inverse Element:** The inverse of any integer *a* is *-a*, since a + (-a) = 0. ✅
This is an **infinite, Abelian group**.

**Example 2: The Set {1, -1, i, -i} under Multiplication, (G, ×)**
*   **Set:** G = {1, -1, i, -i}
*   **Closure:** e.g., i × i = -1 (which is in G), -1 × -1 = 1 (in G). ✅
*   **Associativity:** Holds for complex multiplication. ✅
*   **Identity Element:** 1, since 1 × a = a for any a in G. ✅
*   **Inverse Element:**
    *   Inverse of 1 is 1.
    *   Inverse of -1 is -1.
    *   Inverse of i is -i (since i × -i = -(i²) = -(-1) = 1).
    *   Inverse of -i is i.
    ✅
This is a **finite, Abelian group** of order 4.

**Non-Example: The Set of Integers under Subtraction, (ℤ, –)**
This is **not a group**. Let's check the axioms:
*   **Closure:** Holds. Subtracting two integers gives an integer.
*   **Associativity:** Fails! (a - b) - c ≠ a - (b - c). For example, (5 - 3) - 2 = 0, but 5 - (3 - 2) = 4.
Since associativity fails, (ℤ, –) is not a group.

## 4. Key Points & Summary

For your exam, remember these crucial takeaways:

*   **The Four Axioms are Key:** Memorize Closure, Associativity, Identity, and Inverse. You will be asked to verify if a given set and operation form a group.
*   **Identity is Unique:** A group has exactly one identity element.
*   **Inverses are Unique:** Each element in a group has exactly one inverse.
*   **Abelian means Commutative:** If the operation is commutative (like + or × on real numbers), the group is Abelian.
*   **Common Groups:**
    *   (ℤₙ, +ₙ) - Integers modulo *n* under addition modulo *n*. This is a fundamental **finite, cyclic, Abelian group**.
    *   (ℝ, +) and (ℝ\{0}, ×) - The real numbers under addition and the non-zero reals under multiplication are both **infinite, Abelian groups**.
*   **Notation Matters:** Be comfortable with both multiplicative (a*b, a⁻¹) and additive (a+b, -a) notation. The additive notation is often used for Abelian groups.

**Exam Tip:** When asked to "Prove that (G, *) is a group," methodically check each of the four axioms in order. Provide a specific counterexample if any one of them fails. Practice this with different sets and operations.
