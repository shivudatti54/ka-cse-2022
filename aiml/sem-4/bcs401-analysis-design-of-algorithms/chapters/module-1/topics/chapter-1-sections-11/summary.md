# Algorithms and Data Structures Revision Notes

### Chapter 1: Introduction to Algorithms

#### Section 1.1: What is an Algorithm?

- **Definition:** An algorithm is a well-defined procedure that takes some input and produces a corresponding output.
- **Characteristics:**
  - Finite: It has a finite number of steps.
  - Unambiguous: It has a unique solution.
  - Efficient: It uses minimal time and space.
- **Importance:** Algorithms are used to solve problems, process data, and make decisions in computer science.

### Key Concepts

- **Input:** The data provided to an algorithm.
- **Output:** The result produced by an algorithm.
- **Algorithmic Complexity:** A measure of an algorithm's efficiency, typically measured by its time and space complexity.

### Important Formulas, Definitions, and Theorems

- **Big O Notation:**
  - `O(1)` - Constant time complexity
  - `O(log n)` - Logarithmic time complexity
  - `O(n)` - Linear time complexity
  - `O(n log n)` - Linearithmic time complexity
  - `O(n^2)` - Quadratic time complexity
  - `O(2^n)` - Exponential time complexity
- **Time Complexity:** The amount of time an algorithm takes to complete, typically measured in terms of the size of the input.
- **Space Complexity:** The amount of memory an algorithm uses, typically measured in terms of the size of the input.

### Important Terms

- **Pseudocode:** A high-level representation of an algorithm using a programming language-like syntax.
- **Algorithm Design:** The process of creating a new algorithm or improving an existing one.
- **Algorithm Analysis:** The process of analyzing an algorithm's time and space complexity.
