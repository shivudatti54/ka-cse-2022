# **Chapter 3(Section 3.1) Revision Notes**

## **Key Concepts**

- Definition of an Algorithm:
  - A well-defined procedure for solving a problem or performing a specific task.
  - A set of instructions that is used to solve a problem or perform a computation.
- Characteristics of an Algorithm:
  - Input: A set of values that are used to solve a problem.
  - Output: The result of solving the problem.
  - Step-by-Step Procedure: A sequence of instructions that are used to solve the problem.
  - Finite: An algorithm must have a finite number of steps.
- Types of Algorithms:
  - Recursive Algorithms: Algorithms that use recursion to solve a problem.
  - Iterative Algorithms: Algorithms that use loops to solve a problem.

## **Important Formulas and Definitions**

- Big O Notation:
  - A measure of the complexity of an algorithm.
  - It represents the upper bound of an algorithm's running time or space complexity.
  - Functions that describe the complexity of an algorithm are written in Big O notation.
- Time Complexity:
  - The amount of time an algorithm takes to complete.
  - Measured in terms of the size of the input (e.g., n).
- Space Complexity:
  - The amount of memory an algorithm uses.
  - Measured in terms of the size of the input (e.g., n).

## **Important Theorems**

- **Divide and Conquer Theorem**
  - A method for solving a problem by breaking it down into smaller subproblems.
  - Each subproblem is solved independently, and the solutions are then combined to solve the original problem.
- **Greedy Algorithm Theorem**
  - A method for solving a problem by making the locally optimal choice at each step.
  - The locally optimal choice is made without considering the overall optimality of the solution.

## **Key Terms**

- **Pseudocode**: A high-level representation of an algorithm that is written in a formal but not necessarily executable language.
- **Algorithmic Efficiency**: The ability of an algorithm to solve a problem efficiently in terms of time and space complexity.
- **Well-Defined**: An algorithm that is precisely defined and unambiguous.
