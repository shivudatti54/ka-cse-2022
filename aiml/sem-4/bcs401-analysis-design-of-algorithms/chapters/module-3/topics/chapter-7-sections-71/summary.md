### Chapter 7 (Sections 7.1) Revision Notes: Analysis & Design of Algorithms

#### Key Concepts

- **Time Complexity Analysis**:
  - Big O notation: Upper bound on time complexity
  - Big Ω notation: Lower bound on time complexity
  - Big Θ notation: Tight bound on time complexity
- **Comparison-Based Sorting Algorithms**:
  - **Comparison-based sorting**: Relies on comparing elements to sort
  - **Time complexity**: Measured in terms of comparisons and swaps
- **Recursive Algorithms**:
  - **Recursion**: Function calls itself to solve a problem
  - **Recursion depth**: Number of recursive calls

#### Important Formulas and Definitions

- **Big O Notation**:
  - O(1): Constant time complexity
  - O(log n): Logarithmic time complexity
  - O(n): Linear time complexity
  - O(n log n): Linearithmic time complexity
  - O(n^2): Quadratic time complexity
- **Time Complexity Theorem**:
  - The time complexity of an algorithm is determined by its input size (n)
- **Space Complexity**:
  - O(1): Constant space complexity
  - O(log n): Logarithmic space complexity
  - O(n): Linear space complexity

#### Theorems

- **Master Theorem**:
  - Helps analyze recursive algorithms
  - Used to solve recurrence relations

#### Example Problems

- **Find the time complexity of the following algorithms**:
  - Bubble sort
  - Merge sort
  - Quick sort

#### Key Takeaways

- Understand the concept of time complexity and its notation
- Learn to analyze the time complexity of recursive algorithms
- Familiarize yourself with common sorting algorithms and their time complexities
