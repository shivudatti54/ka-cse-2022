0. INTRODUCTION

Game theory is a mathematical framework for analyzing strategic interactions among rational decision-makers. It has applications in economics, political science, psychology, computer science, and biology.

1. BASIC DEFINITIONS

A game consists of:
- Players: Decision-makers in the game
- Strategies: Possible choices for each player
- Payoffs: Outcomes associated with strategy combinations

2. TYPES OF GAMES

2.1 Cooperative vs Non-cooperative
- Cooperative games: Players can form binding agreements
- Non-cooperative games: Players act independently

2.2 Zero-sum vs Non-zero-sum
- Zero-sum games: One player's gain equals another's loss
- Non-zero-sum games: Players can both gain or both lose

2.3 Simultaneous vs Sequential
- Simultaneous games: Players choose strategies at same time
- Sequential games: Players move in sequence

3. SOLUTION CONCEPTS

3.1 Nash Equilibrium
A strategy profile where no player can improve their payoff by unilaterally changing strategy.

3.2 Dominant Strategies
A strategy that yields the highest payoff regardless of opponents' choices.

3.3 Maximin Strategy
Choosing the strategy that maximizes the minimum possible payoff.

4. EXAMPLE: PRISONER'S DILEMMA

Two suspects are interrogated separately:
- If both confess: 5 years each
- If one confesses, other doesn't: confessor free, silent gets 10 years
- If both remain silent: 1 year each

The Nash equilibrium is both confessing, though mutual silence would be better.

5. APPLICATIONS IN COMPUTER SCIENCE

- Network routing
- Algorithm design
- Multi-agent systems
- Cybersecurity

6. KEY POINTS

- Game theory models strategic decision-making
- Nash equilibrium is a fundamental solution concept
- Different game types require different analytical approaches
- Has wide applications across disciplines

Game theory provides powerful tools for analyzing situations where outcomes depend on interactions between multiple decision-makers.
