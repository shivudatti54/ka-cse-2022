# Wumpus World Revisited Expert Systems: Representing and using domain knowledge

## **Definitions and Notations**

- **World**: The Wumpus World is a classic problem in Artificial Intelligence, consisting of a grid with a Wumpus (a deadly creature), a pit, and an exit.
- **State**: A state is a configuration of the World, represented as a subset of the grid.
- **Action**: An action is a move in the World, either up, down, left, or right.
- **Knowledge**: Domain knowledge is the expert's understanding of the World.

## **Representing Domain Knowledge**

- **Description-based representation**: Representing the World as a set of descriptive relations, such as "the Wumpus is located at (x, y)".
- **Description-based representation**: Using first-order logic to represent the World, with predicates such as "Wumpus(x, y)" and "Pit(x, y)".
- **Description-based representation**: Using a semantic network to represent the World, with nodes and edges representing concepts and relationships.

## **Using Domain Knowledge**

- **Rule-based expert system**: Using a set of rules to reason about the World, such as "if the Wumpus is in the same cell as the user, then the user is in danger".
- **Decision-theoretic frameworks**: Using decision-theoretic frameworks, such as Bayesian networks or influence diagrams, to represent the World and reason about possible actions.

## **Important Formulas and Theorems**

- **Bayes' theorem**: P(A|B) = P(A ∩ B) / P(B)
- **Influence diagram**: Representing the World as a directed graph, where each node represents a state and each edge represents a causal relationship.
- **Markov decision process**: A mathematical framework for representing and reasoning about decision-making under uncertainty.

## **Key Points**

- Domain knowledge is essential for expert systems, as it enables the system to reason about the World.
- Description-based representations are commonly used to represent the World.
- Rule-based expert systems and decision-theoretic frameworks are useful for using domain knowledge.
- Bayes' theorem, influence diagrams, and Markov decision processes are important formulas and theorems for reasoning about the World.
