# Learning Objectives

After studying this topic, you should be able to:

1. Differentiate between informed and uninformed search strategies
2. Define and explain the properties of heuristic functions (admissibility, consistency)
3. Implement and trace the execution of Greedy Best-First Search algorithm
4. Implement and trace the execution of A\* search algorithm
5. Compare and contrast various informed search algorithms based on completeness, optimality, time and space complexity
6. Design effective heuristic functions for practical problems
7. Explain the advantages of memory-bounded search variants like IDA\* and RBFS
