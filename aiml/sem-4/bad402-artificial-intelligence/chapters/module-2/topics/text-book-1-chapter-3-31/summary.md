# **Revision Notes: Textbook 1, Chapter 3-3.1**

## **Artificial Intelligence (AI)**

- **Definition:** The development of computer systems that can perform tasks that typically require human intelligence, such as learning, problem-solving, and decision-making.

## **Problem-Solving Agents**

- **Definition:** A computer program that can search, reason, and solve problems.
- **Types of agents:**
  - **Deterministic agents:** Make decisions based on fixed rules.
  - **Probabilistic agents:** Make decisions based on probabilities.

## **Example Problems**

- **Traveling Salesman Problem (TSP):** Find the shortest possible tour that visits a set of cities and returns to the original city.
- **Sudoku:** Solve a puzzle where numbers are filled into a grid according to certain rules.

## **Searching for Solutions**

- **Breadth-First Search (BFS):** Explore all nodes at a given depth before moving on to the next depth level.
- **Depth-First Search (DFS):** Explore as far as possible along each branch before backtracking.

## **Important Formulas and Definitions**

- **Greedy Algorithm:** A simple algorithm that makes the locally optimal choice at each step, hoping to find a global optimum solution.
- **Heuristic Function:** A function that estimates the distance between a given state and the goal state.

## **Theorems**

- **Travelling Salesman Problem:** NP-hard, meaning that the running time of algorithms for solving TSP increases exponentially with the size of the input.

## **Key Concepts**

- Problem-solving agents
- Searching algorithms (BFS, DFS)
- Greedy algorithm
- Heuristic functions
