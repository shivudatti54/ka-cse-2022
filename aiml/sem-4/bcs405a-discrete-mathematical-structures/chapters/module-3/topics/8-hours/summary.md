# (8 hours) Revision Notes: Relations and Functions

=============================================

### Definitions and Notations

- **Relation**: A subset of the Cartesian product of two sets A and B, denoted as R ⊆ A × B.
- **Function**: A special type of relation where each element in the domain maps to exactly one element in the codomain, denoted as f: A → B.
- **Domain** (A): The set of input values for a function.
- **Codomain** (B): The set of possible output values for a function.
- **Range** (f(A)): The set of actual output values for a function.

### Types of Relations

- **Reflexive**: If (a, a) ∈ R for all a ∈ A.
- **Symmetric**: If (a, b) ∈ R implies (b, a) ∈ R for all a, b ∈ A.
- **Transitive**: If (a, b) ∈ R and (b, c) ∈ R implies (a, c) ∈ R for all a, b, c ∈ A.
- **Equivalence Relation**: A relation that is reflexive, symmetric, and transitive.

### Types of Functions

- **Injective** (One-to-One): If f(x) = f(y) implies x = y for all x, y ∈ A.
- **Surjective** (Onto): If for every y ∈ B, there exists an x ∈ A such that f(x) = y.
- **Bijective** (One-to-One and Onto): If f is both injective and surjective.

### Important Formulas and Theorems

- **Horizontal Line Test**: A horizontal line intersects the graph of a function at most once, indicating that the function is injective.
- **Vertical Line Test**: A vertical line intersects the graph of a function at most once, indicating that the function is injective.
- **Inclusion-Exclusion Principle**: For two sets A and B, |A ∪ B| = |A| + |B| - |A ∩ B|.
- **Distribution Law**: For a relation R and a set A, (R ∘ f)(a) = {(a, b) | ∃c ∈ A, (c, b) ∈ R ∩ f(a)}.

### Key Concepts

- **Partial Order**: A relation that is reflexive, antisymmetric, and transitive.
- **Order Relation**: A partial order where the relation is irreflexive.
- **Cardinality**: The number of elements in a set.
