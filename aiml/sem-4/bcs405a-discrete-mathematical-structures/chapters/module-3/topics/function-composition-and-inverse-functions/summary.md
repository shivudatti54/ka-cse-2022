# Function Composition and Inverse Functions

### Definitions and Notations

- **Function Composition**: A composition of functions `f` and `g` is a function `h = f ∘ g` defined as `h(x) = f(g(x))`.
- **Inverse Function**: A function `f^(-1)` is the inverse of a function `f` if `f(f^(-1)(x)) = x` and `f^(-1)(f(x)) = x`.

### Key Formulas and Theorems

- **Function Composition Formula**: `h(x) = f(g(x))`
- **Inverse Function Formula**: `f^(-1)(x) = y ⇔ f(y) = x`
- **Inverse Function Theorem**: If `f` is a one-to-one function, then `f^(-1)` exists and is unique.

### Properties of Function Composition

- **Associativity**: `(f ∘ g) ∘ h = f ∘ (g ∘ h)`
- **Identity Function**: `f ∘ id = f` and `id ∘ f = f`
- **Inverse of a Composition**: `(f ∘ g)^(-1) = g^(-1) ∘ f^(-1)`

### Important Theorems

- **Inverse of a Product**: If `f` and `g` are one-to-one functions, then `(f ∘ g)^(-1) = g^(-1) ∘ f^(-1)`
- **Inverse of a Quotient**: If `f` and `g` are one-to-one functions, then `(f ∘ g)^(-1) = g^(-1) ∘ f^(-1)`
