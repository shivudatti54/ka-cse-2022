# **Functions – Plain and One-to-One**

## **Introduction**

In the study of discrete mathematical structures, functions play a crucial role in representing relationships between sets. In this topic, we will explore two types of functions: plain functions and one-to-one functions.

## **Plain Functions**

### Definition

A plain function, also known as a function, is a relation between two sets, A and B, such that every element in A is related to exactly one element in B. In other words, a function f: A → B is a relation that assigns to each element in A exactly one element in B.

### Notation

The notation f: A → B represents a function from set A to set B. The function f takes an element a from set A and returns an element b from set B.

### Examples

- A simple example of a plain function is f(x) = 2x, where f takes an element x from the set of integers and returns its double.
- Another example is f(A) = {f(a) | a ∈ A}, where f is a function that maps each element in set A to its square.

**Key Concepts:**

- Every element in the domain (set A) is related to exactly one element in the range (set B).
- Each element in the range is the image of exactly one element in the domain.
- The function is often denoted as f: A → B, where A is the domain and B is the range.

### Properties of Plain Functions

- **Injective**: A plain function is injective if it maps each element in the domain to a unique element in the range.
- **Surjective**: A plain function is surjective if every element in the range is the image of at least one element in the domain.
- **Bijection**: A plain function is a bijection if it is both injective and surjective.

## **One-to-One Functions**

### Definition

A one-to-one function, also known as an injective function, is a plain function that maps each element in the domain to a unique element in the range.

### Notation

The notation f: A → B represents a one-to-one function from set A to set B. The function f takes an element a from set A and returns an element b from set B, such that b is unique.

### Examples

- A simple example of a one-to-one function is f(x) = x^2, where f takes an element x from the set of integers and returns its square.
- Another example is f(A) = {f(a) | a ∈ A}, where f is a one-to-one function that maps each element in set A to its square.

**Key Concepts:**

- A one-to-one function maps each element in the domain to a unique element in the range.
- Every element in the range is the image of at most one element in the domain.
- The function is often denoted as f: A → B, where A is the domain and B is the range.

### Properties of One-to-One Functions

- **Injective**: A one-to-one function is injective by definition.
- **Unique**: A one-to-one function maps each element in the domain to a unique element in the range.

## **Conclusion**

In conclusion, plain functions and one-to-one functions are fundamental concepts in discrete mathematical structures. Understanding the definitions, properties, and examples of these functions is essential for solving problems and developing a deeper understanding of the subject.
