# Functions – Plain and One-to-One

### Definitions

- **Function**: A relation between a set of inputs (called the domain) and a set of possible outputs (called the range) where each input is associated with exactly one output.
- **Plain Function**: A function where every element in the domain is mapped to exactly one element in the range.
- **One-to-One Function**: A function where each element in the range is mapped to at most one element in the domain.

### Key Formulas and Theorems

- **Function Equation**: f(x) = y, where f is the function and x and y are elements of the domain and range respectively.
- **Inverse Function**: A function that undoes the action of another function. If f(x) = y, then f^(-1)(y) = x.
- **Domain and Range**: The set of all input values and output values respectively.

### Properties of Plain Functions

- **Injective (One-to-One)**: A function is injective if f(x) = f(y) implies x = y.
- **Surjective (Onto)**: A function is surjective if for every y in the range, there exists an x in the domain such that f(x) = y.
- **Bijective**: A function is bijective if it is both injective and surjective.

### Key Points

- A plain function is injective but not necessarily surjective.
- A one-to-one function is injective but not necessarily surjective.
- A bijective function is both injective and surjective.
- The function equation f(x) = y can be rewritten as y = f(x).

### Important Theorems

- **Theorem of Inverse Functions**: If f is a bijective function, then f^(-1) is also bijective.
- **Theorem of Injectivity and Surjectivity**: If f is a function that is both injective and surjective, then it is bijective.
