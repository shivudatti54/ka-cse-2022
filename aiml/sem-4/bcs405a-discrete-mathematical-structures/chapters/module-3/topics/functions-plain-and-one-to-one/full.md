### Introduction

In discrete mathematical structures, functions play a crucial role in defining relationships between sets of elements. A function is a relation between a set of inputs, called the domain, and a set of possible outputs, called the range. In this section, we will delve into the concept of plain functions and one-to-one functions, exploring their definitions, properties, and applications.

### Historical Context

The concept of functions dates back to ancient civilizations, with contributions from mathematicians such as Euclid and Archimedes. However, the modern concept of functions as we know it today was developed in the 19th century by mathematicians such as Augustin-Louis Cauchy and Karl Weierstrass.

The term "function" was first used by the German mathematician Carl Friedrich Gauss in the early 19th century. Gauss used the term to describe a mathematical relation between variables, and it was later adopted by other mathematicians to describe the concept of functions.

### Plain Functions

A plain function, also known as a single-valued function, is a relation between a set of inputs and a set of possible outputs that assigns each input to exactly one output.

**Definition:** A function `f` from a set `A` to a set `B` is a relation `R` such that:

1. For every `a` in `A`, there exists exactly one `b` in `B` such that `(a, b)` is in `R`.
2. For every `b` in `B`, there exists at most one `a` in `A` such that `(a, b)` is in `R`.

**Example:** Define a function `f` from the set of integers `Z` to the set of real numbers `R` by `f(x) = x^2`.

| x   | f(x) |
| --- | ---- |
| 1   | 1    |
| 2   | 4    |
| 3   | 9    |

In this example, each input `x` in `Z` is assigned to exactly one output `f(x)` in `R`.

### One-to-One Functions

A one-to-one function, also known as an injective function, is a relation between two sets that assigns each element in the first set to exactly one element in the second set, with no two elements in the first set assigned to the same element in the second set.

**Definition:** A function `f` from a set `A` to a set `B` is one-to-one if:

1. For every `a` in `A`, there exists exactly one `b` in `B` such that `(a, b)` is in `f`.
2. For every `b` in `B`, there exists at most one `a` in `A` such that `(a, b)` is in `f`.

**Example:** Define a function `f` from the set `A = {1, 2, 3}` to the set `B = {a, b, c}` by `f(1) = a`, `f(2) = b`, and `f(3) = c`.

| x   | f(x) |
| --- | ---- |
| 1   | a    |
| 2   | b    |
| 3   | c    |

In this example, each element in `A` is assigned to exactly one element in `B`, with no two elements in `A` assigned to the same element in `B`.

### Properties of Functions

Functions have several important properties, including:

- **Surjectivity:** A function `f` from a set `A` to a set `B` is surjective if every element in `B` is assigned to at least one element in `A`.
- **Injectivity:** A function `f` from a set `A` to a set `B` is injective if every element in `B` is assigned to at most one element in `A`.
- **Bijectivity:** A function `f` from a set `A` to a set `B` is bijective if it is both surjective and injective.

### Examples and Case Studies

1. **Linear Functions:** Define a function `f` from the set of integers `Z` to the set of real numbers `R` by `f(x) = 2x + 1`.

| x   | f(x) |
| --- | ---- |
| 1   | 3    |
| 2   | 5    |
| 3   | 7    |

In this example, the function `f` is both injective and surjective.

2. **Quadratic Functions:** Define a function `f` from the set of integers `Z` to the set of real numbers `R` by `f(x) = x^2`.

| x   | f(x) |
| --- | ---- |
| 1   | 1    |
| 2   | 4    |
| 3   | 9    |

In this example, the function `f` is not injective, as two different inputs can have the same output.

### Applications

Functions have numerous applications in various fields, including:

- **Computer Science:** Functions are used in programming languages to define algorithms and data structures.
- **Economics:** Functions are used to model economic systems and make predictions about economic behavior.
- **Biology:** Functions are used to model population dynamics and make predictions about the spread of diseases.

### Modern Developments

In recent years, there has been significant development in the field of functions, including:

- **Category Theory:** A branch of mathematics that studies the commonalities between different types of functions and relations.
- **Homotopy Theory:** A branch of mathematics that studies the properties of functions and relations that are preserved under continuous deformations.
