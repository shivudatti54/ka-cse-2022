# Quantifiers in Discrete Mathematical Structures

### **Definitions**

- **Quantifier**: A symbol used to express the universality or existence of a property.
- **For All (∀)**: Expresses that a statement is true for all elements in a set.
- **There Exists (∃)**: Expresses that a statement is true for at least one element in a set.

### **Formulas and Theorems**

- **Universal Quantifier (∀)**:
  - ∀x ∈ S, P(x) (For all x in S, P(x) is true)
- **Existential Quantifier (∃)**:
  - ∃x ∈ S, P(x) (There exists x in S, P(x) is true)
- **Quantifier Rules**:
  - De Morgan's Law: ¬(∀x, P(x)) ≡ ∃x, ¬P(x) and ¬(∃x, P(x)) ≡ ∀x, ¬P(x)
  - Distributive Law: ∀x, (P(x) ∧ Q(x)) ≡ (∀x, P(x)) ∧ (∀x, Q(x)) and ∃x, (P(x) ∧ Q(x)) ≡ (∃x, P(x)) ∧ (∃x, Q(x))

### **Important Theorems**

- **Generalization Rule**: If P(x) is true for all values of x in a set, then P is true for all values of x.
- **Existential Instantiation Rule**: If P(x) is true for at least one value of x in a set, then we can infer P.

### **Key Points**

- Understand the meaning and usage of ∀ and ∃.
- Familiarize with De Morgan's Law and Distributive Law.
- Apply Generalization Rule and Existential Instantiation Rule in proof construction.
- Practice solving problems involving quantifiers.
