# **Definitions and the Proofs of Theorems**

## **Introduction**

In discrete mathematical structures, definitions and proofs of theorems play a crucial role in establishing the foundations of mathematics. A definition is a statement that provides a clear and concise description of a mathematical concept, while a proof is a step-by-step argument that demonstrates the validity of a statement.

## **What is a Definition?**

A definition is a statement that provides a clear and concise description of a mathematical concept. It is a way to define a set, a function, a relation, or any other mathematical object.

## **Types of Definitions**

### 1. **Explicit Definition**

An explicit definition is a statement that explicitly defines a mathematical concept.

- Example: The set of all even integers is defined as: {x | x is an integer and x is divisible by 2}

### 2. **Implicit Definition**

An implicit definition is a statement that implies a mathematical concept without explicitly defining it.

- Example: A function f is defined implicitly as: f(x) = x^2

### 3. **Recursive Definition**

A recursive definition is a statement that defines a mathematical concept in terms of itself.

- Example: The Fibonacci sequence is defined recursively as: F(n) = F(n-1) + F(n-2), with initial values F(0) = 0 and F(1) = 1

## **What is a Proof?**

A proof is a step-by-step argument that demonstrates the validity of a statement. It is a way to establish the truth of a mathematical statement using logical reasoning and mathematical principles.

## **Types of Proofs**

### 1. **Direct Proof**

A direct proof is a step-by-step argument that demonstrates the validity of a statement without assuming any other statements.

- Example: Prove that the sum of two even integers is even. Let x and y be two even integers. Then x = 2a and y = 2b for some integers a and b. By the definition of even integers, x + y = 2(a + b). Since a + b is an integer, x + y is even.

### 2. **Indirect Proof**

An indirect proof is a step-by-step argument that demonstrates the invalidity of a statement.

- Example: Prove that there is no integer that is divisible by both 3 and 5. Assume, if possible, that there is an integer x that is divisible by both 3 and 5. Then x = 3k and x = 5m for some integers k and m. Since x is divisible by both 3 and 5, x is divisible by their least common multiple, which is 15. But this contradicts the assumption that x is an integer.

### 3. **Proof by Contradiction**

A proof by contradiction is a step-by-step argument that demonstrates the invalidity of a statement by assuming its negation.

- Example: Prove that the sum of two odd integers is even. Assume, if possible, that the sum of two odd integers is odd. Let x and y be two odd integers. Then x = 2a + 1 and y = 2b + 1 for some integers a and b. By the definition of odd integers, x + y = 2(a + b) + 2. Since a + b is an integer, x + y is even, which contradicts the assumption that x + y is odd.

## **Key Concepts**

- **Axioms**: A set of statements that are assumed to be true without proof.
- **Theorems**: Statements that are proven to be true using logical reasoning and mathematical principles.
- **Proofs**: Step-by-step arguments that demonstrate the validity of a statement.
- **Definitions**: Statements that provide a clear and concise description of a mathematical concept.
- **Recursive definitions**: Definitions that define a mathematical concept in terms of itself.
