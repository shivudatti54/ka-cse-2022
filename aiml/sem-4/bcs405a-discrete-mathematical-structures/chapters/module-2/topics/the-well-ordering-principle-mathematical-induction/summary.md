# **The Well Ordering Principle – Mathematical Induction**

## **Definitions and Formulas**

- **Well Ordering Principle (WOP):** A set of positive integers has a least element.
- **Well-Ordering Principle for Naturals:** Every non-empty set of natural numbers contains a least element.
- **Proof by Mathematical Induction:**
  - **Base Case:** Prove the statement is true for the smallest possible value.
  - **Inductive Step:** Assume the statement is true for k and prove it is true for k+1.

## **Theorems**

- **Well-Ordering Principle Theorem:** A set of positive integers has a least element if and only if it is bounded above.
- **Principle of Mathematical Induction:** If a property P(n) is true for n = 1 and if P(n) implies P(n+1), then P(n) is true for all positive integers n.

## **Key Points**

- **Well Ordering Principle:** Every non-empty set of positive integers has a least element.
- **Mathematical Induction:**
  - Base case: Prove the statement is true for the smallest value.
  - Inductive step: Assume the statement is true for k and prove it is true for k+1.
- **Applications:** Proof by mathematical induction can be used to prove statements about the behavior of sequences and series.
