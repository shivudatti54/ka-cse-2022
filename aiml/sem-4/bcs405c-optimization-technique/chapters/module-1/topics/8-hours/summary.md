# **Revision Notes: (8 hours) - Optimization Technique**

## **Module: Vector Calculus**

### Topics Covered

- **Definition of Optimization**
  - Finding a maximum or minimum value of a function
- **Types of Optimization**
  - Local and global optimization
  - Unconstrained and constrained optimization
- **Optimization Techniques**
  - Gradient Descent
  - Newton's Method
  - Lagrange Multipliers
  - Cauchy-Schwarz Inequality
- **Important Formulas and Theorems**
  - First and Second Derivatives
  - Hessian Matrix
  - Gradient Vector
  - Euler-Lagrange Equation

### Key Points

- Optimization techniques are used to find the maximum or minimum value of a function.
- Gradient Descent and Newton's Method are popular optimization algorithms.
- Lagrange Multipliers are used to solve constrained optimization problems.
- Cauchy-Schwarz Inequality is a useful tool for optimization problems.
- First and Second Derivatives are essential for understanding optimization techniques.
- Hessian Matrix and Gradient Vector are crucial for optimization calculations.

### Important Formulas

- **First Derivative**: `f'(x) = lim(h → 0) [f(x + h) - f(x)]/h`
- **Second Derivative**: `f''(x) = lim(h → 0) [f'(x + h) - f'(x)]/h`
- **Hessian Matrix**: `H = [[f''(x)], [f''(x)]]`
- **Gradient Vector**: `∇f(x) = [f'(x), f'(x)]`

### Important Theorems

- **Lagrange Multiplier Theorem**: `∇f(x) = λ∇g(x)`
- **Cauchy-Schwarz Inequality**: `[a, b]∇f(x)⋅∇g(x) ≤ ||a∇f(x)|| ||g(x)||`
