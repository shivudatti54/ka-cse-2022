# **8 Hours Optimization Technique Revision Notes**

## **Definition**

- Optimization technique used to find the maximum or minimum value of a function subject to certain constraints

## **Key Concepts**

- **8-Hour Rule**: A problem has a solution that can be found in 8 hours or less if:
  - It is a first-order calculus problem
  - It involves only one variable
  - It uses basic calculus rules (e.g., derivatives, integrals)
- **8-Hour Timeframe**: Problems that can be solved within 8 hours are typically easier and more straightforward

## **Important Formulas and Theorems**

- **First Derivative Test**: Used to determine the behavior of a function at a critical point
- **Second Derivative Test**: Used to determine the nature of a critical point
- **Optimization Problems**: Maximize or minimize a function subject to constraints (e.g., equality or inequality constraints)

## **Definitions**

- **Constraint**: A condition that must be satisfied
- **Optimal Solution**: The solution that maximizes or minimizes a function subject to constraints

## **Key Takeaways**

- The 8-Hour Rule is a guideline for identifying problems that can be solved using basic calculus rules
- Optimization problems involve finding the maximum or minimum value of a function subject to constraints
- Understanding first and second derivative tests is crucial for solving optimization problems
