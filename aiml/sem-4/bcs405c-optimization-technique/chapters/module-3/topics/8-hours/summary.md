# **Revision Notes: Optimization Technique - (8 hours)**

### Introduction

- Optimization techniques are used to find the best solution among a set of possible solutions, minimizing or maximizing a given objective function.
- This topic focuses on a specific optimization technique, with 8 hours of study material.

### Key Concepts

- **Linearity**: A function is linear if it can be written in the form `f(x) = ax + b`, where `a` and `b` are constants.
- **Convexity**: A function is convex if its graph is a convex set, meaning that the line segment connecting any two points on the graph lies entirely below the graph.
- **Convex Optimization**: A technique for finding the optimal solution to a convex optimization problem, where the objective function is convex and the constraints are convex.

### Important Formulas and Definitions

- **Optimization Problem**: `minimize or maximize f(x) subject to constraints g(x) = 0, h(x) = 0, ...`
- **Convex Optimization Problem**: `minimize or maximize f(x) subject to constraints g(x) = 0, h(x) = 0, ..., where f(x) and g(x) are convex`
- **KKT Conditions**: Necessary conditions for a point `x` to be a local minimum or maximum, where `K` is the Hessian matrix of the Lagrangian function.

### Important Theorems

- **Weierstrass Theorem**: A continuous function on a compact set attains its minimum and maximum values.
- **Karush-Kuhn-Tucker (KKT) Theorem**: A necessary and sufficient condition for a point `x` to be a local minimum or maximum, where `K` is the Hessian matrix of the Lagrangian function.

### Important Algorithms

- **Gradient Descent**: An iterative algorithm for minimizing a convex function.
- **Newton's Method**: An iterative algorithm for finding the root of a convex function.

### Important Results

- **Strong Duality Theorem**: If an optimization problem has a finite optimal solution, then the dual problem also has a finite optimal solution.
- **Dual Problem**: A problem obtained by reversing the roles of the primal problem and the Lagrange multipliers.
