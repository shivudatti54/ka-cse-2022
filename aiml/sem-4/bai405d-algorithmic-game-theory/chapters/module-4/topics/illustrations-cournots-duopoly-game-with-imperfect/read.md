# **Illustrations: Cournot’s Duopoly Game with Imperfect Information**

## **Introduction**

In this topic, we will explore the classic Cournot's duopoly game with imperfect information. This game is a fundamental model in algorithmic game theory, illustrating how players with imperfect knowledge can make decisions that affect the outcome of the game.

## **Definition of Cournot's Duopoly Game**

Cournot's duopoly game is a two-player game where each player produces a good that is a continuous variable. The game is characterized by two key elements:

- **Imperfect information**: Neither player has complete knowledge of the other's production level.
- **Strategic interdependence**: The output of one player affects the demand for the good, which in turn affects the outcome for both players.

## **Key Concepts**

- **Continuous variable**: The output of each player is a continuous variable, meaning it can take on any value within a certain range.
- **Demand function**: The demand for the good is a function of the total output of both players.
- **Production cost**: Each player incurs a production cost for producing the good.

## **Gameplay**

The game consists of two players, 1 and 2. Each player makes a decision about how much to produce, denoted by x1 and x2, respectively. The objective is to maximize the profit for each player, given the production cost and the demand function.

## **Demand Function**

The demand function is typically represented as a downward-sloping curve, indicating that an increase in total output leads to a decrease in demand. The demand function can be written as:

Q = a - b(x1 + x2)

where Q is the total output, a and b are constants, and x1 and x2 are the production levels of players 1 and 2, respectively.

## **Production Cost**

Each player incurs a production cost for producing the good. The production cost function can be written as:

C(x) = cx

where C(x) is the production cost and c is a constant.

## **Profit Function**

The profit function for each player is the difference between the revenue and the production cost. The revenue is the product of the price and the quantity sold. The profit function can be written as:

π = (p)(a - b(x1 + x2)) - cx

where π is the profit, p is the price, and x is the production level.

## **Equilibrium**

To find the equilibrium, we need to solve the following equations:

∂π/∂x1 = 0
∂π/∂x2 = 0

These equations represent the marginal revenue and marginal cost of each player. Solving these equations, we get:

x1 = x2 = a/2b

This is the Nash equilibrium, where each player produces half of the total output.

## **Imperfect Information**

In the Cournot's duopoly game with imperfect information, neither player has complete knowledge of the other's production level. This leads to a subgame perfect equilibrium, where each player anticipates the other player's action and adjusts their strategy accordingly.

## **Strategic Interdependence**

The output of one player affects the demand for the good, which in turn affects the outcome for both players. This strategic interdependence leads to a more complex game, where each player needs to consider the potential actions of the other player.

## **Example**

Suppose we have two players, Alice and Bob. Alice produces a good that is in high demand, while Bob produces a good that is complementary to Alice's good. The demand function is given by:

Q = 100 - 0.5(x1 + x2)

where Q is the total output, x1 is Alice's production level, and x2 is Bob's production level.

The production cost function for Alice is:

C(x) = 2x

The production cost function for Bob is:

C(x) = 3x

The profit function for Alice is:

π = (20)(100 - 0.5(x1 + x2)) - 2x

The profit function for Bob is:

π = (15)(100 - 0.5(x1 + x2)) - 3x

To find the equilibrium, we need to solve the following equations:

∂π/∂x1 = 0
∂π/∂x2 = 0

Solving these equations, we get:

x1 = x2 = 50

This is the Nash equilibrium, where Alice and Bob produce 50 units of the good each.

Cournot's duopoly game with imperfect information is a classic model in algorithmic game theory.
