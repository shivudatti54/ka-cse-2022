# Equilibrium in a Single Population

=====================================

## Definitions and Theorems

- **Nash Equilibrium**: A state where no player can improve their payoff by unilaterally changing their strategy, assuming all other players keep their strategies unchanged.
- **Mixed Strategy**: A probability distribution over a set of pure strategies, used to randomize one's actions.
- **Payoff Matrix**: A table showing the payoffs for each possible combination of strategies.

## Key Formulas and Concepts

- **Nash Equilibrium Formula**: `NE = {s | ∑∑[u_i(s) - u_i(s')][1 - P_j(s' | s)] = 0 for all j]`
- **Mixed Strategy Nash Equilibrium**: `NE = {s | ∑∑[u_i(s) - u_i(s')][P_j(s' | s)] = 0 for all j]`
- **Expected Payoff**: `E[u] = ∑[u(s) \* P(s)]`

## Important Theorems

- **Nash's Theorem**: If a finite game has a mixed strategy Nash equilibrium, then it has a mixed strategy Nash equilibrium.
- **Parieto's Theorem**: If a finite game has a mixed strategy Nash equilibrium, then every player must use a mixed strategy.

## Key Points

- Equilibrium in a single population refers to a state where no player can improve their payoff by changing their strategy.
- Mixed strategies are used to randomize one's actions and achieve equilibrium.
- The Nash Equilibrium Formula and Mixed Strategy Nash Equilibrium Formula provide conditions for achieving equilibrium.
- Key concepts include expected payoff, payoff matrices, and theorems such as Nash's Theorem and Parieto's Theorem.
