# **Finitely and Infinitely Repeated Prisoner’s Dilemma**

## **Definition**

- The Prisoner's dilemma is a classic game theory problem in which two individuals (prisoners) must decide whether to cooperate or defect.
- Finitely repeated Prisoner's dilemma refers to a repeated game where the same game is played multiple times, with the possibility of a finite number of periods.
- Infinitely repeated Prisoner's dilemma refers to a repeated game where the same game is played an infinite number of times.

## **Key Points**

### Finitely Repeated Prisoner's Dilemma

- **Payoff matrix**
  - Coop: (3,3), Defect: (5,5)
- **Nash Equilibrium**
  - No unique equilibrium in finitely repeated game
- **Revised solution**
  - Cooperate in the last period, defect in all other periods

### Infinitely Repeated Prisoner's Dilemma

- **Payoff matrix**
  - Coop: (3,3), Defect: (5,5)
- **Nash Equilibrium**
  - **Pareto Optimality**: (1,1) - both prisoners cooperate indefinitely
  - **Time-consistent behavior**: both prisoners cooperate in every period

## **Important Formulas**

- **Prisoner's dilemma payoff matrix**
  - A (Coop) | B (Coop) | A (Defect) | B (Defect)
  - ***
  - 3 | (3,3) | 5 | 5
  - 5 | 5 | 3 | (3,3)
- **Nash Equilibrium**
  - (Defect, Defect) - both prisoners defect

## **Theorems**

- **Prisoner's dilemma theorem**
  - In finitely repeated Prisoner's dilemma, the Nash Equilibrium is not unique.
- **Reynolds' theorem**
  - In infinitely repeated Prisoner's dilemma, the Nash Equilibrium is (1,1) - both prisoners cooperate indefinitely.
