### Introduction

Repeated games are a type of game where two or more players make decisions over multiple stages, with the goal of maximizing their payoff.

### Definition of Repeated Games

- A repeated game is a sequence of stages, where each stage is a game.
- The players make decisions at each stage, with the goal of maximizing their payoff.

### Key Concepts

- **Strategic Subgame Perfect Equilibrium (SSPE)**: a Nash equilibrium that is also a subgame perfect equilibrium, meaning that each player's strategy is optimal given the information available to them at each stage.
- **Repeating Game**: a game where the same game is played multiple times, with the same players and the same payoffs.
- **Discount Factor**: a parameter that determines how much a player values future payoffs relative to current payoffs.

### Theorem of Perfect Equilibrium

- Every repeated game has a strategic subgame perfect equilibrium.

### Nash Equilibrium

- A Nash equilibrium is a state where no player can improve their payoff by unilaterally changing their strategy, given the strategies of the other players.

### Key Formulas and Definitions

- **Payoff Matrix**: a table that shows the payoffs for each player at each stage of the game.
- **Strategy Space**: the set of all possible strategies that a player can use.
- **Action Space**: the set of all possible actions that a player can take.
- **Game Theory**: the study of how people make decisions when they are interacting with each other.
