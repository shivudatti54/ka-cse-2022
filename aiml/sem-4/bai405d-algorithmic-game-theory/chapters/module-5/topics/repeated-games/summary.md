# Repeated Games Revision Notes

### Definitions

- **Repeated Game**: A game where the same sequence of actions is repeated multiple times.
- **Strategy**: A plan of action for a player in a game.
- **Payoff Matrix**: A table showing the payoffs for each possible combination of actions.

### Key Concepts

- **N- Person Game**: A game with multiple players.
- **Stationary Game**: A repeated game where players choose actions at the beginning of each period.
- **Dynamic Game**: A repeated game where players adapt their actions over time.
- **Subgame Perfect Equilibrium**: A Nash equilibrium that is also a stationary game equilibrium.

### Important Formulas and Theorems

- **Pareto Optimality**: A state where no player can improve their payoff without making another player worse off.
- **Nash Equilibrium**: A state where no player can improve their payoff by unilaterally changing their action.
- **Ponzi's Theorem**: A state where a player can replicate an action that maximizes their payoff in a subgame perfect equilibrium.
- **Folk Theorem**: A state where a Nash equilibrium can be replicated in a dynamic game.

### Repeated Games Theorems

- **Revisiting Theorem**: A state where a player can improve their payoff by revisiting an action that was previously not optimal.
- **Ponzi's Revision Theorem**: A state where a player can improve their payoff by revisiting an action that was previously not optimal, but only if the opponent's actions do not change.

### Important Theorems

- **Prisoner's Dilemma**: A two-player game where one player can improve their payoff by betraying the other player.
- **Tragedy of the Commons**: A game where multiple players can improve their payoff by overusing a shared resource.

### Key Takeaways

- Repeated games can be solved using stationary game theory.
- Dynamic games can be solved using dynamic game theory.
- Nash equilibrium is a state where no player can improve their payoff by unilaterally changing their action.
