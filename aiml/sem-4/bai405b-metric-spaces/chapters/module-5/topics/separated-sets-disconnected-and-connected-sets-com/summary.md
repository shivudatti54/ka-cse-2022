# **Separated Sets, Disconnected and Connected Sets, Components, Connected Subsets of R, Continuous Functions on Connected Sets**

## **Definitions and Formulas**

- **Separated Set**: A set $A$ in a metric space $X$ is separated if it is not empty and there exist points $x, y \in A$ such that $d(x, y) > 0$.
- **Disconnected Set**: A set $A$ is disconnected if it can be written as $A = A_1 \cup A_2$, where $A_1$ and $A_2$ are non-empty disjoint open sets.
- **Connected Set**: A set $A$ is connected if it cannot be written as $A = A_1 \cup A_2$, where $A_1$ and $A_2$ are non-empty disjoint open sets.
- **Component**: The component of a set $A$ is the largest connected subset of $A$.
- **Connected Subset**: A subset $A$ of a connected set $B$ is connected if it cannot be written as $A = A_1 \cup A_2$, where $A_1$ and $A_2$ are non-empty disjoint open sets.

## **Theorems**

- **Heine-Borel Theorem**: A subset $A$ of $\mathbb{R}$ is compact if and only if it is closed and bounded.
- **Connected Components Theorem**: Every connected subset of $\mathbb{R}$ has a connected component.

## **Properties of Connected Sets**

- **Connected Sets are Open**: If $A$ is connected, then it is open.
- **Connected Sets are Compact**: If $A$ is connected and bounded, then it is compact.

## **Continuous Functions on Connected Sets**

- **Continuous Functions Preserve Connectedness**: If $f: X \to Y$ is continuous and $X$ is connected, then $f(X)$ is connected.
- **Continuous Functions on Compact Sets are Bounded**: If $f: X \to \mathbb{R}$ is continuous and $X$ is compact, then $f(X)$ is bounded.

## **Important Results**

- **Separation Axiom**: A space is disconnected if and only if it can be represented as a union of two disjoint non-empty closed sets.
- **Connectedness Implies Separability**: A space is separable if and only if it is connected.
