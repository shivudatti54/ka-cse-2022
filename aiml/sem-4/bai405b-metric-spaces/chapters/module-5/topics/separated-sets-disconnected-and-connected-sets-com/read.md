# **Separated Sets, Disconnected and Connected Sets, Components, Connected Subsets of R, Continuous Functions on Connected Sets**

## **Introduction**

In the study of metric spaces, connectedness is a fundamental concept that plays a crucial role in understanding the properties of spaces. This topic covers the key concepts of separated sets, disconnected and connected sets, components, connected subsets of R, and continuous functions on connected sets.

## **Separated Sets**

A set of points in a metric space is said to be separated if it is not equal to the entire space and is not empty.

- Definition: A set A is separated if ∅ ≠ A ≠ X, where X is the metric space.
- Example: Let X = R with the standard metric. The set A = {0, 1} is separated since ∅ ≠ A ≠ R.

## **Disconnected Sets**

A set is disconnected if it can be written as the union of two non-empty, disjoint sets.

- Definition: A set A is disconnected if A = A1 ∪ A2, where A1 and A2 are non-empty, disjoint sets, and A1 ∩ A2 = ∅.
- Example: Let X = R with the standard metric. The set A = (0, 1) ∪ (1, 2) is disconnected since A = (0, 1) ∪ (1, 2) = (0, 2) and (0, 1) ∩ (1, 2) = ∅.

## **Connected Sets**

A set is connected if it cannot be written as the union of two non-empty, disjoint sets.

- Definition: A set A is connected if A is not disconnected.
- Example: Let X = R with the standard metric. The set A = (0, 1) is connected since A = (0, 1) and (0, 1) ∩ (0, 1) = (0, 1).

## **Components**

A component of a connected set is a maximal connected subset.

- Definition: A component of a connected set A is a maximal connected subset of A.
- Example: Let X = R with the standard metric. The set A = (0, 1) is connected, and its component is A = (0, 1).

## **Connected Subsets of R**

A connected subset of R is a subset that is connected in the usual Euclidean metric.

- Definition: A subset A of R is connected if A is connected in the usual Euclidean metric.
- Example: Let A = (0, 1) be a connected subset of R.

## **Continuous Functions on Connected Sets**

A continuous function on a connected set is a function that is continuous at every point in the set.

- Definition: A function f: X → Y is continuous on a connected set A if f is continuous at every point x in A.
- Example: Let f: R → R be a function defined by f(x) = x^2. The function f is continuous on R, which is connected.

## **Tips and Tricks**

- To determine if a set is connected or disconnected, try to write it as the union of two non-empty, disjoint sets.
- To determine if a function is continuous on a connected set, check if the function is continuous at every point in the set.
- To find components of a connected set, look for maximal connected subsets.
