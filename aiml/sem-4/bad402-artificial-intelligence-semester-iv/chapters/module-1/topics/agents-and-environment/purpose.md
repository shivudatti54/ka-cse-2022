# Learning Objectives

After studying this topic, you should be able to:

1.  Define an intelligent agent and explain its core components (sensors, actuators, percepts, actions).
2.  Apply the PEAS framework to describe the task environment for a given agent.
3.  Explain the concept of rationality and how it differs from perfection or omniscience.
4.  Classify a given task environment based on its key properties (e.g., observable, deterministic, episodic, etc.).
5.  Compare and contrast the four main agent architectures (Simple Reflex, Model-Based, Goal-Based, Utility-Based), identifying the environments for which each is suited.
6.  Describe the components of a learning agent (performance element, learning element, critic, problem generator) and their functions.
