# Learning Objectives

After studying this topic, you should be able to:

1. Define and distinguish between procedural programming and object-oriented programming paradigms
2. Explain the fundamental characteristics and principles of procedural programming
3. Describe the four pillars of OOP: Encapsulation, Inheritance, Polymorphism, and Abstraction
4. Compare procedural and OOP approaches with respect to code organization, data security, and scalability
5. Write C++ programs demonstrating both procedural and OOP approaches
6. Implement basic classes with constructors, destructors, and member functions
7. Demonstrate inheritance by creating derived classes from base classes
8. Apply the concept of polymorphism using virtual functions and function overriding
