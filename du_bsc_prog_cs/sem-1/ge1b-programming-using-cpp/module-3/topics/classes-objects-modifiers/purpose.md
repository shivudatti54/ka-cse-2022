## Purpose and Learning Objectives  

In modern software development, the ability to model real-world entities using classes and objects is fundamental to creating modular, maintainable, and scalable C++ applications. Understanding access modifiers ensures proper encapsulation, protecting data integrity and facilitating team‑based development.

- **Define** the concepts of classes, objects, and their relationship in C++.  
- **Implement** constructors, destructors, and member functions to initialize and manipulate object state.  
- **Apply** `public`, `private`, and `protected` access specifiers to enforce encapsulation and data hiding.  
- **Differentiate** between object lifetimes, storage durations, and the effects of modifiers.  
- **Create** and use objects dynamically and statically, demonstrating proper memory management.  
- **Design** reusable and well‑structured classes that adhere to OOP principles such as abstraction and modularity.
