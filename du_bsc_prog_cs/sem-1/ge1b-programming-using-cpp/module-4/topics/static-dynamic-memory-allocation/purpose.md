# Learning Objectives

After studying this topic, you should be able to:

1. Explain the different memory segments in a C++ program (stack, heap, data segment, code segment)

2. Distinguish between static and dynamic memory allocation with respect to timing, storage location, and use cases

3. Write C++ code using new, delete, new[], and delete[] operators for dynamic memory management

4. Analyze and identify common memory management problems including memory leaks and dangling pointers

5. Implement dynamic arrays and matrices using pointers in C++

6. Apply best practices for memory management including setting pointers to nullptr after deletion

7. Evaluate when to use static versus dynamic allocation based on problem requirements

8. Understand the advantages of smart pointers (unique_ptr, shared_ptr) for automatic memory management in modern C++
