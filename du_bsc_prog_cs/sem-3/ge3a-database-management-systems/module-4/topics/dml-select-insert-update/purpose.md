# Learning Objectives

After studying this topic, you should be able to:

1. Explain the purpose and importance of DML (Data Manipulation Language) operations in database management systems

2. Write SELECT queries with various clauses including WHERE, ORDER BY, GROUP BY, HAVING, and DISTINCT to retrieve specific data from tables

3. Use comparison operators (=, <>, <, >, <=, >=) and logical operators (AND, OR, NOT) effectively in WHERE conditions

4. Apply pattern matching with LIKE operator and wildcards (% and _) for string searches

5. Implement INSERT statement to add single and multiple rows into database tables

6. Construct INSERT...SELECT statements to transfer data between tables

7. Write UPDATE statements to modify existing data with appropriate WHERE conditions

8. Understand transaction control commands (COMMIT, ROLLBACK, SAVEPOINT) and their role in DML operations

9. Identify and avoid common mistakes such as forgetting WHERE clause in UPDATE operations

10. Apply subqueries within DML statements for complex data manipulation requirements
