### Purpose
Recursive functions are a cornerstone of algorithm design, enabling elegant solutions to problems that naturally split into sub‑problems (e.g., sorting, searching, tree traversals). Mastering linear and binary recursion equips students with the analytical and implementation skills needed for efficient, scalable software development in real‑world CS applications.

### Learning Objectives
- **Explain** the fundamental principles of recursion and differentiate between linear and binary recursion patterns.  
- **Implement** recursive solutions for classic problems such as factorial, Fibonacci numbers, and binary search.  
- **Analyze** the time and space complexity of recursive algorithms using recurrence relations.  
- **Compare** iterative and recursive approaches in terms of performance, memory usage, and code readability.  
- **Design** recursive algorithms for divide‑and‑conquer strategies (e.g., merge sort, quicksort).  
- **Evaluate** the correctness of recursive functions by verifying base cases and applying inductive reasoning.  
- **Apply** recursion to traverse and manipulate binary tree structures effectively.
