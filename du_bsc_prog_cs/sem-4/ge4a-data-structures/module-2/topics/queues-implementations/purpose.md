# Learning Objectives

After studying this topic, you should be able to:

1. Define and explain the queue data structure along with its fundamental FIFO (First-In-First-Out) operational principle.

2. Identify and describe the basic operations performed on a queue: enqueue, dequeue, front, rear, isEmpty, and isFull.

3. Implement a queue using array-based (linear) approach, including initialization, insertion, and deletion operations with proper overflow/underflow handling.

4. Implement a circular queue that overcomes the space limitation of linear queues by wrapping around to the beginning of the array.

5. Implement a queue using linked list representation, demonstrating dynamic memory allocation and pointer manipulation.

6. Compare and contrast linear queue, circular queue, and linked list-based queue implementations in terms of time complexity, space efficiency, and practical applicability.

7. Explain the concept of priority queues and identify their real-world applications in scheduling and optimization algorithms.

8. Apply queue data structures to solve practical problems such as breadth-first search (BFS) in graphs and level-order tree traversal.
