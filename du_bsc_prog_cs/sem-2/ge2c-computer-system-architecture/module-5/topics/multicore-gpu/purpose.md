# Learning Objectives

After studying this topic, you should be able to:

1. Explain the architectural differences between multicore CPUs and GPUs, including their design philosophies and target workloads.

2. Analyze the advantages and limitations of multicore processors in terms of parallel computing performance.

3. Describe the GPU memory hierarchy and explain how thread organization (grids, blocks, warps) enables efficient parallel execution.

4. Apply Amdahl's Law and Gustafson's Law to calculate theoretical speedups for parallel programs.

5. Compare SIMD, SIMT, and MIMD execution models and identify their suitable applications.

6. Design basic parallel solutions using GPU programming concepts like kernels, thread blocks, and memory management.

7. Evaluate when to use CPU-only, GPU-only, or heterogeneous computing approaches for different problem types.

8. Critically assess the future trends in processor architecture including heterogeneous integration and specialized accelerators.
