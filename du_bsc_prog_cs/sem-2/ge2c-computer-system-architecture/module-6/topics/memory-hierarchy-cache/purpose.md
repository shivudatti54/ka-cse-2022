# Learning Objectives

After studying this topic, you should be able to:

1. Explain the concept of memory hierarchy and understand why it is necessary in modern computer systems.

2. Describe the principles of temporal and spatial locality that make caching effective.

3. Differentiate between various cache mapping techniques: direct mapped, fully associative, and set associative.

4. Analyze cache performance using Average Memory Access Time (AMAT) calculations for single and multi-level caches.

5. Evaluate different cache write policies (write-through vs write-back) and replacement policies (LRU, FIFO, Random).

6. Solve numerical problems involving cache address mapping and determine tag, index, and offset bits for given addresses.

7. Understand the trade-offs between cache size, block size, associativity, and overall system performance.

8. Apply knowledge of cache memory to optimize program performance by writing cache-friendly code.
