# Learning Objectives

After studying this topic, you should be able to:

1. Explain the role and characteristics of CPU registers in computer system architecture
2. Distinguish between general purpose and special purpose registers with examples
3. Differentiate between SRAM and DRAM based on their construction, operation, and applications
4. Classify various types of ROM (PROM, EPROM, EEPROM, Flash) and explain their erasing/programming methods
5. Analyze the memory organization of a computer system and calculate maximum addressable memory
6. Understand the concept of memory hierarchy and the rationale behind it
7. Compare volatile and non-volatile memory and explain their applications in computer systems
8. Apply knowledge of registers and memory to trace simple instruction fetch operations
