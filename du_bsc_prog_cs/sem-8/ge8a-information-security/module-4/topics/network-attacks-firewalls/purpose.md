# Learning Objectives

After studying this topic, you should be able to:

1. Classify network attacks into passive and active categories, and identify major attack types including DoS, DDoS, MITM, SQL injection, and XSS.

2. Explain the working mechanism of Denial of Service attacks and their distributed variant, including common attack vectors like SYN flood and ICMP flood.

3. Describe the fundamental purpose and core functions of firewalls in network security architecture.

4. Differentiate between various types of firewalls—packet filtering, stateful inspection, proxy, and next-generation firewalls—based on their operational layer and capabilities.

5. Analyze firewall architectures including DMZ and multi-tiered designs, and explain how they provide defense in depth.

6. Evaluate appropriate firewall rule configurations for given network scenarios, understanding rule ordering and the principle of implicit deny.

7. Identify mitigation strategies for common network attacks, including technical controls and architectural design approaches.

8. Apply knowledge of network security principles to design basic security solutions for enterprise network environments.
