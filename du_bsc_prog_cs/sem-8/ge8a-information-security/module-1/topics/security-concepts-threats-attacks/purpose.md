# Learning Objectives

After studying this topic, you should be able to:

1. Explain the CIA Triad (Confidentiality, Integrity, Availability) and its significance in information security.

2. Distinguish between key security concepts including authentication, authorization, non-repudiation, and accountability.

3. Classify security threats by source (internal/external) and intent (malicious/accidental), with examples of each category.

4. Identify and describe major threat categories such as malware, phishing, social engineering, DoS/DDoS, MITM attacks, and SQL injection.

5. Differentiate between passive and active attacks, providing examples and explaining their detection challenges.

6. Analyze the security attack lifecycle and explain how understanding each phase aids in implementing effective countermeasures.

7. Evaluate real-world scenarios to identify potential vulnerabilities, threats, and appropriate security controls.

8. Apply theoretical security concepts to propose practical solutions for protecting information systems.
