# Learning Objectives

After studying this topic, you should be able to:

1. **Define symmetric encryption** and explain its fundamental components (plaintext, ciphertext, secret key, encryption/decryption algorithms).

2. **Compare and contrast symmetric encryption with asymmetric encryption**, highlighting the respective advantages, disadvantages, and use cases of each approach.

3. **Differentiate between block ciphers and stream ciphers**, explaining their operational mechanisms, performance characteristics, and appropriate applications.

4. **Analyze the major symmetric encryption algorithms** including DES, Triple DES, and AES, with emphasis on key sizes, block sizes, security levels, and current relevance.

5. **Evaluate different modes of operation** (ECB, CBC, CTR, GCM) in terms of security properties, parallelizability, and suitability for various scenarios.

6. **Explain the key distribution problem** in symmetric cryptography and describe how hybrid encryption systems address this challenge in practice.

7. **Identify common vulnerabilities and attacks** against symmetric encryption systems, including ECB pattern leakage, padding oracle attacks, and keystream reuse vulnerabilities.

8. **Apply symmetric encryption concepts** to design basic security solutions for data protection scenarios, selecting appropriate algorithms and modes based on requirements.
