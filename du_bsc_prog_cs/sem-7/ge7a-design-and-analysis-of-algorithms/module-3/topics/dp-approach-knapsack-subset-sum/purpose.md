# Learning Objectives

After studying this topic, you should be able to:

1. Explain the fundamental properties of dynamic programming: optimal substructure and overlapping subproblems.

2. Formulate the 0/1 Knapsack problem mathematically using recurrence relations and understand why the greedy approach fails.

3. Implement the 0/1 Knapsack solution using dynamic programming, including both table-based and space-optimized approaches.

4. Solve the Subset Sum problem using dynamic programming and determine whether a target sum is achievable.

5. Analyze the time and space complexity of both algorithms and explain the trade-offs involved in space optimization.

6. Apply these DP techniques to solve variations and real-world optimization problems, including resource allocation and decision-making scenarios.

7. Trace through DP tables step-by-step and reconstruct solutions (identify which items form the optimal solution).

8. Differentiate between 0/1 Knapsack and Fractional Knapsack, understanding when each algorithmic approach (DP vs. Greedy) is appropriate.
