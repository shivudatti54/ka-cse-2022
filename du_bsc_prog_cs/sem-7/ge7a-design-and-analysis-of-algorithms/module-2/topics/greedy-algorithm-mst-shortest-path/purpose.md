### Purpose
Greedy algorithms provide optimal, efficient solutions to classic problems such as **Minimum Spanning Tree (MST)** and **Single‑Source Shortest Path (SSSP)**, which are core to designing reliable communication networks, transportation systems, and data‑routing protocols. Mastering these techniques equips students with rigorous proof skills and the ability to analyse time‑complexity, essential for tackling real‑world optimisation challenges in computer science.

### Learning Objectives
- **Explain** the greedy‑choice property and optimal‑substructure characteristics of MST and shortest‑path problems.  
- **Apply** Kruskal’s and Prim’s algorithms to construct minimum spanning trees for weighted graphs.  
- **Apply** Dijkstra’s algorithm to compute shortest paths from a single source in graphs with non‑negative edge weights.  
- **Analyze** the correctness of each greedy algorithm using appropriate proof techniques (cut‑and‑paste, induction).  
- **Compare** the runtime and space complexities of MST and shortest‑path algorithms, identifying trade‑offs.  
- **Design** greedy solutions for new problem contexts such as network broadcasting or routing in sensor networks.  
- **Implement** the algorithms in a programming language, testing them on realistic datasets.  
- **Evaluate** the practical performance of the algorithms, considering factors like graph density and edge‑weight distribution.
