# Learning Objectives

After studying this topic, you should be able to:

1. Explain the structure of IPv4 addresses, distinguishing between network and host portions.

2. Identify the five IP address classes and their default subnet masks.

3. Perform subnetting calculations, including determining network address, broadcast address, and usable host range.

4. Apply CIDR notation and VLSM concepts to efficiently allocate IP addresses.

5. Implement route aggregation (supernetting) to reduce routing table size.

6. Compare and contrast distance vector and link state routing algorithms.

7. Analyze the Bellman-Ford and Dijkstra algorithms for shortest path computation.

8. Evaluate the advantages and limitations of RIP and OSPF protocols in different network scenarios.

9. Solve numerical problems related to subnetting and routing table calculations.

10. Design an addressing scheme for a given network requirement using subnetting principles.
