# Learning Objectives

After studying this topic, you should be able to:

1. Explain the fundamental need for flow control in computer networks and distinguish it from congestion control
2. Describe how TCP implements flow control using the sliding window protocol and window size advertisements
3. Analyze the role of the receiver's advertised window (rwnd) in regulating data transmission
4. Explain why UDP does not implement traditional flow control and discuss its implications
5. Compare Stop-and-Wait, Go-Back-N, and Selective Repeat protocols in terms of efficiency and complexity
6. Calculate the efficiency of Stop-and-Wait protocol given propagation delay and transmission time
7. Understand the concept of zero window advertisements and how TCP prevents deadlocks
8. Apply flow control concepts to analyze real-world networking scenarios
