# Learning Objectives

After studying this topic, you should students able to:

1. Define the purpose of the `final` keyword in Java and its relevance to inheritance.
2. Identify the three main applications of the `final` keyword: final classes, final methods, and final variables.
3. Explain how final classes prevent inheritance and provide an example of a final class in Java.
4. Describe the effect of declaring a method as final and demonstrate how it restricts method overriding in subclasses.
5. Use final variables to create constants and justify their importance in programming.
6. Compare and contrast the use of final classes, methods, and variables in different design scenarios.
7. Analyze the role of the `final` keyword in ensuring code stability and security, and assess its impact on API design.
8. Design a simple class hierarchy that incorporates final classes, methods, and variables to achieve specific design goals.
