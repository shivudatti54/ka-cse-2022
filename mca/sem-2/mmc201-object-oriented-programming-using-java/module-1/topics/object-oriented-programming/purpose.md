# Learning Objectives

After studying this topic, you should students able to:

1. Understand the fundamental concepts of Object-Oriented Programming and its advantages over procedural programming paradigms.

2. Explain the relationship between classes and objects, and demonstrate how to create and use objects in Java programs.

3. Implement encapsulation in Java using access modifiers (private, public, protected) and getter/setter methods to protect data.

4. Apply the concept of inheritance to create hierarchies of classes, reusing code and establishing "is-a" relationships between classes.

5. Demonstrate polymorphism through method overloading (compile-time) and method overriding (runtime) with appropriate examples.

6. Implement abstraction using abstract classes and interfaces to hide implementation details and show essential features.

7. Create constructors with different types (default, parameterized, copy) and understand their role in object initialization.
