# Learning Objectives

After studying this topic, you should students able to:

1. Define the terms "pass-by-value" and "pass-by-reference" in the context of Java programming.
2. Identify the type of argument passing used by Java for primitive types and object references.
3. Explain the difference between passing primitive types and object references as method arguments in Java.
4. Describe the behavior of code that modifies parameters inside a method, including the impact on original variables.
5. Use Java code to demonstrate the effect of modifying primitive and reference type parameters within a method.
6. Compare and contrast the behavior of primitive and reference types when passed as arguments to methods in Java.
7. Justify the use of pass-by-value for primitive types and reference types in Java, considering the implications for code predictability and efficiency.
8. Design a simple Java program that correctly passes and modifies primitive and reference type variables as method arguments.
