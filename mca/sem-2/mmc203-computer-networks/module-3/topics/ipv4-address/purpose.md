# Learning Objectives

After studying this topic, you should students able to:

1. Explain the structure of IPv4 addresses and describe the 32-bit addressing scheme with dotted decimal representation.

2. Identify and differentiate between the five classes of IPv4 addresses (A, B, C, D, E) based on their first octet ranges and default subnet masks.

3. Calculate the network address, broadcast address, and number of usable host addresses for a given IP address and subnet mask.

4. Apply subnetting concepts to divide a network into smaller subnetworks, determining appropriate subnet masks for specific requirements.

5. Recognize and explain the purpose of special IPv4 addresses including loopback, private, network, and broadcast addresses.

6. Convert IPv4 addresses between binary and decimal representations and perform bitwise operations to derive network information.

7. Understand CIDR notation and its advantages over classful addressing for efficient IP address allocation.

8. Troubleshoot basic networking issues by analyzing IPv4 address configurations and identifying address conflicts or misconfigurations.
