# Learning Objectives

After studying this topic, you should students able to:

1. Explain the structure and purpose of each field in the IPv4 datagram header
2. Identify the role of Version, IHL, Total Length, and Protocol fields in packet processing
3. Analyze the fragmentation process and calculate fragment sizes, offsets, and the number of fragments
4. Understand the significance of Identification, Flags, and Fragment Offset in reassembly
5. Explain the working of Time to Live (TTL) and its role in preventing routing loops
6. Describe how Header Checksum provides error detection and is recalculated at each router
7. Differentiate between the various IP address classes and special IP addresses
8. Apply the concepts of IPv4 addressing and fragmentation to solve numerical problems
