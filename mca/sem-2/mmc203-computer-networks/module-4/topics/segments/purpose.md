# Learning Objectives

After studying this topic, you should students able to:

1. Describe the structure of a TCP segment header and identify all its fields
2. Explain the purpose of sequence numbers and acknowledgment numbers in reliable data transfer
3. Identify and describe the six TCP control flags and their functions
4. Calculate the sequence number range for a given segment based on data length
5. Understand how the Maximum Segment Size (MSS) affects segment creation
6. Analyze segment exchanges during connection establishment (three-way handshake) and termination
7. Explain the role of the checksum in error detection for TCP segments
8. Apply segment header field knowledge to troubleshoot network communication scenarios
