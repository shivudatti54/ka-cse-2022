# Networks
Computer networks comprise interconnected computing devices that communicate and share resources through standardized protocols. Essential components include nodes, network interface cards, transmission media (guided and unguided), intermediate network devices, and communication protocols. Networks are classified by geographic area into PAN (personal), LAN (local), CAN (campus), MAN (metropolitan), and WAN (wide area) categories, each exhibiting distinct performance characteristics and ownership models. Network topologies—including bus, star, ring, tree, mesh, and hybrid configurations—determine the physical arrangement of nodes and significantly influence network performance, reliability, and fault tolerance. Transmission modes (simplex, half-duplex, full-duplex) define the directionality and timing of data flow between communicating devices. These foundational concepts establish the framework for understanding advanced networking principles and technologies.
