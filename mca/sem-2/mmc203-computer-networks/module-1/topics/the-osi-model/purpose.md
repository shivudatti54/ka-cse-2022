# Learning Objectives

After studying this topic, you should students able to:

1. Explain the concept of the OSI model and describe why it was developed as a standardized framework for network communication.

2. Identify and describe the seven layers of the OSI model in correct order, from the Physical Layer to the Application Layer.

3. Explain the specific functions and responsibilities of each OSI layer, including the protocols and devices operating at each layer.

4. Differentiate between physical addressing (MAC addresses), logical addressing (IP addresses), and port addressing, and identify at which OSI layers each operates.

5. Analyze how data flows through the OSI layers during transmission, including the process of encapsulation and de-encapsulation.

6. Apply the OSI model for troubleshooting network connectivity issues using a systematic, layer-by-layer approach.

7. Compare connection-oriented (TCP) and connectionless (UDP) protocols, and determine appropriate use cases for each based on reliability and efficiency requirements.
