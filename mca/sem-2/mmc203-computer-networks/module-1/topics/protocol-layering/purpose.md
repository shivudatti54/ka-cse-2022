# Learning Objectives

After studying this topic, you should students able to:

1. Explain the concept of protocol layering and its importance in network communication
2. Describe all seven layers of the OSI Reference Model and their functions
3. Explain the four-layer TCP/IP model and how it maps to the OSI model
4. Identify the protocols and devices operating at each layer of the OSI model
5. Understand the process of data encapsulation and decapsulation
6. Differentiate between the various PDUs (Protocol Data Units) at each layer
7. Compare TCP (reliable, connection-oriented) with UDP (unreliable, connectionless)
8. Apply knowledge of layering concepts to analyze network communication scenarios
9. Explain the difference between logical (IP) and physical (MAC) addressing
10. Describe how routers, switches, and hubs operate at different layers
