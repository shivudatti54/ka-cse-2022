# Learning Objectives

After studying this topic, you should students able to:

1. Classify computer networks based on geographical coverage into PAN, LAN, CAN, MAN, and WAN.

2. Explain the characteristics, advantages, and limitations of each network type with respect to coverage area, speed, and cost.

3. Identify appropriate network types for different scenarios such as homes, offices, campuses, cities, and enterprises.

4. Compare the transmission media and technologies used in different network types.

5. Analyze the relationship between network scale and performance parameters like latency, bandwidth, and error rates.

6. Design basic network solutions by selecting appropriate network types for given requirements.

7. Understand the role of network types in the broader context of network architecture and internetworking.

8. Apply knowledge of network types to troubleshoot connectivity issues and optimize network performance.
