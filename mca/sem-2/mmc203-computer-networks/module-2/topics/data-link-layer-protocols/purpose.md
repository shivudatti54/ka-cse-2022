# Learning Objectives

After studying this topic, you should students able to:

1. Explain the purpose and functions of the Data Link Layer in the OSI model
2. Describe and apply character stuffing and bit stuffing techniques for framing
3. Compare and contrast Stop-and-Wait and Sliding Window flow control protocols
4. Calculate the efficiency of Stop-and-Wait protocol using the given formula
5. Apply error detection techniques including parity checking and CRC
6. Implement Hamming code for single-bit error correction
7. Differentiate between HDLC and PPP protocols and their applications
8. Analyze frame structure and identify the role of different fields in Data Link Layer protocols
