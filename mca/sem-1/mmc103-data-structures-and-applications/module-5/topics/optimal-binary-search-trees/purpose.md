# Learning Objectives

After studying this topic, you should students able to:

1. Define the Optimal Binary Search Tree problem and explain its motivation in practical search applications
2. Distinguish between successful and unsuccessful search probabilities (pᵢ and qᵢ) and their roles in the OBST formulation
3. Prove the optimal substructure property for OBST and explain why dynamic programming is applicable
4. Apply the dynamic programming recurrence relation to compute the expected search cost for small instances
5. Construct the complete DP table for given probability distributions and determine the optimal tree structure
6. Analyze the time and space complexity of the OBST algorithm and explain Knuth's optimization
7. Verify the correctness of an OBST solution by manually computing the expected search cost
8. Compare OBST with other tree construction strategies like balanced BSTs and explain when OBST is preferred
