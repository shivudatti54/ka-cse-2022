# Depth First Search is a graph traversal algorithm that explores vertices by going deep along each branch before backtracking, utilizing LIFO behavior via stack or recursion. The algorithm achieves O(V+E) time complexity with adjacency lists and correctly visits all reachable vertices exactly once (proved via induction on path length). Each edge traversed during DFS gets classified as tree, back, forward, or cross based on discovery/finish timestamps, enabling cycle detection and various graph applications. DFS forms the foundation for topological sorting, connected component identification, articulation point detection, and strongly connected component computation.
