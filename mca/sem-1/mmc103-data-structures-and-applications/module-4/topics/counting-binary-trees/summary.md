# Counting Binary Trees
Binary tree counting leads to Catalan numbers, given by C(n) = (2n)! / [(n+1)!n!], with recurrence C(n+1) = Σ(i=0 to n) C(i)C(n-i). This sequence counts: (1) rooted binary trees with n+1 nodes, (2) binary search trees with n distinct keys, and (3) full binary trees with n internal nodes (total nodes = 2n+1). Complete binary trees have exactly 1 structure per n. Key proofs include combinatorial derivation using lattice paths (ballot problem) and recurrence proof via root node selection. Applications include BST search complexity analysis showing average comparisons ≈ 2ln(n). Common errors include confusing node count with formula index, and conflating labeled with unlabeled tree counting.
