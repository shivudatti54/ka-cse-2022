# Breadth First Search (BFS) is a level-by-level graph traversal algorithm that uses a FIFO queue to explore vertices. The key theorem proved in this module establishes that BFS computes the shortest path (minimum edge count) from a source vertex to all reachable vertices in an unweighted graph. The algorithm maintains arrays for visited status, distance, and predecessor, initializing the source with distance 0 and progressively discovering vertices at increasing distances. Each vertex is enqueued exactly once and dequeued exactly once, yielding O(V + E) time complexity with adjacency list representation. BFS finds applications in shortest path computation, cycle detection in undirected graphs, bipartite graph checking, and level-order tree traversals. The module includes a complete C implementation, step-by-step execution trace, comparison with Depth First Search, and handling of disconnected graph components.
