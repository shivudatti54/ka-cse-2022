# Connected Components
This study material provides comprehensive coverage of Connected Components in graphs, covering both undirected and directed cases. It targets students/students/students with formal mathematical definitions, correctness proofs for algorithms, time/space complexity analysis, and multiple solution approaches (DFS, BFS, Union-Find). The content includes Kosaraju's algorithm for strongly connected components with detailed proofs, practical applications across computer science domains, numerical problems with solutions, and analytical multiple-choice questions. The complete C implementation demonstrates practical programming aspects.
