# Dequeues
A deque (double-ended queue) is a linear data structure allowing O(1) insertion and deletion at both front and rear ends. It generalizes stacks and queues, making it a universal structure. The circular array implementation uses modular arithmetic to efficiently wrap indices, achieving constant time complexity for all operations. Key variants include input-restricted (insertion only at rear) and output-restricted (deletion only at front) deques. The ADT specifies eight core operations, and both array-based and linked list implementations provide O(1) time complexity for all operations, with the array version offering better cache locality and the linked list offering dynamic sizing.
