# Stacks Using Dynamic Arrays
This chapter examines stacks implemented using dynamically growing arrays, addressing the critical limitations of fixed-size array stacks. The core technique involves geometric growth (doubling capacity when full), which achieves O(1) amortized time complexity for push operations—proven through both aggregate and accounting methods. A complementary shrink-to-fit policy halves capacity when utilization drops to 25%, preventing memory waste. The chapter provides complete C implementation with all stack operations, comparative analysis between geometric and incremental growth strategies, and practical considerations versus linked list implementations. Eight multiple choice questions test conceptual understanding and analytical skills, while numerical problems require calculating final capacities and total copy costs for sequences of operations.
