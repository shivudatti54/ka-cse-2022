# Strings are formal Abstract Data Types (ADTs) defined as finite sequences of characters. This module examines three primary representations: (1) Fixed-length arrays offering O(1) random access but O(n) insertion/deletion; (2) Dynamic arrays providing amortized O(1) insertion with automatic resizing; (3) Linked lists enabling O(1) concatenation and deletion at known positions but O(n) random access. The KMP algorithm achieves O(n+m) pattern matching using preprocessing to compute the Longest Proper Prefix Suffix (LPS) array, while Rabin-Karp uses hashing for O(n+m) average performance. Key theorems establish complexity proofs for concatenation, insertion shifting, and amortized dynamic array growth. The MCQs test application-level understanding of reallocation counts, LPS computation, worst-case scenarios, and representation trade-offs.
