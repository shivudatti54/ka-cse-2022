# Review Of Pointers And Dynamic Memory Allocation
Pointers are variables storing memory addresses, fundamental to dynamic memory allocation in C. Key operations include address-of (&) and dereference (\*). Pointer arithmetic scales by sizeof(type), enabling array traversal. Array names decay to pointers to first elements. Structure pointers use arrow (->) operator for member access. Self-referential structures enable linked lists. Double pointers (\*\*) enable dynamic 2D arrays and pointer modification in functions. Dynamic memory functions—malloc (uninitialized), calloc (zero-initialized), realloc (resize), and free (deallocate)—manage heap memory. Common pitfalls include memory leaks (unfreed memory), dangling pointers (freed memory access), and buffer overflows. These concepts are essential for implementing dynamic data structures like stacks, queues, linked lists, and trees.
