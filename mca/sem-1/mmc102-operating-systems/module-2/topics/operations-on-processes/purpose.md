# Learning Objectives

After studying this topic, you should be able to:

1. Explain the process creation mechanism in operating systems, including the role of fork() and exec() system calls in UNIX-like systems

2. Describe the process termination process and the importance of the wait() system call in preventing zombie processes

3. Identify and explain the components of a Process Control Block (PCB) and its role in process management

4. Analyze process state transitions and draw the state transition diagram with appropriate event labels

5. Understand the context switching mechanism and its impact on system performance

6. Compare and contrast different interprocess communication models, including message-passing and shared-memory approaches

7. Differentiate between zombie and orphan processes and explain how the operating system handles each case
