# Learning Objectives

After studying this topic, you should be able to:

1. Explain the purpose and functions of directories in operating systems, including the metadata stored in directory entries.

2. Compare and contrast different directory structures including single-level, two-level, hierarchical, and acyclic graph organizations.

3. Analyze the advantages and disadvantages of file allocation methods: contiguous, linked, and indexed allocation.

4. Evaluate free space management techniques including bit vectors, linked lists, grouping, and counting methods.

5. Apply disk scheduling algorithms (FCFS, SSTF, SCAN, C-SCAN, LOOK) to compute head movement and compare algorithm performance.

6. Understand the relationship between logical directory structures and physical disk organization including partitioning and file system implementation.

7. Distinguish between hard links and symbolic links in acyclic graph directory structures and explain their behavior during file operations.
