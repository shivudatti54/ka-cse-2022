# Learning Objectives

After studying this topic, you should be able to:

1. Explain the three primary file allocation methods: contiguous, linked, and indexed allocation.

2. Compare the advantages and disadvantages of each allocation method in terms of access time, storage efficiency, and implementation complexity.

3. Analyze how external and internal fragmentation affect different allocation methods.

4. Evaluate the performance characteristics of sequential access versus random access for each allocation method.

5. Describe how FAT file systems and UNIX inode structures implement variations of linked and indexed allocation.

6. Apply allocation strategies (first-fit, best-fit, worst-fit) to determine block placement in contiguous allocation scenarios.

7. Calculate maximum file sizes supported by different indexed allocation variations based on block size and pointer size.
