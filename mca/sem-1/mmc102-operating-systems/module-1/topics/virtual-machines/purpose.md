# Learning Objectives

After studying this topic, you should be able to:

1. DEFINE VIRTUAL MACHINES AND EXPLAIN THEIR ROLE IN MODERN COMPUTING INFRASTRUCTURE

2. DISTINGUISH BETWEEN PROCESS VIRTUAL MACHINES AND SYSTEM VIRTUAL MACHINES WITH RELEVANT EXAMPLES

3. EXPLAIN THE ARCHITECTURE AND FUNCTIONS OF A VIRTUAL MACHINE MONITOR (VMM) OR HYPERVISOR

4. COMPARE TYPE 1 AND TYPE 2 HYPERVISORS IN TERMS OF ARCHITECTURE, PERFORMANCE, AND USE CASES

5. DESCRIBE CPU VIRTUALIZATION TECHNIQUES INCLUDING HARDWARE ASSISTED VIRTUALIZATION

6. EXPLAIN MEMORY VIRTUALIZATION CONCEPTS INCLUDING GUEST PHYSICAL ADDRESSES AND HOST PHYSICAL ADDRESSES

7. UNDERSTAND THE WORKING OF VIRTUAL STORAGE AND VIRTUAL NETWORKING IN VIRTUALIZED ENVIRONMENTS

8. DIFFERENTIATE BETWEEN TRADITIONAL VIRTUAL MACHINES AND CONTAINER-BASED VIRTUALIZATION
