# Linear Combinations and Span

## Introduction

In linear algebra, the concept of linear combinations and span is fundamental to understanding vector spaces and their properties. A linear combination of vectors is an expression of a vector as the sum of scalar multiples of other vectors. The span of a set of vectors is the set of all possible linear combinations of those vectors. In this topic, we will explore the concept of linear combinations and span, and their importance in computing.

Linear combinations and span have numerous applications in computer science, including computer graphics, machine learning, and data analysis. For instance, in computer graphics, linear combinations of vectors are used to perform transformations on images and 3D models. In machine learning, linear combinations of features are used to make predictions and classify data.

## Key Concepts

* **Linear Combination**: A linear combination of vectors is an expression of a vector as the sum of scalar multiples of other vectors. Mathematically, it can be represented as: `v = a1*u1 + a2*u2 + ... + an*un`, where `v` is the resulting vector, `a1, a2, ..., an` are scalars, and `u1, u2, ..., un` are vectors.
* **Span**: The span of a set of vectors is the set of all possible linear combinations of those vectors. It is denoted as `Span(U)`, where `U` is the set of vectors.
* **Basis**: A basis of a vector space is a set of vectors that spans the space and is linearly independent.
* **Linear Independence**: A set of vectors is said to be linearly independent if none of the vectors can be expressed as a linear combination of the others.

## Examples

### Example 1: Linear Combination of Vectors

Suppose we have two vectors `u = [1, 2]` and `v = [3, 4]`. We can form a linear combination of these vectors by multiplying them with scalars and adding them together. For example: `w = 2*u + 3*v = [2*1 + 3*3, 2*2 + 3*4] = [11, 16]`.

### Example 2: Span of a Set of Vectors

Suppose we have a set of vectors `U = {[1, 0], [0, 1]}`. The span of this set is the set of all possible linear combinations of these vectors. For example: `v = 2*[1, 0] + 3*[0, 1] = [2, 3]` is in the span of `U`.

### Example 3: Basis of a Vector Space

Suppose we have a vector space `V` with a basis `B = {[1, 0], [0, 1]}`. This means that any vector in `V` can be expressed as a linear combination of the vectors in `B`.
