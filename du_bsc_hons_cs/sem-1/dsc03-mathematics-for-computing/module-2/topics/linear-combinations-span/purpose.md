# Learning Objectives

After studying this topic, you should be able to:

1. Define linear combinations and span, and explain their importance in computing.
2. Form linear combinations of vectors and determine the span of a set of vectors.
3. Understand the concept of basis and linear independence, and how they are related to span.
4. Determine if a set of vectors is linearly independent or not.
5. Provide examples of the applications of linear combinations and span in computing.
6. Solve problems involving linear combinations and span, such as finding the span of a set of vectors or determining if a vector is in the span of a set of vectors.
7. Use matrices to solve problems involving linear combinations.
8. Understand the relationship between linear combinations and vector spaces.
