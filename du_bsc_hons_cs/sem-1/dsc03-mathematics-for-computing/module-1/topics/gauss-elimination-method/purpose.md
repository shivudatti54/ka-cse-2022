# Learning Objectives

After studying this topic, you should be able to:

1. Represent a system of linear equations as an augmented matrix.
2. Use elementary row operations to transform the matrix into upper triangular form.
3. Solve for the variables using back-substitution.
4. Check your work by plugging the values of the variables back into the original equations.
5. Apply the Gauss Elimination Method to solve systems of linear equations in various fields.
6. Use a calculator to perform calculations.
7. Identify and correct errors in your work.
8. Solve systems of linear equations with multiple variables.
