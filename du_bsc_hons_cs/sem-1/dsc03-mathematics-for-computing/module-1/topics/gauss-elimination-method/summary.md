# Gauss Elimination Method - Summary

## Key Definitions and Concepts

* Augmented matrix: a matrix that combines the coefficients of the variables and the constants of the system of linear equations.
* Elementary row operations: swap two rows, multiply a row by a non-zero constant, add a multiple of one row to another row.
* Upper triangular form: a matrix where all the elements below the main diagonal are zero.
* Back-substitution: the process of solving for the variables by substituting the values of the variables from the bottom row to the top row.

## Important Formulas and Theorems

* None

## Key Points

* The Gauss Elimination Method is used to solve systems of linear equations.
* The method involves transforming the augmented matrix into upper triangular form using elementary row operations.
* Back-substitution is used to solve for the variables.
* The method can be applied to solve systems of linear equations in various fields.
* A calculator can be used to perform calculations.
* It is essential to check your work by plugging the values of the variables back into the original equations.
* The method can be used to solve systems of linear equations with multiple variables.

## Common Mistakes to Avoid

* Writing the augmented matrix incorrectly.
* Not using elementary row operations correctly.
* Not checking your work by plugging the values of the variables back into the original equations.
* Not using a calculator to perform calculations.
