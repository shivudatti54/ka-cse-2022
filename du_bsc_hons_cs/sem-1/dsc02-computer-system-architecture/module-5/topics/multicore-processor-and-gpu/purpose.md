### Purpose
Modern computing workloads—from real‑time graphics to data‑intensive analytics—demand massive parallelism and energy efficiency that a single‑core processor cannot provide. Understanding multicore CPUs and GPUs is essential for designing scalable, high‑performance systems and for writing software that exploits modern hardware parallelism.

### Learning Objectives
- **Explain** the architectural differences between multicore CPUs and GPUs, including core organization, threading models, and memory hierarchies.  
- **Analyze** the performance characteristics of parallel workloads on multicore processors and GPUs, identifying bottlenecks and scalability limits.  
- **Compare** programming models such as OpenMP, CUDA, and OpenCL, and select the appropriate model for a given application.  
- **Design** parallel algorithms that effectively utilize both CPU cores and GPU execution units while respecting data locality and synchronization requirements.  
- **Implement** basic parallel programs on multicore CPUs and GPUs, measuring execution time and throughput.  
- **Evaluate** the trade‑offs among power consumption, cost, and performance when choosing hardware for specific computing tasks.  
- **Optimize** code through techniques like kernel tuning, memory coalescing, and load balancing to maximize hardware utilization.  
- **Summarize** emerging trends in heterogeneous computing and their implications for future computer system architecture.
