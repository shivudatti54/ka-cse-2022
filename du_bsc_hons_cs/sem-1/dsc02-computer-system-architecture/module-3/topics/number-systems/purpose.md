## Purpose

Understanding number systems is fundamental to computer architecture, as all data processing, storage, and communication in digital systems relies on binary and other positional numeral representations. This topic provides the essential groundwork for grasping how computers perform arithmetic operations, represent signed integers, handle floating-point numbers, and optimize memory usage—all critical skills for software development, system programming, and computer engineering roles.

## Learning Objectives

- **Convert** numbers between binary, octal, decimal, and hexadecimal systems
- **Perform** arithmetic operations (addition, subtraction, multiplication, division) in binary and hexadecimal
- **Explain** the representation of signed integers using sign-magnitude, 1's complement, and 2's complement
- **Analyze** the concept of overflow and underflow in fixed-point arithmetic
- **Illustrate** floating-point representation (IEEE 754 standard) and its components
- **Apply** bitwise logical operations for data manipulation and masking
- **Evaluate** the trade-offs in number representation for various computing applications
