# Learning Objectives

After studying this topic, you should be able to:

1. Define R's complement and (R-1)'s complement.
2. Explain the difference between R's complement and (R-1)'s complement.
3. Find the R's complement and (R-1)'s complement of a binary number.
4. Use R's complement for subtraction.
5. Identify when to use R's complement and when to use (R-1)'s complement.
6. Explain the importance of R's complement and (R-1)'s complement in computer arithmetic.
7. Mathematically represent R's complement and (R-1)'s complement.
8. Solve problems involving R's complement and (R-1)'s complement.
