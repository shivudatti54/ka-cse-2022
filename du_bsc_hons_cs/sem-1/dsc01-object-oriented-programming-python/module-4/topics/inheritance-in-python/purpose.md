## Purpose
Inheritance is a core OOP principle that enables code reuse, promotes extensibility, and lets developers model hierarchical relationships found in real‑world systems. Mastering inheritance in Python prepares students to build scalable, maintainable applications and to work confidently with modern frameworks that rely on class reuse.

## Learning Objectives
- **Explain** the concept of inheritance and its role in object‑oriented design.  
- **Implement** base and derived classes in Python using the `class` syntax.  
- **Demonstrate** method overriding and the use of `super()` to invoke parent constructors.  
- **Analyze** attribute lookup and method resolution order (MRO) in inheritance hierarchies.  
- **Design** class structures that embody appropriate “is‑a” relationships while avoiding excessive coupling.  
- **Apply** inheritance to extend built‑in types and create custom data structures.  
- **Compare** single, multiple, and multilevel inheritance patterns and their implications in Python.  
- **Utilize** abstract base classes (ABCs) from the `abc` module to enforce interface contracts.
