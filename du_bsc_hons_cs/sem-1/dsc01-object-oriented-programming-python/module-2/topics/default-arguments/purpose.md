# Learning Objectives

After studying this topic, you should be able to:

1. Define a function with default arguments in Python.
2. Explain the benefits and pitfalls of using default arguments.
3. Use default arguments to simplify code and provide common values for function arguments.
4. Avoid shared mutable default arguments using the `None` trick.
5. Call a function with default arguments correctly.
6. Add new arguments to a function without breaking existing code using default arguments.
7. Understand the order of arguments when defining a function with default arguments.
8. Use default arguments to improve the readability and maintainability of your code.
