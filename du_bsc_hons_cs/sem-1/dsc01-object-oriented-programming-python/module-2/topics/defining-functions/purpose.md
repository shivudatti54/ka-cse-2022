### Purpose
Functions are the fundamental building blocks of modular, maintainable code in Python. In Object‑Oriented Programming, functions become methods that encapsulate behavior within classes, enabling abstraction, reuse, and polymorphism—core principles in modern software development. Mastering function definition, parameter passing, and scope is essential for building robust, scalable applications in industry and research.

### Learning Objectives
- **Define** Python functions using proper syntax, including positional, keyword, and default arguments.  
- **Implement** return values and handle multiple return values to produce structured outputs.  
- **Apply** `*args` and `**kwargs` to create flexible functions that accept variable numbers of arguments.  
- **Demonstrate** variable scope (global, nonlocal, and local) and understand the lifetime of variables within functions.  
- **Create** instance, class, and static methods in a class, correctly using the `self` and `cls` parameters.  
- **Write** recursive functions to solve problems exhibiting self‑similar structure.  
- **Utilize** lambda functions and higher‑order built‑ins (e.g., `map`, `filter`, `reduce`) for concise data processing.
