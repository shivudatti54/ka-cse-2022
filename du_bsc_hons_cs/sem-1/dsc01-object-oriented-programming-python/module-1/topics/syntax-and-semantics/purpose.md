# Learning Objectives

After studying this topic, you should be able to:

1. Distinguish between syntax and semantics in programming languages, with specific focus on Python.

2. Identify and apply Python's syntax rules including proper indentation, use of colons, and statement structure.

3. Demonstrate understanding of Python's dynamic typing system by creating variables of different data types.

4. Utilize various operators (arithmetic, comparison, logical, assignment) to construct valid Python expressions.

5. Implement control flow structures (if-elif-else, for, while) using correct Python syntax.

6. Explain the concept of namespaces and apply the LEGB rule for variable scope resolution.

7. Write, debug, and execute basic Python programs demonstrating proper syntax and semantic behavior.

8. Prepare for advanced OOP concepts by understanding the fundamental building blocks of Python programming.
