# Learning Objectives

After studying this topic, you should be able to:

1. Explain the difference between interpreted and compiled languages, and describe how the Python interpreter executes programs.

2. Set up a Python development environment and execute programs in both interactive and script modes.

3. Create, save, and run Python programs using proper file extensions (.py) and understand the importance of the `if __name__ == "__main__":` construct.

4. Declare and use variables of different data types (integers, floats, strings, booleans) in Python programs.

5. Apply appropriate type conversion functions (int(), float(), str()) to transform data between different types.

6. Implement input/output operations using the input() and print() functions with various formatting techniques.

7. Write properly structured Python code with correct indentation and identify common programming errors.

8. Debug simple programs by interpreting error messages and applying systematic troubleshooting techniques.
