## Introduction

Understanding its evolution and the major players in the market is essential for any computer science student, as per the Delhi University syllabus for Cloud Computing under NEP 2024 UGCF.

---

## History of Cloud Computing

### Key Milestones

- **1950s-1960s**: Mainframe computing with time-sharing systems
- **1970s-1980s**: Client-server architecture emerged
- **1990s**: Grid computing and utility computing concepts
- **1999**: Salesforce.com pioneered SaaS delivery
- **2002**: Amazon Web Services (AWS) launched
- **2006**: AWS Elastic Compute Cloud (EC2) became first commercial IaaS
- **2008-2010**: Google App Engine, Microsoft Azure launched
- **2010s-Present**: Hybrid cloud, multi-cloud, and edge computing era

### Evolution Drivers

- **Virtualization technology** enabled resource pooling
- **Broadband internet** made remote access feasible
- **Pay-as-you-go model** reduced capital expenditure
- **Scalability demands** from web applications

## Major Cloud Service Providers

### Top Global Providers

- **Amazon Web Services (AWS)** — Pioneer, most comprehensive services
- **Microsoft Azure** — Strong enterprise integration, hybrid cloud
- **Google Cloud Platform (GCP)** — Leader in AI/ML, data analytics
- **IBM Cloud** — Enterprise focus, hybrid cloud solutions
- **Oracle Cloud** — Database and enterprise applications

### Indian Cloud Providers

- **Reliance Jio Cloud**
- **AWS India (Local regions)**
- **Azure India**
