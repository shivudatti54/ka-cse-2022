# Google Cloud Platform (GCP)

## Introduction

Google Cloud Platform is a suite of cloud computing services provided by Google, forming one of the three major cloud platforms alongside Amazon Web Services (AWS) and Microsoft Azure. GCP offers a wide range of cloud-based services for computing, storage, networking, machine learning, and application development, making it essential for modern cloud computing education under the Delhi University NEP 2024 UGCF syllabus.

## Core Concepts

### 1. Computing Services

- **Compute Engine** — Virtual machines (VMs) running in Google's data centers with customizable CPU, memory, and storage
- **App Engine** — Platform-as-a-Service (PaaS) for building and deploying applications without managing infrastructure
- **Cloud Functions** — Serverless computing for running event-driven code functions
- **Google Kubernetes Engine (GKE)** — Managed Kubernetes service for containerized applications

### 2. Storage Services

- **Cloud Storage** — Object storage for storing and retrieving files (Buckets)
- **Persistent Disk** — Block storage for VMs
- **Filestore** — Managed file storage for enterprises
- **Cloud SQL** — Fully managed relational database service (MySQL, PostgreSQL)

### 3. Networking

- **Virtual Private Cloud (VPC)** — Isolated network environment
- **Cloud Load Balancing** — Distributes traffic across multiple instances
- **Cloud CDN** — Content delivery network for low-latency content delivery
- **Cloud DNS** — Scalable domain name system

### 4. Big Data & Analytics

- **BigQuery** — Serverless data warehouse for large-scale analytics
- **Dataflow** — Stream/batch data processing
- **Pub/Sub** — Messaging and ingestion service for real-time data

### 5. Machine Learning & AI

- **Vertex AI** — Unified ML platform for building, deploying, and scaling models
- **Vision AI, Speech AI, Natural Language API** — Pre-trained ML APIs
- **AutoML** — No-code ML model training

### 6. Identity & Security

- **Cloud IAM** — Identity and Access Management for fine-grained permissions
- **Cloud Identity** — Identity as a Service
- **Security Command Center** — Security and risk management

### 7. DevOps & Developer Tools

- **Cloud Build** — CI/CD pipeline service
- **Artifact Registry** — Package and container registry
- **Deployment Manager** — Infrastructure as code (IaC) tool

## Key Features

- **Global Infrastructure** — Distributed across multiple regions and zones worldwide
- **Pay-as-you-go Pricing** — Cost-effective with free tier and sustained use discounts
- **Scalability & Reliability** — Automatic scaling and 99.9% uptime SLA
- **Integration with Google Services** — Seamless connectivity with Gmail, Maps, YouTube APIs

## Conclusion

GCP provides comprehensive cloud solutions covering IaaS, PaaS, and SaaS models. For BSc Computer Science students at Delhi University, understanding GCP is crucial as cloud computing forms a core component of the NEP 2024 UGCF curriculum. Proficiency in GCP services prepares students for roles in cloud architecture, DevOps, data engineering, and machine learning, aligning with industry demands and emerging technology trends.
