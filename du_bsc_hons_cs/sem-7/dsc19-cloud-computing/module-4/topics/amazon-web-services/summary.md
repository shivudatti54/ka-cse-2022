# Amazon Web Services (AWS) - Cloud Computing

## Introduction

Amazon Web Services (AWS) is the leading cloud computing platform launched by Amazon in 2006. It provides on-demand cloud computing resources and services over the internet on a pay-as-you-go basis. AWS has become the foundation for cloud computing education in the Delhi University BSc (Hons) Computer Science syllabus under NEP 2024 UGCF, covering essential cloud concepts required for modern IT infrastructure.

## Key Concepts

**What is AWS?**
- Cloud computing platform offering compute, storage, database, networking, and analytics services
- Provides infrastructure-as-a-service (IaaS), platform-as-a-service (PaaS), and software-as-a-service (SaaS)
- Operates on a subscription/Pay-as-you-go pricing model

**Core AWS Services:**
- **Compute:** EC2 (Elastic Compute Cloud), Lambda, Elastic Beanstalk
- **Storage:** S3 (Simple Storage Service), EBS (Elastic Block Store), Glacier
- **Database:** RDS (Relational Database Service), DynamoDB, ElastiCache
- **Networking:** VPC (Virtual Private Cloud), CloudFront, Route 53
- **Security:** IAM (Identity and Access Management), Security Groups

**AWS Global Infrastructure:**
- **Regions:** Geographic data centers worldwide
- **Availability Zones:** Multiple isolated data centers within a region
- **Edge Locations:** Content delivery network (CDN) endpoints

**Deployment Models:**
- Public Cloud
- Private Cloud (AWS Outposts)
- Hybrid Cloud

**Key Advantages:**
- Scalability and elasticity
- Cost-effective (pay only for what you use)
- Reliability and high availability
- Global infrastructure
- Security and compliance

**Common Use Cases:**
- Web application hosting
- Data backup and disaster recovery
- Big data analytics
- Machine learning and AI
- Mobile and gaming applications

## Conclusion

AWS forms the cornerstone of modern cloud computing education. For Delhi University students, understanding AWS fundamentals—its services, architecture, and practical applications—is essential for cloud certification exams and IT industry readiness. Mastery of AWS concepts enables students to design scalable, cost-effective cloud solutions, making it a critical skill in today's technology-driven job market.
