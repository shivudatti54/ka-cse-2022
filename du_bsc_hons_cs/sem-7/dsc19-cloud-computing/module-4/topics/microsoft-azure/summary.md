# Microsoft Azure - Cloud Computing Summary

## Introduction

Microsoft Azure is a leading cloud computing platform that provides a comprehensive set of cloud services for building, deploying, and managing applications through Microsoft-managed data centers. As per the Delhi University NEP 2024 UGCF syllabus for BSc (Hons) Computer Science, Azure represents enterprise-grade cloud solutions essential for modern IT infrastructure.

## Key Concepts

### Cloud Computing Models
- **Infrastructure as a Service (IaaS)**: Virtual machines, storage, networking (Azure Virtual Machines, Blob Storage)
- **Platform as a Service (PaaS)**: Development frameworks, databases (Azure App Service, Azure SQL Database)
- **Software as a Service (SaaS)**: Ready-to-use applications (Microsoft 365, Azure Active Directory)

### Core Azure Services

**Compute Services**
- Azure Virtual Machines (VMs)
- Azure App Service
- Azure Functions (Serverless)
- Azure Kubernetes Service (AKS)

**Storage Services**
- Azure Blob Storage (unstructured data)
- Azure File Storage (file shares)
- Azure Disk Storage (managed disks)
- Azure Table Storage (NoSQL)

**Networking**
- Azure Virtual Network (VNet)
- Azure Load Balancer
- Azure VPN Gateway
- Azure CDN (Content Delivery Network)

**Database Services**
- Azure SQL Database
- Azure Cosmos DB (multi-model NoSQL)
- Azure Database for MySQL/PostgreSQL
- Azure Redis Cache

### Azure Deployment Models
- **Public Cloud**: Microsoft-owned infrastructure
- **Private Cloud**: Azure Stack (on-premises)
- **Hybrid Cloud**: Combine on-premises with Azure

### Azure Subscriptions and Management
- Subscription types (Free, Pay-as-you-go, Enterprise)
- Resource Groups for logical organization
- Azure Resource Manager (ARM) for deployment
- Azure Portal, CLI, and PowerShell management

### Security and Compliance
- Azure Active Directory (Identity & Access Management)
- Role-Based Access Control (RBAC)
- Azure Security Center
- Encryption (at rest and in transit)
- Compliance certifications (ISO, GDPR, SOC)

### Azure Pricing
- Pay-as-you-go model
- Reserved Instances (1-3 years)
- Free tier services
- Cost Management tools

## Conclusion

Microsoft Azure provides scalable, secure, and cost-effective cloud solutions essential for modern software development and IT operations. Understanding Azure services, deployment models, and management tools is crucial for computer science students preparing for industry-relevant cloud computing roles. Mastery of Azure fundamentals aligns with the Delhi University curriculum's emphasis on practical cloud skills and enterprise-ready competencies.
