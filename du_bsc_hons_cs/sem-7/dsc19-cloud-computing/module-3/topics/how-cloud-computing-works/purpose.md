# Learning Objectives

After studying this topic, you should be able to:

1. Explain the layered architecture of cloud computing and the role of each component (physical, virtualization, resource pooling, management layers).

2. Describe the three primary cloud service models—IaaS, PaaS, and SaaS—with appropriate examples and understand the trade-offs between control and management responsibility.

3. Compare and contrast the four cloud deployment models—public, private, hybrid, and community—with respect to ownership, access, and use cases.

4. Analyze the step-by-step process of how cloud services are provisioned from user request to resource delivery.

5. Identify and explain key enabling technologies in cloud computing, including virtualization, containers, and serverless computing.

6. Evaluate the advantages of cloud computing over traditional on-premises infrastructure, particularly regarding scalability, cost, and agility.

7. Apply cloud computing concepts to design appropriate solutions for different business scenarios.
