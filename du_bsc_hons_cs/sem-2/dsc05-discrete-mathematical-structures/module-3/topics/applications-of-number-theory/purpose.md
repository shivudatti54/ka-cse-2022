# Learning Objectives

After studying this topic, you should be able to:

1. Apply modular arithmetic concepts to solve congruence equations and understand their properties in cryptographic contexts.

2. Compute the greatest common divisor of two integers using the Euclidean algorithm and its extended version.

3. Calculate modular multiplicative inverses using the extended Euclidean algorithm and determine when such inverses exist.

4. Explain Euler's totient function and apply Euler's theorem to solve problems involving modular exponentiation.

5. Demonstrate the RSA encryption and decryption process, including key generation with small prime numbers.

6. Solve systems of simultaneous congruences using the Chinese Remainder Theorem for pairwise coprime moduli.

7. Analyze the computational complexity of number-theoretic algorithms and understand their practical implications in cryptography.

8. Evaluate the security foundations of modern cryptographic systems based on number-theoretic problems.
