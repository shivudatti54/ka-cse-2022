# Learning Objectives

After studying this topic, you should be able to:

1. Define modular exponentiation and explain its mathematical formulation a^b mod n

2. Apply the fundamental properties of modular arithmetic to simplify exponentiation problems

3. Compute modular exponentiation using the repeated squaring method efficiently

4. Apply Fermat's Little Theorem to compute modular exponents when the modulus is prime

5. Use Euler's Theorem to reduce exponents when working with composite moduli

6. Find the modular multiplicative inverse using the Extended Euclidean Algorithm

7. Explain the role of modular exponentiation in cryptographic applications, particularly RSA

8. Solve problems involving large numbers by appropriate reduction techniques
