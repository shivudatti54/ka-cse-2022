# Learning Objectives

After studying this topic, you should be able to:

1. Define and represent binary relations using set notation, matrices, and directed graphs
2. Analyze and determine whether a relation possesses reflexive, symmetric, antisymmetric, or transitive properties
3. Identify equivalence relations by verifying all three properties (reflexive, symmetric, transitive)
4. Compute equivalence classes for a given equivalence relation and understand their properties
5. Explain the relationship between equivalence relations and partitions of a set
6. Perform composition of relations and find the inverse of a relation
7. Compute reflexive, symmetric, and transitive closures of a relation
8. Apply the concept of congruence modulo n to identify equivalence classes in modular arithmetic
