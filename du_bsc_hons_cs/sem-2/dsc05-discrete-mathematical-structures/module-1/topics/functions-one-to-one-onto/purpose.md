# Learning Objectives

After studying this topic, you should be able to:

1. Define a function, identifying its domain, codomain, range, and image elements
2. Determine whether a given function is one-to-one (injective) using the formal definition and counterexamples
3. Determine whether a given function is onto (surjective) by checking if the range equals the codomain
4. Identify bijective functions and explain why they are invertible
5. Find the inverse of a bijective function and verify the result
6. Analyze function composition and determine when the composition preserves injectivity or surjectivity
7. Apply the pigeonhole principle to reason about functions between finite sets of different sizes
8. Connect these concepts to practical applications in computer science, such as hash functions and encryption
