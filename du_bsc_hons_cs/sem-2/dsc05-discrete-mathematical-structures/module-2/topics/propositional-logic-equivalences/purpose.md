# Learning Objectives

After studying this topic, you should be able to:

1. Classify any proposition as a tautology, contradiction, or contingency based on its truth table

2. Define and explain logical equivalence using both the truth table definition and the biconditional tautology definition

3. State and apply all fundamental logical equivalence laws including identity, domination, idempotent, complement, commutative, associative, distributive, De Morgan's, and absorption laws

4. Prove logical equivalences using the truth table method by comparing truth values across all possible interpretations

5. Prove logical equivalences using the algebraic proof method by systematically transforming one expression into another using known laws

6. Apply logical equivalences to simplify complex boolean expressions in practical applications such as digital circuit design

7. Correctly apply De Morgan's laws when negating compound propositions, ensuring proper distribution and connective flipping

8. Distinguish between the biconditional connective (↔) and the logical equivalence relation (≡) and use appropriate notation in proofs
