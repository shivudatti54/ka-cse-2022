# Learning Objectives

After studying this topic, you should be able to:

1. Define and distinguish between Eulerian paths, Eulerian circuits, Hamiltonian paths, and Hamiltonian cycles in graph theory.

2. Apply Euler's theorem to determine whether a given connected graph has an Eulerian path or circuit based on vertex degrees.

3. Execute Fleury's algorithm step-by-step to construct an Eulerian path or circuit in a graph.

4. Execute Hierholzer's algorithm efficiently to find Eulerian circuits in O(V + E) time.

5. Apply Dirac's theorem and Ore's theorem to determine sufficient conditions for a graph to be Hamiltonian.

6. Analyze the fundamental differences between Eulerian and Hamiltonian problems, including their computational complexity classifications.

7. Identify real-world scenarios that correspond to Eulerian and Hamiltonian path problems.

8. Evaluate whether a given graph is Eulerian, semi-Eulerian, Hamiltonian, or neither, providing justification based on relevant theorems.
