# Learning Objectives

After studying this topic, you should be able to:

1. Define and compute conditional probability P(A|B) using the formula P(A ∩ B)/P(B), ensuring P(B) > 0

2. Apply the multiplication rule P(A ∩ B) = P(B) × P(A|B) to find joint probabilities of events

3. Determine whether two events are independent using the criterion P(A ∩ B) = P(A) × P(B)

4. Distinguish between pairwise independence and mutual independence for multiple events

5. Use the Law of Total Probability to compute unconditional probabilities from conditional probabilities

6. Apply Bayes' theorem to invert conditional probabilities and solve real-world inference problems

7. Analyze real-world scenarios involving medical testing, quality control, and classification to extract relevant probabilities

8. Solve examination-style problems involving cards, dice, and random selection with confidence
