# Learning Objectives

After studying this topic, you should be able to:

1. Define and distinguish between discrete and continuous random variables with appropriate examples from computing contexts.

2. Construct and verify the Probability Mass Function (PMF) for a discrete random variable, ensuring it satisfies non-negativity and normalization properties.

3. Calculate and interpret the Cumulative Distribution Function (CDF) for discrete random variables and understand its step-function nature.

4. Compute expected value (mean), variance, and standard deviation of discrete random variables using both definition and shortcut formulas.

5. Identify real-world computing scenarios that follow Bernoulli, Binomial, Poisson, and Geometric distributions.

6. Apply appropriate probability formulas to solve problems involving the four fundamental discrete distributions.

7. Use the linearity of expectation property to simplify complex probability calculations in algorithm analysis and system modeling.

8. Evaluate probability expressions involving discrete random variables and interpret results in the context of computer science applications.
