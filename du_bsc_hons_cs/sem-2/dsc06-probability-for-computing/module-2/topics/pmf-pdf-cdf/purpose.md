# Learning Objectives

After studying this topic, you should be able to:

1. Distinguish between discrete and continuous random variables with computing-related examples

2. Define and explain the Probability Mass Function (PMF), including its properties and valid conditions

3. Define and explain the Probability Density Function (PDF), emphasizing why P(X = a) = 0 for continuous variables

4. Define the Cumulative Distribution Function (CDF) and explain its universal applicability to both discrete and continuous distributions

5. Apply the fundamental relationships between PMF, PDF, and CDF to solve probability problems

6. Compute probabilities using PMFs (via summation) and PDFs (via integration)

7. Verify whether a given function represents a valid PMF or PDF by checking the non-negativity and normalization conditions

8. Interpret probability distributions in practical computing contexts such as algorithm analysis and system modeling
