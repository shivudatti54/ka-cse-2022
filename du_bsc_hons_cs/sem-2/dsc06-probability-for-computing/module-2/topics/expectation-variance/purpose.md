# Learning Objectives

After studying this topic, you should be able to:

1. Define and compute the expected value of discrete and continuous random variables

2. Apply the properties of expectation, including linearity of expectation, to solve complex probability problems

3. Calculate variance and standard deviation using both the definition and computational formulas

4. Understand and apply the properties of variance, including how it transforms under linear transformations

5. Compute and interpret covariance between two random variables

6. Apply the Law of Total Expectation to solve multi-stage probability problems

7. Use expectation and variance concepts in computing applications such as algorithm analysis

8. Verify computational results using alternative methods to ensure accuracy
