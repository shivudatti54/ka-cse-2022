# Learning Objectives

After studying this topic, you should be able to:

1. **Define** continuous random variables and distinguish them from discrete random variables based on their value sets and probability assignment methods.

2. **Explain** the concept of probability density function (PDF) and its two fundamental properties: non-negativity and normalization.

3. **Compute** probabilities for continuous random variables using integration over specified intervals, applying the definition P(a ≤ X ≤ b) = ∫[a to b] f(x) dx.

4. **Derive** and interpret the cumulative distribution function (CDF) from a given PDF and vice versa, understanding their relationship through differentiation and integration.

5. **Calculate** expected value, variance, and standard deviation for continuous distributions using the appropriate integral formulas.

6. **Apply** the uniform, exponential, and normal distributions to solve real-world computing problems, including performance modeling and data analysis scenarios.

7. **Transform** normal random variables to standard normal using z-score standardization and use standard normal tables for probability calculations.

8. **Analyze** the memoryless property of exponential distributions and explain its significance in computing contexts like queuing theory and reliability analysis.
