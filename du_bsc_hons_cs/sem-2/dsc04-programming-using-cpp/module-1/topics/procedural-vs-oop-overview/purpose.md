# Learning Objectives

After studying this topic, you should be able to:

1.  Define Procedural Programming and Object-Oriented Programming.
2.  Explain the characteristics of Procedural Programming and Object-Oriented Programming.
3.  Identify the advantages and disadvantages of each paradigm.
4.  Write programs using both Procedural and OOP styles.
5.  Explain the concept of encapsulation in OOP.
6.  Use classes and objects in OOP.
7.  Compare and contrast Procedural and OOP with examples.
8.  Apply Procedural and OOP concepts to real-world problems.
