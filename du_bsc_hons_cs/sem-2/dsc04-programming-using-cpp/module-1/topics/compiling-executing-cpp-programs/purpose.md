# Learning Objectives

After studying this topic, you should be able to:

1. Explain the different stages of the compilation process, including preprocessing, compilation, assembly, and linking.
2. Compile a C++ program using GCC and G++.
3. Understand the role of the linker in resolving external references in the object code.
4. Execute a C++ program using the command line.
5. Explain the differences between static and dynamic linking.
6. Use the `-c` flag to compile a C++ program without linking.
7. Use the `-o` flag to specify the output file name.
8. Troubleshoot common errors that occur during the compilation process.
