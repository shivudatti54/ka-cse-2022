# Learning Objectives

After studying this topic, you should be able to:

1.  Explain the concept of inheritance in C++.
2.  Describe the different types of inheritance in C++.
3.  Use access modifiers to control member accessibility.
4.  Write C++ code that demonstrates single inheritance.
5.  Write C++ code that demonstrates multiple inheritance.
6.  Write C++ code that demonstrates multilevel inheritance.
7.  Resolve ambiguity in multiple inheritance using the `virtual` keyword.
8.  Explain the order of constructor and destructor calls in a derived class.
