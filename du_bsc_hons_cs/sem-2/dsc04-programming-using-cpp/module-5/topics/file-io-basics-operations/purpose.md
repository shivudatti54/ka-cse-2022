## Purpose

File I/O is fundamental to most software systems, enabling programs to persist data beyond runtime, read configuration files, and process large datasets efficiently. In line with the NEP 2024 UGCF curriculum for BSc (Hons) Computer Science at Delhi University, mastering file operations in C++ equips students with the skills to build robust applications that interact with the file system, handle errors, and manage resources responsibly.

## Learning Objectives

- **Explain** the role and necessity of file I/O in C++ programs.  
- **Use** the standard stream classes (`ifstream`, `ofstream`, `fstream`) to open, read, and write text files.  
- **Perform** both formatted and binary file I/O operations, understanding their differences and use cases.  
- **Handle** file‑related errors and exceptions, ensuring graceful failure and resource cleanup.  
- **Implement** file positioning techniques (`seekg`, `seekp`, `tellg`, `tellp`) to navigate within files.  
- **Apply** best practices for file management, including proper opening, closing, and checking stream state.  
- **Design** programs that read/write structured data (e.g., records) to and from files.  
- **Demonstrate** file I/O in realistic scenarios such as logging, configuration handling, and data archival.
