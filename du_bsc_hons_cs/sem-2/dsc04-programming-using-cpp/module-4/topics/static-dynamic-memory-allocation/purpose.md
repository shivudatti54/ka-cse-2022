# Learning Objectives

After studying this topic, you should be able to:

1. Distinguish between static and dynamic memory allocation in C++, understanding their respective memory regions (stack vs heap).

2. Explain the characteristics, advantages, and limitations of stack-based (static) memory allocation.

3. Implement dynamic memory allocation using the `new` and `delete` operators for both single objects and arrays.

4. Compare C-style memory management functions (`malloc`, `calloc`, `realloc`, `free`) with C++ operators (`new`, `delete`).

5. Identify and prevent common memory-related errors including memory leaks, dangling pointers, and double-deletion bugs.

6. Apply smart pointers (`unique_ptr`, `shared_ptr`) for automatic memory management in modern C++.

7. Analyze code to trace memory allocation and deallocation patterns, identifying potential issues in exam scenarios.

8. Design solutions that appropriately choose between static and dynamic allocation based on problem requirements.
