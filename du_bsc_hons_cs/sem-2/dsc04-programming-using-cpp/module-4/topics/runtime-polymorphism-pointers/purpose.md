## Purpose & Learning Objectives  

Runtime polymorphism, realized through virtual functions and base‑class pointers, is a cornerstone of flexible, maintainable object‑oriented software. It enables a single interface to operate on multiple derived types, supporting extensible codebases, reusable components, and dynamic behavior essential in real‑world applications such as GUI frameworks, game engines, and plugin architectures.

### Learning Objectives  

- **Define** the concepts of runtime polymorphism, virtual functions, and dynamic binding in C++.  
- **Implement** base‑class pointers and references to derived objects to achieve polymorphic behavior.  
- **Apply** pure virtual functions and abstract classes to design extensible interfaces.  
- **Analyze** the internal mechanism of virtual tables (v‑tables) and the runtime dispatch process.  
- **Demonstrate** safe upcasting and downcasting in polymorphic code.  
- **Evaluate** the impact of runtime polymorphism on common design patterns (e.g., Strategy, Factory, Observer).  
- **Design** and develop a C++ program that showcases polymorphic behavior using pointers and virtual functions.
