## Decision Making Iteration  

**Purpose**  
Decision‑making statements and iteration constructs are the backbone of any C++ program, enabling it to respond dynamically to input and process data repeatedly. Mastering these control‑flow mechanisms is essential for writing efficient, maintainable software across domains such as data analysis, embedded systems, game development, and algorithmic problem solving.  

**Learning Objectives**  
- Explain the role of selection statements (`if`, `switch`) and iteration constructs (`for`, `while`, `do‑while`) in C++.  
- Implement decision‑making statements to direct program flow based on conditional expressions.  
- Design and construct loops to handle repetitive tasks and traverse data structures.  
- Select appropriate loop types and condition checks to optimize execution time and resource usage.  
- Analyze the behavior of nested loops and identify potential performance bottlenecks.  
- Debug common loop‑related errors such as off‑by‑one mistakes, infinite loops, and unintended side effects.  
- Apply best practices for code readability, including consistent indentation and meaningful condition naming.  
- Evaluate the time‑complexity impact of various decision‑making and iteration patterns.
