### Purpose & Learning Objectives
Data types, variables, and operators are the building blocks of any C++ program, directly influencing how data is represented, stored, and manipulated in memory. Mastering these fundamentals enables efficient algorithm design and is essential for developing robust, high‑performance software in real‑world computer science applications.

- Explain the role of fundamental and user‑defined data types in C++.  
- Identify variable declaration, initialization, and scope rules.  
- Apply arithmetic, relational, logical, and bitwise operators to construct expressions.  
- Utilize type casting and conversion to ensure correct data handling.  
- Demonstrate the use of constants and enumerations for code readability.  
- Compare primitive data types and select appropriate types for given scenarios.  
- Implement operator precedence and associativity in complex expressions.  
- Analyze the impact of data type selection on memory usage and performance.
