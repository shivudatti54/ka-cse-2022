# Learning Objectives

After studying this topic, you should be able to:

1. Define and explain fundamental tree terminology including node, root, parent, child, sibling, leaf, ancestor, and descendant
2. Differentiate between height of a node, depth of a node, and height of a tree with precise definitions
3. Apply Theorem 1 (n nodes = n-1 edges) to solve tree property problems
4. Classify trees as full binary trees, complete binary trees, perfect binary trees, and balanced binary trees
5. Identify and count leaf nodes and internal nodes in a given tree structure
6. Calculate maximum possible nodes at any level in a binary tree
7. Determine whether a given binary tree satisfies the properties of complete, full, or perfect binary trees
8. Understand the relationship between number of nodes, edges, and tree height in various tree configurations
