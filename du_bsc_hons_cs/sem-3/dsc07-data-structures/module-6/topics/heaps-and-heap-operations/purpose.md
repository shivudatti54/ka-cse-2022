# Learning Objectives

After studying this topic, you should be able to:

1. Define and explain the heap data structure, including max-heap and min-heap variants
2. Identify and apply the heap property and complete binary tree requirements
3. Implement array representation of heaps and calculate parent and child indices
4. Perform heap operations including insertion, deletion, and heapify with correct time complexities
5. Build a heap from an unsorted array using the Build Heap algorithm
6. Apply Heap Sort algorithm to sort elements and analyze its time and space complexity
7. Compare heaps with other data structures like arrays and binary search trees for various operations
8. Evaluate the practical applications of heaps in priority queues and graph algorithms
