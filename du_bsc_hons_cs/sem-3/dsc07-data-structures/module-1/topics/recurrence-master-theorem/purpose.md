# Learning Objectives

After studying this topic, you should be able to:

1. Define recurrence relations and explain their role in algorithm analysis
2. Identify the parameters a, b, and f(n) in a divide-and-conquer recurrence of the form T(n) = aT(n/b) + f(n)
3. Solve recurrence relations using the substitution method (guess and verify)
4. Solve recurrence relations using the iteration method (unrolling)
5. Apply the Master Theorem to solve divide-and-conquer recurrences
6. Distinguish between the three cases of the Master Theorem and determine which case applies to a given recurrence
7. Analyze the time complexity of standard algorithms like Binary Search, Merge Sort, and Strassen's Matrix Multiplication
8. Recognize the limitations of the Master Theorem and identify when it cannot be applied
