# Recursive Functions - Summary

## Key Definitions and Concepts

*   **Recursion**: A function calls itself repeatedly until it reaches a base case.
*   **Base Case**: A condition that stops the recursion.
*   **Recursive Case**: A condition that requires the function to call itself to solve the problem.
*   **Direct Recursion**: A function calls itself directly.
*   **Indirect Recursion**: A function calls another function, which in turn calls the first function.

## Important Formulas and Theorems

*   **Factorial Function**: `n! = n * (n - 1)!`
*   **Fibonacci Series**: `F(n) = F(n - 1) + F(n - 2)`

## Key Points

*   Recursion is a powerful technique for solving problems.
*   A recursive function has a base case and a recursive case.
*   Direct recursion is when a function calls itself directly.
*   Indirect recursion is when a function calls another function, which in turn calls the first function.
*   Recursion can be less efficient than iteration due to the overhead of function calls and returns.
*   Recursion is useful for solving problems that can be broken down into smaller sub-problems of the same type.
*   Practice solving problems using recursion to improve your understanding.

## Common Mistakes to Avoid

*   **Infinite Recursion**: A function calls itself repeatedly without reaching a base case.
*   **Incorrect Base Case**: The base case is not correctly defined, leading to incorrect results.
*   **Incorrect Recursive Case**: The recursive case is not correctly defined, leading to incorrect results.
