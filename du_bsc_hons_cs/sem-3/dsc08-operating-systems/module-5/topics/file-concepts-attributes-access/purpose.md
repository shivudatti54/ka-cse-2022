# Learning Objectives

After studying this topic, you should be able to:

1. Define what a file is in the context of operating systems and explain its role as a fundamental abstraction for secondary storage.

2. Identify and describe the various attributes of a file, including name, identifier, type, location, size, timestamps, owner, and protection information.

3. Explain the sequential access method and its appropriate use cases, including how the file pointer advances during read and write operations.

4. Explain the direct (random) access method and demonstrate how to calculate the physical location of a record given its relative record number and record size.

5. Explain the indexed access method and describe how an index structure improves file access efficiency compared to sequential and direct access.

6. Compare the three file access methods in terms of efficiency, storage requirements, and suitable applications.

7. Understand the concept of a file directory and its role in organizing and managing files within a file system.
