### Purpose
Semantic networks, frames, and scripts are foundational knowledge‑representation paradigms that enable AI systems to model complex domains, support reasoning, and facilitate natural‑language understanding. Mastering these structures equips computer‑science graduates with practical tools to build intelligent applications such as expert systems, chatbots, and autonomous agents that require structured contextual knowledge.

### Learning Objectives
- **Explain** the core concepts and differences between semantic networks, frames, and scripts.  
- **Design** appropriate knowledge‑representation models for a given real‑world problem domain.  
- **Implement** semantic‑network or frame‑based systems using a programming language (e.g., Python, Prolog).  
- **Compare** the representational expressiveness, inference mechanisms, and computational trade‑offs of each paradigm.  
- **Analyze** case studies where these representations have been successfully applied in AI applications.  
- **Apply** reasoning algorithms (e.g., inheritance, constraint propagation) to derive implicit knowledge from explicit structures.  
- **Evaluate** the scalability and maintenance challenges of large‑scale knowledge bases in practical deployments.  
- **Create** a prototype script‑driven narrative system that demonstrates story flow and event sequencing.
