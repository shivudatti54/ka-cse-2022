# Learning Objectives

After studying this topic, you should be able to:

1. Define and explain the four components of a formal grammar: non-terminals, terminals, production rules, and start symbol.

2. Classify formal grammars into the four types of the Chomsky Hierarchy based on their production rule restrictions.

3. Distinguish between Type 0 (Unrestricted), Type 1 (Context-Sensitive), Type 2 (Context-Free), and Type 3 (Regular) grammars with appropriate examples.

4. Explain the correspondence between grammar types and automata models (Turing Machine, Linear Bounded Automaton, Pushdown Automaton, Finite Automaton).

5. Construct grammars for specific formal languages at different levels of the hierarchy.

6. Derive strings from given grammars and verify whether specific strings belong to languages generated by given grammars.

7. Convert context-free grammars to Chomsky Normal Form (CNF) and Greibach Normal Form (GNF).

8. Apply the concepts of Chomsky Hierarchy to practical applications in natural language processing and compiler design.
