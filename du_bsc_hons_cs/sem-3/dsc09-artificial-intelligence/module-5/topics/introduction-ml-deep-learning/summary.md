# Introduction to Machine Learning & Deep Learning

## Introduction

Machine Learning (ML) and Deep Learning (DL) are foundational pillars of modern Artificial Intelligence. As per the Delhi University NEP 2024 UGCF syllabus for BSc (Hons) Computer Science, this unit introduces students to the core concepts, algorithms, and applications that enable computers to learn from data and make intelligent decisions.

---

## Machine Learning Fundamentals

- **Definition**: ML is a subset of AI that enables systems to automatically learn and improve from experience without being explicitly programmed
- **Difference from Traditional Programming**: Traditional programming uses rules + data → output; ML uses data + output → rules
- **Types of Machine Learning**:
  - **Supervised Learning**: Learning from labeled data (Classification, Regression)
  - **Unsupervised Learning**: Learning from unlabeled data (Clustering, Dimensionality Reduction)
  - **Reinforcement Learning**: Learning through rewards and penalties (Agent-based learning)
- **Common Algorithms**:
  - Linear Regression, Logistic Regression
  - Decision Trees, Random Forests
  - K-Nearest Neighbors (KNN)
  - Support Vector Machines (SVM)
  - K-Means Clustering
- **Model Evaluation**: Accuracy, Precision, Recall, F1-Score, Confusion Matrix
- **Overfitting & Underfitting**: Balancing model complexity with generalization

---

## Deep Learning Fundamentals

- **Definition**: DL is a specialized subset of ML that uses artificial neural networks with multiple layers (deep architectures) to learn hierarchical representations of data
- **Key Difference from ML**: DL automatically extracts features, while traditional ML requires manual feature engineering
- **Neural Network Basics**:
  - **Neuron/Perceptron**: Basic computational unit
  - **Layers**: Input, Hidden, Output layers
  - **Activation Functions**: ReLU, Sigmoid, Tanh, Softmax
  - **Backpropagation**: Algorithm for training neural networks
- **Common Architectures**:
  - **Convolutional Neural Networks (CNN)**: Image processing, Computer Vision
  - **Recurrent Neural Networks (RNN)**: Sequential data, NLP
  - **Transformers**: Modern NLP, GPT, BERT models
- **Training Concepts**: Learning rate, epochs, batch size, optimizers (Adam, SGD)
- **Frameworks**: TensorFlow, PyTorch, Keras

---

## Applications

- Computer Vision (face recognition, object detection)
- Natural Language Processing (chatbots, translation)
- Healthcare (disease diagnosis)
- Autonomous vehicles
- Recommendation systems

---

## Conclusion

Machine Learning provides the foundation for intelligent systems, while Deep Learning pushes the boundaries of AI capabilities, particularly in handling complex unstructured data. For exam success, remember: ML learns from data using various algorithms, while DL uses neural networks with multiple layers to automatically learn features. Understanding when to apply each approach is crucial for solving real-world AI problems.
