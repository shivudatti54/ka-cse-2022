## Purpose  
Machine learning (ML) and deep learning (DL) are central to today’s intelligent technologies, powering applications such as image recognition, natural language processing, autonomous systems, and data‑driven decision making. Mastering the fundamentals of ML/DL equips BSc (Hons) Computer Science graduates with the expertise to design, implement, and evaluate AI‑based solutions that address real‑world challenges and meet industry demands.

## Learning Objectives  
- **Explain** the core concepts of machine learning and deep learning, including supervised, unsupervised, and reinforcement learning paradigms.  
- **Describe** the architecture and operation of key neural network models such as feed‑forward, convolutional, and recurrent networks.  
- **Implement** end‑to‑end machine learning pipelines using popular Python libraries (e.g., scikit‑learn, TensorFlow, PyTorch).  
- **Analyze** the effects of data preprocessing, feature engineering, and hyper‑parameter tuning on model performance.  
- **Compare** traditional ML algorithms with deep learning approaches in terms of data requirements, computational complexity, and scalability.  
- **Apply** appropriate evaluation metrics (accuracy, precision, recall, F1‑score, AUC‑ROC) to assess and benchmark model effectiveness.  
- **Evaluate** ethical considerations, bias mitigation strategies, and societal impacts of deploying AI systems.  
- **Design** a basic deep learning solution for a real‑world problem, integrating model selection, training, validation, and testing phases.
