## Purpose
Linear regression with one variable is a foundational supervised‑learning technique that enables computers to predict continuous outcomes based on historical data, forming the basis for more sophisticated predictive models. It equips computer scientists with the skills to uncover and quantify relationships between variables, supporting data‑driven decision making across domains such as finance, healthcare, and technology. Mastering this algorithm also prepares students to tackle regularization, feature engineering, and model‑evaluation concepts that underlie advanced machine‑learning methods.

## Learning Objectives
- **Explain** the conceptual framework of supervised learning and the role of one‑variable linear regression within it.  
- **Derive** the normal equation and understand the geometric interpretation of the least‑squares solution.  
- **Implement** a linear regression model from scratch using Python and NumPy (or a similar library).  
- **Evaluate** model performance using metrics such as Mean Squared Error (MSE), Root Mean Squared Error (RMSE), and the coefficient of determination (R²).  
- **Apply** linear regression to a real‑world dataset (e.g., house‑price prediction) and interpret the resulting coefficients.  
- **Analyze** the underlying assumptions of linear regression (linearity, independence, homoscedasticity, normality of residuals).  
- **Compare** one‑variable linear regression with alternative approaches (e.g., polynomial regression, decision‑tree regression) to appreciate the trade‑offs in predictive power and interpretability.
