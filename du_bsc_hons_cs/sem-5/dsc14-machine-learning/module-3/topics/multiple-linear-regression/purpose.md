# Learning Objectives

After studying this topic, you should be able to:

1. Explain the assumptions of Multiple Linear Regression and how to check them.
2. Apply the Ordinary Least Squares (OLS) method to estimate the coefficients and intercept of a Multiple Linear Regression model.
3. Interpret the coefficients and intercept of a Multiple Linear Regression model in the context of a problem.
4. Evaluate the goodness of fit of a Multiple Linear Regression model using R-squared.
5. Identify the limitations of Multiple Linear Regression, such as multicollinearity and non-linearity.
6. Apply techniques such as regularization to address multicollinearity in Multiple Linear Regression.
7. Consider alternative techniques, such as decision trees or random forests, if the data is non-linear.
8. Use Multiple Linear Regression to solve real-world problems, such as predicting house prices or stock prices.
