## Purpose

Support Vector Machine (SVM) is a powerful supervised learning algorithm that forms the backbone of many real-world computer science applications, from email spam detection and medical diagnosis to image recognition and handwriting analysis. Understanding SVM equips students with essential skills to solve complex classification and regression problems while developing strong foundations in optimization and statistical learning theory.

## Learning Objectives

Upon completion of this unit, students will be able to:

- **Explain** the geometric intuition and mathematical foundations of Support Vector Machines
- **Implement** SVM algorithms for both linear and non-linear classification problems
- **Apply** various kernel functions (linear, polynomial, RBF) to handle complex datasets
- **Optimize** SVM hyperparameters to improve model performance and generalization
- **Compare** SVM with other classification algorithms and select appropriate models for given problems
- **Utilize** SVM libraries and tools to build practical machine learning applications
- **Evaluate** model performance using appropriate metrics such as accuracy, precision, recall, and F1-score
