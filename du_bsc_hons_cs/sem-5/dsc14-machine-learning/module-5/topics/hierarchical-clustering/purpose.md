# Learning Objectives

After studying this topic, you should be able to:

1.  Explain the concept of hierarchical clustering and its types.
2.  Choose the appropriate distance metric and linkage criterion for a given problem.
3.  Create a dendrogram by hand for a small dataset.
4.  Interpret the results of hierarchical clustering, including identifying clusters and understanding the hierarchy.
5.  Implement hierarchical clustering using popular machine learning libraries such as scikit-learn.
6.  Compare and contrast hierarchical clustering with other clustering algorithms.
7.  Apply hierarchical clustering to real-world problems.
8.  Evaluate the strengths and weaknesses of hierarchical clustering.
