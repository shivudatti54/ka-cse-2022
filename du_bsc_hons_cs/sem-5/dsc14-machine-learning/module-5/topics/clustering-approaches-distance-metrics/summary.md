# Clustering Approaches and Distance Metrics - Summary

## Key Definitions and Concepts

* Hierarchical clustering: builds a hierarchy of clusters by merging or splitting existing clusters.
* Non-hierarchical clustering: partitions the data into a fixed number of clusters.
* Distance metric: measures the similarity between data points.
* Euclidean distance: measures the straight-line distance between two points in n-dimensional space.
* Manhattan distance: measures the sum of the absolute differences between corresponding coordinates.
* Minkowski distance: generalization of Euclidean and Manhattan distances.
* Cosine similarity: measures the cosine of the angle between two vectors.

## Important Formulas and Theorems

* Euclidean distance: d(x, y) = √(∑(x_i - y_i)^2)
* Manhattan distance: d(x, y) = ∑|x_i - y_i|
* Minkowski distance: d(x, y) = (∑|x_i - y_i|^p)^(1/p)
* Cosine similarity: sim(x, y) = (x · y) / (||x|| ||y||)

## Key Points

* Clustering is a type of unsupervised machine learning algorithm.
* Hierarchical clustering builds a hierarchy of clusters.
* Non-hierarchical clustering partitions the data into a fixed number of clusters.
* Distance metrics measure the similarity between data points.
* Choosing the right distance metric is crucial for clustering.
* Clustering has numerous applications in real-world problems.
* Clustering algorithms can be evaluated using various metrics.

## Common Mistakes to Avoid

* Confusing hierarchical and non-hierarchical clustering.
* Using the wrong distance metric for a given problem.
* Not normalizing the data before clustering.
* Not evaluating the performance of clustering algorithms.
