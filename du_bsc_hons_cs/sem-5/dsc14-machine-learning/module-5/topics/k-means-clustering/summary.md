# K-Means Clustering - Summary

## Key Definitions and Concepts

* K-Means clustering: a type of unsupervised machine learning algorithm used to group similar data points into clusters based on their features.
* Centroid: the mean of all data points assigned to a cluster.
* Euclidean distance: a measure of the distance between two data points.
* Elbow method: a method for choosing the number of clusters (K) by plotting the sum of squared errors (SSE) against different values of K.

## Important Formulas and Theorems

* K-Means algorithm: initialize centroids, assign data points to closest cluster, update centroids, repeat until convergence.
* Euclidean distance: d(x,y) = sqrt((x1-y1)^2 + (x2-y2)^2 + ... + (xn-yn)^2)
* Sum of squared errors (SSE): SSE = sum((xi-ci)^2) where xi is the ith data point and ci is the centroid of the cluster.

## Key Points

* K-Means clustering is a simple yet powerful algorithm for identifying clusters in data.
* The choice of K is a critical parameter in K-Means clustering.
* The elbow method is a common method for choosing K.
* K-Means clustering is sensitive to outliers and non-spherical clusters.
* K-Means clustering can be applied to real-world problems such as customer segmentation and image compression.
* The performance of K-Means clustering can be evaluated using metrics such as SSE and silhouette score.
* K-Means clustering can be implemented using a programming language such as Python or R.

## Common Mistakes to Avoid

* Not choosing the correct value of K.
* Not handling outliers and non-spherical clusters.
* Not evaluating the performance of K-Means clustering.
* Not implementing K-Means clustering correctly.
