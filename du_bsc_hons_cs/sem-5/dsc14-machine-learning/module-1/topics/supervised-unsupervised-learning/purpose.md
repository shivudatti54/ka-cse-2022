# Learning Objectives

After studying this topic, you should be able to:

1. Define machine learning and explain its role in enabling computers to learn from data without explicit programming.

2. Distinguish between supervised and unsupervised learning paradigms, identifying their key differences in terms of data requirements, learning approach, and output types.

3. Explain the two main categories of supervised learning—classification and regression—with clear definitions, examples, and appropriate use cases.

4. Describe the two main categories of unsupervised learning—clustering and dimensionality reduction—with practical applications for each.

5. Implement or explain the working of at least three supervised learning algorithms (Linear Regression, Logistic Regression, Decision Trees) and at least two unsupervised learning algorithms (K-Means, PCA).

6. Apply appropriate machine learning paradigms to real-world problems by analyzing problem requirements and determining whether supervised or unsupervised learning is suitable.

7. Evaluate machine learning models using appropriate metrics and understand concepts like overfitting, underfitting, and model generalization.

8. Identify common algorithms used in both paradigms and categorize them correctly based on their learning approach and problem type.
