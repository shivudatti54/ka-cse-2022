# Learning Objectives

After studying this topic, you should be able to:

1. Explain the single-source shortest path problem and its significance in graph theory
2. Describe how the Bellman-Ford algorithm handles negative weight edges
3. Implement the Bellman-Ford algorithm with correct initialization, relaxation, and iteration steps
4. Analyze the time and space complexity of the Bellman-Ford algorithm
5. Detect the presence of negative weight cycles in a graph
6. Compare and contrast Bellman-Ford algorithm with Dijkstra's algorithm
7. Apply the Bellman-Ford algorithm to solve real-world problems involving negative weights
8. Trace through the algorithm step-by-step for a given graph to determine shortest paths
