# Learning Objectives

After studying this topic, you should be able to:

1. Define and explain the components of a flow network including source, sink, and capacity function.

2. State and explain the three properties that a valid flow must satisfy: capacity constraint, skew symmetry, and flow conservation.

3. Construct and interpret the residual graph for a given flow, computing residual capacities for both forward and backward edges.

4. Identify augmenting paths in a residual graph and calculate their bottleneck capacities.

5. Apply the Ford-Fulkerson algorithm step-by-step to find the maximum flow in a given network.

6. State and prove the Max-Flow Min-Cut theorem and use it to verify that a flow is maximum.

7. Analyze the time and space complexity of the Ford-Fulkerson algorithm.

8. Apply maximum flow concepts to solve practical problems such as bipartite matching.
