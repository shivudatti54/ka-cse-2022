# Learning Objectives

After studying this topic, you should be able to:

1. Define polynomial time computation and explain its significance in algorithm analysis
2. Identify whether a given algorithm runs in polynomial time by examining its time complexity
3. Distinguish between tractable problems (solvable in polynomial time) and intractable problems (requiring super-polynomial time)
4. Explain the class P and provide examples of problems within this class
5. Analyze why exponential time algorithms become impractical for large inputs
6. Understand the concept of polynomial time reduction and its importance in complexity theory
7. Apply knowledge of polynomial time to make informed decisions about algorithm selection in practical scenarios
8. Explain the theoretical foundations (Church-Turing Thesis, Cook's Thesis) that justify polynomial time as the boundary of efficient computation
