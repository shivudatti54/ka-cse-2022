## Purpose

3D geometric transformations form the backbone of modern computer graphics, enabling the creation of realistic visualizations, animations, and interactive experiences across gaming, CAD, simulations, and VR/AR applications. This topic equips students with essential mathematical and programming skills to manipulate 3D objects in virtual space, preparing them for advanced work in graphics rendering pipelines and real-world application development.

## Learning Objectives

- **Define** the fundamental 2D and 3D transformation operations including translation, rotation, scaling, and shearing
- **Apply** homogeneous coordinates to represent and compute 3D transformations efficiently
- **Implement** transformation matrices for rotating objects around arbitrary axes in 3D space
- **Compose** multiple basic transformations into complex composite transformations using matrix multiplication
- **Analyze** the differences between orthographic and perspective projection techniques
- **Evaluate** the role of transformations in the graphics rendering pipeline from modeling to viewport display
- **Design** hierarchical 3D scenes using nested transformations for articulated models and character animation
