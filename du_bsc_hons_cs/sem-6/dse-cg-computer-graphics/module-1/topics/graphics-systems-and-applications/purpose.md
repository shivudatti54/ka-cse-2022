## Purpose

Computer graphics is fundamental to modern computing, powering everything from video games and animated films to medical imaging, virtual reality, and data visualization. Understanding graphics systems enables computer scientists to create immersive user experiences, simulate real-world phenomena, and develop applications across entertainment, engineering, science, and education.

## Learning Objectives

- Understand the fundamental concepts of 2D and 3D graphics systems and pipelines
- Apply mathematical transformations for graphics rendering and object manipulation
- Implement algorithms for line drawing, filling, clipping, and curve generation
- Analyze different lighting models, shading techniques, and texture mapping methods
- Design interactive graphics applications using industry-standard libraries
- Evaluate rendering techniques for performance and visual quality
- Create realistic visualizations using geometric modeling and scene graphs
- Demonstrate proficiency in graphics programming for real-world problem solving
