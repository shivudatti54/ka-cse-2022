# Input Output Devices
## Dse Cg Computer Graphics – BSc (Hons) Computer Science (DU NEP 2024)

### Introduction
Input and output devices are essential components of any computer graphics system. They facilitate the interaction between users and the computer, enabling the creation, manipulation, and display of graphical data. Understanding these devices is crucial for effective computer graphics applications.

### Input Devices for Graphics

**Pointing Devices**
- **Mouse**: Optical/mechanical device for cursor control and selecting graphical objects
- **Trackball**: Ball-based pointer device for precise positioning
- **Light Pen**: Direct pointing device that detects screen phosphor glow
- **Touch Panel**: Finger-sensitive screen for intuitive graphical interaction
- **Graphics Tablet (Digitizer)**: Converts hand-drawn sketches into digital coordinates

**Scanning Devices**
- **Scanner**: Converts printed images, photos, and documents into digital raster format
- **Digital Camera**: Captures visual data directly in digital form

**Keyboard**
- Alphanumeric input for commands and text-based graphical parameters

### Output Devices for Graphics

**Display Devices**
- **CRT (Cathode Ray Tube)**: Traditional display using electron beam scanning
- **LCD (Liquid Crystal Display)**: Flat-panel technology using liquid crystal cells
- **LED Display**: Light-emitting diode based displays
- **Plasma Display**: Gas plasma technology for large-format displays

**Hardcopy Devices**
- **Dot Matrix Printer**: Impact printing using pins (low resolution, cost-effective)
- **Inkjet Printer**: Sprays liquid ink droplets (good color reproduction)
- **Laser Printer**: Toner-based electrostatic printing (high quality, fast)
- **Plotter**: Vector output device for large-scale technical drawings (pen/plotters, electrostatic, inkjet plotters)

### Key Concepts

- **Refresh Rate**: Frequency of screen image redraw (measured in Hz)
- **Resolution**: Number of pixels per inch (dpi for print, pixels for display)
- **Color Depth**: Number of bits used to represent color (1-bit to 32-bit)
- **Interactivity Level**: Device's capability for real-time user interaction
- **Input/Output Coordination**: Proper handling of data flow between devices

### Conclusion
A thorough understanding of input output devices is fundamental to computer graphics. The selection of appropriate devices directly impacts the quality, efficiency, and versatility of graphical applications. For exam preparation, remember that graphics systems require both precise input mechanisms (for creating/modifying graphics) and high-quality output devices (for accurate visual representation).
