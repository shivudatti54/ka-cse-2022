### Purpose
Understanding Input / Output Devices is essential in computer graphics because these peripherals bridge human perception with digital computation, enabling interactive and immersive visual experiences. Mastery of device characteristics, performance metrics, and integration techniques empowers students to build responsive graphics applications ranging from real‑time games to scientific visualization and virtual reality.

### Learning Objectives
- **Identify** the major input devices used in graphics (e.g., mouse, graphics tablet, stylus, motion sensors, touchscreens).  
- **Describe** the operating principles and key specifications of output devices such as CRT/LCD/LED monitors, projectors, plotters, and VR headsets.  
- **Compare** device attributes like resolution, refresh rate, color depth, latency, and bandwidth to select appropriate hardware for given graphics tasks.  
- **Analyze** how input device precision and ergonomics influence user interaction and design of graphical user interfaces.  
- **Evaluate** the trade‑offs between cost, performance, and power consumption when choosing I/O peripherals for graphics systems.  
- **Design** interface pipelines that integrate multiple input and output devices with graphics APIs.  
- **Implement** driver or middleware code to configure and communicate with custom or specialized I/O hardware.  
- **Select** suitable I/O solutions for specific computer‑graphics applications (e.g., CAD, gaming, augmented reality) based on application requirements.
