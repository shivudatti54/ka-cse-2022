# Learning Objectives

After studying this topic, you should be able to:

1. Explain the fundamental architecture of Convolutional Neural Networks, including convolutional layers, pooling layers, and fully connected layers, with emphasis on their roles in hierarchical feature learning.

2. Describe the complete layer-by-layer architecture of LeNet-5, including input specifications, filter sizes, output dimensions, and the total number of trainable parameters.

3. Analyze AlexNet's architectural innovations including ReLU activation, dropout regularization, data augmentation, and GPU-based training that enabled its breakthrough performance.

4. Calculate output dimensions for convolutional and pooling layers using the appropriate formulas, and compute the number of trainable parameters in a given layer.

5. Compare and contrast LeNet-5 and AlexNet across multiple dimensions including scale, depth, activation functions, regularization techniques, and training methodologies.

6. Explain why AlexNet's success at ILSVRC-2012 marked a paradigm shift in computer vision and deep learning research.

7. Connect the foundational concepts in LeNet and AlexNet to modern CNN architectures and real-world applications including object detection, facial recognition, and medical imaging.
