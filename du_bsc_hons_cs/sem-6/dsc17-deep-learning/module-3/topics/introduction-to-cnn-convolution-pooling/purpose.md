# Learning Objectives

After studying this topic, you should be able to:

1. Explain the convolution operation mathematically and describe how it extracts features from images through kernel sliding.

2. Define and differentiate between key CNN parameters: kernel/filter, stride, padding, and receptive field.

3. Perform manual calculations for convolution and pooling operations to determine output feature map dimensions.

4. Compare max pooling and average pooling, explaining their respective advantages in feature extraction.

5. Analyze how stacking convolutional and pooling layers creates hierarchical feature representations in CNNs.

6. Design a basic CNN architecture by selecting appropriate kernel sizes, stride values, and padding strategies for a given image processing task.

7. Evaluate the computational and memory advantages of using convolution and pooling operations compared to fully-connected layers.

8. Apply CNN concepts to real-world computer vision applications including image classification and object detection.
