# Generative Adversarial Networks - Summary

## Key Definitions and Concepts

- **GAN (Generative Adversarial Network):** A deep learning framework where two neural networks (generator and discriminator) compete in a zero-sum game to learn the data distribution.
- **Generator (G):** Neural network that transforms random noise from latent space into synthetic data samples resembling training data.
- **Discriminator (D):** Binary classifier that distinguishes between real samples from training data and fake samples generated by G.
- **Latent Space:** Low-dimensional space (typically 100-dim) from which random noise is sampled to feed the generator.
- **Nash Equilibrium:** Training equilibrium where D(x) = 0.5 for all inputs—generator perfectly models distribution and discriminator is maximally confused.

## Important Formulas and Theorems

- **Minimax Objective:** min_G max_D V(D,G) = E_{x~p_data}[log D(x)] + E_{z~p_z}[log(1-D(G(z)))]
- Generator minimizes this (fool D), discriminator maximizes (correctly classify)
- At equilibrium: p_g = p_data (generator matches true distribution)

## Key Points

1. GANs are generative models that learn to approximate complex data distributions without explicit likelihood computation.
2. The adversarial training uses game theory—generator and discriminator are adversaries in a minimax optimization.
3. DCGAN uses deep convnets with batch norm, transposed convolutions for generation, and strided convolutions for discrimination.
4. Conditional GANs (cGAN) incorporate class labels for controlled, category-specific generation.
5. WGAN uses Wasserstein distance providing smoother gradients and better training stability.
6. Mode collapse occurs when generator produces limited variety, failing to capture full distribution diversity.
7. GAN training requires careful balance—discriminator shouldn't become too strong (vanishing gradients) or too weak.
8. Applications include image synthesis, data augmentation, art generation, super-resolution, and medical imaging.

## Common Mistakes to Avoid

- Confusing generator and discriminator objectives: Remember G minimizes while D maximizes the value function
- Thinking GANs are only for images: They work with any data type (audio, text, molecules)
- Assuming training converges easily: GANs are notoriously unstable; convergence is not guaranteed
- Overlooking mode collapse: This is the most common failure mode where generator produces only one type of output
