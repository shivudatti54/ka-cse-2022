# Learning Objectives

After studying this topic, you should be able to:

1. **Define confidentiality** in the context of information security and explain how symmetric encryption achieves it.

2. **Differentiate between classical ciphers (Caesar, Vigenère) and modern symmetric algorithms (DES, 3DES, AES)**, understanding the evolution of cryptographic techniques.

3. **Explain the structure and operation of AES**, including key sizes, block size, and the SPN (Substitution-Permutation Network) principles.

4. **Analyze different block cipher modes of operation (ECB, CBC, CTR, GCM)** with respect to security properties, efficiency, and appropriate use cases.

5. **Identify the fundamental key distribution problem** in symmetric cryptography and explain why it necessitates hybrid cryptographic systems.

6. **Evaluate security properties** of various encryption modes, including confidentiality, integrity, and authentication.

7. **Apply cryptographic concepts to real-world scenarios**, selecting appropriate encryption methods for different applications (TLS, disk encryption, database security).
