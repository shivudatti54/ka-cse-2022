# Learning Objectives

After studying this topic, you should be able to:

1. Explain the fundamental concepts of message authentication and its importance in secure communications

2. Define and differentiate between the three security properties of cryptographic hash functions: pre-image resistance, second pre-image resistance, and collision resistance

3. Describe how Message Authentication Codes (MACs) work and explain why they provide both integrity and authentication

4. Analyze the HMAC algorithm and explain how it combines hash functions with secret keys for secure authentication

5. Evaluate the Birthday Attack and its implications for hash function security, understanding why output length matters

6. Compare symmetric-key MACs with asymmetric-key digital signatures, understanding the concept of non-repudiation

7. Apply hash functions and MACs to solve practical problems like file integrity verification and message authentication
