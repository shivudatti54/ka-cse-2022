# Learning Objectives

After studying this topic, you should be able to:

1. Explain the CIA Triad (Confidentiality, Integrity, Availability) and its significance in information security
2. Apply the principle of least privilege to system design and access control scenarios
3. Analyze how defense in depth provides layered security through multiple complementary controls
4. Distinguish between least privilege, separation of duties, and least common mechanism
5. Evaluate system designs for fail-secure behavior versus fail-open vulnerabilities
6. Critically assess why security through obscurity fails and understand Kerckhoffs's principle
7. Design security architectures incorporating complete mediation and economy of mechanism
8. Balance security requirements with psychological acceptability in practical implementations
