# Malicious Software Types

## Introduction

Malicious software (malware) refers to any software designed to damage, disrupt, or gain unauthorized access to computer systems. In the context of Information Security, understanding malware types is crucial for implementing effective cybersecurity measures. This summary covers the key malware types as per the Delhi University BSc (H) Computer Science syllabus (NEP 2024 UGCF).

## Key Malware Types

### 1. **Virus**
- Self-replicating program that attaches itself to legitimate files
- Requires human action to spread (opening infected files)
- Examples: File viruses, boot sector viruses, macro viruses
- Damage: Data corruption, system slowdown, file deletion

### 2. **Worm**
- Self-replicating malware that spreads across networks without user intervention
- Exploits network vulnerabilities
- Examples: Morris Worm, WannaCry (also ransomware)
- Damage: Network congestion, data theft, distributed denial of service

### 3. **Trojan Horse**
- Disguises itself as legitimate software
- Creates backdoors for unauthorized access
- Does not self-replicate
- Examples: Remote Access Trojans (RATs), banker trojans
- Damage: Data theft, system compromise, privacy breach

### 4. **Ransomware**
- Encrypts victim files and demands payment for decryption
- Types: Encrypting and locker ransomware
- Examples: Petya, Locky, REvil
- Damage: Data loss, financial loss, operational disruption

### 5. **Spyware**
- Secretly monitors user activities
- Collects sensitive information without consent
- Types: System monitors, trojan spies, password spies
- Damage: Identity theft, privacy invasion, data harvesting

### 6. **Adware**
- Displays unwanted advertisements
- Often bundled with free software
- Tracks browsing habits for targeted ads
- Damage: System slowdown, privacy concerns, annoying pop-ups

### 7. **Rootkit**
- Hides its presence by modifying operating system
- Provides privileged access to attackers
- Difficult to detect and remove
- Damage: Persistent unauthorized access, system compromise

### 8. **Botnet**
- Network of compromised computers (bots/zombies)
- Controlled by a command-and-control server
- Used for DDoS attacks, spam distribution, cryptomining
- Damage: Large-scale attacks, resource exploitation

### 9. **Keylogger**
- Records keystrokes typed by users
- Captures passwords, credit card details, personal information
- Types: Hardware and software keyloggers
- Damage: Credential theft, identity fraud

## Conclusion

Understanding these malware types is essential for developing comprehensive security strategies. Organizations and individuals must implement multi-layered defenses including antivirus software, firewalls, regular updates, and user awareness training to protect against these evolving threats. For exam preparation, remember the distinguishing characteristics of each type—particularly the propagation method (virus vs. worm), intentional deception (trojan), and specific objectives (ransomware, spyware).
