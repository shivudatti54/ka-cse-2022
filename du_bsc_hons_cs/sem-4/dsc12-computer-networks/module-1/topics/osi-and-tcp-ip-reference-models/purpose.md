# Learning Objectives

After studying this topic, you should be able to:

1. Explain the need for layered network architecture and describe the benefits of the reference model approach to network design.

2. Identify and describe all seven layers of the OSI Reference Model, including their functions, protocols, and devices operating at each layer.

3. Compare and contrast the OSI model with the TCP/IP Reference Model, explaining how they map to each other.

4. Analyze the data encapsulation and de-encapsulation processes as data flows through the network layers.

5. Distinguish between the roles of TCP and UDP protocols in the Transport Layer and select appropriate protocols for different applications.

6. Explain the concept of Protocol Data Units (PDUs) and identify the PDU type at each layer of the OSI model.

7. Describe the functions of key networking devices (hub, switch, router) and identify the OSI layers at which they operate.

8. Apply knowledge of the OSI model to troubleshoot basic network communication problems by identifying which layer is affected.
