### Purpose
Understanding subnetting and routing algorithms is essential for designing scalable, efficient, and secure computer networks. This topic equips students with the practical skills to partition IP address spaces, optimize routing tables, and select appropriate algorithms for real‑world network implementations.

### Learning Objectives
- **Identify** the role of subnetting in modern IP networking and its impact on routing efficiency.  
- **Explain** the operation of key routing algorithms such as Distance Vector, Link State, and Classless Inter‑Domain Routing (CIDR).  
- **Compute** subnetwork addresses, host ranges, and subnet masks for given IPv4 and IPv6 prefixes.  
- **Design** a hierarchical subnetting plan that optimizes address utilization and reduces routing table size.  
- **Analyze** the trade‑offs between static and dynamic routing protocols in subnetted environments.  
- **Compare** the performance and scalability of routing algorithms in small‑scale versus large‑scale networks.
