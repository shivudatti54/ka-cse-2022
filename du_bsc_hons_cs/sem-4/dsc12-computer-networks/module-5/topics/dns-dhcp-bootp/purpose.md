# Learning Objectives

After studying this topic, you should be able to:

1. Explain DNS hierarchy and resolution process with TTL management
2. Configure DHCP server with address pools and lease policies
3. Differentiate BOOTP and DHCP in legacy vs modern networks
4. Analyze DNS packet structure using tools like dig/nslookup
5. Implement DNSSEC for secure DNS operations
6. Troubleshoot common DHCP failure scenarios
7. Design network boot solutions using BOOTP/TFTP
8. Compare IPv4 and IPv6 address allocation mechanisms
