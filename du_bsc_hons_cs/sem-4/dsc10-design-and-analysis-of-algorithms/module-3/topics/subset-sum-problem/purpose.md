# Learning Objectives

After studying this topic, you should be able to:

1. Define the Subset Sum Problem and its variants (decision, optimization, counting)

2. Explain the brute force approach and its exponential time complexity O(n × 2ⁿ)

3. Apply backtracking with pruning to improve practical performance

4. Implement the dynamic programming solution with O(n × T) time complexity

5. Optimize space complexity from O(n × T) to O(T) using 1D DP array

6. Analyze why DP solution is pseudo-polynomial and understand its limitations

7. Identify Subset Sum as an NP-complete problem and explain its significance

8. Distinguish Subset Sum from related problems like Partition and 0/1 Knapsack
