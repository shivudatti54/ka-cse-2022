# Learning Objectives

After studying this topic, you should be able to:

1. **Define** the 0/1 Integer Knapsack Problem and explain its constraints (binary choice per item, weight capacity limit).

2. **Formulate** the dynamic programming recurrence relation dp[i][w] for the knapsack problem with correct base cases.

3. **Apply** the dynamic programming algorithm to compute the maximum profit for given weights, profits, and knapsack capacity.

4. **Optimize** space complexity from O(n×W) to O(W) using a 1D array and explain why backward iteration is essential.

5. **Differentiate** between 0/1 Knapsack, Unbounded Knapsack, and Fractional Knapsack with respect to their solution approaches.

6. **Backtrack** through the DP table to identify which specific items were selected in the optimal solution.

7. **Analyze** the time and space complexity of the knapsack dynamic programming solution and explain why it is pseudo-polynomial.

8. **Evaluate** when to use dynamic programming versus greedy approaches based on problem constraints.
