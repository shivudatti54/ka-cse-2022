# Learning Objectives

After studying this topic, you should be able to:

1. Explain the difference between adjacency matrix and adjacency list representations, and choose appropriate representation based on graph density.

2. Implement Breadth-First Search (BFS) algorithm using a queue, and trace through its execution on a given graph.

3. Implement Depth-First Search (DFS) algorithm using both iterative and recursive approaches, and trace through its execution.

4. Analyze the time and space complexity of BFS and DFS algorithms.

5. Determine whether an undirected graph is connected and identify all connected components using traversal algorithms.

6. Distinguish between weak and strong connectivity in directed graphs.

7. Apply BFS to find the shortest path in unweighted graphs.

8. Identify real-world applications of graph traversals including network analysis, path finding, and cycle detection.
