# Learning Objectives

After studying this topic, you should be able to:

1. Define and explain the concept of a Minimum Spanning Tree (MST) in a weighted, undirected graph
2. State and prove the Cut Property and Cycle Property, which are fundamental to MST algorithms
3. Apply Kruskal's algorithm step-by-step to construct an MST from a given graph
4. Apply Prim's algorithm step-by-step to construct an MST from a given graph
5. Analyze and compare the time and space complexities of Kruskal's and Prim's algorithms
6. Select the appropriate MST algorithm based on graph characteristics (sparse vs. dense)
7. Identify real-world applications where MST concepts are used for cost optimization
8. Determine whether an MST is unique based on edge weight distinctness
