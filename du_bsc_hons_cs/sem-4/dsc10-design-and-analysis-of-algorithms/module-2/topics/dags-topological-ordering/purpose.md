# Learning Objectives

After studying this topic, you should be able to:

1. Define and identify a Directed Acyclic Graph (DAG) and explain its fundamental properties.

2. Explain the concept of topological ordering and understand why it exists only for acyclic graphs.

3. Apply Kahn's Algorithm (BFS-based approach) to compute topological orderings of given DAGs step-by-step.

4. Apply the DFS-based topological sort algorithm to compute valid orderings.

5. Detect cycles in directed graphs using topological sorting principles.

6. Analyze the time and space complexity of both topological sort algorithms.

7. Solve real-world problems modeled as DAGs, such as course prerequisite planning, build system dependencies, and task scheduling.

8. Verify the correctness of a topological ordering by checking that all directed edges respect the ordering.
