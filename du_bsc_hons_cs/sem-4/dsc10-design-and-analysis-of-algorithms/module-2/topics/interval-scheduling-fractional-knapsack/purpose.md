# Learning Objectives

After studying this topic, you should be able to:

1. Explain the Interval Scheduling Problem and describe its real-world applications in scheduling and resource allocation scenarios.

2. Apply the earliest-finish-time greedy algorithm to solve the Interval Scheduling Problem and determine the maximum number of non-overlapping intervals.

3. Analyze and prove why the greedy approach is optimal for Interval Scheduling using exchange arguments.

4. Define the Fractional Knapsack Problem and explain how it differs from the 0/1 Knapsack Problem in terms of constraint flexibility.

5. Solve the Fractional Knapsack Problem by calculating value-to-weight ratios, sorting items in descending order, and selecting items greedily.

6. Compare and contrast the greedy algorithms for Interval Scheduling and Fractional Knapsack, understanding why greedy works for both but fails for 0/1 Knapsack.

7. Compute time and space complexities of both algorithms and explain how sorting affects overall complexity.

8. Apply these algorithms to solve practical problems involving task scheduling, room allocation, and resource maximization with capacity constraints.
