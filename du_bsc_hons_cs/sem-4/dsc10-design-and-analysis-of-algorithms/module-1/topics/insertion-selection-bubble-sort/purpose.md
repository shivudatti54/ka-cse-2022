# Learning Objectives

After studying this topic, you should be able to:

1. Explain the working principle of Bubble Sort, Selection Sort, and Insertion Sort algorithms with clear step-by-step procedures.

2. Analyze and determine the time complexity (best, worst, and average cases) for each of the three sorting algorithms.

3. Trace through any given array using these sorting algorithms and show the state of the array after each iteration or pass.

4. Identify and explain why insertion sort is considered adaptive and how it performs efficiently on nearly sorted data.

5. Distinguish between stable and unstable sorting algorithms, identifying which of these three algorithms exhibit stability.

6. Compare the three algorithms based on criteria such as time complexity, space complexity, number of swaps, and practical use cases.

7. Select an appropriate sorting algorithm for specific scenarios based on data characteristics (e.g., nearly sorted data, small datasets, reverse sorted data).

8. Implement all three sorting algorithms in a programming language (Python/C++) and understand their in-place sorting nature.
