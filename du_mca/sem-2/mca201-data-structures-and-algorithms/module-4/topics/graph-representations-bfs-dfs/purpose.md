# Learning Objectives

After studying this topic, you should be able to:

1. Analyze time and space complexity of adjacency matrix vs list representations
2. Implement BFS and DFS algorithms from scratch
3. Determine appropriate graph representation for given problem constraints
4. Apply BFS to find shortest paths in unweighted graphs
5. Utilize DFS for topological sorting and cycle detection
6. Compare memory requirements of BFS and DFS implementations
7. Convert between different graph representations
8. Solve real-world problems like web crawling (BFS) and maze solving (DFS)
