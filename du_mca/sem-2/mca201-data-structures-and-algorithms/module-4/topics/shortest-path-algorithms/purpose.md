# Learning Objectives

After studying this topic, you should be able to:

1. Analyze and implement Dijkstra's algorithm for weighted graphs
2. Detect negative weight cycles using Bellman-Ford algorithm
3. Compute all-pairs shortest paths using Floyd-Warshall
4. Design admissible heuristics for A* search
5. Compare time and space complexities of different algorithms
6. Apply pathfinding algorithms to network optimization problems
7. Handle edge cases like negative weights and disconnected graphs
8. Prove algorithm correctness using mathematical induction
