# Learning Objectives

After studying this topic, you should be able to:

1. Define recurrence relations and explain their role in algorithm analysis
2. Identify the components a, b, and f(n) in divide-and-conquer recurrences of the form T(n) = aT(n/b) + f(n)
3. Apply the Master Theorem to determine asymptotic time complexity for divide-and-conquer recurrences
4. Distinguish between the three cases of the Master Theorem and correctly identify which case applies to a given recurrence
5. Verify the regularity condition when applying Case 3 of the Master Theorem
6. Solve recurrence relations using the substitution method as an alternative to Master Theorem
7. Analyze common algorithms including binary search, merge sort, and Strassen's matrix multiplication using recurrence relations
8. Apply recursion tree method to verify and understand Master Theorem results
