# Learning Objectives

After studying this topic, you should be able to:

1. Differentiate between process and program states
2. Analyze various CPU scheduling algorithms using Gantt charts
3. Calculate performance metrics: waiting time, turnaround time, throughput
4. Compare preemptive vs non-preemptive scheduling strategies
5. Design appropriate scheduling solutions for given system requirements
6. Explain the role of Process Control Block in context switching
7. Evaluate scheduling algorithms for real-world scenarios like web servers
8. Implement basic scheduling algorithms using queue data structures
