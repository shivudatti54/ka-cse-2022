# Learning Objectives

After studying this topic, you should be able to:

1. Formally define functional dependencies and their role in database design
2. Apply Armstrong's axioms to derive implied dependencies
3. Compute attribute closures and FD set closures
4. Identify candidate keys and superkeys in relational schemas
5. Calculate minimal covers of functional dependency sets
6. Apply FD concepts to normalize relations to 3NF and BCNF
7. Detect and resolve FD violations in real-world database schemas
8. Analyze trade-offs between normalization levels in practical scenarios
