# Learning Objectives

After studying this topic, you should be able to:

1. Explain the CIA triad and its implementation in enterprise systems
2. Classify threats as passive/active and internal/external
3. Analyze attack vectors using STRIDE model
4. Design mitigation strategies for SQLi and XSS attacks
5. Evaluate security controls using NIST Risk Management Framework
6. Apply cryptographic techniques to maintain data integrity
7. Develop incident response plans for DDoS scenarios
8. Interpret security breaches using attack trees
