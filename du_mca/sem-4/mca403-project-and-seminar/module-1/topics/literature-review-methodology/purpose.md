# Learning Objectives

After studying this topic, you should be able to:

1. Differentiate between systematic, scoping, and narrative literature reviews
2. Design effective search strategies using Boolean logic and database-specific syntax
3. Apply PRISMA guidelines to document review processes
4. Synthesize technical findings using thematic analysis and comparative matrices
5. Evaluate study quality using domain-specific appraisal tools (e.g., AMSTAR for ML papers)
6. Utilize reference management software for citation organization
7. Present literature reviews with appropriate visualization tools (flow diagrams, heat maps)
8. Identify ethical considerations in literature synthesis (citation bias, predatory journals)
