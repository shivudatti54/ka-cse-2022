# Literature Review Methodology - Summary

## Key Definitions and Concepts
- **PRISMA**: Evidence-based minimum set of items for systematic reviews
- **Grey Literature**: Non-peer-reviewed sources (technical reports, patents)
- **Snowballing**: Citation tracking method to find related papers
- **MeSH Terms**: Controlled vocabulary for PubMed searches

## Important Formulas and Theorems
- **Cohen's Kappa**: κ = (Pₐ - Pₑ)/(1 - Pₑ) → Measures inter-rater reliability during screening
- **Recall-Precision Tradeoff**: F₁ = 2*(Precision*Recall)/(Precision+Recall) for search strategy optimization

## Key Points
- Systematic reviews require protocol registration (PROSPERO)
- Scoping reviews answer "What evidence exists?" rather than "What is effective?"
- Use PICOS framework: Population, Intervention, Comparison, Outcomes, Study design
- Preferred Reporting Items for Systematic Reviews and Meta-Analyses (PRISMA) has 27-item checklist
- Zotero's CSL editor allows DU-specific citation styles
- Temporal analysis crucial for fast-moving fields (e.g., comparing GPT-3 vs GPT-4 papers)
- Enterprise tools: LexisNexis for legal tech reviews, Crunchbase for startup trends

## Common Mistakes to Avoid
- Using only Google Scholar instead of subject-specific databases
- Failing to document exclusion reasons during paper screening
- Overlooking negative results that contradict hypothesis
- Not updating search date cutoff (e.g., missing 2023 papers in 2024 review)

## Revision Tips
1. Create a PRISMA flow diagram for your semester project
2. Practice writing search strings with Scopus field codes (TITLE-ABS-KEY())
3. Use DU's e-resources portal to access ACM Digital Library
4. Memorize 3 key differences between narrative vs systematic reviews
