# Learning Objectives

After studying this topic, you should be able to:

1. Differentiate between combinational and sequential circuits
2. Analyze and design SR, JK, D, and T flip-flop circuits
3. Implement synchronous counters and shift registers using flip-flops
4. Calculate setup time, hold time, and clock frequency limitations
5. Design finite state machines using state diagrams and excitation tables
6. Resolve metastability issues in cross-clock domain communication
7. Optimize sequential circuits for power and performance
8. Apply flip-flop concepts to real-world memory systems and interface controllers
