# Learning Objectives

After studying this topic, you should be able to:

1. Construct and analyze truth tables for complex logical expressions
2. Prove logical equivalences using algebraic laws and truth tables
3. Translate natural language statements into formal propositional/predicate logic
4. Apply inference rules (Modus Ponens, Universal Instantiation) in formal proofs
5. Analyze nested quantifiers and their relationships
6. Identify tautologies, contradictions, and contingencies
7. Resolve predicate logic statements with multiple variables
8. Apply logical principles to real-world CS problems (circuit design, query optimization)
