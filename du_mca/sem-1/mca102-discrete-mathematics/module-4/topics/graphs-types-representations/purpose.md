# Learning Objectives

After studying this topic, you should be able to:

1. Classify graphs into major types (directed, weighted, cyclic, etc.)
2. Convert between adjacency matrix, adjacency list, and edge list representations
3. Analyze time/space complexity of graph operations for different representations
4. Identify real-world applications of specific graph types (e.g., DAGs in task scheduling)
5. Implement BFS/DFS using appropriate graph representations
6. Prove basic graph theorems (e.g., handshaking lemma)
7. Compare memory efficiency of different representations for sparse/dense graphs
8. Recognize bipartite graphs and trees through structural properties
