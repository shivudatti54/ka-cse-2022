# Learning Objectives

After studying this topic, you should be able to:

1. Analyze time and space complexity of fundamental graph algorithms
2. Apply BFS/DFS to solve connectivity and traversal problems
3. Implement Dijkstra's algorithm for optimal routing in weighted networks
4. Construct Minimum Spanning Trees using Kruskal's and Prim's algorithms
5. Perform topological sorting on dependency graphs
6. Compare and contrast different graph algorithms for specific use cases
7. Translate real-world problems into graph representations
8. Identify when to use adjacency matrix vs list representations
