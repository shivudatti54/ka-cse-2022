# Learning Objectives

After studying this topic, you should be able to:

1. Classify functions as injective, surjective, bijective, or none
2. Construct function compositions and verify their properties
3. Determine inverse functions for bijective mappings
4. Apply function concepts to real-world CS problems like hashing and encryption
5. Prove/disprove function properties using formal mathematical reasoning
6. Analyze computational processes through functional composition
7. Implement characteristic functions for set operations
8. Relate function theory to database systems and algorithm design
