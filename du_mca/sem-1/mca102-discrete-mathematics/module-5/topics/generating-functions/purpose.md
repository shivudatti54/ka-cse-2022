# Learning Objectives

After studying this topic, you should be able to:

1. Define ordinary generating functions (OGF) and exponential generating functions (EGF), and explain the key differences between them with respect to combinatorial interpretations.

2. Construct generating functions for given sequences and interpret the coefficients of generating functions in the context of counting problems.

3. Apply operations on generating functions—addition, multiplication (convolution), shifting, and differentiation—to solve combinatorial problems.

4. Solve linear recurrence relations with constant coefficients using the generating function method, including both homogeneous and non-homogeneous cases.

5. Apply generating functions to solve integer partition problems and restricted composition problems (such as coin change problems).

6. Use partial fraction decomposition to extract coefficients from rational generating functions.

7. Distinguish between scenarios requiring OGF versus EGF and apply the appropriate generating function technique.
