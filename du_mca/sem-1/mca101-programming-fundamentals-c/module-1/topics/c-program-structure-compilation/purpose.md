# Learning Objectives

After studying this topic, you should be able to:

1. Analyze and explain the structure of a standard C program
2. Differentiate between preprocessing, compilation, assembly, and linking stages
3. Use GCC flags to control compilation phases (-E, -S, -c)
4. Diagnose errors occurring at different build stages
5. Implement multi-file programs with proper header inclusion
6. Explain memory layout of C programs during execution
7. Compare static vs dynamic linking trade-offs
8. Apply preprocessor directives for conditional compilation
