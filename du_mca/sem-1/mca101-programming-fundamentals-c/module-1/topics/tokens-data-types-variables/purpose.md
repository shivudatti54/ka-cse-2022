## Purpose
Understanding tokens, data types, and variables is essential because they form the structural foundation of every C program, directly affecting readability, memory usage, and correctness. Mastering these concepts enables developers to write efficient, bug‑free code that can be ported across platforms, which is a core requirement in real‑world software development.

## Learning Objectives
- **Identify** the different token categories (identifiers, keywords, constants, operators, punctuation) in C.  
- **Distinguish** among the primary data types (int, float, char, void) and their memory representations.  
- **Apply** appropriate data types to declare variables that match the problem domain’s requirements.  
- **Write** statements that correctly initialize and use variables, observing scope and lifetime rules.  
- **Analyze** the impact of type conversions and arithmetic promotions on expression evaluation.  
- **Demonstrate** the use of qualifiers (const, volatile) and storage class specifiers (auto, static, extern) in variable declarations.
