# Learning Objectives

After studying this topic, you should be able to:

1. Differentiate between 6 key requirements elicitation techniques
2. Apply MoSCoW prioritization to Indian e-governance projects
3. Create UML use case diagrams with <<include>> and <<extend>> relationships
4. Develop traceability matrices for CMMI Level 3 compliance
5. Conduct requirement reviews using Fagan inspection methodology
6. Analyze stakeholder conflicts using Win-Win negotiation approach
7. Design CRUD matrices for inventory management systems
8. Validate requirements using prototyping tools like Balsamiq
