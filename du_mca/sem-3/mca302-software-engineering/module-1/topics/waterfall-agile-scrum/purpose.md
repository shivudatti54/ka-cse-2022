# Learning Objectives

After studying this topic, you should be able to:

1. Differentiate between predictive (Waterfall) and adaptive (Agile) SDLC models
2. Explain the four values and twelve principles of the Agile Manifesto
3. Design a Scrum process flow with all roles, artifacts, and events
4. Analyze project scenarios to recommend suitable methodologies
5. Calculate Scrum metrics: Velocity, Burn-down Rate, Capacity Planning
6. Critique the limitations of Waterfall in modern software projects
7. Implement hybrid models for regulated environments (healthcare, banking)
8. Apply Change Management techniques in Waterfall vs Agile contexts
