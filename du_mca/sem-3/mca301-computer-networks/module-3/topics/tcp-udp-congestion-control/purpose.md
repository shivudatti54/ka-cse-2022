# Learning Objectives

After studying this topic, you should be able to:

1. Differentiate between TCP and UDP protocol architectures
2. Explain TCP's congestion control algorithms (Slow Start, AIMD, Fast Recovery)
3. Analyze throughput using Jacobson's RTT estimation formula
4. Design appropriate protocol selection strategies for given application requirements
5. Troubleshoot network issues using Wireshark traces of TCP/UDP headers
6. Evaluate performance of Cubic vs BBR congestion control in high-speed networks
7. Implement basic socket programming using both TCP and UDP
8. Apply QoS mechanisms for UDP-based real-time traffic in enterprise networks
