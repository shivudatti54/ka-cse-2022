# Physical Layer Transmission Media - Summary

## Key Definitions and Concepts
- **Guided Media**: Physical path (copper/fiber)
- **Unguided Media**: Wireless transmission
- **Attenuation**: Signal loss (dB/km)
- **Bandwidth**: Max data rate (Hz)
- **Modal Dispersion**: Light paths in MMF

## Important Formulas and Theorems
- **Shannon Capacity**: C = B log₂(1 + SNR)
- **FSPL**: 20 log(d) + 20 log(f) + 92.45 (dB)
- **Attenuation Coefficient**: α = 10 log(P₁/P₂)
- **Fresnel Zone**: r = 8.657√(λd) (60% zone)

## Key Points
- UTP Cat 6A: 10 Gbps @ 100m
- SMF uses laser, MMF uses LED
- 5G FR1 (<6 GHz), FR2 (mmWave)
- Coaxial types: RG-6 (TV), RG-58 (Ethernet)
- Satellite latency: GEO 270ms, LEO 20-40ms
- dB math: 3 dB = 2× power, 10 dB = 10×
- WDM: C-band (1530-1565 nm)

## Common Mistakes to Avoid
- Confusing SMF/MMF core diameters
- Ignoring connector losses in fiber links
- Using FSPL formula for non-free space
- Mixing optical/radio frequency ranges

## Revision Tips
1. Create comparison tables: Media vs Bandwidth vs Distance
2. Practice dB calculations (addition, not multiplication)
3. Memorize IEEE standards: 802.3 (Ethernet), 802.11 (Wi-Fi)
4. Study recent trends: FTTH (Fiber to the Home), mmWave 5G
