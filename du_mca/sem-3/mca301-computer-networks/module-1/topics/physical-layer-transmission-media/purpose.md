# Learning Objectives

After studying this topic, you should be able to:

1. Classify transmission media into guided/unguided categories
2. Explain attenuation, dispersion, and bandwidth characteristics
3. Analyze scenarios to select appropriate media (cost vs performance)
4. Calculate link budgets using dB arithmetic and FSPL formula
5. Compare IEEE standards for copper (802.3) and wireless (802.11)
6. Describe fiber optic multiplexing techniques (WDM, TDM)
7. Evaluate EMI/RFI impacts on different media types
8. Design physical layer solutions for enterprise networks
