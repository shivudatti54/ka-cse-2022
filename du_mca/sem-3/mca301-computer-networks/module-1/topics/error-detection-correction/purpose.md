# Learning Objectives

After studying this topic, you should be able to:

1. Differentiate between error detection and correction mechanisms
2. Design single parity check and 2D parity codes for given data
3. Compute CRC codes using polynomial division
4. Implement Hamming code encoding/decoding processes
5. Analyze checksum effectiveness in IP protocols
6. Compare various ARQ protocols' efficiency
7. Calculate Hamming distance for error detection capability
8. Evaluate error control methods for different network scenarios
