# Learning Objectives

After studying this topic, you should be able to:

1. Design subnets using CIDR and VLSM for given host requirements
2. Convert between binary, decimal, and CIDR notation formats
3. Differentiate between classful and classless addressing schemes
4. Implement IPv6 address compression and expansion
5. Troubleshoot common IP configuration errors
6. Calculate valid host ranges considering network/broadcast addresses
7. Compare IPv4 and IPv6 addressing architectures
8. Apply subnetting strategies in enterprise network scenarios
