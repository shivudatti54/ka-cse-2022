# Learning Objectives

After studying this topic, you should be able to:

1. Analyze and compare time/message complexity of distance vector vs link state algorithms
2. Design OSPF area hierarchies for enterprise networks
3. Evaluate BGP route selection policies in multi-homed AS configurations
4. Configure routing tables using Dijkstra's and Bellman-Ford algorithms
5. Diagnose convergence issues in RIP networks using update sequence diagrams
6. Implement basic SDN routing policies using OpenFlow controllers
7. Troubleshoot common BGP peering problems in ISP environments
8. Apply traffic engineering concepts to MPLS-based routing scenarios
