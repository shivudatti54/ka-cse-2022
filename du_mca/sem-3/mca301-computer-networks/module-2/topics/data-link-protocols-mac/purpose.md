# Learning Objectives

After studying this topic, you should be able to:

1. Differentiate between channel partitioning, random access, and controlled MAC protocols
2. Calculate throughput for ALOHA variants and CSMA systems
3. Explain CSMA/CD operation with binary exponential backoff
4. Design appropriate MAC strategies for given network scenarios
5. Analyze tradeoffs between TDMA, FDMA, and CDMA
6. Describe modern MAC enhancements in 802.11ax and 5G networks
7. Solve propagation delay-related problems in Ethernet
8. Compare token passing vs polling mechanisms
