# Learning Objectives

After studying this topic, you should be able to:

1. Formulate the mathematical foundations of PCA including covariance matrices and eigen decomposition
2. Implement PCA from scratch using Python and NumPy
3. Determine optimal number of principal components using scree plots and cumulative variance
4. Apply PCA for data visualization and noise reduction in real datasets
5. Compare and contrast PCA with other dimensionality reduction techniques (LDA, t-SNE)
6. Analyze trade-offs between information loss and dimensionality reduction
7. Utilize scikit-learn's PCA for practical machine learning pipelines
8. Interpret principal components in terms of original features
