# Learning Objectives

After studying this topic, you should be able to:

1. Formulate SVM optimization problems in both primal and dual forms
2. Implement the kernel trick to transform non-linear problems into linear ones
3. Analyze and compare different kernel functions (Linear, Polynomial, RBF)
4. Apply Mercer's theorem to validate kernel functions
5. Design SVM-based solutions for real-world classification tasks
6. Tune hyperparameters (C, γ, degree) using cross-validation
7. Interpret support vectors and decision boundaries in feature space
8. Extend SVMs for multi-class classification using one-vs-one/one-vs-rest
