# Learning Objectives

After studying this topic, you should be able to:

1. Differentiate between supervised, unsupervised, and reinforcement learning paradigms
2. Implement linear regression and logistic regression with gradient descent
3. Evaluate clustering quality using silhouette score and elbow method
4. Design Q-learning algorithms for Markov Decision Processes
5. Apply PCA for dimensionality reduction in high-dimensional datasets
6. Compare parametric vs non-parametric learning approaches
7. Diagnose overfitting/underfitting using learning curves
8. Implement epsilon-greedy exploration strategies in RL
