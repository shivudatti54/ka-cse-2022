# Learning Objectives

After studying this topic, you should be able to:

1. Select appropriate evaluation metrics for different ML tasks
2. Calculate and interpret classification metrics from confusion matrices
3. Compute regression metrics and analyze model performance
4. Explain the tradeoff between precision and recall
5. Interpret ROC curves and AUC scores
6. Apply probabilistic metrics like log-loss for model comparison
7. Choose metrics aligned with business objectives
8. Perform statistical significance tests for model comparisons
