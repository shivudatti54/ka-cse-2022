# Learning Objectives

After studying this topic, you should be able to:

1. Define semantic role labeling and its position in the NLP pipeline
2. Contrast PropBank and FrameNet annotation frameworks
3. Implement a Bi-LSTM-CRF model for SRL using PyTorch
4. Evaluate SRL systems using CoNLL-2005 metrics
5. Adapt transformer models (BERT, mBERT) for multilingual SRL
6. Analyze challenges in code-mixed SRL for Indian languages
7. Critique recent advances in few-shot/zero-shot SRL approaches
8. Design an SRL-based feature extractor for downstream tasks
