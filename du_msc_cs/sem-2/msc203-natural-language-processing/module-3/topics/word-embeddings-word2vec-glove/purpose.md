# Learning Objectives

After studying this topic, you should be able to:

1. Explain the mathematical foundations of Word2Vec and GloVe
2. Compare and contrast skip-gram, CBOW, and GloVe architectures
3. Implement word embedding models using negative sampling
4. Analyze embedding spaces using vector arithmetic and dimensionality reduction
5. Evaluate embeddings using intrinsic and extrinsic evaluation methods
6. Discuss limitations and biases in traditional embedding approaches
7. Connect classical embeddings to modern contextual representation models
8. Design appropriate preprocessing pipelines for embedding training
