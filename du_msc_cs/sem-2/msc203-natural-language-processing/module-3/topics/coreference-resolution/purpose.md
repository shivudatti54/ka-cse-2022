# Learning Objectives

After studying this topic, you should be able to:

1. Define coreference resolution and its role in discourse analysis
2. Differentiate between anaphoric, cataphoric, and bridging references
3. Implement mention detection using syntactic and semantic features
4. Evaluate clustering algorithms using MUC, B³, and CEAF metrics
5. Analyze challenges in cross-document and multilingual coreference
6. Compare transformer-based architectures (CorefBERT vs SpanBERT)
7. Design a coreference system for low-resource languages
8. Critique current limitations in handling implicit references
