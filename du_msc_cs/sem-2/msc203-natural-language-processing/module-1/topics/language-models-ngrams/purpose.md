# Learning Objectives

After studying this topic, you should be able to:

1. Construct n-gram language models using maximum likelihood estimation
2. Apply various smoothing techniques to handle sparse data
3. Evaluate language model performance using perplexity and cross-entropy
4. Analyze the computational complexity of n-gram models
5. Compare different smoothing methods in practical scenarios
6. Explain how n-gram models handle out-of-vocabulary words
7. Discuss the evolution from statistical to neural language models
8. Implement interpolation techniques for combining n-gram models
