# Learning Objectives

After studying this topic, you should be able to:

1. Explain the architectural components of IPSec including SAs, AH, and ESP
2. Compare and contrast Transport Mode vs Tunnel Mode implementations
3. Analyze IKEv2's four-message exchange protocol and its security properties
4. Design a site-to-site VPN with proper cryptographic algorithm selection
5. Evaluate the role of Diffie-Hellman groups in forward secrecy
6. Assess vulnerabilities in legacy IPSec implementations (e.g., MD5, DES)
7. Discuss current research challenges in post-quantum IPSec migrations
8. Implement NAT-Traversal techniques using Wireshark packet analysis
