# Learning Objectives

After studying this topic, you should be able to:

1. Explain the mathematical foundations of RSA and ECC
2. Perform RSA key generation, encryption, and decryption
3. Implement elliptic curve point addition and scalar multiplication
4. Analyze security strengths of different key sizes in RSA vs ECC
5. Evaluate vulnerabilities in practical implementations of both systems
6. Compare computational complexity of factoring vs ECDLP
7. Discuss post-quantum implications for public-key cryptography
8. Design secure parameter choices for modern cryptographic systems
