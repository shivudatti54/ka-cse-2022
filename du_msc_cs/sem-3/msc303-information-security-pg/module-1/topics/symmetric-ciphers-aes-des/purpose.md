# Learning Objectives

After studying this topic, you should be able to:

1. Analyze the Feistel network structure in DES and SPN in AES
2. Perform step-by-step encryption for one DES round and AES transformation phases
3. Compare security strengths and vulnerabilities of AES vs DES
4. Implement key expansion algorithms for AES-128 and DES
5. Evaluate resistance against differential/linear cryptanalysis in both ciphers
6. Apply cipher modes (CBC, CTR, GCM) in real-world protocols
7. Critically assess post-quantum implications for symmetric cryptography
8. Design optimized S-boxes using criteria from AES/DES research
