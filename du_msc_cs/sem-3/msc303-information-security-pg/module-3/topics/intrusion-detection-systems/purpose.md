# Learning Objectives

After studying this topic, you should be able to:

1. Differentiate between host-based and network-based IDS architectures
2. Design Snort rules for specific attack patterns
3. Implement entropy-based anomaly detection algorithms
4. Evaluate IDS performance using ROC curve analysis
5. Analyze modern evasion techniques against signature-based IDS
6. Propose hybrid IDS architectures combining ML and signature approaches
7. Critique current research challenges in IoT intrusion detection
8. Apply stateful protocol analysis to detect HTTP request smuggling
