# Learning Objectives

After studying this topic, you should be able to:

1. Analyze the limitations of physical clocks in distributed systems
2. Compare and contrast Lamport and vector clocks in capturing causal relationships
3. Implement Cristian's and Berkeley synchronization algorithms mathematically
4. Evaluate NTP's hierarchical structure and clock offset/delay calculations
5. Explain how Google TrueTime achieves external consistency in Spanner
6. Design a Byzantine fault-tolerant clock synchronization protocol
7. Apply happens-before relations to determine event ordering in distributed traces
8. Critically assess recent advances in hybrid logical-physical clocks
