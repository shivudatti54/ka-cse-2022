# Learning Objectives

After studying this topic, you should be able to:

1. Analyze the need for model interpretability in real-world AI systems
2. Explain LIME's methodology for local surrogate modeling
3. Derive Shapley values using cooperative game theory principles
4. Contrast axiomatic properties of SHAP vs heuristic nature of LIME
5. Implement both methods for different data modalities (tabular, image, text)
6. Evaluate attribution methods using quantitative metrics (infidelity, sensitivity)
7. Critique current limitations in feature attribution research
8. Design XAI pipelines compliant with emerging AI regulations
