# Learning Objectives

After studying this topic, you should be able to:

1. Formally define MDPs using the 5-tuple (S, A, P, R, γ)
2. Derive both Bellman expectation and optimality equations from first principles
3. Implement value iteration and policy iteration algorithms
4. Analyze convergence properties using contraction mapping arguments
5. Solve finite-state MDPs with stochastic transitions
6. Connect classical dynamic programming to modern deep reinforcement learning
7. Critically evaluate recent research papers using MDP extensions (e.g., POMDPs, RMDPs)
8. Design reward functions for real-world problems considering ethical implications
