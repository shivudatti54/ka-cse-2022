# Experimental Design - Summary

## Key Definitions and Concepts
- **Factorial Design**: Varying multiple factors simultaneously
- **Confounding**: Uncontrolled variables affecting results
- **Power Analysis**: Probability of detecting true effects
- **Counterbalancing**: Mitigating order effects in trials

## Important Formulas and Theorems
- Power = 1 - β = P(reject H₀ | H₁ true)
- Bonferroni Correction: α' = α/m (m=comparisons)
- Central Composite Design: 2^k + 2k + center points
- ANOVA: F = (Between-group variance)/(Within-group variance)

## Key Points
- Randomization reduces selection bias
- Blocking controls for known nuisance factors
- Repeated measures increase statistical power
- Full factorial designs capture interaction effects
- Bayesian designs enable sequential updating
- Ethical review required for human subjects
- Pre-registration prevents p-hacking

## Common Mistakes to Avoid
- Ignoring temporal effects in longitudinal studies
- Using arbitrary significance thresholds (e.g., p=0.05)
- Overlooking multiple comparison corrections
- Confusing statistical vs practical significance

## Revision Tips
1. Practice creating design matrices for 2^3 factorial experiments
2. Memorize Type I/II error decision matrix
3. Use G*Power software for sample size calculations
4. Analyze case studies from ACM SIGMETRICS papers
