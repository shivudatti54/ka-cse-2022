# Technical Writing & Thesis Development - Summary

## Key Definitions and Concepts
- IMRaD: Introduction-Methodology-Results-Discussion structure
- FID Score: Fréchet Inception Distance for GAN evaluation
- FedAvg: Federated Averaging in distributed learning
- PRISMA: Preferred Reporting Items for Systematic Reviews

## Important Formulas and Theorems
- **Statistical Significance**: p < 0.05 (95% confidence interval)
- **F1-Score**: 2*(Precision*Recall)/(Precision+Recall)
- **LaTeX Theorem Environment**:
  \begin{theorem}[Turing Completeness]
  Any GAN architecture with... 
  \end{theorem}

## Key Points
- Thesis introduction must clearly state research gap using recent citations
- Methodology sections require reproducibility details (hyperparameters, seeds)
- Use non-technical abstracts for interdisciplinary committees
- Maintain version control (Git) for all document drafts
- Ethical clearance documents must be archived for 5 years
- Viva success depends on connecting results to real-world impact
- Appendices should contain complete experiment logs

## Common Mistakes to Avoid
- Using informal language in mathematical proofs
- Overlooking dataset licensing requirements
- Poor figure resolution (<300 dpi) in printed copies
- Inconsistent numbering of equations/tables

## Revision Tips
- Practice peer review by exchanging drafts with classmates
- Use text-to-speech tools to identify awkward phrasing
- Create a checklist from DU's thesis submission guidelines
- Simulate viva sessions with timed presentations
