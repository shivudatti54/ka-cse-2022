# Learning Objectives

After studying this topic, you should be able to:

1. Define flow network, flow, and cut.
2. Explain the max flow problem and the min cut problem.
3. Describe the Ford-Fulkerson algorithm and the Edmonds-Karp algorithm.
4. Solve max flow and min cut problems using these algorithms.
5. Understand the relationship between the max flow problem and the min cut problem.
6. Find the minimum cut that corresponds to the maximum flow.
7. Apply these concepts to real-world problems.
8. Analyze the time complexity of the Ford-Fulkerson algorithm and the Edmonds-Karp algorithm.
