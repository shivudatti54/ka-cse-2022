# Learning Objectives

After studying this topic, you should be able to:

1. Analyze time/space complexity trade-offs between different shortest-path algorithms
2. Implement Johnson's algorithm with correct reweighting and path reconstruction
3. Prove correctness of Floyd-Warshall's dynamic programming formulation
4. Design hybrid algorithms for constrained shortest-path problems
5. Evaluate algorithm suitability for different graph types (sparse vs dense, weighted vs unweighted)
6. Modify traditional algorithms to detect and handle negative cycles
7. Apply contraction hierarchies to real-world routing problems
8. Critically assess recent research papers on shortest-path optimizations
