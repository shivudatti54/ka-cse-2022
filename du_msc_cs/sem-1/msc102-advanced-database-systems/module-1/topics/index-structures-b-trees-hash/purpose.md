# Learning Objectives

After studying this topic, you should be able to:

1. Analyze and compare time/space complexities of B-Trees vs Hash indexes
2. Implement B-Tree insertion/deletion algorithms with proper split/merge handling
3. Design extendible hashing schemes for growing datasets
4. Evaluate index structure choices for different workload patterns (OLTP vs OLAP)
5. Prove mathematical bounds for B-Tree height and hash collision probabilities
6. Critically assess recent research in learned indexes and hybrid structures
7. Optimize index parameters for modern storage hardware (SSDs, NVM)
8. Diagnose performance issues in real-world indexing implementations
