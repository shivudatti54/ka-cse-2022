# Learning Objectives

After studying this topic, you should be able to:

1. Formulate the ELBO objective and implement reparameterization trick in VAEs
2. Design generator/discriminator architectures for specific data modalities
3. Analyze mode collapse and propose modern mitigation strategies (e.g., spectral normalization)
4. Compare and contrast likelihood-based vs implicit generative models
5. Implement Wasserstein GAN with gradient penalty
6. Evaluate generated samples using FID score and precision/recall metrics
7. Adapt existing architectures for conditional generation tasks
8. Critically assess recent research papers in neural generative modeling
